5. **数据复盘** —— 蝉妈妈 / 飞瓜 / 抖音电商罗盘 / 巨量百应

## 6. 警示与边界(写入 SKILL.md 诚实边界)

1. **信息严重不平衡**:几乎全中文,几乎没有 quality 英文一手资料。海外 livestream commerce(Amazon Live / Whatnot / TikTok Shop)生态完全不同,不可迁移。
2. **营销号 + 课程营销占比极高(估 70%+)**:「9.9 转 99 课程」「七天速成主播」等大量噪音,真知识被掩埋。研究时狠筛课程导流内容。
3. **学派分歧大**,同问题不同派给完全不同答案:
   - 直播间纯运营派(场控 + 话术 + 节奏)
   - 流量投放派(千川 + 巨量为王)
   - 选品派(选对品就赢一半)
   - 私域沉淀派(公域只是入口)
   - 主播 IP 派(人设比技术重要)
4. **政策高频且严格**:网信办 2022 起「直播带货新规」+ 国家市场监管总局「明星 / KOL 广告法」+ 国家税务总局对头部主播税务调查(薇娅 2021 罚 13.4 亿 / 雪梨 / 林珊珊 / 邓伦 等)。**月度需重新校准。**
5. **头部腰部断层极端**:头部 0.1% 主播占行业 GMV ~30%+,腰部 + 尾部生存困难。SKILL.md 要诚实写「不适合 90% 想做主播的人」。
6. **数据可信度问题**:蝉妈妈 / 飞瓜 是估算,与抖音内部真实数据有 10-30% 偏差。后台「巨量百应」是真数据但仅商家本人可见。
7. **MCN / 切片 / 矩阵号 等灰色操作面广**:有真实运营策略,也有套路 / 假流量 / 假交易。研究时严格区分。
8. **无 quality 学术资料**:Track 06 academic 预计稀薄,直接在 SKILL.md 标注。
