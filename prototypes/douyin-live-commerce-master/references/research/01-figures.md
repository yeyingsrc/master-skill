## 详细 Figure 卡片

### 1. 罗永浩

- **One-liner**: 把「直播带货是个严肃的好生意」这件事用自己的个人信用背书给整个早期行业的人；他真正的方法论遗产不是「话术」，而是「不依赖单一主播 IP、靠机构品牌 + 重供应链 + 自研工具做直播」的去 IP 化路径。(evidence: [T01-S006, T01-S007])
- **核心身份**: 交个朋友创始人（2020 创立，2023 起逐步退出直播一线转向 AR/产品创业）。当前最相关身份：交个朋友的「奠基人 + 信用符号」，而非现役主播。
- **代表作品**:
  - 交个朋友直播间（2020-04 首播，3 小时 GMV 破 1.1 亿）
  - 「朋友云」——交个朋友自研的选品/洽谈/入驻/直播/复盘全链路线上化工具 (evidence: [T01-S007])
  - 「交个朋友」从 MCN 扩展到供应链 SaaS、品牌代运营等七大板块的机构化样板
- **值得读 / 听 / 看的 3 件事**:
  - 📖 「深度丨罗永浩直播这三年」21 世纪经济报道 https://www.21jingji.com/article/20230407/herald/99a0629851307ca43de1072159be4d52.html （三年完整复盘，⚠️ 二手长稿，非本人长文）
  - 🎙️ ⚠️ 未找到罗永浩本人系统谈直播方法论的长 podcast（他的长内容多在锤子/AR 创业话题上）
  - 🎬 ⚠️ 未找到方法论型 conference talk；可参考新京报专访「老赖罗永浩：我仍是一个理想主义创业者」https://m.bjnews.com.cn/detail/161901124715758.html
- **核心思想关键词**: 去 IP 化（做机构品牌不做单一主播）、重供应链（核心供应链 = 独家/绝对价格优势）、严肃生意（不是网红投机）、信用背书、爆品+主力款+人设款的排品结构
- **voice_samples**:
  - 客户/公众解释样本（对「直播带货是不是一门好生意」的解释，访谈摘录）：「直播起步的时候，虽然我说过，即便它是个严肃的生意，是个好生意，也不是我的兴趣方向。但在实际做的过程中，我获得了巨大的满足感，很快乐。」`(source: T01-S008, 原话——专访直接引语)`
  - 同业/行业讨论样本（谈这门生意的复杂度，访谈摘录）：「其实也没外界想象的那么快，而且里面涉及选品、排期、供应链、售后等方方面面还是很复杂的，难度不小，是个比较细致和费心的工作。」`(source: T01-S008, 原话——专访直接引语)`
  - 团队/操盘讨论样本（对供应链团队定位的描述，媒体转述其团队构成）：交个朋友的核心竞争力是 300 人规模的供应链团队，其中大量人做商务合同、资质审核，几十人做法务；一场直播要同时有「爆品（拉流量）+ 主力款（主播利润）+ 立人设/增信任的商品」三类货。`(source: T01-S006, 转述)`
  - ⚠️ 监管/专业讨论样本：未找到罗永浩公开系统谈监管/标准的 ≥50 字原话片段。
- **sub_skill_candidate**: `false` — 影响力和象征意义极高，但他本人**没有留下系统、自洽、可调研的方法论长材料**（长访谈多围绕锤子/AR/个人创业，直播部分是碎片金句）。真正承载交个朋友方法论的是黄贺、李牧人。罗永浩更适合在 Phase 2 接口列为「行业 founders / 奠基符号」。
- **dual_role**: `"founder + 信用符号"`（不是 thinker，他的价值在「示范」不在「体系」）
- **最近 12 个月动态**: 2024-2025 罗永浩本人持续淡出交个朋友直播一线，重心在 AR 创业（细红线）；交个朋友被反复讨论「不再需要罗永浩」——机构已能脱离他运转（这恰恰验证了他「去 IP 化」路径的成功）。(evidence: [T01-S007])
- **争议 / 批评**: ① 「交个朋友离了罗永浩还转不转」是持续争议——支持方认为去 IP 化成功，质疑方认为机构增长乏力、低价标签难撕；② 早期「真还传」叙事被部分人认为是营销人设。

  - [Secondary] 21 世纪经济报道「深度丨罗永浩直播这三年」https://www.21jingji.com/article/20230407/herald/99a0629851307ca43de1072159be4d52.html
  - [Secondary] 新京报专访「老赖罗永浩：我仍是一个理想主义创业者」https://m.bjnews.com.cn/detail/161901124715758.html
- **可信度自评**: medium — 影响力跨多源确认无疑，但缺「≥30min 本人系统输出方法论」的长材料（可调研性这一项是 ⚠️），按 3✅+1⚠️ 判 medium。

### 2. 董宇辉

- **One-liner**: 用「内容/人设信任本身就是带货发动机」证明了一条与「投流派/运营派」完全不同的路——全程不介绍产品、靠聊天和文化输出也能成交，他定义了「知识型超头」这一物种。(evidence: [T01-S009, T01-S010])
- **核心身份**: 与辉同行 CEO / 第一主播（2023 底从东方甄选「单飞」，与辉同行独立运营）。原新东方英语教师。
- **代表作品**:
  - 与辉同行直播间（单飞一年约 600+ 场直播，带货总额近百亿，2024 抖音最卖座直播间）(evidence: [T01-S010])
  - 「阅山河 / 破万卷 / 好读书 / 爱生活」四大 IP 内容栏目（文旅直播 / 读书 / 嘉宾访谈 / 带货）(evidence: [T01-S010])
  - 巴黎奥运会期间主持《运动健儿说》，访谈杨倩、苏炳添等冠军
- **值得读 / 听 / 看的 3 件事**:
  - 📖 「200亿带货先生董宇辉，可能是最后一个超头主播」界面新闻 https://www.jiemian.com/article/13869211.html （行业定位深度分析，⚠️ 二手）

## Phase 2 提炼提示

### 反复出现 ≥ 3 个 figures 都谈到的关键词（候选行业心智模型）

- **「去 IP 化 / 不赌单一大主播」**（出现于 figures: 罗永浩「做机构品牌」、黄贺「做号不做人」、李牧人「1+N 矩阵·大网红不可复制」、奥利「服务者·超头之外」、雷彬艺「多业务梯队」）—— **5 个 figure 命中，最强候选心智模型**。这一行被超头塌房（薇娅税案、小杨哥合规、董宇辉单飞）反复教育后的共识：个人 IP 是高风险资产，机构的护城河是「供应链 + 可复制矩阵 + 利益分配机制」而非某个人。
- **「重供应链 / 货是天花板」**（出现于 figures: 罗永浩「核心供应链=独家/绝对价格」、尹晨「货是天花板」、奥利「供应链基地+组货服务」，+ Track 04 canon「货是天花板」强共识）—— 候选心智模型：人和场只能在货的上限内优化，MCN/操盘手的真本事在「组货」不在「话术」。
- **「投流是结构性成本，不是增长开关」**（出现于 figures: 尹晨「千川=校准算法的付费数据」、贾乃亮案例「遥望投流 23 亿占采购 53.87% 仍亏」、李牧人「围绕流量+选品」，+ Track 04/06 强证据）—— 候选心智模型 / playbook：投流堆出的 GMV 不等于盈利，明星主播 + 巨额投流的模式（遥望）是反面教材。
- **「合规是生死线」**（出现于 figures: 罗永浩「几十人法务团队」、黄贺方法论六词含「合规」、小杨哥案例「6894.91 万罚款+停播」，+ Track 04/06 监管 canon）—— 候选心智模型：这一行的动作空间由立法+平台规则双重框定，头部的合规投入（罗永浩法务团队）是「能不能活」而非「要不要做」。
- **「抖音是大爆发逻辑 / 算法生意」**（出现于 figures: 尹晨「算法生意」、李牧人「3000 块到 3000 万的大爆发逻辑」，+ Track 04 canon「算法生意」）—— 候选心智模型：抖音直播带货的本质是「向算法证明直播间值得被推」，流量是非线性爆发的，核心变量是流量×选品。

### 显著分歧 / 流派分裂

intake 已点名 5 大流派，本 track 的 figures 给出了**人格化的流派代表**：

- **去 IP 化 / 机构矩阵派**（代表：罗永浩、黄贺、李牧人 — 交个朋友体系）vs **超头 IP 派 / 明星派**（代表：董宇辉 — 知识型超头、贾乃亮 — 明星主播、小杨哥 — 娱乐 IP）。前者认为「人是高风险资产、要去 IP 化」，后者的整个商业模式就建立在「人」上。**这是本 track 最尖锐的分歧**——而且「超头 IP 派」内部三个代表全都出过事（董宇辉翻车/焦虑、贾乃亮养不活遥望、小杨哥合规崩塌），间接给「去 IP 化派」背书。
- **知识 / 内容信任派**（代表：董宇辉 — 「全程不介绍产品也成交」）vs **运营 / 投流派**（代表：尹晨 — 「算法生意+千川校准」、李牧人 — 「流量+选品」）。前者认为内容/人设信任本身是发动机，后者认为发动机是「算法×投流×选品」的工程。
- **MCN「服务者」派**（代表：奥利 — 「服务主播也服务品牌、做基础设施」、雷彬艺 — 「找对路+分好钱」）vs **MCN「造星」派**（传统逻辑，代表：遥望早期 — 签 70 位明星）。服务者派把 MCN 做成「卖水的」，造星派把 MCN 押注在「淘金的人」身上。
- **平台官方叙事**（代表：魏雯雯 — 「全域兴趣电商/内容本质」、林旭灵 — 「三生态协同」）vs **一线操盘手现实**（代表：尹晨、李牧人 — 「算法博弈/投流脏活」）。Track 04/06 已标注这是 canon 内部的「明线 vs 暗线」——官方讲「健康、内容、长期」，操盘手知道「灰色操作真实存在且有效」。
