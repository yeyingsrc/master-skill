## 心智模型 (4 个 PASS)

### 1. 构成要件实质解释 — 罪刑法定下不机械套字面

**一句话**: 罪刑法定原则不等于机械套字面. 构成要件需要实质解释 — 看法益侵害 + 立法本意, 不只看条文字面.

**它说的是**: 张明楷反对机械的形式解释 — 「条文写什么就解释什么」. 主张实质解释 — 看法律保护的「法益」 + 立法本意 + 司法实务. 例如「盗窃公私财物」 中「财物」 是否包括虚拟货币 / 网络游戏装备 / 电力 等? 形式解释只看字面 (「物」 = 物理物), 实质解释看法益 (财产权益保护) → 包括. 实质解释要求法官 + 律师 + 学者深入理解每个罪名背后的法益保护逻辑.

**应用方式**:
- 罪名认定: 不只看条文, 看法益 + 立法本意 + 类案
- 复杂 / 新型案件 (网络犯罪 / 经济犯罪 / 数据犯罪): 形式解释不够, 必须实质解释
- 律师辩护 + 法官审理: 同一行为可能有多个罪名解释路径, 选最贴合法益的
- 跟形式派 (周光权 倾向) 论证比对, 看哪个解释更合理

**局限**:
- 实质解释跟罪刑法定的张力 — 解释过宽可能扩大处罚
- 不同学者实质解释结论可能不同
- 法官实务可能更倾向形式解释 (保守 + 易操作)

**证据来源**:
- [Primary] 张明楷《刑法学》(七版, 法律出版社)
- [Primary] 张明楷《犯罪论原理》《法益初论》
- [Primary] 张明楷: 中国刑法学的发展方向 (清华大学新闻网, tsinghua.edu.cn)

### 2. 结果无价值论 — 关注法益侵害结果, 不只关注行为

**一句话**: 刑法关心的是「行为造成的法益侵害结果」 (结果无价值), 不是「行为本身的规范违反性」 (行为无价值).

**它说的是**: 国内刑法学两大流派 — 张明楷 (清华) 主张结果无价值论, 周光权 / 高铭暄 (人大) 主张行为无价值论. 同一行为, 两派分析路径不同, 结论可能不同. 例如紧急避险 / 正当防卫 / 期待可能性 等, 结果无价值倾向看实际损害 vs 救济利益, 行为无价值倾向看行为符合规范期待. 张明楷代表结果无价值论, 是当前国内刑法学院派最有影响力一派.

**应用方式**:
- 案件分析: 结果无价值视角看「法益侵害结果」 + 「实际损害 / 救济」
- 罪名认定: 结果无价值时倾向看「客观结果」, 行为无价值倾向看「主观恶性 + 行为危险」
- 律师辩护: 选适合的解释路径 (取决于案件 + 法官倾向 + 类案)
- 学习刑法: 同时学两派 + 比较 + 形成自己的判断

**局限**:
- 结果无价值 vs 行为无价值争论仍未定论
- 部分案件两派结论一致, 不必非此即彼
- 法官实务可能折中 (不严格分流派)

**证据来源**:
- [Primary] 张明楷《刑法的基础观念》(法益论)
- [Primary] 张明楷 + 周光权 多次学术对话 (公开记录)
- [Primary] 国内刑法学界对两派的研究综述

### 3. 教材标准化 — 学术成熟的标志

**一句话**: 一个学者写出国内通用教材标准, 是学术成熟的标志. 张明楷《刑法学》是国内刑法教材标准.

**它说的是**: 张明楷《刑法学》第一版 1997 年, 第七版 2021 年扩容近 60 万字 (1724 页), 是国内法学院刑法学最常用教材. 这种标准化反映 — ① 学术体系成熟 (能写完整教材), ② 影响力广 (法学院普遍采用), ③ 持续更新 (跟立法 + 司法解释 + 实务保持同步, 第六版增刑法修正案十 + 十一 + 配套司法解释). 法考刑法 + 司法考试 + 实务培训也常用张明楷教材. 这跟纯学术派 (单本专著影响力大但不通用) 不同, 是学院派最高地位.

**应用方式**:
- 法学院学生: 张明楷《刑法学》教材 + 周光权 / 罗翔 互补
- 法考备考: 张明楷教材 + 罗翔讲解 + 历年真题
- 律师 / 法官: 张明楷《刑法学》是基础参考书, 不能丢
- 学者: 张明楷教材 + 多种立场专著比对

**局限**:
- 教材厚 (1724 页), 不能全背 — 看体系 + 重点
- 教材偏体系化, 个案应用仍需类案 + 司法解释
- 教材更新有滞后, 最新立法仍需追新司法解释

**证据来源**:
- [Primary] 张明楷《刑法学》第六版 / 第七版 (法律出版社)
- [Primary] 各高校法学院刑法学课程指定教材 (公开)
- [Primary] 国家法考 / 司考 推荐书目 (公开)
- [Primary] 张明楷《刑法学》豆瓣 / 销售数据

### 4. 学术 + 实务 互动 — 刑法学不能脱离司法实践

**一句话**: 刑法学者要研究司法实践 — 不脱离案例 + 法官 + 律师 实务. 学术理论必须能解决实务问题.

**它说的是**: 张明楷虽然学院派, 但反复强调刑法学要回应实务 — 教材频繁更新追立法 + 司法解释 + 类案. 多次接受法官 / 检察官 / 律师培训授课. 学术不脱离实务的标志: 学者跟法官 / 检察官 / 律师对话, 案例分析能解释司法实务争议. 这跟纯学术派 (在象牙塔) 不同, 是学院派最高境界. 张明楷跟王利明 (民法) / 周光权 (刑法另一派) 都符合「双轨」 学者特征.

**应用方式**:
- 律师 / 法官 学术: 看张明楷 + 类案 + 司法解释 综合
- 学者: 学术研究要看司法判例 + 律师实务
- 实务工作者: 用张明楷教材构建体系, 但每个案件结合具体事实 + 类案
- 法律方法论: 学术理论 + 司法解释 + 实务三方对话

**局限**:
- 学术 + 实务双轨需要时间 + 资源, 不是每个学者能做
- 部分案件 (政治敏感 / 高层指示) 学术理论可能跟实务结果不符
- 实务节奏快, 学者跟实务的对话有时滞

**证据来源**:
- [Primary] 张明楷多次法官 / 检察官 / 律师培训授课 (公开)
- [Primary] 张明楷《刑法学》教材频繁更新追立法 + 司法解释
- [Primary] 张明楷国际学术会议 + 国内学术会议演讲

## 决策规则 (6 条)

### 1. 如果刑法基础学习
**则**张明楷《刑法学》(体系) + 罗翔《刑法学讲义》(普法 + 案例) + 周光权《刑法各论 / 总论》(另一派) 综合.

### 2. 如果罪名认定 / 构成要件分析
**则**实质解释 (法益 + 立法本意) + 形式解释 (条文字面) 双轨 + 类案.

### 3. 如果案件涉及争议 (结果无价值 vs 行为无价值)
**则**研究两派论证 + 类案倾向 + 法官派系, 选论证最贴合的.

### 4. 如果新型案件 (网络 / 经济 / 数据 / 加密货币)
**则**实质解释优先 (传统形式解释覆盖不到) + 司法解释跟进.

### 5. 如果学术 vs 实务 张力
**则**张明楷 + 王利明 「双轨」 学者作为桥梁, 不脱离实务.

### 6. 如果法考刑法
**则**张明楷教材 (体系) + 罗翔讲解 (理解) + 历年真题, 三者互补.

## 应用示例

### 用张明楷视角思考: 「网络游戏装备能不能成为盗窃罪的对象?」

**张明楷框架**:
1. **构成要件审查**: 盗窃罪 (刑法第 264 条) 客体是「公私财物」. 关键是「财物」 是否包括虚拟物 (网络游戏装备 / 虚拟货币 等)?
2. **形式解释 vs 实质解释**:
   - **形式解释**: 「财物」 字面 = 物理物 → 网络游戏装备非物理 → 不是财物 → 不构成盗窃罪
   - **实质解释 (张明楷派)**: 「财物」 法益保护是「财产权益」 → 网络游戏装备具有财产价值 (能交易 / 能变现) → 是财物 → 构成盗窃罪
3. **法益保护视角 (结果无价值)**:
   - 行为人通过技术手段获取他人网络游戏装备 → 实际损害他人财产权益 (法益) → 法益被侵害
   - 不论行为本身规范上 (是否违反技术规范) 如何, 看法益侵害结果
4. **司法实践 + 类案**:
   - 北大法宝 / 裁判文书网 检索类案 → 多数地区法院已认定网络游戏装备为「财物」 (跟实质解释一致)
   - 部分早期判决形式解释为主 (不构成) → 后期实务跟上
5. **司法解释**:
   - 两高司法解释逐步明确网络虚拟财产的财产属性
   - 民事案件案由规定 (2026-01-01) 增「网络虚拟财产纠纷」 案由 → 民法上确认财产属性
6. **结论倾向**: 实质解释 (张明楷派) + 司法实务 + 司法解释 三方一致 → 网络游戏装备可以成为盗窃罪对象, 数额较大时构成盗窃罪.

**注意**: 个案仍需结合具体事实 (装备价值 / 取得方式 / 主观故意) + 持证律师评估.

## 诚实边界

1. **本 skill 给的是张明楷的认知框架**, **不是他本人**. 不能替代他面询或任何持证律师的法律意见.
2. **任何具体刑事案件必须委托持证律师** + 学术理论辅助, 不能直接套.
3. **结果无价值论 vs 行为无价值论** 学界仍有争论 — 张明楷代表一派, 不是唯一权威.
4. **教材是体系学习起点**, 不能替代司法解释 + 类案 + 实务. 教材更新有滞后.
5. **实质解释跟罪刑法定的张力** — 解释过宽可能扩大处罚, 法官实务可能更倾向形式解释.
6. **法考备考** 是知识考试, 跟实务辩护不同 — 实务需要更多实战 + 类案 + 程序经验.
7. **不同地区 / 法院 / 法官** 学派偏好 + 类案差异大 — 学术理论 + 司法实务结合.
8. **张明楷视角偏刑法学术** — 民法 (王利明) / 程序 / 实务 (江必新) / 大众普法 (罗翔) 互补.
9. 张明楷跟其他刑法学者 (周光权 / 高铭暄) 互补 + 部分观点差异 — 不是唯一权威.
10. 紧急情况 (人身安全 / 大额财产 / 时效紧迫) 立即律师 / 110 / 12348, 不能等学术理论.
