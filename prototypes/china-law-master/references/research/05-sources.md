## A. 数据库 + 法律检索 (6, 必备)

### A1. 国家法律法规数据库 (flk.npc.gov.cn)
- **频率**: 实时更新
- **机构**: 全国人大常委会
- **付费**: 免费
- **核心**: 立法源头, 全部法律 + 行政法规 + 司法解释 + 地方性法规
- **来源**: [Primary] flk.npc.gov.cn

### A2. 中国裁判文书网 (wenshu.court.gov.cn)
- **频率**: 持续更新
- **机构**: 最高人民法院
- **付费**: 免费 (受权限限制)
- **核心**: 全国裁判文书最大库
- **来源**: [Primary] wenshu.court.gov.cn

### A3. 北大法宝 (law.pkulaw.com)
- **频率**: 持续更新
- **机构**: 北京大学 + 北大英华
- **付费**: 律所 / 高校订阅
- **核心**: 国内最权威商业法律数据库
- **来源**: [Primary] law.pkulaw.com

### A4. 威科先行 (china.wolterskluwer.com)
- **频率**: 持续更新
- **机构**: Wolters Kluwer
- **付费**: 高于北大法宝

- **来源**: [Primary] wolterskluwer.com

### A5. 中国法学会专业研究会 (chinalaw.org.cn)
- **频率**: 不定期
- **机构**: 中国法学会
- **核心**: 各专业研究会 (民法 / 刑法 / 经济法 等) 学术资源
- **来源**: [Primary] chinalaw.org.cn

### A6. 中国知网 (CNKI) + 万方 + 维普 法学期刊
- **频率**: 持续更新
- **付费**: 高校 / 律所订阅
- **核心**: 法学期刊 + 学位论文 + 会议论文
- **来源**: [Primary] cnki.net

## G. 信源黑名单 (重要)

| 信源 | 问题 |
|------|------|
| 抖音 / 小红书 普法账号「打赢官司必胜」类 | 标题党 + 个案不能套 |
| 朋友圈个人律师「赢了多少案件」 | 选择性报告, 真实率不明 |
| 微信群 / QQ 群「免费法律咨询」(非持证律师) | 法律意见效力问题 |
| 「100% 包赢」「关系包搞定」类承诺 | 违反律师执业规范 |
| 山寨 / 仿冒法律网站 (非政府 / 非头部所) | 法规过时 / 错误 / 营销目的 |
| AI 直接出法律意见 | AI 不能签字 / 不能代理 + hallucination 风险 |
| 部分老旧法律书 (修正前版本) | 修正后法律已变, 老书过时 |
| 不知名「法律名家」 | 资质不明, 内容不可信 |
| 网络流传「秘密法条」「内部司法解释」 | 真实性可疑, 必查官方 |
| 部分自媒体「翻译」外国法律 | 翻译质量参差 + 文化差异忽略 |

## H. 验证方式

新法律 / 新司法解释 / 新案例出现时, 验证路径:

1. **flk.npc.gov.cn** 官方源头确认 (法律 / 司法解释)
2. **court.gov.cn / spp.gov.cn** 官方解读 (司法实务)
3. **多家头部所** 公开解读对照 (实务理解一致性)
4. **学术期刊** 检索 (中国法学 / 法学研究 等顶刊学者意见)
5. **类案** 检索裁判文书网 (司法实践)
6. **持证律师** 个案面询 (具体应用必查)
