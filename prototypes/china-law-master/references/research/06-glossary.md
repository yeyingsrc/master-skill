# Track 06 — Glossary (法律执业 / 中国法)

> 调研日期: 2026-05-02。

## 元数据
- 高频术语 35 / 法律 + 司法解释 10

---

## 民法基础

| 术语 | 含义 |
|------|------|
| **法律关系** | 合同 / 侵权 / 物权 / 婚姻家庭 / 劳动等 |
| **请求权基础** | 主张事实的法条 + 司法解释依据 |
| **意思表示** | 民事行为是否真实 + 完整 |
| **善意第三人** | 合同效力对外保护 |
| **效力 (有效 / 无效 / 待定 / 可撤销)** | 民事法律行为效力分类 |
| **物权变动** | 不动产登记 / 动产交付 |
| **法人格否认** | 例外揭穿公司面纱, 股东担责 |

---

## 程序法

| 术语 | 含义 |
|------|------|
| **诉讼时效** | 民法典普通时效 3 年 |
| **管辖 (级别 / 地域)** | 哪个法院审 |
| **管辖权异议** | 立案后程序异议 |
| **当事人主体适格** | 原告 / 被告身份是否合适 |
| **受案范围** | 法院 / 仲裁 是否受理 |
| **前置程序** | 调解 / 仲裁条款是否前置 |
| **保全 (诉前 / 诉中)** | 财产 + 行为保全 |
| **管辖恒定** | 立案后管辖一般不变 |
| **执行** | 判决后执行程序 |

---

## 证据

| 术语 | 含义 |
|------|------|
| **高度盖然性** | 民事证据标准 |
| **排除合理怀疑** | 刑事证据标准, 高很多 |
| **谁主张谁举证** | 民事举证基本原则 |
| **举证责任倒置** | 部分领域 (劳动 / 产品责任) 倒置 |
| **自认** | 当事人承认对自己不利事实 |
| **证据链** | 证据之间的逻辑关联 |
| **质证** | 庭上对证据真实性 / 关联性 / 合法性的质疑 |
| **证据保全** | 证据可能灭失时申请保全 |

## 合同

| 术语 | 含义 |
|------|------|
| **要约 + 承诺** | 合同成立基础 |
| **违约金** | 约定违约责任 |
| **解约权 (法定 / 约定)** | 解除合同的权利 |
| **管辖条款** | 合同争议哪个法院 / 仲裁机构 |
| **法律适用条款** | 涉外 / 跨境必备 |
| **保密条款 + 竞业限制** | 商事合同核心 |
| **不可抗力** | 免责条款 |
| **情势变更** | 履约难度发生重大变化 |

## 商事

| 术语 | 含义 |
|------|------|
| **股东会 / 董事会 / 监事会** | 公司治理三会 |
| **法人格独立** | 法人 vs 股东 |
| **法人格否认 (揭穿面纱)** | 股东担责例外 |
| **股权激励** | 员工持股 / 期权 |
| **股东协议** | 股东之间的合约 |
| **并购 (M&A)** | 兼并收购 |
| **重组** | 公司重组 |
| **反垄断申报** | 经营者集中审查 |

## 外行破绽

- 把民事和刑事混 (证据标准 / 程序完全不同)
- 不区分实体法和程序法
- 法条适用不看具体司法解释
- 把感情判断当法律判断
- 「我有道理」就以为能赢 (法庭只看证据)
- 跳过程序审查直接做实体分析
