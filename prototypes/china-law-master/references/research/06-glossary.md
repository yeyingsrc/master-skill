# Track 06 — Glossary (中国法律执业)

> 调研日期: 2026-05-03 (deep upgrade). **严肃声明**: 仅供执业律师参考, 不替代法律意见. 中国大陆法语境.

## 元数据
- 高频术语 100 / 标准 + 司法解释 12
- 一手比例 ≈ 78%

---

## A. 民法术语 (35)

### A1. 总则 (10)
| 术语 | 含义 |
|------|------|
| **民事主体** | 自然人 / 法人 / 非法人组织 |
| **民事法律行为** | 设立 / 变更 / 终止民事权利义务的合法行为 |
| **代理** | 法定 / 委托 / 指定 |
| **诉讼时效** | 一般 3 年, 短期 1 年, 最长 20 年 |
| **除斥期间** | 形成权行使期, 不中断 / 不延长 |
| **善意取得** | 第三人善意 + 合理价格 + 完成登记 / 交付 |
| **公序良俗** | 公共秩序 + 善良风俗, 民事行为基本边界 |
| **诚实信用** | 民法基本原则之一 |
| **意思表示** | 民事法律行为核心 |
| **法律行为效力** | 有效 / 无效 / 可撤销 / 效力待定 |

### A2. 物权 (8)
| 术语 | 含义 |
|------|------|
| **物权法定** | 物权种类 + 内容由法律规定, 不能私设 |
| **登记对抗** vs **登记生效** | 不动产登记效力区分 |
| **抵押 / 质押 / 留置** | 担保物权三种 |
| **善意取得 (物权)** | 不动产 / 动产善意取得 |
| **共有** | 按份共有 + 共同共有 |
| **建筑物区分所有权** | 业主对专有 + 共有部分权利 |
| **地役权** | 利用他人不动产 |
| **居住权** | 民法典新增 (居住但不所有) |

### A3. 合同 (10)
| 术语 | 含义 |
|------|------|
| **要约 + 承诺** | 合同订立 |
| **格式条款** | 重复使用 + 单方拟定 + 不能协商 |
| **合同效力** | 有效 / 无效 / 可撤销 / 效力待定 |
| **不可抗力** | 不能预见 + 不能避免 + 不能克服 |
| **情势变更** | 客观情况重大变化致合同基础动摇 |
| **预期违约** | 履行期前明确表示不履行 |
| **后合同义务** | 通知 / 协助 / 保密 |
| **缔约过失** | 违反先合同义务 |
| **担保** | 保证 / 抵押 / 质押 / 定金 / 留置 |
| **违约责任** | 继续履行 / 赔偿损失 / 违约金 / 定金 |

### A4. 侵权 (4)
| 术语 | 含义 |
|------|------|
| **过错责任** | 一般侵权, 须证明过错 |
| **无过错责任** | 法定特殊情形 (产品 / 高度危险 / 环境 等) |
| **公平责任** | 双方都无过错时分担损失 |
| **共同侵权** | 多人共同造成损害 |

### A5. 人格权 + 婚姻继承 (3)
| 术语 | 含义 |
|------|------|
| **人格权** | 民法典独立成编 (王利明立法贡献) |
| **离婚冷静期** | 协议离婚 30 日 (民法典) |
| **遗嘱形式** | 公证 / 自书 / 代书 / 录音 / 录像 / 打印 / 口头 |

## B. 刑法术语 (25)

### B1. 总则 (10)
| 术语 | 含义 |
|------|------|
| **罪刑法定** | 法律明文规定为犯罪行为的, 才能定罪量刑 |
| **构成要件** | 主体 / 客体 / 主观 / 客观 (四要件) |
| **主观方面** | 故意 / 过失 / 意外 |
| **客观方面** | 行为 / 结果 / 因果 |
| **正当防卫** | 紧迫 + 必要 + 限度 |
| **紧急避险** | 紧迫 + 不得已 |
| **共同犯罪** | 主犯 / 从犯 / 教唆犯 / 胁从犯 |
| **未遂 / 既遂** | 着手实施但未得逞 vs 完成 |
| **结果无价值论** vs **行为无价值论** | 张明楷派 vs 周光权派 |
| **自首 / 立功** | 量刑酌定情节 |

|------|------|
| **故意杀人 / 故意伤害** | 暴力犯罪 |
| **盗窃 / 抢劫 / 抢夺** | 财产犯罪 |
| **诈骗 / 合同诈骗** | 财产犯罪 |
| **挪用资金 / 职务侵占** | 单位人员职务犯罪 |
| **贪污 / 受贿 / 行贿** | 国家工作人员职务犯罪 |
| **非法经营** | 经济类犯罪 |
| **非法吸收公众存款 / 集资诈骗** | 金融犯罪 |
| **强奸 / 强制猥亵** | 性犯罪 |
| **绑架 / 拐卖** | 侵犯公民人身权利 |
| **危险驾驶 / 交通肇事** | 公共安全 |
| **走私** | 走私货物 / 物品 / 武器 / 文物 |
| **生产销售伪劣商品** | 经济犯罪 |
| **侵犯知识产权** | 假冒 / 侵犯著作权 / 侵犯商业秘密 |
| **网络犯罪** | 非法侵入 / 帮助网络 (帮信罪) |

## C. 程序法术语 (20)

### C1. 民事诉讼 (10)
| 术语 | 含义 |
|------|------|
| **起诉 / 应诉** | 诉讼启动 |
| **管辖** | 级别 + 地域 + 专属 + 协议 |
| **诉讼参与人** | 当事人 / 代理人 / 第三人 / 证人 |
| **举证责任** | 谁主张谁举证 (一般原则) |
| **质证** | 对证据的真实性 / 合法性 / 关联性发表意见 |
| **庭前会议** | 重大复杂案件法院召开 |
| **简易程序** vs **普通程序** | 案件难易 + 标的额 决定 |
| **小额诉讼** | 一审终审 (≤ 2 万 / 各地标准) |
| **保全** | 诉前 / 诉中 / 财产 / 行为 |
| **执行** | 一审生效 + 申请 + 强制执行 |

### C2. 刑事诉讼 (10)
| 术语 | 含义 |
|------|------|
| **侦查 / 起诉 / 审判** | 三阶段 |
| **强制措施** | 拘传 / 取保 / 监居 / 拘留 / 逮捕 |
| **嫌疑人 / 被告人 / 罪犯** | 三阶段身份 |
| **辩护人** | 律师 / 法定代理人 |
| **认罪认罚从宽** | 认罪 + 认罚 → 量刑减轻 |
| **不起诉** | 法定 / 酌定 / 存疑 |
| **证据排除规则** | 非法证据排除 (刑讯逼供 等) |
| **死刑复核** | 最高人民法院核准 |
| **二审 / 申诉** | 一审不利时上诉 / 已生效判决申诉 |
| **审判监督** | 已生效判决再审 |

## D. 仲裁 + 涉外术语 (10)

| 术语 | 含义 |
|------|------|
| **仲裁条款** | 选仲裁需明确机构 + 适用法律 + 仲裁地 |
| **一裁终局** | 仲裁不能上诉, 仅撤销 / 不予执行 |
| **仲裁庭** | 单 / 三人, 各方指定 |
| **CIETAC / BAC / SHIAC / SCIA** | 国内主要仲裁机构 |
| **ICC / SIAC / HKIAC / LCIA** | 国际仲裁机构 |
| **New York Convention** | 仲裁裁决跨境承认执行 (中国 1987 加入) |
| **海牙公约 / Apostille** | 文书跨境认证 (2023-11-07 对中国大陆生效) |
| **领事认证** | 海牙公约前的文书认证方式 |
| **强制性规定** | 不能由当事人协议排除的法律规定 |
| **国际商事惯例** | UNIDROIT 商事合同通则 等 |

## E. 律师执业术语 (10)

| 术语 | 含义 |
|------|------|
| **执业证 + 律所** | 持证律师必备 |
| **委托代理合同** | 律师跟客户签订, 明确范围 + 费用 |

| **律师调查令** | 法院授权律师调查取证 |
| **法律意见书** | 律师正式文书, 签字 + 盖章 |

## G. 外行破绽 (15)

1. 「法律」泛指 vs 法律严格定义
2. 不查时效 (一般 3 年 / 短期 1 年)
3. 不审管辖 (立错法院 → 移送 / 重立)
4. 「合同写了就有效」 (受民法典 + 司法解释 + 强制性规定约束)
5. 「赢官司就拿到钱」 (执行难是常态)
6. 「打官司便宜」 (诉讼费 + 律师费 + 鉴定费 + 公证费 + 时间)
7. 「自己打官司省律师费」 (复杂案件律师价值远高于费用)
8. 「认罪从宽就一定轻」 (失去辩护空间)
9. 「微信聊天就是证据」 (电子证据效力受规则限制)
10. 「家里有人是法官 / 律师就行」 (司法独立 + 利益回避)
11. 「公证就万能」 (强制执行公证仅限特定情形)
12. 「仲裁就是私了」 (仲裁是法定程序)
13. 「网上抄来的合同模板」 (公开模板可能漏关键条款)
14. 「AI 法律咨询」 (AI 不出法律意见)
15. 「罗翔讲过」 (普法 ≠ 个案法律意见)
