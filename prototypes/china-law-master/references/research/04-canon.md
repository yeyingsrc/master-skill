## 必读 (基础法律 + 实务)

### 1. 《中华人民共和国民法典》(2020 年公布, 2021 年实施)
- **核心**: 中国民法基础法
- **价值**: 任何民商事执业必读
- **更新**: 持续司法解释更新

### 2. 《最高人民法院关于民事诉讼证据的若干规定》
- **核心**: 民事证据规则核心
- **价值**: 证据规则 + 举证责任分配

### 3. 《中华人民共和国民事诉讼法》
- **核心**: 民事程序法基础
- **价值**: 程序正义底层

### 4. 《刑法》+《刑事诉讼法》(刑辩相关)
- **核心**: 刑事执业基础
- **注**: 本 skill 重点不在刑法, 但需要基础认知

### 5. 王利明《民法学》系列教材
- **核心**: 中国民法学经典教材
- **价值**: 理论 + 体系化

### 6. 江平《罗马法基础》
- **核心**: 大陆法系法理学基础
- **价值**: 法理思想根基

### 7. 君合 / 中伦 / 金杜内训材料 (公开版)
- **核心**: 头部商事律所实务方法论
- **价值**: 大型案件实操参考

### 8. 罗翔《刑法学讲义》(普法版)
- **核心**: 法律普及典范
- **价值**: 法律思维训练入门
- **注**: 不替代专业刑法教科书

### 9. 《最高人民法院司法解释》全集
- **核心**: 司法实践指引
- **价值**: 实务操作必查

### 10. 中国法学会研究会出版物
- **核心**: 各专门法领域研究
- **价值**: 商法 / 知产 / 反垄断 等专项

## 核心概念 (30)

### 民法基础
- **请求权基础**: 主张某个事实的法条 + 司法解释依据
- **法律关系**: 合同 / 侵权 / 物权 / 婚姻家庭 / 劳动等
- **意思表示**: 民事行为基础, 是否真实 + 完整
- **效力**: 有效 / 无效 / 效力待定 / 可撤销
- **善意第三人**: 合同效力对外保护

### 程序法
- **诉讼时效**: 民法典普通时效 3 年
- **管辖**: 级别管辖 + 地域管辖
- **当事人主体适格**: 原告 / 被告身份是否合适
- **受案范围**: 法院 / 仲裁 是否受理
- **前置程序**: 调解 / 仲裁条款是否前置
- **管辖权异议**: 立案后程序异议
- **保全 (诉前 / 诉中)**: 财产 + 行为保全

### 证据
- **高度盖然性**: 民事证据标准, 「比反方更有可能」即可
- **排除合理怀疑**: 刑事证据标准, 高很多
- **谁主张谁举证**: 民事举证基本原则
- **举证责任倒置**: 部分领域 (如劳动 / 产品责任) 倒置
- **自认**: 当事人承认对自己不利的事实
- **证据链**: 证据之间的逻辑关联

### 合同
- **要约 + 承诺**: 合同成立基础
- **违约金**: 约定违约责任
- **解约权**: 法定 / 约定
- **管辖条款 / 法律适用条款**: 涉外 / 跨境必备
- **保密条款 + 竞业限制**: 商事合同核心
- **不可抗力 + 情势变更**: 风险分配条款

### 商事
- **公司治理**: 股东会 / 董事会 / 监事会
- **公司法人格独立**: 法人 vs 股东
- **法人格否认 (揭穿公司面纱)**: 例外情况下股东担责
- **股权激励 + 股东协议**: 商事核心
- **并购 + 重组**: M&A 实务

|------|---------|

| **风险防范派** | 商事律师价值在前期合规 |
