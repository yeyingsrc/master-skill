# Track 02 — Tools (中国法律执业)

> 调研日期: 2026-05-03 (deep upgrade). 法律检索 / 文书 / 证据 / 诉讼 / 仲裁 工具. **严肃声明**: 工具不能替代律师专业判断 + 个案审查.

## 元数据
- 必备 6 / 场景特化 8 / 新兴 5 = 19 工具
- 一手比例 ≈ 78%

## A. 必备 (法律检索 + 文书 + 沟通, 6 个)

### A1. 国家法律法规数据库 (flk.npc.gov.cn)
- **One-liner**: 全国人大官方法律 + 行政法规 + 司法解释 + 地方性法规 全文检索
- **Tier**: 必备 (法律源头)
- **典型场景**: 任何法律意见 + 诉讼 + 合规 第一站
- **价格**: 免费公开
- **优势**:
  - 立法源头, 权威性最高
  - 历次修订对照
  - 立法说明 + 配套司法解释链接
- **避坑**:
  - UI 一般, 检索体验不如商业法律数据库
  - 部分地方性 / 部门规章不全
- **来源**: [Primary] flk.npc.gov.cn

### A2. 北大法宝 (Pkulaw, law.pkulaw.com)
- **One-liner**: 国内最权威商业法律数据库 — 法规 + 案例 + 学术 + 实务汇总
- **Tier**: 必备 (国内执业律师标配)
- **典型场景**:
  - 法律检索 (含历次修订 / 司法解释 / 地方法规 / 部门规章)
  - 案例检索 (人民法院公报 / 指导性案例 / 各地裁判文书)
  - 学术资料 (论文 / 专著)
  - 法律新闻 + 法规变动追踪
- **价格**: 律所 / 高校订阅 (~ ¥几万 / 年)
- **优势**: UI 好 + 检索强大 + 关联推荐 + 知识图谱
- **避坑**:
  - 商业产品, 部分内容延迟 (vs 立法源头)
  - 个人订阅贵, 多通过律所共享
- **来源**: [Primary] law.pkulaw.com

### A3. 中国裁判文书网 (wenshu.court.gov.cn)
- **One-liner**: 最高人民法院全国裁判文书公开 — 司法实务真实案例最大库
- **Tier**: 必备 (案例研究 + 同类案件参考)
- **典型场景**:
  - 类案检索 (类似案件法院怎么判)
  - 法官说理研究
  - 客户案件预测
  - 学术研究
- **价格**: 免费 (受权限限制)
- **避坑**:
  - 公开范围近年收紧 — 不是全部上网, 部分敏感 / 涉密 / 隐私案件不公开
  - 公开延迟 (3-12 月)
  - 部分案件文书不全
- **来源**: [Primary] wenshu.court.gov.cn

### A4. 威科先行 (Wolters Kluwer China — china.wolterskluwer.com)
- **One-liner**: 国际化商业法律数据库 — 中外法律 + 涉外业务
- **Tier**: 必备 (涉外业务 + 大所)
- **典型场景**: 涉外商事 / 跨境交易 / 双语法律检索
- **价格**: 律所订阅 (高于北大法宝)
- **来源**: [Primary] wolterskluwer.com

### A5. 文书 + 合同 起草工具
- **One-liner**: Word + Excel + 专业合同库 + Track changes
- **Tier**: 必备
- **典型场景**: 合同起草 / 法律意见书 / 起诉状 / 答辩状 / 仲裁申请 / 律师函
- **关键技巧**:
  - **Track changes** (修订模式) 跨稿协作必备
  - **Compare** (比较) 双方版本差异
  - **Cross-reference** 长合同条款互引
  - 合同模板库 (律所 / 公开资源)
- **避坑**: 邮件直接发送 .docx 暴露元数据 (修订记录 / 注释), PDF 化前必须清除
- **来源**: [Primary] Microsoft Office + 律所合同库

### A6. 律师沟通 + 电子签 工具
- **One-liner**: 微信 + 钉钉 + 邮箱 + 电子签 (e签宝 / 法大大)
- **Tier**: 必备
- **典型场景**:
  - 客户沟通 (微信 + 邮件)
  - 律所内部协同 (钉钉 / 飞书)
  - 电子合同签署 (e签宝 / 法大大 / 上上签)
- **避坑**:
  - 微信 / 钉钉 沟通有保密风险 (敏感案件用加密邮件 + 必要时面谈)
  - 电子签必须用具备法律效力的平台 (实名认证 + 证书认证 + 时间戳)
  - 电子证据保全 (区块链存证) 跟一般电子签不同
- **来源**: [Primary] 各平台合规资质

## B. 场景特化 (8 个)

### B1. 律师调查令 + 证据收集
- **One-liner**: 法院授权律师调查 (银行 / 房产 / 通信 / 户籍 等)
- **典型场景**: 民商事诉讼 + 部分刑事辩护
- **关键技巧**:
  - 申请前先评估必要性 + 主张关联
  - 调查令对象需明确 (银行 / 工商 / 房产 / 通信 / 公安 等)
  - 部分单位仍需律师函 + 证明文件
- **来源**: [Primary] 民事诉讼法 / 民事证据规定 + 各地法院实施细则

### B2. 司法鉴定 (笔迹 / 印文 / 财务 / 医疗 / 工程 等)
- **One-liner**: 民事 / 刑事 / 行政诉讼证据
- **典型场景**: 笔迹真伪 / 印章真伪 / 财务审计 / 医疗事故 / 工程造价 等
- **价格**: ¥3000 - ¥几十万 (案值相关)
- **避坑**:
  - 鉴定机构必须 / 鉴定人有资质
  - 鉴定材料保全严格 — 链条断 = 证据无效
  - 鉴定意见可申请重新鉴定 (符合法定情形)
- **来源**: [Primary] 全国统一司法鉴定信息平台 + 司法部

### B3. 公证 (Notarization)
- **One-liner**: 公证机构对法律行为 / 文书 / 事实 出具具有证据效力 + 强制执行效力 的证明
- **典型场景**:
  - 强制执行公证 (借款合同 / 租赁合同) — 不经审判直接执行
  - 证据保全公证 (网页截图 / 现场固定 / 物品保全) — 重要电子证据保全
  - 涉外公证 (海外结婚 / 出生 / 学位 / 财产)
  - 房屋买卖 / 继承 / 委托 公证
- **价格**: 几百到几万
- **来源**: [Primary] 中国公证协会 + 司法部

### B4. 仲裁 (Arbitration)
- **One-liner**: 民商事争议替代诉讼的解决方式 — 一裁终局
- **代表机构**:
  - **CIETAC** (中国国际经济贸易仲裁委员会, cietac.org) — 涉外商事 + 国内
  - **北京仲裁委员会** (BAC, bjac.org.cn) — 国内商事
  - **上海国际仲裁中心** (SHIAC, shiac.org)
  - **深圳国际仲裁院** (SCIA, scia.com.cn)
- **典型场景**: 商事合同 / 涉外 / 投资 / 工程 / 知识产权
- **避坑**:
  - 仲裁条款必须明确 (机构 + 适用法律 + 仲裁地)
  - 一裁终局 — 不能上诉, 仅能申请撤销 / 不予执行
  - 涉外仲裁结果跨境执行需走 New York Convention
- **来源**: [Primary] CIETAC / BAC / SHIAC / SCIA 官网 + 仲裁法

### B5. 互联网法院 (Internet Courts)
- **One-liner**: 北京 / 杭州 / 广州 互联网法院 — 网络案件集中审理
- **典型场景** (2025-11-01 新规):
  - 网络数据 / 知识产权 / 合同纠纷
  - 网络人格权 + 隐私权
  - 网络虚拟财产
  - 网络不正当竞争
  - 网络域名纠纷
- **优势**: 全程在线 / 区块链证据 / 智能合约 等数字化
- **避坑**:
  - 管辖范围有限 — 不是所有网络案件都归互联网法院
  - 2025-11-01 新规 调整管辖 (基层一审为主)
- **来源**: [Primary] 最高人民法院 互联网法院案件管辖规定 (2025-11-01 起施行)

### B6. 法律 AI 工具 (国内)
- **代表**: 通义千问 / 文心一言 / 豆包 / 智谱清言 / 法律垂直 AI (LawyerAI 等)
- **典型场景**: 法律检索辅助 / 起草辅助 / 案例摘要 / 论文综述
- **避坑**:
  - **AI 不能出法律意见** — 不能签字 / 不能代理
  - **AI 输出必须人工核校** — 法条 / 案例 / 数字 / 时效
  - **AI 编造法条 / 案例风险** (hallucination) — 美国 ChatGPT 编案例事件已多次发生
  - **保密 + 数据安全** — 客户信息不能传给公网 AI
  - 所内私有部署 / 加密 AI 是更好选择
- **来源**: [Primary] 各平台合规资质 + 行业实务指引

### B7. 区块链 + 电子证据存证
- **代表**: 杭州互联网法院司法链 / 北京互联网法院天平链 / 司法行政机关认可平台
- **典型场景**: 网络著作权 / 合同 / 网页 / 取证
- **优势**: 时间戳 + 哈希值不可篡改, 法院采信度高
- **避坑**:
  - 必须用法院认可平台
  - 存证不等于证据 — 仍需关联证明
- **来源**: [Primary] 杭州 / 北京互联网法院司法链文件

### B8. 涉外业务工具
- **代表**:
  - **HCCH** (海牙国际私法会议) 公约文本
  - **UNCITRAL** 国际贸易法委员会
  - **New York Convention** (纽约公约) 仲裁裁决跨境执行
  - **领事认证 / Apostille** (海牙公约 2023-11-07 起对中国大陆生效)
  - **WIPO** 全球知识产权
- **典型场景**: 跨境合同 / 投资 / 仲裁裁决执行 / 知识产权
- **来源**: [Primary] HCCH / UNCITRAL / WIPO 公约文本 + 中华人民共和国对外缔约公约公开

## D. 选型决策树

| 任务 | 工具优先级 |
|------|----------|
| 法律检索 | 国家数据库 (flk.npc.gov.cn) → 北大法宝 / 威科先行 |
| 类案检索 | 裁判文书网 → 北大法宝案例库 → 类案推送 (各平台) |
| 文书起草 | Office + Track changes + 合同模板库 + 必要时 AI 辅助 (人工核校) |
| 客户沟通 | 邮件优先 (留痕) + 微信 / 钉钉 (敏感面谈) + 文档共享平台 |
| 证据收集 | 律师调查令 + 公证保全 + 司法鉴定 + 区块链存证 (网络) |
| 商事争议 | 仲裁条款看清 — 仲裁 vs 诉讼; 国际 → CIETAC / 涉外仲裁 |
| 网络案件 | 互联网法院 (符合管辖) + 区块链存证 |
| 涉外案件 | 领事认证 / Apostille + 海牙公约 + New York Convention |

## E. 避坑总览

1. **国家数据库优先于商业数据库** — 立法源头, 权威性最高
2. **裁判文书网公开有限** — 不是全部上网, 部分敏感不公开
3. **AI 不能出法律意见** — 律师责任不能转移
4. **微信 / 钉钉 沟通敏感案件保密风险** — 必要时面谈
5. **电子签必须有法律效力平台** (实名 + 证书 + 时间戳)
6. **仲裁条款必须明确** (机构 + 适用法律 + 仲裁地) + 一裁终局
7. **互联网法院管辖有限** — 2025-11-01 新规调整
8. **跨境执行复杂** — 仲裁走纽约公约 / 判决走双边条约 + 海牙公约
9. **司法鉴定材料链** 断 → 证据无效
10. **公证强制执行** 仅特定情形 — 不是所有合同都能直接执行
11. **AI hallucination 风险** — 编造法条 / 案例必须人工核校
12. **元数据泄露** — Word 修订记录 / 邮件原始发件人 / PDF 元数据 PDF 化前清除
