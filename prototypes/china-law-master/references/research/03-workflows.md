# Track 03 — Workflows (法律执业 / 中国法)

> Mini-viable for prototype. 严肃声明: 仅供执业律师参考, 不替代专业法律意见.

### 1. 民商事案件立案前评估

- **One-liner**: 从客户找上门到决定是否立案的标准动作
- **Trigger**: 客户主张需要诉讼解决纠纷
- **Output**: 立案 / 不立案决定 + 风险评估意见书
- **入门 SOP**:
  1. 客户访谈 — 主张事实 + 时间线 + 已有证据清单
  2. 法律关系定性 — 合同 / 侵权 / 物权 / 婚姻家庭 等
  3. 程序前置审查 — 时效 / 管辖 / 主体 / 受案范围 / 仲裁条款
  4. 证据评估 — 是否充分 / 链条闭合度 / 缺口
  5. 实体分析 — 请求权基础 / 法条 + 司法解释 / 类似案例
  6. 出立案前评估意见 — 推荐立案 / 暂缓 / 不立案 + 三种情景预测
- **资深路径**:
  - skip 实体分析在程序审查之前 (新手最常见错), 资深永远先程序后实体
  - optimize 类似案例检索用「裁判文书网 + 北大法宝」交叉, 不只看一家
  - add 给客户三种情景预测 (最佳 / 最差 / 中位), 不只是「能赢能输」
- **常见失败模式**:
  - 没做时效审查就立案 → 对方一抗辩就崩
  - 证据不足硬主张事实 → 反诉风险
  - 给客户百分百能赢承诺 → 出问题客诉 + 律师执业风险
- **Last_updated**: 2025-07-15
- **Decay risk**: low (程序规则相对稳定, 司法解释偶有更新)

### 2. 商事合同审核流程

- **One-liner**: 客户拿合同要求律师审核到出意见的标准动作
- **Trigger**: 客户已经有合同草本 (自己写 / 对方给) 需要审核
- **Output**: 风险点清单 + 修改建议 + 必改 vs 建议改优先级
- **入门 SOP**:
  1. 看合同主体 + 类型 (买卖 / 服务 / 借贷 / 保密 / 涉外)
  2. 重点审违约救济条款 — 违约金 / 解约权 / 管辖 / 法律适用
  3. 审权利义务对等性 — 客户的权利是否充分 / 对方义务是否具体
  4. 审风险分配 — 不可抗力 / 第三方责任 / 知识产权
  5. 审涉外因素 (如适用) — 法律适用 + 争议解决地 + 语言版本优先
  6. 出意见 — 必改 vs 建议改 vs 接受 三档分级
- **资深路径**:
  - skip 删错别字 / 改格式 (新手最常见错, 资深聚焦风险条款)
  - optimize 用合同审核 checklist (按类型不同) 提高效率
  - add 给客户「这家对方 / 行业是否标准 / 能不能改」的市场判断
- **常见失败模式**:
  - 只看权利义务不看违约救济 → 履行出问题没救济路径
  - 涉外合同没审法律适用条款 → 争议时管辖混乱
  - 不分级给客户全部修改建议 → 客户不知道哪个必改
- **Last_updated**: 2025-08-30
- **Decay risk**: low
