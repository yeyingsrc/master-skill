# Track 03 — Workflows (中国法律执业)

> 调研日期: 2026-05-03 (deep upgrade). **严肃声明**: 仅供执业律师参考, **绝不替代律师对个案的法律意见**. 中国大陆法语境.

## 元数据
- 完整工作流 4 个 (民商事案件 / 合同审核 / 刑事辩护 / 涉外争议)
- 一手比例 ≈ 78%

---

### 1. 民商事案件立案前评估

- **One-liner**: 从客户咨询到决定是否代理 + 立案的标准流程 (案值 / 时效 / 管辖 / 证据 / 费用 风险评估)
- **Trigger**: 客户带着主张 / 纠纷找律师, 询问能否打 / 怎么打
- **Output**: 代理决策 (接 / 不接) + 立案策略 (管辖 / 时效 / 证据 / 主张) + 客户费用风险预告

- **入门 SOP**:
  1. **接谈记录** — 时间 / 当事人 / 纠纷概述 / 客户期望 / 是否其他律师代理过 / 是否签订风险代理. 留痕签字
  2. **案件四要素** —
     - **管辖** (哪个法院): 一般原告就被告 + 合同约定管辖 + 不动产专属 + 涉外 / 互联网 等特殊
     - **时效** (诉讼时效): 一般民事 3 年, 短期 1 年 (人身损害), 最长 20 年 (绝对). 商事仲裁不同
     - **证据** (有什么 / 没什么): 客户能拿出的 + 需要调查 + 鉴定 / 公证 必要性
     - **主张** (法条依据 + 案由): 民法典 / 单行法 / 司法解释 / 案由规定 (2026-01-01 修订增数据 / 网络虚拟财产)
  3. **法律检索** — flk.npc.gov.cn (法源) + 北大法宝 (司法解释 + 案例) + 裁判文书网 (类案)
  4. **证据评估** — 现有证据是否充分 + 待查证据可行性 + 司法鉴定 / 公证 必要性
  5. **风险评估** —
     - **胜诉概率** (基于类案 + 证据 + 法律状态)
     - **执行风险** (被告有可执行财产? 不能执行 = 赢了也白赢)
     - **诉讼成本** (诉讼费 + 律师费 + 鉴定费 + 时间)
  6. **代理决策** — 接 / 不接 (接) 风险代理 vs 普通代理 vs 计时
  7. **立案策略** — 起诉状起草 + 证据材料 + 立案登记 + 缴费 + 受理
  8. **诉讼进展** — 答辩 + 举证 + 质证 + 庭审 + 判决 + 执行 (各阶段策略)

- **资深路径**:
  - **skip 客户讲什么就信什么** — 资深要看证据 + 客观核校
  - **skip 类案不查直接评估** — 资深必查类案 + 法官说理
  - **optimize 代理范围明确** — 委托代理合同写清服务范围 + 分阶段计费
  - **add 不接的案件提供咨询服务** — 不一定接代理, 但可以收咨询费
  - **add 「程序先于实体」** — 时效 / 管辖 / 主体 任何一个错 → 实体输
  - **add 调解优先** — 多数民商事案件调解结案比判决经济 + 快
  - **optimize 风险代理审慎** — 不是所有案件可风险代理 (法律明确禁止类型)
  - **add 客户期望管理** — 「赢」不等于「全赢」, 部分赢 / 调解 / 撤诉 都是结果

- **常见失败模式**:
  - **时效错过** — 一般 3 年 / 短期 1 年 错过, 实体上赢了形式上输
  - **管辖错误** — 立错法院 → 移送 / 重新立案, 浪费时间 + 错失程序优势
  - **主体不适格** — 原告 / 被告 / 第三人 错 → 案件被驳回
  - **证据不足 + 没调查** — 客户说的 ≠ 法律事实, 没证据等于没事实
  - **客户期望失控** — 律师讲不清「赢」的范围, 客户落差大
  - **执行风险评估不足** — 赢了判决拿不到钱 = 客户灾难
  - **风险代理乱用** — 部分案件法律禁止 / 客户结果坏律师还要赔
  - **不查类案** — 资深律师必查, 新手忽略 → 策略错配

- **关键决策点**:
  - 接不接: 胜诉概率 + 执行风险 + 客户费用承担 综合决定
  - 风险代理 vs 普通代理: 案值大 + 客户付不起前期费 + 不属法定禁止类型
  - 调解 vs 判决: 经济性 + 时间 + 关系修复 (商事 / 家事考量不同)

- **Last_updated**: 2026-01-01 (民事案件案由规定修订)
- **Decay risk**: medium — 司法解释频繁更新, 案由规定修订, 互联网法院管辖变动

### 2. 合同审核 (Contract Review)

- **One-liner**: 从客户提交合同到出审核意见的标准流程 (条款风险 + 法律风险 + 商业风险 三维度)
- **Trigger**: 客户 (企业 / 个人) 提交合同请审核 / 修改
- **Output**: 审核意见 (red flag / yellow flag / 修改建议) + 修改建议稿 (track changes) + 风险提示书

- **入门 SOP**:
  1. **了解背景** — 当事人 / 业务背景 / 客户角色 (买 / 卖 / 借 / 投 / 雇) + 客户重点关注 (价格 / 履约 / 责任 / 退出)
  2. **结构性审查** —
     - 主体: 双方 / 多方资质 + 适格 + 授权 (法人 / 个人 / 合伙)
     - 标的: 标的物明确 + 数量 + 质量 + 规格
     - 价款: 金额 + 支付 + 货币 + 税费
     - 履约: 时间 + 地点 + 方式 + 验收
     - 风险: 违约责任 / 不可抗力 / 索赔 / 担保
     - 退出: 解除条件 / 终止 / 通知
     - 争议: 适用法律 + 管辖 (法院 vs 仲裁) + 语言
  3. **法律风险审查** —
     - 强制性规定 (法律 / 行政法规 强制条款)
     - 民法典 + 单行法 + 司法解释
     - 行业特殊监管 (金融 / 数据 / 知识产权 / 房地产 / 涉外)
  4. **商业风险审查** —
     - 价格条款合理性 (跟客户业务匹配)
     - 责任 / 赔偿条款公平性 (向客户倾斜?)
     - 退出机制 (灵活 vs 锁定)
     - 模糊语言 (「合理」「适当」「双方协商」 等)
  5. **审核意见输出** —
     - **Red flag** (必须改): 强制性规定违反 / 重大不公平 / 致命漏洞
     - **Yellow flag** (建议改): 商业不利 + 法律风险 + 模糊条款
     - **Green** (可接受 + 建议): 中性条款建议
     - **修改稿** (track changes 明示)
     - **风险提示书** (即使改后仍存的风险)

- **资深路径**:
  - **skip 不看背景直接审条款** — 资深先了解业务 + 客户角色 + 重点关注
  - **skip 「有就行」式审查** — 资深查每条 + 推演执行场景
  - **optimize 用合同模板库 + 客户行业模板** — 不重复造轮子
  - **add 「红黄绿三色标注」清楚** — 客户能快速看到 must / should / nice to have
  - **add 推演履约 + 违约场景** — 静态条款不够, 看动态情景下条款怎么用
  - **add 仲裁条款 + 适用法律 + 管辖** 必查 — 涉外尤甚 (仲裁条款失效会致灾)
  - **add 个保法 + 数据合规** 涉数据合同必查 (2021-11-01 起 + CAC 跨境数据规则)
  - **optimize 谈判轮次估计** — 提交修改稿前预估对方可能反驳 / 反提
  - **add 元数据清除** — Word 修订记录 / 注释 PDF 化前清除

- **常见失败模式**:
  - **不了解业务背景直接审条款** → 形式审查, 错失关键风险
  - **没签字 / 没审核范围** → 范围争议 + 律师责任风险
  - **强制性规定遗漏** → 条款无效, 客户损失
  - **仲裁 / 管辖条款被忽略** → 争议时无法主张
  - **数据合规漏项** → 涉外 / 个保案件后果严重
  - **元数据没清除** → 内部讨论 / 修订记录暴露 给对方
  - **「合理」「适当」「双方协商」 模糊条款不指出** → 履约时分歧无依据
  - **客户告知不充分** → 风险责任律师承担

- **关键决策点**:
  - Red 必须改 vs Yellow 建议改: 强制性规定 / 致命漏洞 vs 商业不利 / 模糊
  - 仲裁 vs 法院: 标的额 + 涉外 + 保密 + 执行地 + 程序速度
  - 适用法律: 涉外多选目的地 / 中立第三方法律, 国内一般中国法

- **Last_updated**: 2026-01-01 (民事案件案由 + 数据 / 网络虚拟财产案由)
- **Decay risk**: low — 合同审核框架稳定, 数据 / 涉外 部分跟监管动

### 3. 刑事辩护流程 (从侦查到审判)

- **One-liner**: 从委托人接受刑事侦查 / 拘留 / 逮捕到法院判决的全流程辩护标准动作
- **Trigger**: 嫌疑人 / 被告人家属委托律师介入 (刑事案件)
- **Output**: 各阶段辩护策略 + 证据收集 + 沟通家属 + 庭审准备

- **入门 SOP**:
  1. **接受委托** — 家属 / 嫌疑人本人委托 (家属优先, 因嫌疑人在押) + 委托手续 + 律师执业证 + 律所介绍信
  2. **会见嫌疑人** —
     - 看守所会见 (法律明确权利, 24-48 小时内)
     - 了解事实 + 嫌疑人陈述 + 是否被刑讯逼供
     - 普及权利 (沉默权 / 不自证其罪)
     - 评估认罪认罚可能 (轻案常见, 重案谨慎)
  3. **侦查阶段策略** —
     - 申请变更强制措施 (取保候审 / 监视居住)
     - 提交辩护意见 (无罪 / 罪轻 / 程序违法)
     - 申请取证 (有利证据)
     - 监督程序合法性 (刑讯逼供 / 超期羁押 / 律师权利)
  4. **审查起诉阶段** —
     - 阅卷 (公诉机关移送的全部案卷)
     - 提交辩护意见 (建议不起诉 / 退查 / 罪轻)
     - 申请变更强制措施
  5. **审判阶段** —
     - 庭前会议 (重大复杂案件)
     - 出庭辩护 (开庭辩护 / 法庭辩论 / 最后陈述)
     - 证据质证 (合法性 / 关联性 / 真实性)
     - 法律适用辩论 (罪名 / 量刑 / 适用法律 / 司法解释)
  6. **二审 + 申诉** — 一审不利时, 上诉 + 申诉 + 必要时再审

- **资深路径**:
  - **skip 嫌疑人讲什么信什么** — 资深要客观核校证据
  - **skip 罪名先入为主** — 资深从构成要件实质解释 (张明楷派) 切入
  - **optimize 家属沟通 + 期望管理** — 家属期望「无罪」 vs 现实「罪轻 / 缓刑 / 量刑减少」
  - **add 程序辩护 + 实体辩护 双轨** — 程序违法可能直接影响证据效力 / 量刑
  - **add 认罪认罚审慎** — 轻案 / 证据确凿可考虑, 重案 + 有辩护空间不轻易认
  - **add 类案研究 + 法官倾向** — 同一罪名不同法院 / 法官量刑差距大
  - **optimize 庭前会议运用** — 重大复杂案件可解决程序 + 证据 + 管辖问题
  - **add 取保候审申请** — 不批准成本低, 批准帮助大, 试一下
  - **add 「不自证其罪 + 沉默权」 普及** — 嫌疑人陈述前必须明确权利

- **常见失败模式**:
  - **会见迟 / 走过场** → 嫌疑人不知权利 / 错过黄金期
  - **不阅卷直接辩护** → 不知公诉证据 = 抓瞎
  - **罪名先入为主** → 错过更轻罪名辩护 (如盗窃 vs 诈骗 / 贪污 vs 受贿)
  - **认罪认罚乱劝** → 不该认的认了 → 量刑虽减, 但失去辩护空间
  - **家属期望失控** → 「无罪」期望 vs 「缓刑」结果, 客户灾难
  - **程序违法漏揭** → 刑讯逼供 / 超期羁押 不主张 → 错过证据排除
  - **不研究类案** → 同罪名量刑差距大, 不知道争取空间
  - **二审上诉决定迟** → 上诉期 10 日 (一审判决送达起算), 不能错过

- **关键决策点**:
  - 取保候审申请: 案情 / 嫌疑人情况 (病 / 怀孕 / 老 / 未成年) / 罪名严重度
  - 认罪认罚: 证据是否确凿 + 罪名是否准确 + 量刑空间 + 嫌疑人意愿
  - 上诉 / 申诉: 一审判决错误 + 新证据 + 程序违法 + 法律适用错误

- **Last_updated**: 2024-03-01 (刑法修正案十二) + 2025 持续
- **Decay risk**: medium — 刑法修正案 + 司法解释频繁

### 4. 涉外商事争议 (国际仲裁 / 跨境执行)

- **One-liner**: 涉外商事合同纠纷 — 国际仲裁 / 跨境诉讼 / 跨境执行 标准流程
- **Trigger**: 涉外商事合同纠纷, 当事人一方在境内 + 一方在境外 (或合同签订 / 履行 / 标的 跨境)
- **Output**: 仲裁 / 诉讼策略 + 跨境执行预案

- **入门 SOP**:
  1. **争议条款审查** — 仲裁 vs 诉讼? 仲裁机构 (CIETAC / SHIAC / SCIA / 国际机构 ICC / SIAC / HKIAC) + 仲裁地 (重要 — 决定程序法 + 撤销 / 不予执行管辖) + 适用法律 + 语言
  2. **管辖权评估** —
     - 仲裁条款有效性 (新加坡公约 / 纽约公约 / 各国国内法)
     - 法院管辖 (一般原告就被告 / 涉外 + 协议管辖)
     - 反诉 / 第三方介入空间
  3. **法律适用** —
     - 实体法 (中国法 / 外国法 / Lex Mercatoria 国际商事惯例)
     - 程序法 (仲裁地法 / 法院地法)
     - 强制性规定 (中国法 / 第三国 / 仲裁地国)
  4. **证据收集** —
     - 跨境取证 (海牙公约 + 各国国内程序)
     - 涉外公证 + 领事认证 / Apostille (海牙公约 2023-11-07 起对中国大陆生效)
     - 跨境调查 (谨慎, 部分国家禁止外国律师调查)
  5. **仲裁 / 诉讼程序** —
     - 仲裁庭组建 (单仲裁员 / 三人 / 各方指定)
     - 程序法律 + 时间表
     - 文书提交 + 证据交换 + 听证
     - 裁决 / 判决 (仲裁一裁终局)
  6. **跨境执行** —
     - 仲裁裁决: New York Convention (170+ 国, 中国 1987 加入)
     - 法院判决: 双边司法协助条约 + 海牙公约
     - 中国境内承认 + 执行: 申请中国法院承认 + 强制执行
     - 境外承认 + 执行: 各国程序

- **资深路径**:
  - **skip 仲裁条款不审清就接案** — 仲裁条款失效会致灾
  - **skip 默认中国法** — 涉外可能选第三国法 / 国际商事惯例
  - **optimize 仲裁地选择** — 中立第三方 (新加坡 / 香港 / 瑞士) 比当事人国仲裁好
  - **add 跨境执行预案** — 接案前评估「赢了能不能拿到钱」, 被告有无境外财产
  - **add 紧急仲裁员 / 临时措施** — 大型案件可申请临时禁令
  - **add 多语种 + 多法律体系沟通** — 涉外律师团队需要语言 + 法律体系对接
  - **add 海牙公约新规** — 2023-11-07 起对中国大陆生效, Apostille 简化文书认证
  - **optimize 国际仲裁规则差异** — CIETAC / ICC / SIAC / HKIAC 程序差异大, 选机构看程序灵活度

- **常见失败模式**:
  - **仲裁条款失效** → 仲裁不成 + 法院管辖也不成 → 灾难
  - **跨境执行困难** → 赢了仲裁 / 判决拿不到钱
  - **法律适用错** → 用中国法解释外国法主张, 判决不利
  - **跨境取证违法** → 部分国家禁止外国律师调查, 操作不当违法
  - **公证 / 认证缺失** → 涉外文书在境外不被采信 (Apostille 简化前更严重)
  - **程序时间表失控** → 国际仲裁 / 跨境诉讼时间长 (1-3 年常见), 客户不耐
  - **多法系沟通失误** → 普通法系 vs 大陆法系 vs 中国法 概念不互通

- **关键决策点**:
  - 仲裁 vs 诉讼: 涉外多倾向仲裁 (机密 + 中立 + 跨境执行 + 程序灵活)
  - 仲裁机构选择: 案件性质 + 当事人偏好 + 仲裁员资源 + 程序成本
  - 仲裁地: 中立 + 程序法成熟 + 法院支持仲裁 (新加坡 / 香港 / 瑞士 / 伦敦 / 纽约)
  - 适用法律: 当事人协议优先 + 国际惯例 + 强制性规定

- **Last_updated**: 2023-11-07 (海牙公约对中国大陆生效)
- **Decay risk**: medium — 国际经贸法律 + 跨境执行规则持续变化

## Phase 2 提炼提示 (跨工作流共性)

1. **程序先于实体** (4 个工作流共性)
2. **法律检索 + 类案研究** 是基本功 (民商 / 刑事 / 涉外 共性)
3. **客户期望管理** 是律师质量隐藏前提 (民商 / 刑事 / 合同 / 涉外 共性)
4. **证据 + 证据规则** 是诉讼核心 (4 个共性)
5. **跨境 / 涉外 程序复杂度高** (合同 + 涉外 共性)
