## A. 民商事法学权威 (现役)

### A1. 王利明 (中国人民大学法学院)
- **职位**: 中国人民大学法学院教授 / 一级教授 / 博士生导师, 历任副校长 + 法学院院长
- **核心一句话**: 「民法典是私权保护的总章程 — 解释民法典必须从中国实际出发, 同时放眼世界」
- **代表作**:
  - 主持起草民法典人格权编 + 侵权责任编 (2001 年起立法专家组核心成员)
  - 《中国民法典释评》十卷本 (2024 年北京哲社一等奖)
  - 200+ 篇核心期刊论文 + 20+ 部专著
  - 最高人民法院 + 最高人民检察院专家咨询委员
  - 两次中央政治局集体学习主讲, 三次全国人大常委会专题讲座
- **思维独特点**: 从中国实际出发 + 比较法视角. 强调民法典实施 + 司法适用一致性. 「人格权独立成编」是其重要立法贡献
- **可信度**: ★★★★★ (民法典核心起草 + 理论实务双轨)
- **代表内容载体**: ruc.edu.cn/wangliming + 人大重阳网长文 + 中国民法典释评 + 最高法学术讲座

### A2. 江平 (中国政法大学, 已故 2023)
- **职位**: 中国政法大学终身教授 (2023 年去世). 「民法泰斗」
- **核心一句话**: 「私权要靠规则, 不靠权力」
- **代表作**:
  - 《罗马法基础》《法人制度通论》
  - 民法通则 / 公司法 / 合同法 / 物权法 起草核心
  - 大量长篇访谈 + 演讲, 思想史地位
- **思维独特点**: 罗马法功底 + 私权独立 + 法治理念. 「法治三老」(江平 / 郭道晖 / 李步云) 之一
- **可信度**: ★★★★★ (历史地位 + 思想代表)

### A3. 梁慧星 (中国社会科学院法学研究所)
- **职位**: 中国社科院学部委员 + 民法学家
- **核心一句话**: 「民法典必须有解释论 — 不是立法论了」
- **代表作**:
  - 《民法解释学》《民法总论》
  - 民法典立法专家组核心成员
  - 大量论著影响国内民法学界
- **思维独特点**: 解释论派, 强调司法适用 + 法律解释方法
- **可信度**: ★★★★★ (社科院学部委员 + 民法学派代表)

### A4. 崔建远 (清华大学法学院)
- **职位**: 清华大学法学院文科资深教授
- **核心一句话**: 「合同法解释要看商事习惯, 不只看条文」
- **代表作**:
  - 《合同法》多版本
  - 民法典合同编立法专家
  - 商事合同 + 物权 双重权威
- **可信度**: ★★★★★ (清华文科资深教授)

### A5. 王轶 (中国人民大学法学院)
- **职位**: 人大法学院院长 (2020s) / 教授

## B. 刑事法学权威 (现役)

### B1. 张明楷 (清华大学法学院)
- **职位**: 清华大学法学院文科资深教授 (2021 年清华第二批文科资深教授) + 中国刑法学研究会副会长
- **核心一句话**: 「构成要件实质解释 — 罪刑法定原则下结果无价值论 + 规范责任论」
- **代表作**:
  - 《刑法学》(七版, 2021 年第六版扩容近 60 万字, 1724 页, 法律出版社) — 国内刑法教材标准
  - 《犯罪论原理》《刑事责任论》《刑法的基础观念》《法益初论》
  - 大量核心期刊论文 + 国际学术会议
- **思维独特点**: 结果无价值论 + 规范责任论 + 构成要件实质解释 (vs 高铭暄等行为无价值论). 国内刑法学方法论代表
- **可信度**: ★★★★★ (清华文科资深 + 教材标准 + 学派代表)

### B2. 罗翔 (中国政法大学)
- **职位**: 中国政法大学刑事司法学院教授 + 刑法学研究所所长. 北大刑法学博士
- **核心一句话**: 「圆圈正义 — 任何具体的圆都不完美, 但圆是存在的. 法律的目标是不断接近正义」
- **代表作**:
  - 《刑法学讲义》(普通读者刑法普及, 豆瓣高分)
  - 《圆圈正义》(随笔, 法律 / 道德 / 权力的现实)
  - 法考刑法名师 — 司法考试 / 法律职业资格考试备考主力
  - 2020 年视频网络爆红 (B站百大 UP / 抖音粉丝千万级)
  - 大量普法访谈 + 媒体专栏
- **思维独特点**: 哲学 + 文学功底 + 刑法功底 + 案例叙事能力. 国民级普法 — 中文圈独特现象, 把刑法学讲到普通人能听懂
- **争议**: 部分学术派认为「普法 ≠ 学术」, 普法影响力大但学术深度跟张明楷 / 周光权等不同
- **可信度**: ★★★★★ (法考主力 + 普法影响力 + 政法大学学术身份)

### B3. 周光权 (清华大学法学院)
- **职位**: 清华大学法学院教授 + 全国人大宪法和法律委员会副主任委员
- **核心一句话**: 「刑法解释要回到立法本意 — 司法不能跟立法脱节」
- **代表作**:
  - 《刑法各论》《刑法总论》(教材)
  - 《刑法学的基本立场》
  - 刑法修正案立法专家
- **思维独特点**: 立法 + 学术双轨, 跟张明楷不同观点 (行为无价值论倾向)
- **可信度**: ★★★★★ (清华 + 全国人大法律委)

### B4. 高铭暄 (中国人民大学法学院, 已故 2024)
- **职位**: 中国人民大学法学院教授 + 一代刑法学宗师 (2024 年去世)
- **核心一句话**: 「刑法是社会治理工具, 严而不苛 + 宽而不滥」
- **代表作**:
  - 《刑法学原理》多卷
  - 主编多部教材
  - 行为无价值论代表
- **可信度**: ★★★★★ (历史地位)

## F. 流派分歧

| 流派 | 核心主张 | 代表 |
|------|---------|------|
| **民法典实施 + 解释论派** | 民法典已立, 重点是司法适用 + 解释统一 | 王利明 / 梁慧星 / 王轶 |
| **比较法 + 国际化派** | 学德 / 法 / 日 + 中国本土化 | 王利明 / 江平 |
| **解释论方法派** | 强调法律解释方法 + 价值判断 | 梁慧星 / 王轶 |
| **结果无价值论 (刑法)** | 关注行为造成的法益侵害结果 | 张明楷 (清华派) |
| **行为无价值论 (刑法)** | 关注行为本身的规范违反性 | 周光权 / 高铭暄 (人大派) |
| **大众普法派** | 把法律讲到普通人能听懂 | 罗翔 / 部分微信公众号 |
| **司法实务派** | 程序 + 证据 + 案例先于学理 | 江必新 / 沈德咏 / 各级法官 |
| **辩护实战派** | 刑辩 / 民商代理 实战经验 | 田文昌 / 李贵方 等头部律师 |

## G. 主流共识 + 争议

**主流共识**:
1. 民法典 (2021-01-01 起施行) 是私法基础, 单行法体系仍有效
2. 程序先于实体 — 民事 / 刑事 / 行政 程序法重要
3. 证据是诉讼核心 — 证据规则严格 (民事新证据规定 2020 起)
4. 刑事辩护权利 — 律师参与是被告人基本权利
5. 数据 / 网络虚拟财产 是新兴民事案由 (2026-01-01 民事案件案由规定增加)

**仍存争议**:
1. 结果无价值论 vs 行为无价值论 (刑法)
2. 民法典人格权独立成编 是否合理 (王利明 vs 部分老一代)
3. 普法 (罗翔) vs 学术 (张明楷) 的边界
4. 民事新证据规定下电子证据效力
5. 互联网法院案件管辖范围 (2025-11-01 新规)
6. 涉外商事 + 仲裁 vs 国内法院
