# 巨量千川官方方法论视角 · Sub-skill

> **来源声明**: 本 skill 基于巨量千川 / 巨量引擎 官方公开材料 (官方文档 / 巨量大学 / WILL keynote / 千川·乘方发布) 蒸馏. 不编造官方没说过的话.

## 角色背景

- **背景**: 巨量千川是字节跳动旗下抖音电商核心广告平台
- **关键事件**:
  - 2021 千川上线
  - 2025-03 整体并入抖音电商 (从巨量引擎商业化部门转出)
  - 2025-12 千川·乘方发布 — 营销工具进入「智能化新阶段」
- **战略转向**: 从「让商家拿曝光」到「让商家做经营」 — 流量服务于 GMV / ROI

---

## 心智模型 (4 个 PASS)

### 1. OCPM 智能出价是主流, 卡价时代过去

**一句话**: OCPM 让算法预估转化价值自动调整出价, 比手动卡价更高效. 老套路 (手动卡价 / 死磕定向) 跟不上算法迭代.

**它说的是**: 算法在前 50 转化里学习人群转化偏好, 后续自动给高价值用户溢价 / 低价值用户降价. 投手要做的是给算法好素材 + 等学习期, 不是手动微操.

**应用方式**:
- OCPM 是默认出价模式
- 学习期 (前 50 转化) 不动出价 / 定向 / 素材
- 学完后只动 1 个变量, 不要同时改

**局限**:
- 极少数极小预算 (¥ 100-500/天) 学习期长, 等不起
- 高客单类目转化数据少, 学习期不稳定

### 2. 5A 人群 + GTA 是中大品牌资产

**一句话**: GTA = GMV → 5A 人群积累 → 投放触点. 中大品牌品牌资产沉淀 + ROI 双优化的官方方法论.

**它说的是**: 5A 人群 (Z世代 / 精致妈妈 / 新锐白领 / 小镇青年 / 小镇中老年) 是定向底盘. 非大促节点累积 → 大促节点放大 — 实测 CPA 降 8% / ROI 升 30%.

**应用方式**:
- 月预算 ≥ ¥ 50k 才有意义跑 GTA
- 不同 5A 不同素材
- 千川·乘方 AI 自动跨人群分配预算

**局限**:
- 中小商家直接跑 ROAS 更实在
- 业态差异大 (本地生活 / 短剧 不适用)

### 3. 直播 3 阶段不同核心目标

**一句话**: 冷启动 (互动 + 人群标签) / 成长 (转化率 + UV 值) / 成熟 (GMV) — 阶段不同 KPI 不同, 不要混着追.

**它说的是**: 巨量千川官方明确给出 3 阶段 KPI. 多数商家失败因为冷启动期硬冲 GMV (但人群没建).

**应用方式**:
- 冷启动期 (前 7 天): 轻投 ¥ 500-2000/天, 看互动数据 + 人群标签
- 成长期 (≥ 50 转化, 7-30 天): 短视频引流 + 直播间承接双轮
- 成熟期 (≥ 30 天稳定): 全产品组合 + GTA 5A 人群

**局限**:
- 头部主播 (罗永浩 / 董宇辉 / 大杨哥) 不走 3 阶段
- 短期事件型直播 (1-3 天) 压缩 3 阶段

### 4. 千川·乘方 — AI 智能化是大方向 (2025-12 起)

**一句话**: 千川·乘方整合广告投放 + 智能优惠券 + 达人佣金结算 + AI 创意, 营销工具进入「智能化新阶段」.

**它说的是**: 2025-12 发布 千川·乘方 是千川并入抖音电商后的全面升级. AI 跨场景 (短视频 / 直播间 / 商品卡 / 搜索) 自动分配预算, 商家只需要给目标 (GMV / ROI), 算法做执行.

**应用方式**:
- 中大品牌优先用千川·乘方 全域推广 (跨场景一键投放)
- AI 创意工具 (智能脚本 / 智能剪辑) 节省 70% 创意人力
- 智能优惠券自动按人群分发
- 达人佣金结算自动化 (CPS 模式)

**局限**:
- 中小商家可能用不到全域推广 (预算门槛高)
- AI 创意全自动直发限流, 必须人工把关

## 决策规则 (6 条)

### 1. 如果新建千川计划
**则**OCPM 智能出价 + 学习期 (前 50 转化) 不动 — 让算法学完.

### 2. 如果月预算 ≥ ¥ 50k
**则**跑 GTA + 5A 人群 — 非大促累积人群, 大促放大投放.

### 3. 如果做直播间投流
**则**按 3 阶段不同 KPI: 冷启动看互动 + 人群 / 成长看转化 + UV / 成熟看 GMV.

### 4. 如果是中大品牌
**则**用千川·乘方 全域推广 (跨场景), 不要分散投流.

### 5. 如果用 AI 创意
**则**辅助 OK (脚本 / 剪辑 / 字幕), 但必须人工把关 — AI 全自动直发限流.

### 6. 如果跑量低
**则**先看创意是否过关 (3 秒钩子 / 转化点 / 真人感), 再调出价 — 创意是因, 出价是果.

## 应用示例

### 用千川官方视角思考: 「我中型商家月预算 ¥ 100k, 跑了 14 天 ROAS 1.5, 怎么优化?」

**千川官方框架**:
1. **学习期过了吗?** 14 天 + ROAS 1.5 = 应该已过学习期 (≥ 50 转化). 进入稳定期可以优化.
2. **创意是否过关?** ROAS 1.5 偏低, 类目均值多少? 如果类目均值 2.5 → 创意是瓶颈, 不是出价.
3. **有没有跑 5A 人群?** 月预算 ¥ 100k 应该跑 GTA + 5A — 非大促累积人群.
4. **直播 + 短视频双轮没?** 单纯短视频投流 ROAS 上限 2-3, 加直播间可以做到 3-5.
5. **千川·乘方 全域推广用了吗?** 中大品牌优先用全域推广, 跨场景算法自动分配比手动分散投放好.

**结论倾向**: ROAS 1.5 是创意 + 人群双瓶颈. 升级创意 (5-10 个角度) + 跑 5A + 上千川·乘方 全域, 目标 ROAS 3.0+.

## 诚实边界

1. 本 skill 给的是「巨量千川官方方法论」, **不是商家实战经验** — 实际效果跟商家情况强相关.
2. 千川官方立场偏「商家用全套千川产品」 — 实际方法论中小商家可以简化.
3. 适合中大商家 / 月预算 ≥ ¥ 50k. 中小商家 GTA / 千川·乘方 用不到.
4. 平台政策 6 个月一变, 千川 2025-03 并入抖音电商 是关键节点 — 之前老套路很多过时.
5. 千川·乘方 (2025-12) 是新功能, 实际效果还在迭代中.
