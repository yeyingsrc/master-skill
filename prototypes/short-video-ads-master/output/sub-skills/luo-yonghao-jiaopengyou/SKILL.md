## 心智模型 (4 个 PASS)

### 1. 头部主播 IP 是抗算法波动的护城河

**一句话**: 算法 6 个月一变, 跑量套路活不长. 真正抗算法的是头部主播 IP — 用户跟你的关系是稳定资产.

**它说的是**: 罗永浩 / 交个朋友能跨平台复刻 (抖音 → 视频号 → TikTok), 因为 IP 内核 (罗永浩本人 + 交个朋友品牌 + 品质感 + 性价比导向) 是可迁移的. 算法换了, 用户还认你这个 IP.

**应用方式**:
- 头部主播培养: 选 1-2 个核心人物长期投入, 不押单点
- IP 内核稳定 — 调性 / 选品 / 话术 风格一致
- 跨平台复刻 — 抖音爆款 → 视频号 → 跨境

**局限**:
- 头部主播 IP 不可复制 — 大多数商家学不来
- IP 培养周期 6-12 月, 短期 ROI 看不到
- IP 翻车风险 (主播说错话 / 商业道德) 是大风险

### 2. 直播 + 投流双轮驱动, 单轮死一半

**一句话**: 直播间靠投流跑流量, 但内容差投流也只是烧钱. 直播 + 投流是双轮, 缺一不可.

**它说的是**: 短视频投流是流量入口 (用户来), 直播间是转化场 (用户买). 投流给曝光, 内容承接转化. 两者必须同步迭代.

**应用方式**:
- 短视频引流到直播间 (头部账号短视频日更 5-10 条)
- 直播间做 3 件套: 选品 + 主播 + 投流
- 月度复盘: 短视频 ROAS + 直播间 ROAS 双数据看

**局限**:
- 短视频内容产能压力大 (头部主播日产 5-10 条)
- 直播间 + 短视频团队人力成本高 (≥ 20 人)

### 3. 性价比 + 品质感是新消费人群的双驱动

**一句话**: 抖音电商主流人群要的是「性价比 + 品质感」 — 不是单纯便宜, 也不是单纯品牌.

**它说的是**: 交个朋友选品逻辑是「品质过关 + 价格让人觉得划算」. 这不是简单白牌竞价, 也不是品牌溢价. 是头部主播 IP 给的「信任背书」让用户接受合理价格.

**应用方式**:
- 选品: 品质过关 (商品质检 + 退货率) + 性价比 (类目均价 70-90%)
- 话术: 不只讲价格, 也讲品质 / 故事 / 真实使用
- 直播间不打硬广, 是「真实测评 + 真实体验」

**局限**:
- 极致低价类目 (9.9 包邮) 不适用 — 竞价为主
- 极致品牌类目 (奢侈品) 不适用 — 品牌溢价为主

### 4. 跨平台复刻 IP 是头部主播标配

**一句话**: 抖音 / 视频号 / TikTok 同 IP 复刻, 跨平台流量协同, 单平台风险全消化.

**它说的是**: 罗永浩从抖音店播起家, 复刻到视频号 (微信生态) + TikTok (跨境). 不押单平台 — 算法变 / 平台政策变都不至于团灭.

**应用方式**:
- 抖音 (主战场) + 视频号 (微信生态加成) + TikTok (跨境) 同步
- IP 内核保留, 适配各平台格式
- 跨平台节奏: 抖音首发 → 视频号同步 → TikTok 国际化

**局限**:
- 跨平台运营复杂度极高, 需要 50+ 人团队
- 不同平台用户偏好不同 (TikTok 海外受众跟抖音差异大)

## 决策规则 (6 条)

### 1. 如果做店播
**则**头部主播 IP 培养是核心, 不押单点 — 至少 2-3 个主播轮换.

### 2. 如果做投流
**则**直播 + 投流双轮 — 投流给曝光, 内容承接转化, 缺一不可.

### 3. 如果选品
**则**品质过关 (退货率 < 类目均值) + 性价比 (类目均价 70-90%).

### 4. 如果跨平台
**则**抖音 (主战场) + 视频号 (微信加成) + TikTok (跨境) 同 IP 复刻.

### 5. 如果做内容
**则**真实测评 + 真实体验 + 真人感 — 不打硬广.

### 6. 如果遇到平台政策变
**则**IP 内核稳定 (调性 / 选品 / 话术), 跨平台分散风险.

## 应用示例

### 用罗永浩视角思考: 「我品牌想做店播, 月预算 ¥ 200k, 怎么打?」

**罗永浩可能的回答框架**:
1. **主播 IP 培养了吗?** 没 → 至少 ¥ 50k/月先培养 2-3 个主播 (6-12 月长期投入)
2. **直播 + 投流双轮搭了吗?** 短视频日产 5-10 条 + 直播间日开 6-8 小时, 双轮启动
3. **选品逻辑是?** 品质过关 + 性价比 (类目均价 70-90%) — 不要走极致低价 / 极致品牌
4. **跨平台规划了吗?** 抖音 (主战场 ¥ 100k) + 视频号 (¥ 30k) + TikTok 海外 (¥ 20k) 三件套
5. **剩下 ¥ 50k**: 内容生产 (创意中台 + 剪映 Pro) + 数据工具 (蝉妈妈 + 巨量百应)

**结论倾向**: ¥ 200k/月 起步够了 — 但要做好 6-12 月不出 ROI 的心理准备. 头部主播 IP 是长期资产, 不是短期跑量.
