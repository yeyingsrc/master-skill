# Track 02 — Tools (短视频投流)

> 调研日期: 2026-05-02。

## 元数据
- 必备 4 / 场景特化 6 / 新兴 3

---

## 必备层

### 1. 巨量引擎后台
- **One-liner**: 抖音 + 头条 + 西瓜 + 番茄全套信息流投放平台
- **License**: 开户费 5 万起 (一般找代理)
- **典型场景**: 所有抖音 / 字节系信息流投放
- **避坑**: 走代理通常更便宜 (有返点)

### 2. 千川 (巨量千川)
- **One-liner**: 抖音电商 + 直播专用投流后台
- **License**: 开户 1k-5k
- **典型场景**: 抖音小店 + 直播带货必装
- **避坑**: 千川 vs 巨量引擎不同体系, 别混用账户

### 3. 巨量云图
- **One-liner**: 巨量人群洞察 + 5A 人群分层工具
- **License**: 月费 (跟广告投放挂钩)
- **典型场景**: 大账户人群拓展 / lookalike

### 4. 飞书 / 钉钉 (内部协同)
- **One-liner**: 投手 + 创意 + 老板沟通必装
- **License**: 免费 (基础)

---

## 场景特化层

### 5. 蝉妈妈
- **One-liner**: 抖音电商数据 + 投流 benchmark
- **License**: ¥200-2000/月
- **典型场景**: 类目 benchmark / 竞品监控

### 6. 飞瓜数据
- **One-liner**: 抖音垂类账号 / 直播间数据
- **License**: ¥100-1500/月
- **典型场景**: 直播间数据复盘

### 7. 巨量算数
- **One-liner**: 巨量原生关键词 + 趋势分析
- **License**: 免费 (基础)
- **典型场景**: 找内容方向 + 关键词测试

### 8. ARK Pro / 微小宝
- **License**: ¥100-500/月
- **典型场景**: 多账号 / 矩阵投放 + 创意管理

### 9. 第三方代理后台
- **One-liner**: 代理服务商提供的策略后台
- **典型场景**: 月广告 ≥ 100 万 RMB 时找代理省力

### 10. 创意素材库 (内部 / 外采)
- **One-liner**: 团队历史素材 + 拆解参考
- **典型场景**: 矩阵账号素材池

---

## 新兴层

### 11. AI 创意生成 (即梦 / 海艺 / Midjourney 等)
- **One-liner**: AI 生成视频 + 数字人 + 配音
- **Status**: 早期 (2024-2025 起势)
- **典型场景**: 大批量素材测试期
- **避坑**: 平台对纯 AI 内容限流, 必须有真人元素

### 12. 智能投放托管 (巨量 / 千川官方 + 第三方)
- **One-liner**: AI 自动调出价 / 定向
- **Status**: 早期
- **典型场景**: 中小账户测试新计划
- **避坑**: 大账户慎用

### 13. AB 测试工具 (内部 / 外采)
- **典型场景**: 创意 / 文案 / 出价的对照测试

---

## 选型决策树

| 场景 | 工具栈 |
|------|-------|
| 中小卖家 / 单账户 (月广告 < 5 万 RMB) | 巨量引擎 / 千川 + 巨量算数 + 飞书 |
| 中型电商 / 多账户 (5-100 万 RMB) | + 蝉妈妈 + ARK + 巨量云图 |
| 大账户 (≥ 100 万 RMB) | + 走代理 + 巨量云图 + 蝉妈妈专业版 + AB 测试工具 |

---

## 避坑

1. 巨量 / 千川账户不要混用, 体系完全不同
2. 智能托管慎用, 大账户尤其要看人工把关
3. AI 生成的素材必须有真人元素
4. 蝉妈妈 / 飞瓜数据有延迟, 实时性场景不能只信榜单
5. 跨账户同素材投放可能被识别为「同质化竞争」
