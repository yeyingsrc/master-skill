## A. 计费 / 出价 (15 项)

- **CPM (Cost Per Mille)**: 千次曝光价
- **CPC (Cost Per Click)**: 单次点击价
- **OCPM (Optimized CPM)**: 优化千次曝光成本, 智能出价 (主流)
- **OCPC (Optimized CPC)**: 优化单次点击成本
- **CPA (Cost Per Action)**: 单次行为成本
- **CPS (Cost Per Sale)**: 按销售抽佣 (CPS 联盟分销)
- **CPL (Cost Per Lead)**: 单次留资成本
- **eCPM (effective CPM)**: 有效千次展示, 排名指标
- **Target ROAS**: 目标 ROAS 直接锁定 ROI
- **Value Optimization**: 价值优化, 算法自由探索
- **智能放量**: 千川官方智能投放
- **全域推广**: 千川·乘方 跨场景投放
- **手动出价**: 老套路, 已不推荐
- **自动出价**: 算法自动调整, OCPM 标配
- **竞价机制**: GSP (Generalized Second Price) 第二价格

## B. 数据指标 (15 项)

- **CTR (Click Through Rate)**: 曝光转点击, 短视频 ≥ 3%
- **CVR (Conversion Rate)**: 点击转转化
- **ROAS (Return on Ad Spend)**: 广告支出回报率
- **ROI**: 总投资回报率
- **GPM (GMV Per Mille)**: 千次曝光 GMV (直播间)
- **UV 值**: 单访客价值 (直播间)
- **进入率**: 直播间曝光转进入
- **停留时长**: 用户在直播间停留秒数
- **互动率**: 直播间互动 / 进入
- **完播率**: 视频完播比例 (短视频)
- **点赞率**: 点赞 / 曝光
- **评论率**: 评论 / 曝光
- **分享率**: 分享 / 曝光
- **关注率**: 涨粉 / 曝光
- **复购率**: 重复购买 / 总用户

## C. 投放策略 (15 项)

- **学习期 (Honeymoon)**: 前 50 转化, 算法学习
- **稳定期**: ≥ 50 转化, 可优化
- **衰减期**: 素材 7-14 天衰减
- **冷启动**: 0 → 50 转化
- **跑量**: 计划放大投放, ¥ 5k+/天
- **卡价**: 手动设定低出价 (老套路)
- **5A 人群**: 5 类人群标签
- **GTA**: GMV → 5A → 投放触点
- **直播 3 阶段**: 冷启动 / 成长 / 成熟
- **千川主图**: 商品卡推广 (流量沉淀)
- **直播间转化推广**: 直播间投流主流
- **短视频转化推广**: 短视频投流主流
- **预热期**: 大促前 30-60 天 5A 人群积累
- **冲刺期**: 大促前 7-14 天放大投放
- **大促当天**: 即时调整 ±10%, 不大动

## D. 创意 / 素材 (10 项)

- **黄金 3 秒**: 前 3 秒抛核心卖点
- **钩子**: 5-15 秒痛点 + 反差
- **转化点**: 15-30 秒产品 + 行动
- **真人感**: 真人出境 / 真实场景
- **B-roll**: 主画面之外的辅助素材
- **同卖点 5-10 角度**: 让系统挑出最优
- **素材衰减**: 7-14 天 ROAS 下滑
- **创意中台**: 头部品牌创意生产中心
- **AI 创意辅助**: 70% AI + 30% 人工
- **AI 直发限流**: 全自动 AI 视频被算法识别

## E. 5A 人群细分 (5 项)

- **小镇青年**: 四线及以下 / < 35 岁
- **小镇中老年**: 四线及以下 / ≥ 35 岁
- **Z 世代**: 三线及以上 / < 24 岁
- **精致妈妈**: 三线及以上 / 25-35 岁 / 备孕或已生育女性
- **新锐白领**: 三线及以上 / 25-35 岁 / 家庭月收入 6-25k+

## F. 平台 / 工具 (15 项)

- **巨量千川**: 抖音电商核心广告平台
- **巨量引擎**: 字节系全产品
- **巨量百应**: 商家 + 达人匹配
- **巨量数据云**: 商家 BI
- **巨量算数**: 行业关键词 / 趋势
- **千川·乘方**: 2025-12 升级版
- **抖店 / 巨量小店**: 商家入驻
- **蝉妈妈**: 第三方数据头部
- **飞瓜数据**: 第三方数据
- **灰豚数据**: 跨平台数据
- **剪映 / CapCut**: 短视频剪辑
- **巨量大学**: 官方培训
- **抖音 OS**: 商家系统全套
- **DOU+**: 个人创作者投流工具 (起投 ¥ 100)
- **粉条**: 抖音个人创作者推广 (类似 DOU+)

## G. 平台规则 / 合规 (15 项)

- **抖音电商商家规范**: 平台官方规范
- **直播带货合规**: 不得虚假宣传 / 价格欺诈
- **明示标识**: 商单需标识 / 直播间不得违规种草
- **禁忌品类**: 医美 / 保健 / 减肥 / 金融 / 证券 / 政治 / 宗教
- **直播话术合规**: 避免极限词 / 限定词
- **优惠券合规**: 不得虚假优惠
- **诱导关注 / 互动**: 平台禁止
- **AI 内容标识**: AI 生成必须标识
- **未成年保护**: 涉及未成年内容严审
- **数据隐私**: 不得滥用用户数据
- **退款 / 维权**: 平台维权机制
- **商家保证金**: 入驻需保证金
- **违规处罚**: 限流 / 降权 / 封禁 / 罚款
- **税务合规**: 商家收入需个税申报
- **跨境合规**: 跨境商品 / 海外购合规

## 外行破绽 (说错就露馅)

1. 把「投流」叫「投广告」
2. 不知道学习期 (前 50 转化) 别折腾
3. 把 ROAS 跟 ROI 混着叫
4. 不知道千川·乘方 (2025-12 升级)
5. 把 5A 人群当成简单人群标签
6. 一个组只放 1 条素材
7. 把卡价当成主流方法 (老套路, 已不推荐)
8. 把 AI 直发当成正常做法
9. 不知道千川 2025-03 已并入抖音电商
10. 把直播 3 阶段混着追 (互动 / 转化 / GMV 不同期)
