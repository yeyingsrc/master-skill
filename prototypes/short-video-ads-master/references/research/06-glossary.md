## 高频术语

### 通用
- **ROAS**: 广告 ROI
- **ROI**: 真实投资回报 (扣成本)
- **GMV**: 总成交额
- **GPM**: 直播间千次曝光成交
- **CPM**: 千次曝光成本
- **CPC**: 单次点击成本
- **CPA**: 单次行动 / 转化成本
- **千展**: 千次曝光
- **跑量 / 起量 / 衰减**: 计划生命周期三阶段
- **冷启动 / 学习期 / 探索期**: 新计划前 3-7 天

### 创意 / 素材
- **完播率**: 3s / 6s / 完整三档
- **互动率**: 综合互动 / 曝光
- **CTR / CVR**: 点击率 / 转化率
- **创意分**: 平台给单条创意打分
- **素材衰减**: 一条素材跑不动了 (一般 3-7 天)
- **跑量 / 起量 / 起飞**: 计划进入大流量阶段
- **ALL-IN / 提预算**: 加预算的两种方式

### 人群定向
- **5A 人群**: A1 (了解) / A2 (吸引) / A3 (问询) / A4 (行动) / A5 (拥护)
- **lookalike (LK 1-9)**: 相似人群层级
- **DMP**: 数据管理平台
- **品牌词 / 兴趣词 / 行为词**: 三层定向

### 出价 / 投放
- **OCPM**: Optimized CPM, 最常用
- **CPC**: 按点击付费
- **CPM**: 按千次展示付费
- **目标 ROI 出价 / 双出价**: 千川独有
- **bid 策略**: 提价 / 降价 / 保 / 起量
- **托管 / 智能投放**: 系统自动出价 / 选人群
- **手动 / 半托管**: 人工调控

### 直播专属
- **千川投放**: 直播专属体系
- **福利单 / 福袋单**: 直播间冷启动用低价单
- **直播间 GMV / ROAS / GPM**: 直播专属财务指标
- **闭环单**: 抖音内完成下单
- **跳转单**: 跳第三方平台下单 (淘宝 / 京东)

### 财务
- **ROAS / ROI / payback period / LTV / CAC**
- **真实 ROI**: 扣去退货 / 平台费 / 物流后

## 平台政策 / 标准 (6)

### 1. 创意审核规则
- **范围**: 所有素材发布前过审
- **罚则**: 违规创意被拒 / 影响计划权重
- **2024 起**: AI 生成内容标识强制

### 2. 跑量计划生命周期 (官方)
- **冷启动期**: 3-7 天, 跑出 ≥ 1k 曝光
- **学习期**: 7-14 天, 形成稳定 ROAS
- **跑量期**: ≥ 14 天, 开始衰减
- **衰减期**: 素材 / 创意需更新

### 3. 直播间黑科技 / 异常流量
- **范围**: 刷数据 / 互动 / 假买等
- **罚则**: 限流 / 关播间 / 账号封禁
- **2025 监管收紧**

### 4. 跨账户同素材识别
- **范围**: 不同账户投放同一素材, 系统识别为「同质化竞争」
- **罚则**: 计划权重下降, 跑量难

### 5. AI 生成创意标识
- **范围**: 2024 Q3 起强制标注
- **罚则**: 不标 → 限流

### 6. 千川资质要求
- **范围**: 不同类目要求不同资质 (医疗 / 金融严)
- **罚则**: 无资质素材审核不过 / 类目降权

## 行业近期更新

| 时间 | 事件 | 影响 |
|------|------|------|
| 2024-Q3 | AI 内容标识强制 | 所有 AI 生成视频必须标 |
| 2024-Q4 | 智能托管 (巨量) 推出 | 中小账户测试新计划路径 |
| 2025-Q1 | 千川「全域投放」上线 | 抖音 + 商城 + 直播间一体投放 |
| 2025-Q3 | 巨量优化目标分层调整 | 浅层 / 深层目标定义变化 |
| 2025-Q4 | 闭环 vs 跳转 ROAS 计算口径调整 | 财务模型重写 |
| 2026-Q1 | 5A 人群替代旧版三层定向 | 投放策略调整 |

## 衰减速度

| 模块 | 衰减 | 原因 |
|------|------|------|
| 后台规则 / 计费 | 极高 (3 月) | 巨量 / 千川 月更 |
| 算法 | 高 (3-6 月) | 算法持续演进 |
| 法规 | 中 (1-2 年) | 国家级法规年度更 |
| 缩写 / 黑话 | 低 | 行业基础语言 |

## 外行破绽

- 把 ROAS 当 ROI (差很多, ROAS 不扣成本)
- 把 OCPM 当 CPC 思路操作 (出价完全不同体系)
- 不知道千川 vs 巨量引擎是两套体系
- 把跑量 = 跑利润 (跑量是表面, ROI 是本质)
- 直播投流跟视频投流逻辑混淆
