## 核心概念 (22 个)

| 概念 | 一句话 |
|------|------|
| **ROAS** | Return on Ad Spend, 广告 ROI |
| **ROI** | 真实投资回报, 跟 ROAS 不同 |
| **CPM / CPC / CPA** | 千次曝光 / 单次点击 / 单次转化成本 |
| **GPM** | 直播间千次曝光成交额 |
| **跑量** | 计划进入大流量池稳定出量 |
| **起量** | 计划开始能跑出量 |
| **衰减** | 计划数据下滑 |
| **冷启动期** | 新计划前 3-7 天系统学习 |
| **学习期** | 系统找人群的阶段 |
| **完播率** | 视频看完比例, 3s / 6s / 完整三档 |
| **互动率** | 点赞 + 评论 + 转发 / 曝光 |
| **CTR** | 点击率 |
| **CVR** | 转化率 |
| **千展** | 千次曝光 |
| **5A 人群** | 巨量云图人群分层 |
| **拉新 / 复购** | 新客 / 老客 |
| **payback** | 单客户回本周期 |
| **OCPM / OCPC** | 优化的千展 / 点击 |
| **手动 / 智能投放** | 人工调控 vs 系统自动 |
| **创意分** | 平台给单条创意打分 |
| **抢量** | 多投手同类素材冲击转化率 |
| **5A 人群** (Aware/Appeal/Ask/Act/Advocate) | 巨量五层人群分类 |

## 流派代表性长篇

- 「素材决定 ROAS, 出价是结果」(老 A)
- 「冷启动期单日数据不重要, 看趋势」(大宇)
- 「智能投放 vs 手动调」(巨量官方 vs 实战代理)

## 怎么用

- **入门 (1-2 周)**: 巨量学堂 + 老 A 入门 30 课
- **进阶 (1-2 月)**: 大宇连载 + 飞瓜月报 + 群响周报
- **资深**: 巨量引擎大会 + 抖音电商罗盘 + 自家 BI 沉淀
