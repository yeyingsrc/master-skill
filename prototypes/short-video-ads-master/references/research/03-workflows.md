### 1. 新计划冷启动 (前 50 转化)

- **One-liner**: 千川新计划从 0 到稳定跑量的标准动作 — 14 天内累积 ≥ 50 转化, 算法学完模型.
- **Trigger**: 新建千川计划 / 新品上线 / 新创意上线
- **Output**: 计划稳定跑量 (≥ 50 转化, ROAS 达标), 算法学习期完成
- **入门 SOP**:
  1. **创意素材准备** (Day 0): 一个广告组准备 3-5 条不同创意素材 (不同角度 / 不同钩子)
  2. **建组建计划** (Day 0): OCPM 智能出价, 出价按目标 ROAS 倒推 (类目 benchmark 找蝉妈妈)
  3. **学习期铁律** (Day 0-7, ≤ 50 转化): 不动出价 / 不动定向 / 不动素材 / 不动预算
  4. **跑量评估** (Day 7-14): 观察曝光 / CTR / CVR / ROAS — 哪条素材跑得动就放大
  5. **跑得动的处理** (Day 14+): 跑出量的素材复制 + 微调出价 (±10%) 持续放大
  6. **跑不动的处理** (Day 14+): 跑不动的计划砍掉, 不要硬撑 — 没有学习期保底数据
- **资深路径**:
  - **skip 通用素材, 直接做差异化创意** — 资深投手第一组就放 5 条不同钩子
  - **optimize OCPM 出价**: 资深用类目 ROAS benchmark 倒推, 不直接抄竞品
  - **add 5A 人群定向** (中大品牌): GTA 模型从历史 5A 人群圈定, 不冷启动跑全人群
  - **add 智能放量 / 全域推广** (千川·乘方): 让算法自由探索, 适合预算 ≥ ¥ 5k/天
- **常见失败模式**:
  - 学习期硬调出价 → 算法重置学习, 50 转化重计
  - 一个组只放 1 条素材 → 没法 A/B 测试
  - 一上来就调高预算 → 模型来不及学习, 跑乱
  - 计划跑不动还硬撑 → 浪费预算, 应该立刻砍
- **Last_updated**: 2026-04-15
- **Decay risk**: high (千川算法 3-6 个月一变)

### 2. 直播间投流 (3 阶段)

- **One-liner**: 直播间投流分冷启动 / 成长 / 成熟 3 阶段, 每阶段核心目标 + 投放策略不同.
- **Trigger**: 开播 / 新主播 / 大促节点
- **Output**: 直播间稳定 GMV + 5A 人群资产沉淀
- **入门 SOP**:
  1. **冷启动期 (新直播间, 前 7 天)**:
     - 核心目标: 直播间互动数据 + 人群标签
     - 投放: 直播间 PV 推广 + 商品卡推广 (轻投, ¥ 500-2000/天)
     - 关键指标: 进入率 / 停留时长 / 互动率
  2. **成长期 (≥ 50 转化, 7-30 天)**:
     - 核心目标: 转化率 + UV 值 + 5A 人群标签丰富
     - 投放: 直播间转化推广 + 短视频投流引流
     - 关键指标: ROAS + CVR + UV 值 + GPM (千次曝光销售额)
  3. **成熟期 (≥ 30 天稳定, GMV ≥ 类目均值)**:
     - 核心目标: GMV 稳定 + 大促节点放大
     - 投放: 全产品组合 (千川 + 巨量引擎 + GTA 5A 人群)
     - 关键指标: GMV / ROAS / 5A 人群 LTV
- **资深路径**:
  - **skip 冷启动手动调** — 资深直接上智能放量, 让算法 7 天内学完
  - **optimize 5A 人群分层投放**: 不同人群 (Z世代 / 精致妈妈 / 新锐白领) 不同素材
  - **add 短视频引流 + 直播间承接双轮**: 短视频产爆款 → 直播间承接 GMV
  - **add 大促预热**: 大促前 30 天累积 5A 人群, 大促节点 CPA 降 8% / ROI 升 30%
- **常见失败模式**:
  - 冷启动期硬冲 GMV → 人群标签没建起, 后续投放跑不动
  - 成长期不上 5A 人群 → 大盘转化效率低
  - 成熟期才想起 5A 人群 → 已经错过黄金期
  - 大促节点临时加预算 → 没有人群积累, 出价跟人抢
- **Last_updated**: 2026-04-20
- **Decay risk**: medium (直播 3 阶段框架稳定, 具体策略随平台动)

### 3. 创意素材生产 (黄金 3 秒法则)

- **One-liner**: 短视频投流素材的标准生产流程 — 黄金 3 秒抛卖点 + 钩子 + 转化点 三件套.
- **Trigger**: 新品 / 新创意 / 素材衰减需要上新
- **Output**: 5-10 条不同角度素材 + 投放跑测找出最优
- **入门 SOP**:
  1. **拆解类目爆款** (Day 1): 蝉小红 / 蝉妈妈 找类目 top 10 跑量素材, 拆其结构 (3 秒 / 钩子 / 转化点)
  2. **写脚本 / 做素材** (Day 2-4): 同卖点 5-10 个不同角度 (痛点开头 / 反差 / 解决方案 / 真人感)
  3. **3 秒钩子设计** (Day 2-4): 前 3 秒抛核心卖点 / 反差 / 痛点, 否则 80% 划走
  4. **15-30 秒结构** (Day 2-4): 5-15 秒钩子 (痛点 + 反差) → 15-30 秒产品 + 转化
  5. **真人感要素**: 真人出境 / 真实场景 / 真实使用 (反对 AI 全自动直发)
  6. **投放跑测** (Day 5-7): 每组 3-5 条投放, 让系统挑出最优
- **资深路径**:
  - **skip 模板套用**: 资深第一稿就做差异化创意, 不抄类目模板
  - **optimize AI 辅助**: 70% AI (脚本 / B-roll / 字幕 / 配音) + 30% 真人创意
  - **add 创意中台**: 头部品牌建创意中台, 月产 50-100 条素材保持轮换
  - **add 素材衰减监控**: 素材 7-14 天衰减, 衰减期前批量上新
- **常见失败模式**:
  - 一个素材投到死 → 衰减后 ROAS 暴跌
  - 全 AI 生成不加人工 → 平台识别限流
  - 没有钩子 / 转化点结构 → 跑不出量
  - 同质化素材 (5 条都用同一钩子) → 系统挑不出最优
- **Last_updated**: 2026-04-10
- **Decay risk**: medium (3 秒法则结构稳定, 具体钩子 / 角度随平台用户偏好变)

### 4. 5A 人群 + GTA 方法论 (中大品牌)

- **One-liner**: GTA = GMV → 5A 人群积累 → 投放触点, 中大品牌品牌资产沉淀 + ROI 双优化的标准方法论.
- **Trigger**: 中大品牌 / 月预算 ≥ ¥ 50k / 大促节点 / 跨阶段长期投放
- **Output**: 5A 人群资产 (5 类人群标签丰富) + 大促节点 ROI 显著优化
- **入门 SOP**:
  1. **5A 人群分层定义** (Week 1):
     - 小镇青年 (四线及以下 / < 35 岁)
     - 小镇中老年 (四线及以下 / ≥ 35 岁)
     - Z 世代 (三线及以上 / < 24 岁)
     - 精致妈妈 (三线及以上 / 25-35 岁 / 备孕或已生育女性)
     - 新锐白领 (三线及以上 / 25-35 岁 / 家庭月收入 6-25k+)
  2. **GTA 目标拆解** (Week 1):
     - GMV 目标 → 拆到 5A 人群积累目标 → 落地到投放触点的预算分配
  3. **非关键节点累积人群** (持续 Week 2-12):
     - 非大促节点持续投放, 累积 5A 人群标签
     - 巨量数据云监控 5A 人群增长
  4. **大促节点放大投放** (Week 12-13, 大促):
     - 用历史 5A 人群圈定投放, 跳过冷启动
     - 实测: CPA 降 8% / ROI 升 30%
  5. **复盘 + 迭代** (Week 13+):
     - 每月看 5A 人群增长 + LTV
     - 调整人群分层占比
- **资深路径**:
  - **skip 全人群冷启动**: 资深直接用历史 5A 圈定, 不重新跑全人群学习期
  - **optimize 不同 5A 不同素材**: Z 世代 (反差幽默) / 精致妈妈 (信任 / 实用) / 新锐白领 (品质 / 效率)
  - **add 千川·乘方 智能化** (2025-12 升级): AI 自动跨人群分配预算
  - **add 跨平台 5A 协同**: 抖音 5A + 小红书 AIPS + 微信私域 协同
- **常见失败模式**:
  - 临时大促才想起累积人群 → 大促当天没人群, 出价跟人抢
  - 5A 人群只用一条素材 → 不同人群偏好不同, 转化效率低
  - 不监控 5A 人群增长 → 人群资产沉淀不下来
  - 全部预算压大促 → 大促节点 CPA 飞涨
- **Last_updated**: 2026-04-25
- **Decay risk**: low (GTA 方法论 2024-2026 稳定, 千川·乘方升级也基于这个框架)

### 5. 大促备战 (双 11 / 618 / 38 节)

- **One-liner**: 大促节点 60 天备战 + 大促当天精细投放 + 大促后复盘 全流程.
- **Trigger**: 大促节点 (双 11 / 618 / 38 / 双 12 等)
- **Output**: 大促 GMV 翻倍 + 5A 人群 LTV 提升
- **入门 SOP**:
  1. **预热期 (大促前 30-60 天)**:
     - 5A 人群积累 (GTA 模型)
     - 短视频内容铺量 (头部 KOL + 中腰部 + KOC)
     - 商品卡推广 (千川商品卡, 流量沉淀)
  2. **冲刺期 (大促前 7-14 天)**:
     - 直播间预告 + 短视频钩子素材
     - 优惠券提前发放 (千川·乘方智能优惠券)
     - 5A 人群放大投放
  3. **大促当天**:
     - 全产品组合 (千川 + 巨量引擎 + GTA + 智能放量)
     - 实时监控 ROAS / CPA / 库存
     - 即时调整出价 (±10% 微调)
  4. **大促后复盘** (大促后 7 天):
     - 巨量数据云全链路复盘
     - 5A 人群 LTV 评估
     - 头部素材 / 头部计划 整理资产
- **资深路径**:
  - **skip 平铺直叙备战**: 资深从 90 天前就开始 5A 人群积累, 不只 30 天
  - **optimize 大促预热素材**: 同卖点 20+ 个角度轮换, 不靠单条爆款
  - **add 跨平台联动**: 抖音 + 小红书 + 微信私域 + 天猫旗舰店 同步预热
  - **add 千川·乘方 全域推广**: AI 跨场景 (短视频 / 直播间 / 商品卡 / 搜索) 自动分配预算
- **常见失败模式**:
  - 大促前 7 天才开始预热 → 5A 人群没积累, 出价跟人抢
  - 大促当天调整过频 (出价 / 定向) → 算法乱
  - 大促后不复盘 → 头部素材 / 计划资产沉淀不下来
  - 单平台押宝 → 平台政策一动就团灭
- **Last_updated**: 2026-04-30
- **Decay risk**: medium (大促节点框架稳定, 具体玩法每年迭代)

## 工作流之间的关系

```
新计划冷启动 (workflow 1)
   │
   │  累积 ≥ 50 转化, 算法学完
   ↓
直播间投流 3 阶段 (workflow 2)
   │
   │  3 阶段递进
   ↓
创意素材生产 (workflow 3)  ←→  5A 人群 + GTA (workflow 4)
   │                              │
   │  素材轮换                    │  人群资产沉淀
   ↓                              ↓
                          大促备战 (workflow 5)
                              │
                              ↓
                          头部品牌闭环
```

**典型路径**:
- 抖音小店新商家: workflow 1 → workflow 3 (素材) → workflow 2 (直播)
- 中型商家: workflow 1-3 全打通 + 月度大促
- 头部品牌: workflow 1-5 全闭环 + 跨平台联动

- workflow 1 → 决策规则: 「学习期 (前 50 转化) 不动出价 / 定向 / 素材 / 预算」

- workflow 4 → 决策规则: 「非大促节点累积 5A 人群, 大促节点 CPA 降 8% / ROI 升 30%」
- workflow 5 → 决策规则: 「大促备战 30-60 天起, 当天微调 (±10%), 大促后必复盘」
