## A. 平台官方代表 (战略级最高一手)

### 1. 巨量千川官方 (并入抖音电商后, 2025-)

- **核心一句话**: 千川从「让商家拿曝光」转向「让商家做经营」 — 流量服务于 GMV / ROI, 不再单纯卖广告位.
- **代表事件**:
  - 2025-03 巨量千川团队整体并入抖音电商 (从巨量引擎商业化部门转出)
  - 2025-12 千川·乘方发布 — 整合广告投放 + 智能优惠券 + 达人佣金结算 + AI 创意, 营销工具全面智能化
  - 2024 起 GTA (GMV → 5A 人群积累 → 投放触点) 方法论推全
- **代表观点**:
  1. 直播带货 3 阶段: 冷启动期 (互动数据 + 人群标签) / 成长期 (转化率 + UV 值) / 成熟期 (GMV)
  2. 5A 人群分层 (小镇青年 / 小镇中老年 / Z世代 / 精致妈妈 / 新锐白领) 是定向投放底盘
  3. 非关键节点累积 5A 人群 → 大促节点放大投放, CPA 降 8% / ROI 升 30%
- **争议**: 算法迭代偏 ROI / GMV, 中小商家短期跑量难度变高
- **来源**: [Primary] 巨量引擎 / 巨量千川官网 + 36氪独家「千川并入抖音电商」报道

### 2. 巨量引擎官方 (品牌广告侧)

- **核心一句话**: 「广告平台 → 增长平台」转型 — 不只卖流量, 也卖人群资产 + 品牌资产沉淀.
- **代表内容**:
  - 巨量引擎年度增长大会 (年度)
  - 巨量数据云 (商家自助 BI)
  - 巨量百应 (达人匹配) + 巨量千川 (电商投放)
- **代表观点**:
  1. 数据驱动 + 人群资产沉淀是品牌方主战场
  2. 广告 / 内容 / 直播 / 达人 四件套联动是头部品牌标配
  3. AI 创意工具 (智能脚本 / 智能剪辑) 是 2025-2026 主推方向
- **来源**: [Primary] 巨量引擎官方 + TopMarketing 报道

## B. 头部投手 / 操盘手 (实战一手)

### 3. 头部投流操盘手代表 (匿名常见, 业内多家)

- **核心一句话**: 创意是因, 出价是果. 没有好素材, 怎么投都跑不动.
- **代表观点**:
  1. 一个广告组放 3-5 条不同创意, 让系统自己挑出最优解
  2. 学习期 (前 50 转化) 别折腾 — 不动出价 / 不动定向 / 不动素材
  3. 稳定期 (≥ 50 转化) 小步跑 — 每次只动 1 个变量
  4. 素材衰减 7-14 天, 稳定期之后必须持续上新
- **代表内容**: 各家投手公众号 + 知识星球 (买量小飞机 / 千灵 / 智点课堂等)
- **来源**: [Primary] 多家头部投手公众号长期跟踪

### 4. 罗永浩 / 交个朋友 (店播头部)

- **核心一句话**: 短视频投流跟内容是一回事 — 没有好内容, 投流只是烧钱.
- **代表案例**: 交个朋友直播间 (抖音店播头部) + 罗永浩 IP 复刻多平台
- **代表观点**:
  1. 头部主播 IP 是抗算法波动护城河
  2. 直播间靠投流 + 内容双轮驱动, 单轮死一半
- **争议**: 头部 IP 不可复制 — 大多数商家学不来
- **来源**: [Primary] 交个朋友官方 + 罗永浩公开发声

### 5. 东方甄选 / 董宇辉 (内容驱动直播)

- **核心一句话**: 直播间也可以做内容, 不一定要靠纯投流跑量.
- **代表案例**: 东方甄选「文化带货」模式 — 内容驱动 + 投流辅助
- **代表观点**:
  1. 内容差异化 = 抗投流通胀
  2. 投流跑得动的直播间内容多数同质化
- **来源**: [Primary] 东方甄选官方 + 36氪 / 晚点深度

### 6. 头部 MCN 投流团队 (谦寻 / 美 ONE / 遥望 等)

- **核心一句话**: 直播 + 投流 + 选品 + 内容是 4 件套, 缺一不可.
- **代表案例**: 谦寻 (薇娅前签约方) / 美 ONE (李佳琦) / 遥望 (大杨哥) 投流团队
- **代表观点**:
  1. 头部主播投流转化路径深 — 必须是品牌方一起打
  2. 中腰部主播跑量靠投流 — 这部分是大盘
- **来源**: [Primary] 各家 MCN 公开发声 + 36氪报道

## C. 内容流派 / 创意操盘手 (跟投流互补)

### 7. 头部创意中台 (亮哥 / 何夕等行业大咖)

- **核心一句话**: 黄金 3 秒法则 + 钩子 + 转化点是抖音创意的标准结构.
- **代表观点**:
  1. 前 3 秒抛核心卖点, 否则 80% 用户划走
  2. 5-15 秒做钩子 (痛点 + 反差), 15-30 秒做产品 + 转化
  3. 同一卖点 5-10 个素材角度, 让系统挑
- **来源**: [Primary] 各家创意中台公众号 + 行业大会 keynote

### 8. AI 创意流派 (2025-2026 新兴)

- **核心一句话**: 70% AI + 30% 方法论是 2025 抖音内容增长密码.
- **代表观点**:
  1. AI 生成脚本 / B-roll / 字幕 / 配音, 节省 70% 人力
  2. 但 30% 核心创意 (钩子 / 转化点 / 真人感) 必须人工把关
- **争议**: AI 直发被算法识别限流, 必须有真人 / 真实场景介入
- **来源**: [Primary] 行业 KOL 知乎专栏 + AI 工具厂商案例

## 流派分裂 (关键认知分歧)

| 流派 | 主张 | 适合 | 风险 |
|------|------|------|------|
| **创意驱动派** | 创意是因, 出价是果. 90% 在素材 | 中小品牌 / 内容能力强 | 没创意时全军覆没 |
| **数据 / 算法派** | OCPM + 5A 人群 + GTA 模型 | 中大品牌 / 投手专业 | 算法变天就被动 |
| **直播 + 投流双轮派** | 直播 + 投流 + 选品 + 内容四件套 | 头部品牌 / MCN | 操盘复杂度极高 |
| **AI 创意派** | 70% AI + 30% 人工 | 中小品牌 / 想跑量 | AI 直发限流 |
| **内容长期派** | 内容资产 / 品牌沉淀 / 用户关系 | 头部品牌 / 抗算法 | 短期 ROI 看不到 |

**核心分歧 1**: **创意 vs 算法** — 决定团队配置 (创意中台 vs 投手中台)
**核心分歧 2**: **跑量 vs 长期资产** — 决定预算路径
**核心分歧 3**: **店播 vs 达人投流** — 决定品牌投放方式

## 可信度判断 (诚实边界)

1. **「巨量引擎 / 千川官方」keynote 是战略宣示**, 实际效果跟商家情况强相关 — GTA / 5A 人群在头部品牌效果明显, 中小品牌不一定吃得到
2. **「头部 MCN 投流团队」公开发声有立场偏见** — 投流团队靠这个吃饭, 不会主动唱衰投流
3. **「头部主播」案例不可复制** — 罗永浩 / 董宇辉 的成功 80% 靠 IP, 不是投流方法论
4. **业态差异极大** — 抖音电商 / 本地生活 / 短剧 / 教育 / 服务业 投流逻辑各不同
5. **平台政策 6 个月一变** — 千川并入抖音电商 (2025-03) 是大变化, 之前的方法论很多过时
6. **AI 内容直发被限流** — AI 创意辅助可以, AI 直发不行
