## 1. 行业定义

**English**:Newsletter / Creator Economy — independent practitioner role building audience via direct-publishing channels and monetizing through paid subscriptions, sponsorships, courses, and downstream products, bypassing traditional publishers.

**Industry type**:`platform_specific_practitioner`

## 2. 核心子领域候选(Phase 1 调研维度)

1. **Niche 选择与定位** — 垂直深 vs 横向洞察 / 个人专家 vs 横向分析 / 海外 vs 中文
2. **写作产出** — 节奏 / 字数 / 标题 / 内容矩阵 / AI 辅助边界
3. **分发与增长** — SEO / 社交 cross-post / Substack Notes / 公众号 / 视频号 / 小红书 / 推荐网络
4. **订阅制** — 免费 vs 付费比例 / 定价 / 续订率 / 流失
5. **广告与赞助** — 议价 / 平台 vs 直签 / 软广 vs 硬广 / 转化数据
6. **课程与下游产品** — 从 newsletter 衍生(推 1 vs 推多)
7. **社群与会员** — Patreon / Discord / 私域回流配合 newsletter
8. **平台选型** — Substack / Beehiiv / Ghost / ConvertKit / 公众号 / 知识星球 / 小报童 — 成本 + 功能 + 数据 + 退出门槛
9. **海外 vs 国内路径** — 订阅制为主 vs 广告 + 知识付费 + 内容矩阵
10. **合规与平台风险** — 微信封号 / Substack 政策 / 自媒体许可证 / 税务
11. **数据与指标** — open rate / CTR / 净增 / churn / LTV / referral
12. **个人品牌 + IP** — 写作以外的延伸(演讲 / 顾问 / 投资 / 出书)

## 6. 警示与边界(写入 SKILL.md 诚实边界)

1. **头部效应极端**:Substack top 0.1% 占订阅收入估 50%+,中位订阅 newsletter 月入低 $100。**SKILL.md 必须诚实写「绝大多数 newsletter 永远做不到全职收入」**。
2. **平台依赖风险高**:Substack 政策变 / 公众号封号 / Ghost 要懂技术。SKILL.md 写「鸡蛋分篮 — newsletter + 自有域名 + 跨平台 cross-post」。
3. **信噪比差**:「7 天搭建 newsletter」「Substack 月入 5 万」营销课大量,真知识被掩埋。研究时狠筛课程导流。
4. **海外 vs 国内路径完全不同**:海外订阅为主 + Substack default + 读者付费习惯成熟;国内广告 + 知识付费 + 公众号生态 + 读者付费习惯差。**SKILL.md 必须按 locale 分章讲**。
5. **「写作就能赚钱」是误导**:真正核心 = (a) niche 选对 (b) 持续产出 (c) 分发 / 增长 (d) 销售环节。写作只占 30%。
6. **4 大学派分歧**,同问题不同答案:
   - Subscription-first(Ben Thompson / Casey Newton — 慢但稳)
   - Ad-supported(Morning Brew / Workweek / 国内绝大多数公众号 — 快但累)
   - Course-from-newsletter(Justin Welsh — newsletter 是 funnel,主收入是课)
   - 个人 IP 多元化(罗振宇 / 阑夕 — newsletter 是入口,生意在演讲 / 顾问 / 投资)
7. **国内监管收紧**:微信公众号自媒体许可证 / 算法备案 / 营销号清理 / 头部税务调查 — 月度校准。
8. **保鲜期 12-24 个月**:平台规则 + 工具生态 + 受众触媒习惯变化,2023 「Substack playbook」2025 部分过时。
9. **收入数据不透明**:头部公开收入多被认为夸大;中小创作者羞于公开真实数据,「幸存者偏差」严重。
