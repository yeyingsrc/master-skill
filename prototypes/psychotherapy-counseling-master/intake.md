## 2. 核心子领域候选(Phase 1 调研维度)

1. **接案与评估** — 初始访谈 / 风险评估 / 心理测评工具 / DSM-5-TR 与 ICD-11 / 诊断 vs 个案概念化
2. **个案概念化** — 跨流派的核心临床能力,把症状、历史、关系、情境组织成可工作的理解
3. **CBT 及第三浪潮** — 认知行为疗法 / DBT / ACT / 正念取向,循证证据最强的一支
4. **精神动力学 / 精神分析取向** — 移情反移情 / 防御 / 早年关系 / 长程工作
5. **人本 - 存在 - 体验取向** — 来访者中心 / 存在主义 / 格式塔 / 聚焦
6. **家庭与伴侣治疗** — 家庭系统 / 结构式 / Bowen / 伴侣治疗(注意与 love-coach 的临床 vs 经营边界)
7. **创伤治疗** — 创伤聚焦 / EMDR / 躯体取向 / 复杂性创伤 / 阶段化治疗
8. **治疗关系与共同要素** — therapeutic alliance / common factors,关系质量对疗效的解释力
9. **伦理 / 法律 / 边界** — 保密及其例外 / 知情同意 / 双重关系 / 胜任力范围 / 记录管理
10. **危机干预** — 自杀 / 自伤 / 虐待 / 急性精神病性发作 的识别、评估、处理与转介(生命安全硬边界)
11. **督导与个人成长** — 临床督导 / 个人体验 / 持续教育 / 职业耗竭与替代性创伤的自我照顾
12. **执业形态与经营** — 机构 vs 私人执业 / 收费排期 / 平台与转介来源 / 中国 vs 欧美 执业路径与监管

## 6. 警示与边界(写入 SKILL.md 诚实边界)

1. **这是临床领域,SKILL.md 绝不能让 agent 或用户假装是治疗师**。明确写『本 skill 不能用于诊断或治疗任何人,包括用户自己』。
2. **危机是生命安全问题,不是技巧**。自杀 / 自伤 / 虐待 / 急性精神病性发作 的内容必须导向『立即转介专业资源 / 危机热线』,不给『自助处理』暗示。
3. **中国与欧美执业体系完全不同**。2017 年人社部取消『心理咨询师』国家职业资格认证,之后认证市场混乱;欧美是严格持照(LMFT / LCSW / LPC / 临床心理学博士,各州各国不同)。**必须按 locale 分章**。
4. **流派之争没有干净答案**。CBT / DBT / ACT 有强 RCT 证据,精神动力 / 人本证据形态不同;共同要素(治疗关系)解释力可能不亚于具体技术。**呈现分歧,不站队**。
5. **培训市场营销噪音极大**。『21 天成为心理咨询师』是骗局。真实成长是数年受训 + 长期督导 + 个人体验。
6. **大众心理内容 ≠ 临床知识**。『原生家庭』『PUA』『NPD』等概念常被滥用;部分高影响力作者学界争议大。**区分科普流行说法与临床共识**。
7. **循证与实践之间有 gap**。RCT 证据来自标准化情境,真实个案复杂、共病多、文化因素重。
8. **文化适配问题**。西方治疗理论不能直接照搬到中国来访者。
9. **伦理与法律是硬约束不是建议**。保密、知情同意、双重关系、胜任力范围 — 违反造成真实伤害且涉及法律责任。
10. **从业者职业健康风险真实存在**。替代性创伤 / 共情耗竭 / 收入不稳定。写真实职业图景,不写鸡汤。
