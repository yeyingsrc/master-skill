## 候选 playbook 规则 (target 8-10)

| Pattern | 来源 |
| P1 接案后不止于 DSM 标签,必做个案概念化(什么在维持/资源在哪/从哪切入阻力最小) | T04(C26/F4), T03-W2/W3, T06-S022 |
| P2 凡涉及执业资质/伦理守则/转介/监管的判断,先确定 locale | T03-W7vsW8, T02 决策树根节点, T06-S015/S016 |
| P3 遇到危机/伦理两难/卡住的个案/强反移情,不独自决定 — 带进督导或同行咨询 | T03-W4/W6/W9, T01-S003, T06-S004 |
| P4 遇到危机信号(自杀/自伤/虐待/急性精神病性),直接、具体地问 + 用结构化工具 + 立即导向相应强度资源 | T03-W4, T02-S005, T06 外行破绽 |
| P5 遇到长期反复人际性创伤,先评估稳定性与资源,缺稳定化时优先建立安全,稳定后再加工 | T04(C16/C17), T03-W2, van der Kolk/Herman |
| P6 来访者突然疏远/迟到/质疑/关系紧张,不回避也不归咎来访者 — 在「此时此地」命名并探索;修复决裂是治疗时刻 | T04(C19/C21), T03-W1, Safran&Muran |
| P7 不要只凭主观感觉判断治疗有没有用 — 用 ORS/SRS 等会谈级监测,数据提示联盟下降/无进展时调整或讨论转介 | T04(C2/C25), T02-S015, T03-W2 |
| P8 「该用哪种疗法」先查循证清单(Div12/NICE/Cochrane),但循证清单不替代个案概念化,且「不在清单上 ≠ 无效」 | T02-S004/S013/S014, T04(E1/E2/E3), T05-S002 |
| P9 临床仪器(量表/诊断系统/风险评估)的选型逻辑是「信效度+临床定位+版权可得性+使用资质」,不是「功能/价格」 | T02-S001/S006/S005/S024, T02 Phase2 |
| P10 被问「哪个流派最好」/纠结认同哪派 — 这是错误问题;按「障碍×证据等级×来访者偏好与文化」匹配,不按流派身份认同 | T04(C3/C28/F1), T01 Phase2, intake warning #4 |

## 候选工具流派分裂 (智识谱系)

| Split | A 派代表 | B 派代表 | 来源 |
| 五大临床流派 | ①CBT与第三浪潮(Beck/Linehan/Hayes) ②精神动力/精神分析(Freud→曾奇峰/施琪嘉) ③人本-存在-体验(Rogers/Yalom) ④家庭系统(Minuchin/Bowen) | ⑤整合/共同要素派(Wampold/Norcross — 横切所有流派) | T01 Phase2, T04 Phase2, intake |
| 特异性技术 vs 共同要素 | CBT 阵营:OCD/PTSD/恐惧症有显著特异性技术效应 | Wampold 阵营:流派差异被高估,关系解释力不亚于技术 | T04(C3/C10/C28/F1) |
| 测量为本 / 循证主义 vs 关系优先 / 过程取向 | ORS/SRS-FIT、PHQ-9、Div12;Scott Miller、Beck/Linehan/Hayes 一脉 | 治疗核心变量(关系/过程/意义)难以也不应被量表化;动力学/人本 | T02 Phase2 |
| 诊断分类之争 DSM vs ICD | DSM(美国精神医学会):细颗粒症状计数 | ICD-11(WHO,中国采用):部分诊断原型+临床判断;收录 C-PTSD | T02-S017, T04(C16/C26/E4) |
| 「证据」的定义之争 | 手册化 RCT 是唯一可接受证据标准 | 精神动力/人本据此被系统性低估 — 评价框架偏倚 ≠ 疗效结论 | T04(E2 疗法偏倚) |
| 平台撮合 / 规模化 vs 自主私人执业 | BetterHelp/Talkspace:治疗规模化为 marketplace | 自建 EHR+合规视频+自有转介:完整掌控关系/数据/合规 | T02-S010/S011/S016 |
| 海外 vs 国内(分叉,不是分歧) | 欧美严格持照(LMFT/LCSW/LPC/博士;BACP/UKCP) | 中国 2017 取消国家职业资格 → 注册系统+连续培训+市场噪音 | T02-B分支, T03-W7vsW8, T06 时间轴 |
| 三浪潮谱系(代际演进) | 行为疗法(一)→ Beck/Ellis 认知疗法(二) | ACT/DBT/MBCT(三,改变与体验的关系) | T06 Phase2 |
| 精神分析内部演化 | 经典驱力 → 自我心理学 → 客体关系(Klein/Winnicott/Kernberg) | 自体心理学(Kohut)→ 关系学派 | T06 Phase2 |

## 候选 Agentic Protocol 维度 (target 7-9,标准-复杂行业)

| 维度 | 推导自哪个 mental model | 互斥性检查 |
| D1 locale 与执业体系判定 | M7 | 决定整套资质/伦理/转介/监管假设;≠ D2(D2 判流派,D1 判地域体系) |
| D2 流派语境与「障碍×证据×来访者」匹配 | M1 + M4 | 判用户在哪派 + 该障碍循证图景;≠ D7(D7 是「这概念是不是大众流行说法」) |
| D3 个案概念化就绪度审计 | M3 | 是否有成文可修订概念化(非只有 DSM 标签);≠ D2(D2 选疗法,D3 审「理解」质量) |
| D4 危机与风险硬边界核查 | M5 | 危机信号识别 + 转介去向 + 法定义务;≠ 其他所有(唯一聚焦生命安全) |
| D5 伦理 / 法律硬约束扫描 | M5 + M6 | 知情同意/保密例外/双重关系/胜任力范围/记录;≠ D4(D4 是急性危机,D5 是结构性伦理) |
| D6 督导与治疗师职业健康评估 | M8 | 督导习惯 + 个人体验 + 替代性创伤风险;≠ D5(D5 审来访者侧伦理,D6 审治疗师侧维护) |
| D7 大众流行说法 vs 临床共识甄别 | M2 + M4 | 这个概念/作者是科普流行说法还是临床共识;≠ D2(D2 是流派选择,D7 是知识真伪分层) |
| D8 工具栈与时效 / 监管变动追踪 | M6 + M7 + 时效信号 | 量表/EHR/平台选型 + 近期标准/监管变动;≠ D1(D1 判 locale,D8 追该 locale 近期变了什么) |
| D9 文化适配审查 | M7 | 西方理论默认假设是否需为该来访者调整;⚠️ 与 D1 可能 overlap → 评估:D1 判「哪套执业体系」,D9 判「理论假设要不要调」 — 若 overlap 则并入 D2 或 D8 |

**互斥性自检结论**:D9 与 D1 边界模糊(都涉文化/地域),且 D9 结论常 = D2 的一个子项 → **D9 并入 D2(匹配时把文化适配作为一个核对项)**。最终保留 D1-D8 共 8 个维度,落在 7-9 区间。每个能独立产出不同结论。
