## 总览（按 decay risk 分组）

> 9 个 workflow。心理治疗工作流**整体偏稳态**——核心骨架（会谈结构、个案弧线、概念化、危机响应、督导）多为数十年成熟的临床训练共识，与 LLM/agent 那类「12 个月级全面过时」的行业完全不同。变化集中在**载体（远程化）+ 记录方式（AI scribe）+ 监管不确定性（中国执业准入、美国跨州协约）**这三条边，不在流程核心。

## Workflows

### W1. 单次会谈的结构（开场 → 工作 → 收尾，一次 50 分钟会谈怎么走）

- **One-liner**：把一次 ~50 分钟的「治疗小时」（therapeutic hour）从「来访者坐下」走到「带着可带走的东西离开 + 治疗师完成记录」，让单次会谈既有结构又留得住治疗关系。
- **Trigger**：一次已排期的常规会谈开始（个案已完成接案、已在工作期内）。
- **Output**：一次有焦点、有情绪工作、有收束的会谈；一份当次进展记录（progress note）；下一次的隐性或显性方向。
- **范围提示**：这是治疗师的会谈结构，不是「和人聊天的步骤」——结构本身（settING / frame）有疗效，乱来会破坏治疗关系。

- **入门 SOP（minimum viable steps）**：
  1. **开场 / 签到（check-in，~5 min）**：问候、确认本次时长与设置、扫一眼来访者状态；如用结果监测，开场施测 ORS（结果评定量表）。**输入**=上次记录 + 来访者当下呈现；**输出**=本次大致状态基线。
     - *跳过会发生什么*：直接扎进内容，会漏掉「来访者这周出过危机 / 状态崩了」的信号，可能在错误的议题上耗掉整次。
  2. **设置本次焦点（agenda / 焦点协商，~5 min）**：和来访者一起确认「今天工作什么」——可以是来访者带来的事，也可以是上次的延续。CBT 取向会显性列 agenda；动力学 / 人本取向更松，但治疗师心里仍有焦点。
     - *跳过会发生什么*：会谈散成「这周流水账」，50 分钟用完没有任何一处被工作到深处。
  3. **工作期（working phase，~30–35 min）**：用所属流派的技术做核心工作——CBT 的思维记录 / 行为实验、动力学的此时此地探索 / 移情工作、人本的情感反映、创伤取向的稳定化或加工。全程**监测治疗联盟**与来访者的情绪窗口。
     - *跳过会发生什么*：没有工作期就不是治疗，是陪聊。
  4. **收尾 / 着陆（closing，~5–8 min）**：把情绪强度降下来（尤其触及创伤 / 强情绪时不能让来访者「带着炸开的状态」离开）、小结今天、确认带走的东西（可能含家庭作业 / 练习）、必要时给反馈量表 SRS（会谈评定量表）。
     - *跳过会发生什么*：来访者带着未着陆的强情绪进入现实生活，是真实的安全与脱落风险；也漏掉了「这次治疗对你怎么样」的反馈机会。
  5. **会谈后记录（progress note，会谈后 ~5–10 min）**：写当次进展记录——主诉、干预、反应、风险状态、下次计划。这是**伦理与法律要求的最小动作，不是可选项**。
     - *跳过会发生什么*：记录缺失是直接的伦理 / 法律风险，且下次会谈会「断片」、督导无据可依、个案交接不可能。
  - evidence: [T02-S003, T02-S015, T04-S025, T06-S004]

- **资深路径（差异点）**：
  - **skip**：资深治疗师常**跳过显性的 agenda 列表步骤（step 2 的形式）**——不是不要焦点，而是焦点在脑中即时形成，不必和来访者逐条过；经验让他们在 check-in 的前两分钟就听出今天的焦点在哪。（对应 step 2）evidence: [T04-S025, T05-S014]
  - **optimize**：资深治疗师把 **check-in（step 1）与焦点设置（step 2）合并**——同一段对话里既收集状态又浮现焦点；并且把**收尾着陆（step 4）做成贯穿性的而非末段补救**，全程留意情绪强度、提前 10 分钟就开始往「可着陆」的方向带。evidence: [T01-S022, T04-S006]
  - **add**：资深治疗师会**额外显性做「过程评论 / process comment」**——在工作期主动命名「此刻房间里在发生什么」（来访者突然沉默 / 转移话题 / 对治疗师的反应），把它当作核心素材而非干扰；并在记录里**额外记一行自己的反移情反应**作为下次督导 / 个人体验的料。新手往往只记内容、忽略过程。evidence: [T01-S011, T01-S022, T04-S006]

- **近期变化（近 12 个月）**：
  - **2020 起、由疫情推动、2024–2026 已常态化**：相当比例的「单次会谈」从面对面变为**远程视频会谈**——开场多了「确认来访者所在物理环境是否私密 / 安全 / 本次紧急联系人」这一步，收尾时「着陆」更难（隔屏无法用空间感帮来访者落地）。触发事件类型：行业事件（疫情）+ 标准更新（远程治疗伦理指引）。evidence: [T02-S012, T05-S006]
  - **2024–2026、由 AI ambient scribe 进入 EHR 推动**：会谈后记录（step 5）出现「AI 自动生成草稿 + 治疗师审核」的新变体——但这是**有伦理争议的新增动作**：需来访者明确知情同意，且记录准确性的最终责任不能外包给 AI。采用率无硬数据（治疗师执业私密、不公开晒工具栈）。触发事件类型：新工具。evidence: [T02-S003, T02-S007]
  - 会谈结构的**核心骨架（开场—工作—收尾—记录）本身是稳态**，5–10 年无重大变化，变的是「载体」（远程）和「记录方式」（AI 草稿）。

- **典型耗时**：
  - 入门 SOP：单次 ~50 min 会谈 + ~10 min 记录 = ~60 min/个案。
  - 资深路径：单次 ~50 min 会谈 + ~5 min 记录 = ~55 min/个案（记录更快、更聚焦）。

- **关键工具（与 Track 02 关联）**：
  - 结构化进展记录模板（必备）、ORS / SRS（场景特化——做反馈知情治疗 FIT 的治疗师每次用）、合规视频工具（远程会谈场景特化）。

- **关键人物（与 Track 01 关联）**：
  - Judith Beck 在《认知疗法：基础与超越》中详细写过 CBT 结构化会谈的标准流程（agenda → 工作 → 作业 → 反馈）；Irvin Yalom 在《The Gift of Therapy》中大量谈「此时此地」与过程评论；Carl Rogers 的核心条件定义了「工作期」里关系的底色。

- **常见失败模式**：
  - **「流水账会谈」**：每次都从「这周发生了什么」开始且不收口——触发原因是跳过焦点设置（step 2）。避免：哪怕不显性列 agenda，治疗师心里要在前 5 分钟锁定一个焦点。
  - **「带着炸开的情绪送走来访者」**：工作期触及创伤 / 强情绪后没留够收尾时间——触发原因是工作期超时、收尾被压缩到 2 分钟。避免：把收尾当硬约束，提前 10 分钟开始降强度，宁可工作期短一点。
  - **「会谈完不记录、攒到周末补」**：记录滞后导致内容失真、风险状态漏记——触发原因是把记录当行政负担。避免：把记录当会谈的一部分，会谈后立刻写。
  - **「只记内容不记过程 / 反移情」**：新手记录只有「来访者说了什么」，没有「关系里发生了什么、我有什么反应」——后果是督导无料、自我成长停滞。避免：每次记录额外留一行过程 / 反移情观察。
  - **CLI 化暗示**：这些失败会被抽进会谈结构自检 checklist——「本次是否锁定了一个焦点？」「收尾是否留了 ≥5 分钟且来访者状态已着陆？」「记录是否在 24 小时内完成且含风险状态 + 一行过程观察？」每项 yes/no。

### W2. 一个个案从接案评估到结案的完整弧线

- **One-liner**：把一个来访者从「第一次联系 / 初始访谈」一路带到「计划性结案」——经过风险评估、个案概念化、治疗计划、工作期、结案，是这一行最大尺度的工作流。
- **Trigger**：一个新来访者发起联系 / 被转介过来 / 被平台派单。
- **Output**：一个完整治疗历程——从知情同意签署到结案小结；中间产出包括风险评估记录、个案概念化、治疗计划、定期进展记录、（理想情况下）结果监测数据曲线。
- **范围提示**：这是治疗师视角的个案管理弧线，不是来访者「自我疗愈的阶段」。

- **入门 SOP（minimum viable steps）**：
  1. **初始访谈 / intake（第 1–2 次会谈）**：建立初步关系、收集主诉与相关史（症状、发展史、关系、家庭、医疗、用药、既往治疗）、签**知情同意**（含保密及其法定例外、收费、设置、远程的特殊条款）。**输出**=一份 intake 记录 + 已签知情同意。
     - *跳过会发生什么*：没签知情同意是伦理 / 法律红线；没收主诉与史就直接「开始治疗」，治疗方向无据。
  2. **风险评估（与 intake 同步，且贯穿全程）**：评估自杀 / 自伤 / 对他人危险 / 受虐 / 急性精神病性等风险——用结构化工具（如 C-SSRS）问到位。**风险评估不是一次性动作，是每次会谈都要扫的**。详见 W4。
     - *跳过会发生什么*：漏评风险是这一行最严重的失败模式，直接关联生命安全与法律责任。
  3. **个案概念化（第 2–4 次会谈逐步成形）**：把症状 / 史 / 关系 / 情境组织成一个「为什么这个人现在这样、什么在维持、从哪入手」的可工作理解。详见 W3。**输出**=一份初步个案概念化（动态、可修订的工作假设，不是诊断标签）。
     - *跳过会发生什么*：跳过概念化直接套技术，等于「拿着锤子找钉子」，治疗无方向、阻力点不明。
  4. **治疗计划（第 2–4 次，与概念化同步）**：基于概念化定治疗目标、大致方法、频率、预估时长，和来访者协商达成一致（目标 + 任务的协商本身就是治疗联盟的一部分）。必要时查循证清单（APA Div12 / NICE）看「这个障碍证据指向哪些选项」。
     - *跳过会发生什么*：没有协商一致的计划，来访者和治疗师可能朝不同方向使劲，脱落率高。
  5. **工作期（多次会谈，数周到数年不等）**：按 W1 的单次结构反复推进；定期回看概念化与计划是否仍成立；处理治疗联盟的**破裂与修复**（rupture & repair）；遇到瓶颈重新概念化。
     - *跳过会发生什么*：没有工作期就没有治疗；不回看计划会导致「治疗惯性」——一直做但早已偏离目标。
  6. **结案（calculated ending，理想是计划性的）**：识别「目标基本达成 / 来访者准备好 / 或需要转介」的信号，留出**结案阶段**（不是突然停）——回顾历程、巩固成果、处理分离情感、讨论复发预警与未来求助路径。
     - *跳过会发生什么*：突然终止（来访者脱落或治疗师草率结案）丢掉了巩固与分离工作；尤其依恋议题的来访者，草率结案可能本身就是一次创伤性分离。
  - evidence: [T02-S003, T02-S004, T03-S002, T04-S016, T06-S004]

- **资深路径（差异点）**：
  - **optimize**：资深治疗师把 **intake / 风险 / 概念化 / 计划（step 1–4）压缩并行**——不是用 3–4 次会谈线性走完，而是在第 1–2 次就形成「足够开工的」概念化与计划，把精细化留到工作期边做边修；他们的 intake 收集更聚焦（知道哪些史与当下问题真正相关，不做百科式问询）。evidence: [T01-S001, T04-S002]
  - **add**：资深治疗师**额外做系统性的结果监测（FIT / ORS-SRS）并据数据调整**——不靠主观感觉判断「治疗有没有用」，数据提示联盟下降 / 无进展时主动重新概念化或讨论转介；还会**额外把「结案」当成贯穿全程的视角**——从一开始就朝「来访者不再需要我」工作，而非把结案当末尾事件。evidence: [T02-S015, T04-S020]
  - **skip**：对清晰、单一、证据明确的问题（如单纯特定恐惧症），资深治疗师会**跳过冗长的发展史采集**——经验告诉他们这类个案不需要深挖早年关系就能用循证技术高效解决；新手往往对所有个案都做同样深度的 intake。evidence: [T02-S004, T04-S016]

- **近期变化（近 12 个月）**：
  - **2022 起、由 DSM-5-TR（2022-03）与 ICD-11（2022-01 生效）推动**：intake 与概念化环节的诊断语言、计费编码、与精神科协作的沟通都在更新——尤其复杂性 PTSD（C-PTSD）进了 ICD-11 未进 DSM-5-TR，用哪套系统会影响怎么命名同一个来访者。触发事件类型：标准更新。evidence: [T02-S001, T02-S017]
  - **2020 起、远程治疗常态化**：intake 多了「确认来访者所在司法辖区（治疗师执照是否覆盖）+ 本地危机资源 + 紧急联系人」环节；2024–2026 美国跨州执业协约（PSYPACT / Counseling Compact）推进，正在改变私人执业的地理边界。触发事件类型：行业事件 + 法规变化。evidence: [T05-S006, T06-S016]
  - **核心弧线（接案→评估→概念化→计划→工作→结案）本身是稳态**，几十年的临床训练共识，变的是诊断语言版本与执业地理边界。

- **典型耗时**：
  - 入门 SOP：intake + 概念化 + 计划 ~3–5 次会谈（3–5 周）；工作期数周到数年（短程 CBT 8–20 次 / 长程动力学可数年）；结案 2–4 次。
  - 资深路径：intake + 概念化 + 计划压到 1–2 次会谈起步（后续边做边修）；工作期靠结果监测动态判断长度而非套平均曲线。

- **关键工具（与 Track 02 关联）**：
  - 知情同意 / 结构化记录模板（必备）、C-SSRS（风险，必备配套）、DSM-5-TR / ICD-11（诊断语言，必备）、APA Div12 / NICE / Cochrane（治疗计划查证据）、ORS / SRS（结果监测，场景特化但资深默认）、EHR 执业管理平台（海外必备 / 国内基本缺位）。

- **关键人物（与 Track 01 关联）**：
  - Judith Beck 的 CBT 教材给出了「评估→概念化→计划→干预」的标准模板；Judith Herman 的创伤三阶段（稳定化→处理→重新联结）是「创伤个案弧线」的专门版本；Bruce Wampold 的共同要素研究提醒「弧线里关系质量的解释力可能不亚于具体技术」。

- **常见失败模式**：
  - **「跳过概念化，直接套手册技术」**：把 DSM 标签当治疗计划——触发原因是把诊断误当成对来访者的完整理解。避免：诊断后必问「什么在维持、资源在哪、从哪切入阻力最小」。
  - **「治疗惯性 / 没有结案」**：一直做下去，早已偏离目标也不停——触发原因是不做定期回看与结果监测。避免：定期（如每 8–12 次）回看计划，长期无进展就重新概念化或转介，而非默认「继续做就会好」。
  - **「突然终止当结案」**：来访者脱落 / 治疗师草率收尾，丢掉巩固与分离工作——触发原因是不把结案当一个需要时间的阶段。避免：识别结案信号后留出 2–4 次结案会谈。
  - **「百科式 intake」**：对所有个案都做同等深度的史采集，第一次会谈变成审讯——后果是关系还没建立就把来访者问跑了。避免：intake 聚焦「与当下问题真正相关」的史。
  - **CLI 化暗示**：抽进个案弧线自检——「是否签了知情同意（含保密例外）？」「是否有一份成文的、可修订的个案概念化（不是只有 DSM 标签）？」「治疗目标是否和来访者协商一致？」「最近是否回看过计划是否仍成立？」「结案是计划性的、留了阶段，还是突然停的？」

### W3. 个案概念化流程（把症状/史/关系/情境组织成可工作的理解 + 计划）

- **One-liner**：把一堆零散的信息（症状、发展史、关系模式、当下情境、来访者的资源）组织成一个「为什么这个人现在这样、什么在维持这个问题、从哪入手」的可工作因果理解——这是**跨流派的核心临床能力**，不是某一派的专利。
- **Trigger**：intake 收集完初步信息后，需要把信息变成「能指导治疗」的理解；或工作期遇到瓶颈、原概念化不再解释来访者的反应时，需要重新概念化。
- **Output**：一份个案概念化——一个**动态、可证伪、可修订的工作假设**（不是诊断标签，不是一次写死的文档）。
- **范围提示**：这是治疗师的临床推理流程；它需要受训与督导才能做好，不是「分析一个人」的通用方法。

- **入门 SOP（minimum viable steps）**：
  1. **描述问题（presenting problem）**：把来访者带来的困扰具体化——什么症状 / 行为 / 痛苦，频率、强度、情境、对生活的影响。**输出**=问题的清晰描述（而非模糊的「情绪不好」）。
     - *跳过会发生什么*：问题没描述清楚，后面整个概念化都建在沙子上。
  2. **收集相关信息（4 个维度）**：症状层 + 发展 / 早年史 + 关系模式（含治疗关系里观察到的）+ 当下情境（压力源、资源、文化背景）。**输出**=四个维度的素材。
     - *跳过会发生什么*：只看症状不看史 / 关系 / 情境，会得到一个扁平的、只能套技术的「理解」。
  3. **找模式 / 提假设（formulation 核心动作）**：问「**什么在维持这个问题**」——是认知模式（CBT）、早年关系的重复（动力学）、家庭系统的功能（系统式）、还是创伤的身体印记（创伤取向）？不同流派提供不同的「解释模板」，但动作相同：从素材里提出一个因果假设。
     - *跳过会发生什么*：没有维持机制的假设，治疗就是「试试这个、试试那个」。
  4. **写成可工作的概念化 + 接出治疗计划**：把假设落成文字（很多取向有概念化图 / 模板），明确「从哪个点切入阻力最小」，治疗计划直接从概念化导出。
     - *跳过会发生什么*：概念化只在脑子里 = 无法和督导讨论、无法验证、无法在偏离时被发现。
  5. **持续修订（概念化是假设不是结论）**：每次会谈拿新信息回测概念化——对了就深化，不对就改。把它当**可证伪的工作假设**。
     - *跳过会发生什么*：把第一版概念化当定论，会对「不符合假设的来访者反应」视而不见，错失重要信息。
  - evidence: [T04-S002, T06-S022, T01-S001]

- **资深路径（差异点）**：
  - **optimize**：资深治疗师**边访谈边概念化**——不是「先收集完再分析」，而是在 intake 进行中就动态形成假设、并用接下来的提问去检验；他们的概念化更早成形、更精炼。evidence: [T01-S001, T04-S002]
  - **add**：资深治疗师**额外把治疗关系本身当概念化素材**——来访者怎么对待治疗师、治疗师产生什么反移情反应，被直接读进「这个人的关系模式」；新手往往只从来访者「讲述的内容」里提取，忽略「房间里正在发生的」。也会**额外做跨流派整合**——不被单一流派的解释模板锁死，按来访者情况借用不同视角。evidence: [T01-S011, T01-S022, T04-S006]
  - **skip**：资深治疗师对清晰的循证适配个案，会**跳过深度的早年史挖掘**——若是单纯焦虑 / 特定恐惧且有明确循证技术，他们不会把概念化做成一篇「童年溯源」，直接做一个聚焦维持机制的轻量概念化。evidence: [T02-S004]

- **近期变化（近 12 个月）**：
  - 个案概念化的**方法论本身高度稳态**——「描述问题→收集四维度→提维持机制假设→落成可修订文档」是几十年的临床训练共识，**近 12 个月无重大变化**。
  - 唯一的边缘变化：**2022 起 DSM-5-TR / ICD-11 更新**使概念化里「诊断语言」这一层换了版本（如 C-PTSD 在 ICD-11 有、DSM-5-TR 无），但概念化的核心动作不受影响。触发事件类型：标准更新。evidence: [T02-S001]
  - 这是一个**稳态 workflow**——最近一次显著变化是「从『只下诊断』转向『诊断 + 概念化并重』」的临床共识演进，而那已是数十年的成熟共识。

- **典型耗时**：
  - 入门 SOP：初版概念化需 2–4 次会谈逐步成形；写成文档每个案 ~30–60 min。
  - 资深路径：初版概念化在第 1–2 次会谈就「足够开工」；写成文档更快（~15–30 min），但持续修订贯穿全程。

- **关键工具（与 Track 02 关联）**：
  - 个案概念化图 / 模板（必备——CBT 的认知概念化图、动力学的关系模式图等，取向不同模板不同）、DSM-5-TR / ICD-11（提供诊断语言但不替代概念化）、测评量表（PHQ-9 / GAD-7 等提供症状层的量化素材）。

- **关键人物（与 Track 01 关联）**：
  - Judith Beck 把 CBT 概念化写成了可教学、可督导的「认知概念化图」；Salvador Minuchin 的结构式家庭治疗把「家庭结构 / 边界」做成可观察的概念化单位；David Wallin《心理治疗中的依恋》示范了用依恋视角做概念化。

- **常见失败模式**：
  - **「概念化 = 诊断」**：把「来访者是抑郁症」当成概念化——后果是治疗只剩「对抑郁症做点什么」，没有「对这个人」的理解。避免：诊断完，必答「什么在维持、资源在哪、从哪切入」。
  - **「概念化写死不改」**：第一版当定论——后果是对不符合假设的信息视而不见。避免：每次会谈拿新信息回测，把它当可证伪假设。
  - **「单一流派锁死」**：只会用自己流派的解释模板硬套所有个案——后果是有些来访者被概念化得很勉强。避免：知道自己流派的模板是「一种视角」不是「真相」。
  - **「只从内容提取、忽略过程」**：只分析来访者讲的事，不看治疗关系里的模式与自己的反移情——丢掉了最直接的关系样本。避免：把「房间里发生的」纳入概念化素材。
  - **CLI 化暗示**：抽进概念化自检——「我能用一两句话说出『什么在维持这个问题』吗？」「我的概念化是文字成形的、还是只在脑子里？」「最近一次会谈的信息，符不符合我的概念化？不符合的话我改了吗？」

- **来源（≥3）**：

  - [secondary] Guilford《认知疗法：基础与超越》（认知概念化图模板）：https://www.guilford.com/books/Cognitive-Behavior-Therapy/Judith-Beck/9781462544196 (T04-S002)
  - [secondary] Guilford《心理治疗中的依恋》（依恋视角概念化）：https://www.guilford.com/books/Attachment-in-Psychotherapy/David-Wallin/9781462522712 (T04-S004)

- **Decay risk**：**low** — 个案概念化是这一行最稳态的核心能力之一，方法论数十年成熟，无 12 个月级衰减风险。

### W4. 危机干预流程（自杀 / 自伤 / 虐待 / 急性精神病性发作 的识别 → 评估 → 立即导向危机资源 / 转介 / 上级督导）

> **⚠️ 硬边界 workflow**：危机是**生命安全问题，不是技巧**。本 workflow 写成**硬边界导向转介与上级支持**。它描述的是**持照 / 受督导的从业者**在临床场景中的应对流程，**不是给任何人「自助处理危机」的步骤**。任何读者遇到危机：请立即联系当地危机热线 / 急诊 / 紧急服务，不要把本节当作处理指南。资深治疗师与新手的差异**不是「跳过转介」**——而是「更快识别、更稳处理、转介衔接更扎实」。

- **One-liner**：当来访者出现自杀 / 自伤 / 对他人危险 / 受虐 / 急性精神病性发作的信号时，治疗师从「识别」走到「评估严重度」再走到「立即导向危机资源 / 转介 / 调动上级督导」——目标是把来访者安全地接入比单次治疗会谈更高强度的安全网。
- **Trigger**：来访者透露自杀念头 / 计划 / 既往尝试；自伤行为升级；透露正在受虐或施虐；出现急性精神病性症状（幻觉、妄想、现实检验崩解）；或治疗师从状态、内容、缺席模式中察觉危机信号。
- **Output**：一次完成的危机响应——结构化风险评估记录 + 一个**已落实的下一步安全去向**（危机热线 / 急诊 / 精神科 / 上级督导介入 / 安全计划 + 加密随访），以及完整的危机记录。
- **范围提示（再次强调）**：这是从业者工作流，不是用户处理自己或他人危机的步骤。

- **入门 SOP（minimum viable steps，硬边界导向转介）**：
  1. **识别（不回避地问）**：察觉危机信号后，**直接、具体地问**——不要因为「怕问了反而诱导」而回避（这是被反复证伪的误解）。自杀风险要直接问到念头、计划、手段可得性、时间线、既往尝试。
     - *跳过会发生什么*：不敢问 = 漏掉危机 = 最严重的临床失败，直接关联生命。
  2. **用结构化工具评估严重度**：用 C-SSRS（哥伦比亚自杀严重程度评定量表）等结构化工具把「问到位」标准化——评估的目的是**分诊**（决定下一步去向的强度），不是「做完就免责」。**没有任何工具能预测自杀**。
     - *跳过会发生什么*：凭感觉评估会漏问关键维度（如手段可得性），且无可记录、无法交接。
  3. **立即导向相应强度的资源 / 转介**（这是核心步骤，硬边界）：
     - 高危 / 急性 → **立即导向急诊 / 危机热线（如美国 988）/ 必要时调动紧急服务**，不让来访者「自己回去想想」。
     - 中等风险 → 与来访者**协作制定安全计划**（识别预警信号、应对策略、移除 / 限制手段可得性、列出可联系的人与资源、危机热线号码），并提高随访密度。
     - 涉及虐待 / 对他人危险 → 触发**保密的法定例外**（强制报告义务 / 警告与保护义务，如 Tarasoff 类判例情境）——这是法律义务，不是治疗师的自由裁量。
     - 急性精神病性发作 → 导向**精神科评估**（治疗师做心理治疗，不开药、不做精神科诊断处置）。
     - *跳过会发生什么*：把高危来访者当常规会谈处理、不导向更高强度资源，是直接的生命安全失败与法律责任。
  4. **调动上级督导 / 同行支持**：危机个案**不独自扛**——尽快（理想是当天）联系督导 / 上级 / 同行咨询，把决策放进专业支持网。
     - *跳过会发生什么*：独自处理危机既增加来访者风险，也让治疗师暴露在法律与职业耗竭风险中。
  5. **完整记录 + 安排随访**：把识别到的信号、评估结果、采取的行动、转介去向、安全计划、随访安排全部记录（危机记录的法律与伦理要求比常规记录更严）；落实**加密的、明确时间的随访**。
     - *跳过会发生什么*：危机记录缺失在法律上极其危险；不安排随访 = 危机响应没有闭环。
  - evidence: [T02-S005, T03-S003, T03-S004, T03-S005, T06-S012, T06-S035]

- **资深路径（差异点 —— 不是「跳过转介」，是「更快识别 + 更稳处理 + 衔接更扎实」）**：
  - **optimize**：资深治疗师**识别更快、评估更流畅**——能在来访者还没明说时就从语气、内容、缺席模式里捕捉危机信号，把 C-SSRS 类评估自然地融进对话而非「拿出一张表来念」，来访者的防御更低、信息更真实。**这是把 step 1–2 做得更稳，不是省略。**evidence: [T02-S005, T03-S003]
  - **add**：资深治疗师**额外做的是「衔接质量」**——不止给来访者一个危机热线号码，而是确认转介真的接上了（必要时陪同打电话、和接收方沟通、跟进急诊结果）；还会**额外做自己的情绪处理**——危机个案对治疗师是高替代性创伤负荷，资深者会主动把它带进督导 / 个人体验 / 同行支持，而不是硬扛。evidence: [T03-S002, T06-S033]
  - **add**：资深治疗师**额外维护一份「实时本地危机资源清单」**——本地急诊、危机热线、精神科转介渠道、强制报告流程都是预先备好的，不是危机当下现查；新手往往到了危机才发现不知道本地资源。evidence: [T03-S005, T06-S012]
  - **明确不做的**：资深治疗师**不会**因为「我经验丰富所以能自己 hold 住高危个案」而跳过转介 / 跳过调动上级——经验越多越清楚危机的不可预测性，转介与上级支持是**硬边界，不随经验增长而松动**。

- **近期变化（近 12 个月）**：
  - **2022-07 起、美国 988 自杀与危机生命线启用**：危机干预的转介路径更新——三位数号码替代旧的十位数热线，治疗师必须更新「立即导向资源」的具体信息。触发事件类型：危机基础设施变化。evidence: [T06-S012, T06-S013]
  - **2020 起、远程治疗常态化带来的危机响应难题**：隔屏做危机干预时，治疗师无法直接确保来访者物理环境安全、无法快速调动现场资源——这推动 intake 阶段就必须采集「来访者所在辖区 + 本地危机资源 + 紧急联系人 + 物理地址」。触发事件类型：行业事件。evidence: [T05-S006]
  - **核心流程（识别→结构化评估→立即导向资源→调动上级→记录随访）是稳态的硬边界**——变的是「转介到哪个号码 / 哪个本地资源」，不是流程本身。术语层面有「不自杀契约」→「安全计划」的演进（后者更被循证支持），这一演进已基本完成。

- **典型耗时**：
  - 入门 SOP：危机响应本身在一次会谈内 + 当天的督导联系 + 后续密集随访；时间不固定，由风险等级决定。
  - 资深路径：识别与评估更快，但**衔接与跟进会刻意花更多时间**（确认转介接上了）——资深路径在危机 workflow 上不是「更省时」，而是「更扎实」。

- **关键工具（与 Track 02 关联）**：
  - C-SSRS（必备——自杀风险结构化分诊）、安全计划模板（必备）、本地危机资源清单 + 危机热线（必备，如美国 988）、合规视频工具（远程危机场景）。

- **关键人物（与 Track 01 关联）**：
  - Marsha Linehan 的 DBT 正是为「被判定无法治疗」的高自杀风险来访者造出来的，她对危机的处理框架影响深远，且她公开过自己的边缘性人格与自杀危机经历；Judith Herman 的「先建立安全」原则贯穿危机与创伤工作。

- **常见失败模式**：
  - **「不敢问自杀」**：怕「问了反而诱导」而回避直接询问——这是被反复证伪的误解，后果是漏评危机。避免：危机信号出现就直接、具体地问，回避才是危险的。
  - **「把风险评估当免责清单」**：做完 C-SSRS 就觉得「我尽到责任了」——后果是把分诊工具误当预测工具、误当法律护身符。避免：评估是为了决定下一步去向的强度，且风险是动态的、要反复评。
  - **「独自扛高危个案」**：不调动督导 / 上级 / 同行——后果是来访者风险增加 + 治疗师法律与耗竭风险。避免：危机个案当天联系专业支持网。
  - **「给个号码就算转介了」**：不确认转介是否真的接上——后果是来访者卡在转介缝隙里。避免：跟进到转介确实衔接。
  - **「危机资源清单是危机当下才现查」**：没有预先维护的本地资源清单——后果是危机时手忙脚乱。避免：预先备好本地急诊 / 热线 / 精神科 / 强制报告流程。
  - **CLI 化暗示**：抽进危机响应自检——「我是否直接、具体地问了风险（含计划 / 手段可得性 / 时间线 / 既往尝试）？」「我是否用了结构化工具并记录？」「来访者的下一步安全去向是否已落实、是否与风险等级匹配？」「我是否已（或将在当天）联系督导 / 上级？」「随访是否已安排了明确时间？」「涉及虐待 / 对他人危险时，我是否履行了法定报告 / 警告义务？」——**每一项都是 yes 才算危机响应闭环。**

### W5. 转介流程（识别超出胜任力范围 / 需要其他资源 → 衔接到合适的同行或机构）

- **One-liner**：当一个个案超出治疗师的胜任力范围、需要不同治疗形态、或需要医疗 / 精神科 / 社会服务等额外资源时，治疗师从「识别需要转介」走到「把来访者扎实地衔接到合适的人 / 机构」——转介**不是把人推走，是负责任地接力**。
- **Trigger**：评估发现个案超出自己的训练 / 胜任范围（如需专科创伤治疗、需药物评估、需团体或住院级别照护）；治疗联盟持续无法建立；来访者地理 / 排期 / 费用与自己不匹配；或危机情境需要更高强度资源（与 W4 衔接）。
- **Output**：一次完成的转介——来访者带着**具体的、可落实的下一步去向**（一个名字 / 一个机构 / 一条明确路径），治疗师完成转介记录，理想情况下确认衔接成功。
- **范围提示**：这是从业者「知道自己能力边界」的工作流——胜任力范围（scope of competence）是伦理硬约束，不是「想接什么都行」。

- **入门 SOP（minimum viable steps）**：
  1. **识别转介需要**：诚实判断「这个个案是不是超出我的胜任力范围 / 需要我提供不了的东西」——胜任力 = 训练 + 督导经验 + 持续教育覆盖的范围，不是「我愿不愿意试」。
     - *跳过会发生什么*：超出胜任力还硬接，是直接的伦理违规且可能伤害来访者（intake warning：胜任力范围是硬约束）。
  2. **和来访者坦诚沟通转介理由**：不让来访者感觉「被嫌弃 / 被放弃」——说清楚「为什么转介对你更好」，处理可能的被拒感（尤其依恋议题来访者）。
     - *跳过会发生什么*：突兀的转介会被来访者体验为拒绝 / 抛弃，可能造成二次伤害、也可能让来访者彻底放弃求助。
  3. **给出具体、可落实的去向**：不是「你去找个创伤专家吧」，而是给到具体的同行名字 / 机构 / 注册系统名录 / 明确的查找路径；考虑来访者的地理、费用、排期现实。
     - *跳过会发生什么*：模糊的转介建议 = 来访者大概率卡住、不了了之。
  4. **完成转介衔接 + 记录**：在来访者同意下与接收方沟通必要信息（守保密、有知情同意）；记录转介理由、去向、沟通内容。
     - *跳过会发生什么*：转介无记录、无衔接，个案在缝隙里掉落，连续性断裂。
  - evidence: [T02-S003, T03-S002, T06-S004]

- **资深路径（差异点）**：
  - **add**：资深治疗师**额外维护一个「转介网络」**——预先和不同专长的同行（创伤、伴侣、儿童青少年、精神科、成瘾等）建立关系，转介时不是现查，而是调用已有信任网络；还会**额外做转介后的跟进**——确认来访者真的接上了，必要时和接收方简短交接。新手往往给个建议就结束。evidence: [T03-S002, T05-S007]
  - **optimize**：资深治疗师**更早识别转介需要**——在 intake 阶段就能判断「这个个案不适合我 / 超出我范围」，而不是做了十几次会谈、卡住了才想到转介；早识别省掉了来访者的时间与失望。evidence: [T02-S004]
  - **skip**：对明确匹配自己胜任范围的常规个案，资深治疗师**跳过「要不要转介」的反复纠结**——经验让他们对自己的能力边界有清晰、稳定的认知，不会对每个个案都焦虑「我是不是该转出去」。（注意：这是「跳过不必要的自我怀疑」，不是「跳过该做的转介」。）evidence: [T03-S002]

- **近期变化（近 12 个月）**：
  - **2020 起、远程治疗常态化 + 2024–2026 美国跨州执业协约推进**：转介的「地理可及性」逻辑在变——远程让转介池从「本地」扩到「本州 / 协约覆盖区」，治疗师需要更新「能转给谁」的地理边界认知。触发事件类型：法规变化 + 行业事件。evidence: [T05-S006, T06-S016]
  - **平台派单生态（BetterHelp / Talkspace / 国内简单心理等）改变了部分从业者的转介逻辑**——平台内转介受算法与商业规则影响，转介自主性弱于自建执业。触发事件类型：行业事件。evidence: [T02-S010, T02-S011]
  - 转介的**核心动作（识别边界→坦诚沟通→给具体去向→衔接记录）是稳态**，变的是「转介池的地理范围」和「平台 vs 自主执业的转介自由度」。

- **典型耗时**：
  - 入门 SOP：识别 + 沟通在 1–2 次会谈内；衔接 + 记录额外 ~30 min。
  - 资深路径：识别更早（intake 阶段）；衔接靠已有网络更快，但跟进会多花时间。

- **关键工具（与 Track 02 关联）**：
  - 治疗师目录 / 转介平台（场景特化——海外如 Theravive、专业协会名录；国内中国心理学会注册系统名录）、结构化记录模板（必备）、循证清单 APA Div12（帮判断「这个障碍需要什么专长」）。

- **关键人物（与 Track 01 关联）**：
  - 转介流程没有单一「代表人物」——它是所有流派共享的伦理实践。Marsha Linehan 的 DBT 框架里「治疗师团队 / 转介到更适配资源」是显性环节；危机转介与 W4 强衔接（Linehan 的危机处理）。

- **常见失败模式**：
  - **「超出胜任力还硬接」**：舍不得放手 / 高估自己——伦理违规 + 伤害风险。避免：胜任力是训练 + 督导覆盖的范围，不是意愿；超出就转。
  - **「转介让来访者感觉被抛弃」**：突兀沟通、不处理被拒感——后果是二次伤害、来访者放弃求助。避免：说清「为什么对你更好」，处理分离与被拒情感。
  - **「模糊建议当转介」**：「你去找个专家」——后果是来访者卡住、不了了之。避免：给具体名字 / 机构 / 路径，考虑地理费用排期现实。
  - **「转介后撒手」**：不跟进是否接上——后果是个案在缝隙里掉落。避免：确认衔接，必要时和接收方交接。
  - **CLI 化暗示**：抽进转介自检——「这个个案是否在我的胜任力范围内（训练 + 督导 + 继续教育覆盖）？」「我给来访者的去向是具体可落实的（一个名字 / 机构 / 明确路径），还是模糊建议？」「我是否处理了来访者可能的被拒 / 被抛弃感？」「转介是否记录、是否跟进了衔接？」

- **来源（≥3）**：
  - [surrogate_primary] APA 伦理守则（胜任力范围 / 转介 / 记录条款）：https://www.apa.org/ethics/code (T06-S004)
  - [surrogate_primary] APA 伦理守则（同上 — 转介相关）：https://www.apa.org/ethics/code (T03-S002)
  - [surrogate_primary] APA Division 12（判断障碍所需专长）：https://www.div12.org/treatments/ (T03-S001)
  - [secondary] Theravive 治疗师目录（转介平台生态）：https://www.theravive.com/ (T02-S019)

- **Last_updated**：2026-05-14（核心动作稳态；地理范围与平台逻辑为 2020–2026 信号）
- **Decay risk**：**low** — 转介作为伦理实践高度稳态；唯一会变的是「转介池地理范围」（随远程 / 跨州协约），但这是参数变化不是流程变化。

### W6. 伦理决策流程（遇到伦理两难——保密例外 / 双重关系 / 胜任力边界——怎么走）

- **One-liner**：当治疗师遇到一个伦理两难（保密该不该破、这个关系算不算双重关系、要不要接超出范围的个案、AI 记录该不该用），从「察觉这是个伦理问题」走到「一个有依据、有咨询、有记录的决策」——伦理是**硬约束不是建议**，但具体情境常需要结构化推理而非套公式。
- **Trigger**：出现伦理张力的情境——来访者透露可能触发保密例外的信息（自伤 / 伤人 / 虐待）；和来访者有治疗外的关系交集（双重关系 / 多重关系）；被请求接一个边缘超出胜任力的个案；礼物 / 自我暴露 / 边界跨越的判断；新技术（AI scribe、远程平台数据）的伦理判断；记录调取请求等。
- **Output**：一个**有依据、可辩护、有记录**的伦理决策——附理由、查过的伦理守则条款、咨询过的督导 / 同行 / 伦理顾问、考虑过的选项。
- **范围提示**：这是从业者的伦理推理流程；它建立在受训、对所属伦理守则的掌握、和督导关系之上，不是「凭直觉做个好人」。

- **入门 SOP（minimum viable steps）**：
  1. **识别「这是一个伦理问题」**：先意识到当前情境有伦理张力——很多伦理失误的根源是根本没把它当伦理问题（如不假思索地接受了来访者的礼物 / 进入了双重关系）。
     - *跳过会发生什么*：没识别 = 在不知情中违规，事后才发现已造成伤害。
  2. **查所属伦理守则的相关条款**：明确自己受哪套守则约束（美国 APA 伦理守则 / 英国 BACP 伦理框架 / 中国心理学会注册系统伦理守则——locale 不同，守则不同），查保密、知情同意、双重关系、胜任力、记录的具体条款。
     - *跳过会发生什么*：凭印象判断会漏掉守则里的明确规定（如保密的法定例外边界、双重关系的禁止情形）。
  3. **识别这是「硬规定」还是「需要权衡的两难」**：有些情形守则有明确硬规定（如与当前来访者发生性关系 = 绝对禁止；强制报告 = 法定义务），直接照规定走；有些是真两难（多重关系在小社区不可避免、自我暴露的度），需要结构化权衡。
     - *跳过会发生什么*：把硬规定当「可商量」（危险），或把真两难当「有标准答案」（僵化、可能仍伤害来访者）。
  4. **咨询督导 / 同行 / 伦理顾问**：伦理两难**不独自决定**——带进督导、找同行咨询、必要时联系专业协会的伦理咨询服务。
     - *跳过会发生什么*：独自决定的伦理判断容易被自己的盲点 / 利益 / 焦虑带偏，且事后难辩护。
  5. **做决策 + 记录推理过程**：落实决策，并把「识别了什么张力、查了哪些条款、咨询了谁、考虑了哪些选项、为什么这样决定」记录下来。
     - *跳过会发生什么*：没有记录的伦理决策在被质疑 / 投诉 / 法律审查时无法辩护——记录是「负责任地决策过」的证据。
  - evidence: [T02-S003, T03-S002, T06-S004, T06-S008, T06-S028, T06-S035]

- **资深路径（差异点）**：
  - **optimize**：资深治疗师**伦理推理更快、识别更早**——很多两难在「还没成为问题」时就被预判和规避（如在 intake 设置时就把可能的双重关系风险讲清楚、把保密例外写进知情同意），把「事后救火」变成「事前设计」。evidence: [T03-S002, T06-S004]
  - **add**：资深治疗师**额外做的是「制度化的伦理咨询习惯」**——不是遇到大事才找伦理顾问，而是把伦理议题作为常规督导内容；还会**额外关注新兴伦理灰区**（AI scribe、远程平台数据去向、社交媒体边界），主动更新而非等出事。evidence: [T02-S003, T02-S016, T05-S006]
  - **skip**：对守则有明确硬规定的情形，资深治疗师**跳过「权衡纠结」**——硬规定（如强制报告义务、绝对禁止的关系）直接照走，不浪费精力假装「这是个开放问题」；他们把推理精力集中在真正的两难上。evidence: [T06-S004, T06-S035]
  - **明确不做的**：资深治疗师**不会**因为「我经验丰富所以我的直觉就是伦理」而跳过查守则 / 跳过咨询——伦理守则与同行咨询是硬环节，不随经验松动。

- **近期变化（近 12 个月）**：
  - **2024–2026、由 AI ambient scribe 进入 EHR 推动**：新增一类高频伦理两难——「会谈内容用 AI 处理是否需要、如何取得知情同意；记录准确性责任如何归属；数据去向如何尽调」。这是**新增的伦理决策动作，不是默认可做的事**。触发事件类型：新工具。evidence: [T02-S003, T02-S007]
  - **2023 起、由 FTC 对 BetterHelp 的执法令推动**：远程平台的「来访者数据被共享给广告平台」成为显性伦理问题——治疗师在选择执业平台时多了一层数据伦理尽调。触发事件类型：监管 / 行业事件。evidence: [T02-S016]
  - 伦理决策的**核心流程（识别张力→查守则→分硬规定 vs 两难→咨询→决策记录）是稳态**，伦理守则数年一修（APA 守则 2017 版仍现行），变的是「需要被这套流程处理的新议题」（AI、平台数据）在增多。

- **典型耗时**：
  - 入门 SOP：简单情形当次会谈内 + 一次督导咨询；复杂两难可能跨数周（多轮咨询）。
  - 资深路径：多数两难在事前设计阶段就规避；真两难的推理更快，但仍走完查守则 + 咨询的硬环节。

- **关键工具（与 Track 02 关联）**：
  - 所属伦理守则文本（必备——APA / BACP / 中国心理学会注册系统，locale 决定用哪套）、知情同意模板（把保密例外等写在前面）、专业协会的伦理咨询服务（场景特化）。

- **关键人物（与 Track 01 关联）**：
  - 伦理决策流程没有单一「流派创立者」式的代表人物——它由专业协会（APA / BACP / 中国心理学会）的伦理委员会承载。判例层面，Tarasoff 案塑造了「保密 vs 警告与保护义务」这一最经典的伦理 / 法律两难。

- **常见失败模式**：
  - **「没把它当伦理问题」**：不假思索进入双重关系 / 接受礼物 / 做自我暴露——后果是在不知情中违规。避免：对「有张力的情境」保持伦理敏感，先停下来问「这里有伦理问题吗」。
  - **「凭直觉 / 印象判断，不查守则」**：以为自己记得守则——后果是漏掉明确规定。避免：真遇到就查所属守则的具体条款。
  - **「独自决定伦理两难」**：不咨询督导 / 同行——后果是被自己的盲点带偏、事后难辩护。避免：伦理两难必带进督导或同行咨询。
  - **「决策不记录推理」**：做了决定但没写「为什么」——后果是被质疑时无法辩护。避免：记录识别、查证、咨询、选项、理由的全过程。
  - **「把硬规定当可商量 / 把两难当有标准答案」**：两个方向的错误——前者危险，后者僵化。避免：先分清这是硬规定还是真两难，再决定照走还是权衡。
  - **CLI 化暗示**：抽进伦理决策自检——「我是否意识到这是个伦理问题？」「我是否查了我所属伦理守则的相关条款（APA / BACP / 中国心理学会，按 locale）？」「这是硬规定还是需要权衡的两难？」「我是否咨询了督导 / 同行 / 伦理顾问？」「我是否记录了决策的推理过程？」

- **来源（≥3）**：
  - [surrogate_primary] APA 伦理守则官方页：https://www.apa.org/ethics/code (T06-S004)
  - [surrogate_primary] APA 伦理守则 2017 修订全文 PDF：https://www.apa.org/ethics/code/ethics-code-2017.pdf (T06-S005)
  - [surrogate_primary] BACP 伦理框架（英国 locale）：https://www.bacp.co.uk/about-us/about-the-bacp/our-ethical-framework/ (T06-S008)
  - [secondary] Wikipedia 双重关系（伦理两难入门）：https://en.wikipedia.org/wiki/Dual_relationship (T06-S028)
  - [secondary] Tarasoff 案（保密 vs 警告义务经典两难）：https://en.wikipedia.org/wiki/Tarasoff_v._Regents_of_the_University_of_California (T06-S035)

- **Last_updated**：2026-05-14（流程稳态；AI / 平台数据为 2023–2026 新增议题）
- **Decay risk**：**low**（流程本身）/ **medium**（需要被处理的新议题——AI scribe、远程平台数据伦理在快速增加，从业者需持续更新「灰区清单」）。

### W7. 从受训到独立执业的成长路径（欧美持照体系 locale）

> **locale 提示**：本 workflow 是**欧美持照体系**的路径（美国 LMFT / LCSW / LPC / 临床心理学博士；英国 BACP / UKCP 注册）。**绝不能套到中国大陆**——中国路径见 W8，两者差异极大。即便在欧美内部，美国各州、英国各协会的具体要求也不同，本 workflow 给的是**共性骨架**，具体数字以本辖区为准。

- **One-liner**：把一个想入行的人从「进入研究生 / 受训项目」一路带到「独立持照执业」——经过学位 / 受训、实习见习、督导下的累计执业时数、（多数取向要求或强烈建议的）个人体验、执照考试，是这一行最长尺度的成长工作流，通常**以年计**。
- **Trigger**：一个人决定进入心理治疗 / 咨询行业，开始申请研究生项目 / 受训项目。
- **Output**：一个独立持照的从业者——有学位 / 受训资格证书、有执照、有累计的受督导执业时数、有持续的继续教育（CE）与（多数情况下）持续督导习惯。
- **范围提示**：这是「成为从业者」的路径地图，本身就是在说明——**这一行没有速成，真实门槛是数年受训 + 长期督导 + 个人体验**（intake warning 明确点名「21 天速成」是骗局）。

- **入门 SOP（minimum viable steps）**：
  1. **学位 / 受训项目**：进入并完成相应学位 / 受训项目——美国是临床 / 咨询心理学硕博、社工硕士（MSW）、婚姻家庭治疗硕士等；英国是 BACP / UKCP 认可的咨询 / 心理治疗培训课程。**输出**=学位 / 受训完成资格。
     - *跳过会发生什么*：无认可学位 / 受训 = 无法进入后续持照流程，根本起不了步。
  2. **临床实习 / 见习（practicum / internship）**：在督导下接触真实来访者，开始累计临床小时——这是「从课堂到治疗室」的关键过渡。
     - *跳过会发生什么*：没有督导下的真实临床经验，无法形成临床能力，也凑不够持照所需时数。
  3. **督导下累计执业时数（post-degree supervised hours）**：毕业后在持证督导师的督导下执业，累计辖区要求的执业小时（美国多数州数千小时量级，分直接来访者小时与督导小时）。
     - *跳过会发生什么*：时数不够 = 不能考执照 / 不能独立执业，这是硬性准入门槛。
  4. **个人体验 / 受训分析（personal therapy / training analysis）**：精神分析 / 动力学取向**强制要求**个人受训分析；其他取向多为强烈建议或项目要求。目的是治疗师「了解自己作为工具」。
     - *跳过会发生什么*：对动力学取向是硬性缺失；对其他取向，跳过会留下「治疗师未处理的盲点」风险（反移情未被加工）。
  5. **执照考试 + 申请执照**：通过辖区要求的执照考试（如美国各州的执照考试），向监管委员会申请执照。
     - *跳过会发生什么*：无执照独立执业在持照体系下是违法的。
  6. **持续教育 + 持续督导（独立执业后）**：持照后仍需定期修 CE credits 维持执照；督导虽不再强制，但持续督导 / 同行咨询是负责任执业的常态。
     - *跳过会发生什么*：CE 不够会被吊照；停止一切督导 / 同行咨询会让盲点累积、职业孤立。
  - evidence: [T02-S020, T03-S010, T03-S011, T06-S008, T06-S009, T06-S036]

- **资深路径（差异点 —— 这里「资深」= 已持照的成熟从业者回看 / 指导这条路）**：
  - **add**：成熟从业者会告诉新人**额外重视「督导师选择」和「个人体验」**——而不是只盯着「凑够时数 / 通过考试」这两个硬门槛；督导关系的质量、个人体验的深度，决定的是「能不能成为一个好治疗师」，而时数和执照只是「能不能合法执业」。新人容易把流程当「打卡」。evidence: [T03-S010, T06-S009]
  - **optimize**：成熟从业者会**更早确定取向 / 专长方向**——在受训中后期就开始往一个或两个流派深耕、选对应的督导和继续教育，而不是「什么都学一点」；聚焦让后续的督导时数和 CE 更有累积效应。evidence: [T03-S008, T03-S009]
  - **skip**：成熟从业者回看会说，新人**不必在受训期就追求「掌握所有流派」**——经验告诉他们「先扎实一个取向 + 理解共同要素」远比「样样通样样松」有用；广度可以在执业后逐步加。evidence: [T03-S009]
  - **明确不做的**：成熟从业者**不会**告诉新人「时数 / 个人体验 / 执照可以走捷径」——这条路径的长度本身就是专业性的一部分，「速成」在持照体系里既不可能也不合法。

- **近期变化（近 12 个月）**：
  - **2020 起、远程治疗常态化推动**：受训和督导本身大量转为远程——实习见习的来访者、督导会谈都可能是远程的，受训项目在更新「远程临床训练」的要求与伦理。触发事件类型：行业事件。evidence: [T05-S006]
  - **2024–2026、美国跨州执业协约（PSYPACT / Counseling Compact）推进**：持照后的执业地理范围在扩大，影响「持照→独立执业」之后的执业边界（不影响受训→持照的核心流程，但影响成熟从业者的执业形态）。触发事件类型：法规变化。evidence: [T06-S016]
  - 受训→持照的**核心骨架（学位 / 受训→实习→督导时数→个人体验→执照→持续 CE）是高度稳态**——这是持照体系几十年的制度设计，近 12 个月无重大变化，最近一次系统性变化是远程化对「训练载体」的影响（不是流程本身）。

- **典型耗时**：
  - 入门 SOP：学位 / 受训 2–6 年（视项目，博士更长）+ 毕业后督导时数累计常需 2 年左右 + 个人体验贯穿（动力学取向可数年）→ **从入学到独立持照通常 5–10 年量级**。
  - 资深路径：路径长度不可压缩（是制度硬要求），「资深」体现在更早聚焦方向、更重视督导与个人体验的质量，让同样的年数产出更高的临床能力。

- **关键工具（与 Track 02 关联）**：
  - 培训 / 认证机构（必备——Beck Institute、ACBS、IFS Institute、EMDRIA 等各流派认证体系）、CE credits 体系（必备，维持执照）、督导师名录 / 注册系统（找合格督导）。

- **关键人物（与 Track 01 关联）**：
  - 各流派创立者的「培训体系」就是这条路径的载体——Judith Beck 的 Beck Institute（CBT 培训认证）、Steven Hayes 一脉的 ACBS（ACT/RFT 培训）、Richard Schwartz 的 IFS Institute（Level 1–3）；Marsha Linehan 的 DBT 也有结构化的强化训练路径。

- **常见失败模式**：
  - **「把流程当打卡」**：只盯时数和考试，忽略督导质量与个人体验深度——后果是「合法执业」但不是「好治疗师」。避免：把督导师选择和个人体验当成长的核心，不是凑数。
  - **「受训期什么都学一点」**：广度优先、不扎实任何一个取向——后果是样样松、督导和 CE 无累积效应。避免：受训中后期聚焦一两个取向深耕。
  - **「相信速成营销」**：被「快速拿证 / 缩短路径」吸引——后果是钱白花、且在持照体系里这种路径不被认可。避免：认清这条路径的年数是制度硬要求。
  - **「持照后停止一切督导 / 同行咨询」**：把「不再强制督导」当成「不需要督导」——后果是盲点累积、职业孤立。避免：持续督导 / 同行咨询是负责任执业的常态。
  - **CLI 化暗示**：抽进成长路径自检（给受训中的人）——「我所在辖区的持照路径要求我清楚吗（学位 / 时数 / 考试 / CE）？」「我的督导师是不是我主动选的、关系质量如何，还是随便分配的？」「我的取向 / 专长方向开始聚焦了吗？」「我有没有把个人体验当回事？」

### W8. 从受训到独立执业的成长路径（中国大陆 locale）

> **locale 提示 + 信号薄弱声明**：本 workflow 是**中国大陆**的路径，与 W7（欧美持照体系）**差异极大**——中国**没有**像美国各州 / 英国 BACP 那样的统一法定持照准入。2017 年人社部取消「心理咨询师」国家职业资格认证后，**没有中心化的法定执业资格**，路径是「市场化 + 半行业自律」的混合态。本 workflow 的一手深度信息源结构性偏薄（中国心理学会注册系统官网访问不稳定、培训项目无统一官网、相关内容大量在黑名单渠道），下述路径是基于公开制度信息 + Wave 1 Track 02/06 交叉的**综合推断**，不是逐字考证——Phase 2.8 诚实边界须据此标注。

- **One-liner**：在「没有法定统一准入」的中国大陆，把一个想入行的人从「学习 / 受训」带到「能相对被信任地执业」——路径不是「考一个证就上岗」，而是「学历 / 系统受训 + 长期督导 + 个人体验 + 在认证混乱市场里锚定专业身份」的拼图，且充满营销噪音。
- **Trigger**：一个人决定在中国大陆进入心理咨询 / 心理治疗行业。
- **Output**：一个相对能被同行与机构信任的从业者——通常有相关学历或系统连续培训经历、有持续督导、有个人体验，可能进入了中国心理学会注册系统（注册心理师 / 督导师名录）作为专业身份锚点。
- **范围提示**：这是「在中国成为从业者」的现实路径地图。它本身就在说明 intake warning——**「21 天速成」「考证躺赚」是骗局，真实成长是数年受训 + 长期督导 + 个人体验**；且国内早期从业者收入不稳定是真实职业图景。

- **入门 SOP（minimum viable steps）**：
  1. **认清「没有法定统一准入」这一前提**：先理解 2017 年后市场现状——大量「山寨证书 / 速成班」是营销噪音，「考个证」既不是法定准入、也不等于胜任力。这一步是**认知前提**，跳过它会被营销带进坑。
     - *跳过会发生什么*：不理解这个前提，会把钱和时间投进「速成考证」骗局，且对自己的资质背书有错误预期。
  2. **学历 / 系统连续培训**：走相对扎实的路径——临床与咨询心理学的学历教育（北师大 / 北大 / 华东师大等院校项目），或长程系统连续培训（中德 / 中挪 / 中美等精神分析连续培训项目、简单心理 Uni / 壹心理学院等的系统课程）。**输出**=系统的理论与技术训练基础。
     - *跳过会发生什么*：没有系统训练，只有碎片知识，无法形成临床能力。
  3. **督导下执业 + 长期督导**：在督导师的督导下接案、累计实务经验；督导是国内最实质的专业把关机制之一（中国心理学会注册系统的督导师名录是找合格督导的相对可信入口）。
     - *跳过会发生什么*：无督导执业是高风险的——无人把关临床判断、伦理边界、危机处理，对来访者和从业者都危险。
  4. **个人体验**：接受个人体验 / 受训分析（动力学取向尤其强调）——了解自己作为工具。
     - *跳过会发生什么*：跳过会留下未处理的盲点（反移情未被加工），且多数正规连续培训项目本身要求个人体验时数。
  5. **在认证混乱市场里锚定专业身份**：进入**中国心理学会临床与咨询心理学注册系统**（注册心理师 / 督导师身份及其继续教育、督导时数要求）——这是 2017 取消国家证后**国内最接近行业公认门槛的标准**，是对抗市场噪音的工具。
     - *跳过会发生什么*：在混乱市场里没有任何专业身份锚点，难以与「速成证」从业者区分，来访者和机构难以判断你的专业性。
  6. **执业形态选择 + 持续教育**：选择机构执业 / 平台执业（简单心理 / 壹心理等平台后台 + 抽成派单规则）/ 私人执业（国内缺一体化 EHR，需自行拼凑合规工具）；持续受训、督导、继续教育是常态。
     - *跳过会发生什么*：不持续受训督导会让能力停滞；不理解平台规则会在执业经营上吃亏。
  - evidence: [T02-S018, T03-S012, T06-S015, T06-S016, T06-S017]

- **资深路径（差异点）**：
  - **add**：国内成熟从业者会告诉新人**额外把「注册系统身份 + 督导关系质量 + 个人体验」当真正的专业锚点**——而不是去追逐市场上五花八门的「证书」；在没有法定准入的环境里，**同行认可与受训谱系才是信任的来源**。新人最容易在「考哪个证」上浪费钱和时间。evidence: [T03-S012, T06-S017]
  - **add**：国内成熟从业者**额外要主动管理职业经济现实**——国内早期从业者收入不稳定是真实的，资深者会更早规划「机构 / 平台 / 私人执业」的组合、不被「助人很美好」的鸡汤掩盖收入与职业耗竭的真实风险。evidence: [T02-S018, T03-S012]
  - **optimize**：国内成熟从业者**更早确定取向并选对连续培训项目**——而不是在多个零散工作坊之间漂；连续培训项目的谱系（中德 / 中挪等）本身就是国内的「受训身份」。evidence: [T03-S012]
  - **skip**：国内成熟从业者会让新人**跳过「集邮式考证」**——经验告诉他们市场上的多数证书对专业性几乎无背书价值，把精力投在系统受训 + 督导 + 注册系统上更有效。evidence: [T03-S012]

- **近期变化（近 12 个月）**：
  - **2017 人社部取消「心理咨询师」国家职业资格——这是整条 workflow 的根本性背景**：它改变了中国从业者的入行路径、资质背书逻辑、行业信任机制，催生了认证市场的混乱。这不是「近 12 个月的变化」，但它是理解这条 workflow 现状的前提。触发事件类型：法规变化。evidence: [T06-S015]
  - **decay risk high 的活跃议题**：**中国是否会重建国家级心理咨询执业准入制度**——这是国内这条 workflow 最不确定、最需要周期性复查的点。一旦重建，整条「从受训到独立执业」的路径会被重写。触发事件类型：（潜在的）法规变化。evidence: [T06-S015, T06-S016]
  - **2020 起远程治疗常态化 + 平台生态扩张**：平台执业（简单心理 / 壹心理等）成为国内从业者更主流的执业入口，平台的派单规则、抽成、远程合规要求都在演化。触发事件类型：行业事件。evidence: [T05-S006]
  - 与 W7 不同，**这条 workflow 不是稳态的**——它建立在「国家职业资格已取消、新准入制度未建立」的过渡态上，本身就处在政策不确定性中。

- **典型耗时**：
  - 入门 SOP：系统学历 / 连续培训 2–5 年（连续培训项目本身多为数年制）+ 督导下执业与个人体验贯穿多年 → 走相对扎实的路径同样是**数年量级**，没有合法的「速成」。
  - 资深路径：路径长度同样不可压缩，「资深」体现在不在「考证」上浪费时间、更早锚定注册系统与受训谱系、更早规划职业经济。

- **关键工具（与 Track 02 关联）**：
  - 中国心理学会临床与咨询心理学注册系统（必备——专业身份锚点 + 督导师名录）、连续培训项目（必备——中德 / 中挪 / 中美等 + 简单心理 Uni / 壹心理学院）、国内执业平台后台（场景特化——简单心理 / 壹心理等，国内缺一体化 EHR）。

- **常见失败模式**：
  - **「考证躺赚 / 21 天速成」**：被营销骗——后果是钱白花、对自己资质有错误预期、且对来访者不负责。避免：认清 2017 后无法定统一准入、证书 ≠ 胜任力。
  - **「无督导执业」**：自己接案、没人把关——后果是临床判断 / 伦理 / 危机处理无人把关，对来访者和从业者都危险。避免：长期督导是国内最实质的把关机制，不能省。
  - **「集邮式考证」**：在零散工作坊和各种证书之间漂——后果是无系统受训谱系、无累积。避免：聚焦学历 / 连续培训 + 注册系统 + 督导。
  - **「被『助人很美好』鸡汤掩盖职业现实」**：忽视收入不稳定、替代性创伤、共情耗竭——后果是职业规划缺位、耗竭风险高。避免：把职业经济现实与自我照顾当成长的一部分。

### W9. 临床督导流程（带个案进督导 → 呈现 → 加工 → 把督导转化为下一次会谈的调整）

- **One-liner**：把一个卡住的 / 需要第二双眼睛的个案，从「带进督导」走到「呈现 → 在督导关系里加工 → 把督导所得转化为下一次会谈的具体调整」——督导是这一行**贯穿整个职业生涯**的质量保障与成长机制，不是只有受训期才需要。
- **Trigger**：受训 / 受督导阶段的规定督导时数；个案卡住、来访者无进展、治疗联盟出现破裂；治疗师感到强烈的反移情反应；遇到伦理两难（与 W6 衔接）；遇到危机个案（与 W4 衔接，危机个案当天就要带进督导 / 同行咨询）；或常规的同行咨询。
- **Output**：对个案的新理解 / 新方向——具体到「下一次会谈我会做什么不同」；治疗师自身的成长（对自己作为工具的觉察）；以及（受督导阶段）累计的督导时数与督导记录。
- **范围提示**：这是从业者之间的专业流程——督导 ≠ 个人体验 ≠ 受训分析（这三者在专业上严格区分）。督导聚焦「你和你的个案」，不是治疗督导对象本人。

- **入门 SOP（minimum viable steps）**：
  1. **选个案 + 准备呈现**：决定带哪个个案进督导（通常是卡住的 / 有风险的 / 有强反移情的）、准备呈现材料——个案概念化、近期进展、卡点、（在知情同意与保密处理下的）会谈片段 / 逐字稿 / 录音。
     - *跳过会发生什么*：没准备的督导变成漫谈，督导师无据可依，时间浪费。
  2. **呈现个案**：向督导师 / 督导小组呈现——既呈现「来访者发生了什么」（内容），也呈现「治疗关系里发生了什么、我有什么反应」（过程与反移情）。
     - *跳过会发生什么*：只呈现内容不呈现过程 / 反移情，督导就只能给「技术建议」，碰不到最有价值的关系层面。
  3. **在督导关系里加工**：和督导师一起重新看这个个案——可能是重新概念化、识别盲点、加工反移情、讨论伦理或危机维度、或就治疗联盟的破裂找修复路径。
     - *跳过会发生什么*：把督导当「要答案」而非「加工」，会错过督导真正的价值——督导是协作的理解过程，不是上级派活。
  4. **转化为下一次会谈的具体调整**：把督导所得落成「下一次会谈我会具体做什么不同」——督导如果不落到行动上就只是清谈。
     - *跳过会发生什么*：督导和实际治疗脱节，一边督导一边治疗室里照旧，督导白做。
  5. **记录（受督导阶段）+ 持续性**：受督导阶段记录督导时数与内容；把督导当**贯穿职业生涯**的常态，而非「受训期的任务」。
     - *跳过会发生什么*：受督导阶段记录缺失影响持照 / 注册时数核算；持照后停止一切督导会导致盲点累积、职业孤立。
  - evidence: [T01-S003, T04-S006, T06-S004]

- **资深路径（差异点）**：
  - **optimize**：资深治疗师**带进督导的东西更聚焦**——不是「讲一遍整个个案」，而是精准带出「我卡在哪、我想加工什么」；他们也更善于**主动呈现自己的反移情与盲点**（这恰恰是新手最回避的——怕暴露「没做好」）。evidence: [T04-S006, T01-S022]
  - **add**：资深治疗师**额外把同辈督导 / 同行咨询制度化**——不只依赖「有一个上级督导师」，而是有稳定的同辈督导小组、同行咨询关系；还会**额外把督导当成自我照顾的一部分**——督导也是处理替代性创伤、防职业耗竭的场所，不只是「改进个案」。evidence: [T06-S033, T01-S003]
  - **skip**：资深治疗师对常规、顺利推进的个案，**跳过逐个案的密集督导**——经验让他们对「哪些个案真正需要督导」有判断，把督导资源集中在卡点、风险、强反移情、伦理两难的个案上；新手（也因受训时数要求）倾向于平均地督导所有个案。evidence: [T04-S006]
  - **明确不做的**：资深治疗师**不会**对危机个案、严重伦理两难「省略督导 / 同行咨询」——这些是硬环节，与 W4/W6 衔接，不随经验松动。

- **近期变化（近 12 个月）**：
  - **2020 起、远程督导常态化**：督导会谈本身大量转为远程视频——远程督导改变了督导关系的质感（隔屏的临场感弱），也带来「呈现录音 / 逐字稿的数据安全」要求的更新。触发事件类型：行业事件。evidence: [T05-S006]
  - **2024–2026、AI 转写工具进入会谈记录流程**：督导呈现材料的「逐字稿」可能由 AI 生成——但这继承了 W1/W6 的伦理争议（知情同意、数据安全、准确性责任），督导呈现用 AI 转写材料是新增的伦理动作。触发事件类型：新工具。evidence: [T02-S003, T02-S007]
  - 督导的**核心流程（选个案→呈现内容与过程→在关系里加工→转化为下次会谈调整）是稳态**——这是临床训练几十年的成熟设计，变的是「载体（远程）」和「呈现材料的生成方式（AI 转写）」。

- **典型耗时**：
  - 入门 SOP：单次督导 ~50–90 min（个体督导）/ 团体督导更长；受督导阶段按辖区 / 注册系统要求的督导频率（如每周 / 每若干小时执业配一定督导时数）。
  - 资深路径：督导频率随成熟下降到「按需 + 稳定的同辈督导」，但不归零；单次督导更聚焦、更高效。

- **关键工具（与 Track 02 关联）**：
  - 督导师名录 / 注册系统（必备——海外 CE 体系内的督导资质、中国心理学会注册系统督导师名录）、个案概念化材料 / 逐字稿 / 录音（必备，需知情同意与保密处理）、合规视频工具（远程督导场景）。

- **关键人物（与 Track 01 关联）**：
  - Marsha Linehan 的 DBT 把「咨询师咨询团队（consultation team）」做成了治疗本身的结构性环节——「治疗师也需要被支持」在 DBT 里是显性设计；国内施琪嘉等连续培训项目的督导组是这条 workflow 在中国的实际载体。

- **常见失败模式**：
  - **「不准备就来督导」**：把督导当漫谈——后果是督导师无据可依、时间浪费。避免：带着具体卡点和准备好的呈现材料。
  - **「只呈现内容不呈现过程 / 反移情」**：怕暴露「没做好」而回避——后果是督导只能给技术建议，碰不到关系层。避免：主动呈现自己的反应与盲点，这才是督导最有价值的地方。
  - **「把督导当要答案」**：等督导师给标准答案——后果是错过「加工」的过程价值。避免：督导是协作的理解过程。
  - **「督导不落到行动」**：督导完治疗室里照旧——后果是督导与治疗脱节、白做。避免：每次督导收尾问「下次会谈我会做什么不同」。
  - **「持照后停止一切督导」**：把督导当受训期任务——后果是盲点累积、职业孤立、耗竭风险升高。避免：把督导 / 同辈咨询当贯穿职业生涯的常态。
  - **CLI 化暗示**：抽进督导自检——「我带进督导的个案是有具体卡点的、还是泛泛而谈？」「我有没有呈现治疗关系里发生的、我自己的反移情反应？」「这次督导有没有落成『下次会谈我会做什么不同』？」「危机 / 伦理两难个案，我有没有及时带进督导 / 同行咨询？」

### 入门 SOP 和资深路径之间最大的差距（候选 心智模型）

- **入门 SOP 平均 5–6 步，资深路径几乎不「压缩步数」，而是「重排 + 加深 + 选择性投入」**——这与 LLM/工程类行业很不同。心理治疗的资深差异**不是「少做几步」**，而是：
  1. **把线性步骤变并行 / 提前**（W2、W3：intake/风险/概念化/计划在第 1–2 次就「足够开工」，精细化留到工作期边做边修）；
  2. **额外做新手忽略的「过程层」**（W1、W3、W9：过程评论、把治疗关系当素材、呈现反移情——新手只做「内容层」）；
  3. **把资源选择性地集中在高风险 / 高价值处**（W5、W9：资深者对「哪些个案真正需要转介 / 密集督导」有判断，不平均用力）；
  4. **把「硬环节」做得更稳而非省略**（W4、W6：危机识别更快、伦理事前设计——但转介、查守则、咨询督导这些硬环节资深者**绝不跳过**）。
  → **核心心智模型候选**：「这一行的资深 ≠ 走捷径。资深是『判断力』——判断哪步可以并行 / 提前、哪些个案值得多投入、哪些是绝不能松的硬边界；而硬边界（危机转介、胜任力范围、伦理守则、记录、督导）不随经验增长而松动。」
- **次级心智模型候选**：「治疗师本人是工具，需要被维护」——W1（记反移情）、W7/W8（个人体验是成长核心）、W9（督导兼自我照顾）反复指向：资深路径里「维护治疗师自己」（个人体验、反移情加工、防替代性创伤）和「精进技术」同等重要。与 Wave 1 Track 04 的同名 Phase 2 信号强对齐。

### 近期工作流变化的根本原因（触发事件类型分布）

9 个 workflow 的近期变化触发事件统计：
- **行业事件（远程治疗常态化，2020 起）**：影响 W1 / W2 / W4 / W5 / W7 / W8 / W9 —— **7 个，最大的单一驱动力**。远程化改变的是几乎所有 workflow 的「载体」（会谈、督导、危机响应、受训都可远程），并连带推高了「数据安全 / 知情同意」的权重。
- **新工具（AI ambient scribe 进入 EHR，2024–2026）**：影响 W1 / W6 / W9 —— 3 个。改变「记录的生成方式」，且**在本领域是伦理不确定性而非技术不确定性**（知情同意、准确性责任、数据去向）。
- **标准更新（DSM-5-TR 2022 / ICD-11 2022）**：影响 W2 / W3 —— 2 个。改变「诊断语言版本」，是参数更新不是流程改写。
- **法规变化**：① 美国跨州执业协约（PSYPACT / Counseling Compact）影响 W2 / W5 / W7 的执业地理边界；② **中国 2017 取消国家职业资格** 是 W8 整条 workflow 的根本性背景，且「是否重建准入」是 high decay 活跃议题。
- **危机基础设施变化（美国 988，2022-07）**：影响 W4 的转介路径——参数更新（号码变了），流程不变。
- **主要驱动力结论（→ Phase 2「外部驱动力」）**：本行业工作流的外部驱动力**不是「新技术让流程更高效」（那是工程行业的模式）**，而是 **(a) 载体迁移（远程化）+ (b) 监管 / 制度不确定性（中国准入、美国跨州协约）+ (c) 伦理灰区扩张（AI、平台数据）**。流程骨架本身极稳，「衰减」主要发生在参数层和监管层，不在方法论层。

### 失败模式清单（跨 workflow 汇总 → Phase 2.6 反模式节 + cli_writer 自检项）

按「危害等级」排序：

**生命安全 / 法律级（绝不能犯）**：
- 不敢问自杀 / 回避直接询问危机（W4）——被反复证伪的误解，回避才是危险的。
- 漏评 / 不持续评估风险（W2、W4）——风险是动态的。
- 把高危个案当常规会谈、不导向更高强度资源（W4）。
- 独自扛危机 / 严重伦理两难，不调动督导（W4、W6、W9）。
- 涉及虐待 / 对他人危险时不履行法定报告 / 警告义务（W4、W6）。
- 无知情同意就执业 / 用 AI 处理会谈内容（W2、W6）。
- 危机记录 / 临床记录缺失（W1、W4——被质疑时无法辩护）。

**伦理 / 胜任力级**：
- 超出胜任力范围还硬接个案（W5）。
- 没把有张力的情境「当成伦理问题」（W6）——不知情中违规。
- 凭直觉判断、不查所属伦理守则（W6）。
- 把硬规定当可商量 / 把真两难当有标准答案（W6）。
- 用消费级工具（微信视频等）做远程治疗 / 督导（W1、W9——继承自 Track 02）。

**临床有效性级**：
- 跳过个案概念化、把 DSM 标签当治疗计划（W2、W3）。
- 概念化写死不改 / 单一流派锁死（W3）。
- 治疗惯性——一直做、早已偏离目标也不停（W2）。
- 突然终止当结案，丢掉巩固与分离工作（W2）。
- 带着炸开的情绪送走来访者（W1）。
- 只记 / 只呈现内容，忽略过程与反移情（W1、W3、W9）。

**职业可持续性级**：
- 把流程当打卡、忽略督导质量与个人体验（W7、W8、W9）。
- 持照 / 成熟后停止一切督导与同行咨询（W7、W9——盲点累积、职业孤立）。
- 被「21 天速成 / 考证躺赚 / 助人很美好」营销与鸡汤带偏（W8）——尤其中国 locale。
- 忽视收入不稳定 / 替代性创伤 / 共情耗竭的真实风险（W8）。
