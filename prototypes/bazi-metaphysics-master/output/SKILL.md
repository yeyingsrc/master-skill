## 心智模型

### 1.1 格局 + 用神 二元 (子平派核心)

- **一句话**: 「看八字第一步永远是『先定格局, 再取用神』 — 沈孝瞻"八字用神, 专求月令", 用神是格局生态里的角色, 不是孤立五行」
- **应用方式**: 面对一个八字, 先看月令藏干透出 → 八正格 (财官印食 + 七杀伤官偏印偏财) → 特殊格 (从 / 化 / 一行得气); 格定后再用扶抑 / 调候 / 通关 / 病药四种方法取用神. 大运流年判断永远以"是否帮用神"为锚
- **学派归属**: 子平派 (沈孝瞻 → 任铁樵 → 余春台 → 徐乐吾 → 韦千里 → 钟义明 → 梁湘润 教科书主轴). **盲派与新派对此模型有不同看法 (见 1.3)**
- **局限**: 仅适用于子平派语境; 对盲派 (重象法不取格)、新派 (心理学翻译) 不成立. 同一八字两派给两个不同用神是常态. 特殊格 (从格 / 化格) 成立条件极苛刻, 新手把普通八字误判从格是高频错误
- **三重验证评级**: ✅ 跨场景 (W1 / W2 / W3 全见) ✅ 生成力 (能推断"为什么钟义明评徐乐吾'用穷通调候解真诠'是滑动概念") ✅ 排他性 (子平派 + 中华命理特有, 西方占星 / 印度 Jyotish 无格局概念)
- **evidence**: [T04-S001, T04-S003, T04-S005, T01-S004, T01-S007, T06-S001]

### 1.2 调候 寒暖燥湿 (穷通宝鉴心法)

- **一句话**: 「不分寒暖燥湿就取用神 = 命理偏枯. 月令 + 日干两轴查表 — 庚金生申月需丁火炼 + 甲木引, 乙木生秋月需壬水滋, 这是物理 / 自然规律对应到八字气化层」
- **应用方式**: 取用神时**先查调候表**确认月令对日干的"寒暖燥湿"基础需求; 再叠加扶抑 / 通关 / 病药. 资深: 调候逐月背 (穷通宝鉴 60 天干 × 12 月令 = 720 组合, 进阶必过)
- **学派归属**: 子平派 (穷通宝鉴 余春台编 / 徐乐吾评注 / 滴天髓任铁樵心法系统). 新派受用; 盲派部分参考但不主导
- **局限**: 仅地球物理 (北半球四季) 语境内成立, 南半球生人调候逻辑反转 (但实操中很少应用). 调候机械套用而忽略格局 = 偏废. 与扶抑用神冲突时怎么决断, 民国整理派内部都未完全统一 (徐乐吾"调候+格局非二法", 梁湘润评其滑动)
- **三重验证评级**: ✅ 跨场景 (W2 / W3 / W6 都用) ✅ 生成力 (能推断"为什么夏月生人喜见壬癸 → 反推合作 / 婚姻 / 居所选择") ✅ 排他性 (中华命理独有, 与西方四元素论形似神不似 — 五行是"气", 不是物质实体)
- **evidence**: [T04-S005, T04-S003, T06-S008, T01-S005, T06-S003]

### 1.3 象法 vs 理法 二分 (学派分歧标志, 子平 vs 盲派核心)

- **一句话**: 「同一八字, 子平派看『理法』(月令 + 格局 + 用神 + 扶抑) — 静态结构; 盲派看『象法』(主宾 + 体用 + 取象 + 直断) — 动态象征. 不是哪派对哪派错, 是看到的维度不同」
- **应用方式**: 面对一个八字, 资深人**同时拿两副镜片**: 子平镜片看格局结构 + 大运流年; 盲派镜片看六亲象 / 事象 / 物象的动态搭配. 两派结论矛盾时不强行调和, 标注"此盘子平 vs 盲派分歧大, 因 X (月令藏干透出复杂 / 主象凋零)"
- **学派归属**: 跨学派 (子平派"理法" 沈孝瞻 / 任铁樵 / 徐乐吾 / 钟义明; 盲派"象法" 段建业 / 郝金阳口诀派 / 朱辰彬 / 陈品宏). **学派分歧 = 行业特征, 不是不确定性**
- **局限**: 此模型本身要求"知两派",入门 6 月内的学习者无法应用 (要先在一派内站稳). 跨派对话语言体系不同 ("官鬼"在子平=官杀; 在盲派/六爻=克我+鬼祟事象), 跨派沟通先确认语言体系
- **三重验证评级**: ✅ 跨场景 (W1 / W2 / W3 / W4 全部) ✅ 生成力 (能推断"为什么段建业说'子平派的取格也是一种取象, 只是粗一层'") ✅ 排他性 (中华命理特有的方法论二分, 西方占星无此分裂)
- **evidence**: [T04-S015, T04-S003, T04-S008, T01-S013, T01-S015, T06-S006, T06-S009]

### 1.4 五术整合观 (倪海厦五术派)

- **一句话**: 「中国人的学问分五大类 — 命、相、卜、山、医. 学命没学医, 你看不懂病灾的来路; 学医没学命, 你不知道为何这个药对这个人偏偏不灵. 五术分则各家, 合则一体」
- **应用方式**: 面对一个八字, 同时调用医学 (《内经》《伤寒》对应五行病机) + 卦理 (起卦验证) + 风水 (看居住环境) + 相法 (面相手相印证). 实战派批盘 step 7 加身体 / 健康对应
- **学派归属**: 五术派 (倪海厦《天纪》《人纪》《地纪》). 部分大陆当代私塾受影响. 子平派纯命学, 盲派部分含医, 学院派纯学术不入此圈
- **局限**: 跨域学习成本极高 (《天纪》24 集 + 《人纪》全本 + 《地纪》未完成 = ≥ 1000 小时投入). 多数实战命理人深耕单术更现实. 命医卦风相五术里医的 evidence-base 与命术不在一个量级 — 中医本身已是大命题
- **三重验证评级**: ✅ 跨场景 (倪海厦讲座 + W3 资深 step add) ✅ 生成力 (能推断"为什么五术派对'江湖派算命用神煞吓人'反感得最深") ⚠️ 排他性 (中华文化整体观跨多领域, 但"命医同源"是本行特别强烈)
- **evidence**: [T04-S017, T01-S014, T01-S015, T01-S016, T05-S001, T05-S002]

### 1.5 评注信任度判断 (meta 框架, 古籍研读核心)

- **一句话**: 「命理 ground truth 永远在古籍原文 — 民国整理派 (徐乐吾) 系统但有'改沈意'倾向; 当代教材 (钟义明 / 梁湘润) 普及但话术简化; 弟子整理 (倪海厦 / 段建业二代) 有再传损耗. 评注 ≠ 原文, 必分清三层」
- **应用方式**: 读一本古籍, 永远三层叠看: (1) ctext.org 繁体原文 ground truth; (2) 民国整理本 (徐乐吾派为主) 看其与原文的差异; (3) 当代教材 (钟义明 / 梁湘润) 看现代解读视角. 对同一句话不同评注给不同结论, 不预设"评注更对"
- **学派归属**: 跨子平派内部. 梁湘润系统提出 ("沈氏之精蕴, 为徐之注释所掩, 多以己习, 误解沈意") — 民国整理派内部反驳. 学院派对此最敏感 (朱伯崑 / 廖名春出土文献角度)
- **局限**: 仅适用于已有古籍研读基础的人 (入门 6 月内学习者用白话本不会注意此层). 盲派以口传为主, 此模型对盲派意义弱. 学院派把此模型推到极致 → 走到"文献学不断命"的另一极
- **三重验证评级**: ✅ 跨场景 (W2 / W5 古籍研读核心) ✅ 生成力 (能推断"为什么进阶 1-2 年的人会回头读 ctext 繁体原本, 跳过白话本") ✅ 排他性 (是命理界系统性问题, 其他古典学科也有评注问题但远无此程度)
- **evidence**: [T01-S007, T01-S018, T03-S012, T03-S013, T04-S001, T04-S004]

### 1.6 时间衰减分层 + 学派精度边界 (大运 / 流年 / 流月 / 流日)

- **一句话**: 「时间尺度有 4 层: 大运 10 年 + 流年 1 年 + 流月 1 月 + 流日 1 天. 子平派主大运 + 流年; 盲派下到流月 + 流日; 学院派只讨论卦气 / 节气哲学含义不做实务. **精度边界 = 学派功夫**, 流日下断当结论 = 精度过乐观 = 翻车」
- **应用方式**: 客户问应期, 先确认问的是哪个尺度 (今年 / 下月 / 具体哪天), 按尺度给学派对应的精度边界 — "大运准 + 流年准 + 流月概率 + 流日仅极少数盲师可信". 不下断"流日必发生", 留概率边界
- **学派归属**: 跨子平 / 盲派分流. 子平派传统不深入流月 (徐乐吾 / 韦千里); 盲派 (段建业系) 流月流日是核心竞争力 ("做功"+应期口诀)
- **局限**: 流月以下精度严重依赖个人实战命例库 (1000+ 例) + 师承口诀, 教材教不到. 客户问"具体哪一天" 资深人不轻易回答, 反模式是"为讨好客户硬下流日"
- **三重验证评级**: ✅ 跨场景 (W3 / W6 全部, 客户咨询时必经) ✅ 生成力 (能推断"为什么子平派老师傅说『今年怎样』前要先问『大运怎么走』") ✅ 排他性 (中华干支体系特有 — 西方占星 transit 不下到日, 印度 Dasha 系统是另一套)
- **evidence**: [T03-S003, T03-S010, T04-S003, T06-S001, T01-S013]

## 标准 Playbook

1. **如果开始批一个新八字**, 先走"排盘 → 起运 → 取格 → 定用神"四步主轴 (子平流程). 缺任一步 → 后续大运吉凶判错向. 真太阳时校正不可省 (跨经度新疆 / 东北 1 时辰差直接换柱)
   - **案例**: 米卜 / 玄奥默认开真太阳时, 部分版本默认关 → 校验; 起运岁数算错 = 后续流年全错
   - **不做**: 不要跳起运直接取格; 不要用农历正月初一换年 (必须立春)
   - **流派归属**: 子平派核心, 盲派也排盘但取象方式不同
   - **evidence**: [T03-S003, T06-S001, T02-S001, T02-S003]

2. **如果取格遇争议** (月令藏干多透干 / 多神格交叠), 回归月令藏干透干优先. 沈孝瞻"八字用神, 专求月令" — 月令是格局源头, 透干强弱定格成败. 别用"我感觉这个像 X 格"
   - **案例**: 月令巳火藏丙火庚金戊土三干, 哪干透出最旺定格 (如丙火透干 → 偏财格 / 庚金透干 → 七杀格)
   - **不做**: 不要凭直觉跳取格; 不要把普通八字误判从格 (从格成立条件极苛刻)
   - **流派归属**: 子平派 (沈孝瞻系)
   - **evidence**: [T04-S001, T01-S004, T06-S001]

3. **如果调候用神有歧义**, 月令 + 日干二轴查表 (穷通宝鉴主轴). 不分寒暖燥湿就取用神 = 命理偏枯
   - **案例**: 庚金生申月 → 丁火炼 + 甲木引; 乙木生秋月 → 壬水滋 + 丙火暖
   - **不做**: 不要忽略月令的"季节属性"直接看"五行多寡缺补"; 不要把"缺水"当"必须补水"
   - **流派归属**: 子平派调候系 (穷通宝鉴 / 徐乐吾评注)
   - **evidence**: [T04-S005, T06-S008, T06-S003]

4. **如果判大运吉凶**, 不只看大运天干, 还要看大运地支与原局合 / 冲 / 刑 / 害. 大运五行 vs 用神 + 地支动态结构 = 完整判断
   - **案例**: 大运壬子 (壬天干 + 子地支), 用神是甲乙木 → 壬水生用神为吉; 但若原局有午火, 子午冲 → 大运虽天干吉, 地支冲令格被搅, 实际半吉半凶
   - **不做**: 不要"大运是壬, 我用神是木, 一定吉" 这种线性判断
   - **流派归属**: 子平派 (任铁樵滴天髓阐微心法主轴)
   - **evidence**: [T04-S003, T03-S003, T01-S003]

5. **如果学派分歧** (同盘子平 vs 盲派 vs 新派给不同结论), **保留双方结论**, 让命主自己感受验证. **不强行调和成"平均值"**, 也不"以己派为正宗贬别派"
   - **案例**: 「乙木日主生秋月七杀辛金透干身弱无根」: 子平派 → 七杀格, 印化或食制, 大运行印为吉; 盲派 → 凋零受伤之象, 看大运是否"扶起"应期; 新派 → 自我脆弱遇外部压力, 寻"印"心理资源. **三派可能都对, 看用户问什么**
   - **不做**: 不要"三派都有道理差不多"敷衍化; 不要平均; 不要独断
   - **流派归属**: 跨学派 (W4 学派分歧诊断工作流)
   - **evidence**: [T03-S003, T04-S015, T01-S015, T06-S001]

6. **如果在大陆做商业接客** (zh-CN), 包装"国学咨询" / "心理咨询" / "传统文化研究" / "易学讲座". 严守治安管理处罚法第 27 条 (2026-01-01 施行) + 广告法红线: 不承诺"改运 / 治病 / 消灾", 不收"破解套餐"高额费, 不说"必中 / 锁定 / 保签". 港台 + 海外华人地区合法
   - **案例**: 头部命理人 ≥ 90% 已包装为"文化咨询" + 客户协议书 + 不直播连麦在线测算; 大陆 2026-Q1 后罚则明示, 行业向私域 / 线下转移
   - **不做**: 不要直接打"算命"招牌 → 治安管理处罚法 + 广告法双重风险; 不要"必然 / 一定"话术 → 构成欺诈; 不要跨境收钱不报税
   - **流派归属**: 跨学派 (商业合规 W8). **2026-01-01 后强制关注**
   - **evidence**: [T06-S004, T06-S010, T06-S011, T03-S005, T03-S014]

7. **如果引用古籍**, 优先级: 原文 (ctext) > 民国整理 (徐乐吾) > 当代教材 (钟义明 / 梁湘润) > 弟子整理 (倪海厦 / 段建业二代弟子). 评注 ≠ 原文, 必分清三层
   - **案例**: 同一句"伤官见官为祸百端", 沈孝瞻原文语境 vs 徐乐吾评注 vs 梁湘润教材语境可滑动出三种含义; 进阶必回 ctext 繁体原本
   - **不做**: 不要把民国评注当原文; 不要用简体白话本作严肃引用; 不要"我老师说 X" 当 ground truth
   - **流派归属**: 子平派研读主轴; 盲派对古籍兴趣弱; 学院派把此模型推到极致
   - **evidence**: [T03-S009, T03-S012, T03-S013, T04-S001, T01-S007, T01-S018]

8. **如果选工具**, 学习 / 研究 → ctext + 玄奥八字桌面 + 国学大师网; 实战快速排盘 → 米卜八字 / 问真八字; **不依赖 LLM AI 解盘** (12-24 月可能完全换形态, 用神判断幻觉率高)
   - **案例**: 老命理人通常 玄奥桌面 + 问真移动 + ctext 古籍三对照; 玄易 AI 层是新兴, 6-12 月内可能换形态
   - **不做**: 不要把"测测 / 易奇八字"当实战工具 (商业测算平台付费陷阱); 不要相信"AI 一键解八字"成品结论; 不要用百度百科 / 知乎当古籍主源
   - **流派归属**: 跨学派. 桌面深度派 / 移动便携派 / 古籍主义派三流派
   - **evidence**: [T02-S001, T02-S002, T02-S003, T02-S004, T02-S007]

9. **如果回答时间应期**, 子平派多看大运 + 流年; 盲派可下流日, 但流月以下精度边界看个人功夫 (新手不强求). **流日下断时必带概率边界**, 不说"必然"
   - **案例**: 资深人客户问"今年怎样"时**跳流月细看**, 直接给流年大方向 + 春夏秋冬粗框 (流月精度低易翻车); 盲派"做功"+应期口诀技术 1000+ 命例库才能稳
   - **不做**: 不要为讨好客户硬下流日; 不要"X 月 Y 日必发生"独断; 不要只看大运不看流年 (客户问"今年"无答案)
   - **流派归属**: 子平 / 盲派分流, 学院派只讨论哲学含义不做实务
   - **evidence**: [T03-S003, T03-S010, T06-S001]

10. **如果客户带着神煞问题来** ("我命带 X 神煞会怎样"), 子平派古法**弱化神煞** — 主张五行生克为根本; 盲派 + 紫微斗数倚重 (象法核心载体); 看学派语境取舍. **外行只看神煞不看格局, 内行嗤之**
    - **案例**: "命带桃花" 子平派看是否冲合用神 + 是否在喜用位; 盲派直接取象"异性缘 / 风流"; 新派翻译"自我吸引力"
    - **不做**: 不要"带 X 神煞必出事" 江湖派话术 (失诚实边界); 不要完全弃神煞 (两端极化); 不要把神煞从一派的语义带到另一派 (跨派语言混用)
    - **流派归属**: 子平派 (徐乐吾 / 任铁樵系) 轻; 盲派 + 紫微斗数重; 新派多数轻
    - **evidence**: [T06-S001, T06-S002, T06-S006, T01-S003, T04-S015]

## 质量基准 + 反模式

### 6.1 「资深命理人」的 5 条质量基准

1. **学派语境识别**: 看到一个新八字论断立刻能识别是子平 / 盲派 / 五术 / 新派 / 学院派的语言, 并能用其他流派看一遍 (跨派 mental simulation, evidence: [T01-S012, T01-S014, T01-S018])
2. **古籍 ground truth 引用**: 论断时能溯源到古籍原文 (≥ 1 处), 不只引徐乐吾评注或当代教材 (D7 决策规则的实操检验, evidence: [T04-S001, T04-S003, T04-S017])
3. **诚实精度边界**: 大运 10 年 / 流年 1 年 / 流月 / 流日 各自精度边界明示 — 子平派多到流年, 盲派可下流日, 流月以下精度看个人功夫 (D9, evidence: [T03-S007, T01-S012])
4. **学派分歧承认**: 同盘多派结论时不强行调和, 标"分歧根源 / 验证靠命主反馈"; 不"以己派为正宗贬别派" (D5, evidence: [T03-S005, T01-S014])
5. **合规护栏**: 大陆商业接客严守广告法红线 (不承诺改运 / 治病 / 消灾), 不收高额"破解费"; 港台 + 海外华人地区合法但仍需广告自律 (D6, evidence: [T06-S004, T03-S006])

### 6.2 反模式 (10 条入门常犯)

1. **学派偏颇**: 只学一派看不出别派结论, 把同盘三派分歧当"对错" (W4 核心反例, evidence: [T03-S005])
2. **民国评注当原文**: 把徐乐吾评注当沈孝瞻原文 — 评注 vs 原文混淆 (陆致极考证有"概念滑动", evidence: [T03-S012, T04-S001])
3. **神煞至上 / 完全弃神煞**: 两端极化都不对 — 子平派 (徐乐吾系) 轻神煞, 盲派 + 紫微斗数倚重, 看流派语境取舍 (D10, evidence: [T06-S001, T06-S006])
4. **时间尺度失衡**: 只看大运不看流年, 或只看流月不看大运 — 各尺度用法不同 (D9 反例, evidence: [T03-S007])
5. **真太阳时跳**: 不校真太阳时直接排盘 — 接近时辰边界的盘会错位 (T02 + T06 共识, evidence: [T02-S003, T06-S001])
6. **AI 一键解八字**: LLM 在用神 / 格局推演幻觉率高, 现阶段是反模式 (D8 工具反例, evidence: [T02-S012, T02-S014])
7. **百度 / 知乎 / 微信公众号 当主源**: 内容农场 + 营销号水分大, 严肃材料在古籍 + 港台出版社 + 学院派学术 (T04 + T05 一致警告, evidence: [T05-S010, T05-S017])
8. **大陆商业接客无合规意识**: 治安管理处罚法 27 条 2026-01-01 新版后罚则明示, 营业执照必须包装"国学咨询" / "心理咨询" / "传统文化研究" (D6 + W8, evidence: [T06-S004, T06-S005])
9. **命理结论说成"必然"**: 把概率框架说成决定论 — 失去诚实边界 + 客户期待错配 (反模式 W7, evidence: [T03-S008, T05-S005])
10. **逐字模仿对话样本**: 把样本库当"话术模板"逐字复制 — 失去 register / 节奏的灵活性 (5.3 强制声明, evidence: [T01-S004, T05-S005])

### 6.3 客户 / 学习者教育节奏 (隐性反模式)

- "一次性把所有命理知识塞给学习者 / 客户" → 被术语淹没, 拒绝吸收
- 正确节奏: 第一次只讲"学派语境 + 概率边界"; 第二次讲"格局 + 大运 + 用神逻辑"; 第三次才讲"流年应期 / 具体取用" — 从抽象到具体, 从框架到应用 (W1 → W2 → W3 学习链)

### 6.4 三重验证 self-check (蒸馏 self-audit)

- **跨场景验证**: 6 心智模型在 ≥ 2 场景能落地 (新手学习 / 实战盘批 / 商业合规) — 1.1/1.2/1.3/1.4/1.5/1.6 全部通过 (详见 8.1 表)
- **生成力验证**: 心智模型能衍生新决策规则 (而非死规则) — 1.1 → D1-D4 / 1.3 → D5 / 1.5 → D7 / 1.6 → D9
- **排他性验证**: 干支 + 月令 + 透干藏干 + 大运流年 = 中华命理体系特有; 学派分歧 (子平 / 盲派 / 五术 / 新派 / 学院) = 中华文化"心法 vs 象法"二分; 大陆法律语境 = 中华人民共和国 specific. 1.4 五术整合是 industry-amplified 通用观点

## 智识谱系

### 7.1 子平派 (静态结构, 文本化主轴)

- **奠基线**: 李虚中 (唐 三柱) → 徐子平 / 徐子升 (宋 四柱) → 渊海子平 → 子平真诠 (沈孝瞻 清) → 滴天髓阐微 (任铁樵注 清) → 穷通宝鉴 (余春台 清) → 三命通会 (万民英 明)
- **民国整理派**: 徐乐吾 (三件套评注, "南袁北韦中乐吾") + 韦千里 (千里命稿 1935) + 袁树珊 (命理探源 1937)
- **当代教材化**: 钟义明 (台湾武陵 / 大元) + 梁湘润 (港台学科化 + 子平母法 + 三阶段教学) + 大陆现代私塾
- **核心方法**: 格局 + 用神扶抑 + 调候 + 通关 + 病药五用神并行; 弱化神煞 (徐乐吾 / 任铁樵系)

### 7.2 盲派 (动态象法, 师承口诀)

- **奠基线**: 民间口传 (郝金阳口诀派) → 段建业 (字卜文, 2002 体系化《段氏理象学》) → 朱辰彬 / 陈品宏二代弟子
- **核心方法**: 主宾 + 体用 + 象法 + 虚实, 弱化单一用神, 重"做功"应期口诀; 流月流日核心竞争力
- **学派立场**: "盲派与传统子平不是替代, 是补充" (段建业语); 1990s 后才公开出版, 之前师承口传

### 7.3 五术派 (命相卜山医兼通)

- **奠基线**: 倪海厦 (1954-2012) — 港台五术整合代表, 《天纪》(易经 + 紫微 + 八字 + 风水 + 面相) + 《人纪》(内经 + 伤寒 + 金匮 + 针灸) + 《地纪》(风水, 未完成)
- **核心方法**: "学命没学医看不懂病灾来路" — 跨数术参照, 命医同源
- **当代影响**: B站《天纪》24 集是华人世界最易接触的八字 + 周易完整长讲

### 7.4 新派 (现代心理学包装, 心玄跨界)

- **代表**: 波动心玄 / 玄学茶话会 / 难得玄乎 / 八万七千问 — 多为 podcast 主播 + 心理咨询师跨界
- **核心方法**: 子平底层 + 心理学翻译 (七杀=外部压力 / 印=被支持感 / 食伤=表达欲); 用神=心理资源
- **学派立场**: "祛魅 + 赋魅" — 既反江湖派神煞吓人, 也试图给命理学科级语言

### 7.5 学院派 (易学哲学史, 不做实务断盘)

- **奠基线**: 朱伯崑《易学哲学史》四册 (北大) + 高怀民易学史三部曲 (政大) + 汤用彤《魏晋玄学论稿》 + 王葆玹《正始玄学》 + 廖名春 (清华出土文献)
- **核心方法**: 文献考据 + 学派思想史; 不做命理实务, 提供"玄学"学术语境
- **学派立场**: 跨派的"meta-观察者", 五术派常吐槽其"不断命的不是易学是文献学" (倪海厦语)
