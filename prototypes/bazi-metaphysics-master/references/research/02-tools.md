## 工具卡片 (要点)

### 排盘工具

- **米卜八字** (T02-S001, 必备, decay=low): 网页+APP 双端, 新手零成本上手. 适合: 学习初期/出门快速复盘. 不适合: 盲派深度研究 (神煞标注简化). 失败模式: 真太阳时部分版本默认关. 评价: 子平派默认 友好.
- **玄奥八字 桌面版** (T02-S002, 必备, decay=low): 港台老牌, 神煞/调候/格局/多盘对比最完整. 适合: 子平派深度学习+实战命理人多盘对比. 不适合: 完全免费用户 (注册码门槛). 失败模式: 大陆 Win11 偶有兼容问题. 评价: 老命理人黄金标准.
- **问真八字** (T02-S003, 必备, decay=low): 真太阳时校正 + 滴天髓/穷通宝鉴/三命通会智能匹配. 适合: 中级命理人想看古籍对应解释. 不适合: 完全相信文献匹配的新手 (需自己判断). 失败模式: 文献匹配生硬, 需当参考非结论.
- **元亨利贞网** (T02-S005, 场景特化, decay=medium): 大陆元老级门户, 排盘+论坛+古籍混合. 适合: 老网友圈层 + 找历史讨论. 不适合: 移动优先用户 (UI 老旧).
- **玄易 (ixybz)** (T02-S004, 场景特化+新兴, decay=medium-high): 八字+合盘+六爻+罗盘+黄历+AI 一站式. 适合: 多数术覆盖+AI 解释层尝鲜. 不适合: 单纯子平深度. 失败模式: AI 解读层是新兴, 6-12 月内可能换形态.
- **易奇八字** (T02-S014, reference, decay=high): 商业测算平台, 商城+起名+付费会员. 适合: 大众消费者. 不适合: 实战命理人 (功能藏在付费墙). 评价: 用户投诉「付费陷阱」, 仅入 reference.
- **测测** (T02-S015, reference): 星座+八字+AI 问答 大众向, 实战命理人不依赖.

### 紫微斗数 (边界数术)

- **紫微斗数实验室 iziwei.com.cn** (T02-S011, 场景特化): 网页免费, 原局/大限/流年/流月/流日全. 适合: 紫微入门+辅助八字. 失败模式: 飞星派/三合派切换需手动配置.
- **陈剑紫微 / 文墨天机** (T02-S012, 场景特化): 桌面+移动, 三方四正/十二宫盘完整. 适合: 紫微深度研究. 不适合: 临时网页用户.
- **(原候选「清水居士」「紫微闲人」 — 检索未找到独立官网, 退入二手列表 — 由当代主流 文墨天机 / 陈剑 替代)**

### 古籍 / 学术 (实战命理人 30% 时间)

- **ctext.org** (T02-S007, 必备, decay=low): 中国哲学书电子化计划, 3 万+ 著作 50 亿字, 三命通会 / 渊海子平 / 滴天髓 繁体原文最准. 适合: 学术研究 + 古籍引用 ground truth. 不适合: 简体白话用户 (无翻译).

## 选型决策树

### 决策树 A: 用户的核心目标是什么？

#### Branch 1: 新手入门, 想验证「八字能做什么」
→ 推荐: **米卜八字** (T02-S001) 网页/APP, 零门槛
→ 不推荐: 玄奥桌面 (学习曲线陡); 易奇八字 (付费陷阱)
→ 配套: B 站 (T02-S013) 倪海厦天纪 + 段建业盲派 长讲沉浸

#### Branch 2: 子平派深度学习, 看格局/用神/调候
→ 推荐: **玄奥八字 桌面版** (T02-S002) — 神煞 + 调候 + 多盘对比 完整
→ 替代: **问真八字** (T02-S003) 移动端 — 文献智能匹配
→ 古籍底座: **ctext.org** (T02-S007) 三命通会 / 子平真诠 原文校对

#### Branch 3: 盲派 (象法 / 实战速断)
→ 推荐: 老盲师多手排不依赖软件; 软件作辅助仅取干支
→ 实战工具: **玄奥** 或 **米卜** 都可 (盲派看重干支组合本身)
→ 学习材料: B 站 段建业弟子整理频道 (T02-S013)

#### Branch 4: 学术研究 / 古籍考据
→ 推荐: **ctext.org** (T02-S007) + **archive.org** (T02-S008) + **维基文库** (T02-S010) 三源对照
→ 简体环境: **国学大师网** (T02-S006) 备查
→ 工具垫底: 玄奥 桌面排盘 (验证古籍提到的盘例)

#### Branch 5: 紫微斗数 (边界数术)
→ 网页轻量: **紫微斗数实验室 iziwei** (T02-S011)
→ 桌面深度: **陈剑紫微 / 文墨天机** (T02-S012)
→ 警告: 原候选「清水居士 / 紫微闲人」未找到现役官网, 退入历史

#### Branch 6: 多数术覆盖 + AI 解释尝鲜
→ 实验性: **玄易 (ixybz)** (T02-S004) — 八字+合盘+六爻+罗盘+AI 一站式
→ 标注: AI 解读层 6-12 月内可能换形态, 不依赖此层做严肃判断

## 避坑清单

- 不要把 **测测 / 易奇八字** 当成实战工具 — 商业测算平台为大众设计, 把高级功能藏在付费墙后, 实战命理人圈层不用 (T02-S014, T02-S015)
- 不要相信任何「AI 一键解八字」的成品结论 — 包括玄易的 AI 层 — LLM 在用神/格局推演上幻觉率高, 仅作交叉参考
- 不要跳过真太阳时校正 — 米卜部分版本默认关闭, 跨经度 (新疆/东北) 偏差可达 1 时辰, 直接换柱 (T02-S001)
- 不要把百度百科 / 知乎 当古籍主源 — 简体白话转译多有讹误, 严肃引用必须回到 ctext.org 繁体原文 (T02-S007)
- 不要用单一软件结论下断 — 实战命理人通常 玄奥 桌面 + 问真 移动 + ctext 古籍 三对照
- 不要把紫微斗数 / 八字 软件混同 — 排盘逻辑不同, 紫微用 iziwei / 文墨天机, 八字用 玄奥 / 米卜
- B 站 / YouTube 长讲资料随时被删 — 重要长讲先离线备份再消化 (T02-S013)
