## A. 古籍核心 — 子平派必读四大 (低衰减 / 永恒)
### A1. 子平真诠


- **A1. 子平真诠** 沈孝瞻 清乾隆 (T04-S001/S002). 子平派教科书, 格局 + 用神扶抑核心. 进阶. 版本: **徐乐吾评注本**最流通; 中医古籍社 2012 白话本入门; ctext 繁体为校勘 ground truth. 徐评注有改动古意之争 — 进阶后回看清原本. 读完得到: 看八字先定格局 (正官 / 七杀 / 财 / 印 / 食伤 八格), 再取用神
### A2. 滴天髓

- **A2. 滴天髓** 京图原 / 任铁樵 增注 清道光 (T04-S003/S004). 子平派**高阶心法** — 中和 / 寒暖燥湿 / 通关. 高阶 (需真诠 + 3 年实盘). 1933 袁树珊《阐微》; 1935 徐乐吾《征义》. **流派分歧**: 子平派论格局气势; 盲派只取"象法"片段不重格局
### A3. 穷通宝鉴

- **A3. 穷通宝鉴** 余春台 编 清光绪 (T04-S005). 调候用神核心 (寒暖燥湿). 别名: 栏江网 → 造化元钥 → 穷通宝鉴. 进阶, 按月令调候逐字背. **徐乐吾评注本** + 中医古籍社白话本. 读完: 不同月令日主调候用神速查 (例: 庚金生申月需丁火炼 + 甲木引)
### A4. 渊海子平

- **A4. 渊海子平** 徐升 宋末 (T04-S010). 子平派早期总结, 系统性次于真诠, 含赋文诀法 (元理赋 / 喜忌篇). 进阶, 海南 / 中医古籍社多家本均可

## H. 学习路径 (canon 必出)

**新手 6 个月 (入门)**: 韦千里《千里命稿》(1.5 月) → 邵伟华《四柱预测学》普及 (0.5 月, 可选) → 徐乐吾《子平真诠评注》白话本 (3 月) → 周易经传中华书局十三经注疏 (1 月). **目标**: 起盘 + 看八格 + 取浅层用神 + 大运流年初判.

**进阶 1-2 年**: 沈孝瞻《子平真诠》清原本 + 徐评注对读 (3 月) → 余春台《穷通宝鉴评注》调候逐月背 (3 月) → 任铁樵《滴天髓》阐微 + 征义对读 (4 月) → 渊海子平赋文 (2 月) → 段建业《段氏理象学》盲派开眼界 (3 月) → 钟义明现代破译 (3 月). **目标**: 三派各有判盘视角, 知道同盘不同解.

**高阶 3-5 年**: 万民英《三命通会》查阅 → 廖中《五行精纪》+ 李虚中《命书》源流 → 朱伯崑《易学哲学史》四卷 (1 年) → 高怀民易学史三部曲 → 汤用彤 / 王葆玹 魏晋玄学 → 倪海厦 天纪 + 人纪 跨领域. **目标**: 四派 (子平 / 盲派 / 新派 / 学院派) 分别定位, 知道分歧从哪儿来, 不被某一派话术 hijack.

## I. 学派分歧 (canon 必保留, 不要平均化)

| 同一本 / 同一概念 | 子平派 | 盲派 | 新派 |
|---|---|---|---|
| 滴天髓 | 论格局气势 / 中和 | 取象法片段 | 心理学化解读 |
| 用神 | 扶抑 + 调候 + 通关 + 病药 | 弱化用神, 重"象" | 引入心理需求 |
| 同一八字 | 先定格局 → 用神 → 喜忌 | 直接看象 / 速断 | 看心理结构 |
| 邵伟华 | 学术性不足 | 不入流 | 普及功大 |

**蒸馏要点**: 三派对同一八字给不同结论. SKILL.md 必须明示 **学派分歧 = 领域特征**, 非"探索中的不确定性".

## Phase 2 接口

**核心 idea (≥ 3 canon 共讨论, 候选心智模型)**: **格局 + 用神 二元** (真诠 / 渊海 / 滴天髓 / 千里 / 命理探源) · **调候 寒暖燥湿** (穷通 / 滴天髓 / 三命) · **象法 vs 理法** (段氏 / 滴天髓 / 李虚中) — 学派分歧标志.

**智识谱系**: **子平派** 李虚中 (唐 三柱) → 徐子平 (宋 四柱) → 渊海子平 → 子平真诠 → 滴天髓 → 穷通 → 徐乐吾 / 韦千里 / 袁树珊 (民国) → 钟义明 (台湾) → 现代私塾. **盲派** 民间口传 → 段建业 (2000s 公开化) → 朱辰彬等. **学院派** 朱伯崑 / 高怀民 / 汤用彤 / 王葆玹 (不做实务, 提供"玄学"学术语境). **主要分歧** 格局派 vs 象法派 / 古法 (三柱) vs 今法 (四柱) / 普及派 vs 严谨派.

**冷僻信号**: canon 维度并不薄 — 一手 73%, 学派分歧清晰, 学习路径成形 (新手 6 月 / 进阶 1-2 年 / 高阶 3-5 年). 数据厚度高于一般冷僻行业 (古籍永恒 + douban 主页可计 verified_primary).
