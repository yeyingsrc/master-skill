## 一、官方 / 数据源(最高权重)

> 这一层是房产投资决策的**数据底座**。共识:**官方网签数据 > 中介挂牌价**,**流动性指标(成交量 / 挂牌量 / 去化周期 / 成交周期)> 价格指标**。(evidence: [T05-S001, T05-S010])

### 1. 国家统计局 — 70 城房价指数

- **Type**: dataset / 官方发布
- **URL**: https://www.stats.gov.cn/sj/zxfb/(最新发布) · https://data.stats.gov.cn/(国家数据库可下载)
- **机构**: 国家统计局(T05-S001 / T05-S002)
- **Cadence**: 月度(每月中旬发布上月「70 个大中城市商品住宅销售价格变动情况」)
- **Last activity**: 2026-04(滚动月度发布,长期稳定)
- **权威度**: 最高 — 这是中国房价唯一的官方口径,所有研究院 / 媒体 / 自媒体的「全国房价」叙事都回溯到它
- **流动性指标?**: 部分 — 主看价格(新房 / 二手房环比同比),配合统计局月度「房地产开发投资 / 商品房销售面积 / 销售额」可看全国成交量趋势,但**不到城市板块颗粒度**
- **One-liner**: 全国房价的官方锚 —— 看「方向和分化」(一线 / 二线 / 三线分组、环比连跌连涨月数),不看它做单城买卖决策
- **典型每期内容**: 70 城新建商品住宅与二手住宅价格环比 / 同比指数,按一线 / 二线 / 三线城市分组汇总;附文字解读。投资视角重点:二手房环比下跌城市数、连续下跌月数 —— 这是判断「下行还是企稳」的官方信号。
- **投入产出比**: high —— 月度必看,5 分钟扫完即知大势,是所有判断的起点(evidence: [T05-S001])
- **Paywall**: none
- **可信度**: high · **Decay risk**: low(机构级,统计口径多年稳定)
- **局限**: 指数经过抽样和环比链式计算,**与你所在小区的实际成交价可能差很大**;城市分化下「全国数据」对单城决策意义有限,必须下沉到城市 / 板块。

### 2. 中国人民银行 — LPR 与房贷利率政策

- **Type**: 官方发布 / 政策原文
- **URL**: https://www.pbc.gov.cn/(T05-S003)
- **机构**: 中国人民银行
- **Cadence**: LPR 每月 20 日左右报价;货币政策 / 房贷利率政策不定期
- **Last activity**: 2026-04(LPR 月度滚动)
- **权威度**: 最高 — 房贷成本、首付比例、利率下限的政策源头
- **流动性指标?**: 否 — 但 LPR 与利率政策直接决定杠杆成本和月供
- **One-liner**: 杠杆成本的政策源头 —— LPR 调降 / 首付与利率下限放松,是判断「政策松紧」最硬的一手信号
- **典型每期内容**: LPR 月度报价(1 年期 / 5 年期以上,房贷挂钩 5 年期以上);阶段性发布首套 / 二套房贷利率政策下限调整、存量房贷利率下调、公积金利率调整等。投资视角:LPR + 加减点 = 你的实际房贷利率,提前还贷决策要算清 LPR 走势。
- **投入产出比**: high —— LPR 公布日 + 房贷政策调整日必看,直接进月供和杠杆纪律的算账(evidence: [T05-S003])
- **Paywall**: none
- **可信度**: high · **Decay risk**: low
- **局限**: 央行给的是全国框架和下限,**各城市实际执行利率、首付比例由地方因城施策**,要回到地方住建 / 当地银行口径核实。

### 3. 住建部 — 行业定调与保交楼

- **Type**: 官方发布 / 政策原文
- **URL**: https://www.mohurd.gov.cn/(T05-S004)
- **机构**: 住房和城乡建设部
- **Cadence**: 不定期(重要政策 / 年度工作会议 / 保交楼进展)
- **Last activity**: 滚动(政策驱动)
- **权威度**: 最高 — 房地产行业管理与「保交楼」「白名单」「房地产发展新模式」的政策定调机构
- **流动性指标?**: 否 — 政策与风险信号源
- **One-liner**: 行业方向与烂尾风险的政策源头 —— 保交楼、融资白名单、现房销售试点等定调都从这里出
- **典型每期内容**: 全国住房城乡建设工作会议精神、保交楼 / 保交房进展、房企融资协调机制(白名单)、租购并举、城中村改造、房地产发展新模式等。投资视角:买预售房前看保交楼与白名单政策环境,是烂尾风险评估的一环。
- **投入产出比**: medium —— 不必日更跟,但重大定调(年末工作会议、保交楼新政)必看(evidence: [T05-S004])
- **Paywall**: none
- **可信度**: high · **Decay risk**: low
- **局限**: 部委定调偏宏观,**落到「你这个楼盘能不能按时交付」要查具体项目的网签 / 资金监管 / 白名单状态**,部委文件代替不了项目尽调。

### 4. 中国政府网 — 国常会 / 政治局会议房地产定调

- **Type**: 官方发布 / 政策原文
- **URL**: https://www.gov.cn/(T05-S005)
- **机构**: 中国政府网(国务院)
- **Cadence**: 不定期(国常会、政治局会议涉房表述)
- **Last activity**: 滚动(政策驱动)
- **权威度**: 最高 — 「房住不炒」「止跌回稳」等顶层表述的原文出处
- **流动性指标?**: 否 — 顶层政策风向标
- **One-liner**: 房地产顶层风向的原文出处 —— 政治局会议 / 国常会的涉房一句话,往往领先地方一轮政策松紧
- **典型每期内容**: 中央经济工作会议、政治局会议、国务院常务会议中涉及房地产的表述(如「促进房地产市场平稳健康发展」「止跌回稳」「构建房地产发展新模式」)。投资视角:**顶层表述的措辞变化是政策周期的领先指标**,比地方文件早。
- **投入产出比**: high —— 重大会议(每季度政治局会议、年末中央经济工作会议)涉房段落必读原文,不看自媒体转述(evidence: [T05-S005])
- **Paywall**: none
- **可信度**: high · **Decay risk**: low
- **局限**: 顶层表述抽象,需要结合地方落地文件才能转成「能不能买、首付几成」的具体动作 —— 一手定调 + 地方一手执行,两个都要。

### 5. 地方住建局 / 房管局 — 网签数据与限购限贷文件(以北京 / 深圳为样板)

- **Type**: 官方发布 / 政策原文 / 数据
- **URL**: https://www.bjjs.gov.cn/(北京住建委 T05-S006) · https://zjj.sz.gov.cn/(深圳住建局 T05-S007) · 其余城市同构:搜「{城市}住建局 / 房管局」官网
- **机构**: 各地住房和城乡建设主管部门
- **Cadence**: 网签 / 成交数据多为日 / 周 / 月更;限购限贷限售文件不定期
- **Last activity**: 滚动(各城市独立)
- **权威度**: 最高(本城市口径)— 网签备案数据是**比中介挂牌量更权威的成交量真相**,地方限购限贷限售原文也在这里
- **流动性指标?**: **是 — 这是城市级流动性的核心一手来源**:一手 / 二手网签套数与面积、可售房源、成交结构,部分城市公开二手房成交周期
- **One-liner**: 城市级流动性与政策的一手来源 —— 网签数据 = 真实成交量(中介挂牌价骗不了网签),限购限贷文件 = 政策原文
- **典型每期内容**: 当日 / 当月一手房与二手房网签套数及面积、商品房可售存量、地方版限购 / 限贷 / 限售 / 认房认贷 / 公积金 / 人才购房政策文件。投资视角:**网签量是判断城市流动性的硬指标**,挂牌量暴增 + 网签量低迷 = 流动性恶化。
- **投入产出比**: high —— 对你目标城市,网签数据建议周度跟,政策文件出台即看原文(evidence: [T05-S006, T05-S007])
- **Paywall**: none
- **可信度**: high · **Decay risk**: low-medium(机构稳定,但部分城市公开口径可能收窄 / 数据栏目改版)
- **局限**: 各城市公开程度差异大 —— 有的城市网签数据透明到日,有的只给月度笼统数;**挂牌量、成交周期这两个流动性指标官方往往不公开**,要靠贝壳研究院等民间机构补(见第二节)。

## 二、研究院

> 研究院是**官方数据的延伸和细化** —— 它们补官方不给的颗粒度(城市 / 板块二手房、挂牌量、去化周期、房企财务)。可信度高于媒体二手解读,但**研究院与中介 / 房企有商业关联**(贝壳研究院属贝壳找房,克而瑞服务房企),报告口径要带一点「卖方视角」的折扣看。bucket 标 `surrogate_primary`「机构数据」。(evidence: [T05-S010, T05-S012])

### 1. 贝壳研究院

- **Type**: 研究院 / dataset
- **URL**: https://research.beike.cn/(T05-S010)
- **机构**: 贝壳找房旗下研究院
- **Cadence**: 月度市场报告 + 不定期专题(KMI 指数、二手房景气、租赁)
- **Last activity**: 2026 年持续(月度滚动)
- **权威度**: 高(民间最强二手房数据)— 贝壳 / 链家掌握全国海量二手房真实带看与成交数据,**二手房流动性指标的最强民间口径**
- **流动性指标?**: **是 — 核心价值所在**:二手房成交量、带看量、KMI(贝壳二手房景气指数)、挂牌 / 成交价差、业主调价比例、成交周期 —— 这些官方基本不给
- **One-liner**: 二手房流动性的最强民间口径 —— 看带看量、成交周期、业主调价比例,判断「市场冷热」比看价格指数灵敏
- **典型每期内容**: 重点城市二手房成交量环比、带看指数、KMI 景气指数、新增挂牌与去化、业主预期(涨 / 跌 / 持平的调价占比)。投资视角:**带看量是成交量的领先指标,业主调价比例是价格的领先指标** —— 这是「流动性优先于价格」最实操的抓手。
- **投入产出比**: high —— 月度报告必看,尤其目标城市的二手房景气数据(evidence: [T05-S010])
- **Paywall**: 报告多数公开摘要免费;细分数据 / API 商用收费
- **可信度**: high(数据真实性高)· **Decay risk**: low-medium(机构级,但属贝壳商业实体,栏目与口径可能调整)
- **替代品**: 中指研究院 CREIS、克而瑞 CRIC(房企 / 一手视角更强);官方网签数据(权威但无挂牌 / 带看)
- **利益关联提示**: 贝壳是中介平台,「市场回暖」类结论天然有交易撮合的立场,**数据信、强结论打折**。

## 四、政策跟踪源

> 中国房产是**政策驱动多于市场驱动**,「读政策、留余量、不赌政策」是核心能力。铁律:**一手政策原文 > 自媒体二手解读**。本节列「专门用来追政策」的源 —— 与第一节官方源有重叠(政策原文本就在官方网站),这里强调「怎么组合着追」。(evidence: [T05-S005, T05-S006])

### 1. 地方住建局 / 房管局公告栏(政策原文层 — 最该盯的)

- **Type**: 政策原文
- **URL**: 各城市住建局 / 房管局官网「政策文件 / 通知公告」栏(样板:北京 https://www.bjjs.gov.cn/ T05-S006、深圳 https://zjj.sz.gov.cn/ T05-S007)
- **Cadence**: 不定期(因城施策,松紧政策随时出)
- **覆盖**: 限购 / 限贷 / 限售 / 认房认贷 / 公积金 / 人才购房 / 契税补贴 / 现房销售试点等**地方落地文件**
- **One-liner**: 政策落地的原文层 —— 你能不能买、首付几成、卖出要不要满几年,答案在地方住建局公告里,不在公众号解读里
- **典型内容**: 地方版购房资格、首付比例、贷款认定、限售年限、公积金额度与认定、人才 / 多孩家庭购房支持等。投资视角:**任何买卖决策前,先到目标城市住建局官网核对最新的限购限贷限售原文** —— 政策可以一夜调整。
- **投入产出比**: high —— 对目标城市必盯,出新政即看原文(evidence: [T05-S006, T05-S007])
- **可信度**: high · **Decay risk**: low-medium(各城市公告栏可能改版)
- **追法**: 把目标城市住建局公告栏加入定期检查;不要等自媒体「解读完」再行动,解读常有偏差和滞后。

### 2. 央行 LPR 与房贷利率政策(利率政策层)

- **Type**: 政策原文
- **URL**: https://www.pbc.gov.cn/(T05-S003)
- **Cadence**: LPR 月度(每月 20 日左右);房贷利率 / 首付政策不定期
- **覆盖**: LPR 报价、首套 / 二套利率下限、存量房贷利率调整、公积金贷款利率
- **One-liner**: 杠杆成本的政策层 —— LPR 和利率下限决定你的月供和提前还贷决策
- **典型内容**: LPR 月度报价、阶段性房贷政策(利率下限取消 / 调整、存量房贷下调、公积金利率调整)。投资视角:**LPR + 加减点 = 实际房贷利率**;政策放松周期里首付与利率下限的变化是「政策松」最硬的证据。
- **投入产出比**: high —— LPR 公布日 + 房贷新政日必看(evidence: [T05-S003])
- **可信度**: high · **Decay risk**: low
- **追法**: LPR 日历化(每月 20 日附近);配合各城市当地银行实际执行利率核实。

### 3. 中国政府网 — 顶层会议涉房表述(政策风向层)

- **Type**: 政策原文
- **URL**: https://www.gov.cn/(T05-S005)
- **Cadence**: 不定期(政治局会议每季度、中央经济工作会议年度、国常会不定期)
- **覆盖**: 政治局会议 / 中央经济工作会议 / 国常会的房地产定调措辞
- **One-liner**: 政策周期的风向层 —— 顶层措辞从「房住不炒」到「止跌回稳」的变化,领先地方一轮政策松紧
- **典型内容**: 顶层会议涉房一句话(定调、新模式、保交楼、止跌回稳等)。投资视角:**措辞变化是政策周期的领先指标**,比地方文件早,但抽象 —— 用它判方向,用地方文件判动作。
- **投入产出比**: high —— 重大会议涉房段落读原文(evidence: [T05-S005])
- **可信度**: high · **Decay risk**: low
- **追法**: 跟季度政治局会议 + 年末中央经济工作会议的房地产通稿原文。

### 4. 房产税试点文件 — 上海 / 重庆市政府(持有环节政策层)

- **Type**: 政策原文
- **URL**: 上海 https://www.shanghai.gov.cn/(T05-S008) · 重庆 https://www.cq.gov.cn/(T05-S009)
- **Cadence**: 不定期(试点文件 + 全国房产税立法进展通过中国政府网 / 全国人大跟）
- **覆盖**: 上海 / 重庆房产税试点征收办法、税率、免征面积、征收范围;全国房地产税立法进展
- **One-liner**: 持有环节成本的政策层 —— 房产税试点(沪渝)+ 全国立法节奏,决定长期持有成本测算要不要留余量
- **典型内容**: 上海 / 重庆房产税试点的征收对象、税率、免征条件;全国房地产税立法的官方表态。投资视角:**房产税是仍在演进的持有成本变量** —— 长期持有测算必须把「未来可能开征 / 扩围」作为留余量项,不能按「永远不收」算。
- **投入产出比**: medium —— 不必高频跟,有立法 / 试点新动向时看原文(evidence: [T05-S008, T05-S009])
- **可信度**: high · **Decay risk**: low(政策演进慢,但方向不确定性高)
- **追法**: 关注沪渝试点调整 + 全国人大立法规划中的房地产税进展;一手表态优先于自媒体「房产税要来了 / 不会来了」的情绪化判断。

## 五、自媒体样本(噪音极大 — 作情绪 / 现状样本甄别)

> **严正声明 — 这一节和上面四节的性质完全不同。**
>
> - 上面四节(官方 / 数据 / 研究院 / 严肃媒体)是**决策底座**;这一节是**情绪与现状的样本**,用来「了解市场上在传什么、散户在怕什么 / 贪什么」,**不作决策依据**。
> - **房产「大师」绝大多数是利益相关方**:卖课的、卖咨询的、和中介 / 开发商分佣的、带学员团购的。他们**只晒对不晒错** —— 喊对的截图天天发,喊错的删掉当没说过。
> - **公开喊单 = 立场**。任何「跟我买就赚」「现在抄底」「这个盘闭眼入」都应触发警报。收录这些人**只为呈现「自媒体派在说什么」这个现状切片**,不是背书,不是「另一种可信观点」。
> - **平台是黑名单**:国内房产自媒体主阵地是微信公众号和知乎,均为 Source Manifest 黑名单。**所以这一节不挂任何 URL 作 evidence,不引用其文章内容**;只做「派别 / 风格 / 利益结构」的样本描述。一手信号在黑名单平台 = 国内房产自媒体的结构性问题。
> - 甄别动作:**把「分析框架」和「喊单 / 带货 / 卖课」分开** —— 同一个人可能前半段讲了有用的框架(如人口 / 库存 / 流动性怎么看),后半段就开始荐盘卖课。取框架,弃荐盘。

### 1. 真叫卢俊(样本:相对专业的「行业观察」派)

- **Type**: 自媒体样本(公众号 / 视频)— 平台属黑名单,不引用文章
- **Author**: 卢俊团队(上海)
- **类型**: 行业观察 + 一线市场视角,相对专业 —— 在自媒体里属「框架成分较高」的一档
- **为什么作样本**: 代表「自媒体里相对克制、有行业观察视角」的一类。可用来感知一线城市市场情绪、行业动态的民间叙事。
- **怎么用**: 取它对「市场冷热、板块逻辑、行业变化」的观察当**情绪 / 现状参考**;**但它仍是房产内容生意**(团队化运营、有商业合作、做房产服务),涉及具体荐盘 / 带看 / 课程时一律按利益相关方处理。
- **甄别要点**: 「分析框架」部分可听(如怎么看城市分化、板块);「具体某盘值不值得买」「跟团购房」部分 = 商业立场,不采信。
- **可信度**: low-medium(作样本)· **Decay risk**: medium(个人 / 团队自媒体,内容方向随商业模式变)
- **强制标注**: 房产自媒体多为利益相关方,只晒对不晒错;此条仅作「自媒体派现状样本」,非决策依据。

### 2. 子木(子木聊房)(样本:高影响力的「判断 / 喊单」派)

- **Type**: 自媒体样本(公众号 / 视频)— 平台属黑名单,不引用文章
- **Author**: 子木
- **类型**: 影响力大,**偏判断 / 喊单风格** —— intake 已明确点名其「偏判断 / 喊单」
- **为什么作样本**: 代表「高影响力、强判断、偏喊单」的自媒体典型。它的走红和话术本身就是「散户被什么样的叙事吸引」的研究样本。
- **怎么用**: **不作任何买卖参考**。可用来观察「自媒体强判断叙事长什么样」「市场情绪被怎样调动」。看到「抄底」「闭眼入」「错过再等十年」这类话术时,正是 SKILL.md 要教用户警惕的对象。
- **甄别要点**: 强判断 + 时间紧迫感 + 「跟上车」叙事 = 典型喊单结构;**只晒对不晒错**的问题在这类账号尤其突出。
- **可信度**: low(作样本)· **Decay risk**: medium-high(喊单类账号生命周期与判断准确率 / 平台政策绑定)
- **强制标注**: 房产自媒体多为利益相关方,只晒对不晒错;此条作「喊单派现状 / 风险样本」收录,**明确不背书**。

### 3. 各地房产论坛 / 知乎房产话题 / 小红书房产(样本:散户情绪与一线信息的「噪音池」)

- **Type**: 社区样本(篱笆网 / 19 楼等地方论坛、知乎「买房 / 房地产」话题、小红书房产)— 知乎 / 小红书 / 公众号均黑名单,不引用内容
- **类型**: 散户讨论、看房经历、维权信息、情绪宣泄的混合池 —— **噪音占绝对多数**
- **为什么作样本**: 这是「市场情绪温度计」和「一线零散信息」的来源:某小区降价、某楼盘维权、某板块挂牌暴增、某政策传闻 —— 散户的真实感受和零散信号会先出现在这里。
- **怎么用**: **只作线索,不作结论,不作决策依据**。论坛 / 知乎 / 小红书的房产内容混杂大量中介营销、房托、个体焦虑、未经核实的传闻。**任何在这里看到的「数据 / 政策 / 价格」都必须回到官方 / 研究院核实**。
- **甄别要点**: (1) 区分「亲历者描述」(有一定参考价值,如某盘交付质量、某中介体验)和「判断 / 荐盘 / 喊涨喊跌」(噪音);(2) 警惕伪装成「素人分享」的中介营销和房托;(3) 维权 / 烂尾信息有现实参考价值但需交叉核实。
- **可信度**: low(作样本)· **Decay risk**: high(平台政策、话题热度、马甲号生态都不稳定)
- **强制标注**: 社区是情绪与零散线索的噪音池,中介营销与房托混杂其中;仅作样本与线索,**严禁作决策依据**。

> **三个样本的共同甄别框架(供 SKILL.md 直接用)**:
> 1. **看立场**:他靠什么赚钱?卖课 / 卖咨询 / 带团购 / 中介分佣 / 流量变现 —— 立场决定他不会告诉你「别买」。
> 2. **看「晒对不晒错」**:只有「我喊对了」的截图,没有「我喊错了」的复盘 = 不可信。要求看完整记录,看不到就当没有。
> 3. **分离框架与荐盘**:取「怎么分析(人口 / 库存 / 流动性 / 政策)」的方法论,**扔掉「买哪个盘 / 现在是不是底」的结论**。
> 4. **绝不让自媒体替代官方**:数据回到统计局 / 网签,政策回到原文,流动性回到研究院。自媒体只回答「市场上在传什么」,不回答「我该怎么做」。

### 冷僻 / 信号薄弱判定

- 官方 / 数据源信号:**强** —— gov 域名一手数据源充足(统计局 / 央行 / 住建部 / 政府网 / 多个地方住建局),9 个 URL 全部 `verified_primary`;研究院 4 个 `surrogate_primary`(机构数据)。官方层是本 track 最扎实的部分。
- 严肃媒体信号:**中强** —— 财新作为深度调查标杆质量高,5 个媒体源覆盖足够。
- 自媒体信号:**结构性偏薄(诚实标注)** —— 国内房产自媒体一手信号集中在微信公众号 / 知乎 / 小红书等**黑名单平台**,无法进 manifest 作 evidence,只能做派别 / 风格 / 利益结构的样本描述。这是本行业 sources 维度的**结构性局限**,Phase 2.8 诚实边界应标注:「自媒体一手信号在黑名单平台、结构性偏薄;且房产自媒体噪音 + 利益相关问题极重,本就不应作决策源 —— 此处的『薄』不影响决策质量,因为决策本就该靠官方 + 研究院 + 严肃媒体。」
- Newsletter / Podcast / Conference 维度:本行业以「机构数据 + 政策原文 + 严肃媒体」为信息源主形态,**没有西方意义上的高质量个人 newsletter / podcast 生态**(优质长内容主要是经济学家的书与公开演讲,已在 Track 04 canon)。这是行业特性,非调研缺漏 —— Phase 2 应说明本行业「信息源」节以官方 / 数据 / 研究院 / 媒体为主轴,而非订阅制个人内容。
- 综合:**官方 / 数据 / 研究院 / 严肃媒体维度信号充足,不触发冷僻协议**;自媒体维度按行业特性诚实标注「结构性偏薄且本不应作决策依据」。
