## Tier 1 — 核心必懂术语（所有房产投资人必懂）

### 1. 限购 — 购房资格限制

- **Type**: term + regulation
- **Tier**: tier-1
- **Definition (insider)**: 地方政府按户籍、社保 / 个税年限、名下房产套数对「能不能买、能买几套」设的硬门槛 —— 投资人第一步先查「我有没有资格」，没资格后面全白算。
- **Definition (outsider-friendly)**: 不是有钱就能买，城市规定了谁能买、能买几套。
- **来源**: 2010 年起一线城市率先实施，地方住建局发文，因城施策（evidence: [T06-S010, T06-S001]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「限购 = 全国统一规则」；内行知道限购是**一城一策、随时调整、甚至一区一策**，2023-2024 多城已大幅松绑或取消，必须查最新文件（evidence: [T06-S010]）。
- **关联术语**: 限贷、限售、认房认贷、落户
- **Decay risk**: high — 政策驱动，12 个月内常有调整。

### 2. 限贷 — 贷款资格 / 杠杆限制

- **Type**: term + regulation
- **Tier**: tier-1
- **Definition (insider)**: 对「能不能贷、首付几成、利率上浮多少」的限制 —— 和限购是两套独立门槛，限购管资格、限贷管杠杆。
- **Definition (outsider-friendly)**: 规定你买房能借多少钱、要先付几成。
- **来源**: 央行 + 地方银保监 + 地方政府三方叠加，认房认贷是其核心判定逻辑（evidence: [T06-S003, T06-S005]）。
- **常见误用 (outsider-tell)**: `semantic_tell` 外行把「限购」和「限贷」混为一谈；内行严格区分 —— 有资格（不限购）≠ 能贷到款（可能因征信 / 名下贷款记录被限贷）（evidence: [T06-S005]）。
- **关联术语**: 限购、认房认贷、首付比例、LPR
- **Decay risk**: high。

### 3. 限售 — 取得产权后的转让锁定期

- **Type**: term + regulation
- **Tier**: tier-1
- **Definition (insider)**: 买到手后规定年限内不能卖（常见 2-5 年）—— 直接锁死流动性，投资人测算退出时必须算进去。
- **Definition (outsider-friendly)**: 买了之后几年内不准卖。
- **来源**: 2017 年起多城推出，地方政府发文，目的是抑制短炒（evidence: [T06-S001]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行只关注「买」的成本，忽略限售对「卖」的锁定；内行把限售年限当作流动性折价的硬约束 —— 限售 5 年意味着这 5 年这套房不是「资产」是「负债占用」（evidence: [T06-S001]）。
- **关联术语**: 限购、限贷、流动性、去化周期
- **Decay risk**: high。

### 4. 认房认贷 — 二套房 / 首套房的判定口径

- **Type**: term + regulation
- **Tier**: tier-1
- **Definition (insider)**: 判定你是「首套」还是「二套」的规则 —— 「认房」看本地名下有无房，「认贷」看全国征信有无房贷记录。最严是「认房又认贷」，最松是「认房不认贷」。
- **Definition (outsider-friendly)**: 银行怎么算你是不是第一次买房，决定首付和利率。
- **来源**: 央行 / 住建部框架 + 地方执行，2023 年 9 月多城从「认房认贷」松绑为「认房不认贷」是标志性政策转向（evidence: [T06-S003, T06-S001]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「卖了旧房就算首套」；内行知道在「认贷」口径下，只要征信里有过房贷记录（哪怕已结清、房已卖），仍可能被算二套 —— 首付直接差 10-40 个百分点（evidence: [T06-S003]）。
- **关联术语**: 限贷、首付比例、征信
- **Decay risk**: high — 2023 起松绑频繁。

### 5. LPR — 贷款市场报价利率（Loan Prime Rate）

- **Type**: term + acronym + standard
- **Tier**: tier-1
- **Definition (insider)**: 房贷利率的定价锚 —— 实际房贷利率 = LPR + / − 基点（BP）。LPR 由央行授权全国银行间同业拆借中心每月 20 日公布。
- **Definition (outsider-friendly)**: 房贷利率跟着这个基准走，它降你月供降。
- **Etymology / 来源**: 2019 年 8 月央行 LPR 形成机制改革，取代原「贷款基准利率」；由 18 家报价行报价、去掉最高最低后算术平均（evidence: [T06-S003, T06-S012]）。
- **常见误用 (outsider-tell)**: `pronunciation_tell` 外行常念「L-P-R 三个字母」没问题，但 `semantic_tell` 外行以为「LPR 降了我月供马上降」；内行知道存量房贷利率**通常一年才重定价一次**（多数在每年 1 月 1 日或贷款发放日对应月），且实际利率还要看当初签的加减点（evidence: [T06-S003]）。
- **关联术语**: 房贷利率、商贷、提前还贷、重定价日
- **Decay risk**: medium — 机制稳定，但报价值每月变。

### 6. 首付比例 — 购房贷款的自有资金门槛

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 总价中必须用自有资金支付的比例，其余可贷款 —— 由限贷政策 + 首套 / 二套 + 商贷 / 公积金共同决定，是杠杆倍数的直接体现。
- **Definition (outsider-friendly)**: 买房自己先掏的那部分钱占总价的比例。
- **来源**: 央行 + 地方政策，2024 年多地首套首付降至 15%-20%（历史低位）（evidence: [T06-S003, T06-S005]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行把「首付低」当利好（「门槛低、好上车」）；内行知道首付越低杠杆越高，下行期房价跌 20% 就把 20% 首付的本金全部抹平甚至「资不抵债」—— 首付比例是风险刻度不是优惠力度（evidence: [T06-S005]）。
- **关联术语**: 限贷、杠杆、断供、首付来源
- **Decay risk**: high。

### 7. 公积金贷 / 组合贷 / 商贷 — 三种房贷资金来源

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 商贷 = 商业银行贷款（利率高、额度大、审批快）；公积金贷 = 用住房公积金账户贷款（利率显著低、额度有上限、审批慢）；组合贷 = 公积金额度用满后剩余部分用商贷补，两者叠加。
- **Definition (outsider-friendly)**: 三种借钱买房的方式，公积金最便宜但能借的少。
- **来源**: 公积金由地方公积金管理中心管理，商贷归商业银行（evidence: [T06-S003]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「能用公积金就全用公积金」；内行知道公积金贷额度普遍偏低、放款慢（卖家可能不愿等），组合贷流程更复杂，二手房交易里卖家常因此压价或拒贷款方式 —— 资金来源会影响议价（evidence: [T06-S003]）。
- **关联术语**: LPR、首付比例、网签、提前还贷
- **Decay risk**: low — 制度稳定。

### 8. 提前还贷 — 提前偿还房贷本金

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 在贷款期内主动归还部分或全部本金 —— 2022-2024 因理财收益普遍低于房贷利率，出现「提前还贷潮」，部分银行甚至限号排队。
- **Definition (outsider-friendly)**: 手上有钱了，提前把房贷还掉一部分。
- **来源**: 各银行自行规定违约金 / 等待期规则（evidence: [T06-S003]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行纠结「还款额减少」还是「年限缩短」；内行先看的是**机会成本** —— 提前还贷等于「锁定一笔收益率 = 房贷利率的无风险投资」，只有当其他低风险收益低于房贷利率、且不影响应急流动性时才划算（evidence: [T06-S003]）。
- **关联术语**: LPR、房贷利率、流动性、断供
- **Decay risk**: low。

### 9. 网签 — 网上签约备案

- **Type**: term + standard
- **Tier**: tier-1
- **Definition (insider)**: 买卖双方在房管局网上系统签订合同并备案的环节 —— 网签成功该房即被锁定，是「一房二卖」的防火墙，也是官方成交数据的来源。
- **Definition (outsider-friendly)**: 在政府系统里正式登记这套房卖给你了。
- **来源**: 各地房管局 / 住建局系统，商品房销售管理制度的一环（evidence: [T06-S002]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「签了买卖合同 = 房子是我的了」；内行知道顺序是 网签 → 资金监管 → 贷款审批 → 过户，**没过户前产权都不是你的**，网签只是锁定不是确权（evidence: [T06-S002]）。
- **关联术语**: 备案、资金监管、过户、备案价
- **Decay risk**: low。

### 10. 备案 / 备案价 — 价格备案与限价

- **Type**: term + regulation
- **Tier**: tier-1
- **Definition (insider)**: 新房售价须向住建部门备案，实际网签价不得超备案价（一手房限价工具）；二手房则有「指导价」机制（部分城市）压低网签价进而压低可贷额度。
- **Definition (outsider-friendly)**: 政府给房价设了上限，不能随便标价卖。
- **来源**: 住建部门，商品房预售 / 销售管理制度（evidence: [T06-S002]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行看到「备案价低于市场价」以为捡漏；内行知道二手房指导价会**压低银行评估价 → 压低可贷款额度 → 实际首付要补更多现金**，备案 / 指导价低反而抬高了上车的现金门槛（evidence: [T06-S002]）。
- **关联术语**: 网签、限价、阴阳合同（红线）、资金监管
- **Decay risk**: medium。

### 11. 资金监管 — 交易资金第三方监管

- **Type**: term + standard
- **Tier**: tier-1
- **Definition (insider)**: 买方房款先打入房管局指定的监管账户，过户完成后才划给卖方 —— 防止「钱房两失」的核心制度，二手房交易绝不走私下转账。
- **Definition (outsider-friendly)**: 买房的钱先冻在政府监管账户里，过完户卖家才拿得到。
- **来源**: 各地房管局 / 住建局存量房交易资金监管制度（evidence: [T06-S002]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行为「省手续费 / 图快」答应卖方走私下转账或「半监管」；内行把「全程资金监管」当不可妥协的硬规矩 —— 绕开监管 = 把几百万本金敞口暴露给卖方违约和房屋查封风险（evidence: [T06-S002]）。
- **关联术语**: 网签、过户、阴阳合同（红线）、中介费
- **Decay risk**: low。

### 12. 过户 — 不动产权属转移登记

- **Type**: term + standard
- **Tier**: tier-1
- **Definition (insider)**: 在不动产登记中心办理产权人变更，拿到新的不动产权证 —— **过户完成那一刻产权才真正归你**，是交易的法律终点。
- **Definition (outsider-friendly)**: 去政府把房本上的名字改成你，房子才真正是你的。
- **来源**: 不动产统一登记制度，民法典 / 原物权法（evidence: [T06-S008]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为交了钱、拿了钥匙就完成了；内行知道**未过户前一切都可逆**（卖方可被查封、可反悔、可一房二卖），交钥匙、交定金、网签都不等于确权（evidence: [T06-S008]）。
- **关联术语**: 网签、资金监管、不动产登记、产权核查
- **Decay risk**: low。

### 13. 契税 — 购房法定税种

- **Type**: term + regulation
- **Tier**: tier-1
- **Definition (insider)**: 买方按成交价（或评估价）一定比例缴纳的税，首套 / 二套、面积大小适用不同税率（普遍 1%-3%）—— 买方必交，是交易成本的固定项。
- **Definition (outsider-friendly)**: 买房时要交给国家的一笔税。
- **来源**: 《契税法》（2021 年 9 月施行），税务总局执行（evidence: [T06-S009]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行常把「契税、增值税、个税、中介费」笼统说成「税费」一笔糊涂账；内行会逐项拆 —— 谁交、按什么基数、首套二套差多少，二手房里增值税 / 个税常被卖方「转嫁」给买方实际承担，谈判前必须算清（evidence: [T06-S009]）。
- **关联术语**: 增值税、个税、中介费、满五唯一
- **Decay risk**: medium。

### 14. 增值税 / 个税 — 二手房交易税种（卖方税，常被转嫁）

- **Type**: term + regulation
- **Tier**: tier-1
- **Definition (insider)**: 增值税（含附加，约 5%+）+ 个人所得税（差额 20% 或全额 1%）法定由卖方承担，但二手房市场惯例「实收价」—— 卖方报净到手价，税费由买方实际买单。
- **Definition (outsider-friendly)**: 卖二手房本该卖家交的税，市场上常变成买家替交。
- **来源**: 增值税 / 个税相关法规，税务总局；「满五唯一」可免个税、「满二」可免增值税（evidence: [T06-S009]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行看挂牌价高低；内行算的是「到手价 + 全部税费 + 中介费」的总成本，且会确认房子「满几年、是否唯一」—— 一套不满二的房，税费可能比满五唯一的房高出总价 6%-8%（evidence: [T06-S009]）。
- **关联术语**: 契税、满五唯一、满二、中介费
- **Decay risk**: medium。

### 15. 满五唯一 / 满二 — 二手房税费优惠资格

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 「满五唯一」= 产证满 5 年 + 卖方家庭名下唯一住房，可免个税；「满二」= 产证满 2 年，可免增值税。是二手房税费高低的分水岭。
- **Definition (outsider-friendly)**: 房子持有够年限、又是卖家唯一的房，交易时税少很多。
- **来源**: 个税 / 增值税优惠政策，税务总局（evidence: [T06-S009]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「满五唯一」是房子的固有属性；内行知道「唯一」是针对**卖方家庭当下的持有状态**，会变 —— 卖方再买一套就不唯一了，且「满五」从产证 / 契税票时间算，必须验原始凭证（evidence: [T06-S009]）。
- **关联术语**: 增值税、个税、契税、房龄
- **Decay risk**: medium。

### 16. 70 年产权 / 40 年产权（商住）— 土地使用权年限

- **Type**: term + regulation
- **Tier**: tier-1
- **Definition (insider)**: 中国土地国有，房屋买的是土地「使用权」：住宅 70 年、商业 / 综合用地 40 年（商住公寓）、工业 50 年。年限从开发商**拿地**起算，不是从你买房起算。
- **Definition (outsider-friendly)**: 房子下面的地是租国家的，住宅租 70 年，商住公寓只租 40 年。
- **来源**: 民法典 / 原物权法关于建设用地使用权（evidence: [T06-S008]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「70 年从我买房算起」；内行知道剩余年限 = 70 −（你买房时距开发商拿地的年数），一套二手房可能只剩 50 多年。且 40 年商住公寓还有水电商用、不能落户、税费重、转手难的叠加劣势（evidence: [T06-S008]）。
- **关联术语**: 剩余产权年限、商住公寓、不动产登记、房龄
- **Decay risk**: low — 法律层面稳定（住宅到期续期原则已立法，细则待定）。

### 17. 预售 / 期房 / 现房 — 一手房的交付状态

- **Type**: term + standard
- **Tier**: tier-1
- **Definition (insider)**: 预售 / 期房 = 房子还没建好就先卖、买家先付全款 / 月供等交付；现房 = 已竣工验收、可即买即交付。中国新房以预售为主。
- **Definition (outsider-friendly)**: 期房是买「还没盖好的房」，现房是买「已经盖好的房」。
- **来源**: 商品房预售制度（住建部门），需开发商取得《商品房预售许可证》（evidence: [T06-S002]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行默认「买预售房 = 一定能交房」；内行知道预售制把建设风险转嫁给购房者，2021 年后开发商爆雷导致大量期房**烂尾**，买期房必须额外核查开发商资金链、预售资金监管、保交楼进度（evidence: [T06-S002, T06-S013]）。
- **关联术语**: 烂尾、保交楼、资金监管、开发商爆雷
- **Decay risk**: medium — 现房销售试点在扩大。

### 18. 烂尾 / 保交楼 — 期房交付失败与政府托底

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 烂尾 = 开发商资金链断裂、楼盘停工无法交付；保交楼 = 2022 年起中央定调、地方政府主导推动停工项目复工交付的政策行动 —— 是 2021 年后中国房产最尖锐的现实风险。
- **Definition (outsider-friendly)**: 房子盖一半开发商没钱了；保交楼是政府想办法把它盖完交给业主。
- **来源**: 2022 年中央政治局会议首提「保交楼」，住建部门 + 地方政府执行（evidence: [T06-S002, T06-S013]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「保交楼 = 政府兜底，买期房没风险了」；内行知道保交楼是「尽力推动」不是「刚性兑付」，进度慢、且烂尾期间业主仍要还房贷 —— 保交楼缓释风险但远不消除风险（evidence: [T06-S013]）。
- **关联术语**: 预售、期房、开发商爆雷、资金监管
- **Decay risk**: medium。

### 19. 法拍房 — 司法拍卖房产

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 因债务纠纷被法院强制拍卖的房产，通过阿里 / 京东司法拍卖平台竞拍 —— 价格可能低于市场，但尽调难度远高于普通二手房。
- **Definition (outsider-friendly)**: 法院把欠债人的房子拍卖掉，普通人也能去竞拍。
- **来源**: 司法拍卖制度，法院主导（evidence: [T06-S011]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行只看「比市场价便宜」；内行知道法拍房有腾退（原住户 / 租户赖着不走）、欠费（物业 / 水电 / 大额税费）、户口（占着学位无法迁出）、限购占资格等隐藏风险，「便宜」常被这些坑吃掉甚至倒亏（evidence: [T06-S011]）。
- **关联术语**: 二手房、产权核查、腾退、限购
- **Decay risk**: low。

### 20. 小产权房 — 无合法产权的房屋

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 建在集体土地上、未缴土地出让金、**没有国家认可的不动产权证**的房屋 —— 不受法律保护，不能办按揭、不能正常过户。
- **Definition (outsider-friendly)**: 没有正规房本的房子，法律上不算你的。
- **来源**: 国家多次明确小产权房不予确权登记（evidence: [T06-S008]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行被「便宜一半」「村委会盖章」吸引；内行直接一票否决 —— 小产权房不是「打折的商品房」，是法律上不被承认的资产，遇拆迁 / 纠纷可能血本无归。对投资人这是绝对红线（evidence: [T06-S008]）。
- **关联术语**: 不动产登记、产权核查、过户、商住公寓
- **Decay risk**: low。

### 21. 租金回报率 — 年租金 / 房价

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 年租金收入 ÷ 房屋总价（或投入成本），衡量房产作为「收租资产」的收益率 —— 中国一二线住宅普遍仅 1.5%-2.5%，低于房贷利率和无风险理财。
- **Definition (outsider-friendly)**: 把房子租出去，一年收的租金占房价的百分比。
- **来源**: 行业通用估值指标（evidence: [T06-S011]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行只算「租金 ÷ 房价」毛回报；内行算**净回报** —— 要扣物业费、维修、空置期、税费、资金占用成本，且和房贷利率对比：当租金回报率（约 2%）低于房贷利率（约 3%-4%）时，**租金根本覆盖不了月供利息**，靠涨价才不亏（evidence: [T06-S011]）。
- **关联术语**: 持有成本、空置率、LPR、流动性
- **Decay risk**: low。

### 22. 城市能级（一线 / 新一线 / 三四线）— 城市分层

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 按经济体量、人口、产业、行政地位对城市分层：一线（北上广深）、新一线 / 二线、三四线 —— 2021 年后不同能级城市房价走势分化甚至相反，「买对城市」是第一性问题。
- **Definition (outsider-friendly)**: 城市按大小和发展程度分等级。
- **来源**: 非官方但业内通用分层（部分榜单来自媒体机构），与统计局人口 / 经济数据相关（evidence: [T06-S004, T06-S011]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行谈论「房价涨没涨」时默认有「全国房价」；内行知道**没有「全国房价」这回事** —— 同一时间一线核心区在涨、三四线在跌、人口流出城市在阴跌，任何笼统判断都是错的，必须落到具体城市具体板块（evidence: [T06-S004]）。
- **关联术语**: 人口流入流出、板块、地段、土地财政
- **Decay risk**: low（分层稳定，但具体城市走势变化快）。

### 23. 房住不炒 — 「房子是用来住的、不是用来炒的」

- **Type**: term + regulation（政策定调）
- **Tier**: tier-1
- **Definition (insider)**: 2016 年中央经济工作会议提出的房地产长期定位，是限购 / 限贷 / 限售 / 房产税试点等所有调控工具的总纲 —— 意味着投机性炒房有真实、长期的政策风险。
- **Definition (outsider-friendly)**: 国家定的大方向：买房主要是为了住，不是拿来炒来炒去赚差价。
- **来源**: 2016 年中央经济工作会议，多次写入政府工作报告（evidence: [T06-S001]）。
- **常见误用 (outsider-tell)**: `register_tell` 外行 / 销售把它当一句可以无视的「口号」；内行把它当**政策底层操作系统** —— 它解释了为什么限购限贷会反复出现、为什么不能「赌政策一定全面放开」。把房住不炒当空话的人，会在政策急转时被打懵（evidence: [T06-S001]）。
- **关联术语**: 因城施策、限购、限贷、房产税
- **Decay risk**: low — 长期定调。

### 24. 因城施策 — 调控权下放地方

- **Type**: term + regulation（政策机制）
- **Tier**: tier-1
- **Definition (insider)**: 中央定总基调（房住不炒），具体松紧由各城市根据自身情况自行调整 —— 这就是为什么限购限贷「一城一策」、政策碎片化、必须查到「你那个城市那个区」的最新文件。
- **Definition (outsider-friendly)**: 每个城市可以根据自己情况定自己的买房规则。
- **来源**: 政策长期表述，写入多份中央文件（evidence: [T06-S001]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行看到「某市放松限购」就以为全国都放松了；内行知道因城施策意味着政策**不可跨城套用**，A 市的经验在 B 市可能完全失效，连本城不同区都可能不同（evidence: [T06-S001]）。
- **关联术语**: 房住不炒、限购、限贷、城市能级
- **Decay risk**: low（机制稳定）。

### 25. 去化周期 / 成交周期 — 流动性核心指标

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 去化周期 = 当前在售库存 ÷ 月均成交量（多少个月能卖完，新房口径）；成交周期 = 一套二手房从挂牌到成交的平均天数 —— 这两个数字直接刻画「房子好不好卖」，2024-2025 普遍拉长。
- **Definition (outsider-friendly)**: 衡量房子卖得快不快、市场上房子多久能消化完。
- **来源**: 行业通用指标，部分数据见研究机构报告，官方成交量见房管局网签数据（evidence: [T06-S004, T06-S002]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行只盯「房价涨跌」；内行先看流动性 —— 在成交周期拉长、挂牌量暴增的市场，「账面价格」是虚的，真正能成交的价格往往低于挂牌价 10%-20%，**流动性指标比价格指标更早预警**（evidence: [T06-S004]）。
- **关联术语**: 挂牌量、成交量、流动性、止损
- **Decay risk**: low（指标定义稳定）。

### 26. 挂牌量 / 成交量 — 二手房市场冷热的对照指标

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 挂牌量 = 市场上正在出售的二手房套数；成交量 = 实际完成网签的套数。挂牌量飙升 + 成交量低迷 = 卖方多买方少 = 价格承压。
- **Definition (outsider-friendly)**: 在卖的房子有多少、真正卖出去的有多少。
- **来源**: 成交量以房管局网签数据为权威口径，挂牌量见中介平台（evidence: [T06-S002]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行用中介平台的「挂牌价」推断市场；内行知道挂牌价是「卖方想要的价」、网签价才是「真实成交的价」，且要看挂牌量 / 成交量的**比值和趋势** —— 挂牌量三年翻几倍是 2023-2025 多城的真实图景（evidence: [T06-S002]）。
- **关联术语**: 去化周期、成交周期、网签、流动性
- **Decay risk**: low。

### 27. 杠杆 — 用借入资金放大购房规模

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 用首付撬动总价（首付 3 成 = 约 3.3 倍杠杆）—— 杠杆**同时**放大收益和亏损，下行期是毁灭性的：房价跌幅一旦超过首付比例，本金归零并「资不抵债」。
- **Definition (outsider-friendly)**: 自己出一部分、借一大部分来买，借得越多杠杆越高。
- **来源**: 金融通用概念，房贷是中国家庭最主要的杠杆工具（evidence: [T06-S005]）。
- **常见误用 (outsider-tell)**: `semantic_tell` 外行 / 大师话术只说「杠杆放大收益」；内行知道这是**半句话** —— 必须同等强调「杠杆放大亏损 + 现金流断裂 + 断供」，下行期高杠杆让大量人深套甚至断供，杠杆纪律是硬规矩不是可选项（evidence: [T06-S005]）。
- **关联术语**: 首付比例、断供、流动性、首付贷（红线）
- **Decay risk**: low。

### 28. 断供 — 无力偿还房贷而停止还款

- **Type**: term
- **Tier**: tier-1
- **Definition (insider)**: 借款人现金流断裂、连续多月不还月供 —— 后果是征信受损、房子被银行起诉拍卖，且**拍卖价不够还贷时仍要补差额**（房子没了、债还在）。
- **Definition (outsider-friendly)**: 还不起房贷了，停止还款。
- **来源**: 民法典 / 借款合同，银行处置流程（evidence: [T06-S008, T06-S005]）。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「实在还不起就把房子还给银行、一了百了」；内行知道断供后房子法拍，**若拍卖款 + 已还款 < 贷款本息，差额仍须偿还**，同时征信黑名单数年 —— 断供不是「退出」是「灾难性结局」（evidence: [T06-S008]）。
- **关联术语**: 杠杆、首付比例、流动性、止损
- **Decay risk**: low。

## Tier 2 — 周边熟知术语（资深投资人熟知）

### 29. 房贷利率 — 实际房贷执行利率

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: = LPR ± 基点（BP），首套常低于二套；存量房贷一年重定价一次。
- **常见误用 (outsider-tell)**: `usage_tell` 外行把「利率高低」当唯一变量；内行同时看利率 + 期限 + 还款方式（等额本息 vs 等额本金）的总利息差（evidence: [T06-S003]）。
- **关联术语**: LPR、商贷、提前还贷
- **Decay risk**: medium。

### 30. 中介费 — 二手房居间服务费

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 二手房交易中支付给中介的佣金，普遍 1%-3%（贝壳 / 链家曾约 2.5%-3%，2023 后部分城市下调），是交易总成本的固定大项。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为中介费「可有可无 / 一定能砍掉」；内行把它算进总成本，并知道砍中介费的代价可能是失去资金监管、产权核查协助等服务保障（evidence: [T06-S011]）。
- **关联术语**: 资金监管、契税、网签
- **Decay risk**: medium。

### 31. 房龄 — 房屋建成年数

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 从房屋竣工年份算起的年数 —— 影响贷款年限（房龄 + 贷款年限常有上限）、剩余产权年限、维修成本、转手难度。
- **常见误用 (outsider-tell)**: `usage_tell` 外行只看「装修新旧」；内行知道房龄大会导致「房龄 + 贷款年限 ≤ 银行上限」从而压缩可贷年限、抬高月供，老房还有管线老化、电梯 / 加装难题（evidence: [T06-S011]）。
- **关联术语**: 剩余产权年限、70 年产权、房贷利率
- **Decay risk**: low。

### 32. 剩余产权年限 — 土地使用权剩余年数

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 70（或 40）年 − 自开发商拿地起已过的年数；二手房买的是「剩下的年限」。
- **常见误用 (outsider-tell)**: `usage_tell` 外行忽略它，只看房龄；内行查土地证 / 不动产证上的「土地使用起止日期」，剩余年限短的房（尤其 40 年商住二手）流动性和估值都打折（evidence: [T06-S008]）。
- **关联术语**: 70 年产权、房龄、商住公寓
- **Decay risk**: low。

### 33. 容积率 — 总建筑面积 / 用地面积

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 衡量小区开发强度，数字越低越「低密」（别墅 / 洋房），越高越「高密」（高层 / 塔楼）—— 影响居住舒适度和小区档次。
- **常见误用 (outsider-tell)**: `usage_tell` 外行不知道这个词；内行用它快速判断小区品质 —— 高容积率 + 塔楼可能意味着采光差、公摊大、人多电梯挤（evidence: [T06-S011]）。
- **关联术语**: 得房率、公摊、地段
- **Decay risk**: low。

### 34. 得房率 / 公摊 — 套内面积占比与公共分摊面积

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 公摊 = 电梯、楼道、设备间等分摊到每户的面积；得房率 = 套内面积 ÷ 建筑面积。建筑面积 100㎡、得房率 75%，实际套内只有 75㎡。
- **常见误用 (outsider-tell)**: `usage_tell` 外行只比「建筑面积单价」；内行换算成「套内单价」再比 —— 两套建面同价的房，得房率差 10% 等于实际多付了 10% 的钱（evidence: [T06-S011]）。
- **关联术语**: 容积率、户型、备案价
- **Decay risk**: medium（部分地区在推「按套内面积计价」）。

### 35. 学区房 — 对应优质学校学位的住房

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 因绑定优质中小学入学资格而产生显著溢价的房产 —— 溢价完全依赖学区政策，政策一变（多校划片 / 学位锁定 / 公民同招）溢价可瞬间蒸发。
- **常见误用 (outsider-tell)**: `usage_tell` 外行把学区溢价当「永久的硬通货」；内行知道学区房是「政策资产」不是「地段资产」，多校划片、六年一学位、教师轮岗等政策已让多地学区房大幅回调 —— 买学区房 = 买一份政策风险敞口（evidence: [T06-S010]）。
- **关联术语**: 落户、城市能级、地段
- **Decay risk**: high — 学区政策频繁调整。

### 36. 落户 — 取得城市户籍

- **Type**: term + regulation
- **Tier**: tier-2
- **Definition (insider)**: 通过购房 / 积分 / 人才引进等渠道取得城市户口 —— 部分城市购房可落户，户口又关联购房资格、子女入学，是「人房政策」的连接点。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「买房就能落户」全国通用；内行知道购房落户政策一城一策、且在收紧 / 放松间反复，是吸引人口的工具不是承诺（evidence: [T06-S010]）。
- **关联术语**: 限购、学区房、人口流入流出
- **Decay risk**: high。

### 37. 持有成本 — 持有房产期间的全部支出

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 物业费 + 维修 + 空置损失 + 资金占用成本（机会成本）+ 潜在房产税 —— 投资测算里和「租金收入」对冲的那一侧。
- **常见误用 (outsider-tell)**: `usage_tell` 外行只算「房价涨没涨」；内行做「总持有成本 vs 总收益 vs 流动性」的完整账，持有 5 年的隐性成本可能吃掉名义涨幅的一大块（evidence: [T06-S011]）。
- **关联术语**: 租金回报率、空置率、房产税
- **Decay risk**: low。

### 38. 空置率 — 房屋闲置未出租的比例

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 一年中房屋空着收不到租金的时间占比 —— 直接侵蚀实际租金回报率，冷门地段 / 户型空置期可长达数月。
- **常见误用 (outsider-tell)**: `usage_tell` 外行用「满租」假设算回报率；内行按现实空置期打折，并把「找租客的难度」纳入选房标准（evidence: [T06-S011]）。
- **关联术语**: 租金回报率、持有成本、流动性
- **Decay risk**: low。

### 39. 板块 / 地段 — 城市内部的区域价值单元

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 板块 = 城市内有相对独立产业 / 配套 / 规划的片区；地段 = 具体位置的综合价值（交通、配套、环境）。同一城市不同板块走势可天差地别。
- **常见误用 (outsider-tell)**: `usage_tell` 外行说「我看好这个城市」；内行追问「哪个板块、什么地段」—— 「买对城市」之后还要「买对板块」，城市内分化和城市间分化一样大（evidence: [T06-S004]）。
- **关联术语**: 城市能级、人口流入流出、规划
- **Decay risk**: low。

### 40. 人口流入 / 流出 — 城市人口净变动

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 衡量一个城市「有没有持续的真实住房需求」的根本指标 —— 任泽平框架「长期看人口」即指此。人口净流出的城市，房产需求端是塌的。
- **常见误用 (outsider-tell)**: `usage_tell` 外行看「这个城市要建地铁 / 有新规划」就看多；内行先看常住人口是不是真在增长 —— 规划是「供给侧故事」，人口流入才是「需求侧事实」（evidence: [T06-S004]）。
- **关联术语**: 城市能级、板块、土地财政
- **Decay risk**: low。

### 41. 土地财政 / 招拍挂 — 地方政府卖地收入与土地出让方式

- **Type**: term + regulation
- **Tier**: tier-2
- **Definition (insider)**: 土地财政 = 地方政府依赖出让国有土地使用权获取财政收入的模式；招拍挂 = 招标 / 拍卖 / 挂牌出让土地的法定方式。2021 年后土地出让大幅萎缩，土地财政转型是楼市深层背景。
- **常见误用 (outsider-tell)**: `usage_tell` 外行不关心地方财政；内行知道土地财政承压会反过来影响一个城市基建、配套兑现能力和救市意愿 —— 黄奇帆等对此有体系性论述（evidence: [T06-S001]）。
- **关联术语**: 城市能级、房住不炒、因城施策
- **Decay risk**: medium。

### 42. 不动产登记 — 不动产统一登记制度

- **Type**: term + standard
- **Tier**: tier-2
- **Definition (insider)**: 2015 年起全国统一的不动产权属登记制度，由不动产登记中心办理，发《不动产权证书》—— 产权核查（查抵押 / 查封 / 产权人）的官方入口。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「卖方拿出房本就行」；内行一定去不动产登记中心**查档**，确认有无抵押、查封、共有人、产权性质 —— 房本在手不代表房子能干净过户（evidence: [T06-S008]）。
- **关联术语**: 过户、产权核查、小产权房
- **Decay risk**: low。

### 43. 商住公寓 — 商业 / 综合用地上的类住宅

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 建在商办用地上、用作居住的公寓，40 年产权 —— 商水商电、首付高利率高、税费重、多数不能落户和上学、转手极难。
- **常见误用 (outsider-tell)**: `usage_tell` 外行被「不限购、总价低、地段好」吸引；内行知道商住公寓是「劝退型产品」，持有成本高、退出几乎无接盘 —— 总价低背后是流动性几乎归零（evidence: [T06-S008]）。
- **关联术语**: 40 年产权、剩余产权年限、流动性
- **Decay risk**: low。

### 44. 置换 — 卖一买一的换房操作

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 卖掉现有房产、再买入另一套（常为改善居住）—— 下行期「卖旧」难、周期长，置换链条极易卡住。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「先买新的、慢慢卖旧的」很轻松；内行知道下行期「卖旧」是最大堵点，置换要先确保旧房能在合理价位卖出、并算好两套房的过桥资金和时间差（evidence: [T06-S004]）。
- **关联术语**: 流动性、成交周期、认房认贷
- **Decay risk**: low。

### 45. 接盘 / 抄底 / 止损 — 投资者的几种入 / 出场心态词

- **Type**: term（含口语 register）
- **Tier**: tier-2
- **Definition (insider)**: 接盘 = 从前手买入（常含「高位接最后一棒」的贬义）；抄底 = 试图在最低点买入；止损 = 承担亏损卖出以避免更大损失。
- **常见误用 (outsider-tell)**: `register_tell` 外行 / 大师话术爱说「跟我抄底」「现在是底部」；内行知道**底部只能事后确认**，「抄底」常变「接盘」，且把「止损」当不可启齿的失败 —— 成熟投资人把止损当纪律工具，不是耻辱（evidence: [T06-S004]）。
- **关联术语**: 流动性、断供、杠杆
- **Decay risk**: low。

### 46. 开发商爆雷 — 房企债务违约 / 资金链断裂

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 房企无法偿还到期债务、项目停工 —— 2021 年恒大、2023 年碧桂园等头部房企相继出险，是期房烂尾的直接源头。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「大开发商 = 安全」；2021 年后内行知道**头部房企照样爆雷**，买期房要查的是「这个项目的预售资金监管和工程进度」，不是「开发商名气大不大」（evidence: [T06-S013]）。
- **关联术语**: 烂尾、保交楼、预售、资金监管
- **Decay risk**: medium。

### 47. 写字楼 / 商铺 — 商业地产物业类型

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 写字楼 = 办公用房；商铺 = 商业经营用房。40-50 年产权，「一铺养三代」是旧叙事 —— 电商冲击 + 供应过剩下，多地商铺写字楼空置率高、租金 / 价格双跌。
- **常见误用 (outsider-tell)**: `usage_tell` 外行还信「商铺收租稳」的老话；内行知道商铺写字楼的风险结构和住宅完全不同 —— 高空置、招租难、流动性极差，「一铺养三代」在很多城市已变「一铺套三代」（evidence: [T06-S011]）。
- **关联术语**: 商住公寓、租金回报率、空置率
- **Decay risk**: low。

### 48. 重定价日 — 存量房贷利率调整的生效日

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 存量浮动利率房贷按 LPR 重新计算利率的日期，每年一次，常见为「每年 1 月 1 日」或「贷款发放日对应月」—— 这天之前 LPR 怎么变，月供都不动。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「LPR 一降月供马上降」；内行知道要等到自己的重定价日才生效，中间的时间差是常识（evidence: [T06-S003]）。
- **关联术语**: LPR、房贷利率、提前还贷
- **Decay risk**: low。

### 49. 旅居 / 养老地产 — 文旅康养类房产

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 以度假 / 养老为卖点、多位于风景区或远郊的房产 —— 自住效用可能有，但作为投资品流动性极差、配套兑现常落空。
- **常见误用 (outsider-tell)**: `usage_tell` 外行被「面朝大海 / 候鸟养老 + 投资双丰收」话术打动；内行知道这类房几乎只有「使用价值」没有「投资价值」，远郊文旅盘是接盘最难的品类之一（evidence: [T06-S011]）。
- **关联术语**: 流动性、商住公寓、空置率
- **Decay risk**: low。

### 50. 评估价 — 银行 / 评估机构认定的房屋价值

- **Type**: term
- **Tier**: tier-2
- **Definition (insider)**: 银行放贷时认定的房屋价值，可贷额度 = 评估价 × 贷款成数 —— 评估价常低于成交价，差额要买方用现金补足首付。
- **常见误用 (outsider-tell)**: `usage_tell` 外行以为「成交价 × 7 成 = 我能贷的钱」；内行知道是「**评估价** × 成数」，评估价被压低（指导价城市尤甚）时实际首付远高于名义首付（evidence: [T06-S002]）。
- **关联术语**: 备案价、首付比例、网签
- **Decay risk**: medium。

## 灰色 / 红线操作（违规甚至违法 — 只标注后果，不教做法）

> **本节定位**：以下操作部分「技术流大师」公开宣讲甚至「包教」。本 skill 的立场是 **明确标注其违规 / 违法性质 + 讲清后果，绝不教做法、绝不提供操作路径**。识别它们是为了让投资人**避开**，以及在听到「大师包教这些」时立刻判定对方不可信。

### R1. 经营贷置换房贷 —— 红线

- **是什么（仅识别用）**: 注册 / 借壳一个企业，用利率较低的「企业经营性贷款」套出资金去偿还房贷或买房。经营贷本应用于企业经营。
- **性质**: **违规**。信贷资金违规流入房地产是监管明令禁止的，监管总局多次专项排查（evidence: [T06-S005]）。
- **后果**: 银行一旦核查发现资金挪用，可**立即抽贷 / 提前收回全部贷款**（借款人需短期内一次性还清，极易资金链断裂）；纳入征信不良；企业 / 个人面临罚息；情节严重涉嫌骗取贷款，**可能承担法律责任**（evidence: [T06-S005]）。
- **大师话术对应**: 「教你把 4% 的房贷换成 3% 的经营贷」「省下几十万利息」—— 听到「包教经营贷置换」，直接判定对方在教你违规、不可信。
- **本 skill 立场**: 不教做法。投资人应**主动远离**，不因「利差诱惑」入局。

### R2. 首付贷 —— 红线

- **是什么（仅识别用）**: 通过消费贷、信用贷、网贷、民间借贷等渠道借钱来凑首付。首付按规定必须是自有资金。
- **性质**: **违规**。首付资金来源造假、信贷资金违规作首付，是监管严禁的（evidence: [T06-S005]）。
- **后果**: 真实杠杆远高于名义杠杆（首付都是借的 = 接近「零首付」的极端杠杆），月供 + 首付贷还款双重现金流压力，下行期极易断供；银行核查到首付来源违规可拒贷 / 抽贷；征信受损（evidence: [T06-S005]）。
- **大师话术对应**: 「首付不够也能上车」「教你解决首付」—— 这是把人推向最危险的杠杆结构。
- **本 skill 立场**: 不教做法。首付必须是「亏得起的自有资金」，凑首付本身就是「不该买」的信号。

### R3. 阴阳合同 —— 红线

- **是什么（仅识别用）**: 同一笔交易签两份价格不同的合同 —— 一份「阳合同」报低价给政府用于过户避税，一份「阴合同」是双方真实成交价。
- **性质**: **违法**。属于偷逃税款，违反税收征管法；合同效力存在重大瑕疵（evidence: [T06-S009]）。
- **后果**: 被税务稽查可**补缴税款 + 罚款 + 滞纳金**，情节严重涉嫌逃税犯罪；阴阳合同纠纷时「低价的阳合同」可能被认定为准，买方 / 卖方都可能吃大亏；银行按低价放贷导致可贷额度不足（evidence: [T06-S009]）。
- **大师话术对应**: 「做低合同价省税费」「中介都这么操作」—— 是把客户和自己一起架在违法的火上。
- **本 skill 立场**: 不教做法。坚持「网签价 = 真实成交价」，全程资金监管。

### R4. 假流水 / 收入证明造假 —— 红线

- **是什么（仅识别用）**: 伪造银行流水、虚开收入证明，把不达标的收入「包装」成符合银行贷款要求的样子。
- **性质**: **违法**。属于提供虚假材料骗取贷款（evidence: [T06-S005]）。
- **后果**: 银行核查发现可拒贷 / 抽贷；情节严重涉嫌**骗取贷款罪 / 贷款诈骗罪**，需承担刑事责任；本质上是用造假掩盖「还款能力不足」，买到的是自己根本供不起的房，断供风险被人为放大（evidence: [T06-S005]）。
- **大师话术对应**: 「流水不够我帮你做」「包装资质，包过」—— 中介 / 大师提供这类服务即是同谋。
- **本 skill 立场**: 不教做法。还款能力不足就是不该贷这么多，造假只是把风险延后并放大。

### R5. 首付来源造假（赠与 / 借款伪装为自有资金）—— 红线

- **是什么（仅识别用）**: 把借来的钱通过「转一手、做时间差」等方式伪装成长期持有的自有资金，规避银行对首付来源的核查。
- **性质**: **违规**，与 R2 首付贷同源 —— 本质是首付资金来源造假（evidence: [T06-S005]）。
- **后果**: 同 R2 —— 隐性高杠杆、双重还款压力、核查到即拒贷 / 抽贷、征信受损（evidence: [T06-S005]）。
- **本 skill 立场**: 不教做法。首付的「真实性」就是杠杆纪律的第一道闸。

> **小结**：以上 5 条的共性是 ——「用造假 / 违规手段绕开监管对杠杆和资金来源的约束」。它们让真实杠杆远高于纸面、让风险被隐藏，下行期集中爆发为抽贷和断供。任何「大师 / 中介」主动兜售这些操作，都应被立即判定为 **利益相关 + 不负责任 + 把法律风险转嫁给客户**。

## 政策 / 标准 / 制度（含时间线）

> 中国房产是「政策驱动多于市场驱动」的市场，读政策机制是核心能力。以下为投资人必须理解的制度框架。

### P1. 限购 / 限贷 / 限售 文件机制

- **Type**: regulation
- **机制**: 中央定基调（房住不炒）→ 因城施策 → 各地住建局 / 房管局发文 → 银行 / 不动产登记中心执行。三限相互独立：限购管资格、限贷管杠杆、限售管退出（evidence: [T06-S010, T06-S001]）。
- **Issued / 演变**: 限购始于 2010 年（一线城市），限售始于 2017 年。2022-2024 进入大面积松绑期 —— 多城取消或大幅放宽限购、2023 年 9 月多城「认房不认贷」。
- **Decay**: high — 政策碎片化、调整频繁，**必须查到「具体城市 + 具体时间」的最新文件**，二手解读不可靠。
- **影响的 workflow 假设**: 「选城市 / 算资格」环节的输入随时可能变。

### P2. LPR 形成机制

- **Type**: standard
- **机制**: 18 家报价行在 MLF 利率基础上加点报价 → 去掉最高最低 → 算术平均 → 央行授权全国银行间同业拆借中心每月 20 日公布 1 年期、5 年期以上两个品种。房贷利率 = 5 年期以上 LPR ± 基点（evidence: [T06-S003, T06-S012]）。
- **Issued**: 2019 年 8 月 LPR 形成机制改革（取代「贷款基准利率」）。
- **Decay**: medium — 机制本身稳定，但报价值每月可变；存量房贷一年重定价一次。
- **影响的 workflow 假设**: 「算账 / 测算月供」环节需用当期 LPR + 自己合同的加减点。

### P3. 商品房预售制度

- **Type**: regulation + standard
- **机制**: 开发商取得《商品房预售许可证》后可对未竣工房屋预售，购房者先付款、后交付 —— 建设期风险事实上由购房者承担。配套有《预售资金监管》制度（预售款进监管账户、按工程节点拨付）（evidence: [T06-S002]）。
- **Issued**: 长期存在的核心制度；2021 年后因烂尾频发，「现房销售」试点在多地扩大。
- **Decay**: medium — 预售制存废 / 改革是行业级讨论中议题。
- **影响的 workflow 假设**: 「买一手房」环节必须新增「核查预售许可 + 预售资金监管 + 工程进度」。

### P4. 预售资金监管制度

- **Type**: regulation
- **机制**: 商品房预售款须存入专用监管账户，按施工进度（如封顶、竣工、交付）分节点拨付给开发商，防止资金被挪作他用导致烂尾（evidence: [T06-S002]）。
- **Issued**: 长期制度，2022 年后因保交楼而被强调和收紧。
- **Decay**: medium — 各地监管力度和细则在动态调整。
- **影响的 workflow 假设**: 「期房尽调」的关键核查点。

### P5. 资金监管制度（二手房存量交易）

- **Type**: regulation
- **机制**: 二手房买方房款打入房管局指定监管账户，过户完成后划付卖方，防止「钱房两失」（evidence: [T06-S002]）。
- **Issued**: 各地长期推行的存量房交易资金监管制度。
- **Decay**: low — 制度稳定。
- **影响的 workflow 假设**: 「二手房交易」环节的不可妥协硬规矩。

### P6. 不动产统一登记制度

- **Type**: standard + regulation
- **机制**: 全国统一的不动产权属登记，由不动产登记中心办理、发《不动产权证书》，可查档核验抵押 / 查封 / 产权人 / 产权性质。法律底座是民法典物权编（原《物权法》）（evidence: [T06-S008]）。
- **Issued**: 2015 年全国统一不动产登记制度落地（《物权法》2007 年施行，《民法典》2021 年施行）。
- **Decay**: low — 制度稳定。
- **影响的 workflow 假设**: 「产权尽调」与「过户」环节的官方入口。

### P7. 保交楼政策

- **Type**: regulation（政策行动）
- **机制**: 2022 年中央政治局会议首次提出「保交楼」，住建部门牵头、地方政府主导，通过专项借款 / 纾困基金等推动停工项目复工交付（evidence: [T06-S002, T06-S013]）。
- **Issued**: 2022 年 7 月中央政治局会议提出，此后持续推进。
- **Decay**: high — 是动态推进的政策行动，进度和工具仍在演变。
- **影响的 workflow 假设**: 改变了「期房风险」的评估方式 —— 保交楼缓释但不消除烂尾风险。

### P8. 「房住不炒」定调

- **Type**: regulation（政策定位）
- **机制**: 2016 年中央经济工作会议提出「房子是用来住的、不是用来炒的」，成为此后所有调控工具的总纲，多次写入政府工作报告（evidence: [T06-S001]）。
- **Issued**: 2016 年 12 月。
- **Decay**: low — 长期定调。注：近年表述有微调（强调「促进房地产平稳健康发展」），但「不炒」的基调未变 —— 投机性买房有长期政策风险这一判断成立。
- **影响的 workflow 假设**: 整个投资框架的底层约束 —— 不赌政策全面转向、给政策留余量。

### P9. 房产税试点（上海 / 重庆）

- **Type**: regulation
- **机制**: 2011 年 1 月上海、重庆同步启动对个人住房征收房产税试点（两地方案不同：上海主要针对新购第二套及以上、重庆主要针对高端住房和无户籍人士购房）。属持有环节税（evidence: [T06-S006, T06-S007]）。
- **Issued**: 2011 年 1 月；全国扩围曾被多次提及但节奏不确定。
- **Decay**: medium-high — 全国推进节奏不明，是「持有成本」长期测算里必须留余量的不确定项。
- **影响的 workflow 假设**: 「持有成本测算」需为潜在房产税预留弹性。

## 行业「外行破绽」高亮（outsider-tell top 10）

> 一说就暴露「不是这行的」的破绽。Phase 2.5 表达 DNA 节核心素材。

| # | 外行的说法 / 误区 | 内行的实际理解 | 频率 |
|---|------------------|----------------|------|
| 1 | 「现在房价怎么样？」（默认有「全国房价」） | 没有「全国房价」这回事 —— 一线核心、三四线、人口流出城市可能同时涨 / 跌 / 阴跌，必须落到具体城市具体板块（evidence: [T06-S004]） | 极高 |
| 2 | 「房子永远涨 / 房产是最保值的资产」 | 这是 2021 年前的旧叙事，2021-2025 是深度下行期，多地房价下跌、流动性变差 —— 还说这话直接暴露认知停在 2017 年（evidence: [T06-S013]） | 极高 |
| 3 | 「杠杆能放大收益」（只说半句） | 杠杆**同时**放大亏损 —— 跌幅超过首付比例本金即归零并资不抵债，下行期是毁灭性的（evidence: [T06-S005]） | 高 |
| 4 | 「买了预售房就等着交房」 | 预售制把建设风险转给购房者，会烂尾；买期房必须查预售资金监管 + 开发商资金链 + 工程进度（evidence: [T06-S002]） | 高 |
| 5 | 「签了买卖合同房子就是我的了」 | 顺序是 网签 → 资金监管 → 贷款 → 过户，**没过户产权都不是你的**，网签 / 交钥匙 / 交定金都不等于确权（evidence: [T06-S008]） | 高 |
| 6 | 「卖了旧房我就算首套」 | 在「认贷」口径下，征信里有过房贷记录（哪怕已结清、房已卖）仍可能算二套，首付差 10-40 个百分点（evidence: [T06-S003]） | 高 |
| 7 | 「LPR 降了我月供马上降」 | 存量房贷通常一年才重定价一次，要等到自己的重定价日才生效（evidence: [T06-S003]） | 高 |
| 8 | 用中介「挂牌价」判断市场 / 算能贷多少 | 挂牌价是「卖方想要的价」，网签价才是真实成交价，可贷额度按「评估价」算 —— 内行看官方网签数据和流动性指标（挂牌量 / 成交周期）（evidence: [T06-S002, T06-S004]） | 高 |
| 9 | 「便宜一半的小产权房 / 商铺收租稳 / 文旅养老房投资双丰收」 | 小产权房法律不认、商铺写字楼高空置流动性极差、远郊文旅盘几乎无接盘 —— 内行对这几类基本一票否决或极度审慎（evidence: [T06-S008, T06-S011]） | 中高 |
| 10 | 「70 年产权从我买房算起」「学区房溢价是永久硬通货」 | 产权年限从开发商拿地起算（二手房可能只剩 50 多年）；学区溢价完全依赖学区政策，多校划片 / 学位锁定一变就蒸发（evidence: [T06-S008, T06-S010]） | 中高 |

> 补充破绽：把「契税 / 增值税 / 个税 / 中介费」笼统说成「税费」一笔糊涂账（内行逐项拆：谁交、按什么基数、首套二套差多少）；把「自住」和「投资」混为一谈用一套逻辑（内行严格分开 —— 自住看居住效用，投资看回报率 / 流动性 / 退出）。

## 行业拒绝的话术（大师 / 销售营销话术 top 5 — Phase 2 表达 DNA 反例版用）

> **这些不是术语，是营销话术** —— 负责任的房产投资视角**拒绝**这些表达。拒绝它们 = 反映这一行的价值观和反模式。Phase 2.5 表达 DNA「反例版」直接用。

| # | 被拒绝的话术 | 为什么拒绝 / 内行会怎么说 |
|---|-------------|--------------------------|
| 1 | **「房产永远涨 / 闭眼买就对了」** | 2021-2025 深度下行期证伪了这套旧叙事。内行说：没有「永远涨」的资产，要分城市、分板块、分周期，且要算流动性 —— 笼统看多本身就是不专业（evidence: [T06-S013]） |
| 2 | **「跟我抄底 / 现在就是底部」** | 底部只能事后确认，「抄底」常变「接盘」。且喊单者多为利益相关方（卖课 / 卖咨询 / 和中介开发商分佣 / 只晒对不晒错）。内行说：没人能精准抄底，要做的是算账、留余量、按纪律行动（evidence: [T06-S004]） |
| 3 | **「杠杆放大收益 / 用足杠杆才聪明」** | 这是只说半句的危险话术。内行说：杠杆同等放大亏损和现金流断裂风险，下行期高杠杆是毁灭性的，杠杆纪律是硬规矩 —— 必须同时讲「放大亏损 + 断供」（evidence: [T06-S005]） |
| 4 | **「闭眼买核心资产 / 核心地段永远不会错」** | 「核心资产长期看好」是 2024-2025 最大分歧之一，没有干净答案；「闭眼买」回避了估值、流动性、杠杆纪律。内行说：核心地段也有买贵的价格、也要做尽调和算账，「闭眼」二字本身就是反专业（evidence: [T06-S004]） |
| 5 | **「包教经营贷置换 / 帮你解决首付 / 流水不够我来做」** | 这是在兜售违规甚至违法操作（R1-R5）。内行说：听到「包教这些」立刻判定对方不可信 —— 这类大师 / 中介在把法律风险和断供风险转嫁给客户（evidence: [T06-S005, T06-S009]） |

> 共同反模式：**确定性叙事（永远涨 / 一定是底）+ 回避风险（闭眼买 / 不提杠杆下行）+ 利益相关（卖课卖咨询分佣）+ 兜售违规（经营贷 / 首付贷）**。负责任的房产投资语言的特征恰好相反：分城市分板块、强调流动性和杠杆双向风险、严格区分自住与投资、读政策不赌政策、把灰色操作划为红线。

### 「智识谱系」线索（喂给 Phase 2.7 智识谱系）

- **政策演变暗示范式更替**：「房住不炒」（2016）→「认房认贷」普遍 →「认房不认贷」松绑（2023）→ 多城取消限购（2024）—— 反映从「全面抑制」到「因城施策、托而不举」的调控哲学转变。
- **预售制 → 现房销售试点扩大**：反映对「购房者承担建设风险」这一旧制度的反思。
- **多流派竞争（硬件层）**：高杠杆技术流（水库系，对应灰色操作 R1-R5）vs 低杠杆稳健派；「核心资产长期看好」vs「房地产高增长时代结束」；「一线核心永远涨」vs「人口拐点 + 老龄化全面承压」 —— glossary 里「杠杆」「抄底 / 止损」「城市能级」「人口流入流出」是这些流派之争的术语战场。

### 「时效性」信号（喂给 Phase 2.8 诚实边界）

- **高 decay、12 个月内大概率再变的政策 / 标准**：限购 / 限贷 / 限售（持续松绑中）、认房认贷口径、保交楼政策（动态推进）、学区政策、落户政策、LPR 报价值。
- **2022-2024 已发生的重大修订**：2023-09 多城「认房不认贷」、2024 多城取消 / 放宽限购、2024 首套首付降至 15%-20% 历史低位 —— master skill 引用任何「具体首付比例 / 限购规则」时必须标「以当地最新文件为准」。
- **节奏不确定的长期不确定项**：房产税全国扩围、预售制改革、住宅土地到期续期细则。

### 「工作流变化触发」种子（喂给 wave 3 Track 03）

- **限购 / 限贷 / 限售松绑** → 改变「选城市 / 算资格 / 算杠杆」环节的输入，且因城施策导致必须「逐城逐区查最新文件」成为固定动作。
- **保交楼政策 + 开发商爆雷常态化** → 「买一手房」工作流新增硬环节：核查预售许可 + 预售资金监管 + 工程进度 + 开发商资金链（「大开发商 = 安全」假设失效）。
- **LPR 机制 + 重定价规则** → 「算账 / 测算月供」环节必须用当期 LPR + 自己合同的加减点 + 重定价日。
- **二手房指导价机制** → 「算首付」环节要按「评估价」而非「成交价」算可贷额度。
- **房产税试点 / 潜在扩围** → 「持有成本测算」需为持有环节税预留弹性。
