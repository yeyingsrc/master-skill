## 本 track 的元判断(先读这个 —— 它决定每个 workflow 怎么写)

国内房产投资的工作流不是「找房 → 砍价 → 签约」的交易动作清单,是**一条「先把可控的做到最好,再面对不可控」的投资弧线**。三条贯穿全文的原则:

1. **每个环节都必须挂「政策检查点 + 风险检查点」**。中国房产是政策驱动 > 市场驱动(canon 强候选心智模型),且 2021-2025 是深度下行期 —— 任何一个环节跳过政策核对或风险核对,都可能让整笔投资归零(烂尾 / 查封 / 断供 / 卖不掉)。这是 intake 明确要求,不是可选项。
2. **资深差异不是「教抄底 / 教加杠杆 / 教灰色操作」**。经营贷置换房贷 / 首付贷 / 阴阳合同 / 假流水是红线(glossary R1-R5),workflow 里出现就标红线、讲后果、不教做法。资深的 skip/optimize/add 全部是「把可控部分(选城市 / 尽调 / 算账 / 留余量)做到极致 + 杠杆纪律 + 不赌政策」。
3. **2021 年是分水岭,旧叙事全部作废**。「买房致富 / 闭眼买 / 核心地段永远涨」是 glossary 明确列出的「行业拒绝话术」。退出 workflow 必须正视「2024-2025 流动性显著变差,可能卖不掉或大幅折价」;选城市 workflow 必须强调「没有『全国房价』」。

> ⚠️ 红线提示:本文任何 workflow 都不收录、不教学「经营贷置换 / 首付贷 / 阴阳合同 / 假流水 / 首付来源造假」—— 这些是违规甚至违法操作(glossary R1-R5),出现处一律标红线 + 讲后果。

### W1. 选城市(判断「这个城市的房子有没有基本面」)

- **One-liner**: 从「我想买房投资」到「锁定 1-3 个值得继续往下看的候选城市」—— 「买对城市」是国内房产投资的第一性问题,2021 年后城市间走势分化极大,选错城市后面所有努力都白费。 (evidence: [T01-S001, T05-S011, T05-S013])
- **Trigger**: 准备配置一笔房产投资 / 考虑跨城买房 / 现持有城市基本面恶化想换城市。
- **Output**: 一份「候选城市短名单(1-3 个)+ 每个城市的人口/产业/土地/政策画像 + 排除掉的城市及理由」,作为 W2(选板块)的输入。
- **入门 SOP(minimum viable steps)**:
  1. **查城市能级与房价方向**:用国家统计局 70 城房价指数,看目标城市所在能级(一线/二线/三线)是涨是跌、连续涨跌几个月 —— 看「环比」不看「同比」(同比滞后一年)。inputs: 候选城市名单;outputs: 每城价格趋势标签。
  2. **查人口净流入还是净流出**:看常住人口近 3-5 年变化、小学在校生数变化(比户籍人口更真实)、是否有产业/就业在聚集。人口持续净流出的城市,房价没有基本面。inputs: 统计公报/人口数据;outputs: 人口流向标签。
  3. **查土地与库存(供给端)**:用中指 CREIS 看土地流拍率/溢价率,用易居研究院看存销比(去化周期月数)—— 存销比 > 18-24 个月 = 严重供过于求,这是识别三四线去库存陷阱的关键指标。inputs: 研究院报告;outputs: 供给压力标签。
  4. **查城市级政策环境**:到目标城市住建局官网查最新限购/限贷/限售文件,确认自己是否有购房资格、首付几成。inputs: 地方住建局公告;outputs: 政策资格标签。
  5. **三关合一出短名单**:价格方向 + 人口流向 + 供给压力 + 政策资格四个标签同时过关的城市才进短名单;任一硬伤(人口流出 / 存销比超高 / 无购房资格)直接排除。
  - **每一步如果跳过会发生什么**:跳过步骤 1 → 在能级整体下行的城市里找「洼地」,接整城下跌的刀;跳过步骤 2 → 买进人口流出城市,长期没有接盘需求,租不掉也卖不掉;跳过步骤 3 → 踩进三四线去库存陷阱,新房持续放量压制二手房永远卖不上价;跳过步骤 4 → 看完房才发现没资格买 / 首付差一大截,白忙一场。
- **资深路径（差异点）**:
  - **skip**:资深人会**跳过「找全国洼地 / 城市轮动抄底」这一整类动作** —— 经验(及 Wave 1 跨源共识)告诉他们「没有任何公开工具能可靠支撑精准抄底」,城市轮动是叙事不是方法。他们把精力直接收敛到「自己能持续跟踪、有真实需求支撑」的少数城市,不做大范围扫货。
  - **optimize**:资深人会**优化步骤 2 的人口判断** —— 不只看总量,而是看「人口结构」:净流入里有多少是育龄/就业人口、城市是不是在「抢人」(落户政策松紧)、产业是单一支柱还是多元。他们用「小学在校生连续增长」这类难造假的代理指标,而不是地方政府宣传口径。
  - **add**:资深人会**额外做「同一城市内部已经在分化」的预判** —— 即使一个城市整体能级在涨,也会提前判断「这个城市哪些板块在被抽血、哪些在虹吸」,避免「买对城市、买错板块」(直接衔接 W2)。初学者把城市当一个均质单元,资深人知道城市内部分化和城市间分化一样大。
- **政策检查点**:
  - 目标城市最新**限购**文件(社保/户籍年限、套数限制)—— 决定有没有资格(evidence: [T02-S014, T02-S015])。
  - 目标城市最新**限贷**文件(认房认贷口径、首付比例)—— 决定要准备多少自有资金。
  - **落户/人才购房**政策 —— 既影响购房资格,也是判断城市「抢人」力度的政策信号。
  - ⚠️ 因城施策导致政策一城一策、可一夜调整 —— 「逐城逐时点查最新文件」是固定动作,不能用 A 城逻辑套 B 城(evidence: [T03-S001])。
- **风险检查点**:
  - **人口流出风险**:常住人口/小学生数下滑的城市,基本面缺失,优先排除。
  - **三四线去库存陷阱**:存销比超 18-24 个月、土地流拍率高 —— 新房持续放量会长期压制二手房价格与流动性(evidence: [T05-S013])。
  - **「全国房价」幻觉风险**:不存在「全国房价怎么样」,任何笼统判断都是错的,必须落到具体城市(intake 警示 #6)。
  - **政策资格风险**:确认资格再看房,避免无资格白忙。
- **近期变化 / last_updated**:
  - 2024 起多城取消/放宽限购、首套首付降至 15%-20% 历史低位 —— 「算购房资格 / 算杠杆」环节的输入持续变化,因城施策使「逐城查最新文件」从可选变成必做(evidence: [T03-S001],触发事件类型:法规变化)。
  - 2021 年后城市间走势深度分化成为常态 —— 「选城市」从过去「闭眼买一线」变成必须做人口/土地/政策三查的硬环节(触发事件类型:行业事件)。
  - last_updated: 2026-05-14。
- **典型耗时**:入门 SOP 约 1-2 周(查数据 + 读政策);资深路径首次建立跟踪框架约 2-4 周,之后是低成本的持续监控。
- **关键工具(与 Track 02 关联)**:国家统计局 70 城房价指数(必备)、中指 CREIS / 克而瑞 / 易居研究院报告(场景特化)、地方住建局公告(必备)。
- **关键人物(与 Track 01 关联)**:任泽平「长期看人口、中期看土地、短期看金融」框架对应本 workflow 的查人口→查土地→查信贷顺序(用框架,不背书其人);李迅雷「人口负增长 + 城市分化」、黄奇帆制度框架在公开演讲/著作中谈过城市分化的结构性。
- **常见失败模式**:
  - **在整城下行能级里找「洼地」**:触发原因 = 把「跌得多」当「便宜」;怎么避免 = 先用统计局指数确认城市能级方向,能级在跌就不在里面找洼地。Don't: 看到某三四线城市房价腰斩就觉得「抄底机会」,买进后发现人口外流、库存巨大,十年阴跌还卖不掉。
  - **信地方政府宣传口径的人口数据**:触发原因 = 用易造假的口径;怎么避免 = 用小学在校生数、社保参保人数等难造假的代理指标交叉。Do: 拉目标城市近 5 年小学在校生数曲线,下滑就不碰。
  - **用中介挂牌价做城市横向比较**:触发原因 = 各城口径不一 + 卖方报价;怎么避免 = 横向比较只用统计局 70 城指数(evidence: [T02-S001])。
- **Decay risk**: medium —— 方法论(人口/土地/政策三查)稳定,但「具体城市的政策资格 + 库存数据」12 个月内会变,需定期刷新。

### W2. 选板块与地段(在选定城市内部锁定区域价值单元)

- **One-liner**: 从「选定了城市」到「锁定 2-4 个值得具体看房的板块/地段」—— 城市内部分化和城市间分化一样大,板块逻辑(产业/交通/学区/规划/配套)决定了同城不同板块可能一个虹吸一个被抽血。 (evidence: [T05-S010, T05-S011])
- **Trigger**: W1 选出候选城市后,需要在城市内部缩小范围 / 城市内某板块出现规划利好或利空传闻。
- **Output**: 一份「目标板块短名单 + 每个板块的产业-交通-学区-规划-配套画像 + 板块二手房流动性读数(挂牌量/成交周期)」,作为 W3(选房)的输入。
- **入门 SOP(minimum viable steps)**:
  1. **建立板块价值地图**:把城市分成板块/地段单元,标注每个板块的核心驱动 —— 产业就业(有没有大企业/园区)、交通(地铁/通勤时间)、配套(商业/医院)成熟度。inputs: 城市规划 + 地图;outputs: 板块价值排序。
  2. **核查规划(利好与利空都要查)**:到城市规划/自然资源部门官网查目标板块的规划 —— 地铁线路、学校、商业是否落地;同时查「利空规划」:垃圾站、变电站、高压线、高架。inputs: 规划部门公示;outputs: 规划利好/利空清单。
  3. **核查学区(如果板块靠学区溢价)**:确认目标板块对应的学校、最新的学区划片政策(多校划片/摇号)、入学年限要求 —— 学区政策是高 decay 项,会变。inputs: 教育局划片文件;outputs: 学区有效性判断。
  4. **读板块二手房流动性**:用贝壳研究院/找房平台看目标板块的二手房挂牌量、成交周期、带看量 —— 同城里挂牌量暴增、成交周期拉到 12 个月的板块,流动性已经在恶化。inputs: 贝壳研究院/平台数据;outputs: 板块流动性标签。
  5. **筛出短名单**:产业/交通/配套有支撑 + 规划无重大利空 + (如靠学区)学区有效 + 流动性未崩 —— 同时满足才进短名单。
  - **每一步如果跳过会发生什么**:跳过步骤 1 → 在没有产业支撑的「睡城」买房,空置率高、租不掉;跳过步骤 2 → 买进后才发现门口要建垃圾站/高压线,或指望的地铁/学校根本没规划;跳过步骤 3 → 学区是中介编的,或政策一改片区学位作废,学区溢价归零;跳过步骤 4 → 买进同城流动性最差的板块,要卖时才发现根本没人接。
- **资深路径（差异点）**:
  - **optimize**:资深人会**优化步骤 2 的规划核查** —— 不只查「有没有规划」,而是判断「规划兑现概率和兑现时间」:规划上写了地铁但十年没动工 = 不能算进估值;土地已出让、开工在即的配套才算数。他们区分「规划画的饼」和「在建的实物」。
  - **add**:资深人会**额外做「板块在被虹吸还是被抽血」的资金流向判断** —— 看城市新房供应在往哪个板块集中、产业/人口在往哪迁移、政府重点投入在哪。一个城市的「钱」会从老板块流向新板块,买在被抽血的板块即使城市整体不跌也会跑输。初学者只看板块「现在好不好」,资深人看「钱在往哪流」。
  - **skip**:资深人会**跳过「纯靠规划利好预期下注还没兑现的远郊板块」** —— 经验告诉他们,下行期「规划兑现」被无限期推迟、远郊新区库存最先崩,「炒预期」在 2021 后是高危动作。他们要么买已兑现配套的成熟板块,要么不碰。
- **政策检查点**:
  - **规划政策**:目标板块的控制性详细规划、近期重点建设项目 —— 规划部门公示原文,不信中介「这里要建地铁」的口头承诺。
  - **学区政策**:最新划片方案、多校划片/摇号政策、入学年限 —— 高 decay,买入前查最新文件(evidence: 06-glossary 学区房条目)。
  - **板块级限购/限售差异**:部分城市分区限购,核对目标板块是否在限制区。
- **风险检查点**:
  - **「睡城」/无产业板块风险**:没有产业就业支撑的纯住宅板块,长期空置高、租金低、流动性差。
  - **规划利空风险**:垃圾站/变电站/高压线/高架/殡仪馆等邻避设施 —— 买前实地 + 查规划。
  - **学区政策变动风险**:学区房溢价不是永久的,划片可改、政策可变(glossary 外行破绽 #9 学区溢价永久)。
  - **板块流动性恶化风险**:同城里挂牌量暴增、成交周期最长的板块要先排除(evidence: [T05-S010])。
- **近期变化 / last_updated**:
  - 2021 后城市内部分化加剧 —— 「选板块」从过去「市区随便买」变成必须做资金流向判断;远郊新区/「睡城」库存在下行期最先承压(触发事件类型:行业事件)。
  - 多校划片/摇号在多城推广 —— 「学区」从「买对小区锁定学位」变成「买对小区也不一定锁定学位」,学区核查环节复杂度上升(触发事件类型:法规变化)。
  - last_updated: 2026-05-14。
- **典型耗时**:入门 SOP 约 1-2 周(规划核查 + 实地踏勘);资深路径相近,但资金流向判断需要长期积累的城市认知。
- **关键工具(与 Track 02 关联)**:贝壳研究院/找房平台(板块流动性)、中指 CREIS(土地/供求板块数据)、城市规划部门官网(规划核查)。
- **关键人物(与 Track 01 关联)**:真叫卢俊在公开演讲/文章中谈过城市板块逻辑与产品视角(作一线观察样本,涉具体荐盘不采信)。
- **常见失败模式**:
  - **把规划的「饼」算进估值**:触发原因 = 听信「这里要通地铁/建名校」;怎么避免 = 只把已出让土地、已开工的配套算数,规划利好打折看。Don't: 因为「规划有地铁」溢价买入,十年后地铁仍未动工,板块价值兑现遥遥无期。
  - **买「睡城」当价值洼地**:触发原因 = 单价低误判为便宜;怎么避免 = 先看板块有没有产业就业,无产业不碰。
  - **信中介的学区承诺**:触发原因 = 中介「保证能上 XX 学校」;怎么避免 = 查教育局最新划片文件原文,政策可变。
- **Decay risk**: medium —— 板块价值逻辑稳定,但学区政策、规划进度、板块流动性是 12 个月内会变的高频项。

### W3. 选房(一手房 vs 二手房的标的筛选与初判)

- **One-liner**: 从「锁定了板块」到「圈定 3-8 套值得进入正式尽调的具体标的」—— 一手房的核心风险是烂尾与开发商爆雷,二手房的核心是房龄/户型/税费资格,两条路线的筛选项完全不同。 (evidence: [T02-S009, T05-S014])
- **Trigger**: W2 选定板块后开始具体看房 / 某楼盘开盘 / 某二手房源挂出。
- **Output**: 一份「候选标的清单(3-8 套)+ 每套的一手/二手类型标签 + 初判通过项与待核项」,作为 W4(估值与尽调)的输入。
- **入门 SOP(minimum viable steps)**:
  1. **分流:一手还是二手**:先确定走一手房还是二手房路线 —— 两条路线的筛选清单不同,不能混着筛。
  2. **【一手房路线】核查开发商与预售安全**:查开发商是否有预售许可证、预售资金是否进了监管账户、工程进度、开发商在克而瑞的销售排名/债务/爆雷状态 —— 「大开发商 = 安全」的假设在 2021 后已失效。优先考虑现房或准现房。inputs: 预售许可公示 + 克而瑞房企数据;outputs: 一手标的安全标签。
  3. **【二手房路线】核查房龄/户型/楼层 + 税费资格**:看房龄(影响贷款年限和残值)、户型(南北通透/采光/格局)、楼层;核对是否「满五唯一」「满二」(直接决定增值税/个税要不要交、谁交)。inputs: 产权证 + 房源信息;outputs: 二手标的初判标签。
  4. **建立候选清单并初步比价**:把通过初判的标的列清单,用同板块同户型横向看挂牌价分布建立价格感(注意:挂牌价不是成交价)。inputs: 找房平台;outputs: 候选标的清单。
  - **每一步如果跳过会发生什么**:跳过步骤 1 → 用二手房的逻辑筛一手房(或反之),漏掉对方路线的关键风险;跳过步骤 2 → 买了预售房遇上开发商爆雷,房子烂尾、首付和已还月供打水漂;跳过步骤 3 → 买到房龄过老贷不了款/残值低的房,或满二不满五要多缴一大笔增值税才发现;跳过步骤 4 → 没有横向参照,被单套报价锚定,买贵。
- **资深路径（差异点）**:
  - **add**:资深人买一手房会**额外做开发商资金链尽调** —— 不只看「是不是大牌」,而是查开发商的债务到期表、有没有商票逾期、其他项目有没有停工、在克而瑞/财新的爆雷追踪里有没有出现。2021 后大开发商爆雷成常态(碧桂园、恒大),「品牌」不再是安全凭证(evidence: [T02-S009, T05-S014])。
  - **skip**:资深人在下行期会**大幅跳过 / 收缩「买预售期房」这一选项** —— 经验(及行业反面案例)告诉他们,期房承担了「开发商建不成」的全部风险而买方无对价,下行期这个风险被放大。他们优先现房/准现房,或干脆只买二手 —— 用「确定能拿到的房」换掉「期房折扣」。
  - **optimize**:资深人会**优化二手房户型/楼层的筛选标准** —— 从「自己喜欢」升级为「未来好不好卖」:户型方正、南北通透、楼层适中的房子接盘面最广;「老破小」「奇葩户型」「顶楼/一楼」在下行期流动性折价最大。他们按「退出友好度」选房,不按个人偏好。
- **政策检查点**:
  - **【一手】预售许可与预售资金监管政策**:核对楼盘是否取得预售许可、本城预售资金监管制度执行情况(evidence: [T02-S004])。
  - **【一手】保交楼/白名单政策环境**:查住建部及本城保交楼、房企融资白名单政策 —— 是烂尾风险评估的政策背景(evidence: [T02-S004])。
  - **【二手】满五唯一/满二税费政策**:核对最新二手房税费优惠资格口径(evidence: 06-glossary 满五唯一条目)。
  - **【二手】房龄相关的贷款政策**:部分城市对房龄过老的二手房限制贷款年限甚至不予贷款。
- **风险检查点**:
  - **烂尾风险(一手核心)**:开发商资金链断裂 → 停工 → 烂尾。买预售房 ≠ 安全,必须核查预售许可+资金监管+工程进度+开发商资质(intake 警示 #7)。
  - **开发商爆雷风险**:头部房企爆雷常态化,品牌不等于安全(evidence: [T05-S014])。
  - **房龄/残值风险(二手核心)**:房龄过老 → 贷款年限短/贷不到款 → 接盘面窄 → 退出难。
  - **户型/楼层流动性折价风险**:「老破小」、奇葩户型、顶楼一楼在下行期折价最大,退出困难。
- **近期变化 / last_updated**:
  - 2021 后开发商爆雷常态化(恒大 2021、碧桂园 2023 等)—— 「买一手房」工作流新增「核查开发商资金链」硬环节,「大开发商 = 安全」假设失效(evidence: [T05-S014],触发事件类型:行业事件)。
  - 现房销售试点扩大、保交楼政策推进 —— 反映对「购房者承担建设风险」旧制度的反思,资深人加速向现房/二手倾斜(evidence: [T02-S004],触发事件类型:法规变化)。
  - last_updated: 2026-05-14。
- **典型耗时**:入门 SOP 约 1-3 周(看房 + 初查);资深路径因加做开发商资金链尽调,一手房路线多 3-7 天。
- **关键工具(与 Track 02 关联)**:克而瑞 CRIC(开发商爆雷/排名)、贝壳/安居客找房平台(房源初筛比价)、住建部/地方住建局(预售许可与保交楼政策)。
- **关键人物(与 Track 01 关联)**:财新报道的恒大/碧桂园爆雷调查是「开发商爆雷如何发生」的现实教材(Track 05 关联);黄奇帆谈预售制改革与开发商负债率上限。
- **常见失败模式**:
  - **把「品牌开发商」当烂尾免疫**:触发原因 = 用 2015 年的经验(大开发商不会倒);怎么避免 = 查开发商当前债务/爆雷状态,品牌只是起点不是结论。Don't: 因为是百强房企就买预售,结果该房企资金链断裂、项目停工。
  - **为「期房折扣」承担烂尾全险**:触发原因 = 被期房比现房便宜吸引;怎么避免 = 下行期优先现房/准现房,期房折扣不足以覆盖烂尾风险。
  - **按个人偏好而非退出友好度选二手房**:触发原因 = 「我喜欢这个户型」;怎么避免 = 按「未来谁来接、好不好卖」选房,户型方正/楼层适中优先。
- **Decay risk**: medium —— 一手 vs 二手的筛选逻辑稳定,但开发商爆雷名单、保交楼政策、二手房贷款政策是 12 个月内会变的高频项。

### W4. 估值与尽职调查(合理价格判断 + 产权法律核查)

- **One-liner**: 从「圈定了候选标的」到「对每套房给出『合理价格区间 + 产权是否干净 + 能不能买』的结论」—— 估值定「值不值这个价」,尽调定「能不能碰」,产权尽调是不可跳过的硬环节,跳过等于踩雷。 (evidence: [T02-S013, T02-S016, T05-S010])
- **Trigger**: W3 圈定候选标的后,准备出价/谈判前 —— 这是出手前的最后一道关。
- **Output**: 每套候选标的的「合理价格区间 + 产权核查结论(干净/有瑕疵/不可碰)+ 完整持有账测算 + 最终保留或淘汰」。
- **入门 SOP(minimum viable steps)**:
  1. **锚定真实成交价**:用各地房管局网签备案的真实成交价(法律意义的成交)做价格锚,辅以贝壳「成交记录」和中国房价行情网小区历史曲线 —— 绝不用中介挂牌价当估值依据(下行期挂牌价系统性偏高 10-20%)。inputs: 网签数据 + 平台成交记录;outputs: 合理价格区间。
  2. **产权核查(不动产登记中心查档)**:查「他项权利」(有没有抵押、抵押给谁、金额)、「查封」(有没有被法院查封、有没有轮候查封)、「权利人」(卖方身份与产权证一致、共有产权人是否齐全)、「规划用途/性质」(住宅 vs 商办 40 年产权 vs 划拨地)、「面积与四至」。inputs: 不动产登记查档;outputs: 产权核查结论。
  3. **算完整持有账(自建 Excel)**:买入端(首付+契税+增值税+个税+中介费+贷款手续费)+ 持有端(月供利息+物业+维修+空置损失+资金占用机会成本+房产税试点城市加这项)+ 收益端(扣空置的真实租金,资本利得在 2024-2025 敢填 0 甚至负)+ 退出端(卖出税费+中介费+流动性折价)。inputs: LPR+税费政策+租金数据;outputs: 净租金回报率、盈亏平衡持有年限。
  4. **跑压力测试**:模型必须跑「房价跌 20% / 租金跌 30% / 利率升 1%」三个压力场景,看现金流会不会断。inputs: 持有账模型;outputs: 压力情景下的现金流结论。
  5. **出尽调结论**:价格在合理区间 + 产权干净 + 完整账划算 + 压力测试能扛 —— 四项全过才保留;产权有任何瑕疵(抵押未明、查封、性质不符)直接淘汰或要求卖方先解决。
  - **每一步如果跳过会发生什么**:跳过步骤 1 → 用挂牌价估值,买贵 10-20%;跳过步骤 2 → 买到抵押房/查封房(过不了户、钱付了房拿不到)、买到商办性质当住宅(税费/落户/水电全不同)、买到「假房主」的房;跳过步骤 3 → 只算了月供没算完整持有成本,以为划算实则年年贴钱;跳过步骤 4 → 顺周期假设下买入,房价/租金/利率一动现金流就断,滑向断供。
- **资深路径（差异点）**:
  - **add**:资深人会**额外把「流动性折价」明确写进估值模型** —— 不是算「现在值多少」,而是算「急用钱时 3 个月内能变现的价值是多少」。2024-2025 挂 12 个月卖不掉、要降价 X% 才成交,这个折价必须进模型。初学者算账面价值,资深人算可变现价值(evidence: [T01-S015])。
  - **optimize**:资深人会**优化产权核查的深度** —— 不止查「有没有抵押」,还查抵押的具体情况决定交易路径(带押过户 vs 先解押再过户,资金风险完全不同);查到「轮候查封」直接判定不可碰;把「实测面积 vs 证载面积」的差额(违建/赠送面积无产权)从估值里扣掉。他们把查档结果直接转成「交易方案 + 估值调整」。
  - **skip**:资深人会**跳过对「明显有硬伤标的」的精细估值** —— 一旦查档发现产权瑕疵(抵押说不清、轮候查封、性质是小产权/划拨)或标的是已被定性的高风险类型,直接淘汰,不浪费时间做精细估值和谈判。初学者舍不得放弃已看中的房,资深人「一票否决」干脆。
- **政策检查点**:
  - **税费政策**:契税(首套/二套税率不同)、增值税/个税(满五唯一/满二影响)、本城最新税费优惠口径 —— 直接进估值模型的买入端和退出端(evidence: 06-glossary 契税/增值税条目)。
  - **二手房核验价/指导价政策**:部分城市银行按「评估价/参考价」而非成交价放贷 —— 必须查本城是否有此政策、目标小区参考价,直接影响首付要多掏多少(evidence: [T02-S014, T02-S015])。
  - **房产税试点政策**:上海/重庆已试点,全国立法仍在推进 —— 长期持有测算必须为持有环节税预留弹性,不能按「永远不收」算(evidence: [T03-S002])。
  - **剩余产权年限政策**:70 年/40 年产权的剩余年限、到期续期细则仍不明确 —— 影响残值和贷款。
- **风险检查点**:
  - **产权瑕疵风险(尽调硬核心)**:抵押、查封、轮候查封、产权人不符、共有人不全 —— 任一存在不可裸买(intake 警示 #9)。
  - **小产权房风险**:无合法产权、不受法律保护、不能贷款不能正常过户 —— 直接排除(glossary 小产权房条目)。
  - **性质风险**:商住 40 年产权 / 划拨地 / 划拨性质 —— 税费、落户、水电价格、再交易难度全不同,不能当普通住宅估值。
  - **估值锚错误风险**:用挂牌价当成交价 = 系统性买贵(evidence: [T05-S010])。
  - **顺周期假设风险**:估值模型用「房价永远涨」假设 = 没有压力测试 = 现金流脆弱。
- **近期变化 / last_updated**:
  - 2024-2025 下行期 —— 估值模型重点从「算回报率」转向「算流动性折价 + 现金流断裂风险」,资本利得项要敢填 0/负数(触发事件类型:行业事件)。
  - 二手房参考价/指导价机制(深圳 2021 首推,后多地反复)—— 「算首付」环节要按评估价而非成交价算可贷额度;2024-2025 部分城市又淡化/暂停,高度不稳定(evidence: [T02-S015],触发事件类型:法规变化)。
  - 带押过户在多城推广 —— 查档发现有抵押时,交易路径多了「带押过户」选项,降低资金风险(evidence: [T02-S013],触发事件类型:标准更新)。
  - last_updated: 2026-05-14。
- **典型耗时**:入门 SOP 约 3-7 天/套(查档 + 算账);资深路径因加做流动性折价建模和深度查档,相近或略多,但「一票否决」筛掉硬伤标的反而总耗时更短。
- **关键工具(与 Track 02 关联)**:不动产登记中心查档(必备)、各地房管局网签备案数据(必备,成交价锚)、自建持有成本/回报率 Excel(必备)、中国房价行情网(小区微观估值)、央行 LPR(算账杠杆成本输入)。
- **关键人物(与 Track 01 关联)**:香帅《钱从哪里来》强调把房产放进家庭资产负债表算「占比与流动性」,对应本 workflow 的流动性折价建模;谨慎投资人社群以「现金流压力测试表」作出手硬门槛。
- **常见失败模式**:
  - **信中介一句「房子很干净」就跳过查档**:触发原因 = 嫌查档麻烦 + 信任中介;怎么避免 = 每套房无论一手二手法拍都必须自己到不动产登记中心查档。Don't: 听中介说「产权没问题」就付定金,过户时才发现房子有抵押/被查封,定金和时间全打水漂,甚至卷入诉讼。
  - **用挂牌价当成交价估值**:触发原因 = 平台上只看得到挂牌价;怎么避免 = 成交价锚定回房管局网签,挂牌价只看「卖家心理预期」。
  - **只算月供当完整账**:触发原因 = 用市面房贷计算器;怎么避免 = 自建 Excel 算完整持有账并跑压力测试,月供能扛 ≠ 投资划算。Don't: 算完月供觉得「能负担」就买,忽略物业/空置/资金占用/退出税费/流动性折价,年年净流出。
  - **估值不留政策余量**:触发原因 = 按「房产税永远不收」「政策永远松」算;怎么避免 = 持有成本测算为房产税扩围预留弹性,读政策、留余量。
- **Decay risk**: medium —— 产权核查方法稳定(法定登记制度,low),但税费税率/参考价政策/房产税进展/利率参数是 12 个月内会变的高频项,整体 medium。

### W5. 交易与税费(网签 → 资金监管 → 缴税 → 过户 → 放款)

- **One-liner**: 从「尽调通过、谈妥价格」到「完成过户、拿到不动产权证、贷款放款」—— 这是把「钱」和「房」安全交换的执行链条,每个环节都对应着「钱什么时候付、税什么时候缴、风险敞口在哪」。 (evidence: [T02-S007, T02-S013])
- **Trigger**: W4 尽调通过 + 买卖双方就价格/付款方式/交付时间达成一致。
- **Output**: 完成过户登记、买方取得不动产权属证书、贷款放款到卖方、资金监管账户解冻 —— 交易闭环。
- **入门 SOP(minimum viable steps)**:
  1. **签合同 + 网签备案**:签订买卖合同,办理网上签约备案(网签)—— 网签后房源被锁定,这是交易合法性的前置环节。签合同 ≠ 房子是你的,过户登记完成才是。inputs: 买卖合同;outputs: 网签备案号。
  2. **首付进资金监管账户**:首付款打入第三方资金监管账户(不是直接打给卖方),房款被监管,过户完成等条件满足才放款给卖方 —— 绝不走私下交易、无监管转账。inputs: 首付款;outputs: 资金进监管。
  3. **办贷款审批**:向银行提交贷款申请,银行审批(评估、征信、流水核查)—— 用真实材料,首付贷/假流水/经营贷置换是红线。inputs: 贷款材料;outputs: 贷款批复。
  4. **缴税过户**:在不动产登记中心办理过户登记,同步缴纳契税(买方)、增值税/个税(卖方税,常被转嫁给买方,合同里要写清谁承担)、中介费 —— 拿到新的不动产权证。inputs: 完税凭证;outputs: 不动产权属证书。
  5. **放款 + 监管解冻**:过户完成、抵押登记办妥后,银行放贷款尾款给卖方,资金监管账户按约定解冻 —— 交易完成。inputs: 过户证明;outputs: 交易闭环。
  - **每一步如果跳过会发生什么**:跳过步骤 1(网签)→ 房源未锁定,卖方可能一房二卖;跳过步骤 2(资金监管)→ 钱付了房没过户、卖方拿钱跑路,这是二手房交易最大的资金风险敞口;跳过步骤 3 的合规性 → 走首付贷/假流水,后果是抽贷、征信污点、法律责任;跳过步骤 4 的「谁承担税费」约定 → 卖方应缴的增值税/个税被默默转嫁,实际成本远超报价;跳过步骤 5 顺序(先放款后办抵押)→ 资金风险错配。
- **资深路径（差异点）**:
  - **optimize**:资深人会**优化资金交付与过户的衔接顺序** —— 把「付款节点」严格绑定「过户/抵押节点」:首付进监管→网签→过户→抵押登记→放款,每一笔钱的释放都对应一个已完成的法律动作。带抵押的房优先走「带押过户」(本城本行支持时),省掉「买方垫资帮卖方解押」这个高风险环节(evidence: [T02-S013])。
  - **add**:资深人会**额外把全部税费、谁承担、各环节时间写进合同的明确条款** —— 增值税/个税名义是卖方税但市场惯例常转嫁,资深人在合同里白纸黑字写清「税费由谁承担、金额上限、逾期责任」,避免临到过户被加价。初学者口头约定,资深人合同锁死。
  - **skip**:资深人**绝不跳过资金监管去「省」点钱,也绝不碰任何「避税技巧」** —— 这不是 skip 而是反向的「绝不 skip」:阴阳合同避税、私下交易省监管费,在资深人这里是红线,不是「认知差」。资深人「skip」的是中介推销的灰色操作,守的是合规底线。
- **政策检查点**:
  - **网签/备案/资金监管制度**:核对本城网签流程、资金监管账户要求 —— 制度原文在地方住建局(evidence: [T02-S014, T02-S015])。
  - **契税/增值税/个税政策**:核对最新税率、首套二套差异、满五唯一/满二优惠 —— 各环节的钱(evidence: 06-glossary 契税条目)。
  - **贷款政策**:LPR + 本城本套加点 = 实际利率;首付比例下限、认房认贷口径 —— 决定贷款额度和月供(evidence: [T02-S003])。
  - **带押过户政策**:核对本城本行是否支持带押过户 —— 不支持时仍需走传统「解押再过户」(evidence: [T02-S013])。
- **风险检查点**:
  - **资金安全风险(交易核心)**:不走资金监管 = 「钱付了房没过户」「解押后卖方反悔」的巨大敞口(evidence: [T02-S007])。
  - **一房二卖风险**:未及时网签锁定,卖方可能重复出售。
  - **税费转嫁风险**:卖方税(增值税/个税)被默默转嫁,实际成本超预算 —— 合同未写清是主因。
  - **贷款批不下来风险**:贷款审批未过 → 交易卡住甚至违约 —— 出手前应预审贷款资格。
  - **红线风险**:任何「阴阳合同避税」「经营贷置换」「首付贷」「包装流水」—— 一律拒绝,后果是抽贷/征信/法律责任(glossary R1-R5)。
- **近期变化 / last_updated**:
  - 带押过户 2022-2023 起多地试点、2024-2025 加速推广 —— 带抵押二手房交易新增「带押过户」路径,大幅降低买方垫资风险(evidence: [T02-S013],触发事件类型:标准更新)。
  - 2024-2025 LPR 多次下调、首套首付降至历史低位、存量房贷利率批量下调 —— 「算账/测算月供」环节必须用当期 LPR + 自己合同的加减点 + 重定价日(evidence: [T02-S003],触发事件类型:法规变化)。
  - last_updated: 2026-05-14。
- **典型耗时**:入门 SOP 约 1-3 个月(网签到放款全流程,贷款审批是主要变量);资深路径相近,流程时长主要由银行和登记中心节奏决定。
- **关键工具(与 Track 02 关联)**:带资金监管的正规交易服务(贝壳/链家/正规地方中介)、不动产登记中心(过户)、央行 LPR + 地方房贷利率公告(贷款测算)、全国一体化政务服务平台(带押过户线上办理)。
- **关键人物(与 Track 01 关联)**:无特定 figure 集中谈交易流程 —— 这是实操链条,canon 关联点是「产权核查 → 网签备案 → 资金监管 → 过户」链条由 canon 概念直接组装(04-canon Track 03 协作 seed)。
- **常见失败模式**:
  - **为省中介费走私下交易、不用资金监管**:触发原因 = 觉得「认识的人不会骗」;怎么避免 = 任何交易都走资金监管账户,房款进监管、过户完成才放款。Don't: 为省几千元监管/中介费私下转账给卖方,卖方拿钱后房子因抵押/查封过不了户,钱房两空。
  - **税费谁承担只口头约定**:触发原因 = 嫌合同写太细;怎么避免 = 全部税费、承担方、上限、时间写进合同条款。
  - **碰中介推销的「避税/融资技巧」**:触发原因 = 被「能省一大笔」吸引;怎么避免 = 阴阳合同/经营贷置换/首付贷一律识别为红线、拒绝。
  - **没预审贷款资格就签约**:触发原因 = 默认「能贷下来」;怎么避免 = 出手前确认认房认贷口径和自己的贷款资格。
- **Decay risk**: medium —— 网签/资金监管/过户的核心流程稳定(low),但带押过户覆盖范围、LPR/首付/利率政策是高频变动项,整体 medium。

### W6. 持有与运营(出租 + 持有成本管理)

- **One-liner**: 从「拿到房、成为业主」到「建立一套『租金收入 - 持有成本』的持续运营机制」—— 投资房在持有期不是「躺着等涨」,而是要把净租金回报率和持有成本一笔笔管起来,尤其在资本利得不再可靠的 2024-2025。 (evidence: [T02-S003, T01-S015])
- **Trigger**: W5 交易完成、房产交付 / 已持有房产需要重新评估持有 vs 出售。
- **Output**: 一套「出租安排(租客/租金/租期)+ 年度持有成本台账 + 净租金回报率实测值 + 是否继续持有的定期复盘结论」。
- **入门 SOP(minimum viable steps)**:
  1. **决定出租策略**:长租(稳定、省心、回报率低)还是短租(回报率潜在高、空置和管理成本高、受政策影响大)—— 多数投资房选长租。inputs: 房产位置/类型;outputs: 出租模式。
  2. **定价 + 找租客 + 签租约**:参照同板块同户型真实成交租金(不是挂牌租金)定价,通过平台/中介找租客,签规范租约(租期、租金、押金、维修责任、违约条款)。inputs: 周边租金行情;outputs: 已出租 + 租约。
  3. **建持有成本台账**:逐项记录持有成本 —— 月供利息部分、物业费、取暖费、维修、空置月份损失、资金占用机会成本、(试点城市)房产税。inputs: 各项支出;outputs: 年度持有成本。
  4. **算净租金回报率并定期复盘**:净租金回报率 =(年租金 - 年持有成本)/ 总投入;定期(至少年度)复盘「这笔投资还值不值得继续持有」,而不是默认一直拿着。inputs: 台账 + 租金;outputs: 净回报率 + 持有/退出建议。
  - **每一步如果跳过会发生什么**:跳过步骤 1 → 盲目做短租,遇上政策收紧或淡季空置,回报不如长租还更折腾;跳过步骤 2 的真实租金参照 → 定价过高长期空置,定价过低少收租;跳过步骤 3 → 不记台账,以为「有租金收入就是赚」,实则持有成本吃掉全部租金还倒贴;跳过步骤 4 → 从不复盘,在一个早该退出的标的上沉没多年。
- **资深路径（差异点）**:
  - **add**:资深人会**额外算「资金占用机会成本」并把它写进真实回报** —— 首付那笔钱如果不买房、放无风险理财能赚多少,这是持有房产的隐性成本。算上它之后,很多「有正租金」的房其实是负回报。初学者算「租金 > 现金支出」就满意,资深人算「净租金回报率 vs 无风险收益」(evidence: [T02-S003])。
  - **optimize**:资深人会**优化空置率的真实估算** —— 不按「满租 12 个月」算,而是按真实出租周期(换租客常有 1-2 个月空窗、淡季更长)算;租金现金流也按「会下行」而非「会涨」做保守假设。2024-2025 租金普遍承压,顺周期假设会高估回报。
  - **add**:资深人会**额外把「持有 vs 卖出」做成定期决策而非默认动作** —— 把这套房放进家庭整体资产负债表,定期问「占比是否过高、流动性是否够、有没有更好的去处」,而不是「买了就一直拿」。这是把 W6 和 W7(退出)主动衔接起来。初学者持有是惯性,资深人持有是每期重新选择的结果(evidence: [T01-S015])。
- **政策检查点**:
  - **租赁政策**:本城租购并举、租赁备案、租金指导等政策 —— 影响出租合规和短租可行性。
  - **房产税试点政策**:上海/重庆已试点、全国立法推进中 —— 持有成本台账要为房产税扩围预留弹性(evidence: [T03-S002])。
  - **LPR/重定价政策**:存量房贷利率随重定价日按当期 LPR 调整 —— 月供利息这项持有成本会变(evidence: [T02-S003])。
  - **短租相关政策**:部分城市对民宿/短租有备案或限制要求 —— 做短租前核查。
- **风险检查点**:
  - **空置风险**:租不出去或长期空窗 —— 板块没有租赁需求(产业/人口支撑不足)是根因,在 W1/W2 就该排除。
  - **租金下行风险**:2024-2025 多地租金承压,租金现金流不能按「会涨」假设。
  - **持有成本侵蚀风险**:物业/维修/空置/资金占用机会成本逐项加起来,可能吃掉全部租金 —— 不记台账就看不见。
  - **持有惯性风险**:从不复盘、默认一直持有,在该退出的标的上沉没。
- **近期变化 / last_updated**:
  - 2024-2025 资本利得不再可靠 + 多地租金承压 —— 持有期运营重点从「等升值」转向「老老实实算净租金回报率 + 控持有成本」(触发事件类型:行业事件)。
  - 房产税试点 + 潜在扩围 —— 「持有成本测算」需为持有环节税预留弹性(evidence: [T03-S002],触发事件类型:法规变化)。
  - last_updated: 2026-05-14。
- **典型耗时**:入门 SOP 出租安排约 2-6 周,持有成本台账是持续性低强度工作;资深路径相近,资金占用核算和定期复盘是额外的轻量动作。
- **关键工具(与 Track 02 关联)**:自建持有成本/回报率 Excel(必备)、贝壳/找房平台(租金行情参照)、央行 LPR(月供利息测算)。
- **关键人物(与 Track 01 关联)**:香帅《钱从哪里来》把房产放进家庭整体资产负债表、强调流动性与占比 —— 直接对应本 workflow「持有是每期重新选择」的视角。
- **常见失败模式**:
  - **「有租金就是赚」的错觉**:触发原因 = 不记完整持有成本台账;怎么避免 = 逐项记账并算上资金占用机会成本,净回报率为负就是亏。Don't: 看到每月有租金到账就觉得这房「在帮我赚钱」,没算物业/空置/首付的机会成本,实际年年净亏。
  - **按满租算回报**:触发原因 = 忽略换租空窗;怎么避免 = 按真实出租周期(留 1-2 个月空置)算。
  - **持有变惯性**:触发原因 = 「买都买了」;怎么避免 = 至少年度复盘持有 vs 退出,主动衔接 W7。
  - **盲目做短租**:触发原因 = 被短租高回报吸引;怎么避免 = 核查本城短租政策 + 真实空置/管理成本,多数投资房长租更稳。
- **Decay risk**: medium —— 出租运营和算账框架稳定(low),但租金行情、房产税进展、LPR/重定价是 12 个月内会变的项,整体 medium。

### W7. 退出与流动性(卖出时机判断 + 挂牌成交 + 止损决策)

- **One-liner**: 从「决定要卖 / 该不该卖」到「真正成交拿到钱」—— 这是国内房产投资里最被低估、2024-2025 最残酷的一环:房产是低流动性资产,挂出去可能 12 个月卖不掉或必须大幅折价,「想卖」不等于「卖得掉」。 (evidence: [T05-S010, T05-S013, T01-S015])
- **Trigger**: 需要变现(资金需求/资产再配置)/ 持有复盘判断该退出 / 城市或板块基本面恶化 / 现金流压力。
- **Output**: 完成出售并拿到回款,或得出「以可接受价格无法在可接受时间内卖出」的明确结论(进而触发降价 / 继续持有 / 止损的二次决策)。
- **入门 SOP(minimum viable steps)**:
  1. **判断流动性环境(先看卖不卖得掉)**:用贝壳研究院看目标城市/板块的二手房挂牌量(暴增=供给堆积)、成交周期(从挂牌到成交几个月)、带看量、业主调价比例;用易居研究院看存销比 —— 先确认「这个市场现在能不能卖」,而不是先想「能卖多少钱」。inputs: 研究院流动性数据;outputs: 流动性环境标签。
  2. **定一个「能成交」的挂牌价**:参照同板块同户型的真实网签成交价(不是别人的挂牌价)定价 —— 下行期挂「卖家心理价」等于不卖。定价要对标成交,不对标挂牌。inputs: 网签成交价;outputs: 挂牌价。
  3. **挂牌 + 观察带看反馈**:多平台挂牌,观察带看量 —— 挂了一段时间带看个位数 = 价格虚高或有硬伤,带看多不成交 = 价格还在博弈区间。inputs: 平台挂牌;outputs: 带看反馈。
  4. **根据反馈做降价 / 继续持有 / 止损决策**:带看冷清就要么降价对标成交、要么承认「现在卖不掉」继续持有;若是现金流压力驱动的卖出,要算清「降到能成交的价 = 亏多少」,做止损决策 —— 止损是承认判断错误,不是失败。inputs: 带看反馈 + 财务约束;outputs: 降价/持有/止损结论。
  5. **成交 + 走资金监管完成交易**:接受报价后,同样走网签→资金监管→缴税→过户→收款的流程(W5 的镜像,这次你是卖方)。inputs: 买方;outputs: 回款到账。
  - **每一步如果跳过会发生什么**:跳过步骤 1 → 不看流动性环境就挂牌,在「成交冰封」的市场里干等,白白错过时间;跳过步骤 2 → 按挂牌价/心理价定价,挂一年零带看;跳过步骤 3 → 不看带看反馈,不知道是价格问题还是房子问题,无法调整;跳过步骤 4 → 在「该止损」时死扛「等回本」,流动性继续恶化、亏得更多;跳过步骤 5 的资金监管 → 卖方一样有「过户了没收到钱」的风险。
- **资深路径（差异点）**:
  - **optimize**:资深人会**优化「流动性优先于价格」的判断顺序** —— 入门者卖房先问「能卖多少钱」,资深人先问「这个市场现在卖得掉吗、卖一套要多久」。他们把成交周期、挂牌量、带看量当成比价格指数更早、更真实的信号,在流动性还没崩时就动手,而不是等价格跌明显了才想卖(那时已经卖不动)(evidence: [T05-S010])。
  - **add**:资深人会**额外把「止损」作为一个正式的、有触发线的决策,而不是情绪上的回避** —— 在买入时(W4)就想清楚「什么情况下我会卖、能接受亏多少」,持有期定期对照。该止损时按计划执行,把「沉没成本」和「未来现金流」分开算 —— 「等回本」是用更大的流动性风险赌一个不确定的反弹。初学者把止损当失败、无限期回避,资深人把止损当纪律(evidence: [T01-S015])。
  - **skip**:资深人会**跳过「等绝对高点 / 等市场反转再卖」的择时幻想** —— 与 W1 的「跳过抄底」对称:经验和跨源共识告诉他们「没有工具能可靠预测顶和底」。他们按「自己的资金需求 + 标的基本面 + 当前流动性」决定卖不卖,不按「赌一个更好的卖点」决定。
- **政策检查点**:
  - **限售政策**:目标房产是否还在限售锁定期内 —— 没满限售年限根本不能卖(evidence: 06-glossary 限售条目)。
  - **卖方税费政策**:增值税/个税(满五唯一/满二影响)—— 直接影响到手净额,要在定价时算进去。
  - **二手房参考价政策**:有参考价的城市,买方可贷额度受参考价限制,会影响买方出价能力和成交难度(evidence: [T02-S015])。
  - **顶层政策风向**:中国政府网顶层会议涉房定调(「止跌回稳」等)是政策周期的领先指标,但「政策托底 ≠ 你的房能卖掉」—— 读政策判方向,不赌政策救场(evidence: [T02-S005])。
- **风险检查点**:
  - **流动性风险(退出核心,2024-2025 最致命)**:挂牌量暴增 + 成交周期拉长 = 可能 12 个月卖不掉,「想卖」≠「卖得掉」(intake 警示 #8)。
  - **折价风险**:能成交的价可能远低于心理价、甚至低于买入价 —— 流动性折价是真实的。
  - **死扛「等回本」风险**:在该止损时无限期回避,用更大流动性风险赌不确定反弹。
  - **择时幻想风险**:等「绝对高点」「市场反转」再卖 —— 没有工具能可靠预测,等待中流动性可能进一步恶化。
  - **限售/税费风险**:没满限售期不能卖;卖方税费没算进定价导致到手缩水。
- **近期变化 / last_updated**:
  - 2024-2025 二手房流动性显著变差 —— 挂牌量暴增、成交周期从 3 个月拉到 12 个月甚至更长,「卖不掉」成为退出环节的核心现实。退出 workflow 必须正视「可能卖不掉或大幅折价」,不能沿用「房产随时能变现」的旧假设(evidence: [T05-S010, T05-S013],触发事件类型:行业事件)。
  - 2024 多城取消/放宽限购限售 —— 部分锁定期被解除,但「能挂牌」不等于「卖得掉」,政策放松没有改变流动性变差的基本面(触发事件类型:法规变化)。
  - last_updated: 2026-05-14。
- **典型耗时**:入门 SOP 在 2024-2025 下行期可能 6-18 个月甚至更长(成交周期是最大不确定项);资深路径因「流动性优先、早动手」可能略短,但下行期没有人能保证快速退出。
- **关键工具(与 Track 02 关联)**:贝壳研究院(挂牌量/成交周期/带看/调价比例)、易居研究院(存销比/去化周期)、各地房管局网签数据(真实成交价定价锚)、带资金监管的交易服务。
- **关键人物(与 Track 01 关联)**:香帅强调房产是低流动性资产、家庭配置里要单独算流动性;李迅雷谈需求透支导致成交难;真叫卢俊团队对小中介生存困难的调查侧面反映成交困难(均作框架/现状参考)。
- **常见失败模式**:
  - **按心理价挂牌,在冰封市场干等**:触发原因 = 不接受「现在的市场价低于我的预期」;怎么避免 = 定价对标真实网签成交价,先确认流动性环境再挂牌。Don't: 按「我当年买入价 + 应得收益」定挂牌价,挂 12 个月零成交,期间市场继续下行,最后卖得比一开始就定合理价还低。
  - **该止损时死扛「等回本」**:触发原因 = 把止损当失败、回避面对亏损;怎么避免 = 买入时就设止损触发线,该执行时按纪律执行,沉没成本和未来现金流分开算。
  - **等「市场反转」再卖**:触发原因 = 择时幻想;怎么避免 = 按自己的资金需求和标的基本面决定,不赌不可预测的拐点。
  - **只看价格指数不看流动性指标**:触发原因 = 以为「价格没大跌就还能卖」;怎么避免 = 看挂牌量/成交周期/带看量,流动性崩塌早于价格下跌(evidence: [T05-S010])。
- **Decay risk**: high —— 流动性环境、成交周期、限售/限购政策都是 12 个月内会显著变化的项;2024-2025 流动性变差是当前最大变量,本 workflow 是首要刷新对象。

### W8. 政策跟踪(政策原文 → 影响判断 → 留余量)

- **One-liner**: 从「听到一个政策消息/传闻」到「确认政策原文 + 判断对自己的具体影响 + 留出余量」—— 中国房产是政策驱动 > 市场驱动,读政策是核心能力;这是一条贯穿 W1-W7 全程的横切工作流,不是某个环节的一次性动作。 (evidence: [T02-S005, T03-S001])
- **Trigger**: 看到政策传闻/自媒体解读 / 顶层会议召开 / LPR 报价日 / 准备买卖决策前的政策核对。
- **Output**: 一份「政策原文确认 + 对自己当前/计划中持仓的具体影响判断 + 已留出的余量/已调整的动作」。
- **入门 SOP(minimum viable steps)**:
  1. **回到政策原文**:看到任何政策消息,先回官方原文 —— 顶层定调看中国政府网(政治局会议/中央经济工作会议),全国性政策看住建部/央行,城市政策看地方住建局/房管局,房产税看全国人大/沪渝市政府。不把自媒体解读当政策本身。inputs: 政策传闻;outputs: 政策原文。
  2. **判断政策性质与层级**:是顶层定调(领先、抽象)还是地方落地文件(具体、可执行)?是松还是紧?适用范围(全国/本城/本区)?inputs: 政策原文;outputs: 政策性质标签。
  3. **判断对自己的具体影响**:把政策落到「我能不能买/首付几成/能不能卖/月供变不变/持有成本变不变」—— 抽象定调要等地方落地文件才能转成动作。inputs: 政策标签 + 自己的持仓/计划;outputs: 影响判断。
  4. **留余量,不赌政策**:政策可以急转(收紧或放松),决策时为「政策可能往反方向走」留余量 —— 不基于「政策一定会继续放松」或「一定会救市」下注。inputs: 影响判断;outputs: 调整后的动作 + 余量。
  - **每一步如果跳过会发生什么**:跳过步骤 1 → 把自媒体夹带立场的解读当政策,基于误读决策;跳过步骤 2 → 把顶层抽象定调当成可执行动作,或把 A 城政策套到 B 城;跳过步骤 3 → 知道「有政策」但不知道「对我意味着什么」,等于没读;跳过步骤 4 → 赌政策(赌放松回到炒房时代/赌政策救市),政策一急转就被套。
- **资深路径（差异点）**:
  - **optimize**:资深人会**优化政策的「分层阅读」** —— 顶层措辞变化(如从「房住不炒」到「止跌回稳」)当方向性领先指标看,地方落地文件当具体动作依据看,两者分开用。他们读顶层判方向、读地方判动作,不混为一谈,也不指望顶层一句话立刻变成柜台执行(evidence: [T02-S005])。
  - **skip**:资深人会**跳过几乎所有自媒体政策解读和「政策传闻」的情绪反应** —— 经验告诉他们解读会夹带立场、简化甚至误读,「传闻」在尘埃落定前不可行动。他们直接读原文、等正式文件,跳过中间的二手转述和情绪波动(evidence: 02-tools 决策树 Branch 7)。
  - **add**:资深人会**额外把「政策可能急转」做成决策前提而非意外** —— 「房住不炒」是政策定调,投机性买房有真实政策风险;限购/限贷/限售/房产税都是可随时启用的工具。资深人在每个买卖决策里预留「政策反向」的余量,不把「政策会一直对我有利」当默认。初学者把当前政策当永久背景,资深人把政策当随时会变的变量(evidence: intake 警示 #5)。
- **政策检查点**(本 workflow 本身就是「政策检查点」的方法论,这里列要持续盯的源):
  - **顶层风向**:中国政府网 —— 政治局会议(季度)、中央经济工作会议(年度)涉房定调原文(evidence: [T02-S005])。
  - **全国性政策**:住建部(保交楼/白名单/现房销售试点)、央行(LPR/利率下限/首付比例)(evidence: [T02-S003, T02-S004])。
  - **城市政策**:目标城市住建局/房管局公告栏(限购/限贷/限售/认房认贷/公积金/人才购房)—— 出新政即看原文(evidence: [T02-S014, T02-S015])。
  - **持有环节政策**:全国人大(房地产税立法进展)、沪渝市政府(房产税试点调整)(evidence: [T03-S002])。
- **风险检查点**:
  - **误读风险**:把自媒体夹带立场的解读当政策原文。
  - **赌政策风险**:基于「政策一定继续放松/一定救市」下注 —— 政策可以急转(intake 警示 #5)。
  - **层级错配风险**:把顶层抽象定调当可执行动作,或把别城政策套到本城。
  - **滞后风险**:等自媒体「解读完」再行动,解读常有偏差和滞后,错过窗口或被误导。
- **近期变化 / last_updated**:
  - 2016「房住不炒」→ 2023「认房不认贷」松绑 → 2024 多城取消限购 → 顶层「止跌回稳」表述 —— 调控哲学从「全面抑制」转向「因城施策、托而不举」;政策本身高频变动,「逐城逐时点查最新文件」成为固定动作(evidence: [T03-S001],触发事件类型:法规变化)。
  - 限购/限贷/限售、认房认贷口径、保交楼政策、学区/落户政策、LPR 报价值 —— 均为 12 个月内大概率再变的高 decay 项(evidence: 06-glossary 时效性信号,触发事件类型:法规变化)。
  - last_updated: 2026-05-14。
- **典型耗时**:入门 SOP 单次政策核对约 0.5-2 小时(读原文 + 判影响);作为持续性横切工作流,是低强度但需要纪律的常态动作。
- **关键工具(与 Track 02 关联)**:中国政府网/住建部/央行(政策原文)、地方住建局/房管局公告栏(城市政策)、全国人大/沪渝市政府(房产税)、央行 LPR 发布。
- **关键人物(与 Track 01 关联)**:黄奇帆对「长效机制/房住不炒制度落地」的体系性论述提供读政策的框架视角;任泽平「短期看金融」框架对应紧盯信贷与利率政策(用框架不背书)。
- **常见失败模式**:
  - **把自媒体解读当政策原文**:触发原因 = 原文难找/看不懂、解读省事;怎么避免 = 限购/限贷/限售/利率/参考价一律以官方公告原文为准,读到传闻先回官方渠道证实再行动。Don't: 看到公众号「XX 城要全面放开限购」就赶紧出手,结果是误读或传闻,正式文件根本不是那个意思。
  - **赌政策放松/救市**:触发原因 = 「政策不会让房价跌」;怎么避免 = 不赌政策,每个决策为政策反向留余量,「政策托底 ≠ 你的房保值」。
  - **顶层定调当执行动作**:触发原因 = 看到「止跌回稳」就以为市场已稳;怎么避免 = 顶层判方向、地方判动作,等地方落地文件。
  - **等自媒体解读完再动**:触发原因 = 依赖二手转述;怎么避免 = 政策出台即看原文,不等解读。
- **Decay risk**: high —— 工作流方法论本身稳定(读原文→判影响→留余量),但「当前有效的具体政策」是 12 个月内持续变动的,作为政策跟踪工作流必须高频刷新所跟踪的政策内容。

### W9. 自住 vs 投资决策分流 + 杠杆/断供应对(两条逻辑严格分开)

- **One-liner**: 从「我要不要买/已经买了房有压力」到「先把自住逻辑和投资逻辑分清,再设定杠杆纪律和断供应对预案」—— 这是一条「元决策 + 风险底线」工作流:自住看居住效用、投资看回报率/流动性/退出,两套逻辑不能混;杠杆纪律是硬规矩。 (evidence: [T01-S015, T01-S001, T03-S003])
- **Trigger**: 准备买房(自住还是投资还没想清楚)/ 已持有房产出现现金流压力 / 评估家庭杠杆水平 / 收入下降或失业风险上升。
- **Output**: 一份「自住 vs 投资的明确定性 + 对应的决策指标集 + 杠杆纪律(断供线在哪、现金流断了能撑几个月)+ 断供应对预案」。
- **入门 SOP(minimum viable steps)**:
  1. **先定性:这套房是自住还是投资**:买之前先回答「这是自住还是投资」—— 这决定用哪套逻辑,不能含糊。混合用途的也要拆出「自住部分」和「投资部分」分别评估。inputs: 购房意图;outputs: 用途定性。
  2. **【自住】按居住效用决策**:自住房看通勤、学区、居住舒适度、家庭需求、长期稳定性 —— 不用投资回报率绑架自住决策,「买得起、住得舒服、长期不折腾」是核心。inputs: 家庭需求;outputs: 自住决策。
  3. **【投资】按回报率/流动性/退出决策**:投资房走 W1-W7 完整弧线,看净租金回报率、流动性、退出可行性 —— 不用「反正能自住」掩盖投资亏损。inputs: W1-W7 输出;outputs: 投资决策。
  4. **设定杠杆纪律**:上杠杆前算清「断供线在哪、首付来源是不是 100% 自有资金、现金流断了能撑几个月」—— 而不是只算「最多能贷多少」。首付贷/经营贷置换/假流水是红线,不碰。inputs: 收入/负债/储备;outputs: 杠杆纪律线。
  5. **做断供应对预案**:万一收入中断,预案是什么 —— 动用储备金、与银行协商、必要时主动卖出(W7)止损,而不是等到断供。断供后果是房子被处置 + 征信污点 + 可能被追差额。inputs: 压力情景;outputs: 应对预案。
  - **每一步如果跳过会发生什么**:跳过步骤 1 → 自住投资混为一谈,用投资逻辑买了不宜居的房、或用「反正自住」掩盖了一笔失败投资;跳过步骤 2/3 的逻辑分离 → 决策指标用错,自住的舒适度需求和投资的回报率需求互相打架;跳过步骤 4 → 杠杆上到极限,房价/利率/收入一动就断;跳过步骤 5 → 收入中断时毫无准备,直接滑向断供。
- **资深路径（差异点）**:
  - **add**:资深人会**额外把每套房放进家庭整体资产负债表,算「房产占家庭资产的比例」** —— 中国家庭资产过度集中在房产是结构性风险,资深人会主动控制房产占比、保留流动性,而不是默认「all in 房产」。初学者一套套地看「这套划不划算」,资深人看「房产在我整个资产里是不是太重了」(evidence: [T01-S015])。
  - **add**:资深人会**额外把杠杆的「双向性」明确摆出来** —— 不只算「杠杆能放大多少收益」,同等地算「杠杆放大多少亏损 + 现金流断裂的概率 + 断供的后果」。下行周期里高杠杆是毁灭性的(行业反面案例:水库系高杠杆技术流让大量跟随者深套断供),资深人「留余量」优先于「放大收益」。初学者杠杆只听到「放大收益」,资深人先看「能不能扛住放大的亏损」(evidence: [T01-S001],intake 警示 #3)。
  - **skip**:资深人会**跳过一切「教你绕过杠杆限制」的灰色路径** —— 经营贷置换房贷、首付贷、假流水、首付来源造假,部分「技术流大师」公开教这些。这不是「资深技巧」,是红线。资深人 skip 的是这些违规操作本身,守的是「首付必须是自有资金、杠杆必须在能扛住的范围内」的底线 —— 把灰色操作当「skip 项」是这一行最危险的认知错误(evidence: 06-glossary R1-R5)。
- **政策检查点**:
  - **认房认贷/限贷政策**:决定算首套还是二套、首付比例、利率 —— 直接影响杠杆水平(evidence: 06-glossary 认房认贷条目)。
  - **LPR/重定价政策**:存量房贷利率随重定价日调整,月供会变 —— 杠杆纪律线要按「利率可能上行」算(evidence: [T02-S003])。
  - **公积金政策**:公积金贷/组合贷的额度与认定 —— 合规融资渠道的政策依据。
  - ⚠️ 政策检查点不包括任何「如何利用政策漏洞绕过杠杆限制」—— 那是红线不是政策运用。
- **风险检查点**:
  - **自住投资混淆风险**:用投资逻辑绑架自住、或用「反正自住」掩盖投资亏损(intake 警示 #10)。
  - **高杠杆风险(硬规矩)**:杠杆放大亏损、现金流断裂、断供 —— 下行周期高杠杆是毁灭性的,必须和「放大收益」同等强调(intake 警示 #3)。
  - **断供风险**:收入中断 + 无储备 + 无预案 = 滑向断供,后果是房产被处置 + 征信污点 + 可能被追差额(evidence: [T03-S003])。
  - **家庭资产过度集中风险**:房产占家庭资产比例过高 = 中国式结构性风险,流动性和抗风险能力都差(intake 警示 #12)。
  - **红线风险**:首付贷/经营贷置换/假流水/首付来源造假 —— 后果是抽贷、征信、法律责任(glossary R1-R5)。
- **近期变化 / last_updated**:
  - 2024-2025 收入预期下行 + 房价下行 —— 杠杆的「双向性」从抽象警示变成大量人正在经历的现实(深套、断供案例增多),「杠杆纪律 + 断供预案」的权重显著上升(触发事件类型:行业事件)。
  - 2024 首套首付降至 15%-20% 历史低位、LPR 多次下调 —— 杠杆成本降低,但「能贷更多」不等于「该贷更多」,杠杆纪律反而更重要(evidence: [T02-S003],触发事件类型:法规变化)。
  - 经营贷置换房贷等灰色操作在监管收紧下风险暴露(银行抽贷案例)—— 红线的后果更具体(触发事件类型:法规变化)。
  - last_updated: 2026-05-14。
- **典型耗时**:入门 SOP 决策分流 + 杠杆纪律设定约 1-2 周(主要是想清楚和算账);资深路径相近,家庭资产负债表视角是额外的轻量但持续的动作。
- **关键工具(与 Track 02 关联)**:自建持有成本/回报率 Excel(必备,跑杠杆压力测试)、央行 LPR(杠杆成本输入)、央行征信中心(理解断供后果)。
- **关键人物(与 Track 01 关联)**:香帅《钱从哪里来》—— 家庭资产配置视角、房产占比与流动性;黄奇帆谈开发商负债率上限(杠杆纪律的制度侧印证);欧成效/水库论坛作为「高杠杆技术流让跟随者深套断供」的反面强警示样本(明确不背书,只作红线引用)。
- **常见失败模式**:
  - **用「反正自住」掩盖投资亏损**:触发原因 = 不愿承认投资判断错了;怎么避免 = 买入时就定性,投资房就按投资标准复盘,自住效用不能用来掩盖回报率为负。Don't: 一套明显是投资目的的房跌了,改口说「反正是自住」,从此不再正视这笔亏损也不做退出决策。
  - **杠杆只听「放大收益」**:触发原因 = 上行期叙事残留 + 大师话术;怎么避免 = 每次上杠杆同等算「放大的亏损能不能扛、现金流会不会断」。Don't: 因为「用足杠杆才聪明」上到极限,房价跌 + 收入降双击,现金流断裂滑向断供。
  - **把灰色操作当「资深技巧」**:触发原因 = 「技术流大师」包装成认知差;怎么避免 = 经营贷置换/首付贷/假流水识别为红线,首付必须自有资金。
  - **all in 房产、不留流动性**:触发原因 = 「房产最保值」的旧信念;怎么避免 = 把房产放进家庭资产负债表控占比、留流动性。
- **Decay risk**: medium —— 决策分流逻辑和杠杆纪律框架是稳态的(low),但认房认贷口径、首付比例、LPR、灰色操作的监管环境是 12 个月内会变的项,整体 medium。

### 近期工作流变化的根本原因（触发事件类型分布）

- **因「行业事件」(2021 后深度下行 + 开发商爆雷常态化 + 流动性变差)而变**:W1（城市分化成常态)、W2（板块分化加剧/远郊先崩)、W3（开发商爆雷新增尽调环节)、W4（估值转向算流动性折价)、W6（从等升值转向算净回报)、W7（流动性显著变差是退出核心现实)、W9（杠杆双向性成现实)—— **7 个 workflow**。这是主驱动力。

- **主要驱动力结论**:本行业工作流变化的根本原因是「**深度下行的市场现实 + 高频变动的政策**」双重驱动,几乎没有「技术驱动」的成分(与 02-tools 发现「工具创新来自监管不来自创业公司」一致)。Phase 2 识别外部驱动力时,应把「政策」和「2021 分水岭后的市场下行」作为贯穿 SKILL.md 心智模型与反模式的两大外部力量。

### 失败模式清单（跨 workflow 汇总 — 喂给 Phase 2.6 反模式节 + cli_writer 失败模式自检）

- **F1 用挂牌价/心理价当成交价**:估值买贵(W4)、挂牌零成交(W7)。Actionable 写法:定价/估值一律对标房管局网签真实成交价,挂牌价只看「卖家心理预期」。

- **F3 只算月供不算完整账**:以为划算实则年年净亏(W4/W6)。Actionable:自建 Excel 算买入+持有+收益+退出完整账并跑房价跌20%/租金跌30%/利率升1%三压力场景。
- **F4 在整城下行能级里找洼地 / 买睡城当洼地**:接整城下跌的刀(W1)、空置租不掉(W2)。Actionable:先用统计局指数确认城级方向 + 查人口流向 + 查产业支撑,人口流出/无产业不碰。
- **F5 把品牌开发商当烂尾免疫 / 为期房折扣承担烂尾全险**:遇爆雷房子烂尾首付打水漂(W3)。Actionable:查开发商当前债务/爆雷状态,下行期优先现房/准现房。
- **F6 把规划的「饼」算进估值**:溢价买入十年不兑现(W2)。Actionable:只把已出让土地、已开工配套算数,规划利好打折看。
- **F7 该止损时死扛「等回本」/ 等市场反转再卖**:用更大流动性风险赌不确定反弹(W7)。Actionable:买入时(W4)就设止损触发线,该执行时按纪律执行,沉没成本和未来现金流分开算。
- **F8 杠杆只听「放大收益」/ 把灰色操作当资深技巧**:高杠杆 + 双击滑向断供(W9);经营贷置换/首付贷后果是抽贷征信法律责任(W5/W9)。Actionable:每次上杠杆同等算放大亏损和现金流断裂;首付贷/经营贷置换/阴阳合同/假流水识别为红线、拒绝。
- **F9 把自媒体解读当政策原文 / 赌政策救市**:基于误读决策(W8);政策急转被套(W1/W8)。Actionable:政策一律以官方公告原文为准,读到传闻先回官方证实;每个决策为政策反向留余量。
- **F10 自住投资混为一谈 / 用「反正自住」掩盖投资亏损**:决策指标用错(W9);失败投资不被正视(W9)。Actionable:买入前先定性自住还是投资,投资房按投资标准复盘,自住效用不能掩盖回报率为负。

  1. **本 track 无 low decay workflow** —— 9 个 workflow 全部 medium/high decay,因为每个环节都挂在高频变动的政策和 2021 后深度下行的市场上。SKILL.md 工作流节预计是衰减最快的部分,强烈建议每 3 个月跑 update,重点刷新 W7(退出/流动性)和 W8(政策跟踪)。
  2. **「资深路径」在本行业的特殊形态**:本行业的资深差异不是「少做几步更快做完」,而是「换判断标准 + 把隐性成本/尾部风险显性化 + 跳过择时幻想 + 守住灰色操作红线」。Phase 2 接口统计差异类型时,会看到 add 类显著多于 skip 类,且 skip 类里混有「反向的绝不 skip」—— 这是「深度下行 + 强政策 + 利益相关噪音极大」行业的真实形态,不是数据异常。
  3. **本 track 的工作流是「严筛后的诚实弧线」,不是「买房致富指南」** —— 退出 workflow 正视「可能卖不掉/大幅折价」,杠杆 workflow 同等强调「放大亏损 + 断供」,所有 workflow 拒绝「闭眼买/抄底/灰色操作」。Phase 2 整合时绝不能为 SKILL.md「好看」而软化成旧叙事 —— 这个领域的错误工作流会让人亏掉一辈子积蓄甚至背债。
  4. **seed 来源说明**:本 track 在 Wave 2 启动,Wave 1 全部 5 个产出均已落盘,seed 充足(05-sources 的流动性指标提示、06-glossary 的「工作流变化触发种子」、02-tools 的决策树与避坑清单、04-canon 的「核心概念→候选 playbook」、01-figures 的框架派论述,合计远超 15-30 个 workflow 片段)。本 track 未做独立 web crawl,工作流步骤主要由 Wave 1 跨源信号 + intake 的 12 子领域弧线拼成 —— 若 Phase 1.5 review 发现某环节步骤需更细的一手工程描述(如某城市具体网签操作流程),可定向补充,但这不影响工作流骨架的可信度(骨架来自 gov 制度 + 跨源共识)。
