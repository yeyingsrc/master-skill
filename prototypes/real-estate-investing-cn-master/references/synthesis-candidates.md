| Pattern | 来源 |

| P-e 上杠杆先算「断供线在哪、现金流断了能撑几个月」,不只算「能贷多少」 | 04 概念12 / 03 W9 / 06 杠杆 |

## 候选工具流派分裂 (智识谱系)

| Split | A 派代表 | B 派代表 |
|-------|---------|---------|
| 「核心资产长期看好」vs「房地产高增长时代结束」(2024-2025 最大分歧) | 黄奇帆(止跌回稳)/李迅雷(一二线结构性短缺,均带限定词) | 香帅(微利社会)/马光远(最佳投资品时代结束)/人口学术研究 |
| 高杠杆技术流 vs 低杠杆稳健派(一种是红线、一种是正道) | 欧成效/水库系(反面强警示样本,绝不背书) | 黄奇帆(开发商负债率70%上限)/李迅雷(反对再加杠杆)/香帅 |
| 「一线核心永远涨」vs「人口拐点+老龄化=全面承压」 | 任泽平早期框架结论 | 李迅雷/城镇化人口研究 |
| 自住派(房住不炒)vs 投资派 | 房住不炒政策定调 / canon | 投资视角(本 skill 视角,但严格分流) |
| 「抄底」派 vs「现金为王、等企稳」派 | 子木一类自媒体喊单 | canon 不给定论 + intake 明确不教抄底 |
| 经济学家框架派 vs 自媒体喊单派 | 黄奇帆/李迅雷/香帅 | 真叫卢俊(相对克制现状样本)/子木(喊单样本) |
| 官方数据派 vs 中介平台派(信源口径) | 统计局+网签+不动产登记骨架 | 贝壳挂牌价/成交记录为主 |

## 候选 Agentic Protocol 维度

| 维度 | 推导自哪个 mental model |
|------|------------------------|
| D1 城市基本面研判(人口/土地/政策三查) | C1 没有全国房价 + C2 政策驱动 |
| D2 板块与流动性研判(城市内部分化 + 流动性指标) | C1 + C3 流动性优先 |
| D3 政策原文核查与「留余量」 | C2 政策驱动 > 市场驱动 |
| D4 产权与标的尽调(查档 + 一手/二手/法拍风险) | C14(降级 playbook,但反推为独立研究维度) |
| D5 完整持有账与杠杆压力测试 | C4 杠杆双向 + C13 算完整账 |
| D6 信源利益结构甄别(figure/大师/中介/自媒体) | C6 信源利益结构决定可信度 |
| D7 退出与流动性研判(卖得掉吗/止损线) | C3 流动性优先 + C22 止损是纪律 |
| D8 资产配置占比与自住/投资分流 | C5 自住与投资分开 + C7 房产占比 |
| D9 红线操作识别 | C11 灰色操作是红线 |

> 互斥性自检:D1 城市级 ≠ D2 板块级 ≠ D7 退出时点;D3 查政策外部变动 ≠ D6 甄别信源利益;D4 查产权事实 ≠ D5 算账;D8 配置层 ≠ D5 单笔算账;D9 识别红线 ≠ D6 甄别可信度(D9 是法律红线/二元,D6 是利益结构/程度)。9 个维度产出 9 个不同结论。最终取 D1-D9 全部(复杂行业 7-10 区间)。
