## 2. 核心子领域候选(Phase 1 调研维度)

1. **城市与地段选择** — 城市能级 / 人口流入流出 / 产业与就业 / 板块与地段逻辑 / 学区 / 规划。2021 年后城市间走势分化极大,「买对城市」是第一性问题
2. **政策与调控解读** — 限购 / 限贷 / 限售 / 认房认贷 / 房贷利率与 LPR / 公积金 / 落户 / 房产税试点。中国房产是政策驱动多于市场驱动
3. **一手房 vs 二手房** — 新房(预售 / 现房 / 烂尾与保交楼风险 / 开发商资质)、二手房(房龄 / 满五唯一 / 税费 / 中介 / 议价)
4. **房产金融与杠杆** — 商贷 / 公积金贷 / 组合贷 / 首付来源 / 利率 / 提前还贷 / 杠杆纪律。经营贷置换、首付贷等灰色操作单列为红线,不教
5. **估值与尽职调查** — 合理价格判断 / 小区楼栋楼层户型 / 周边配套与规划 / 产权核查(抵押查封产权瑕疵小产权房)/ 剩余产权年限
6. **交易流程与税费** — 网签 / 资金监管 / 过户 / 契税 / 增值税 / 个税 / 中介费 / 贷款流程
7. **持有与运营** — 出租(租金回报率 / 长租短租 / 租赁政策)、持有成本(物业 / 维修 / 空置 / 资金占用)
8. **退出与流动性** — 卖出时机 / 二手房挂牌量与成交周期 / 折价与止损决策。2024-2025 流动性显著变差是核心现实
9. **特殊类型** — 法拍房 / 商铺写字楼 / 商住公寓 40 年产权 / 旅居养老地产 / 小产权房,风险与普通住宅不同,逐类标注
10. **风险识别** — 烂尾与开发商爆雷 / 产权瑕疵 / 政策急转 / 三四线去库存 / 接盘陷阱 / 高杠杆断供风险
11. **房产在资产配置中的位置** — 房产占家庭资产比例 / 房产 vs 股债存款 / 流动性 vs 收益 / 中国家庭房产占比过高的结构性风险
12. **投资者的决策框架与心态** — 自住 vs 投资的严格区分 / 不被销售话术与「大师喊单」裹挟 / 长周期视角 / 杠杆纪律 / 不赌政策

## 6. 警示与边界(写入 SKILL.md 诚实边界 —— 本领域权重最高的部分)

1. **2021-2025 是中国房地产的深度下行期,不是「买房致富」的时代**。恒大 / 碧桂园等头部房企爆雷,多地房价下跌,二手房挂牌量暴增、成交周期拉长、流动性显著变差,烂尾楼与保交楼是真实问题。SKILL.md 绝不能沿用「房产永远涨」的旧叙事。
2. **经营贷置换房贷、首付贷、阴阳合同、假流水 等是违规甚至违法的灰色 / 黑色操作**。部分「技术流」大师公开教这些。必须明确标为红线,不教,并讲清后果(银行抽贷、征信、法律责任)。
3. **高杠杆在下行周期是毁灭性的**。「水库」式高杠杆技术流在 2021 年后让大量人深套甚至断供。不能只讲「杠杆放大收益」,必须同等强调「杠杆放大亏损 + 现金流断裂 + 断供」。
4. **房产投资「大师」大量是利益相关方**。卖课、卖咨询、和中介 / 开发商分佣、公开喊单只晒对不晒错。收录人物是为「呈现不同分析框架」,不是背书。警惕一切「跟我买就赚」。
5. **「房住不炒」是政策定调,投机性买房有真实政策风险**。限购 / 限贷 / 限售 / 房产税试点都是政策工具,可以急转。不能教「赌政策」,要教「读政策、留余量」。
6. **城市与地段分化极大,没有「全国房价」这回事**。一线核心 vs 三四线 vs 人口流出城市,走势可能完全相反。任何笼统判断都是错的。
7. **烂尾风险真实存在,买预售房 ≠ 安全**。开发商资金链、保交楼进度、网签备案、资金监管账户都要核查。
8. **房产是低流动性资产**。急用钱时可能卖不掉或大幅折价,2024-2025 尤其明显。资产配置里必须单独算流动性。
9. **产权 / 法律尽调是不可跳过的硬环节**。抵押、查封、产权瑕疵、小产权房、商住房 40 年产权、学区政策变动 —— 跳过尽调等于踩雷。
10. **自住和投资是两回事,不能混**。自住看居住效用,投资看回报率 / 流动性 / 退出。不能用投资逻辑绑架自住决策,也不能用「反正自住」掩盖投资亏损。
11. **房产税等持有环节政策仍在演进**。上海 / 重庆试点,全国推进节奏不确定。长期测算要留余量。
12. **家庭资产过度集中在房产是中国式结构性风险**。SKILL.md 应有「房产在资产配置中合理占比」的视角,而不是默认「all in 房产」。
