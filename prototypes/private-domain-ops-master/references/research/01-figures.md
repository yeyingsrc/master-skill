## Figures 详细卡片

### 1. 徐志斌

- **One-liner**: 见实科技 CEO，把「社交流量 / 小群效应」从数据研究做成一整套私域社群方法论 —— 私域「媒体派 / 操盘手派」的奠基者，靠「私域大调研 + 年度大会」持续输出一手案例。(evidence: [T01-S002], [T01-S003], [T01-S008])
- **核心身份**: 见实科技 CEO / 创办人之一（私域专门媒体 + 见实年度私域大会）；曾在腾讯任职 8 年（微博、微信生态相关），后任新浪微博、微播易副总裁。注：intake 把见实负责人写成「林军」是错的 —— 林军是见实联合创办人，但对外发声 / 出书 / 做年度趋势演讲的是徐志斌。
- **代表作品**:
  - 《社交红利》（2013，中信出版社）—— 社交流量怎么从社交网络里「免费」拿
  - 《小群效应：席卷海量用户的隐性力量》（2017-2018，中信出版社）—— 私域社群运营最常被引用的上游读物
  - 《即时引爆》（社交传播机制）+ 见实「私域大调研」年度趋势报告
- **值得读 / 听 / 看的 3 件事**:
  - 📖 《小群效应》豆瓣 entity 页: https://book.douban.com/subject/27168595/ （evidence: [T01-S007]）—— 研究了 QQ 音乐 / 王者荣耀 / 狼人杀等 69 个国民级产品 + 腾讯/百度/豆瓣数据
  - 🎙️ ⚠️ 未找到稳定的长 podcast 专访（徐本人主阵地是见实公众号 + 大会，公众号黑名单不收录）
  - 🎬 见实年度私域大会徐志斌「私域大调研」趋势 keynote（每年底，2024 场主题「硬核增长」）—— 一手洞察密度最高的公开场合；2017 全球产品经理大会讲者页 evidence: [T01-S008]
- **核心思想关键词** (3-5 个): 小群效应（4-6 人小群活跃度 / 传播效率最高）、社交红利、连接 vs 内容 vs 工具（社群三大驱动力）、私域大调研（年度行业一手数据）、关系链传播
- **voice_samples**:
  - 客户解释样本: ⚠️ 暂未找到 ≥ 30 字直接原话片段（徐的长内容主阵地是见实公众号，黑名单不收录；书籍原文不存）
  - 同业讨论样本: 转述其核心论断 ——「用户在 4-6 人的小群里最活跃，大群是死的、小群才有传播力」（source: T01-S003，转述自《小群效应》核心论点）
  - 监管 / 专业讨论样本: 转述其私域定义 ——「私域指企业能自主收集信息、完全掌控、可持续优化升级的用户群；公域是不在企业掌控里的流量和数据」（source: T01-S003，转述）
- **sub_skill_candidate**: `true` —— 有《小群效应》《社交红利》两本可作长素材的出版书 + 见实年度趋势演讲，思想体系（小群效应 / 社交红利）自洽且原创，是「媒体 / 操盘手派」的奠基人，与 focus（operational）对齐。⚠️ 但长访谈类材料偏弱，sub-skill 蒸馏需主要依赖书籍。
- **dual_role**: `"founder + thinker"`（见实科技 CEO + 私域社群方法论原创者）
- **最近 12 个月动态**: 见实持续运营，2024-12 办年度私域大会「硬核增长」场；2025-2026 见实从纯媒体扩展为「大会 + 数据 + 私享会」组合，徐志斌仍以「私域大调研」做年度趋势输出。

### 2. 鉴锋

- **One-liner**: 零一裂变 / 零一数科创始人，把「微信生态裂变玩法」做成可复用的方法论和 to-B 代运营服务 —— 私域「增长 / 工具 / agency 派」的代表，4 年从草根团队做到近 700 人公司。(evidence: [T01-S009], [T01-S010], [T01-S011])
- **核心身份**: 零一数科（原零一裂变）CEO。公司定位「O+O 全链路私域数字化运营服务商」，业务从「裂变玩法开发」（2018 起，最早服务教育行业）→「私域代运营」→「视频号直播代播」一路演进。
- **代表作品**:
  - 「微信生态裂变设计」方法论（任务宝 / 分销 / 拼团类玩法的工程化）
  - 视频号「FOCUS 模型」+ 各行业「O+O 增长体系」（美妆 / 鞋服 / 餐饮连锁 / 美业大健康 / 快消）
  - 鉴锋组织的「私域大咖连麦分享会」（多人对谈私域趋势）
- **值得读 / 听 / 看的 3 件事**:
  - 📖 中国日报网零一数科署名稿「快消零售行业：如何利用微信生态重做全域消费者运营」: https://cn.chinadaily.com.cn/a/202403/07/WS65e957caa3109f7860dd4654.html （evidence: [T01-S011]）
  - 🎙️ ⚠️ 未找到稳定的长 podcast 专访（鉴锋主阵地是「鉴锋」公众号 + 行业连麦，公众号黑名单不收录）
  - 🎬 鉴锋组织的私域大咖连麦分享（行业趋势多人对谈）；数英长稿「创业4年零一从草根到700人」: https://www.digitaling.com/articles/801198.html （evidence: [T01-S009]）
- **核心思想关键词** (3-5 个): 裂变玩法工程化（把裂变做成可复用产品而非一次性活动）、O+O（线上线下全链路）、视频号 FOCUS 模型、私域代运营（agency 视角的精细化 SOP）、单用户月均消费提升（私域 ROI 抓手）
- **voice_samples**:
  - 客户解释样本: ⚠️ 暂未找到 ≥ 30 字直接原话片段（鉴锋长内容在「鉴锋」公众号 + 付费连麦，均黑名单 / 半闭源）
  - 同业讨论样本: 转述其 agency 实操观点 ——「裂变不是做一场活动，是把一套玩法做成可复用的产品，每个行业线套不同的 O+O 增长模型」（source: T01-S009 + T01-S011，转述）
  - 监管 / 专业讨论样本: 转述其业务演进逻辑 ——「裂变玩法拉新成本最低，但保鲜期短、合规风险高，所以零一一路从裂变开发往私域代运营、视频号代播转」（source: T01-S009，转述/推断）
- **sub_skill_candidate**: `false` —— 重要（增长 / agency 派代表），但鉴锋的影响力主要靠**零一数科这家公司的工程产品 / 代运营服务**输出，而非个人成体系的思想著作；无出版书、无长访谈，长素材严重不足，不够独立成 sub-skill。
- **dual_role**: `"founder + operator"`（零一数科 CEO + 一线裂变 / 代运营操盘）
- **最近 12 个月动态**: 业务重心 2023-2024 转向视频号代运营 / 代播；2024 起以「零一数科」品牌对外（原「零一裂变」），强调 O+O 全域；公司规模近 700 人。
- **争议 / 批评**: 裂变玩法类内容**保鲜期 3-6 个月**（微信对裂变屡次合规收紧，鉴锋早期赖以起家的「个人号裂变」打法已大面积过时）；agency 模式天然带「卖服务」导向，其方法论输出需对冲商业目的看；「单用户消费提升 7 倍」类数字按行业惯例需脱水。

### 显著分歧 / 流派分裂

intake warnings 识别的「五大流派」在 figures 层面得到验证，且 figures 之间有真实的、给完全相反建议的分歧：

- **增长 / 裂变 / 工具派**（代表：鉴锋、范冰<上游>）vs **平台 / 合规派**（代表：腾讯企微官方）
  → **核心分歧：裂变玩法的边界**。增长派把裂变当核心引擎、追求拉新成本最低；平台派视激进裂变为「诱导分享红线」要封号。同一个「任务宝该不该做」给完全相反答案。
- **复购 / 会员（品牌）派**（代表：晏涛、完美日记·名创优品）vs **社群 / 操盘手付费社群派**（代表：刘思毅）
  → **核心分歧：私域是「品牌长期资产」还是「销售放大器」**。品牌派要慢养信任、终点是「超级用户」；操盘手/IP 派要快变现、强「搞钱」直给。同一个「私域话术该多激进」给相反建议。
- **媒体 / 操盘手派**（代表：徐志斌）vs **运营通识派**（代表：黄有璨）
  → **较弱分歧：私域是「需要持续拆解标杆案例的专门赛道」还是「互联网运营整体框架里的一段」**。徐志斌靠「私域大调研 + 年度大会」持续输出一手案例，认为私域无固定真理；黄有璨认为私域脱离运营整体框架单看是舍本逐末。
- **工具能多大程度替代人**：鉴锋（agency 视角，信「精细化 SOP + 代运营」可规模化）vs 刘思毅/徐志斌（信「私域核心是人对人 / 小群里的真实关系」）。
