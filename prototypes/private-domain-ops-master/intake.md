## 1. 行业定义

**中文**:微信私域运营 — 在微信生态(企微 + 个人号 + 公众号 + 视频号 + 小程序 + 朋友圈 + 社群)做用户引流 → 沉淀 → 1v1 / 1vN 触达 → 转化 → 复购 → 裂变的全链路操盘。

**English**:WeChat Private-Domain Operations — operator role in WeChat ecosystem (WeCom / personal accounts / Official Accounts / Channels / Mini Programs / Moments / groups), spanning acquisition → consolidation → engagement → conversion → repeat purchase → viral, distinct from public-traffic ops.

**Industry type**:`platform_specific_practitioner`
**Locale**:`zh-CN`(海外无对应物,WhatsApp/LINE/Discord 概念接近但生态不同)
**Profile**:`practitioner`(私域操盘手 / 社群运营 / SCRM 运营 / CRM 操盘手)
**Focus**:`operational` 为主 + `business` 为辅

## 2. 核心子领域候选(Phase 1 调研维度)

1. **公转私漏斗** — Douyin / 小红书 / 视频号 → 企微 / 个人号 / 公众号 承接
2. **沉淀场选择** — 个人号 vs 企微 vs 公众号 vs 群(规则 / KPI / 工具差异)

7. **小程序商城** — 私域承接交易闭环(微盟 / 有赞 / 自研)
8. **转化与销售** — 1v1 销售 vs 群秒杀 vs 直播带货 vs 分销裂变
9. **复购 + 会员体系** — RFM / 标签 / 等级 / 积分 / 储值 / 老带新
10. **裂变玩法** — 任务宝 / 拼团 / 砍价 / 邀请有礼(注意合规红线)
11. **数据与归因** — SCRM 数据 / GMV 归因 / LTV / 标签体系
12. **合规与封号风险** — 微信规则 / 个人号自动化红线 / 企微 vs 个人号迁移

## 6. 警示与边界(写入 SKILL.md 诚实边界)

1. **微信规则极严且频繁变**:个人号自动化(自动加好友 / 自动回复 / 多开)2023-2025 屡次大封,企微也对裂变 / 数据真实性收紧。**玩法保鲜期 3-6 个月**。
2. **信息严重不平衡**:全中文,海外无对应物(WhatsApp Business / LINE OA / Discord 概念接近但生态完全不同),强行迁移会失败。
3. **营销课程占比极高(估 70%+)**:「私域操盘手月入 5 万」「7 天搭建私域」满天飞。研究时狠筛课程导流。
4. **「私域」一词被滥用**:有的指企微社群,有的指公众号粉丝,有的指自营 APP 用户。SKILL.md 必须先定义清楚。
5. **学派分歧大**,同问题不同派给完全不同建议:
   - 品牌派(慢但稳)
   - 销售派(1v1 销售放大器,激进话术)
   - 内容派(公众号 / 视频号驱动,慢热低成本)
   - 工具派(SCRM + 自动化 + 数据驱动一切)
   - 社群派(KOC 裂变 + 群活跃度)
6. **暗黑面广**:刷量 / 假人 / 加好友机器人 / 数据造假 / 假裂变 / 假分销;研究时严格区分白帽 / 黑帽 / 灰帽。**SKILL.md 默认走白帽 + 标注哪些是灰区**。
7. **ROI 不透明**:头部品牌(完美日记 / 名创优品 / 屈臣氏)公开私域 GMV 多被认为有夸大;中小商家私域 ROI 真实数据极少公开。
8. **「个人号 vs 企微」战略已定局**:2024 起企微 default,SKILL.md 不要给「个人号无脑加好友」式过时建议。
9. **头部腰部断层**:头部品牌 / KOC 私域 GMV 占大头,中小商家私域常做不起来。**SKILL.md 要诚实写「不适合 70% 商家」+「需要什么前置条件」**。
