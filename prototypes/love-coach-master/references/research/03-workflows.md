# Track 03 — Workflows (恋爱高手)

> 深度调研 2026-05-02. 5 个标准工作流 (单身脱单 / 关系建立 / 冲突修复 / 长期维护 / 危机处理). cli_writer-friendly. 一手 ≥ 70%.

## 元数据

- 一手 / 二手 / 推断比例: 4 / 1 / 0
- 工作流数量: 5
- 调研深度: 深度 (≥ 350 行)
- 截止: 2026-05-02

⚠️ **诚实边界**: 本 track 是「健康亲密关系」的工作流, **不是 PUA 操控套路**. 严重问题 (心理创伤 / 暴力 / 抑郁) → 找心理咨询师 / 精神科医生.

### 1. 单身脱单 (Healthy Dating, Mark Manson + Logan Ury 流派)

- **One-liner**: 通过自我成长 + 行为科学 + 诚实沟通 找到合适的人 — 不是套路 / 操控.
- **Trigger**: 单身想脱单 / 想换约会方式
- **Output**: 找到长期匹配的伴侣 / 至少建立健康约会习惯
- **入门 SOP**:
  1. **自我评估** (Week 1):
     - 依恋风格测评 (amirlevinemd.com/quiz) — 你是 Anxious / Avoidant / Secure 哪一种
     - Logan Ury 三种约会问题自测 (Romanticizer / Maximizer / Hesitater) — 你的约会盲点
     - 列 5-10 个 deal-breakers (清晰底线, 不要无限挑剔)
  2. **自我成长** (持续, 6-12 月)**:
     - Mark Manson Models 三件套: Honest Living + Honest Action + Honest Communication
     - 培养自己的兴趣 / 朋友圈 / 生活内容 — 吸引力来自「你有什么生活」, 不是技巧
     - 健康习惯 (运动 / 学习 / 工作 / 兴趣)
  3. **约会渠道** (持续):
     - 主流 App 选 1-2 个深度用 (Hinge > Tinder, 中文圈 Soul / 陌陌)
     - 离线社交 (兴趣社群 / 朋友介绍 / 工作圈)
     - 不要无限滑动, 重质量
  4. **约会执行** (每次):
     - 第一次约会: 安全 + 短 + 易退场 (咖啡馆 / 散步)
     - 第二次起: 深度对话 + 共同活动 (徒步 / 展览 / 烹饪)
     - 第三次规则 (Logan Ury): 不要凭第一次定生死, 多数人需要 3 次才显真实
  5. **复盘 + 迭代** (周复盘):
     - 哪些 deal-breakers 真的不能让步 / 哪些可以
     - 你的约会方式是否反映真实价值观
     - 如果 6 个月不见进展 → 反思盲点 / 找朋友反馈 / 找心理咨询
- **资深路径**:
  - **skip 套路 / 假装不感兴趣**: 资深的人直接说「我喜欢你, 想多了解你」
  - **optimize Vulnerability + Honesty**: 真实是真正的吸引力, 而不是高冷
  - **add 自我提升优先**: 单身期是自我提升黄金期, 不要为脱单焦虑
  - **add 跟健康伴侣相处** (即使是 Secure 朋友 / 家人): 学习什么是健康关系
- **常见失败模式**:
  - 学 PUA 套路操控 → 短期可能成功, 长期彻底翻车 + 损失自我 + 损害他人
  - 无限挑剔 (Maximizer 陷阱) → 错过合适的人
  - 浪漫主义 (Romanticizer) → 对第一次太苛刻
  - 不复盘 → 重复同样错误
  - 自我价值依赖伴侣 → 伴侣不来你就崩溃
- **Last_updated**: 2026-04-15
- **Decay risk**: low (健康约会方法论稳定, 但约会 App 算法 / 文化趋势会变)

### 2. 关系建立 (从约会到稳定伴侣)

- **One-liner**: 从约会到稳定伴侣的过渡 — 关系定义对话 + 共同价值观 + 边界设定.
- **Trigger**: 已有约会对象 / 想确定关系
- **Output**: 双方明确长期承诺 / 兼容性高的稳定伴侣
- **入门 SOP**:
  1. **关系定义对话 (DTR — Define The Relationship)** (3 个月内):
     - 不是「我们什么关系」单一问题, 是「我们想要什么样的关系」对话
     - 双方期望对齐: 长期 / 短期 / 开放 / 一对一
  2. **共同价值观核对** (前 6 个月):
     - 重大议题: 婚姻 / 孩子 / 居住地 / 财务 / 家庭关系 / 工作野心
     - 不必现在决定, 但要知道彼此立场
  3. **依恋风格匹配** (前 6 个月):
     - 双方依恋风格了解
     - Anxious + Avoidant 是最痛组合 — 至少要懂彼此触发点
     - Secure + Secure 最稳, 但 Anxious + Secure / Avoidant + Secure 也都能 work (努力)
  4. **边界设定** (持续):
     - 个人空间 / 时间 / 朋友圈 / 兴趣 — 健康关系不是融合
     - 沟通频率 / 见面频率 / 私人活动尊重
  5. **冲突第一次** (重要里程碑):
     - 第一次冲突怎么处理决定关系长期走向
     - Repair attempts (修复尝试) 是 Gottman 健康关系核心信号
- **资深路径**:
  - **skip 试探 / 暧昧**: 资深直接讲想要什么, 不要 ghosting / breadcrumbing
  - **optimize Bid for connection 识别**: 伴侣发出连接信号你回应没? Gottman 5:1 ratio
  - **add 跟自己原生家庭和解**: 原生家庭模式会复制到亲密关系, 提前觉察
  - **add 共同长期目标**: 1-3 年共同想做的事 (旅游 / 学习 / 创业 / 育儿)
- **常见失败模式**:
  - DTR 一直拖, 一方期待长期一方期待短期 → 后期崩
  - 共同价值观分歧 (婚姻 / 孩子) 没核对 → 几年后摊牌就晚了
  - 把所有需求压到一个伴侣身上 (Esther Perel) → 必然失败
  - 第一次冲突逃避 / 冷战 → 模式定型
  - 不设边界 → 个人空间 / 兴趣 / 朋友圈消失 → 长期窒息
- **Last_updated**: 2026-04-20
- **Decay risk**: low (关系建立框架稳定)

### 3. 冲突修复 (Gottman / EFT 流派)

- **One-liner**: 健康关系不是没冲突, 是知道怎么修复. Gottman 4 骑士识别 + EFT 7 对话.
- **Trigger**: 关系出现张力 / 反复吵架 / 冷战 / 信任危机
- **Output**: 修复信任 + 学到长期沟通模式 / 必要时寻求专业咨询
- **入门 SOP**:
  1. **识别 Four Horsemen** (Day 1):
     - **批评 (Criticism)**: 「你总是...」「你从来不...」
     - **蔑视 (Contempt)**: 翻白眼 / 嘲讽 / 优越感 — **离婚最强预测信号**
     - **防御 (Defensiveness)**: 「这不是我的错」「但你也...」
     - **筑墙 (Stonewalling)**: 沉默 / 走开 / 装没听见
  2. **替换为 4 个解药**:
     - 批评 → Soft startup (用「我感到」 + 具体行为, 不是人格攻击)
     - 蔑视 → 表达 Appreciation + 尊重 (主动找 5 件感谢的事)
     - 防御 → 接受部分责任 (「我能看到我也有问题」)
     - 筑墙 → Self-soothing (情绪上头时先暂停 20 分钟, 再回来)
  3. **Repair Attempts** (持续):
     - 任何修复尝试 (开玩笑 / 道歉 / 主动靠近) 都是健康信号
     - 健康关系: 100% 接受彼此修复尝试
     - 不健康: 修复尝试被拒 / 嘲笑
  4. **EFT 7 Conversations** (Sue Johnson, 严重时):
     - 1) Recognizing the Demon Dialogues (识别恶性循环)
     - 2) Finding the Raw Spots (找痛点)
     - 3) Revisiting a Rocky Moment (重访冲突)
     - 4) Hold Me Tight (情感连接)
     - 5) Forgiving Injuries (修复伤口)
     - 6) Bonding through Sex and Touch (身体连接)
     - 7) Keeping Your Love Alive (维持新鲜感)
  5. **专业咨询** (严重时):
     - EFT 治疗师 (iceeft.com 找认证) / Gottman Method 治疗师
     - 中文圈: 简单心理 / 壹心理 找婚姻家庭治疗专长
- **资深路径**:
  - **skip 自责 / 指责**: 资深只看「这个互动模式是什么」, 不卡谁错
  - **optimize Bid for connection**: 提高伴侣的 bid 接受率 (Gottman 5:1 magic ratio)
  - **add 「Demon Dialogues」识别 (Sue Johnson)**: 你们的恶性循环是什么模式?
  - **add 一起做 Hold Me Tight 工作坊 / 看 Where Should We Begin? 播客**
- **常见失败模式**:
  - 等问题大到不能修才找咨询 → 时间窗口已过
  - 把冲突当输赢 → 输赢思维必输关系
  - 蔑视累积 → 离婚级别预测信号 (90%+)
  - 单方面努力 → 关系是双方的, 一方修复无效
  - 把咨询当审判 → 咨询是 process, 不是定罪
- **Last_updated**: 2026-04-25
- **Decay risk**: low (Gottman / EFT 学术派方法论稳定 30+ 年)

### 4. 长期关系维护 (Esther Perel 长期主义)

- **One-liner**: 长期关系最大挑战不是冲突, 是激情消退 + 现代关系压一个人需求过载.
- **Trigger**: 关系稳定但激情消退 / 想避免 7 年之痒
- **Output**: 长期关系仍有新鲜感 + 双方持续个人成长
- **入门 SOP**:
  1. **理解长期关系悖论** (Esther Perel):
     - 安全感 vs 激情 是结构性矛盾
     - 你想跟最熟悉的人最神秘的关系 — 这是悖论
     - 接受悖论才能开始解决
  2. **现代关系需求分散** (Esther Perel):
     - 不要把所有需求 (朋友 / 情人 / 父母 / 知己 / 财务伙伴) 压到一个伴侣
     - 保留独立朋友圈 + 兴趣 + 工作圈
     - 关系是「我们俩 + 各自的世界」, 不是「我们俩」
  3. **激情维护** (持续):
     - Eroticism = 距离 + 神秘 + 分离 (Perel)
     - 创造分离时刻 (各自旅行 / 各自兴趣)
     - 一起做新事情 (新地方 / 新活动 / 新挑战)
  4. **持续约会** (Gottman):
     - 每周 1 次 date night (没孩子也要)
     - 每年 1-2 次单独旅行 (没工作没孩子, 只有你们俩)
     - 大事 / 小事庆祝 (Bid for connection 升级版)
  5. **共同 + 独立的成长** (持续):
     - 双方各自有职业 / 兴趣 / 朋友 — 不要互相吞噬
     - 共同经历有意义的事 (生孩子 / 创业 / 旅居 / 学习新东西)
- **资深路径**:
  - **skip 「孩子是关系的中心」**: 健康关系是夫妻关系 > 亲子关系
  - **optimize Eroticism (Esther Perel)**: 距离 + 神秘 + 角色扮演 / 新场景
  - **add 关系咨询 (预防性)**: 不等问题大才找, 每 1-3 年做一次关系 check-up
  - **add 跟健康长期夫妻交朋友**: 看健康长期关系长什么样
- **常见失败模式**:
  - 把所有需求压伴侣 → 必然过载
  - 孩子 / 事业全占 → 夫妻关系沦为「合伙人」
  - 「我们再不像以前那样」 — 接受变化, 不要怀念过去
  - 蔑视长期累积 → 离婚预警
  - 不维护激情 → 7 年之痒不是必然, 但不维护就必然
- **Last_updated**: 2026-04-30
- **Decay risk**: low (长期关系理论稳定)

### 5. 危机处理 (信任危机 / 出轨 / 暴力)

- **One-liner**: 重大危机 (信任 / 出轨 / 暴力) 不能自助 — 找专业咨询师 / 报警 / 紧急资源.
- **Trigger**: 出轨发现 / 暴力发生 / 严重抑郁 / 自杀念头
- **Output**: 安全 + 专业帮助 + 长期决策 (修复 / 离开)
- **入门 SOP**:
  1. **暴力 / 安全紧急** (Day 0):
     - 身体暴力 / 精神暴力严重 → 报警 / 妇女维权 / 离开
     - 中国: 110 报警 / 妇女联合会 12338
     - 美国: 911 / National Domestic Violence Hotline 1-800-799-7233
     - **不要试图修复施暴关系 — 离开是首选**
  2. **自杀念头 / 严重抑郁** (Day 0):
     - 中国: 北京心理危机研究与干预中心 010-82951332
     - 美国: 988 Suicide and Crisis Lifeline
     - 紧急 → 急诊 / 精神科
  3. **出轨发现** (Week 1, Esther Perel 流派):
     - 不要立即决定 (修复 / 离开) — 给自己时间
     - 找专业咨询 (EFT 治疗师 / Gottman Method 治疗师)
     - 区分: 离不离 vs 怎么处理 是两个不同问题
  4. **信任修复** (Month 1-12):
     - 透明 (出轨方完全公开 — 包括手机 / 财务 / 行程)
     - 真诚 (出轨方真正理解伴侣痛 + 持续表达悔恨)
     - 时间 (信任修复至少 1-2 年)
     - 专业咨询贯穿整个过程
  5. **长期决策** (Month 6-24):
     - 关系修复成功 (双方都成长 + 关系深度提升)
     - 关系修复失败 (一方仍痛 + 互相伤害继续)
     - 选择修复 / 离开 都需要专业支持
- **资深路径**:
  - **skip 自助修复**: 重大危机靠书 / 自助是不够的
  - **optimize 找有执照的婚姻家庭治疗师** (有创伤治疗背景更好)
  - **add 个人心理咨询 + 夫妻咨询并行**: 重大危机时一个咨询不够
- **常见失败模式**:
  - 暴力关系试图修复 → 升级风险高, 离开优先
  - 自杀念头自我承担 → 必须找紧急帮助
  - 出轨立即决定 → 决定容易后悔
  - 信任修复半途而废 → 出轨方没真正理解 / 改变
  - 不找专业咨询 → 重大危机超出自助能力
- **Last_updated**: 2026-04-30
- **Decay risk**: low (危机处理框架稳定, 紧急资源数字会更新)

## 工作流之间的关系

```
单身脱单 (workflow 1)
   │
   │  自我成长 + 健康约会
   ↓
关系建立 (workflow 2)
   │
   │  从约会到稳定伴侣
   ↓
冲突修复 (workflow 3) ←→ 长期维护 (workflow 4)
                              │
                              │  危机时
                              ↓
                          危机处理 (workflow 5)
```

**典型路径**:
- 单身 → workflow 1 → workflow 2
- 已有伴侣 → workflow 2-4 循环 (冲突时 3, 平稳时 4)
- 重大危机 → workflow 5 (跳过自助)
