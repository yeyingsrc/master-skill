# Track 03 — Workflows (恋爱高手)

> Mini-viable for prototype.

### 1. 单身期自我建设

- **One-liner**: 从「焦虑找对象」到「自己生活完整, 缘分自来」的过渡过程
- **Trigger**: 主动想脱单 / 刚分手 / 长期单身焦虑
- **Output**: 内核稳定 + 有 2-3 个独立爱好 + 主动社交渠道
- **入门 SOP**:
  1. 自我评估 — 列清楚自己「想要的关系长什么样」+「不可妥协 3-5 项」
  2. 反复模式诊断 — 看自己历史关系里有什么重复模式 (童年依恋反演)
  3. 自我建设 — 培养 2-3 个独立爱好, 强壮自己的生活
  4. 主动社交 — 每周至少 1-2 次新场景 (兴趣班 / 朋友聚会 / 社交 app)
  5. 真实记录 + 调整 — 每月写一次情感日记复盘
- **资深路径**:
  - skip 急于「找到对象」, 资深做法是放下结果, 享受过程
  - optimize 直接做依恋类型测试 (ECR-R), 不要靠模糊的自我感觉
  - add 心理咨询作为长期投资 (不等到关系出问题再做)
- **常见失败模式**:
  - 急着脱单 → 重复历史模式, 又遇到不合适的人
  - 把所有期待寄托在恋爱上 → 找到对象也维持不久
  - 不愿做自我评估 → 一直问「为什么我老遇到渣」
- **Last_updated**: 2025-09-01
- **Decay risk**: low

### 2. 关系冲突修复

- **One-liner**: 关系出现裂痕到重新建立连接的标准动作
- **Trigger**: 吵架 / 冷战 / 感觉关系变冷
- **Output**: 双方重新对齐 + 至少识别 1 个根本问题
- **入门 SOP**:
  1. 各自冷静 24-48 小时, 不要立刻谈
  2. 主动提出谈一次 — 时间 / 场景双方都舒服
  3. 用 "我感受 + 我希望" 句式开场, 不指责
  4. 听对方说完不打断, 复述确认理解准确
  5. 一起写下「下次怎么避免」, 不只口头承诺
- **资深路径**:
  - skip 试图说服对方你是对的, 资深目标是连接而不是赢
  - optimize 用 Gottman 的「四骑士」识别 (批评 / 蔑视 / 防御 / 筑墙)
  - add 重大冲突后预约一次 couple therapy, 不要拖
- **常见失败模式**:
  - 不冷静直接吵 → 放大伤害
  - 吵完不复盘 → 同样的问题反复出现
  - 一方修复一方躺平 → 长期失衡
- **Last_updated**: 2025-10-15
- **Decay risk**: medium
