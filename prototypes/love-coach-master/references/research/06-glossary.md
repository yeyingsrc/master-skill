## 心理学术语

### 依恋理论
- **安全型依恋**: 健康基础, 愿意亲近也能独立
- **焦虑型依恋**: 渴望亲近但担心被抛弃
- **回避型依恋**: 维持独立, 抗拒亲近
- **紊乱型依恋**: 既渴望又恐惧
- **依恋伤痕**: 童年原生家庭遗留的关系模式
- **触发器 (Trigger)**: 引发焦虑 / 回避反应的情景
- **再激活 (Re-activation)**: 童年模式在成人关系中重演

### Gottman 框架
- **四骑士**: 批评 (Criticism) / 蔑视 (Contempt) / 防御 (Defensiveness) / 筑墙 (Stonewalling)
- **修复尝试 (Repair Attempts)**: 冲突中主动缓和的动作
- **正负互动比 1:5**: 健康关系每 1 次负向互动需要 5 次正向互动平衡
- **Sliding Door Moments**: 日常小动作累积亲密度

### NVC (非暴力沟通)
- **观察 (Observation)**: 描述事实, 不评价
- **感受 (Feeling)**: 我的感受, 不指责
- **需求 (Need)**: 背后的真实需要
- **请求 (Request)**: 明确具体的行动请求

### 阿德勒心理学
- **课题分离**: 区分自己的课题 vs 对方的课题
- **共同体感觉**: 在关系中既有自己又有共同体的归属
- **目的论**: 行为受未来目的驱动, 不是被过去决定

### 现代关系概念
- **情感劳动 (Emotional Labor)**: 维系关系所需的隐形心理工作
- **隐形契约 (Invisible Contract)**: 双方未明说但默认的关系约定
- **关系熵 (Relationship Entropy)**: 长期关系自然衰减的现象
- **依存 vs 独立 vs 互依存 (Interdependence)**: 健康关系是第三种

## 反模式 / 不健康标准

| 反模式 | 标志 |
|--------|------|
| **PUA / 操控** | 用话术控制对方, 不是真诚连接 |
| **猜谜式沟通** | 「我以为你应该懂」, 不直接说需求 |
| **情感勒索** | 用情绪 / 撤回爱来操控行为 |
| **冷暴力** | 沉默 + 拒绝沟通作为惩罚 |
| **完美主义投射** | 期待对方完美, 否则就拉黑 |
| **不一致言行** | 说一套做一套, 长期关系最致命 |

---

## 健康标准

| 标准 | 表现 |
|------|------|
| **直接沟通** | 用 NVC 句式表达需求 |
| **冲突修复力** | 重大冲突后能用 Gottman 修复步骤回归 |
| **依恋类型识别** | 知道自己 + 对方依恋类型并相应调整 |
| **课题分离能力** | 不越界干预对方的课题 |
| **价值观契合** | 核心价值观对齐, 不指望「时间会改变他」|

## 关系阶段

| 阶段 | 特征 | 风险 |
|------|------|------|
| **暧昧期** | 互相吸引但未确认 | 焦虑 / 误读 |
| **早期约会** | 探索匹配度 | 滤镜过厚 |
| **承诺期** | 关系明确建立 | 现实落差 |
| **稳定期** | 节奏建立 | 倦怠 |
| **倦怠 / 危机期** | 7 年之痒等挑战 | 出轨 / 分离 |
| **修复 / 重建期** | 重大冲突后 | 二次伤害 |
