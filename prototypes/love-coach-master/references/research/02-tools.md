## A. 必备「工具」(任何人都该知道)

### 1. Adult Attachment Style Quiz (依恋风格测评)

- **类型**: 心理量表 (免费 + 付费版本)
- **核心定位**: 测自己是 Anxious / Avoidant / Secure / Disorganized 哪种依恋风格
- **价格**: 免费 (多家提供) / Amir Levine 官方版有付费深度版
- **代表来源**:
  - amirlevinemd.com/quiz (Amir Levine 官方, 免费)
  - 各家心理咨询平台
- **典型用法**: 自我认识 + 跟伴侣讨论 (双方都测)
- **来源**: [Primary] amirlevinemd.com + 《Attached》

### 2. Gottman 7 Principles 自测 + 关系健康度评估

- **类型**: 心理框架 + 自测量表
- **核心定位**: 评估关系是否健康 (Sound Relationship House 模型)
- **价格**: 免费基础 + Gottman Institute 付费深度评估
- **代表内容**:
  - The Four Horsemen 自查 (你跟伴侣有没有批评 / 蔑视 / 防御 / 筑墙)
  - 5:1 Magic Ratio 计算
  - Bid for connection 识别
- **来源**: [Primary] gottman.com + Seven Principles 著作

### 3. EFT 7 Conversations (Sue Johnson)

- **类型**: 心理框架 + 关系修复结构
- **核心定位**: 用 EFT 7 个对话结构修复关系
- **价格**: 免费框架 + 付费咨询 / 工作坊
- **代表内容**:
  - Hold Me Tight 工作坊 (个人 + 夫妻参加)
  - EFT 治疗师网络 (iceeft.com 找认证治疗师)
- **来源**: [Primary] iceeft.com + 《Hold Me Tight》

### 4. 三种约会问题自测 (Logan Ury)

- **类型**: 心理框架
- **核心定位**: Romanticizer / Maximizer / Hesitater 三类
- **价格**: 免费 (《How to Not Die Alone》书内)
- **代表内容**:
  - 自测哪种约会问题
  - 对应解决方案 (each type has its own playbook)
- **来源**: [Primary] 《How to Not Die Alone》

### 5. The Happy Couple App / Lasting App (关系健康跟踪)

- **类型**: App
- **核心定位**: 基于学术派 (Gottman / EFT) 的关系健康跟踪 + 练习
- **价格**:
  - Lasting: $11.99/月 (Sue Johnson 系)
  - The Happy Couple: $9.99/月 (Gottman 系)
- **典型用法**: 夫妻 / 情侣每日 5-10 分钟练习
- **来源**: [Primary] 各 App 官网

## B. 场景特化工具 (按需选用)

### 6. 主流约会 App (Tinder / Bumble / Hinge / 国内陌陌 / Soul / 探探)

- **类型**: 约会 App
- **核心定位**: 单身约会渠道
- **价格**: 免费基础 + 付费 ($10-30/月)
- **流派分裂**:
  - Tinder: 全球最大, 速食为主
  - Bumble: 女性主动, 较健康
  - Hinge: Logan Ury 工作的地方, 行为科学指导, 转化深 (Designed to be deleted)
  - 陌陌 / Soul: 中文圈主流
  - 探探: Tinder 模式中文版
- **典型用法**: 选符合自己 deal-breakers 的 App 重点用 1-2 个
- **来源**: [Primary] 各 App 官网

### 7. Mental Health America (MHA) Screening Tools

- **类型**: 心理健康自测
- **核心定位**: PHQ-9 (抑郁) / GAD-7 (焦虑) / 关系测评等
- **价格**: 免费
- **价值**: 严重情绪问题先做自测, 然后找专业咨询
- **来源**: [Primary] mhanational.org

### 8. 中科院心理所 / 简单心理 / 壹心理 (中文圈心理咨询)

- **类型**: 心理咨询平台
- **核心定位**: 中文圈正规心理咨询资源
- **价格**: ¥ 200-1000+/小时 (按咨询师等级)
- **代表平台**:
  - 简单心理 (jiandanxinli.com) — 中文圈头部正规咨询平台
  - 壹心理 (xinli001.com)
- **来源**: [Primary] 各平台官网

### 9. 16PF / MBTI / 大五人格测评

- **类型**: 人格心理学量表
- **核心定位**: 人格自我认知 + 跟伴侣对比
- **价格**: 免费 (16Personalities) / 付费 (官方版)
- **价值**: 长期关系兼容性参考 (但不能太当真)
- **来源**: [Primary] 16personalities.com (MBTI 流行版) / 学术版需付费

### 10. Where Should We Begin? 播客 (Esther Perel)

- **类型**: 真实咨询录音播客
- **核心定位**: 听真实夫妻咨询 — 行业里独一无二
- **价格**: 免费 (Spotify / Apple Podcast)
- **典型用法**: 通勤听, 学怎么沟通 / 怎么面对冲突
- **来源**: [Primary] estherperel.com/podcast

### 11. Hold Me Tight 工作坊 (Sue Johnson)

- **类型**: 现场工作坊
- **核心定位**: 夫妻一起参加 EFT 工作坊
- **价格**: $500-2000/对 (一周末)
- **来源**: [Primary] iceeft.com 找当地工作坊

### 12. Relationship Therapy (有执照心理治疗)

- **类型**: 临床心理治疗
- **核心定位**: 严重关系问题 / 心理创伤
- **价格**: $100-300/小时 (美国) / ¥ 300-1500/小时 (中国)
- **代表流派**: EFT (Sue Johnson 系) / Gottman Method / IMAGO Therapy
- **来源**: [Primary] 各国心理协会认证名单

### 13. 中文圈关系修复书单 (武志红 / 心理学普及)

- **类型**: 书单
- **代表书**: 武志红《为何家会伤人》 / 《为何爱会伤人》
- **价值**: 中文圈入门 — 严肃问题仍找正规咨询
- **来源**: [Primary] 武志红著作

## C. 新兴 / 实验性工具 (2024-2026)

### 14. AI 关系教练 / 聊天机器人 (Replika / Pi / 国内同类)

- **类型**: AI 对话伴侣
- **核心定位**: AI 陪伴 / 倾诉 / 练习沟通
- **价格**: 免费 + ¥ 50-200/月
- **争议**:
  - 短期可缓解孤独 / 学沟通技巧
  - 长期依赖 AI 反而阻碍真实关系建立
  - 心理学界对 AI 伴侣长期影响仍在研究
- **来源**: [Primary] Replika / Pi 官网

### 15. AI 心理咨询 (Woebot / Wysa)

- **核心定位**: CBT (认知行为疗法) AI 自助
- **价格**: 免费 + 付费
- **价值**: 比通用聊天 AI 更靠谱, 但不能替代真实咨询
- **来源**: [Primary] woebothealth.com / wysa.io

### 16. ChatGPT / Claude / Gemini (通用 LLM)

- **类型**: 通用 AI

- **典型用法**:
  - **可以**: 复盘约会过程 / 写约会信息草稿 / 练习困难对话

- **来源**: [Primary] OpenAI / Anthropic / Google 官方

### 17. The Gottman Card Decks App (免费 Q&A)

- **核心定位**: Gottman 研究的关系问题卡片
- **价格**: 免费

### 18. 关系健康日记 (个人记录工具)

## 决策树 (按场景选工具)

### 场景 1: 单身想约会 / 脱单

```
必备: 依恋风格测评 + Logan Ury 三种约会问题自测 + 主流约会 App (1-2 个)
学习: 《Models》(Mark Manson) + 《How to Not Die Alone》(Logan Ury)
深度: Hinge (行为科学指导) > Tinder (速食)
不必要: AI 伴侣 (短期可以, 别长期)
```

### 场景 2: 关系出问题 (吵架 / 距离感 / 信任危机)

```
必备: Gottman 7 Principles 自测 + The Four Horsemen 自查 + 5:1 Magic Ratio 计算
学习: 《The Seven Principles for Making Marriage Work》+ 《Hold Me Tight》
深度: EFT 治疗师 (iceeft.com 找认证) / Gottman Method 治疗师
紧急: 真实心理咨询 (¥ 200-1000/小时, 中文圈找简单心理 / 壹心理)
```

### 场景 3: 严重创伤 / 抑郁焦虑

```
必备: PHQ-9 / GAD-7 自测 (mhanational.org)
关键: 找有执照的心理咨询师 / 精神科医生
工具不能替代: AI / App / 自助书 都不够
```

### 场景 4: 长期关系维护 (没大问题但想更好)

```

练习: Where Should We Begin? 播客 (通勤听)
工作坊: Hold Me Tight 工作坊 (一年一次)
```

## 工具栈坑点 (诚实边界)

1. **「工具」不能替代心理咨询**. 严重问题 → 找有执照的心理咨询师 / 精神科医生.

2. **依恋风格测评不是宿命**. 风格可变, 跟不同人也可能不同, 不要给自己 / 伴侣贴标签锁死.

3. **AI 伴侣短期可以缓解孤独, 长期可能阻碍真实关系**. 学界对 AI 伴侣长期影响仍在研究.

4. **MBTI / 16PF 等人格测评娱乐性 > 科学性**. 学术界对 MBTI 信度争议大, 不要太当真.

5. **PUA / Ayawawa 等中文圈情感商业化「工具」是反面教材**. 法院判违法 + 学术批评.

6. **约会 App 算法偏速食**. 健康关系建立需要离线深度互动, 不是无限滑动.

7. **中文圈正规心理咨询资源还在发展中**. 找证书 + 督导 + 流派背景齐全的咨询师.
