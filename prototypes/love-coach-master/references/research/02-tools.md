# Track 02 — Tools (恋爱高手)

> 调研日期: 2026-05-02。这一行的「工具」更多是评估量表 + 概念框架, 不是 SaaS。

## 元数据
- 必备 4 / 场景特化 5 / 新兴 3

---

## 必备 (心理学认可的核心工具)

### 1. 依恋类型测试 (ECR-R / AAS)
- **One-liner**: 评估自己是安全 / 焦虑 / 回避 / 紊乱 哪种依恋类型
- **Tier**: 必备
- **典型场景**: 自我了解 + 解释关系反复模式
- **来源**: Hazan & Shaver 1987 原始研究
- **避坑**: 依恋类型可以变化, 不是终身标签

### 2. 5 Love Languages 评估 (Gary Chapman)
- **One-liner**: 5 种爱的语言 (肯定 / 服务 / 礼物 / 时间 / 触摸)
- **Tier**: 必备
- **典型场景**: 了解自己 + 对方表达 / 接收爱的偏好
- **付费**: 免费在线测
- **避坑**: 简化模型, 不要套用太死

### 3. Gottman 关系评估问卷
- **One-liner**: 关系健康度多维评估
- **Tier**: 必备
- **付费**: 部分免费
- **价值**: 实证基础, 可定位关系问题维度

### 4. 非暴力沟通 (NVC) 句式工具
- **One-liner**: 「观察 / 感受 / 需求 / 请求」四步沟通法
- **Tier**: 必备 (理论 + 实操工具)
- **来源**: Marshall Rosenberg 《非暴力沟通》

---

## 场景特化

### 5. MBTI / Big Five 性格测试
- **典型场景**: 了解性格特质对关系的影响
- **避坑**: MBTI 学术界争议大, Big Five 更可信

### 6. Imago 对话工具 (Imago Therapy)
- **典型场景**: 深度沟通 + 童年模式探索
- **谁用**: couple therapy 实践者

### 7. 「冲突修复」步骤法 (Gottman)
- **典型场景**: 吵架后的修复对话
- **价值**: 系统化的修复流程

### 8. 情感日记
- **典型场景**: 反思自己的情绪模式
- **付费**: 笔记本 / Day One / Notion 模板

### 9. Coupling Apps (Lasting / Paired / Relish)
- **典型场景**: 数字化关系训练
- **付费**: 月费
- **状态**: 工具有用, 但不能替代真实沟通

---

## 新兴 / 实验

### 10. AI 情感咨询 (Replika / 各家)
- **典型场景**: 模拟对话 / 倾诉
- **风险**: 容易形成依赖, 不能替代真人关系

### 11. AI 关系反思工具
- **典型场景**: 用 AI 帮你梳理关系问题
- **风险**: AI 不知道完整 context, 建议需谨慎

### 12. 在线咨询平台 (壹心理 / 简单心理)
- **典型场景**: 国内心理咨询师在线预约
- **价值**: 真正需要专业帮助时的入口

---

## 选型决策树

| 状况 | 工具 |
|------|------|
| 单身期反思 | 依恋类型 + Big Five + 情感日记 |
| 暧昧期 / 早期约会 | 5 Love Languages + 价值观对齐 |
| 稳定关系 | Gottman 评估 + NVC + Imago |
| 冲突期 | NVC + Gottman 修复步骤 + 必要时咨询师 |

## 避坑

1. PUA 话术 / 套路工具一律不用 — 短期可能有效, 长期毁关系
2. 性格测试别套死 — MBTI 类不严谨
3. AI 情感工具是辅助, 不能替代真实关系
4. 单点测评不够, 要长时间观察实际行为
5. 真正的心理问题 (创伤 / 抑郁 / 焦虑) 找专业咨询师
