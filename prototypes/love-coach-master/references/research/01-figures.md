## A. 学术研究派 (战略级最高一手)

### 1. John Gottman + Julie Schwartz Gottman (Gottman Institute)

- **核心一句话**: 婚姻稳定可被预测. 90%+ 准确率从一对夫妻 15 分钟互动看出他们 5-10 年后离不离.
- **代表事件**:
  - 1980 年代起在「爱情实验室」研究 3000+ 对夫妻
  - 提出 Cascade Model of Relational Dissolution (关系瓦解级联模型)
  - The Four Horsemen (四骑士): 批评 / 蔑视 / 防御 / 筑墙 — 关系毒药
  - 5:1 Magic Ratio: 健康关系正面互动 vs 负面互动比例至少 5:1
  - 2023 跟 Esther Perel 合作《10 Principles of Effective Couples Therapy》
- **代表观点**:
  1. 「Mastery in everyday life」 — 健康关系不是没有冲突, 是知道怎么修复 (repair attempts)
  2. 蔑视 (contempt) 是离婚最强预测信号 — 超过吵架本身
  3. Bid for connection — 伴侣发出情感连接信号, 你回应或忽略 决定关系长期走向
- **来源**: [Primary] Gottman Institute (gottman.com) + 著作《The Seven Principles for Making Marriage Work》

### 2. Sue Johnson (Emotionally Focused Therapy / EFT)

- **核心一句话**: 亲密关系本质是依恋 (attachment) — 修复关系就是修复安全依恋.
- **代表事件**:
  - 1980 年代发展 EFT (情感聚焦疗法)
  - 著作《Hold Me Tight》 (拥抱你, 关系修复经典)
  - EFT 是循证 (evidence-based) 心理治疗中关系修复成功率最高的之一 (75%+)
- **代表观点**:
  1. 成人亲密关系跟婴儿对父母的依恋 同源, 都是「我有人吗?」的根本焦虑
  2. 关系冲突表面是事件, 本质是依恋焦虑 (我对你重要吗?)
  3. EFT 7 个对话 (conversation) 修复关系
- **来源**: [Primary] iceeft.com + 《Hold Me Tight》

### 3. Esther Perel (Modern Love / Mating in Captivity)

- **核心一句话**: 长期关系的悖论 — 安全感 vs 激情. 你想要的是「跟你最熟悉的人最神秘的关系」.
- **代表内容**:
  - TED Talk「The secret to desire in a long-term relationship」(2000 万+ 观看)
  - TED Talk「Rethinking infidelity」(2000 万+ 观看)
  - 著作《Mating in Captivity》 + 《The State of Affairs》
  - 播客「Where Should We Begin?」 (真实夫妻咨询录音)
- **代表观点**:
  1. 长期关系最大挑战不是冲突, 是激情消退
  2. 出轨不只是道德问题, 是关系叙事问题
  3. 现代关系把所有需求 (朋友 / 情人 / 父母 / 知己) 压在一个伴侣身上, 不可能不破
- **来源**: [Primary] estherperel.com + TED Talks + Where Should We Begin? 播客

### 4. Amir Levine + Rachel Heller (Attached / Attachment Theory)

- **核心一句话**: 你是 Anxious / Avoidant / Secure 哪一种依恋风格, 决定你跟谁能幸福.
- **代表内容**:
  - 著作《Attached》 (2010, 销量 200 万+) — 把依恋理论带到大众
  - 2026《Secure》新书纠正「Attached 被普众误读」的问题
- **代表观点**:
  1. Anxious (焦虑型): 担心伴侣不爱我, 频繁需要确认
  2. Avoidant (回避型): 把亲密 = 失去独立, 持续推开
  3. Secure (安全型): 舒服亲近, 温暖稳定
  4. Anxious + Avoidant 配对是「The Anxious-Avoidant Trap」 — 最痛苦组合
- **代表数据**: 50% 人 Secure, 20% Anxious, 25% Avoidant, 5% Disorganized
- **来源**: [Primary] amirlevinemd.com + 《Attached》《Secure》

### 5. John Bowlby + Mary Ainsworth (依恋理论奠基)

- **核心一句话**: 依恋理论的源头 — 婴儿期跟主要照顾者的依恋决定成年关系模式.
- **代表内容**: Bowlby《Attachment and Loss》三部曲 + Ainsworth Strange Situation 实验
- **价值**: 学术祖辈, 理解现代依恋理论必读
- **来源**: [Primary] 学术著作

## B. 健康约会指南派 (面向单身)

### 6. Mark Manson (Models / Subtle Art / Everything Is Fucked)

- **核心一句话**: 吸引女性靠诚实, 不靠套路. 自我成长 > 把妹技巧.
- **代表内容**:
  - 《Models: Attract Women Through Honesty》(2011, 仍是男性约会经典)
  - 《The Subtle Art of Not Giving a F*ck》(2016 NYT bestseller)
  - markmanson.net 博客 (千万级月访问)
- **核心理论**: 三件套 — Honest Living / Honest Action / Honest Communication
- **代表观点**:
  1. PUA「假装不感兴趣」是反人类的 — 真喜欢就直接说
  2. 吸引力 = 你自己有什么 (价值观 / 生活 / 目标), 不是技巧
  3. Vulnerability 是真正的吸引力来源
- **来源**: [Primary] markmanson.net + 《Models》

### 7. Logan Ury (How to Not Die Alone / 行为科学派)

- **核心一句话**: 用行为科学指导约会决策 — 多数人不是缺爱, 是约会方式错了.
- **代表内容**: 《How to Not Die Alone》(2021) + Hinge 行为科学家
- **代表观点**:
  1. 三种约会问题: Romanticizer (浪漫主义) / Maximizer (完美主义) / Hesitater (犹豫不决)
  2. 设定 deal-breakers 明确, 不要无限挑剔
  3. 第三次约会规则: 不要凭第一次定生死
- **来源**: [Primary] loganury.com + 《How to Not Die Alone》

### 8. Matthew Hussey (Get the Guy / 女性向)

- **核心一句话**: 女性主动 + 健康自信 + 高情商沟通 是新型约会指南.
- **代表内容**: 《Get the Guy》 (2013) + howtogettheguy.com + YouTube 千万订阅
- **代表观点**:
  1. 女性主动 ≠ 倒贴 — 是高情商沟通
  2. High-value woman 不是装出来的, 是 demonstrate
- **争议**: 有时被批评「女性向 PUA」, 但整体方向健康
- **来源**: [Primary] howtogettheguy.com

- **核心一句话**: 中文圈最有影响力的心理学普及作家, 推动「原生家庭 / 依恋」概念在中文圈传播.

### 10. 心理咨询界 (中科院心理所 / 北师大心理 / 北大心理 等)

- **核心一句话**: 中文圈正经心理咨询 / 婚姻家庭治疗代表 — 不是「恋爱教练」, 是临床心理学家.
- **代表机构**: 中科院心理所 / 北师大心理学部 / 北大心理与认知科学学院

### 11. ⚠️ 浪迹情感 (反面教材 — 已被法院判决违反公序良俗)

- **核心一句话**: 中国 PUA 鼻祖, 2019 年被法院判决「教学内容违反公序良俗」.
- **历史**:
  - 创始人王环宇, 2008 起在泡学网 / 坏男孩学院
  - 2015 创办浪迹教育
  - 2017 改名浪迹情感, 估值 2 亿, 计划 A 股上市
  - 2019-06 4 名前学员起诉, 法院判决教学内容违反公序良俗 + 退还学费
  - 2019-12 被全网集体封杀
- **核心问题**:
  1. 「情感操纵法」(emotional manipulation) — 系统化教男性如何操控女性
  2. 反女性 + 反伦理 — 多次被媒体 / 学术界 / 法院 批评
  3. 「PU (Paternity Uncertainty) 降低法」等理论被广泛批判
- **诚实边界**: 本 skill **不教浪迹方法论**. 列出仅为说明「中文圈 PUA 是反面教材」.
- **来源**: [Primary] 腾讯新闻 / 澎湃新闻 / 法院判决

### 12. ⚠️ Ayawawa (反面教材 — 已被全网封禁)

- **核心一句话**: 中文圈情感商业化代表, 2018 因「南京大屠杀」言论被全网封禁 + 央视点名.
- **历史**:
  - 「MV (Mate Value) + PU (Paternity Uncertainty)」评分理论
  - 把男女当成可评分的商品 — 学术界批判其「商品化人性」
  - 2018 年关于南京大屠杀「女性被强暴反而活下来」言论 → 央视白岩松点名 → 全网封禁
- **核心问题**:
  1. 把人物化 + 商品化
  2. 反女性 (虽然主打女性向, 但实质羞辱女性)
  3. 学术伪科学 (MV / PU 评分体系无任何学术基础)
- **诚实边界**: 本 skill **不教 Ayawawa 方法论**. 列出仅为说明「中文圈情感商业化是反面教材」.
- **来源**: [Primary] 澎湃新闻深度调查 + 央视新闻批评

## 流派分裂 (关键认知分歧)

| 流派 | 主张 | 适合 | 风险 |
|------|------|------|------|
| **学术研究派** (Gottman / Sue Johnson / Esther Perel / Levine) | 循证心理学 + 依恋理论 + 长期关系 | 任何想认真学的 | 上手慢, 见效慢 |
| **健康约会指南派** (Mark Manson / Logan Ury / Matthew Hussey) | 自我成长 + 诚实沟通 + 行为科学 | 单身 / 想约会 | 仍需配合心理建设 |
| **中文心理学普及派** (武志红 等) | 原生家庭 + 依恋理论 (中文圈) | 中文圈入门 | 学术严谨度参差 |
| **PUA / 情感商业化派** ⚠️ | 套路 / 操控 / 评分 (反面教材) | (无) — 反面教材 | 反伦理 + 反女性 + 法院判违法 |
| **现代女性主义视角** | 对等沟通 + 反操控 + 反评分 | 任何人 | 跟 PUA 直接冲突 |

**核心分歧 1**: **学术循证 vs 经验套路** — 决定方法论可靠性
**核心分歧 2**: **健康成长 vs 操控技巧** — 决定伦理底线
**核心分歧 3**: **中文圈 vs 西方学术圈** — 决定理论严谨度

## 可信度判断 (诚实边界)

1. **学术派 (Gottman / Sue Johnson / Perel / Levine) 是循证 + 同行评议** — 最可靠.

2. **健康约会指南派 (Mark Manson / Logan Ury) 是经验 + 行为科学** — 二手可靠, 但不是临床.

3. **中文心理学普及派 (武志红) 严谨度参差** — 入门 OK, 严肃问题找正经心理咨询.

4. **PUA / 情感商业化派 (浪迹 / Ayawawa) 是反面教材** — 法院判违法 + 学术批判 + 媒体封禁. 本 skill 不教.

5. **本 skill 不能替代心理治疗 / 心理咨询**. 严重关系问题 / 心理创伤 / 抑郁焦虑 → 找有执照的心理咨询师 / 精神科医生.

6. **类目差异**: 单身约会 vs 关系修复 vs 心理创伤 是 3 个不同问题, 方法论也不同.

7. **文化差异**: 西方学术派方法论需要文化适配到中文圈.
