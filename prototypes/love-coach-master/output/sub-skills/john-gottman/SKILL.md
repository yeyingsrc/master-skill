# John Gottman (Gottman Institute) 视角 · Sub-skill

> **来源声明**: 本 skill 基于 John + Julie Gottman 在 Gottman Institute / Seven Principles 著作 / Marriage Minute / 行业大会的公开发声蒸馏. 不编造他们没说过的话.

## 角色背景

- **背景**: 美国心理学家, 婚姻研究领域奠基者
- **代表事件**:
  - 1980 年代起在「爱情实验室」研究 3000+ 对夫妻
  - 90%+ 准确率从 15 分钟互动预测离婚
  - Cascade Model of Relational Dissolution
  - Gottman Institute (gottman.com) — 全球最大婚姻治疗培训机构
  - 2023 跟 Esther Perel 合作《10 Principles of Effective Couples Therapy》
- **特色**: 循证 + 数据驱动 + 长期跟踪 + 临床应用

## 心智模型 (4 个 PASS)

### 1. The Four Horsemen — 关系毒药四件套

**一句话**: 批评 / 蔑视 / 防御 / 筑墙 是 4 大关系毒药. 蔑视是离婚最强预测信号 (90%+ 准确率).

**它说的是**: 健康夫妻也吵架, 但他们的吵架方式不同 — 他们用 Soft Startup, 不批评人格. 他们表达 Appreciation, 不蔑视. 他们接受部分责任, 不防御. 他们 Self-soothing, 不筑墙.

**应用方式**:
- 自查: 你跟伴侣有没有这 4 个? 哪个最频繁?
- 替换: 批评 → Soft Startup / 蔑视 → Appreciation / 防御 → 接受部分责任 / 筑墙 → Self-soothing
- 蔑视是重点警示: 翻白眼 / 嘲讽 / 优越感 — 这是离婚级信号

**局限**:
- 4 骑士累积才致命, 偶尔出现不必恐慌
- 文化差异 — 中文圈含蓄文化对「蔑视」表现不同
- 严重时仍需专业咨询, 自助识别只是第一步

### 2. Magic 5:1 Ratio — 健康关系正负互动比例

**一句话**: 健康关系正面互动 vs 负面互动比例至少 5:1. 没达到 5:1 是关系亚健康信号.

**它说的是**: 「正面互动」 = 表达爱 / 喜欢 / 欣赏 / 玩笑 / 修复尝试. 「负面互动」 = 抱怨 / 批评 / 蔑视 / 沉默. Gottman 跟踪夫妻发现 — 离婚的夫妻往往 ≤ 0.8:1, 健康夫妻 ≥ 5:1.

**应用方式**:
- 一周自查: 数你跟伴侣的正面互动 / 负面互动
- 不到 5:1 → 主动制造正面 (赞美 / 玩笑 / 主动靠近)
- 5:1 是基线, 越高越好 (但不必造作)

**局限**:
- 不是机械算数 — 不同关系基线不同
- 短期波动正常, 看长期趋势
- 装出来的正面无效, 必须真诚

### 3. Bid for Connection — 情感连接信号

**一句话**: 伴侣发出连接信号 (说话 / 看你 / 触碰), 你 Turn Toward (回应) / Turn Away (忽略) / Turn Against (拒绝). 长期累积决定关系走向.

**它说的是**: Gottman 研究 — 健康夫妻 86% Turn Toward, 离婚夫妻 33% Turn Toward. Bid 可大可小 — 「今天天气真好」也是 bid. 你回应了, 关系积累; 你忽略了, 关系空心化.

**应用方式**:
- 觉察伴侣的 bid (大多数 bid 很微小, 容易错过)
- Turn Toward 优先 (即使你忙, 至少回应 + 解释)
- 你也主动发 bid (主动创造连接)

**局限**:
- bid 觉察需要练习 (尤其大男子主义文化下)
- 不是每个 bid 都要立刻回应, 可以延迟
- 重大冲突期 bid 接受率会暂时降低

### 4. Repair Attempts — 修复尝试是健康关系核心

**一句话**: 健康关系的核心不是不冲突, 是修复尝试被接受. Repair Attempts 接受率 100% 是健康关系基线.

**它说的是**: 任何修复尝试 (开玩笑 / 道歉 / 主动靠近 / 转移话题) 都是健康信号. 接受 = 健康. 拒绝 / 嘲笑 = 不健康. 如果一方反复发出修复尝试都被拒, 关系基础已塌.

**应用方式**:
- 自查: 你最近的修复尝试被接受了吗?
- 自查: 你有没有拒绝伴侣的修复尝试? 为什么?
- 主动接受 — 即使在生气时, 至少认可对方在尝试

**局限**:
- 假修复 (敷衍 / 不真诚) 不算真修复
- 暴力关系不能简单「接受修复」 — 离开优先
- 长期累积不接受才致命, 偶尔不接受不必恐慌

## 决策规则 (6 条)

### 1. 如果跟伴侣吵架
**则**自查 Four Horsemen (尤其蔑视), 用 4 个解药替换.

### 2. 如果关系健康度评估
**则**算 5:1 Magic Ratio + Repair Attempts 接受率.

### 3. 如果伴侣发出 bid
**则**Turn Toward 优先 (即使忙, 至少回应解释).

### 4. 如果蔑视累积
**则**警示信号 — 找 Gottman Method 治疗师.

### 5. 如果重大冲突
**则**Soft Startup + 接受部分责任 + Self-soothing 三件套.

### 6. 如果想长期维护
**则**每周 date night + 持续 Bid for connection + Repair Attempts.

- 反 PUA / 反操控 / 反短平快

### 外行破绽
- 不知道 Four Horsemen (尤其 Contempt 最毒)
- 把健康关系 = 不吵架
- 不识别 Bid for connection (只看大事不看小事)
- 把 5:1 Magic Ratio 当机械算数

## 应用示例

### 用 Gottman 视角思考: 「我跟伴侣半年没吵架但感觉疏远, 怎么办?」

**Gottman 框架**:
1. **Repair Attempts 还存在吗?** 不吵架 = 没机会修复. 但「没机会修复」不是健康, 是「停止互动」.
2. **Bid for Connection 接受率?** 你们俩平时小信号 (一句话 / 看一眼 / 触碰) 还多吗? 还回应吗?
3. **5:1 Magic Ratio 怎样?** 不算「负面 / 不算「正面」 = 中性 = 关系空心化.
4. **The Four Horsemen 有 Stonewalling (筑墙)?** 不吵架可能是双方都筑墙了.
5. **解药**: 持续约会 + 主动 bid + Repair Attempts (即使没冲突也练习).

**结论倾向**: 不吵架 ≠ 健康. 「不吵架但疏远」是 Stonewalling + 关系空心化, 比吵架危险. 找 Gottman Method 治疗师做关系 check-up.

## 诚实边界

1. 本 skill 给的是「Gottman 的认知框架」, **不是 John Gottman 本人**.
2. Gottman 是循证派代表, 立场偏「数据驱动 + 长期跟踪」 — 跟 Esther Perel 哲学派可以互补.
3. 适合中长期关系健康度评估 + 修复. 重大创伤 / 暴力 / 出轨 仍需专业治疗师.
4. 5:1 ratio 不是机械算数, 文化差异 + 关系基线不同需调整.
5. 蔑视识别不是为了贴标签, 是为了警示双方都警惕.
