## 1. 行业定义

**中文**:风险投资人 / 早期投资判断(GP + Principal — Deal sourcing / 尽调 / 估值与 term sheet / 投后服务 / 退出 + LP 关系与 fundraising — 涵盖 US 美元基金 + 国内 RMB / 美元基金)

**English**:Venture Capital — practitioner role of GPs / Partners / Principals / Associates making early-stage equity investments: sourcing, diligence, valuation + term sheet, post-investment support, exits, plus fund-level activities (LP relations, fundraising, portfolio construction).

**Industry type**:`high_capital_specialist`
**Locale**:`global`(US 美元 / 国内美元 / RMB 三条路径,海外内地差异巨大)
**Profile**:`practitioner`(GP / Partner / Principal / Associate / 想转 VC 的 operator)
**Focus**:`business` 为主 + `operational` 为辅

## 2. 核心子领域候选(Phase 1 调研维度)

1. **Deal sourcing** — 网络 / inbound / outbound / scout networks / accelerator / AI 辅助
2. **Due diligence** — founder fit / market sizing / product / competition / unit economics / legal / cap table / reference call
3. **估值 + term sheet** — SAFE / convertible note / priced round / pro-rata / liquidation preference / anti-dilution
4. **Thesis-driven 投资** — 基金 sector / stage / geo 主题选择 + 演化
5. **投后服务** — board / hiring / 客户介绍 / 下一轮 fundraising / 危机处理
6. **Exit 退出** — IPO / strategic M&A / secondary / write-off
7. **LP 关系 + fundraising** — institutional / family office / sovereign / endowment / fund-of-funds 各类 LP 的 selection
8. **Portfolio construction** — check size / reserve / concentration vs diversification / power law math
9. **Fund operations** — carry vesting / management fee / GP commitment / co-invest / SPV
10. **Geographic + stage 差异** — US / 国内美元 / RMB 三条路径
11. **时代周期** — 2021 高点 vs 2022-2024 寒冬 vs 2025 AI 拐点的工作节奏 + 风险偏好变化
12. **VC vs 创始人 关系动态** — 权力 / 信息不对称 / 利益对齐

## 6. 警示与边界(写入 SKILL.md 诚实边界)

1. **Power-law business**:90%+ deals 失败或回本,头部 1-2 个 deals 占 fund return 80%。**「VC 不是稳定职业」**。
2. **公开访谈 vs 真实工作 gap 极大**:多数「VC playbook」是 GP 个人品牌营销,不是实际工作。SKILL.md 区分「公开人设」vs「实际工作内容」。
3. **国内外路径差异极大**:US 美元 VC 退出靠 IPO / M&A;国内 RMB 退出难,美元面临中美脱钩 + IPO 收紧。**SKILL.md 按 locale 分章**。
4. **2022-2024 寒冬尚未完全过去**:全球 down cycle,LP 钱少,GP 解散 / merge。2025-2026 AI cycle 反弹但其他赛道仍冷。**SKILL.md 写时间戳**。
5. **学派分歧大**(Thesis / Spray-and-pray / Concentrated / Solo GP / 国内多元化)。
6. **个人 brand ≠ 投资能力**:LinkedIn 网红 VC 真实业绩多平淡。「公开多 ≠ 投得好」。
7. **数据透明度极差**:GP fund returns 多不公开,公开数据(PitchBook / IT 桔子)偏头部偏后期,种子 / 早期天使 deal 大量缺失。
8. **信息源严重精英化**:深度知识在 deal team 会议 / GP-LP 闭门会 / Stanford GSB 圈子流通,公开内容是 sanitize 过的版本。**SKILL.md 诚实写「外部能拿到的 70 分,顶级 20 分拿不到」**。
9. **职业上升路径不透明**:Associate → Principal → Partner 极少数能走通,「up or out」。**SKILL.md 写真实就业市场而非励志故事**。
10. **Carry 兑现周期长**:工资靠 management fee(2%/yr × fund),carry(20% profit)7-10 年后才兑现。
11. **VC 与创始人利益不对齐**:GP 想 outlier exit,founder 可能想 long-term build。**SKILL.md 写「VC playbook 不一定对 founder 最优」**。
