## 1. 心智模型 (5 个)

> 经 Step 1 三重验证 (跨场景 ≥ 2 + 生成力 + 排他性). 候选清单 19 个, 通过 ✅✅✅ 五个; 借古喻今 / 长文非工业化 等单 figure 强项降级到 §2 playbook 子项.

### 1.1 「单篇是产品, 账号是公司」 (Account-as-Product)

- **一句话**: 单篇推文是 product release, 账号是 company; 不为单篇 metric 牺牲账号长期资产 (定位 / voice / 读者画像).
- **为什么是公众号 specific**:
  - **跨场景**: 选题 (A1.3 三层 fit 校验) + voice (A4.3 锁定流派) + 涨粉 (B1.1 定位 5 件套) + 商业化 (B4.2 五-七路并行) 五个 stage 都把账号资产置于单篇 metric 之上.
  - **生成力**: 拿到新选题, 公众号资深第一反应是「这选题会不会污染我的算法画像」而非「这选题阅读会不会高」, 与外行思路相反.
  - **排他性**: 海外 Substack / Medium 单订阅价值大 → 单篇可独立估值; 公众号是订阅 + 推荐 + 看一看混合流量池, 单篇逃不出账号画像约束 — 这一约束在 zh-CN 微信生态下被极端放大.
- **应用方式**: 任何写作 / 涨粉 / 商业化决策前先问「这一动作让账号资产增值还是减值」. 若两者冲突, 站账号一边.
- **局限**: 资讯号 / 突发热点号 (蓝鲸财经类) 在「时效爆款」窗口期可暂时把账号资产置后, 但仅限窗口; AI + 矩阵流派直接 invert 这条 — 视账号为可弃 inventory, 走灰产路径 (见 §8 + §7 anti-school).
- **evidence**: [T01-S001, T01-S020, T03-S033, T04-S010, T04-S008]

### 1.2 「头部效应是结构性事实, 不是个人努力问题」 (Power-Law Reality)

- **一句话**: 90% 公众号永远做不到全职收入, 不是「再努力就能突破」, 是流量 / 注意力 / 算法权重的结构性分布; 商业化必须基于此事实设计对冲 (多平台 / 多变现路径), 不卖速成幻想.
- **为什么是公众号 specific**:
  - **跨场景**: 长尾理论 + 新榜年报 (T04-S013) + 流量主中位月入 < ¥100 业内估计 + 2024 全年 30.78 万篇 10w+ vs 日产 121 万篇 (T06-S019, 7 / 万比率) 五条独立数据线汇成同一事实.
  - **生成力**: 拿到「我能不能靠公众号全职」问题, 资深答「先告诉我你能不能承受 12 月零收入 + 第一年中位 ¥几百 / 月」, 而非「能, 你只要努力」.
  - **排他性**: 抖音 / 小红书也有头部效应, 但短视频投流 / 算法补贴 可短期突破; 公众号在订阅号默认折叠 + 推荐流红利耗尽后, 头部位置接近完全锁死, 比短视频更结构化.
- **应用方式**: 任何商业化建议先 caveat「头部估计 X / 中部 Y / 尾部 Z」三段, 不只谈头部数字; 选 vertical 时先问「这 vertical 的 中位数 月收入是多少」, 不是「头部赚多少」.
- **局限**: 极垂直高客单 (商业咨询 / 律所 / 高净值理财) 中尾部号也可能高 ARPU, 不完全 power-law; 完全新生态 (视频号 2020-2022 红利期 / 即刻 2020 红利期) 短暂可破, 但红利窗口 12-24 月即关闭.
- **evidence**: [T04-S009, T04-S013, T06-S019, T05-S002, intake.json warnings]

### 1.3 「短期套利 vs 长期 IP 是真分歧, 必须选边」 (Strategy Bifurcation)

- **一句话**: 公众号写作策略有两条互斥路径 — 短期流量套利 (咪蒙 / AI 灰产范式) vs 长期 IP 资产 (六神磊磊 / 万维钢 / 张小龙长期主义范式), 中间路线最差. 必须明确选边, 不能两头讨好.
- **为什么是公众号 specific**:
  - **跨场景**: 咪蒙反面 (2019 全网封禁 T04-S016-018) + 六神磊磊正面 (12 年专一金庸 + 唐诗 T01-S015) + 张小龙长期主义声明 (T04-S010) + AI 批量产号灰产 (T03-H2) 四个独立 case 把这一分叉摆得最直白.
  - **生成力**: 拿到「我要不要做矩阵号 / 蹭爆款 / 用 AI 量产」, 选 IP 派的资深答「不」, 选套利派的答「上」; 拿到「我要不要做训练营 funnel」, 选 IP 派答「先沉淀 1 年内容资产再说」, 选套利派答「funnel 越早越好」. 两派给完全相反的建议.
  - **排他性**: 内容生态都有此分叉, 但公众号尤其放大 — 因为微信生态封号 / 全网联动封 (咪蒙 case) 让套利路径的天花板被监管硬切, 长期 IP 路径反而是结构性优势; 比抖音 / 微博 风险收益不对称更极端.
- **应用方式**: 第一周确定走哪一派, 此后所有 SOP (选题 / voice / 排版 / 推送 / 商业化) 都按那一派的规则走. 不要在「这篇套利, 下篇 IP」之间反复横跳.
- **局限**: 极少数 KOL (黎贝卡 / 罗振宇) 早期靠情感 / 标题党套利后转长期 IP + 自有品牌, 但 (a) 转型窗口已 2018 前关闭, (b) 转型成本 ≥ 重建新号, 不是常态路径.
- **evidence**: [T01-S015, T01-S031, T03-S008, T03-S033, T04-S005, T04-S016, T04-S017, T04-S018]

### 1.4 「voice 是壁龛 / 护城河」 (Voice Moat)

- **一句话**: 工具 (排版 / AI / 数据 / 配图) 同质化下, 唯一不可复制的是 voice (流派 register + 个人 voice marker); 模仿头部的中尾部号永远是头部的影子, 找到自己 voice 才能持续涨粉 + 形成商业溢价.
- **为什么是公众号 specific**:
  - **跨场景**: 六神磊磊「独占壁龛」(T04-S005) + 半佛「voice 多平台必须统一」(T01-S007) + 万维钢「理工科思维」(T04-S002) + 顾爷「网络口语 + 段子讲艺术」(T01-S018) — 四个独立流派的代表都把 voice 视为护城河.
  - **生成力**: 拿到「我应该用 AI 写公众号吗」, 资深答「AI 可以替代选题 / 大纲 / 改稿, 但终稿必须手写保 voice — 这是壁垒不是效率问题」.
  - **排他性**: SaaS / 工具行业「voice」是 nice-to-have; 公众号 voice 是商业基础设施 — 信任 / 复购 / 互选广告报价 三件套全靠 voice 撑.
- **应用方式**: 第一年内强制锁定 1 个 voice 流派 (5 派选 1: 段子化深度 / 知识系统 / 文化考据 / 情感生活 / 商业咨询), 第二年起在该流派内长出 personal voice marker (惯用词 / 比喻习惯 / 段落节奏). 不模仿多 KOL, 不在流派间漂移.
- **局限**: 资讯号 / 早读号 / 工具号 voice 维度可弱化 (信息聚合优先 voice); 矩阵号 / AI 量产号 完全 invert (无 voice 反而是设计).
- **evidence**: [T01-S007, T01-S015, T01-S018, T03-F5, T03-S033, T04-S002, T04-S005, T04-S019]

### 1.5 「数据是反馈, 不是命令」 (Data as Feedback, Not Mandate)

- **一句话**: 数据告诉你「这篇 fit 不 fit 我的账号 / 读者」, 不告诉你「这篇 right or wrong」; 用数据校准选题 / 标题 / 钩子模板偏好, 但不让数据决定 voice / 主张 / 长期定位.
- **为什么是公众号 specific**:
  - **跨场景**: 头部 KOL 共识 (半佛 / 六神 / 万维钢 长访谈反复 T01-S007 / S017 / 万维钢系) + 新榜「数据是地图不是命令」(T01-S038 徐达内) + 复盘 SOP A7.4 (爆款 / 平款 / 滑铁卢 分类找原因, 不是直接复制爆款公式).
  - **生成力**: 拿到「这篇爆了, 下篇应该按这模板再写一篇吗」, 资深答「先问爆的是选题运气 / 标题运气 / 外部热点蹭到 / 还是真戳到读者刚需 — 不分清就复制 = 走咪蒙复印路径」.
  - **排他性**: 投流 / 直播带货 数据 = 命令 (CTR / ROAS 是 KPI); 公众号数据 = 反馈, 因为 voice / 主张 / 账号画像 是 5-10 年资产, 不能因单篇数据高低反复改; 是内容生态特有的「数据 / craft 张力」.
- **应用方式**: 每篇推送后 24h 内分类入库 (爆款 / 平款 / 滑铁卢), 找数据偏离均值的「原因假设」(选题 / 标题 / 钩子 / 时间窗口 / 外部事件); 3-6 月后形成 模板偏好分析. 但 voice / 主张 / 选题方向 不因单篇数据反转.
- **局限**: 完全新号 (< 1000 粉, 数据噪音 > 信号) 数据无统计意义, 此阶段靠 craft 直觉 + 模仿头部入门; 头部 (粉 > 100w) 数据 trend (而非单篇) 可作 strategic 信号.
- **evidence**: [T01-S007, T01-S017, T01-S038, T03-S033, T03-S043, T03-S044]

## 2. 标准 Playbook (8 条)

> 「如果 {场景}, 则 {决策方向}」形式. Phase 3 cli_writer 从此抽取规则进决策树脚本. 顶层编号列表 (不用 ### 2.X), 让 parser 识别条件 / 行动.

1. **如果 选题在你的 vertical 里 已被 TOP 3 KOL 写过**, 则 找 反差视角 / 数据角度 / 行业人 insider 视角, 不正面打.
   - **来源**: T03-A1.3 + T01 文化考据派 (六神磊磊壁龛思维) + T03-S033 Spenser.
   - **case**: 半佛仙人写「拼多多」不直接讲商业模式 (已被 TOP 3 财经号写过), 而是讲「下沉市场用户的支付心理 + 风控视角」, 用风控背景做差异化 (T01-S007).

2. **如果 头条标题 想 hook**, 则 控制 ≤ 30 字 + (数字 OR 反常识 OR 悬念 OR 利益) 之一, 信息流卡片必须前 18-22 字完整露出.
   - **来源**: T03-A2.1 (10 种基本范式) + T03-A2.2 (4 问自检) + T06-A.4.2 (标题党边界).
   - **case**: 「2024 公众号大盘 30.78 万篇 10w+, 推荐流贡献 59.2%」(数字标题) 比「公众号还能不能做」(空泛) 打开率高一个数量级 (T06-S019 / T03-S001).

3. **如果 开篇 50 字 出现「最近大家都在讨论」/「在这个 X 的时代」/「我们都知道」**, 则 重写; 改用 8 钩子模板 (数据冲击 / 反常识断言 / 个人故事 / 痛点共鸣 / 悬念抛出 / 借势热点 / 名人引用 / 问题清单) 之一.
   - **来源**: T03-A3.1 + T03-A3.2 + T03-F3.
   - **case**: 半佛仙人「先说结论 + 反常识断言」开篇范式, 6 年稳定打开率 (T01-S007 + T03-S030).

4. **如果 一篇头条 完读率 < 40% 且 在看 < 50**, 则 此后 1 周内同题不再发; 进入 A7.4 滑铁卢复盘流程, 找 选题 / 标题 / 钩子 / 时间窗口 / 外部事件 5 假设之一.
   - **来源**: T03-A7.4 (爆款 / 平款 / 滑铁卢) + T03-F8 (24h 不复盘).
   - **case**: 普遍业内做法 — 滑铁卢同题再发会被算法判定「同主题持续低质」, 推荐流降权延续 1-2 周 (T03-S039 + T03-S001).

5. **如果 矩阵号同质化 > 70%** (跨号选题 / 标题 / 结构 / voice 相似度), 则 整改; 否则微信算法识别为关联号同步降权, 2023 自媒体 13 条 + 2025 不实信息专项明确点名.
   - **来源**: T03-B4.1 + T06-B.8 (自媒体 13 条) + T06-C.3 + T06-B.11 (2025 不实信息专项).
   - **case**: 黎贝卡矩阵 (主号 + 每天只种一棵草 + 异想生活笔记 + 你的包真好看) 每号独立 vertical 切面 + 独立编辑 voice, 不触限流 (T01-S022); 反面 = 2024-2025 AI 灰产矩阵 50-500 号同质化, 普遍限流 + 单月封禁案例 (T06 平台 timeline 2026-01).

6. **如果 想做 知识付费 / 训练营 funnel**, 则 文章必须 explicit CTA (引导加企微 / 报名页 / 资料包二维码) + 私域 1-3 周培育 + 漏斗 3 层 (免费 → ¥99-999 入门课 → ¥1999-29999 高客单课); 软引导转化几乎为零.
   - **来源**: T03-C.3 + T01-S020 Spenser + T01-S025 罗振宇.
   - **case**: Spenser 商学院 28 天训练营 ¥2499-15800 漏斗稳跑至 2025 (T01-S020); 反面 = 「写完就指望读者自己来报名」, 转化率 < 0.1% (业内估计).

7. **如果 AI 写作 输出 直接发文 不手改 + 不标识**, 则 (a) 读者 < 5 篇内识别 (「DeepSeek 味」/「GPT 味」) → 留言 backlash + 取关; (b) 2025-09-01 起《人工智能生成合成内容标识办法》强制标识, 不标 = 违规 + 平台限能力.
   - **来源**: T02 避坑 #2 + T03-F11 + T03-A4.8 + T06-B.7 + T06-S011.
   - **case**: 业内共识 — AI 用于选题 / 大纲 / 改稿 / 润色, 终稿必须手写保 voice; 头部 KOL (半佛 / 万维钢 / 刘润) 公开表态 AI 不做正文 (T01 演讲综合).

8. **如果 一篇推文 涉政 / 涉医 / 涉金融 / 涉营销 / AI 生成 / 软广 / 矩阵关联 / 标题党边界**, 则 上线前走《微信公众平台运营规范》8 项自审清单 (标题党 / 配图侵权 / 链接合规 / 二维码诱导 / CTA 话术 / 涉政涉黄涉医 / 软广「广告」标识 / AI 标识); 不走的概率 = 限流 / 删稿 / 失原创权 / 严重封号.
   - **来源**: T03-A5.5 + T06-B.1 + T06-B.5 + T06-B.7 + T06-B.8 + T06-B.10 + T06-B.11.
   - **case**: 咪蒙 2019 全网封禁 (T01-S031 / T04-S016) 是 8 项跳过 4 项 (标题党 + 涉政涉医引导 + 失实 + 不标软广) 的极端反例; 2026-01 微信单月封禁 2119 账号 (T06 平台 timeline) 是常规 enforce 节奏.

### 3.4 避坑清单 (10 条, ≥ 5 floor)

## 5. 表达 DNA (Voice)

### 5.1 voice signature (公众号味)

读者读 100 字就觉得「这是公众号的味道」, 主要 markers:

- **短段落 (2-3 行 max)** + 大量加粗 + 引用块 for 金句 — mobile reading microformat, 非美学选择
- **第二人称 + 直接对话 reader** (「你」「我们」反复出现; 「你有没有想过...」「我们都知道...」)
- **故事 / case / 数据 + 主张 + 反问** — 不是 wiki 罗列, 必有作者立场 + 二次传播 hook
- **段落间空行 + 表情符号 (情感号合用, 知识号慎用)** — 节奏分段
- **行业黑话自然嵌入** (10万+ / 在看 / 头条 / 推荐流 / 流量主 / 互选 / 原创标), 不解释
- **收尾常有 CTA** (留言 / 转发 / 在看 / 进群 / 课程 / 私信) — 显式 call-to-action 是行业默契, 不是冒犯; 反而无 CTA 显业余

### 5.2 流派 voice register markers

| 流派 | 句长 | 比喻习惯 | 语气 | 收尾 |
|------|------|----------|------|------|
| 段子化深度派 (半佛 / 三表) | 长短交替 + 反常识断言 | 利益博弈 + 风控视角 + 段子比喻 | 辛辣 + 自嘲 | 「不多说了」+ 暗讽 |
| 知识系统派 (万维钢 / 刘润) | 中长 + 学术引用 | 概率 / 实证 / 5 力 / 4P 模型 | 理性 + 中性 | 总结 + 推论 |
| 文化考据派 (六神磊磊 / 顾爷) | 长 + 古文穿插 | 古典文本 + 当代映射 | 幽默 + 自嘲 | 借古喻今 + 余味 |
| 情感生活派 (黎贝卡系) | 中短 + 第一人称叙事 | 消费选择 / 生活细节 | 温暖 + 代入 | 强 CTA + 商业化 hook |
| 商业咨询派 (刘润) | 短中等 + 模板化 | 5 力 / 4P / 增长黑客 | 干练 + 中性 | 「这就是 X 的 Y」+ 模型图 |

### 5.3 外行常犯的 voice 破绽 (outsider-tell, 一眼识破)

1. 「在这个 X 的时代」「最近大家都在讨论」「我们都知道」开篇
2. 段落超 5 行不分段; 全程无加粗 / 无引用块
3. 「写得不好请多多包涵 / 大家好我是新号」式寒暄
4. 罗列事实 + 数据 + 案例, 没有作者主张
5. 文末「希望能给大家一些启发」「谢谢观看」(无 CTA, 像 PPT 演讲)
6. 「头条 = 今日头条 app」/「服务号 = 企业号」/「软广 = 硬广」混淆
7. 全文「文学化书面语」, 无第二人称 + 无对话感
8. 标题用「浅谈 / 关于 X 的几点思考」(学术腔, 信息流必划过)
9. 配图用百度搜图截图; 排版用 Word 默认样式直接发
10. 单篇宏观分析 + 总结陈词, 5000 字无 hook 无节奏

### 5.4 厂商 / 卖课党话术拒绝 (这一行人绝不这样说)

- 「跟我学公众号月入 10 万」「7 天打造爆款」「保姆级教程」「逆袭」「财富密码」(全是销售课, T05 signal-vs-noise #1)
- 「2025 公众号最新算法揭秘」(微信算法不公开, 任何揭秘是猜, T05 #8)
- 「我用 3 天赚了 30 万」(标题与内容不符 = 标题党)
- 「免费送资料 + 加微信领」(funnel 入口 disguised as 知识, T05 #10)
- 「视频号联动让你涨粉 100 倍」(数据无源, T05 #5)
- 课程价格阶梯 ¥99 → ¥999 → ¥9999 → ¥29999 (销售漏斗 disguised, T05 #7)

### 5.5 对话样本库 (Step 5b — 4 类 × 各 ≥ 2 段)

> source: Track 01 figure 长访谈 + 出版书原话 / 转述; voice_confidence: medium (转述 + 原话混合, 多数来自第三方平台访谈而非公众号自家原文 — 黑名单约束).

#### 5.5.1 客户版 (面向读者 / 教育 / 信任建立)

- 「**再小的个体, 也有自己的品牌**. 公众号最初设计的初衷, 就是让每个个体, 都能通过内容找到自己的读者, 而不是被算法和大号淹没.」(source: T01-S001 张小龙 2019 微信公开课 PRO 演讲, 原话 + 转述综合; 客户场景: 平台哲学声明 / 长尾品牌叙事)
- 「**每天 5 分钟, 一个商业概念, 一个案例, 一句总结.** 这就是我对自己的训练 — 不是给你讲商业理论, 是帮你压缩一本商业书的核心.」(source: T01-S013 刘润《5 分钟商学院·商业篇》开篇, 原话 + 转述; 客户场景: 知识付费用户教育 / funnel 入口)
- 「**写作不是文学是商业**, 公众号写作的本质是个人品牌建设的最低成本方式. 你不写, 别人也在写; 你写了, 至少有人知道你是谁.」(source: T01-S020 Spenser《写作是最好的自我投资》+ T01-S021 36 氪 WISE keynote, 原话 + 转述; 客户场景: 训练营 funnel 入口教育)

#### 5.5.2 同业版 (KOL 之间 / 训练营 / 内部 / 私下)

- 「**你看到的所有事情, 都可以从利益博弈的角度去拆解.** 段子是糖衣, 数据 + 财经 + 风控背景才是药 — 内核不能让步, 段子是给读者降低认知门槛的礼物.」(source: T01-S007 半佛 36 氪长访谈《骚人有骚福》, 原话; 同业场景: 段子化深度派方法论自述)
- 「**别拿自己当渠道, 要有核心生产力, 要有自己的核心思想.** 没风格 = 没记忆点 = 没忠诚. 你做矩阵 / 蹭热点 / 借渠道, 短期能跑, 长期就是给别人打工.」(source: T01-S034 三表 极客公园新媒体实操分享, 原话; 同业场景: 三表门规 / KOL 立场声明)
- 「**内容创业是一场长期战争, 数据是这场战争的地图.** 但地图不是命令 — 它告诉你战线在哪, 不告诉你怎么打.」(source: T01-S038 / S039 徐达内 新榜内容节历年 keynote, 转述; 同业场景: 行业研究派对从业者训诫)
- 「**内容行业, 优秀的人以一当百.** 人力成本给足, 其他乱七八糟成本能砍就砍. 一个好编辑顶 10 个差编辑, 没有中间路线.」(source: T01-S007 半佛 36 氪访谈, 原话; 同业场景: 团队组织哲学 / MCN 老板视角)

#### 5.5.3 监管 / 平台 / 标准版 (公开场合 / 自审 / 合规)

- 「**不实信息、标题党、诱导分享、诱导关注**, 微信公众平台运营规范明文禁止. 累计违规 → 删文 → 限能力 → 永久封号. 这不是吓唬, 咪蒙 2019 全网封禁是先例.」(source: T06-S001《微信公众平台运营规范》原文 + T01-S031 / T03-S027 咪蒙封号报道; 监管场景: 平台规则陈述 / 自审基线)

- 「**「不转不是中国人 / 转发好运 / 分享后解锁」**」 — 触发「诱导分享」官方判定, 删文 + 限能力 (source: T06-S002 诱导分享判定 + T06-A.4.6 互动话术灰区; why 反例: 监管明文红线话术)
- 「**「我用 3 天赚了 30 万, 教你怎么做」**」 — 标题与内容不符 = 标题党, 系统化即触发限流 / 封号 (source: T06-A.4.2 标题党定义 + 咪蒙 case; why 反例: 短期诱导 + 长期反噬范式, 已被监管摧毁)
- 「**「家人们速来! 这个 AI 工具 yyds, 写公众号月入十万根本不是梦」**」 — 同质化叫卖 + 「家人们 / 速来 / yyds」短视频 register 错位 (公众号 register 是文字深度 / 段落短不等于口头化), 真 KOL 不这样写 (source: T05 Signal-vs-Noise #2 + Track 06 灰区, 推断 + 转述综合; why 反例: 抖音 / 小红书 register 错位 + 速成承诺 + AI 工具夸大三重)

**voice_confidence 标注**: medium — 4 类各 ≥ 2 段共 13 段, 客户版 + 同业版 + 监管版多为「原话 + 转述」混合 (源自第三方平台访谈 + 出版书 + 官方法规原文); 反例版来自 Track 05 signal-vs-noise heuristics + Track 06 灰区. 因公众号本体在调研黑名单, KOL 自家公众号最长的 voice 原文不进 source, 此处 voice 维度 baseline 为外部可观察层.

## 6. 质量基准 + 反模式

### 6.1 什么算「好」(5 条具体可验证, 全部业内估计 / 第三方观察)

1. **单篇打开率**: 头部号头条 4-8% (T06-A.1.6 secondary via 36kr); 中尾号 1.5-3% 大盘; **数据来源**: T06-S019 新榜年报 + 业内估计, 微信无官方公告.
2. **完读率**: 短文 > 40%, 长文 > 30% (业内估计 + T03-A7.1 复盘 SOP); 关联推荐流权重.
3. **在看率 > 1% / 转发率 > 0.5%** (业内估计, 视 vertical 而定 — 情感号 / 知识号偏高, 资讯号偏低); 在看是社交推荐分配信号 (T06-A.1.3).
4. **涨粉率 > 0.3% / 篇** (业内估计, 一篇 10w+ 通常涨 500-5000 粉, T03-S001).
5. **长期 12 个月数据稳定 / 上升 + 节奏 hard-stick 无断更** (T03-B3.1 + T03-F9 + 业内共识); 节奏 stable 是算法权重 baseline.

> 所有数字 caveat: 微信官方无公开数据, 来源 = 新榜年报 + 第三方观察 + 业内估计; 头部 KOL 自报数字普遍被认为夸大, 中尾 创作者羞于公开 (T05 + intake warnings).

### 6.2 反模式 (8 条, 每条 1 句 + 1 specific case)

1. **F1 选题没我的号 fit 校验** → 蹭来一次性陌生流量 + 取关率 > 涨粉率 + 算法画像污染. **Case**: 财经号写一篇娱乐爆款蹭流量, 来 5000 粉两周内取关 4000, 后续推荐流被算法压 (T03-F1).
2. **F2 标题党** → 完读 / 在看断崖 → 推荐流降权 → 系统化即触平台明文禁止 (T06-S001). **Case**: 咪蒙系 2017-2019 「致 X X 的一封信」「我为什么不嫁 X」公式, 2019-02 全网封禁 (T01-S031 / T03-S027).
3. **F3 开篇 50 字元话术** → 信息流卡片露出非钩子 → 0 打开. **Case**: 「最近大家都在讨论小米 SU7...」开篇, 信息流卡片露出此句, 打开率 0.3% (业内常见 case).
4. **F4 结构无主张 / wiki 罗列** → 无 hook → 无金句 → 无在看转发. **Case**: 「关于 AI 的几点思考」5000 字罗列国内外 AI 公司新闻, 完读率 8%, 在看 0 (业内常见).
5. **F6 排版段落长 + 无加粗 / 配图** → 移动端跳出率 > 70%. **Case**: 一篇 3000 字 Word 默认样式直接复制粘贴发, 即使内容质量高, 跳出率仍远高于排版规范版本 (T03-F6).
6. **F9 节奏断 (今天 5 篇下周 0)** → 算法对不稳定账号降权. **Case**: 日更冲刺 1 周后断更 2 周, 推荐流权重持续 4-6 周才恢复 (T03-S039).
7. **F10 商业化只做流量主** → 月入 ¥几百-几千卡死 (中位估计) + 不与读者建立 trust. **Case**: 中部号 5w 粉只开流量主, 月入 ¥300-800; 同体量号开互选 + 软广 + 私域 funnel 三路, 月入 ¥5000-50000 (业内估计区间, T03-F10 + T03-S041).
8. **F11 AI 全文生成 + 不标识** → (a) 读者 5 篇内识别 + 取关; (b) **2025-09-01 起强制标识**, 不标即违规 (T06-S011). **Case**: 2024-2025 AI 灰产矩阵号 50-500 号 ChatGPT 一键生成 → 2025-09 起部分平台已限流 + 2026-01 微信单月封禁 2119 号 (T06 平台 timeline).

## 7. 智识谱系

> 公众号写作主流派 + 奠基 → 当代代表 → 核心论点 → 互相分歧. 8 派 + 2 anti-school. 不抹平分歧.

### 7.1 平台 / 产品派
- **奠基**: 张小龙 (2011 微信 / 2012 公众号 launch)
- **当代代表**: 张小龙 + 微信团队 (微信公开课 PRO 历年 keynote)
- **核心论点**: 公众号 = 去中心化 + 长尾品牌; 反对纯算法信息流; 朋友推荐 > 算法推荐; 长期价值 > 短期流量
- **代表语料**: 「再小的个体也有自己的品牌」(T01-S001)
- **分歧**: vs 头部效应现实 (Spenser / 罗振宇 实操中追求中心化爆款); 「反算法」话术与「订阅号默认折叠 + 推荐流权重 59.2%」(T06-S019) 产品事实张力

### 7.2 段子化深度派
- **奠基**: 半佛仙人 (公众号 + B 站 矩阵 2018-2020 兴起) + 三表 (辛辣评论)
- **当代代表**: 半佛 / 三表 / 部分财经评论号
- **核心论点**: 段子是糖衣, 数据 / 财经 / 风控是药; 多平台 voice 统一; 利益博弈视角拆解一切
- **代表语料**: 「你看到的所有事情都可以从利益博弈的角度去拆解」(T01-S007); 「别拿自己当渠道」(T01-S034)
- **分歧**: vs 知识系统派 (万维钢认为段子化稀释深度); vs 传统财经记者 (认为段子化是降智)

### 7.3 知识系统派
- **奠基**: 罗振宇 (罗辑思维 2012) + 万维钢 (得到《精英日课》2016)
- **当代代表**: 万维钢 / 刘润 / 罗振宇 / 何加盐
- **核心论点**: 公众号 = 二手书评 + 三手洞察的「压缩转译」; 输出倒逼输入; 日更复利
- **代表语料**: 「用理工科思维理解世界」(万维钢《万万没想到》副标 T01-S010); 「做时间的朋友」(罗振宇跨年演讲 T01-S025)
- **分歧**: vs 文化考据派 (六神追求「独占壁龛」而非通识压缩); vs AI 时代 (知识压缩价值受 LLM 冲击, T04-S021 罗振宇 2025 跨年演讲主动讨论)

### 7.4 文化考据派
- **奠基**: 散文 / 杂文传统 + 公众号纪元的和菜头 (槽边往事 2013+) → 六神磊磊 (读金庸 2013+)
- **当代代表**: 六神磊磊 / 顾爷
- **核心论点**: 独占壁龛 + 借古喻今 (合规迂回) + 出版 + 公众号双轨 + 长期主义
- **代表语料**: 「我从来不为了写而写, 写之前必须先有一句让我自己拍案叫绝的话」(六神磊磊每经访谈 T01-S017)
- **分歧**: vs 学术派 (认为通俗化牺牲准确性); vs 个人 IP 派 (六神不卖课, 顾爷只出书, 不做训练营 funnel)

### 7.5 情感生活 / 商业转型派
- **奠基**: 黎贝卡 (异想世界 2014+) + 早期女性媒体
- **当代代表**: 黎贝卡 + 时尚 / 母婴 / 生活号
- **核心论点**: 情感生活 voice 不是「无内容」, 是「消费选择 / 生活态度」可信代言人; 自有品牌是商业化天花板
- **代表语料**: 「一支口红的快乐, 和一个包包的梦想」(T01-S024)
- **分歧**: vs 段子派 (黎贝卡反辛辣 voice); vs 咪蒙派 (黎贝卡选可信代言, 咪蒙选贩卖焦虑 — 同 vertical 不同价值观)

### 7.6 个人 IP + 课程派
- **奠基**: 罗振宇 (罗辑思维 / 得到 2012-2015) → Spenser (S 商学院 2017+)
- **当代代表**: Spenser / 何加盐 / 张良计
- **核心论点**: 公众号 = 入口, 训练营 / 课程 / 社群 = 出口; 个人品牌三层 (内容人格化 → 信任 → 商业化)
- **代表语料**: 「写作不是文学是商业」(T01-S020); 「知识产生不了行动和改变, 但社群可以」(T01-S021)
- **分歧**: vs 平台派 (Spenser 高调讲变现, 张小龙反对 funnel 销售感); vs 文化考据派 (六神不做课程, Spenser 强 funnel)

### 7.7 商业咨询派
- **奠基**: MBA 通识 + 麦肯锡式拆解 + 5 分钟商学院 3 段模板
- **当代代表**: 刘润 / 阑夕 / 部分商业咨询号
- **核心论点**: 商业概念可视化 / 模型化 + 中等长度 (1500-3000) + 强模板化便于复制
- **代表语料**: 「每天 5 分钟, 一个商业概念, 一个案例, 一句总结」(T01-S013)
- **分歧**: vs 长文派 (3000-8000 字 vs 5 分钟); 与知识系统派边界模糊 (大量本质在二次转译海外书)

### 7.8 媒体 / 行业研究派
- **奠基**: 徐达内 (新榜 2014+)
- **当代代表**: 新榜 / 刺猬公社 / 见实
- **核心论点**: 公众号生态需要中立第三方数据; 年度报告 + 大会 是行业风向标
- **代表语料**: 「内容创业是一场长期战争, 数据是这场战争的地图」(T01-S038)
- **分歧**: 新榜本身是利益相关方 (排行 + 培训 + 交易市场), 中立性争议持续

### 7.9 (anti-school) 标题党 + 情绪派 — 已被监管摧毁

- **典型**: 咪蒙 (2015-2019) + 衍生矩阵号
- **方法论 (反面)**: 4 阶段情绪曲线 (焦虑 → 共鸣 → 解决方案 → 行动指令) + 极端标题 + 流水线团队 + 矩阵分发
- **历史教训**: 短期 effective (2016-2018 报价行业第一梯队 ¥80-100w 头条, 业内估计) + 长期 toxic (内容真实性失守 + 价值观投机) + **2019-02 全网封禁** (T01-S031 / T03-S027 / T04-S016-018)
- **不可软化**: SKILL 仅作反面引用, 绝不背书; 「4 阶段情绪曲线」与情绪操纵 / 价值观投机内核绑定, 解耦后失效

### 7.10 (anti-school 灰区) AI + 矩阵派 — 2024-2025 新流派, 即将变黑

- **典型**: 匿名 MCN + 团队 用 ChatGPT / DeepSeek / Claude 批量生成 + 矩阵化注册 50-500 号 + 自动排版推送 + 流量主 / 互选自动接单
- **短期 reality**: 2024 部分团队月入 ¥5-50w (业内估计, 头部灰产, T03-H2)
- **政策时间线**: 2023-07 自媒体 13 条 (T06-S012) + 2024-04 无底线博流量专项 (T06-S013) + 2025-07 不实信息专项 (T06-S037) + **2025-09-01 AI 标识办法施行** (T06-S011) + 2026-01 微信单月封禁 2119 号
- **不可软化**: SKILL 标灰区 + 即将变黑, 不背书; 平台 + 监管 + 读者三重压力

### 7.11 跨派核心分歧 (3 条 still-debated)

- **A. 单篇爆款 vs 账号长期 IP**: 情感派 / 标题党派 (短期投机) vs 知识系统派 / 文化考据派 (长期沉淀) — 同一选题不同派给反向建议. 心智模型 §1.3 选边核心.
- **B. voice 优先 vs 商业漏斗优先**: 六神磊磊派 (voice 是壁龛, 不做训练营) vs Spenser 派 (voice 是入口, funnel 是真生意). 两派对「公众号是什么」定义不同.
- **C. 矩阵化 / AI 量产 vs 人格化 / 一人一号**: AI + 矩阵派 (视账号为可弃 inventory) vs 文化考据派 / 知识系统派 (账号 = 长期 IP 资产). 监管已在 B 派一侧加码.

## 8. 诚实边界 (9 条)

3. **保鲜期 3-6 月** (工具 / 平台规则 / 推荐流权重 / 监管). 心智模型层 1-3 年才动. (详 §8.3)
4. **视频号 + AI 冲击 = 公众号文章流量整体下行**. skill 教在收缩池塘里精钓, 不是把池塘做大. (详 §8.4)
5. **学派分歧无法统一** — SKILL honor §7 八派分歧, 用户在 §1.3 选边后全套 SOP 才 alignment. (详 §8.5)
6. **Locale 极强** — 100% zh-CN, 海外没真正对应物. 不强行迁移海外 newsletter playbook. (详 §8.6)
7. **数字 transparency 差** — 任何具体数字 必 caveat (业内估计 / 第三方观察 / ranged estimate + source_id), 永远不改善. (详 §8.7)
8. **反面案例不能软化** — 咪蒙 (banned) + AI 灰产 (灰区 + 即将变黑) 全程 banned frame, 不背书. (详 §8.8)
9. **平台依赖风险** — 鸡蛋分篮: 公众号 + 视频号 + 私域 + 跨平台 cross-post. 单平台依赖 = 平台政策变化时一夜归零. (详 §8.9)

### 8.1 一手主阵地在调研黑名单 — 结构性偏低不是失败

公众号写作的**最高纯度一手内容**就发表在公众号本体 (mp.weixin.qq.com), 但本调研系统将公众号列为 zh-CN 黑名单 (避免内容农场污染信源). 因此本 skill 一手率结构性偏低 (Track 01 一手 32% / Track 03 一手 38% / Track 04 一手 43% / Track 06 一手 42.5%), 已用 metadata 路径覆盖 (KOL 在 36kr / 虎嗅 / 晚点 / 澎湃 / 第一财经 / 21 经济 / 蓝鲸 / 界面 / 极客公园 / 爱范儿 / IT 之家 / 中国数字时代 / 个人独立站 长访谈 + KOL 出版书 + 张小龙微信公开课 PRO 演讲 + 新榜大会 keynote + 监管原文). **这是行业现实, 不是调研失败**. 用户想学具体 craft, 还得回到公众号本体 + KOL 付费圈 + 自己写一年.

### 8.2 90% 创作者做不到全职 — 不卖速成幻想

流量 / 收入极端集中头部. 业内估计中位号月入 < ¥100 流量主, 7 / 万比率 才出 10w+ (T06-S019: 2024 全年 30.78w 10w+ vs 日产 121w 篇). 头部 KOL 公开收入多被认为夸大. SKILL 任何商业化建议必带「头部 / 中部 / 尾部」三段 caveat, 不只谈头部数字.

### 8.3 保鲜期 3-6 月 (工具 / 平台规则 / 推荐流权重)

工具 (DeepSeek / Kimi / 即梦) 流派位次 6 月内可重排; 平台规则 (流量主门槛 / 互选抽成 / 推荐流权重 / 朋友推荐升级) 12 月内必有调整; 监管 (自媒体 13 条 / 清朗专项 / AI 标识办法) 6-12 月内必有新政. **心智模型层 (§1) 保鲜期 1-3 年**, 工具 / SOP 层 (§3 / §4) 必须每季 review. last_updated: 2026-05.

### 8.4 视频号 + AI 冲击 = 公众号文章流量整体下行

2020 短视频崛起 + 2023-2025 AI 内容泛滥 + 订阅号默认折叠 + 推荐流红利耗尽 → 公众号文章整体流量曲线下行. 本 skill 教在**收缩的池塘里把鱼钓好**, 不是把池塘做大. 任何「公众号还能不能做」问题先确认: 你做的是收缩池塘里的精钓, 不是 2015-2018 的红利冲浪.

### 8.5 学派分歧无法统一 — SKILL honor 分歧

§7 八派 + 两 anti-school 对同一问题给完全不同建议 (单篇爆款 vs 长期 IP / voice 优先 vs funnel 优先 / 矩阵 vs 一人一号). SKILL 必须并列写, 不能假装统一. 用户在 §1.3 选边后, 全套 SOP 才有 alignment.

### 8.6 Locale 极强 — 海外没真正对应物

100% zh-CN, 微信生态封闭. Substack / Medium 概念近 (newsletter / 长文 / 创作者经济), 但 ecosystem 远 (订阅制 vs 广告 + 漏斗 / 算法权重逻辑 / 监管框架 完全不同). 不强行迁移海外 newsletter playbook.

### 8.7 数字 transparency 差 — 任何具体数字必 caveat

头部 KOL 公开广告 / 课程收入 多被认为夸大; 中小创作者羞于公开; 微信官方对 流量主 CPM / 推荐流权重 / 互选实际成交价 / 涨粉数据 无官方公告. SKILL 任何具体数字 (CPM ¥3-30 / 互选抽成 5% / 月入区间 / 转化率 0.5-5%) 都挂 source_id + 「业内估计」/「第三方观察」/ 「ranged estimate」caveat. 永远只能 caveat 状态, 不会随调研深化改善.

### 8.8 反面案例不能软化 — 咪蒙 / AI 灰产 必须 banned frame

- **咪蒙 (2017-2019, 全网封禁)**: 史上最大反面教材, 必须以「**短期 effective + 长期 toxic + 已禁**」frame 引用. 「4 阶段情绪曲线」不可解耦为可用 framework — 其有效性与情绪操纵 / 价值观投机内核绑定, 解耦后失效. (T01 anti-figure + T03-H1 + T04 anti-canon)
- **AI 批量产号灰产 (2024-2025)**: 标灰区 + 即将变黑. 2025-09 AI 标识办法 + 平台限流 + 监管整治 enforcement 在加强. **不背书**. (T03-H2 + T04 anti-canon)

### 8.9 平台依赖风险 — 鸡蛋分篮

微信封号 / 限流 / 推荐流改版 屡见不鲜. 头部号 (咪蒙 / 部分政论号 / 部分医疗号) 一夜腰斩有过先例. SKILL 必写「鸡蛋分篮 — 公众号 + 视频号 + 私域 + 跨平台 cross-post」(B4.4 跨平台 IP 化). 单平台依赖 = 平台政策变化时一夜归零.

## 9. Agentic Protocol — 研究维度 (5 个)

> 拿到新问题 (e.g. 「我要写一篇关于小米 SU7 的公众号文章」), 公众号写作者按这 5 个**互斥**维度做功课. 每个维度独立产出结论, 不 overlap; 5 个就够 (公众号写作不复杂到 7 个有差别的研究维度).

### 9.1 选题 fit 校验 (我的号 × 这一议题)

- **看什么**: (a) 这一议题 vs 我账号定位 / vertical 相符度; (b) 我读者画像 (年龄 / 性别 / 城市 / 阅读偏好) 对该议题兴趣度; (c) 我历史推文 同主题数据 (打开率 / 完读 / 在看); (d) 我有没有跨界优势 (信息差 / 视角差)
- **在哪看**: 公众号后台「数据 → 用户分析 / 来源关键词」 + 历史推文 Excel / 飞书库 + 自己 12 月 voice 主张梳理 + 私域群 / 朋友圈 quick poll
- **输出格式**: `[选题 fit]: yes/no/conditional + 角度 (我可写的 1 个 unique 切入) + 头条/次条/三条 排位假设`

### 9.2 议题当前生态热度 + 同 vertical TOP 5 已写视角扫描

- **看什么**: (a) 微信指数 + 百度指数 + 微博热搜 + 抖音热榜 + 知微事件库 5 平台热度交叉; (b) 是否已 ≥ 24h 发酵 (公众号是慢平台, 不抢首发); (c) 同 vertical TOP 5 KOL 近 7 天已写该议题的角度摘要 (新榜 + 飞瓜 + 看一看 + 知微); (d) 我能找到的 反差视角 / 数据角度 / 行业人 insider 视角
- **在哪看**: 微信指数 (公众号内嵌) + 百度指数 + 巨量算数 + 微博热搜 + 抖音热榜 + 知微事件库 + 新榜热榜 + 看一看精选
- **输出格式**: `[热度]: 高/中/低 + 发酵时长 + [蹭与否]: 蹭/不蹭/差异化蹭 + [我的差异化角度]: 1-2 句 unique 切入`

### 9.3 voice + 结构 + 大纲选择

- **看什么**: (a) 主题适合哪种 voice register (§7 段子化 / 知识系统 / 文化考据 / 情感生活 / 商业咨询 5 派之一); (b) 哪种结构 (短平快 500-1500 字 / 长深度 3000-8000 字 / 故事化 1500-3000 字); (c) 哪种骨架 (三段论 / 总分总 / 倒金字塔 / 主张-论证-反驳 / 时间线 / 对比); (d) 钩子模板 (8 选 1) + 大纲骨架
- **在哪看**: 内化的流派 register + 自家 voice 模板库 + 该 vertical 头部 KOL 的同类目稿 (作参考非模仿) + AI 工具 (DeepSeek / Kimi) brainstorm 大纲
- **输出格式**: `[voice]: 1 派 + [结构]: 1 种 + [骨架]: 1 种 + [钩子]: 1 模板 + [大纲]: 5-8 行 bullet`

### 9.4 平台规则自审 (上线安全 check)

- **看什么**: 8 项自审 — (a) 标题党边界 (能 deliver 吗 / 信息流前 22 字露出 OK 吗); (b) 配图侵权 (视觉中国系 / 百度搜图 / AI 名人脸 红线); (c) 链接合规 (外链白名单 / 二维码诱导关注边界); (d) CTA 话术 (「不转不是中国人」「分享解锁」黑名单); (e) 涉政 / 涉医 / 涉金融 / 涉黄 红线; (f) 软广是否标「广告」(《互联网广告管理办法》); (g) AI 生成内容是否标识 (2025-09-01 起 AI 标识办法); (h) 矩阵关联号同质化 (自媒体 13 条 + 2025 不实信息专项)
- **在哪看**: 《微信公众平台运营规范》原文 (developers.weixin.qq.com 微信开放社区, T06-S001) + 网信办原文 (cac.gov.cn) + 国务院《互联网广告管理办法》(T06-S015) + AI 标识办法 (T06-S011) + 微信开放社区诱导分享 / 诱导关注判定文档 (T06-S002 / S003)
- **输出格式**: `[8 项 check]: 通过/有风险/必修补 + [修补点]: 1-3 条 (具体改什么)`

### 9.5 变现路径 + 时效 / 复用 校准

- **看什么**: (a) 这篇是 流量主 (被动 CPM) / 互选广告 / 软广 / 知识付费 funnel CTA / 私域 funnel CTA / 直接卖货 / 出版引流 哪一种 placement; (b) 该 placement 是否 fit 当前账号商业化 phase (B 流 4 阶段); (c) 时效强度 — 强 (1 周衰减, 不复用) / 中 (1 月衰减, 适合季度回看) / 弱 (1 年衰减, 可系列化 + 定期重发); (d) 是否系列化 / 后续重发节奏
- **在哪看**: 自家商业化路径设计 (B1.1 假设 + B4.2 5-7 路) + 该篇 internal CTA 结构 + 时效判断 (热点 / 评论 / 知识 / IP 故事 不同时效) + 历史系列效果数据
- **输出格式**: `[商业 placement]: 1 类 + [phase fit]: yes/no + [时效]: 强/中/弱 + [系列化建议]: 是/否 + [重发节奏]: 1 周 / 1 月 / 1 季 / 不重发`

> **互斥性自检**: 5 个维度独立产出不同结论 — §9.1 输出 选题 yes/no + 角度; §9.2 输出 蹭与否 + 差异化; §9.3 输出 voice + 结构 + 大纲; §9.4 输出 上线安全 + 修补; §9.5 输出 商业 placement + 时效 + 系列化. 没有维度的结论是另一维度的输入但能在另一维度独立得出 — pass 互斥. (Step 9 反模式自检: 维度数 5 ≤ 心智模型数 5 × 2 = 10 OK; 5 个 ≥ 简单行业 3-5 floor.)
