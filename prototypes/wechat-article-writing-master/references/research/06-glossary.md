## Layer A — 行业黑话 (33 terms)

### A.1 流量 / 数据 系 (8 terms)

#### A.1.1 10万+ (10w+)
- 中英: 10万+ / 10w+ / 100k+ reads
- 一句话: 单篇文章阅读量突破 10 万的爆款 — 公众号生态的最低「爆款」门槛 (evidence: [T06-S019])
- 谁在用: 全行业, 是 KOL / 广告主 / 编辑 心中的隐性 KPI
- 易混淆: 外行以为「10 万 = 10 万独立读者」, 内行知道实际是 PV (含同人多次打开 + 第三方平台导流计数), 与 UV 不严格对应
- First-time context: 2014-2015 公众号红利期, 阅读量超过 10w 显示「10万+」字样而非具体数字 (微信产品逻辑保护隐私) — 自此 10w+ 成行业 social proof 通货
- Evidence: [T06-S019, T06-S020] — 2024 全年 30.78 万篇 10w+, 日产 121 万篇, 即每 1 万篇出 7 篇 10w+

#### A.1.2 头条 / 次条 / 三条 / 次次条
- 中英: 头条 (top-slot) / 次条 (second slot) / 三条 / 次次条 (slot 4+)
- 一句话: 一次群发推文中的位置序号 — 头条价值远大于次条, 越往后越衰减 (evidence: [T06-S019, secondary])
- 谁在用: 接广告报价 / 派稿决策 / 自媒体内部「头条留给商业 vs 留给情怀」的内部讨论
- 易混淆: 「头条」也指「今日头条 app」, 公众号内部说「头条」专指当次推送第一篇位置
- First-time context: 订阅号一天可群发 1 次 8 篇, 服务号一月 4 次 — 内部分配产生位置经济
- Evidence: 头条打开率约 4.3%, 第 8 条约 0.4% — 数量级差异 (secondary 报道 via 36kr)

#### A.1.3 在看
- 中英: 在看 / 「Wow」(2018-2019 短暂叫法)
- 一句话: 公众号文末点击「在看」按钮, 触发该文进入「看一看」社交推荐流 — 朋友能看到你 in-看的文章 (evidence: [T06-S022])
- 谁在用: 编辑 / 作者求转化的明文 CTA, 文末「点个在看再走」是模板话术
- 易混淆: 不是「赞」, 是社交推荐信号; 老 user 仍会说「好看」 (旧版按钮文案)
- First-time context: 2018 微信把「点赞」改为「好看」, 2019 改为「在看」, 同步推「看一看 — 朋友在看」 (社交推荐流)
- Evidence: [T06-S022]

#### A.1.4 看一看 / 推荐流
- 中英: 看一看 / Top Stories / 推荐流 (algorithmic feed)
- 一句话: 微信发现页里的算法 + 社交推荐内容流 — 2017 上线, 2021-2023 逐步并入订阅号信息流 (evidence: [T06-S022, T06-S035])
- 谁在用: 投放 / 数据分析师 / 编辑研究「为什么我这篇被推荐了 / 为什么没被推」
- 易混淆: 外行把「看一看」与「搜一搜」混; 看一看是 push, 搜一搜是 pull
- First-time context: 2017 推出, 含「朋友在看」(社交) + 「精选」(算法) 两栏
- Evidence: [T06-S022, T06-S025]

#### A.1.5 订阅流 (折叠后) vs 推荐流
- 中英: 订阅流 (subscriber stream) vs 推荐流 (recommended stream)
- 一句话: 2020 后订阅号默认按时间倒序 + 部分折叠, 2021-2024 在订阅号入口加入「推荐」tab — 创作者从「拼粉丝量」变成「拼算法推荐分」 (evidence: [T06-S019, T06-S022])
- 谁在用: 任何关心打开率断崖的运营, 数据后台明确分 「公众号消息」「聊天会话」「朋友在看」「看一看」「推荐」等流量来源
- 易混淆: 「订阅流」≠「订阅号」; 订阅号是账号类型, 订阅流是用户端浏览路径
- First-time context: 2021-03 公众号大改版, 信息流「乱序」, 「推荐」tab 上位
- Evidence: 2024 公众号后台数据显示「推荐」占阅读 59.2% — 已超过自然订阅流量 [T06-S019]

#### A.1.6 打开率
- 一句话: 群发抵达粉丝中, 真正点开文章的比例 — 公众号生态 1.5%-2% 是行业大盘, 顶号 4-8% (evidence: secondary via 36kr)
- 易混淆: 邮件 EDM 行业的 open rate 概念在公众号场景定义不同 (公众号「抵达」≠「送达」, 折叠后只有打开订阅号入口才算抵达)
- 首次接触: 公众号后台 → 数据 → 群发分析

#### A.1.7 完读率 / 阅读完成度
- 一句话: 用户从开篇读到结尾的比例 — 长文号 vs 短文号 differential 巨大, 与算法推荐分相关
- 谁在用: 长文派 (万维钢 / 刘润) 关键指标; 短平快派不重视
- 易混淆: 不等于「停留时长」 — 跳读 / 翻看图都算停留, 但完读率算到底部 anchor

#### A.1.8 涨粉率 / 取关率
- 一句话: 单篇 / 单日的关注 + 取关人数 / 比率 — 涨粉 - 取关 = 净增 (evidence: T06-S040)
- 谁在用: 内容定位调整 / 商业广告效果评估
- 易混淆: 取关高不一定是内容差, 也可能是矩阵号导流后的自然流失

### A.2 平台规则 系 (7 terms)

#### A.2.1 原创标 / 原创权
- 中英: 原创标 (Original Mark)
- 一句话: 注册满 X 天 + 持续输出 + 无违规 后由微信官方授予的「原创声明」权限, 文章左上角带「原创」二字标识 (evidence: [T06-S017])
- 谁在用: 新号最痛苦的第一关 — 没原创权就没流量主资格 + 易被洗稿
- 易混淆: 外行以为「我写的就是原创」, 内行知道「原创标」是平台机制 (无标 = 平台不替你保护抄袭, 也不开流量主)
- First-time context: 2015 微信公众平台引入原创保护机制, 2018 一年标记原创 3900 万篇 + 拦截涉抄袭 13 万篇 [T06-S017]

#### A.2.2 白名单转载 / 转载授权
- 一句话: 原创号通过后台「转载授权」开放白名单, 第三方号可在不修改的前提下转载 + 自动署名 + 不计入抄袭 (evidence: [T06-S004])
- 谁在用: 矩阵号互推 / 行业合作 / 知识号互换流量
- 易混淆: 「白名单」不等于「转给某号」 — 是「允许这个号转我所有原创文」的批量授权
- Evidence: [T06-S004]

#### A.2.3 留言 / 留言精选
- 一句话: 公众号文末读者评论 — 作者后台「精选」后才会展示给所有读者 (evidence: T06-S001)
- 谁在用: 互动设计 / 二次传播抓手 — 高赞精选评论本身可成爆款延展
- 易混淆: 2018-08 后新注册号无留言功能, 老号才有 (是行业内常识但外行不知, 是隐性资产分水岭)
- First-time context: 2018-08-15 后新申请公众号默认无留言功能

#### A.2.4 赞赏
- 一句话: 读者主动给作者打赏 ¥1-¥256 不等 — 作者收入直达 (无平台抽成, 微信支付通道费除外) (evidence: T06-S001)
- 谁在用: 高内容价值号 (六神磊磊 / 万维钢) 的次要变现
- 易混淆: 不是「赞」(thumbs up), 是金钱打赏 — 命名歧义
- First-time context: 2015 推出, 2018 因苹果税中断 iOS 端, 2019 恢复 (微信对苹果作了妥协)

#### A.2.5 服务号 vs 订阅号
- 一句话: 公众号两大账号类型 — 服务号月 4 次群发 / 消息直达对话框 / 可接微信支付; 订阅号日 1 次群发 / 折叠在订阅号入口 (evidence: [T06-S001, secondary])
- 谁在用: 注册前决策点 — 个人 / 自媒体几乎必选订阅号, 企业服务选服务号
- 易混淆: 外行以为「服务号是企业号」 — 实际是产品形态选择, 不是企业认证级别
- First-time context: 2013-08 微信公众平台分化为 服务号 + 订阅号

#### A.2.6 折叠
- 一句话: 订阅号消息默认收缩为一个入口, 用户主动点开才能看到逐条推送 — 2020 信息流改版核心 (evidence: [T06-S025])
- 谁在用: 所有运营痛点的根, 「打开率断崖」直接归因
- First-time context: 2020-05 启用「乱序信息流」, 「时间维度」被部分剥离

#### A.2.7 限流 / 封号 / 违规
- 一句话: 微信对违规账号 3 级处分: 1) 删文 + 警告; 2) 限制部分能力 (失原创权 / 限流量主); 3) 永久封号 (evidence: [T06-S001, T06-S003])
- 谁在用: 任何接触政治 / 灰色 / 标题党边界的号
- 易混淆: 限流 ≠ 限号; 限流是单篇推荐流被压低, 限号是全号能力冻结
- Evidence: [T06-S001] 公众平台运营规范 全文 / [T06-S003] 诱导关注案例集锦

### A.3 变现 系 (8 terms)

#### A.3.1 流量主
- 中英: 流量主 / Ad-Share Publisher
- 一句话: 微信广告联盟 — 文末 / 文中插入广告位 (CPM / 点击) , 平台分成给账号 (evidence: [T06-S007, T06-S009])
- 谁在用: 中长尾号主要变现 (头部不靠这个)
- 门槛: 历史 5000 粉 → 2023 降到 500 粉 (一阶段) → 2024 100 粉 (返佣商品) / 500 粉 (程序化广告) (evidence: [T06-S027])
- 易混淆: 「流量主」≠「广点通」 — 广点通是腾讯广告体系内品牌名, 流量主是公众号接入广点通的「身份」
- Evidence: [T06-S007, T06-S009, T06-S027]

#### A.3.2 互选广告
- 中英: 互选广告 / Mutual Selection / 腾讯互选
- 一句话: 微信官方撮合 广告主 ↔ 账号 的「报价 + 接单 + 投放 + 结算」平台 — 强制走平台过水, 不能私下结算 (evidence: [T06-S006, T06-S008])
- 谁在用: 中头部带广告位的账号, 是 2023 之后 公众号商单的「合规出口」
- 抽成: 历史 30% → 2023-06 政策窗口 0% → 2023-07 起 5% (evidence: [T06-S008])
- 易混淆: 外行听「互选」以为是「互相选择朋友推广」, 内行知道是 微信广告平台的撮合服务
- Evidence: [T06-S006, T06-S008]

#### A.3.3 软广 / 软文 / 品牌植入
- 一句话: 在内容里隐性带品牌信息的广告 — 公众号最古老的变现形态, 2023 互联网广告管理办法 后强制标「广告」(evidence: [T06-S015])
- 谁在用: 几乎全行业, 报价从 ¥500 (尾部) 到 ¥50 万 (头部头条) 不等
- 易混淆: 「软广」≠「植入硬广」; 软广要求读者读完才发现是广告
- 灰色边界: 不标「广告」二字 → 违反互联网广告管理办法 第 9 条 (evidence: [T06-S015])

#### A.3.4 硬广
- 一句话: 直接标注「广告」+ 品牌 banner 形式的广告内容 — 多见于头部号「广告报价」专项位
- 易混淆: 公众号语境里「硬广」≠ 信息流硬广, 多指公众号文末的整版品牌广告

#### A.3.5 接单 / 商单
- 一句话: 自媒体接广告主商业合作的统称 — 含报价 / 排期 / 标的物审核 / 平台过水
- 谁在用: 全行业, 是变现的核心动词
- 易混淆: 「商单」与「投流商单」不同 — 后者是付费推流形态

#### A.3.6 流量主 CPM / eCPM
- 一句话: 千次曝光收益 — 公众号流量主 CPM 普遍 ¥3-¥30, 头部内容向号可到 ¥50+ (evidence: secondary)
- 谁在用: 计算流量主 ROI 时的核心单位
- 易混淆: 公众号 CPM ≠ 信息流 CPM (后者市场化竞价, 前者腾讯统一池)

#### A.3.7 知识付费 funnel
- 一句话: 公众号引流到 课程 / 训练营 / 社群 等付费产品的转化漏斗 — 万维钢 / 刘润 / Spenser 代表
- 谁在用: 知识系统派 / 商业咨询派
- 易混淆: 不是「公众号变现」, 是「公众号作为漏斗顶端」, 真正收钱在外部产品

#### A.3.8 付费阅读 / 付费会员 (灰度)
- 一句话: 2020 推出, 单篇可设 ¥1-¥208 付费墙 — 注册 > 3 月 + 无严重违规 + 已发 3 篇原创 的订阅号可申请, 仅个人号灰度 (evidence: [T06-S031, T06-S039])
- 谁在用: 少数原创深度号实验; 整体生态接受度低
- First-time context: 2017 张小龙在公开课提过, 2020-01 灰度开放
- Evidence: [T06-S031] / 36kr 反思「公众号缺这点钱?」[T06-S039]

### A.4 内容 craft 系 (6 terms)

#### A.4.1 选题
- 一句话: 决定写什么 — 「自上而下」(热点 / 节日 / 平台政策) vs 「自下而上」(读者反馈 / 后台搜索词) 两路, 头条 / 次条选题战略不同
- 谁在用: 所有自媒体, 占创作时间 30-50%
- 易混淆: 外行以为「选题 = 想点子」, 内行知道是「热度 × 调性 × 角度 × 时效」的乘积决策

#### A.4.2 标题党
- 一句话: 用极端 / 误导 / 悬念标题诱导点击, 内容与标题严重不符 — 平台明文禁止「不实信息 + 标题党」, 屡犯永久封号 (evidence: [T06-S001, T06-S028])
- 灰色边界: 「数字标题」「悬念标题」与「标题党」之间无明确量化线 — 行业内默认「能 deliver 不算标题党, 不能 deliver 才算」
- 反面案例: 咪蒙 2019 全网封禁 [T06-S028, T06-S029]

#### A.4.3 钩子 / 开篇
- 一句话: 文章前 50-150 字 — 公众号信息流卡片 + 摘要只露这一段, 直接决定打开率
- 谁在用: 高阶编辑必抠的位置, 内部「钩子三步」「钩子模板库」是产业级 know-how

#### A.4.4 金句
- 一句话: 文章中适合截图传播 / 引用 / 朋友圈 caption 的高浓缩句子 — 二次传播抓手
- 谁在用: 故事 / 鸡汤 / 商业咨询 派最重视; 资讯号弱化
- 易混淆: 不是名人名言, 是作者自己设计的「能脱离上下文也通的句子」

#### A.4.5 排版 / 配图
- 一句话: 段落 2-3 行 / 1-2 张配图 / 加粗 + 引用块 + 表情符号 + 段间空行 — 移动阅读优化 (microformat) 不是设计美学
- 谁在用: 全行业, 三大工具 135 编辑器 / 秀米 / 壹伴
- 易混淆: 「排版」≠「美编」 — 排版是阅读节奏工程, 美编是视觉创作

#### A.4.6 互动话术 / 文末 CTA
- 一句话: 文末固定的「点赞 + 在看 + 转发」三连引导 — 显式 call to action, 与「诱导分享」红线擦边但合规版被默许 (evidence: [T06-S002, T06-S005])
- 灰色: 「不转不是中国人」「转发好运」「分享后解锁」等明显跨线 — 触发删文 + 限能力 (evidence: [T06-S002])

### A.5 生态联动 系 (4 terms)

#### A.5.1 视频号联动
- 一句话: 公众号 ↔ 视频号「一体号」绑定 — 公众号底部挂视频号卡片 + 视频号直播间关注转公众号粉丝 (evidence: [T06-S036])
- 谁在用: 2024-2025 几乎所有头部号必做的二次曝光 + 涨粉路径
- First-time context: 2020 视频号上线, 2022-2023 逐步开放 公众号 ↔ 视频号 互导

#### A.5.2 搜一搜 SEO
- 一句话: 微信内搜索框 (搜一搜) 的内容 SEO — 优化标题 / 摘要 / 标签让内容被搜出 (evidence: secondary)
- 谁在用: 2023-2025 红利期, 长尾流量重要补给
- 易混淆: 不是 百度 SEO, 算法逻辑完全不同 (微信生态封闭 + 优先订阅 + 社交信号权重高)

#### A.5.3 朋友圈分享卡片
- 一句话: 公众号文章被转到朋友圈时, 自动生成的「封面图 + 标题 + 摘要」卡片 — 卡片设计直接影响朋友圈打开率
- 谁在用: 所有重视二次传播的号

#### A.5.4 服务号 模板消息 (回流)
- 一句话: 服务号通过「模板消息」push 已互动用户回到对话框 — 仅限服务通知 (订单 / 提醒 / 行程), 不可营销 (evidence: secondary)
- 谁在用: 电商类 / 工具类服务号; 内容号几乎不用
- 易混淆: 「模板消息」≠「群发消息」 — 模板消息可一天 10 万条但限服务场景

## Layer B — 平台规则 / 标准 / 政策 (12 entries)

### B.1 《微信公众平台运营规范》
- 发布机构: 微信团队 (腾讯)
- 发布日期: 持续更新, 历次重大修订 (2018 / 2020 / 2022 / 2024)
- 一句话核心: 公众号「行为准则总则」 — 含诱导分享 / 诱导关注 / 标题党 / 抄袭 / 刷量 等全部红线 (evidence: [T06-S001])
- 对公众号写作的直接影响: 是所有违规判定的依据 — 累计违规 → 删文 → 限能力 → 永久封号
- 当前 enforce 强度: 高 — 自动化机器扫描 + 人工合议双轨
- Source: [T06-S001]

### B.2 微信原创保护机制
- 发布: 微信团队 (2015 上线, 2018 引入「洗稿合议机制」)
- 一句话: 原创声明 + 抄袭拦截 + 洗稿合议 三件套 (evidence: [T06-S004, T06-S017, T06-S030])
- 对写作的影响: 原创权 = 流量主资格 + 抄袭维权 + 转载授权管理 的底层 prerequisite
- 当前强度: 高 (2018 拦截 13 万篇), 但「洗稿」(非字面抄袭, 思想 / 结构挪用) 维权仍困难 — 现行著作权法不保护思想
- Source: [T06-S004, T06-S017, T06-S030]

### B.3 流量主开通门槛
- 发布: 微信团队 (历次调整 2017 / 2020 / 2023 / 2024)
- 历年变化:
  - 早期 (2014-2017): 5000 粉 + 内容审核
  - 2020 前后: 500 粉 (新手扶持) (evidence: [T06-S027])
  - 2024: 100 粉 (返佣商品) / 500 粉 (程序化广告) (evidence: search T06-S027 series)
- 抽成: 服务费 10% (返佣商品) / 平台对广告主分成不公开
- 对写作影响: 门槛降低意味着 long-tail 都能变现, 但 CPM 单价被稀释
- Source: [T06-S007, T06-S009, T06-S027]

### B.4 互选广告 (腾讯互选)
- 发布: 微信团队 / 腾讯营销 — 2023-06-30 后强制走平台
- 抽成历程: 30% → 0% (2023-06 政策窗口) → 5% (2023-07 起)
- 类目限制: 医疗 / 金融 / 招商 等强监管类目受限
- 对写作影响: 接广告不能再私下 PA, 必须走互选 — 报价 / 排期 / 结算全透明
- Source: [T06-S006, T06-S008]

### B.5 《互联网广告管理办法》 (2023-05-01 施行)
- 发布机构: 国家市场监督管理总局 (令第 72 号)
- 一句话: 互联网广告必须可识别 — 软文 / 测评 / 探店 一律标「广告」二字 (evidence: [T06-S015])
- 对公众号写作的影响: 软广不标 = 违规, 最高罚款 ¥10 万, 微信已配套 2023-06-30 强制接入互选广告
- 当前强度: 中高 (重点查头部账号 + 行业举报触发)
- Source: [T06-S015]

### B.6 《生成式人工智能服务管理暂行办法》 (2023-08-15 施行)
- 发布机构: 国家网信办 + 7 部门

- 对公众号写作影响: 间接 — 监管对象是 AI 服务提供方而非创作者, 但配套 2025 标识办法直接影响创作者

### B.7 《人工智能生成合成内容标识办法》 (2025-09-01 施行)

- 对公众号写作直接影响: 写完整 AI 生成的推文 必须标 — 否则违规; 微信生态尚在落地 enforcement 中
- 当前强度: 新规, 2025-09 起强制, 平台正搭检测能力
- Source: [T06-S011, T06-S034]

### B.8 《关于加强「自媒体」管理的通知》 (2023-07, 「自媒体 13 条」)

### B.10 「清朗·整治『自媒体』无底线博流量」专项行动 (2024)
- 发布: 国家网信办 (2024-04)

### B.12 《中华人民共和国个人信息保护法》 (PIPL, 2021-11-01 施行)

- 对公众号写作影响: 间接 — 公众号收集读者数据 (跳转表单 / SCRM / 留资) 必须明示用途 + 取得同意; 头部账号自营商城 / 训练营 受影响最大

### B.13 网络主播 / 头部自媒体 偷税漏税 公开案例 (2021-2026)

- Source: [T06-S018, T06-S032, T06-S038]

## Layer C — 反模式 / 灰区 / 黑话陷阱 (8 entries)

### C.1 「刷量 / 机刷 / 点击党」(刷阅读 + 刷在看 + 刷粉)
- 表面意思: 短期人为放大数据
- 实际操作: 通过第三方平台 ¥4/千阅读 灰产服务, 注水阅读 + 注水粉丝
- 风险等级: 黑 — 微信明文禁止, 2016-2024 多次大规模清零 + 永久封号
- 已被处罚: 2016-09 微信屏蔽刷量工具 一夜戳破大量「10w+ 大号」; 头部号被处罚案例公开 (secondary 报道)
- 当前 enforce: 持续高压, 异常涨粉 / 阅读 1-3 天限号
- Evidence: [T06-S001] 运营规范 + 央视报道 secondary

### C.2 「打开率优化 / 钩子工程」过度 → 标题党
- 表面意思: 优化点击率
- 实际: 起标题与内容偏离 — 「我用 3 天赚了 30 万」(内容是发广告)
- 风险等级: 灰 → 黑 (个体单文灰, 系统化即黑)
- 已被处罚: 咪蒙 2019 全网封禁 (evidence: [T06-S028, T06-S029])

### C.3 「同质化矩阵 + 矩阵分发」
- 表面意思: 一个 IP 多号扩大覆盖
- 实际: 同一团队多账号发结构 / 内容近似文 — 互相引用洗白来源
- 风险等级: 灰 → 黑 (2023 「自媒体 13 条」+ 2025 不实信息专项行动 都点名 evidence: [T06-S012, T06-S037])
- 当前: 严管「一人一号 / 一企两号」 — 但执法以平台为主, 不痛不痒矩阵仍存

### C.4 「AI 批量生成 + 矩阵分发」
- 表面意思: AI 提效
- 实际: ChatGPT / DeepSeek 一键生成 5000 字 + 多号分发 SEO
- 风险等级: 灰 (2024-2025), 即将变黑 — 2025-09 AI 标识办法 + 网信办整治矩阵号 (evidence: [T06-S011, T06-S012])
- 当前 enforce: 微信生态 AI 内容标识检测刚开始落地, 处罚先例尚少, 但政策已就位

### C.5 「私域闭环 + 公众号 leak」
- 表面意思: 私域运营
- 实际: 公众号文末 / 菜单引导加企微 + 卖课卖货, 用奖品 / 资料包诱导关注 + 私聊
- 风险等级: 灰 — 触线 「诱导关注」(evidence: [T06-S003, T06-S005])
- 当前: 「资料包 + 二维码」型仍普遍, 微信对个例反应慢, 但矩阵化导流被严查

### C.6 「咪蒙式情绪化操控」(2017-2019 反面案例)
- 表面意思: 共情能力强
- 实际: 系统化制造焦虑 + 制造对立 + 极端故事 (含虚构) + 卖 ¥99-¥1999 课程
- 风险等级: 黑 — 直接处罚先例
- 已被处罚: 2019-02 咪蒙公众号注销 + 全平台封禁; 团队多账号被关停 (evidence: [T06-S028, T06-S029])

### C.7 「软广不标 sponsored / 广告」
- 表面意思: 软广「无痕」植入
- 实际: 不标「广告」二字, 让读者以为是真心推荐
- 风险等级: 黑 — 《互联网广告管理办法》明文违规, 最高 ¥10 万罚 (evidence: [T06-S015])
- 当前: 2023 起市场监管出手, 已有判罚先例; 公众号 2023-06-30 后强制接入互选广告标「广告」

### C.8 「假原创 / 洗稿」
- 表面意思: 原创
- 实际: 改写他人作品的表达, 保留思想 + 结构 — 不构成著作权侵权, 但违平台原创规则
- 风险等级: 灰 (法律层面) + 黑 (平台规则层面)
