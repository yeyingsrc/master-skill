## Books / 系统性著作

### 📖 1. 《新媒体运营》(秦阳 / 秋叶 系列)

- **Author / 作者**: 秦阳 + 秋叶 团队 (秋叶 PPT 体系延伸)
- **Year**: 2017 首版, 持续修订 (近版 2018-2020)
- **Type**: textbook (大学教材级 + 培训教材双用)
- **One-liner**: 国内系统性教 "新媒体岗 / 公众号运营从入门到能干活" 的工具书第一阵营, 涵盖选题 / 标题 / 排版 / 涨粉 / 转化全链路, 教学版而非思想版.
- **核心论点 (3-5)**:
  1. 公众号是新媒体岗职业能力的核心 artifact, 围绕单篇产出能力构建工作 SOP
  2. 选题 = 自上而下 (热点 / 节日) + 自下而上 (用户后台 / 留言) 双轨
  3. 标题 + 摘要 + 首图 = 信息流卡片 3 件套, 不可分割考虑
  4. 排版规范化 (短段落 / 加粗 / 引用块 / 空行) 是基础合规, 不是加分项
  5. 涨粉 = 内容自然 + 互推 + 跨平台引流; 中尾部不可依赖自然涨粉单一路径
- **读完得到什么**: 新人能独立完成"选题 → 起标题 → 写开篇 → 排版 → 推送 → 看数据复盘"一个完整 cycle, 不再问"我下一步该做什么". (evidence: [T04-S006])
- **难度**: 入门
- **Endorsement evidence**:
  - [type: blog_secondary] 数英 / 知乎书单 / 行业内多人引用为"入门第一本" (evidence: [T04-S006, T04-S030])
  - [type: course_syllabus] 大量大学新闻传播学院 / 培训机构选作"新媒体运营"课参考教材 (evidence: [T04-S006])
  - [type: figure_short] 秋叶 PPT 体系本身是大 IP, 跨书系交叉背书 (evidence: [T04-S006])
- **一手书 vs 二手转述**: 一手 (作者直接写)
- **是否被新版 supersede**: 持续修订, 当前推荐取近 3 年新版 (老版讲的 2017 推荐机制已显著过时)
- **可信度**: medium-high
- **Decay risk**: medium (平台规则节会衰减, 写作 craft 节稳定)
- **Phase 2 use**: textbook 基底, 教 SOP / 工具栈; mental model 不期待来自此书

  5. 微信生态内必须把多个 touchpoint (公众号 + 视频号 + 小程序) 串成网
- **读完得到什么**: 对 WeChat 平台机制有结构化认知, 不再"听说"而是"知道为什么".
- **难度**: 入门 → 进阶
- **Endorsement evidence**:

  - [type: figure_short] 秋叶 IP 背书 (evidence: [T04-S006])
- **一手书 vs 二手转述**: 一手
- **是否被新版 supersede**: 持续修订
- **可信度**: medium-high
- **Decay risk**: medium-high (微信生态变化快, 当前版应取近 2 年内)
- **Phase 2 use**: 平台机制章节直接用作"业内通识"

### 📖 3. 《写作是最好的自我投资》(Spenser)

- **Author / 作者**: Spenser (陈立飞)
- **Year**: 2018 / 中信出版社
- **Type**: 方法论 + 自传混合
- **One-liner**: 个人 IP + 课程变现派的代表方法论书 — 把"用公众号写作建立个人品牌, 把品牌变成课程 / 训练营 / 私域收入"这条漏斗讲透, 写作技巧只是入口.
- **核心论点 (3-5)**:
  1. 写作不是文学是商业 — 公众号写作的本质是"个人品牌建设的最低成本方式"
  2. 个人 IP 三层: 内容人格化 → 信任建立 → 商业化 funnel
  3. 写作 9 个角度 (定位 / 选题 / 结构 / 开篇 / 标题 / 文字 / 案例 / 金句 / 复盘)
  4. 公众号是入口, 训练营 / 知识付费是出口
  5. 个人 IP 的天花板比"做号"高, 但门槛是"敢露脸"
- **读完得到什么**: 理解为什么"光做内容不变现"的中尾部公众号永远不赚钱; 知道商业漏斗每一层的转化逻辑.
- **难度**: 入门 → 进阶 (商业部分高于写作部分)
- **Endorsement evidence**:
  - [type: figure_long] Spenser 本人在多个商业访谈中重述本书框架 (evidence: [T04-S004])
  - [type: blog_secondary] 多平台 / 知识付费课程引用其"写作变现"框架 (evidence: [T04-S004])
  - [type: course_syllabus] Spenser 写作训练营 / 个人 IP 课的官方配套书 (evidence: [T04-S004])
- **一手书 vs 二手转述**: 一手
- **可信度**: medium (商业部分有"幸存者偏差"风险, 课程派 ROI 不公开)
- **Decay risk**: low (个人 IP 框架稳定; 具体平台规则会变)
- **Phase 2 use**: 变现漏斗章节直接用作"个人 IP 派 mental model"种子

## Core Concepts (20-30)

### 💡 20. 头条 / 次条 / 三条 推送位

- **Tier**: tier-1
- **One-liner**: 公众号订阅号一次推送多条内容时, 第 1 条 (头条) 与第 2-N 条 (次条 / 三条 / 次次条) 是分隔的注意力位, 头条流量通常占总推送阅读 ~70-80% (业内估计).
- **来源**: 微信公众号订阅号原生产品设计 (一手 T04-S010 张小龙演讲 + T04-S013 新榜数据反推)
- **关联概念**: 单条推送 (服务号) / 推送频次 / 信息流卡片 / 推荐流
- **行业内当前理解 vs 原始定义**: 原始设计就是"主次"逻辑; 当前理解还叠加了推荐流时代的"算法 vs 人工排序"博弈 (头条 = 主编决策, 次条 = 试错位)
- **为什么进入 canon**: 是公众号选题最基础的资源分配单位, 任何"今天发什么"决策都从这里开始
- **常见误用**: 新人把所有好选题堆头条 → 次条永远只发凑数 → 次条 / 三条数据永远不会积累
- **Endorsement evidence**: (evidence: [T04-S006, T04-S007, T04-S013])

### 💡 21. 订阅流 vs 推荐流 vs 看一看

- **Tier**: tier-1
- **One-liner**: 公众号文章被读到的 3 个主要来源 — 订阅号信息流 (你关注的号) / 推荐流 (算法 + 朋友推荐) / 看一看 (社交背书). 三者背后的流量逻辑完全不同, 写作 / 标题策略也要分别适配.
- **来源**: 微信原生产品 (T04-S010 张小龙演讲 + T04-S022 36 氪 2025 改版分析)
- **关联概念**: 在看 / 朋友推荐 / 算法推荐 / 折叠订阅
- **行业内当前理解 vs 原始定义**: 张小龙原本反对"信息流"标签 (T04-S010), 但 2020+ 算法权重持续提升; 2025 又转向"朋友推荐"权重 (T04-S022)
- **为什么进入 canon**: 公众号文章为什么会被读到, 90% 在这 3 个池子里决定
- **常见误用**: 只为订阅流写 → 推荐流卡片永远不点; 只为推荐流写 → 老读者疏远
- **Endorsement evidence**: (evidence: [T04-S010, T04-S010, T04-S022, T04-S023])

### 💡 22. 在看 + 留言 + 转发 (社交推荐三件套)

- **Tier**: tier-1
- **One-liner**: 微信生态内一篇文章的"社交动力学"三个动作 — 在看 (轻量背书) / 留言 (深参与) / 转发 (跨场域分发). 三者背后的二次曝光机制不同.
- **来源**: 微信原生产品 + 张小龙 2019 演讲对"点赞 → 在看"演化的解释 (T04-S010)
- **关联概念**: 看一看 (在看的下游) / 朋友推荐 / 群分享 / 朋友圈
- **行业内当前理解 vs 原始定义**: 原"点赞"→ 2018 改"在看" (强调社交而非点赞奖励作者), 演化中
- **为什么进入 canon**: 文章发出后 6 小时内的命运基本由这三件套决定
- **常见误用**: 在结尾求点赞 (旧逻辑) — 当前更有效是求留言 / 转发
- **Endorsement evidence**: (evidence: [T04-S010, T04-S013])

### 💡 23. 原创标 + 白名单 转载

- **Tier**: tier-1
- **One-liner**: 公众号原创权 (申请准入 + 标记原创) + 白名单 (允许其它号转载并标注源头) 是中头部号的内容护城河, 没有原创标 = 无法防御抄袭 + 无法获取转载流量分成.
- **来源**: 微信原生产品 (评估期门槛持续变化, 早期较易申请, 现需粉丝 / 历史质量门槛)
- **关联概念**: 抄袭维权 / 转载分成 / 流量主门槛
- **行业内当前理解 vs 原始定义**: 原始设计是为打击抄袭; 演化为内容方议价筹码
- **为什么进入 canon**: 决定一个号是"内容生产方"还是"被搬运方"
- **常见误用**: 新号天天发但不申请 → 错过窗口 / 标准提高后更难申请
- **Endorsement evidence**: (evidence: [T04-S006, T04-S007])

### 💡 24. 流量主 (CPM 广告分成)

- **Tier**: tier-1
- **One-liner**: 公众号底部 / 文中插入腾讯广告, 按 CPM (千次曝光) 给作者分成. 准入门槛历史为 5000 粉, 中尾部号月入流量主**业内估计 < ¥100**, 头部号可月入万元 — 高度头部集中.
- **来源**: 微信公众号原生功能 (T04-S007 教材 + T04-S013 行业数据)
- **关联概念**: 互选广告 / 软广 / 知识付费 funnel
- **行业内当前理解 vs 原始定义**: 原是普惠创作者激励; 现实是头部效应极端
- **为什么进入 canon**: 99% 中尾部创作者唯一接触到的"被动收入"来源, 但 ROI 真相是: 业内 / 公开 估计 90% 创作者永远做不到月入 ¥1000+ 流量主 (evidence: [T04-S009 长尾理论 + T04-S013 新榜年报])
- **常见误用**: 把流量主当主收入 → 永远等不到; 应当作"通胀对冲零钱", 主收入靠互选 / 知识付费 / 私域
- **Endorsement evidence**: (evidence: [T04-S006, T04-S007, T04-S013])

### 💡 25. 互选广告 (品牌 - KOL 撮合 + 平台抽成)

- **Tier**: tier-1
- **One-liner**: 腾讯官方 KOL 与广告主撮合平台, **业内估计平台抽成约 50% 左右** (历史调整中, 范围 30-50%). 是中头部公众号的主流变现.
- **来源**: 微信公众号 / 腾讯广告 原生功能
- **关联概念**: 软广 / 流量主 / 议价空间
- **行业内当前理解 vs 原始定义**: 原是品牌透明化采买; 实际操作中头部仍通过私下议价绕过
- **为什么进入 canon**: 决定一个号能否进入"中头部商业化"
- **常见误用**: 新号期望接互选 → 门槛达不到; 中部号不报互选 → 错过透明定价锚点
- **Endorsement evidence**: (evidence: [T04-S006, T04-S007])

### 💡 26. 知识付费 funnel (公众号 → 课程 / 训练营)

- **Tier**: tier-1
- **One-liner**: 公众号 = 引流入口, 真正变现在课程 (得到 / 自营) / 训练营 / 社群 / 出书. 罗振宇 / 万维钢 / 刘润 / Spenser / 何加盐 都是此模式.
- **来源**: 罗辑思维 → 得到 (2012-2016 早期实践 T04-S021), 后被广泛复制
- **关联概念**: 个人 IP / 私域 funnel / 社群运营 / 跨平台分发
- **行业内当前理解 vs 原始定义**: 早期 "知识付费红利"已退潮 (2019+), 当前是存量博弈 (T04-S021)
- **为什么进入 canon**: 头部 / 中头部公众号最重要的商业化路径
- **常见误用**: 没有持续内容资产就开课 → 韭菜投诉率高; 不做免费内容养鱼就直接收费
- **Endorsement evidence**: (evidence: [T04-S004, T04-S021, T04-S026])

### 💡 27. 私域 funnel (公众号 → 企微 / 个人号 → 卖货)

- **Tier**: tier-1
- **One-liner**: 公众号 = 公域入口, 引流到企微 / 个人号 (私域), 在私域里销售产品 (实物 / 课程 / 服务). 与 SCRM / 私域运营 skill 接口.
- **来源**: 微商 → 完美日记 / 瑞幸 等品牌私域案例 → 自媒体迁移
- **关联概念**: 公众号矩阵 / 视频号联动 / 草料二维码 / 凡科活码
- **行业内当前理解 vs 原始定义**: 2020+ 私域是公众号最大变现增量 (T04-S023)
- **为什么进入 canon**: 中部 / 长尾公众号的现实主力变现
- **常见误用**: 引流到私域却没有产品 / 服务 → 私域死池; 引流暴力 → 微信封号
- **Endorsement evidence**: (evidence: [T04-S023, T04-S004])

### 💡 28. 短平快 vs 长深度 vs 故事化 (文章结构三型)

- **Tier**: tier-1
- **One-liner**: 公众号文章结构三种范式 — 短平快 (500-1500 字 / 情感号 / 资讯号) / 长深度 (3000-8000 字 / 知识号 / 商业号 / 半佛 万维钢式) / 故事化 (case study / 人物特写 / 黎贝卡 顾爷式).
- **来源**: 跨流派归纳 (T04-S002 + T04-S004 + T04-S019 共同样本)
- **关联概念**: voice 模板 / 选题深浅 / 阅读完成率
- **为什么进入 canon**: 写作前的"格式选择"是任何选题落地的第一个决策
- **常见误用**: 一种结构走天下 → 选题与结构错配 (例: 财经分析硬写成故事化, 失去信任)
- **Endorsement evidence**: (evidence: [T04-S002, T04-S004, T04-S005, T04-S019])

### 💡 29. 开篇钩子 (前 50 字)

- **Tier**: tier-1
- **One-liner**: 信息流卡片 / 公众号摘要只露文章前 50-100 字, 这 50 字决定打开率. "钩子句" craft 是当代公众号写作核心技能 (与传统纸媒"导语"不同 — 后者读者已下决心读, 前者还没).
- **来源**: 信息流时代产物 (T04-S007 教材 + T04-S022 推荐流改版下更重要)
- **关联概念**: 标题工程 / 摘要 / 首图 / 信息流卡片
- **为什么进入 canon**: 推荐流时代决定打开率的第一杠杆
- **常见误用**: 开篇先寒暄"大家好我是 X X" → 信息流卡片露出的是寒暄, 0 打开
- **Endorsement evidence**: (evidence: [T04-S006, T04-S007, T04-S004])

### 💡 30. 标题工程 (数字 / 反差 / 悬念 / 利益 / 情绪 / 名人)

- **Tier**: tier-1
- **One-liner**: 标题 6 种基本范式 (数字 / 反差 / 悬念 / 利益 / 情绪 / 名人借势) + 微信推荐流 / 看一看的标题适配规则 — 不是技术, 是工程化决策树.
- **来源**: 跨教材归纳 (T04-S006 + T04-S007) + 反面案例咪蒙 (T04-S016)
- **关联概念**: 标题党 (反模式) / 开篇钩子 / 信息流卡片
- **行业内当前理解 vs 原始定义**: 早期"标题党"红利期 (2014-2018) → 咪蒙封号后行业 self-regulate + 平台监管 → 标题党红利显著下降
- **为什么进入 canon**: 这是公众号写作可以学的最 mechanical 的部分
- **常见误用**: 把"标题工程"等同"标题党" → 短期数据好, 长期掉粉 / 限流
- **Endorsement evidence**: (evidence: [T04-S006, T04-S007, T04-S016, T04-S017])

### 💡 31. 排版规范 (短段落 / 加粗 / 引用块 / 配图)

- **Tier**: tier-1
- **One-liner**: 公众号默认排版 = 短段落 (2-3 行) + 关键词加粗 + 引用块用于金句 + 段落间空行 + 每 500-800 字一张配图. 不是"加分项", 是"不踩坑底线".

- **关联概念**: 配图 / 工具栈 / voice 模板
- **为什么进入 canon**: 移动端阅读 + 信息流时代, 排版不规范 = 跳出率高
- **常见误用**: 套用 135 编辑器最花哨模板 → 视觉污染 → 反而掉粉
- **Endorsement evidence**: (evidence: [T04-S006, T04-S007])

### 💡 32. 视频号联动 (公众号文末挂视频号 + 反向导流)

- **Tier**: tier-1
- **One-liner**: 2020+ 微信推荐视频号成为重点, 公众号文章底部 / 中间挂视频号卡片, 视频号直播 / 短视频引流到公众号 — 是 2024-2025 微信生态官方鼓励路径.

- **行业内当前理解 vs 原始定义**: 2020 才有视频号, 完全新概念
- **为什么进入 canon**: 公众号单独做 = 流量天花板; 与视频号联动是 2024-2025 增长杠杆
- **常见误用**: 公众号文章硬塞无关视频号 → 用户体验差; 视频号内容不是为公众号读者准备 → 反向引流失败

### 💡 33. 折叠订阅号 (2020 改版)

- **来源**: 微信官方改版 + 业内复盘 (T04-S010 张小龙 2019 演讲已铺垫)

- **为什么进入 canon**: 公众号"阅读量普跌"的最直接产品原因
- **常见误用**: 把阅读量下跌归因于"自己内容不好" → 错过结构性事实, 跟错方向

### 💡 35. 涨粉漏斗 (内容自然 / 互推 / 矩阵 / 跨平台 / 视频号联动)

- **Endorsement evidence**: (evidence: [T04-S006, T04-S007, T04-S019])

### 💡 36. 矩阵号 + IP 化运营

- **Tier**: tier-2
- **One-liner**: 主号 + 子号 + 矩阵分发, 子号定位差异化 (同 IP 不同垂类, 或不同 IP 共主理人), 矩阵之间互相导流 + 广告分摊. 是头部团队 / MCN 的标配.
- **来源**: 行业实践归纳 (T04-S013 新榜年报 + T04-S016 咪蒙旗下矩阵案例 + T04-S017)

- **Endorsement evidence**: (evidence: [T04-S013, T04-S016, T04-S017])

### 💡 37. AI 写作辅助边界 (选题 / 大纲 / 改稿, 不直接生成)

## Anti-Canon / 反面案例

### 反面案例 R1. 咪蒙现象 (2015-2019)

- **标签**: short-term effective + long-term toxic + ultimately banned (短期有效 + 长期有毒 + 终被封禁)
- **时间线** (evidence: [T04-S017 澎湃])
  - 2015: 马凌化名咪蒙开公众号
  - 2016-2018: 巅峰期粉丝过千万, 流量 / 广告报价行业第一梯队, 团队孵化矩阵号 ("才华有限青年"等)
  - 2019-01-29: "才华有限青年"发《一个出身寒门的状元之死》, 因虚构嫌疑 / 煽动焦虑被群嘲, 当晚被微信屏蔽
  - 2019-02-01: 咪蒙团队道歉, 公众号停更 2 月, 微博永久关停
  - 2019-02-21: 咪蒙公众号注销, 头条 / 微博 / 知乎 / 凤凰等同步全网封禁
- **方法论核心 (反面)** (evidence: [T04-S016, T04-S017, T04-S018])
  - "热点 + 金钱 + 性 + 暴力" 选题公式
  - 标题"态度鲜明 + 一定冒犯性 + 引起读者站队" — 标题党升级版
  - 4 阶段情绪曲线 (焦虑 → 共鸣 → 解决方案 → 行动指令) — 让读者完成转发 / 评论闭环
  - 写作团队化 / 流水线化, 主笔 + 数据 + 选题分工
  - 矩阵分发, 同方法论复制多号
- **为什么是反面**:
  1. 内容真实性失守 (《寒门状元之死》虚构嫌疑) → 信任崩塌
  2. 价值观短期投机 (贩卖焦虑 / 性别对立) → 监管 + 平台 + 公众三重反弹
  3. 方法论本身是"可重复的", 任何人学了都能短期爆款 — 但这也意味着任何人都会被同样的反作用力打中
  4. 全网同步封禁说明这不是单一平台的偶发处理, 是行业共识的红线
- **为什么仍然入 canon (反面)**:
  - 咪蒙团队的"4 阶段情绪曲线"方法论 2015-2018 被几乎所有情感号 / 营销号借鉴, 是这个行业当时最有影响力的 playbook — 不读 ≠ 不存在
  - 咪蒙封号是行业自我规范 + 监管的分水岭事件 — 任何后来者必须知道这条红线在哪
  - 当代仍有"咪蒙化"的灰色操作 (尤其 AI 批量产号 2024-2025 流派) — 反面案例帮助新人识别
- **不可软化背书**:
  - SKILL.md 必须明确写: "咪蒙方法论短期有效是真实的, 但长期反噬 + 终被封禁是结构性 — 不要尝试"
  - 不要把"4 阶段情绪曲线"简化为可用 framework — 它的有效性与其"情绪操纵 / 价值观投机"内核绑定, 解耦后失效
- **Endorsement evidence**: (evidence: [T04-S016, T04-S017, T04-S018])

### 反面案例 R2. AI 批量产号灰产 (2024-2025)

- **标签**: 灰色, 短期可走 / 长期被平台规则收紧
- **现象**: 2024+ 用 ChatGPT / DeepSeek / Claude 批量生成公众号文章, 矩阵化注册数百号自动发布, 主要靠流量主 / 互选广告分成. 内容质量低, 读者识别率高, 但因量大 + 长尾, 短期能收益.
- **为什么是反面**:
  1. 平台已开始针对"AI 痕迹明显 + 矩阵化"账号限流 (业内观察, 暂无官方公告) (evidence: [T04-S022, T04-S023] 间接)
  2. 互选广告 / 流量主政策已收紧虚假账号
  3. 价值: 给整个行业制造噪音, 拉低读者对"公众号原创"的信任度 — 是公共资源的负外部性
- **入 canon 的理由**: 新创作者必须知道这条赛道存在 + 为什么不应该走
- **Endorsement evidence**: (evidence: [T04-S022, T04-S023] — 行业观察, 一手信号薄弱, 待 Track 02 / 03 补充)

## 智识谱系种子

> 公众号写作行业的思想流派, 每派 = 奠基 → 当代代表 → 核心分歧.

### 派 1. 科技自媒体派 (财经 / 商业评论 / 行业观察)

- **奠基**: 长尾理论 (Chris Anderson) + 早期 IT 评论 (阑夕 / 三表 / 见识) — Chris Anderson 的"长尾"框架让"小众科技垂类号能活"成为合理预期
- **当代代表 (与 Track 01 figures 对接)**: 半佛仙人 (段子化深度) + 阑夕 (科技评论) + 三表 (辛辣评论)
- **核心方法**: 数据 / 实证 + 段子 / 比喻 (降低认知成本) + 多平台分发 (公众号 + B 站 + 知乎 矩阵)
- **核心分歧**:
  - "深度 vs 段子": 半佛派坚持"段子是糖衣, 数据是内核"; 部分传统财经记者认为段子化是降智
  - "议题选择": 是追每日热点 (流量) vs 守垂类 (信任)
- **Endorsement evidence**: (evidence: [T04-S019, T04-S020, T04-S009])

### 派 2. 知识付费 / 知识系统派

- **奠基**: 罗振宇 (罗辑思维) 2012-2016 + 万维钢的"科普 + 学术转译"模式
- **当代代表 (Track 01 对接)**: 罗振宇 / 万维钢 / 刘润 / 何加盐 — "免费内容养鱼 → 付费课程变现"漏斗
- **核心方法**: 把英美学术 / 商业书系统化转译 + 持续日更 / 长更 + 课程 / 出书 复利
- **核心分歧**:
  - "通识 vs 垂直": 罗振宇 / 万维钢走通识跨界; 刘润 / 何加盐走商业 / 营销垂直
  - "课程 vs 出书": 罗振宇 / 万维钢长期出书, Spenser 偏训练营 (高 ARPU)
  - "AI 时代的位置": 知识付费的"知识压缩"价值在 AI 全面普及后受冲击 (T04-S021 罗振宇 2025 跨年演讲)
- **Endorsement evidence**: (evidence: [T04-S002, T04-S003, T04-S001, T04-S021, T04-S026])

### 派 3. 文化考据 / 历史评论派

- **奠基**: 散文 / 杂文传统 (鲁迅 / 林语堂) + 当代延伸 (王力 / 易中天) + 公众号代纪元的六神磊磊
- **当代代表 (Track 01 对接)**: 六神磊磊 (金庸 + 唐诗) + 顾爷 (艺术普及) + 部分历史向号
- **核心方法**: 锁定一个古典 / 文化壁龛 + 借古喻今 + 幽默语调 + 长期主义
- **核心分歧**:
  - "考据严谨 vs 通俗化": 学术派认为通俗化牺牲准确性; 通俗派认为不通俗等于不传播
  - "时事映射 vs 纯古典": 时事映射收益高但风险大 (敏感词); 纯古典安全但流量天花板低
- **Endorsement evidence**: (evidence: [T04-S005])

### 派 4. 情感 / 生活流派 (含个人 IP 商业化)

- **奠基**: 早期女性媒体 (鲍鲸鲸 / 苏芩) + 黎贝卡 (时尚) + Spenser (个人 IP 商业化集大成)
- **当代代表 (Track 01 对接)**: 黎贝卡 (时尚 → 电商私域) + Spenser (个人 IP → 训练营) + 部分情感号
- **核心方法**: 第一人称叙事 + 生活细节 + 强人格化 + 商业化漏斗 (私域 / 课程 / 商品)
- **核心分歧**:
  - "情感 vs 投机": Spenser 派认为"教别人变好"是正向; 咪蒙派曾走"贩卖焦虑"反面 (T04-S016)
  - "私域转化 vs 知识付费": 黎贝卡走电商私域, Spenser 走训练营, 不同 ROI 曲线
- **Endorsement evidence**: (evidence: [T04-S004])

### 派 5. 商业咨询 / 系统派

- **奠基**: MBA 通识 + 麦肯锡式拆解 + 5 分钟商学院的"概念 + 案例 + 总结" 3 段模板
- **当代代表 (Track 01 对接)**: 刘润 + 李善友 + 部分商业咨询公众号
- **核心方法**: 商业概念可视化 / 模型化 + 中等长度 (1500-3000 字) + 强模板化便于复制
- **核心分歧**:
  - "深度 vs 频率": 刘润日更模板化 vs 部分号坚持低频长文 (3000-8000 字)
  - "原创 vs 转译": 大量商业咨询号本质在"二次转译"海外内容, 与万维钢的"知识系统派"边界模糊
- **Endorsement evidence**: (evidence: [T04-S001, T04-S025, T04-S027])

### 派 6. 媒体研究 / 行业观察派

- **奠基**: 传统媒体研究 + 新榜 (徐达内, 自媒体行业第一数据机构)
- **当代代表 (Track 01 对接)**: 新榜 / 刺猬公社 / 见实 / 极客公园创作者经济条线
- **核心方法**: 数据驱动 + 第三方中立 + 服务行业内部决策
- **核心分歧**: 数据机构 (新榜) 是利益相关方 (帮助交易 / 卖广告) vs 评论方 (刺猬公社) 是行业批评者, 立场不同
- **Endorsement evidence**: (evidence: [T04-S013, T04-S015])

## 诚实边界 (结构性, 必须写入 SKILL.md)

公众号写作行业的最佳一手素材就发表在公众号 (mp.weixin.qq.com) 上, 但本调研系统将公众号列为 zh-CN locale 的黑名单 (避免内容农场 / 转述爬虫污染信源). 因此本 Track 04 canon 收集采用的是 **metadata-level 路径**:

1. **书** — 通过出版社 + ISBN + douban subject page (verified_primary) 引用, 而非公众号原文
2. **KOL 长访谈 / 方法论** — 优先取 KOL 在第三方平台 (36 氪 / 虎嗅 / 数英 / 澎湃 / 极客公园 / IT 之家) 的访谈或文字版, 而非 KOL 自己公众号原文
3. **张小龙等腾讯官方演讲** — 优先取第三方平台 (IT 之家 / 数英 / 雷锋网 / 博客园) 的转载全文, 而非微信公开课公众号原帖
4. **新榜等行业数据** — 取新榜官网 (www.newrank.cn) 而非新榜公众号

**这导致**:
- 本 canon 一手率 (verified_primary 比例) 结构性偏低 (~43%), 这是 zh-CN 公众号写作行业**调研结构特征**, 不是研究缺陷
- 部分 KOL 最 hot 的方法论文章 (如咪蒙的"4 阶段情绪曲线"原始陈述) 只能通过第三方深度复盘 (T04-S016, T04-S017, T04-S018) 间接引用, 信号有轻微损失
- 行业 "业内估计" / "公开数据" 类数字 (如流量主 CPM 实际值, 互选广告平台抽成实际百分比) 因平台无官方公告, **永远只能以 caveat 状态存在**, 这不会随调研深化而改善

**SKILL.md 必须明确告知用户**: "本 skill 给出的方法论 80%+ 来自 KOL 在第三方平台公开访谈 + 出版的书 + 腾讯官方公开课 + 新榜等第三方数据机构; 因调研策略排除了公众号本体, **你看到的 canon 是该行业 KOL 公开陈述的过滤版本** — 任何想要更深入的细节 (尤其 KOL 在公众号本号的具体技法演化), 建议直接订阅 + 读他们的公众号. 这条边界源于调研系统的内容农场防御策略, 不是 KOL 的能力代表."
