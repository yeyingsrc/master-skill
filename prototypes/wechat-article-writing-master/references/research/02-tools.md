#### 🛠️ 1.1 公众号后台 (wechat-mp-backend)

- **Vendor / Maker**: 腾讯 / 微信团队
- **Category**: 1. 公众号后台 + 微信官方
- **Tier breakdown**: 免费 (全功能 — 群发 / 数据 / 流量主 / 互选 / 原创 / 评论 / 视频号联动 / 付费阅读 灰度)
- **Core function** (在公众号写作中的具体场景): 唯一合法发文入口; 编辑器 + 群发 (24h 内可改错别字 + 撤回) + 数据后台 (阅读 / 在看 / 转发 / 完读率) + 留言精选 + 流量主 / 互选 接入 + 原创声明 + 白名单转载授权 (evidence: [T02-S001, T02-S004])
- **2025 default for what segment**: 全 segment 必备, 0 替代品
- **Strength**: 唯一合法入口; 数据真实 (其他工具数据反推, 永远不如后台准)
- **Weakness / Pitfall**: 编辑器极弱 (移动端预览差 + 富文本不稳); 历史只能在 PC 端发文, 移动端发文 2024 起灰度但功能受限; 数据后台 cohort 维度弱 (没有真正的「这批新粉 30 天留存」视图)

## 避坑清单

1. **❌ 视觉中国 直接复制图片用于公众号** — 历史多起 KOL 因图片侵权被索赔 ¥几千-几万 / 张; 「先发再起诉」是行业 negative 共识; 法务团队保护下才能用 (evidence: [T02-S031, T02-S047])

2. **❌ AI 写作直接全文复制粘贴发文** — 读者识别率高 (套路化句式 + 「DeepSeek 味」/ 「GPT 味」), 即使绕过原创检测, 长期掉粉; 业内共识: AI 仅做选题 / 大纲 / 改稿, 终稿手写 + 保留个人 voice (evidence: Track 04 canon AI 章节 + T02-S046)

3. **❌ 流量主当主收入** — 中位号月入 < ¥100 是常态, CPM 极低 (¥1-5/千次曝光); 「跟我学公众号月入十万 (靠流量主)」99% 是销售课 (evidence: [T02-S003, T02-S011])

4. **❌ 矩阵号机械批量发布** — 微信对「内容同质化 + 批量发布」一直在风控, 矩阵号有限流 / 封号风险 (业内 / 公开案例); 工具 (微小宝 / 易撰) 合规但用法触雷 (evidence: [T02-S041, T02-S042])

5. **❌ 标题党工具滥用** — 易撰等历史爆款标题库套用 → 标题党 + 同质化; 微信对「夸张标题 / 标题与内容不符 / 诱导分享」一直在风控, 咪蒙 case (2019 封号) 是史上最大反面教材 (evidence: Track 04 canon 咪蒙反面)

6. **❌ 直接百度搜图右键保存** — 90% 侵权风险, 摄图网 / Unsplash 是底线; 即使免费图也要核 license (Pixabay 部分图源不明)

7. **❌ AI 文生图涉真实人物 / 名人脸** — Midjourney / 即梦 生成名人脸 / 政要脸 → 侵权 + 涉政风险; AI 抽象概念图 OK, 具体真人脸高风险

8. **❌ 公众号引流到个人号 / 企微 用「先关注后回复领资料」** — 微信对「诱导关注」一直罚款 + 警告; 草料活码 / 金数据 工具合规但用法 (二维码 + 利益诱导) 触雷 (evidence: [T02-S043, T02-S044])

9. **❌ 只看微信指数选题, 不交叉百度 + 抖音** — 微信指数算法不公开, 单看一平台数据偏; 三平台 (微信 / 百度 / 巨量算数) 交叉用是标准做法 (evidence: [T02-S001, T02-S027, T02-S028])

10. **❌ 排版工具花哨模板套用** — 头部知识号反而少用 135 / 秀米 花式模板, 倾向极简; 「模板模板套套」→ 视觉污染 → 读者反感 (evidence: Track 04 canon 排版规范节)
