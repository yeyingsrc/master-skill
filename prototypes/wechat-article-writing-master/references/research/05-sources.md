## Newsletter / 行业研究 媒体

### 1. 新榜（newrank.cn 主站）
- **URL**: https://www.newrank.cn
- **Type**: 行业数据 + 研究 + 媒体 + 榜单（主战场是网站，**非**同名公众号）
- **Author / Maintainer**: 徐达内 + 新榜研究院团队
- **Cadence**: 数据看板 daily 刷新；行业报告 monthly-quarterly；公众号头条访谈 weekly
- **Last activity**: 2026-05（持续活跃，2025 全年发布 ≥ 12 份内容生态报告）evidence: [T05-S001, T05-S002]
- **Audience tier**: practitioner + senior
- **典型每期内容**: 公众号 / 视频号 / 小红书 多平台日 / 周 / 月榜单 + 年度公众号生态报告（2024 社媒内容生态报告 / 2024 内容创作者生态报告 / 矩阵纪元 等）+ 创作者 / 机构访谈
- **签名信号**: 公众号日榜 / 月榜 / 飙升榜（行业内引用率最高的「号有多大」标准），公众号交易市场（中尾部号转手实价）evidence: [T05-S001, T05-S003]
- **投入产出比**: high
- **Paywall**: 基础排行免费；新榜有数高级查询 paywall (约 ¥3000-12000/年, 业内估计) evidence: [T05-S003]
- **为什么读**: 把「市场怎么定价公众号」量化下来，把公众号写作从 craft 升级到 ops 层必备。一手在公众号本体被屏蔽时，新榜是行业内最被认作 ground truth 的 metadata-level 信源
- **替代品**: 西瓜数据 S030 / 飞瓜 S029（但权威性 < 新榜）
- **可信度**: high
- **Decay risk**: low（机构级 + 商业化稳定）

## Signal vs Noise Filters

公众号写作领域信噪比格外差 ——「7 天打造爆款 / 月入十万」割韭菜内容压倒性多于真实方法论。鉴别 heuristics：

1. **标题含「月入 X 万」「7 天速成」「保姆级」「逆袭」「财富密码」几乎全是韭菜课** —— 直接 skip; 真正的从业者写「我做了三年, 这件事我做错了」, 不写承诺式标题

2. **推一个工具但绝口不提工具失败场景** —— 商务合作 disguised; 真方法论必然 trade-off, 推 135 编辑器 而不提它 IE 浏览器兼容 / 移动端响应 局限的, 90% 是软广

3. **数据图无 source 标注 + 时间戳** —— 不可信; 新榜 / QuestMobile / CNNIC 的数据图都标 source + 截止日期, 截图就用的 PPT 图 99% 是 KOL 编的

4. **口口声声「微信官方说」但拿不出 微信公开课 PRO / 微信派 原帖 / 腾讯财报 / 官方通告 链接** —— 99% 编的; 微信团队任何公开表态都能在 [T05-S019] [T05-S020] 找到原始链接, 找不到的「官方说」一律存疑

5. **谈变现只谈「头部年入 X 千万」不谈「中位数月入 ¥X」** —— 幸存者偏差; 真信源（新榜年报 [T05-S002] / 36 氪 [T05-S005]）会标尾部 / 中位 / 头部 三段, 只谈头部的是卖课的

6. **谈涨粉只谈互推 / 矩阵 / 投流, 不谈内容本身打磨成本** —— ops 思维, 不是 craft 思维; 头部公众号 KOL 在 [T05-S012] [T05-S014] 谈写作都至少 50% 时间在讲 craft, 只讲流量术的是 ops 课不是写作课

7. **课程 / 训练营 / 知识星球 价格阶梯设计极复杂（¥99 → ¥999 → ¥9999 → ¥29999）** —— 销售漏斗 disguised as 知识产品; 真课程 / 真知识星球 一般定一档价（[T05-S024] [T05-S025] 主流是 ¥1699-2999/年单价）

8. **「2025 公众号最新算法揭秘」类标题** —— 微信算法不公开, 任何「揭秘」都是猜测 + 编故事; 真信源 [T05-S001] [T05-S019] 都讲「平台政策方向」, 不讲「算法细节」

9. **嘉宾履历只有「N 个 10W+ 爆款」没具体公众号名 / 阅读 / 涨粉时间线** —— 履历水分; 真实从业者在 [T05-S012] [T05-S014] [T05-S017] 受访时, 主持人会逼出具体号名 / 时间线 / 数字

10. **「免费送资料 / 加微信领」的引流必然是 funnel 入口** —— 不是说必假, 但定位是 sales lead 不是知识; 别把 funnel 当 source

## 诚实边界（必读）

公众号写作的一手讨论 大量发生在三个闭环渠道:

1. **公众号本体**（被研究黑名单排除, 不进 source 池）
2. **知识星球 / 微信群 / 付费社群**（部分公开, 大部分付费墙 + 群闭环）
3. **私下饭局 + 操盘手互助**（完全闭环）

本 source 清单覆盖的是 **metadata-level 窗口** —— 新榜的数据 / 第三方平台对 KOL 的访谈 / 大会公开演讲 / 投资人播客对创作者的访问 / 数据厂商的 vendor 报告 / 监管协会的统计。这是「外部 + 公开 + 可追溯」的 source 池, 不是「这一行从业者每天实际看的全集」。

**实际从业者每天接触的 source 池 ≈ 公众号 80% + 微信群 / 知识星球 15% + 本清单 5%。本清单值不值得跟? 值, 但要清楚位置: 它是「校准 + 视野」用的, 不是「日常 craft」用的; 它告诉你行业怎么定价 / 平台怎么定调 / 大盘怎么收缩, 但写一篇 10W+ 的具体技法, 它教不了你**。

Skill 用户预期: 本清单是「想跟最新动态 + 想懂行业定位」的入口; 想学具体写作 craft, 还得回到公众号 + KOL 付费圈 + 实战练手。
