## A. 必读 / 必看 (12)

### A1. Mann's Surgery of the Foot and Ankle (足踝外科教科书)
- **领域**: 足踝外科综合教科书
- **作者**: Coughlin / Mann (历史) / Saltzman 多版主编
- **核心**: 解剖 / 病理 / 手术 / 康复全覆盖, 全球足踝住院医 + 主治标配
- **更新**: 多版本, 5-7 年大改 (最新 9th Edition 2024)
- **价格**: $400-600 (国际) / 国内引进版 ¥2000-3000
- **谁推**: 全球足踝外科住院医 + 主治标配
- **章节亮点**: 拇外翻 (Coughlin 章) / 踝关节置换 / 跟腱 / 创伤
- **来源**: [Primary] Mann's 9th Edition 出版信息

## B. 核心概念 (65 个)

### B1. 解剖 (15)
| 术语 | 含义 |
|------|------|
| **足弓 (内侧 / 外侧)** | 足底力学结构, 内侧高, 外侧低 |
| **距下关节 (Subtalar joint)** | 距骨 + 跟骨, 内外翻核心 |
| **距舟关节 (Talonavicular)** | 距骨 + 舟骨, 中足主轴 |
| **跟骰关节 (Calcaneocuboid)** | 跟骨 + 骰骨, 外侧中足 |
| **Lisfranc 关节 (TMT)** | 跖跗关节, 损伤易漏诊 |
| **第一跖趾关节 (1st MTP)** | 拇趾根, 拇外翻 / 关节炎 |
| **跟腱 (Achilles)** | 人体最强韧带, 腓肠肌 + 比目鱼肌合并 |
| **足底筋膜 (Plantar Fascia)** | 足底支撑, 跟骨内侧结节止点 |
| **三角韧带 (Deltoid Ligament)** | 内侧踝主韧带 |
| **距腓前韧带 (ATFL)** | 外侧踝最常损伤韧带 |
| **跟腓韧带 (CFL)** | 外侧踝中韧带 |
| **距腓后韧带 (PTFL)** | 外侧踝后韧带 (相对少见损伤) |
| **后胫肌 (Posterior Tibialis)** | 内侧足弓动力肌 (PTTD = 平足) |
| **腓骨肌群 (Peroneals)** | 外侧足踝动力 / 抗内翻 |
| **胫骨后神经 (Tibial nerve)** | 踝管 (Tarsal tunnel) 经过 |

### B2. 常见病变 (20)
| 术语 | 含义 |
|------|------|
| **跟腱病 (Achilles tendinopathy)** | 慢性退变, 不是急性炎症 |
| **跟腱炎 (Achilles tendinitis)** | 急性炎症, 短期 (跟病不同) |
| **跟腱断裂 (Achilles Rupture)** | 急性外伤, 多需评估 |
| **足底筋膜炎 (Plantar Fasciitis)** | 跟痛症常见原因 |
| **跟骨脂肪垫病变** | 跟痛症另一原因 (鉴别足底筋膜炎) |
| **拇外翻 (Hallux Valgus / Bunion)** | 第一跖趾畸形 |
| **拇僵硬 (Hallux Rigidus)** | 第一 MTP 关节炎 / 强直 |
| **平足 (Pes Planus / PTTD)** | 内侧足弓塌陷 / 后胫肌功能不全 |
| **高弓足 (Pes Cavus)** | 内侧足弓过高 |
| **马蹄足 (Equinus)** | 跟腱挛缩 / 踝背伸不足 |
| **踝关节不稳 (CAI)** | 慢性踝扭伤后遗症 |
| **踝关节冲击综合征 (Impingement)** | 软组织 / 骨性 (前 / 后) |
| **应力性骨折 (Stress Fracture)** | 重复负荷骨折 |
| **跗骨联合 (Tarsal Coalition)** | 先天跗骨融合 |
| **Charcot 足** | 神经性关节病变 (糖尿病) |
| **Lisfranc 损伤** | 跖跗关节损伤, 易漏诊 |
| **距骨骨软骨损伤 (OLT)** | 软骨 + 软骨下骨 |
| **跟骨骨折** | 高能量损伤, 复杂 |
| **Pilon 骨折** | 胫骨远端关节内高能量 |
| **Morton's 神经瘤** | 跖间神经瘤, 第 3-4 跖间常见 |
| **踝管综合征 (TTS)** | 胫后神经卡压 |

### B3. 评估 / 测试 (10)
| 术语 | 含义 |
|------|------|
| **Ottawa Ankle Rules** | 急性扭伤 X 线决策 (5-55 岁) |
| **Lauge-Hansen 分型** | 踝关节骨折分型 |
| **Weber 分型** | 腓骨外踝骨折分型 (A/B/C) |
| **Sanders 分型** | 跟骨骨折 CT 分型 |
| **抽屉试验 (Anterior Drawer)** | ATFL 评估 |
| **内翻应力试验** | 外侧韧带评估 |
| **Coleman Block Test** | 高弓足后足是否柔软 |
| **Silverskiöld 测试** | 跟腱挛缩 vs 腓肠肌挛缩 |
| **Thompson 试验** | 跟腱完整性 (急诊金标准) |
| **HVA / IMA / DMAA** | 拇外翻角度 (负重 X 线) |
| **AOFAS Score** | 足踝功能评分 (各部位特化) |
| **VAS / FAOS / FAAM / VISA-A** | 疼痛 / 功能 / 跟腱 评分 |

### B4. 治疗概念 (15)
| 术语 | 含义 |
|------|------|
| **保守治疗 (Conservative)** | 非手术 |
| **PRICE+ 早期活动** | Protect/Rest/Ice/Compression/Elevation + functional rehab |
| **拉伸 (Stretching)** | 跟腱 / 足底筋膜 — 多数慢性病基础 |
| **NSAIDs** | 短期止痛 / 抗炎 (长期不行) |
| **糖皮质激素封闭** | 局部 — 限次数 (足底筋膜终身 ≤ 3 次) |
| **PRP** | 富血小板血浆, 慢性病变 (B 级证据居多) |
| **体外冲击波 (ESWT)** | 慢性肌腱病 / 足底筋膜炎 (A 级) |
| **关节镜 (Arthroscopy)** | 微创关节内手术 |
| **关节融合 (Arthrodesis)** | 严重退变最终方案 |
| **关节置换 (Arthroplasty / TAA)** | 踝关节置换术 |
| **截骨术 (Osteotomy)** | 力线矫正 (Chevron / Scarf / Lapidus) |
| **跟腱修复 (Achilles Repair)** | 急性断裂手术 |
| **韧带重建 (Ligament Reconstruction)** | 慢性踝不稳 (Brostrom-Gould) |
| **腓肠肌松解** | 足底筋膜炎 / 跟腱病变手术选项 |
| **第三代微创拇外翻 (MIS)** | 经皮截骨 + 螺钉 |
| **functional bracing** | 跟腱断裂保守 — 非石膏制动 |

### B5. 力学 / 步态 (8)
| 术语 | 含义 |
|------|------|
| **gait (步态)** | 整体下肢运动模式 |
| **足旋前 (Pronation)** | 内翻 + 外旋 + 背屈复合 |
| **足旋后 (Supination)** | 外翻 + 内旋 + 跖屈复合 |
| **过度旋前 (Overpronation)** | 平足 / 跑步损伤相关 |
| **过度旋后 (Oversupination)** | 高弓足相关 |
| **生物力学链 (Kinetic Chain)** | 足 → 踝 → 膝 → 髋 → 腰 |
| **Windlass 机制** | 拇趾背伸时足底筋膜张力 |
| **步态周期 (Gait Cycle)** | 站立期 60% + 摆动期 40% |
| **足底压力分布** | 足跟 → 中足 → 前足 重心转移 |

## C. 标准 + 指南 (12)

### C1. AOFAS Clinical Practice Guidelines
- **覆盖**: 各类足踝病变循证诊疗推荐
- **更新**: 部分每 3-5 年
- **可及**: aofas.org 公开 (部分会员限定)

### C2. Ottawa Ankle Rules
- **覆盖**: 急性踝扭伤 X 线决策 (5-55 岁)
- **价值**: 减少不必要影像 + 辐射
- **验证**: 30+ 年, 多个 meta-analysis 支持
- **来源**: [Primary] Stiell 1992 + 多次更新

### C3. AAOS Clinical Practice Guideline: Achilles Tendon Rupture
- **覆盖**: 跟腱断裂手术 vs 保守治疗
- **更新**: 2024 修订
- **核心**: 保守治疗近 10 年证据完善, 跟手术效果接近

### C4. ACFAS Heel Pain Clinical Consensus Statement
- **覆盖**: 足底筋膜炎 / 跟痛症
- **核心**: 多 tier 治疗顺序 (拉伸 → 鞋垫 → 封闭 → ESWT → 手术)

### C5. ESWT 临床应用指南
- **覆盖**: 慢性跟腱病 / 足底筋膜炎 / 钙化性肌腱炎
- **来源**: 多国医学会 + 国际 ESWT 协会
- **证据等级**: A 级 (慢性足底筋膜炎 + 跟腱病)

### C6. PRP 注射临床应用指南
- **覆盖**: 慢性肌腱病变
- **状态**: B 级证据居多 (部分情况 A)
- **来源**: AOFAS PRP position statement / FAI 期刊综述

### C7. 中华足踝外科学会临床指南
- **覆盖**: 拇外翻 / 跟腱 / 踝关节置换 / 平足
- **更新**: 不定期, 国内主要参考

### C8. AOFAS 影像学应用指南
- **覆盖**: 何时该 X 线 / MRI / CT, incidental finding
- **价值**: 减少影像滥用 (Choosing Wisely)

### C9. 第三代微创拇外翻手术国际临床指南 (2024)
- **覆盖**: MIS 拇外翻指南 (适应症 / 禁忌 / 术式 / 术后)
- **来源**: 上海六院牵头制定 (柴益民团队)
- **价值**: 国际 + 国内首份 MIS 拇外翻指南

### C10. 国际足踝关节镜培训标准
- **覆盖**: 关节镜医生培训 (案例量 / 师从 / 培训路径)

## D. 外行破绽

1. **把 X 线异常当病因** — 没结合临床, 跟腱信号异常 / 跟骨骨刺 ≠ 必然症状原因
2. **跟患者说「先做手术再说」** — 跳过保守治疗, 大忌
3. **不解释保守治疗时间窗** — 患者 1 周不见效就放弃, 依从性崩溃
4. **跟腱炎 vs 跟腱病不分** — 一个急性 (NSAIDs / 短期), 一个慢性 (拉伸 + ESWT + PRP), 治疗策略完全不同
5. **拇外翻角度大就手术** — 美观不是单独适应症, 必须有疼痛 / 功能影响
6. **6 周石膏不负重治跟腱断裂** — 现代 functional bracing + 早期负重时代落后做法
7. **不查整体力学只在症状点处理** — 足底筋膜炎只在止点封闭 / 跟腱病只在跟腱按摩
8. **关节置换乱用** — 不严格选患者, 失败率高
9. **跟腱内激素注射** — 诱发断裂, 周围 ≠ 内部
10. **氟喹诺酮 / 慢性类固醇用药史漏问** — 跟腱断裂高风险
11. **MRI 报告写啥就治啥** — 不结合临床体征
12. **AOFAS Score 当患者主观评分** — 包含医生评估部分
13. **第三代微创直接上不练习** — 学习曲线 50-100 例
14. **HINTEGRA H3 在 US 仍植入** — FDA 2023 警示后部分医院暂停, 国内仍在引进 (长期 data 待)
