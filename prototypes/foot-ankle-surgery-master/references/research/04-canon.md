## 核心概念 (25 个)

### 解剖
- **足弓 (内侧 / 外侧)**: 足底力学结构
- **距下关节**: 距骨 + 跟骨之间的关节, 内外翻核心
- **跟腱 (Achilles)**: 人体最强韧带
- **足底筋膜 (Plantar Fascia)**: 足底支撑
- **Lisfranc 关节**: 跖跗关节, 损伤易漏诊
- **Charcot 足**: 神经性关节病变 (糖尿病常见)

### 常见病变
- **跟腱病 (Achilles tendinopathy)**: 慢性, 力学相关
- **足底筋膜炎**: 跟痛症常见原因
- **拇外翻 (Hallux Valgus)**: 第一跖趾关节畸形
- **平足 (Flatfoot)**: 内侧足弓塌陷
- **高弓足 (Pes Cavus)**: 内侧足弓过高
- **踝关节不稳**: 慢性踝扭伤后遗症
- **踝关节冲击综合征**: 软组织 / 骨性冲击

### 评估
- **Ottawa Ankle Rules**: 急性扭伤是否需 X 线
- **Lauge-Hansen 分型**: 踝关节骨折分型
- **Coleman Block Test**: 高弓足力学测试
- **Silverskiöld 测试**: 跟腱挛缩 / 腓肠肌测试

### 治疗概念
- **保守治疗**: 拉伸 / 鞋垫 / NSAIDs / 封闭 / 体外冲击波
- **PRP 注射**: 富血小板血浆, 慢性病变常用
- **关节镜**: 微创关节内手术
- **关节融合 (Arthrodesis)**: 严重退变最终方案
- **关节置换 (Arthroplasty)**: 踝关节置换术

### 力学
- **gait (步态)**: 整体下肢运动模式
- **足旋前 / 旋后 (Pronation / Supination)**: 足部三维运动
- **生物力学链**: 足 → 踝 → 膝 → 髋 全链路
- **鞋具 / 矫形器**: 力学修正工具
