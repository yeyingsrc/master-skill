## F. 信源黑名单 (重要)

| 信源 | 问题 |
|------|------|
| 微信公众号「治好足底筋膜炎」类 | 大部分营销, 单一案例无对照 |
| 朋友圈个人医生「治好的患者」 | 选择性报告, 无对照 |
| 抖音 / 小红书足踝健康类 | 标题党 + 缺乏严谨性, 算法奖励夸张 |
| 患者论坛 | 经验分享 OK, 不能作为治疗依据 |
| 「干细胞包治百病」营销 | 国家药监局警示, 非法 / 无证临床应用 |
| 不知名「秘方 / 偏方」 | 缺乏成分透明 + 监管, 风险高 |
| 私人医生 / 工作室 长期跟腱内激素注射 | 诱发断裂, 高风险操作 |
| HINTEGRA H3 在 US 仍在植入医院 (2023+) | FDA 警示, 失败率高于预期 |

## G. 验证方式

新观点 / 治疗方案出现时, 验证路径:
1. **PubMed 检索 RCT + meta-analysis** — 看证据等级
2. **AOFAS / ACFAS / EFAS / 中华学会指南**对照
3. **多名同行专家** 独立确认 (国内 + 国际)
4. **临床注册** (ClinicalTrials.gov) — 看是否在试验阶段
5. **FDA / NMPA 注册** — 假体 / 器械是否合规
6. **长期 follow-up data** (5+ 年) — 假体 / 大手术
7. **患者人群匹配** — 美国 / 欧洲数据是否适合中国患者
