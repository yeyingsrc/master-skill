# Track 03 — Workflows (足踝外科)

> Mini-viable for prototype. 严肃声明: 仅供执业医生参考, 不替代临床判断.

### 1. 急性踝扭伤评估 + 保守治疗启动

- **One-liner**: 从急诊 / 门诊接到急性踝扭伤患者, 到完成评估 + 启动保守治疗的标准动作
- **Trigger**: 急性踝扭伤 (24-72 小时内)
- **Output**: 严重度分级 + 保守治疗方案启动 + 患者教育
- **入门 SOP**:
  1. 病史 — 受伤机制 (内翻 / 外翻 / 旋转) + 当时声音 / 疼痛
  2. 体格检查 — 肿胀范围 / 压痛点 / 抽屉试验 / 内翻应力试验
  3. Ottawa Ankle Rules 判断是否需要 X 线
  4. X 线确认无骨折 (必要时 MRI 排除韧带完全断裂)
  5. 启动 PRICE 方案 (Protect / Rest / Ice / Compression / Elevation) + 早期活动 + 康复计划
- **资深路径**:
  - skip 全套 X 线 (Ottawa rules 阴性可省, 减少辐射 + 成本)
  - optimize 第一周强调早期渐进性活动, 不是绝对制动
  - add 给患者明确复诊节点 (1 周 / 4 周 / 12 周) + 警示信号
- **常见失败模式**:
  - 不分级一律石膏制动 → 韧带挛缩, 慢性踝关节不稳
  - 跳过 Ottawa rules 直接 X 线 → 浪费 + 辐射
  - 不给康复方案 → 二次扭伤率 30%+
- **Last_updated**: 2025-08-01
- **Decay risk**: low

### 2. 慢性足底筋膜炎管理

- **One-liner**: 从足底筋膜炎确诊到症状缓解的 3-6 月标准管理流程
- **Trigger**: 晨起足跟疼痛 ≥ 6 周, 排除其他病因
- **Output**: 60-80% 症状缓解, 失败者进入手术评估
- **入门 SOP**:
  1. 确认诊断 — 病史 (晨起疼痛) + 体格 (足底筋膜止点压痛) + 影像 (排除跟骨骨刺关联差)
  2. 整体力学评估 — gait / 鞋具 / 足弓 / 小腿肌张力
  3. 启动多模式保守治疗 — 拉伸 (跟腱 + 足底筋膜) + 鞋垫 + NSAIDs + 必要时局部封闭
  4. 4-6 周复诊评估反应
  5. 3-6 月规范保守失败 → 进入手术评估 (体外冲击波 / 内镜下筋膜松解)
- **资深路径**:
  - skip 跳过整体力学评估 (新手最常见错), 资深一律查
  - optimize 拉伸方案要个体化 (晨起前足底拉伸 vs 跟腱拉伸的优先级)
  - add 患者教育要具体到「每天怎么做」, 不是「多休息」
- **常见失败模式**:
  - 不查整体力学, 只在筋膜止点打封闭 → 失衡不解决, 复发
  - 患者教育模糊 ("多休息") → 患者不知道怎么做, 依从性差
  - 保守治疗时间不到 3 个月就跳手术 → 跳过自愈窗口
- **Last_updated**: 2025-09-15
- **Decay risk**: low
