# Track 03 — Workflows (足踝外科)

> 调研日期: 2026-05-03 (deep upgrade). 严肃声明: 仅供执业医生参考, 不替代临床判断 + 患者面诊评估.

## 元数据
- 完整工作流 5 个 (急性扭伤 / 足底筋膜炎 / 拇外翻 / 跟腱断裂 / 终末期踝关节炎)
- 一手比例 ≈ 78%

---

### 1. 急性踝扭伤评估 + 保守治疗启动

- **One-liner**: 从急诊 / 门诊接到急性踝扭伤患者, 到完成评估 + 启动保守治疗的标准动作 (24-72 小时窗口)
- **Trigger**: 急性踝扭伤 (24-72 小时内), 主诉踝部疼痛 + 肿胀 + 活动受限
- **Output**: 严重度分级 (Grade I-III) + 个体化保守治疗方案 + 患者教育 + 复诊计划

- **入门 SOP**:
  1. **病史采集** — 受伤机制 (内翻 / 外翻 / 旋转 / 高位扭伤?) + 当时是否听到 "pop" 声 + 当下能否承重 + 既往扭伤史 + 患者活动需求 (运动员? 老年? 高龄独居?)
  2. **体格检查** — 肿胀范围 (踝外侧 vs 内侧 vs 高位) + 压痛点 (ATFL 远端 / CFL / 三角韧带 / 5th 跖骨基底 / 腓骨远端) + 抽屉试验 + 内翻应力试验 + Squeeze 试验 (高位踝扭伤排查)
  3. **Ottawa Ankle Rules 判断**:
     - 阳性 (任一): 踝外侧腓骨远端 6cm 内压痛 / 内踝远端 6cm 内压痛 / 第 5 跖骨基底压痛 / 舟骨压痛 / 受伤后 + 当下都不能承重 4 步 → 拍 X 线
     - 阴性 → 不必 X 线 (节约 + 减辐射)
  4. **必要时 MRI** — 高位踝扭伤怀疑 syndesmosis 损伤 / 临床体征怀疑韧带完全断裂 / 6 周以上无改善
  5. **严重度分级**:
     - Grade I (轻度): 韧带轻微撕裂, 无松弛, 能承重
     - Grade II (中度): 部分撕裂, 中度松弛, 承重困难
     - Grade III (重度): 完全断裂, 明显松弛, 不能承重
  6. **启动 PRICE+ 方案**:
     - **P**rotect (支具 / 弹力绷带 / 必要时 walker boot)
     - **R**est (相对休息, 不是绝对制动)
     - **I**ce (急性期 24-72 小时 每 2-3h 冰敷 15-20 min)
     - **C**ompression (弹力绷带 / 加压袜)
     - **E**levation (抬高超过心脏)
     - **早期活动** (现代证据: 急性期过后早期渐进性活动比绝对制动好, 减韧带挛缩 + 二次扭伤)
  7. **启动康复** — 急性期过后 (3-7 天) 开始 ROM + 渐进负重 + 平衡训练 + 力量训练. 推荐 PT 介入
  8. **复诊计划** — 1 周 / 4 周 / 12 周, 每次评估恢复度 + 是否需调整

- **资深路径**:
  - **skip 全套 X 线** — Ottawa rules 阴性时省 X 线, 节约 + 减辐射
  - **skip 全制动 / 石膏** — 现代循证支持早期渐进活动, 减韧带挛缩 + 慢性踝不稳风险
  - **optimize Grade III 早期评估** — 完全断裂可能需手术 (年轻 / 运动员) vs 保守 (普通人), 早期决策, 不要拖
  - **add 高位踝扭伤识别** (Squeeze + 旋转外翻应力) — 漏诊率高, 处理跟一般踝扭伤完全不同
  - **add Lisfranc 损伤排查** — 中足压痛 + 「Plantar ecchymosis sign (足底瘀斑)」 + 足背扭伤机制 必须高度警惕
  - **optimize 患者教育具体化** — 「具体几点冰几次 + 几天后开始什么动作 + 什么情况打电话」, 不是「多休息」
  - **add 既往扭伤 ≥ 3 次** → 慢性踝关节不稳 (CAI) 评估 + 韧带重建讨论

- **常见失败模式**:
  - **不分级一律石膏** → 韧带挛缩, 慢性踝关节不稳率 30%+
  - **跳过 Ottawa rules 直接 X 线** → 浪费医保 + 辐射
  - **不给康复方案** → 二次扭伤率 30%+
  - **漏诊高位踝扭伤** → 慢性踝痛 + syndesmosis 不稳
  - **漏诊 Lisfranc 损伤** → 中足关节炎远期后果
  - **运动员太早回归** → 二次扭伤 / 慢性不稳 / 中年关节炎
  - **患者教育模糊「多休息 + 少走」** → 患者不知道怎么做, 依从性差

- **关键决策点**:
  - 何时手术: Grade III 完全断裂 + 年轻 / 运动员 → 手术评估; 普通人 / Grade II 以下 → 一律保守
  - 何时进 PT: 急性期过后 (3-7 天) 立即, 不要等 6 周
  - 何时怀疑慢性踝不稳: 复发性扭伤 ≥ 3 次或 6 个月以上仍主观「打软腿」

- **Last_updated**: 2025-09-01
- **Decay risk**: low — Ottawa rules 30+ 年验证, PRICE 早期活动是国际共识

### 2. 慢性足底筋膜炎管理

- **One-liner**: 从足底筋膜炎确诊到症状缓解的 3-6 月标准管理流程
- **Trigger**: 晨起足跟疼痛 ≥ 6 周, 症状典型 (晨起 / 久坐起立第一步剧痛, 走开几步缓解)
- **Output**: 60-80% 症状缓解 (循证数据), 失败者进入手术评估

- **入门 SOP**:
  1. **确认诊断** —
     - 病史: 晨起 / 久坐起立第一步疼痛 (典型的 "post-static dyskinesia") + 6 周以上 + 晚上不痛或减轻
     - 体格: 足底筋膜止点 (跟骨内侧结节) 压痛 + Windlass 试验阳性
     - 影像: X 线见跟骨骨刺 (但 — 骨刺跟症状关联差, 不是因果) / 排除应力性骨折
     - 鉴别: 跟骨骨折 / 跗管综合征 / S1 神经根病变 / 类风湿 / 痛风
  2. **整体力学评估** — gait (步态) + 鞋具 (磨损模式 / 鞋楦) + 足弓 (高弓 vs 平足) + 小腿肌张力 (Silverskiöld 跟腱挛缩 / 腓肠肌单独挛缩)
  3. **启动多模式保守治疗 (Tier 1, 6 周)**:
     - **拉伸** (核心): 跟腱拉伸 (gastroc/soleus) + 足底筋膜拉伸 (晨起前)
     - **鞋垫**: OTC 足弓支撑 / 跟杯 (heel cup)
     - **NSAIDs**: 短期 (2-4 周), 长期不行
     - **冰敷 + 滚瓶训练**: 冰冻矿泉水瓶足底滚动
     - **必要时局部封闭**: 糖皮质激素 + 局麻药 (跟骨骨刺周围, 不是筋膜内深处)
  4. **4-6 周复诊评估反应**:
     - 改善 60-80% → 继续保守 + 维持训练
     - 改善有限 → 进入 Tier 2
  5. **Tier 2 (Tier 1 失败 → 6-12 周)**:
     - **夜间夹板 (night splint)**: 维持背屈, 避免晨起紧张
     - **半定制 / 定制鞋垫**: 步态分析 + 力学修正
     - **PT 强化**: 离心拉伸 + 力量训练 + 步态修正
     - **PRP 注射** (有限证据)
  6. **Tier 3 (3 月 Tier 2 失败 → 6-12 月)**:
     - **体外冲击波 (ESWT)**: 焦点式 3-5 次, A 级证据
     - **超声引导下封闭** (跟骨骨刺周围 + 跟骨脂肪垫)
  7. **手术评估 (3-6 月规范保守完整失败)**:
     - **腓肠肌松解 (Gastrocnemius recession)**: 文献支持效果好于单纯筋膜松解
     - **内镜下足底筋膜部分松解 (EPFR)**: 微创, 效果好但需选对患者
     - **开放手术**: 现代少用 (除合并神经卡压)
  8. **术后康复**: 渐进负重 + 拉伸训练, 6-12 周

- **资深路径**:
  - **skip 跳过整体力学评估** (新手最常见错), 资深一律查 — 失衡不解决, 复发
  - **optimize 拉伸方案个体化** — 晨起前足底拉伸 vs 跟腱拉伸优先级 (Silverskiöld 测试决定)
  - **add 患者教育「具体每天怎么做」** — 不是「多休息 + 多拉伸」, 是「每天起床前 10 个足底拉伸 + 每天 3 次跟腱拉伸 30 秒 × 5」
  - **optimize 避免反复封闭** — 同一位置封闭 ≤ 3 次终身 (诱发筋膜萎缩 + 跟骨脂肪垫萎缩, 后患无穷)
  - **add 排查 Baxter's nerve 卡压** (足底外侧神经 1st 分支) — 慢性足跟痛的鉴别诊断
  - **add 鞋具问题不解决直接手术 = 复发** — 鞋具是基础, 不能跳过

- **常见失败模式**:
  - **不查整体力学只在筋膜止点打封闭** → 失衡不解决, 复发
  - **患者教育模糊「多休息」** → 依从性差
  - **保守治疗 < 3 个月就跳手术** → 跳过自愈窗口
  - **同一位置反复封闭 5+ 次** → 诱发筋膜萎缩 + 跟骨脂肪垫萎缩
  - **不区分腓肠肌挛缩 vs 全跟腱挛缩** (Silverskiöld) → 拉伸方案错配
  - **忽略鞋具 + 直接做手术** → 复发率高
  - **Baxter's nerve 漏诊** → 患者跑遍科室

- **关键决策点**:
  - Tier 1 → Tier 2: 4-6 周改善 < 50%
  - Tier 2 → Tier 3: 12 周改善 < 50%
  - Tier 3 → 手术评估: 3-6 月仍 < 50% 改善
  - 手术内镜 vs 腓肠肌松解: Silverskiöld 阳性 (跟腱挛缩) 优先腓肠肌松解

- **Last_updated**: 2025-10-15
- **Decay risk**: low — ACFAS Heel Pain CCS 是国际共识, 多个 meta-analysis 支持 ESWT + 腓肠肌松解

### 3. 拇外翻 (Hallux Valgus) 评估 + 决策

- **One-liner**: 从拇外翻初诊 (角度 + 症状 + 期望评估) 到术式选择 (保守 vs 第三代微创 vs 传统切开) 的标准流程
- **Trigger**: 第一跖趾关节畸形 + 疼痛 / 鞋具不耐受 / 美观需求
- **Output**: 个体化方案 (保守 / 微创 / 开放术式) + 患者教育 (期望 + 风险)

- **入门 SOP**:
  1. **病史采集** — 主诉 (疼痛位置 / 严重度 / 进展速度 + 鞋具问题 + 美观期望) + 既往保守治疗 + 家族史 (拇外翻强遗传倾向) + 患者职业 / 活动需求
  2. **负重 X 线** (必须负重位):
     - **HVA** (Hallux Valgus Angle): 第一跖骨 + 第一近节趾骨夹角. 正常 < 15°
     - **IMA** (Inter-Metatarsal Angle): 第一 + 第二跖骨夹角. 正常 < 9°
     - **DMAA** (Distal Metatarsal Articular Angle): 第一跖骨远端关节面角度
     - **籽骨位置** (Sesamoid 位): 籽骨相对跖骨头位置
  3. **严重度分级**:
     - 轻 (HVA 15-20°, IMA 9-11°): 多数保守
     - 中 (HVA 20-40°, IMA 11-16°): 视症状 + 期望
     - 重 (HVA > 40°, IMA > 16°): 多数手术
  4. **体格检查**: 第一 MTP 关节炎 (僵硬 / 疼痛弧度) + 第二 MTP 转移性痛 + 平足 / 平足合并 (常见) + 跟腱挛缩 (Silverskiöld)
  5. **保守治疗 (3-6 月)**:
     - 鞋具调整 (宽楦头 + 柔软上)
     - 趾间垫 + 拇外翻夜间矫形器 (有限证据, 主要止痛)
     - 足弓支撑 (合并平足时)
     - NSAIDs 短期止痛
  6. **手术评估** (保守失败 / 重度 / 患者强烈期望):
     - **第三代微创 (MIS)**: 经皮截骨 + 螺钉固定. 切口小 + 恢复快. 适合轻 + 中度. 上海六院 2024 国际指南推荐
     - **传统开放 (Open)**: Chevron / Scarf / Lapidus 等多种术式. 适合中 + 重度 + 复杂畸形 / 关节炎
     - **关节融合 (1st MTP fusion)**: 重度 + 关节炎晚期
     - **关节置换 (1st MTP arthroplasty)**: 谨慎选, 失败率高
  7. **术后管理**:
     - 第三代微创: 即刻负重 (硬底鞋) + 6-8 周回正常鞋
     - 传统开放: 部分负重 4-6 周 + 12 周回正常活动
     - 康复训练 + 复发预防 (鞋具 + 趾间训练)

- **资深路径**:
  - **skip 不分级一律手术** — 轻度 + 无症状不需要手术 (美观不是单独适应症)
  - **optimize 复杂畸形术式选择** — 单纯 HVA 高 vs IMA 高 vs DMAA 高 vs 关节炎 选不同术式
  - **add Lapidus (1st TMT 融合)** — 严重 IMA + 第一跗跖关节松弛 / Hypermobility
  - **add 第二趾转移性疼痛**评估 — 拇外翻常合并第二趾跖骨头痛, 单纯矫拇外翻可能不解决
  - **add 平足合并矫正** — 30%+ 拇外翻合并平足, 不矫平足 → 拇外翻复发
  - **optimize 患者期望管理** — 「手术目的是止痛 + 功能, 美观是副产品. 不能保证回到原来的鞋」 (张建中 多次强调)
  - **add 手术经验门槛** — 第三代微创学习曲线 50-100 例, 不是上手就做

- **常见失败模式**:
  - **保守治疗够都不试就手术** → 轻度患者不必要手术
  - **单纯按 HVA 选术式** → 忽略 IMA / DMAA / 关节炎 / 合并畸形
  - **不矫平足 / 不矫挛缩** → 复发率高
  - **患者期望管理失败** (期望「漂亮的脚」) → 投诉 + 满意度低
  - **第三代微创学习曲线没过就上** → 复杂畸形效果差
  - **关节置换乱用** → 失败率高 (vs 融合)

- **关键决策点**:
  - 保守 vs 手术: 6 月规范保守失败 + 影响日常 / 鞋具 / 工作
  - 微创 vs 开放: 轻 + 中度 → 微创首选; 重度 / 关节炎 / 复杂 → 开放
  - 融合 vs 截骨: 关节炎晚期 / 严重畸形 / 翻修 → 融合; 否则截骨

- **Last_updated**: 2024-03-13 (上海六院国际指南)
- **Decay risk**: low — 第三代微创趋势确立, 但传统术式仍是复杂畸形主力

### 4. 跟腱断裂 (Achilles Rupture) 评估 + 治疗决策

- **One-liner**: 从急性跟腱断裂诊断到手术 vs 保守 vs 微创 决策的标准流程
- **Trigger**: 急性 (多数) — 「被踢一脚 / 听到 pop 声」 + 不能踮脚 + 跟腱区凹陷
- **Output**: 手术 / 保守 / 微创 决策 + 个体化康复方案

- **入门 SOP**:
  1. **病史**: 受伤机制 (突然蹬地 / 急停急起) + 「pop 声」 + 不能踮脚 + 既往跟腱病史 + 既往氟喹诺酮 / 类固醇使用 (跟腱风险因素)
  2. **体格** (Thompson 试验是 gold standard):
     - **Thompson 试验**: 患者俯卧, 检查者捏小腿肚, 正常足跖屈, 跟腱断裂时跖屈消失或减弱
     - 跟腱区凹陷 (palpable gap)
     - 单足踮脚不能 (single heel raise)
     - 足跖屈无力
  3. **影像确认**:
     - **超声**: 床旁可用, 看断裂位置 + 范围 + 残端间隙
     - **MRI**: 复杂或不典型病例 / 决策手术方案
  4. **决策 — 手术 vs 保守**:
     - **保守 (Functional bracing 早期活动)**: 现代证据近 10 年支持保守治疗效果跟手术接近 (再断裂率 5-10%, 跟手术差距缩小). 适合: 普通人 / 老年 / 高 BMI / 控制不良糖尿病 / 不愿手术
     - **微创 (Mini-open)**: 切口 3-4 cm, 跟腱缝合. 兼顾手术效果 + 创伤小. 适合: 中年 + 健身需求 + 健康
     - **开放手术 (Open repair)**: 大切口 + 缝合 + 必要时增强 (FHL 转位等). 适合: 复杂断裂 / 慢性 / 翻修
     - **慢性 (≥ 4 周)**: 多需手术 + 增强 (FHL 转位 / 跟腱前移 / V-Y 推进)
  5. **术后 / 保守康复**:
     - **早期活动 (functional rehab)**: 现代主流 — 术后 / 保守起即开始
     - **2-4 周**: walker boot 跖屈位 + 部分负重 (不再是 6 周石膏)
     - **4-8 周**: 渐进 ROM + 渐进负重 + 跟腱垫减除
     - **8-12 周**: 渐进力量 + 步态修正
     - **3-6 月**: 渐进回运动 (普通人 6 月, 运动员 6-9 月)
     - **6-12 月**: 完全恢复 (跑跳)
  6. **复发预防**: 永久 awareness — 跟腱断裂后再断率 5-10%, 对侧断率 5-10%

- **资深路径**:
  - **optimize 不要默认手术** — 保守治疗近 10 年证据完善, 不再是次选
  - **add 患者偏好 + 风险评估** — BMI / 糖尿病 / 吸烟 / 依从性 影响手术 vs 保守选择
  - **optimize 康复 functional > 制动** — 古老的「6 周石膏 + 不负重」已被 functional bracing + 早期负重替代
  - **add 慢性断裂 ≥ 4 周复杂处理** — V-Y 推进 / FHL 转位 / 自体或异体跟腱
  - **add 跟腱后凸 / Haglund's** — 部分慢性病变合并骨性后凸, 同期处理
  - **add 慢性病史警示** — 反复氟喹诺酮 (左氧氟沙星 / 莫西沙星) / 慢性类固醇 / 慢性肌腱病 是高风险, 警惕「无创伤断裂」

- **常见失败模式**:
  - **默认手术 / 默认保守** — 不个体化决策
  - **6 周石膏不负重** — 现代 functional 早期负重时代落后做法
  - **过早回运动** (普通人 < 6 月 / 运动员 < 9 月) → 再断率高
  - **慢性断裂当急性处理** → 单纯端端缝合不够, 需增强
  - **跟腱挛缩没矫** (Silverskiöld 阳性) → 复发风险
  - **氟喹诺酮 / 慢性类固醇用药史漏问** → 风险评估不足

- **关键决策点**:
  - 急性 ≤ 4 周 + 普通人 / 老年 / 高风险 → 保守 (functional bracing)
  - 急性 ≤ 4 周 + 中年健康 / 运动需求 → 微创 mini-open
  - 急性 ≤ 4 周 + 复杂 / 高需求 → 开放 + 必要增强
  - 慢性 > 4 周 → 多需手术 + 增强术式

- **Last_updated**: 2024-AAOS 跟腱断裂指南修订
- **Decay risk**: medium — 保守 vs 手术争议仍有, 新证据持续

### 5. 终末期踝关节炎 — 关节置换 vs 关节融合 决策

- **One-liner**: 终末期踝关节炎 (post-traumatic / OA / RA) 选关节置换 (TAA) vs 关节融合 (TAR) 的决策框架
- **Trigger**: 影像 + 临床终末期踝关节炎 + 保守失败 + 严重影响功能
- **Output**: 个体化决策 + 患者期望管理 + 术后长期 follow-up

- **入门 SOP**:
  1. **病史 + 体格**: 病因 (外伤 / OA / RA / 神经性 / 骨坏死) + 严重度 (VAS / AOFAS Score) + 既往治疗 + 患者期望 (运动需求? 生活质量?) + 合并畸形 (力线 / 韧带不稳)
  2. **影像评估**:
     - **负重 X 线**: 关节间隙消失程度 + 力线 (内翻 / 外翻 / 中立) + 后足代偿
     - **CT 三维重建**: 骨缺损 + 力线精确测量
     - **MRI**: 软组织 + 骨髓水肿 (骨坏死 evaluation)
     - **EOS / 全长力线 X 线**: 整下肢力线 (上至髋, 下至足底)
  3. **保守治疗已失败确认**: 鞋具 / 矫形器 / 注射 / NSAIDs / 物理治疗 全做过且无效
  4. **决策 — 置换 (TAA) vs 融合 (TAR)**:
     - **TAA 适应症**:
       - 中老年 (≥ 50 岁理想, 但近年扩展到年轻)
       - 中立力线 / 轻度内外翻 (中度可矫)
       - 良好骨质 + 完整韧带
       - 双下肢已有融合 / 邻近关节炎
       - 现代假体 (HINTEGRA / Salto / Inbone / Star 等) 5 年生存率 90%+, 10 年 80%+
     - **TAR 适应症**:
       - 年轻 + 高需求 (运动 / 体力) — 失败率低于置换长期
       - 严重畸形 / 骨缺损 / 严重韧带不稳
       - 感染史 (置换感染处理棘手)
       - 神经性关节病 (Charcot)
       - 失败置换翻修
     - **不能 TAA 的禁忌**: 严重感染 / 严重血管病变 / 严重骨质疏松 / Charcot / 神经性病变
  5. **术前规划**:
     - 假体选择 (国际多款, 国内引进部分)
     - 力线矫正预演 (CT + 3D 打印预演)
     - 软组织 + 韧带平衡评估
     - 患者期望管理: 不能完全恢复正常踝功能, 是改善功能 + 减痛
  6. **术后管理 + 长期 follow-up**:
     - 早期: 制动 / 部分负重 / 渐进 ROM (TAA 4-6 周, TAR 12 周)
     - 长期: 每年 X 线 follow up (松动 / 下沉 / 假体磨损)
     - 患者教育: 不要负重跑跳 / 体育剧烈活动 (TAA 限制)

- **资深路径**:
  - **skip 默认融合 / 默认置换** — 个体化才是关键
  - **optimize 「Patient selection > implant selection」** (Hintermann 反复强调) — 选错患者比选错假体后果大
  - **add 术中力线矫正预演** — CT 3D 打印模型 + 预演截骨 + 假体放置
  - **add 双下肢评估** — 对侧情况影响决策 (双侧融合是 disability, 双侧置换风险高)
  - **add 邻近关节代偿** — 距下关节 / 距舟关节 是否炎症, 影响 TAR (融合) 长期效果
  - **add 长期 follow-up 必须性** — TAA 假体松动 / 下沉 / 磨损是 inevitable, 5-10 年内可能需翻修
  - **add 翻修策略预案** — 「a great surgeon plans the bailout before the operation」 (Easley 名言)

- **常见失败模式**:
  - **不严格选患者** — 严重畸形 / 韧带不稳 / 骨缺损 强行 TAA → 失败率高
  - **不矫力线** — TAA 不矫力线 → 假体偏载 + 早期磨损松动
  - **不评估邻近关节** — TAR 后距下 / 距舟 应力大 → 加速退变
  - **不患教 + 期望管理** — 患者期望「跑跳如常」 → 投诉
  - **不长期 follow up** → 错过翻修窗口
  - **HINTEGRA H3 (Mobile-bearing) 在 US 失败率高** — FDA 2023 警示, 部分医院暂停植入 (国内引进早, 长期 data 仍待)

- **关键决策点**:
  - 年轻 (< 50) + 高需求 → 倾向融合 (失败长期低)
  - 中老年 (≥ 50) + 适应症 → 倾向置换
  - 严重畸形 / 不稳 / 翻修 → 融合
  - 双下肢 / 多关节融合 → 倾向置换 (避免完全 disability)

- **Last_updated**: 2025 (HINTEGRA 20-year follow up + FDA H3 communication)
- **Decay risk**: medium-high — 假体迭代 + 长期数据仍在积累, 5-10 年可能有较大变化

## Phase 2 提炼提示 (跨工作流共性)

1. **影像 + 体征不一致时, 跟体征走** (5 个工作流共性)
2. **保守治疗 3-6 月是黄金期** (急性高风险手术情况除外)
3. **术后康复 ≥ 手术本身重要** (5 个工作流共性)
4. **患者期望管理是手术成功的隐藏前提** (拇外翻 + 跟腱 + 置换 共性)
5. **整体力学 / 整体功能视角** (足底筋膜炎 + 拇外翻 + 跟腱 + 置换 共性)
6. **复杂 / 高风险病例从规划开始预演翻修** (置换 + 跟腱慢性 共性)
