# Track 06 — Glossary (足踝外科)

> 调研日期: 2026-05-03 (deep upgrade). 严肃声明: 仅供执业医生参考, 不替代临床判断.

## 元数据
- 高频术语 110 / 标准 + 指南 12
- 一手比例 ≈ 78%

---

## A. 解剖术语 (40)

### A1. 骨 (10)
| 术语 | 含义 |
|------|------|
| **距骨 (Talus)** | 踝关节核心骨 |
| **跟骨 (Calcaneus)** | 后足支撑 |
| **舟骨 (Navicular)** | 内侧足弓关键 |
| **骰骨 (Cuboid)** | 外侧足弓 |
| **楔骨 (Cuneiform)** | 中足 (内 / 中 / 外 三块) |
| **跖骨 (Metatarsal)** | 前足 (1-5) |
| **趾骨 (Phalanx)** | 脚趾 (近 / 中 / 远节) |
| **籽骨 (Sesamoid)** | 第一跖骨头下 (拇外翻评估关键) |
| **腓骨 (Fibula)** | 外踝来源 |
| **胫骨 (Tibia)** | 内踝来源 |

### A2. 关节 (8)
| 术语 | 含义 |
|------|------|
| **踝关节 (Tibiotalar)** | 胫骨 + 距骨 |
| **距下关节 (Subtalar)** | 距骨 + 跟骨 (内外翻) |
| **距舟关节 (Talonavicular)** | 距骨 + 舟骨 (中足主轴) |
| **跟骰关节 (Calcaneocuboid)** | 跟骨 + 骰骨 |
| **Chopart 关节** | 距舟 + 跟骰 (中足横关节) |
| **Lisfranc 关节 (TMT)** | 跖跗关节 |
| **第一跖趾关节 (1st MTP)** | 拇趾根 |
| **第二跖趾关节 (2nd MTP)** | 第二趾根 (拇外翻并发常见) |

### A3. 韧带 (10)
| 术语 | 含义 |
|------|------|
| **三角韧带 (Deltoid)** | 内侧踝主韧带 |
| **距腓前韧带 (ATFL)** | 外侧踝最常损伤 |
| **跟腓韧带 (CFL)** | 外侧踝中韧带 |
| **距腓后韧带 (PTFL)** | 外侧踝后韧带 |
| **下胫腓前韧带 (AITFL)** | syndesmosis 前 |
| **下胫腓后韧带 (PITFL)** | syndesmosis 后 |
| **骨间膜 (IOM)** | 胫腓骨间膜 |
| **Spring 韧带 (Calcaneonavicular)** | 跟舟韧带, 内侧足弓 |
| **Lisfranc 韧带** | 第二跖骨基底 + 内侧楔骨 |
| **拇外侧韧带 / 内侧囊** | 拇外翻松弛 |

### A4. 肌腱 (8)
| 术语 | 含义 |
|------|------|
| **跟腱 (Achilles)** | 人体最强韧带, 腓肠肌 + 比目鱼 |
| **后胫肌 (Posterior Tibialis)** | 内侧足弓动力肌 (PTTD = 平足) |
| **腓骨长肌 (Peroneus Longus)** | 外侧抗内翻 |
| **腓骨短肌 (Peroneus Brevis)** | 外侧抗内翻 |
| **胫前肌 (Tibialis Anterior)** | 踝背伸主肌 |
| **趾长伸肌 (EDL)** | 足趾背伸 |
| **拇长屈肌 (FHL)** | 拇趾跖屈 (跟腱断裂可转位用) |
| **足底筋膜 (Plantar Fascia)** | 足底支撑 |

### A5. 神经 + 血管 (4)
| 术语 | 含义 |
|------|------|
| **胫后神经 (Posterior Tibial Nerve)** | 踝管内 |
| **腓总神经 (Common Peroneal)** | 外侧 (腓骨小头处易卡压) |
| **腓肠神经 (Sural Nerve)** | 跟腱外侧 (手术保护) |
| **足底外侧神经 1st 分支 (Baxter's)** | 慢性足跟痛鉴别 |

## B. 病变术语 (35)

### B1. 慢性病变 (15)
| 术语 | 含义 |
|------|------|
| **跟腱病 (Achilles tendinopathy)** | 慢性退变, 不是急性炎症 |
| **跟腱炎 (Achilles tendinitis)** | 急性炎症, 不同于跟腱病 |
| **足底筋膜炎 (Plantar Fasciitis)** | 跟痛症常见原因 |
| **跟骨脂肪垫病变** | 跟痛症另一原因 (鉴别足底筋膜炎) |
| **拇外翻 (Hallux Valgus / Bunion)** | 第一 MTP 畸形 |
| **拇僵硬 (Hallux Rigidus)** | 第一 MTP 关节炎 / 强直 |
| **平足 (Pes Planus / PTTD)** | 内侧足弓塌陷 / 后胫肌功能不全 |
| **高弓足 (Pes Cavus)** | 内侧足弓过高 |
| **马蹄足 (Equinus)** | 跟腱挛缩 / 踝背伸不足 |
| **腓肠肌挛缩 (Gastrocnemius equinus)** | 单独腓肠肌挛缩 (Silverskiöld 阳性) |
| **踝关节不稳 (CAI)** | 慢性踝扭伤后遗症 |
| **踝关节冲击综合征 (Impingement)** | 软组织 / 骨性 (前 / 后) |
| **Haglund's 畸形** | 跟骨后上方骨性突起 (跟腱后凸) |
| **跖骨痛 (Metatarsalgia)** | 前足底跖骨头疼痛 |
| **Morton's 神经瘤** | 跖间神经瘤 (3-4 跖间常见) |

### B2. 急性损伤 (10)
| 术语 | 含义 |
|------|------|
| **跟腱断裂 (Achilles Rupture)** | 急性外伤 |
| **踝扭伤 (Ankle Sprain)** | 韧带损伤 (Grade I-III) |
| **高位踝扭伤 (High Ankle Sprain / Syndesmosis injury)** | syndesmosis 损伤, 漏诊率高 |
| **跟骨骨折 (Calcaneal Fracture)** | 高能量损伤 |
| **Pilon 骨折** | 胫骨远端关节内 |
| **Lisfranc 损伤** | 跖跗关节, 易漏诊 |
| **Jones 骨折** | 第 5 跖骨基底骨折 (愈合慢) |
| **应力性骨折 (Stress Fracture)** | 重复负荷 |
| **第 5 跖骨基底骨折** | Jones / Avulsion 区分 |
| **腓骨肌腱半脱位** | 跑跳运动员常见 |

### B3. 骨与软骨 (5)
| 术语 | 含义 |
|------|------|
| **距骨骨软骨损伤 (OLT)** | 软骨 + 软骨下骨 (内 / 外侧) |
| **跟骨骨突炎 (Sever's disease)** | 儿童 |
| **骨坏死 (Avascular Necrosis / AVN)** | 距骨 / 舟骨 |
| **骨性关节炎 (OA)** | 终末期需置换 / 融合 |
| **痛风 / 假性痛风** | 第一 MTP 急性疼痛鉴别 |

### B4. 神经 + 系统性 (5)
| 术语 | 含义 |
|------|------|
| **踝管综合征 (TTS)** | 胫后神经卡压 |
| **腓总神经卡压** | 腓骨小头处 |
| **Charcot 足** | 神经性关节病变 (糖尿病) |
| **类风湿足 (RA foot)** | 系统性自身免疫累及 |
| **糖尿病足** | 神经病变 + 血管病变 + 感染综合 |

## C. 评估 / 测试 (15)

| 术语 | 含义 |
|------|------|
| **Ottawa Ankle Rules** | 急性扭伤 X 线决策 (5-55 岁) |
| **Lauge-Hansen 分型** | 踝关节骨折分型 |
| **Weber 分型** | 腓骨外踝骨折分型 (A/B/C) |
| **Sanders 分型** | 跟骨骨折 CT 分型 |
| **Myerson 分型** | Lisfranc 损伤分型 |
| **抽屉试验 (Anterior Drawer)** | ATFL 评估 |
| **内翻应力试验** | 外侧韧带评估 |
| **Squeeze 试验** | 高位踝扭伤 syndesmosis |
| **Coleman Block Test** | 高弓足后足是否柔软 |
| **Silverskiöld 测试** | 跟腱挛缩 vs 腓肠肌挛缩 |
| **Thompson 试验** | 跟腱完整性 (急诊金标准) |
| **Mulder's sign** | Morton's 神经瘤 |
| **Tinel 征** | 神经卡压定位 |
| **HVA / IMA / DMAA** | 拇外翻角度 (负重 X 线) |
| **AOFAS Score / VAS / FAOS / FAAM / VISA-A** | 功能评分系统 |

## D. 治疗 / 手术术语 (25)

### D1. 保守治疗 (8)
| 术语 | 含义 |
|------|------|
| **保守治疗 (Conservative)** | 非手术 |
| **PRICE+ 早期活动** | Protect/Rest/Ice/Compression/Elevation + functional rehab |
| **拉伸 (Stretching)** | 跟腱 / 足底筋膜 |
| **NSAIDs** | 短期止痛 / 抗炎 |
| **糖皮质激素封闭** | 局部 — 限次数 |
| **PRP** | 富血小板血浆 |
| **体外冲击波 (ESWT)** | 慢性肌腱病 |
| **functional bracing** | 跟腱断裂保守现代术式 |

### D2. 关节镜 + 微创 (5)
| 术语 | 含义 |
|------|------|
| **关节镜 (Arthroscopy)** | 微创关节内手术 |
| **后踝关节镜 (Posterior arthroscopy)** | 后踝软组织 / 距骨后突 |
| **距下关节镜 (Subtalar arthroscopy)** | 距下关节 |
| **微创筋膜松解 (EPFR)** | 内镜下足底筋膜部分松解 |
| **第三代微创拇外翻 (MIS)** | 经皮截骨 + 螺钉 |

### D3. 截骨术 (5)
| 术语 | 含义 |
|------|------|
| **Chevron 截骨** | 第一跖骨远端 V 形截骨 (拇外翻轻 / 中度) |
| **Scarf 截骨** | 第一跖骨干 Z 形截骨 (拇外翻中 / 重度) |
| **Lapidus 融合** | 第一跗跖关节融合 (严重 IMA / hypermobility) |
| **Akin 截骨** | 拇趾近节趾骨 (合并 Chevron 用) |
| **Ludloff 截骨** | 第一跖骨干斜形 (拇外翻重度) |

### D4. 韧带 + 跟腱手术 (5)
| 术语 | 含义 |
|------|------|
| **Brostrom-Gould 重建** | 慢性踝不稳外侧韧带重建 (gold standard) |
| **解剖韧带重建 (anatomic ligament reconstruction)** | 通用术式名 |
| **跟腱修复 (Achilles Repair)** | 急性断裂手术 |
| **跟腱 V-Y 推进 / FHL 转位** | 慢性跟腱断裂 |
| **腓肠肌松解 (Gastrocnemius recession / Strayer)** | 跟腱挛缩相关 |

### D5. 关节融合 + 置换 (4)
| 术语 | 含义 |
|------|------|
| **关节融合 (Arthrodesis)** | 严重退变最终方案 |
| **踝关节置换 (TAA / Total Ankle Arthroplasty)** | 终末期踝关节炎选项 |
| **三关节融合 (Triple arthrodesis)** | 距下 + 距舟 + 跟骰 (僵硬平足 / 严重畸形) |
| **第一 MTP 融合** | 拇僵硬严重 / 拇外翻关节炎晚期 |

### D6. 创伤 + 修复 (3)
| 术语 | 含义 |
|------|------|
| **ORIF** | 切开复位内固定 |
| **外固定支架** | 软组织未恢复时分期手术过渡 |
| **皮瓣 / 软组织修复** | 跟腱区 / 足底缺损 (显微外科) |

## E. 力学 / 步态 (10)

| 术语 | 含义 |
|------|------|
| **步态周期 (Gait Cycle)** | 站立期 60% + 摆动期 40% |
| **足旋前 (Pronation)** | 内翻 + 外旋 + 背屈复合 |
| **足旋后 (Supination)** | 外翻 + 内旋 + 跖屈复合 |
| **过度旋前 (Overpronation)** | 平足 / 跑步损伤相关 |
| **过度旋后 (Oversupination)** | 高弓足相关 |
| **生物力学链 (Kinetic Chain)** | 足 → 踝 → 膝 → 髋 → 腰 |
| **Windlass 机制** | 拇趾背伸时足底筋膜张力 |
| **足底压力分布** | 足跟 → 中足 → 前足 重心转移 |
| **rocker bottom shoe** | 鞋底前后摇摆 (糖尿病足 / 拇僵硬) |
| **heel-to-toe drop** | 鞋跟跟前掌高度差 (跑鞋设计) |

## F. 标准 / 指南 (12)

### F1. AOFAS Clinical Practice Guidelines
- **覆盖**: 各类足踝病变循证诊疗
- **更新**: 部分每 3-5 年

### F2. Ottawa Ankle Rules
- **覆盖**: 急性踝扭伤 X 线决策
- **价值**: 减少不必要影像 + 辐射

### F3. ACFAS Heel Pain Clinical Consensus Statement
- **覆盖**: 足底筋膜炎 / 跟痛症
- **核心**: 多 tier 治疗顺序

### F4. AAOS Achilles Tendon Rupture Guideline (2024)
- **覆盖**: 跟腱断裂手术 vs 保守
- **核心**: 保守证据完善

### F5. ESWT 临床应用指南
- **覆盖**: 慢性跟腱病 / 足底筋膜炎
- **证据等级**: A 级

### F6. PRP 注射临床应用指南
- **覆盖**: 慢性肌腱病
- **证据等级**: B 级居多

### F7. 中华足踝外科学会临床指南
- **覆盖**: 国内拇外翻 / 跟腱 / 置换 / 平足

### F8. AOFAS 影像学应用指南
- **覆盖**: X 线 / MRI / CT 适应症, incidental finding 处理

### F9. AAOS 跟腱断裂临床指南 (2024 修订)
- **覆盖**: 手术 vs 保守

### F10. 国际足踝关节镜培训标准
- **覆盖**: 关节镜医生培训, AOFAS / EFAS 联合

### F11. 第三代微创拇外翻手术国际临床指南 (2024)
- **来源**: 上海六院牵头

### F12. AAOS / AOFAS 踝关节置换 (TAA) 指南
- **覆盖**: TAA 适应症 / 禁忌 / 假体选择 / 术后

## G. 外行破绽 (15)

1. 把 X 线异常当病因 (incidental finding 不分)
2. 跟患者说「先做手术再说」(跳保守)
3. 不解释保守治疗时间窗 (患者中途放弃)
4. 跟腱炎 vs 跟腱病不分 (一急一慢, 治疗策略不同)
5. 拇外翻角度大就手术 (美观非单独适应症)
6. 6 周石膏不负重治跟腱断裂 (现代 functional bracing 取代)
7. 不查整体力学只在症状点处理
8. 关节置换乱用 (不严格选患者)
9. 跟腱内激素注射 (诱发断裂)
10. 氟喹诺酮 / 慢性类固醇用药史漏问
11. MRI 报告写啥就治啥 (不结合临床)
12. AOFAS Score 当患者主观评分 (含医生评估)
13. 第三代微创直接上不练习 (学习曲线 50-100 例)
14. 同一位置反复封闭 (足底 ≤ 3 次终身)
15. HINTEGRA H3 在 US 仍植入 (FDA 警示后部分医院暂停, 国内引进早, 长期 data 待)
