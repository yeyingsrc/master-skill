# Track 06 — Glossary (足踝外科)

> 调研日期: 2026-05-02。**严肃声明**: 仅供执业医生参考。

## 元数据
- 高频术语 35 / 标准 + 指南 8

---

## 解剖术语

### 骨
- **距骨 (Talus)**: 踝关节核心骨
- **跟骨 (Calcaneus)**: 后足支撑
- **舟骨 (Navicular)**: 内侧足弓关键
- **骰骨 (Cuboid)**: 外侧足弓
- **楔骨 (Cuneiform)**: 中足
- **跖骨 (Metatarsal)**: 前足
- **趾骨 (Phalanx)**: 脚趾

### 关节
- **踝关节 (Tibiotalar)**: 胫骨 + 距骨
- **距下关节 (Subtalar)**: 距骨 + 跟骨
- **距舟关节 (Talonavicular)**: 距骨 + 舟骨
- **Lisfranc 关节**: 跖跗关节
- **第一跖趾关节 (1st MTP)**: 拇趾根

### 软组织
- **跟腱 (Achilles)**: 人体最强韧带
- **足底筋膜 (Plantar Fascia)**: 足底支撑
- **三角韧带 (Deltoid Ligament)**: 内侧踝韧带
- **距腓前韧带 (ATFL)**: 外侧踝最常损伤韧带
- **跟腓韧带 (CFL)**: 外侧踝
- **后胫肌 (Posterior Tibialis)**: 内侧足弓动力肌

## 病变术语

| 术语 | 含义 |
|------|------|
| **跟腱病 (Tendinopathy)** | 慢性退变, 不是急性炎症 |
| **跟腱断裂 (Achilles Rupture)** | 急性外伤, 多需手术 |
| **足底筋膜炎 (Plantar Fasciitis)** | 跟痛症常见原因 |
| **拇外翻 (Hallux Valgus / Bunion)** | 第一跖趾关节畸形 |
| **平足 (Pes Planus / Flatfoot)** | 内侧足弓塌陷 |
| **高弓足 (Pes Cavus)** | 内侧足弓过高 |
| **马蹄足 (Equinus)** | 跟腱挛缩 / 踝背伸不足 |
| **踝关节不稳 (Chronic Ankle Instability)** | 慢性踝扭伤后遗症 |
| **踝关节冲击综合征 (Impingement)** | 软组织 / 骨性冲击 |
| **应力性骨折 (Stress Fracture)** | 重复负荷骨折 |
| **跗骨联合 (Tarsal Coalition)** | 先天跗骨融合 |
| **Charcot 足** | 神经性关节病变 |
| **Lisfranc 损伤** | 跖跗关节损伤, 易漏诊 |

## 评估 / 测试术语

| 术语 | 含义 |
|------|------|
| **Ottawa Ankle Rules** | 急性扭伤是否需 X 线规则 |
| **Lauge-Hansen 分型** | 踝关节骨折分型 |
| **Weber 分型** | 腓骨外踝骨折分型 |
| **抽屉试验 (Anterior Drawer)** | 距腓前韧带评估 |
| **内翻应力试验 (Inversion Stress)** | 外侧韧带评估 |
| **Coleman Block Test** | 高弓足力学测试 |
| **Silverskiöld** | 跟腱挛缩测试 |
| **VAS / FAOS** | 疼痛 / 功能评分 |

## 治疗 / 手术术语

| 术语 | 含义 |
|------|------|
| **保守治疗 (Conservative)** | 非手术 — 拉伸 / 鞋垫 / NSAIDs / 封闭 |
| **PRP** | 富血小板血浆注射 |
| **体外冲击波 (ESWT)** | 慢性病变物理治疗 |
| **关节镜 (Arthroscopy)** | 微创关节内手术 |
| **关节融合 (Arthrodesis)** | 严重退变最终方案 |
| **关节置换 (Arthroplasty)** | 踝关节置换术 |
| **截骨术 (Osteotomy)** | 力线矫正 |
| **跟腱修复 (Achilles Repair)** | 急性断裂手术 |
| **韧带重建 (Ligament Reconstruction)** | 慢性踝不稳手术 |

---

## 力学 / 步态

| 术语 | 含义 |
|------|------|
| **步态周期 (Gait Cycle)** | 站立期 + 摆动期 |
| **足旋前 (Pronation)** | 足内翻 + 外旋 + 背屈复合 |
| **足旋后 (Supination)** | 足外翻 + 内旋 + 跖屈复合 |
| **过度旋前 (Overpronation)** | 平足 / 跑步损伤相关 |
| **过度旋后 (Oversupination)** | 高弓足相关 |
| **生物力学链 (Kinetic Chain)** | 足 → 踝 → 膝 → 髋 → 腰 |

## 标准 / 指南 (8)

### 1. AOFAS Clinical Practice Guidelines
- **覆盖**: 各类足踝病变循证诊疗推荐
- **更新**: 部分指南每 3-5 年

### 2. Ottawa Ankle Rules
- **覆盖**: 急性踝扭伤 X 线决策
- **价值**: 减少不必要影像 + 辐射

### 3. ESWT (体外冲击波) 临床应用指南
- **覆盖**: 慢性跟腱病 / 足底筋膜炎
- **来源**: 多国医学会

### 4. PRP 注射临床应用指南
- **覆盖**: 慢性肌腱病变
- **状态**: 持续更新, 部分类目证据强度仍 B 级

### 5. 中华足踝外科学会临床指南
- **覆盖**: 国内诊疗规范
- **更新**: 不定期

### 6. AOFAS 影像学应用指南
- **覆盖**: 何时该 X 线 / MRI / CT
- **价值**: 减少影像滥用

### 7. AAOS 跟腱断裂临床指南
- **覆盖**: 跟腱断裂手术 vs 保守治疗
- **更新**: 2024-2025 修订

### 8. 国际足踝关节镜培训标准
- **覆盖**: 关节镜医生培训
- **来源**: AOFAS / EFAS 联合

## 外行破绽

- 把 X 线异常当病因 (没结合临床)
- 跟患者说「先做手术再说」(跳过保守治疗)
- 不解释保守治疗时间窗 (患者中途放弃)
- 把跟腱炎跟跟腱病混 (一个急性, 一个慢性, 治疗策略完全不同)
