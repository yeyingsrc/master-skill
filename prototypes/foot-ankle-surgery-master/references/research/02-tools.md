# Track 02 — Tools (足踝外科)

> 调研日期: 2026-05-02。诊断 / 评估 / 治疗 / 康复工具。

## 元数据
- 必备 5 / 场景特化 5 / 新兴 3

---

## 必备 (诊断 + 评估)

### 1. X 线 (足踝标准位 + 负重位)
- **One-liner**: 第一线影像, 任何足踝主诉必查
- **Tier**: 必备
- **典型场景**: 骨折 / 关节排列 / 退行性变 / 钙化点
- **避坑**: Ottawa Ankle Rules 阴性可省 X 线 (减少辐射)

### 2. MRI (足踝)
- **One-liner**: 软组织 + 骨髓水肿评估金标准
- **Tier**: 必备 (慢性 / 复杂病例)
- **典型场景**: 跟腱病变 / 韧带损伤 / 骨髓水肿 / 软骨损伤
- **避坑**: 影像异常 ≠ 症状原因, 区分 incidental finding

### 3. 临床体格检查工具
- **One-liner**: 足踝活动度测角器 / 抽屉试验 / 内翻应力试验
- **Tier**: 必备
- **典型场景**: 任何足踝评估的核心
- **价值**: 体格 + 病史 > 单纯影像

### 4. CT (含三维重建)
- **One-liner**: 复杂骨折 / 关节解剖术前评估
- **Tier**: 必备 (手术计划)
- **典型场景**: 骨折分型 / 内固定计划 / 复杂解剖

### 5. 超声 (足踝)
- **One-liner**: 实时动态 + 软组织评估, 床旁可用
- **Tier**: 必备 (急诊 / 跟腱评估)
- **典型场景**: 跟腱断裂 / 韧带 / 滑囊 / 神经

---

## 场景特化

### 6. 足底压力板 (步态分析)
- **典型场景**: 慢性足底筋膜炎 / 平足 / 高弓足 力学评估
- **谁用**: 三级医院运动医学 + 私立康复中心

### 7. 三维步态实验室
- **典型场景**: 复杂力学问题 / 运动员伤后回归
- **谁用**: 大型三甲医院运动医学中心

### 8. 关节镜
- **典型场景**: 踝关节冲击综合征 / 软骨损伤 / 滑膜病变
- **谁用**: 接受过关节镜培训的足踝外科医生

### 9. 超声引导下注射
- **典型场景**: 局部封闭 / PRP 注射
- **优势**: 比盲打精准

### 10. EMG / NCV (神经电生理)
- **典型场景**: 怀疑神经卡压 (如腓总神经)
- **谁用**: 神经科 + 足踝外科联合

## 新兴 / 实验

### 11. AI 辅助影像诊断
- **One-liner**: AI 看 X 线 / MRI 标记可疑区域
- **Status**: 早期 (≤ 24 个月)
- **避坑**: 必须由医生最终判读, 不能裸用

### 12. 可穿戴步态监测
- **One-liner**: 鞋垫 + 传感器持续监测步态
- **Status**: 起势中
- **典型场景**: 运动员长期伤后回归监测

### 13. 3D 打印 (定制鞋垫 + 矫形器)
- **One-liner**: 个性化鞋垫 / 矫形器
- **Status**: 部分专科应用

---

## 选型决策树

| 主诉 | 工具优先级 |
|------|----------|
| 急性扭伤 | 病史 + Ottawa rules → X 线 (如阳性) → 必要时 MRI |
| 慢性足底痛 | 体格检查 + X 线 → 整体力学评估 (步态 / 鞋具) → 必要时 MRI |
| 跑步损伤 | 病史 + 体格 + 步态分析 → 必要时 MRI |
| 术后随访 | X 线 (规范节点) + 体格 + 必要时 MRI |

---

## 避坑

1. 影像异常不一定是病因 — 临床体征 + 病史先行
2. Ottawa rules 阴性不要做 X 线 (减少辐射 + 成本)
3. AI 辅助诊断不能替代医生判读
4. 慢性病变只查症状点, 不查整体力学 — 失衡不解决, 复发率高
