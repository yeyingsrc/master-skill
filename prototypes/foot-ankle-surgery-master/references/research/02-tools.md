# Track 02 — Tools (足踝外科)

> 调研日期: 2026-05-03 (deep upgrade). 诊断 / 评估 / 治疗 / 康复工具. 严肃声明: 本 skill 给的是工具认知框架, 不替代医生临床判断.

## 元数据
- 必备 6 / 场景特化 8 / 新兴 5 = 19 工具
- 一手比例 ≈ 75%

---

## A. 必备 (诊断 + 评估, 6 个)

### A1. X 线 (足踝标准位 + 负重位)
- **One-liner**: 第一线影像, 任何足踝主诉 (排除 Ottawa rules 阴性) 都该有
- **Tier**: 必备
- **典型场景**:
  - 急性损伤 (Ottawa rules 阳性) 排骨折
  - 拇外翻 / 平足 角度测量 (HVA / IMA / Meary's angle)
  - 关节排列 / 退变评估
  - 钙化点 (跟腱止点 / 足底筋膜)
- **关键技术细节**:
  - **负重位** (weight-bearing) 比非负重重要 — 平足 / 拇外翻必须负重
  - 标准位: AP / Lateral / Mortise (踝穴位) / Sesamoid (籽骨位, 拇外翻用)
  - 跟骨骨折用 Broden's view + axial 跟骨位
- **避坑**:
  - Ottawa Ankle Rules 阴性时不必 X 线 (减辐射 + 成本)
  - 非负重位不能用来评估力线 (拇外翻 / 平足)
  - 影像阴性 ≠ 没有损伤 (软组织 / 应力性骨折早期 X 线不显)
- **来源**: AOFAS imaging guidelines / Mann's 影像章节 [Primary]

### A2. MRI (足踝)
- **One-liner**: 软组织 + 骨髓水肿评估金标准
- **Tier**: 必备 (慢性 / 复杂病例)
- **典型场景**:
  - 跟腱病变 / 韧带损伤 / 软骨损伤 / 骨髓水肿
  - 应力性骨折早期 (X 线阴性时)
  - 距骨骨软骨损伤 (OLT) 分级
  - 后胫肌功能不全 (PTTD) 肌腱评估
  - 慢性踝痛 6 周以上无改善
- **关键技术细节**:
  - 1.5T 跟 3.0T 都够用, 关键是足踝专用线圈 + 序列
  - 关键序列: T1 + T2 fat-sat (FS) + STIR + PD-FS
  - 跟腱断裂位置 / 程度: 矢状位 + 横断位
- **避坑**:
  - **影像异常 ≠ 症状原因** (incidental finding 极常见, 50+ 岁人群跟腱信号异常率 30%+)
  - 急性踝扭伤多数不需 MRI (除非有特殊指征)
  - MRI 报告写「肌腱信号异常 / 骨髓水肿」需结合临床, 不是直接病因
  - MRI 看不到神经卡压 (须 EMG/NCV)
- **价格 / 可及性**:
  - 国内三甲: ~¥ 800-1500 (单部位)
  - 美国: $1500-3000 (out of pocket without insurance)
  - 排队周期: 国内门诊 1-2 周, 美国 1-4 周
- **来源**: AOFAS Position Statement on imaging / Mann's [Primary]

### A3. 临床体格检查工具 (基础 + 不可替代)
- **One-liner**: 测角器 / 抽屉试验 / 应力试验, 足踝医生的基本功
- **Tier**: 必备 (无可替代)
- **关键工具**:
  - **测角器 (Goniometer)**: 关节活动度 (ROM) — 踝背伸 / 跖屈 / 内翻 / 外翻
  - **抽屉试验 (Anterior Drawer)**: 距腓前韧带 (ATFL) 完整性
  - **内翻应力试验 (Inversion Stress)**: 外侧韧带复合体
  - **Coleman Block Test**: 高弓足后足是否柔软可矫
  - **Silverskiöld 测试**: 跟腱挛缩 / 腓肠肌单独挛缩 vs 全跟腱
  - **Squeeze 试验**: 腓骨远端 syndesmosis (高位踝扭伤)
  - **Thompson 试验**: 跟腱完整性 (急诊跟腱断裂)
  - **Tinel 征**: 神经卡压 (Tarsal tunnel / Morton's neuroma)
  - **足弓测试 (Single heel raise)**: PTTD 检测金标准
- **避坑**:
  - 没做体格直接看影像 = 本末倒置
  - 患者疼痛 / 肿胀严重时部分体征不可信, 须复查
  - 跟腱完全断裂 Thompson 试验是 gold standard, 不要只看 「能不能踮脚」
- **来源**: Mann's 体格检查章 / AOFAS basic exam guidelines [Primary]

### A4. CT (含三维重建)
- **One-liner**: 复杂骨折 / 关节解剖术前评估
- **Tier**: 必备 (手术计划)
- **典型场景**:
  - 跟骨骨折 Sanders 分型 (CT 是金标准)
  - Pilon 骨折术前 CT
  - Lisfranc 损伤 (X 线易漏诊, CT 看微小移位)
  - 复杂关节置换术前 (骨缺损 + 力线)
  - 距骨骨软骨损伤 (OLT) 大小 / 位置
- **关键技术细节**:
  - 三维重建对手术规划很有用
  - SPECT-CT (放射核素 + CT) 用于不明痛点定位 (高级用法)
- **避坑**: 辐射剂量比 X 线高 — 不要常规筛查用
- **来源**: Mann's 影像章 [Primary]

### A5. 超声 (足踝)
- **One-liner**: 实时动态 + 软组织, 床旁可用 + 廉价
- **Tier**: 必备 (急诊 / 跟腱评估)
- **典型场景**:
  - 跟腱断裂诊断 (Thompson 阳性时确认)
  - 韧带损伤分级
  - 滑囊积液 / 神经鞘瘤
  - 超声引导下注射 (PRP / 封闭 / 神经阻滞)
- **优势**:
  - 无辐射 + 实时动态
  - 价格远低于 MRI (~¥200-500)
  - 床旁可用, 急诊优势
- **限制**:
  - 高度依赖操作者技术
  - 不能看骨内 / 关节内深处
- **来源**: AOFAS 超声应用 + 国内三甲超声科规程 [Primary]

### A6. 疼痛 + 功能评分量表
- **One-liner**: 客观化主诉 + 治疗反应追踪
- **Tier**: 必备 (临床记录 + 研究)
- **关键量表**:
  - **VAS** (Visual Analog Scale): 疼痛 0-10
  - **AOFAS Score**: 100 分制, 各部位特化 (踝 / 后足 / 中足 / 前足)
  - **FAOS** (Foot and Ankle Outcome Score): 综合功能 5 维
  - **FAAM** (Foot and Ankle Ability Measure): 日常 + 运动两套
  - **VISA-A**: Achilles tendon 病变特化
- **避坑**:
  - AOFAS Score 受医生评估部分影响 (不是纯患者主观)
  - 单次评分意义有限, 治疗前 + 后对比才有意义
- **来源**: AOFAS 官方 + FAI 期刊评分量表汇总 [Primary]

## B. 场景特化 (8 个)

### B1. 足底压力板 (Plantar Pressure / Pedobarography)
- **典型场景**: 慢性足底筋膜炎 / 平足 / 高弓足 / 糖尿病足 / 足部畸形 力学评估
- **谁用**: 三级医院运动医学中心 + 私立康复
- **国内代表机器**: Footscan / Tekscan / 国产 Anqi 等
- **价格**: 检查费 ~¥200-500 (国内)
- **来源**: 物理治疗 + 运动医学交叉文献 [Primary]

### B2. 三维步态实验室 (Gait Lab)
- **典型场景**: 复杂力学问题 / 运动员伤后回归 / 神经性步态 / 儿童脑瘫
- **关键技术**: 红外动作捕捉 + 力台 + EMG 同步
- **谁用**: 大型三甲医院运动医学中心 + 体育科研 + 部分骨科 (北医三院 / 华西 / 上海六院 / 国家体育总局体科所)
- **门槛**: 设备百万级 + 专业团队
- **来源**: gait lab 行业资料 + 国内体科所公开资料 [Primary]

### B3. 关节镜 (Arthroscopy)
- **典型场景**:
  - 踝关节冲击综合征 (Anterior/Posterior impingement)
  - 距骨骨软骨损伤 (OLT) 微骨折 / 修复
  - 滑膜病变 / 软骨损伤
  - 韧带辅助重建 (部分技术)
- **谁用**: 接受过关节镜培训的足踝外科医生 (国际有 AOFAS / EFAS 联合培训标准)
- **设备**: 4.0 mm 标准关节镜 / 2.7 mm 小关节镜 (后足 + 距下关节常用)
- **学习曲线**: 50-100 例后开始稳定, 200+ 例称熟练 (Glazebrook + COFAS 数据)
- **来源**: AANA + Glazebrook 教材 [Primary]

### B4. 超声引导下注射
- **典型场景**:
  - 局部封闭 (糖皮质激素 + 局麻药) — 足底筋膜炎 / 跟腱止点
  - PRP 注射 — 慢性肌腱病
  - 神经阻滞 — 术中 / 术后镇痛
- **优势**: 比盲打精准 (尤其细小结构如 Morton's 神经瘤 / 跟腱周围)
- **避坑**: 跟腱内激素注射禁忌 (诱发断裂) — 周围 ≠ 内部
- **来源**: AOFAS injection guidelines / 国内介入治疗规程 [Primary]

### B5. EMG / NCV (神经电生理)
- **典型场景**:
  - 踝管综合征 (Tarsal Tunnel Syndrome) 胫后神经卡压
  - 腓总神经卡压
  - 复杂周围神经损伤分级
- **谁用**: 神经科 + 足踝外科联合
- **价格**: ¥500-1500 (国内)
- **避坑**: 早期神经病变 EMG 可能阴性, 不能排除诊断
- **来源**: 神经电生理学指南 + 足踝交叉文献 [Primary]

### B6. SPECT-CT / Bone Scan (骨扫描)
- **典型场景**:
  - 不明定位的足踝痛 (e.g. 多关节退变, 不知道哪个关节是症状源)
  - 应力性骨折早期 (X 线阴性, MRI 不可及时)
  - 关节融合术后融合评估
- **优势**: 功能影像 — 看代谢, 不只看结构
- **限制**: 辐射 + 成本高 + 特异性低于 MRI
- **来源**: AOFAS 影像高阶应用 [Primary]

### B7. 矫形器 + 鞋具 (Orthotics + Footwear)
- **典型场景**:
  - 慢性足底筋膜炎 / 平足 / 高弓足 — 鞋垫 + 鞋具修正
  - 拇外翻 — 趾间垫 / 矫形器 (保守治疗)
  - 跟腱病 — 跟腱垫 (heel lift)
- **类型**:
  - **预制 (OTC)**: 药店 / 网购, ~¥50-300
  - **半定制 (Semi-custom)**: 鞋店 + 测足型, ~¥300-1500
  - **定制 (Custom)**: 步态分析 + 3D 打印 / 模具, ~¥1500-5000
- **避坑**:
  - 矫形器只在患者愿意穿才有效 (依从性是首要)
  - 急性期足太肿, 等消肿再选 — 否则换季就不合脚
  - 鞋面 / 鞋底 / 楦头 个体差异大, 不要照搬别人推荐
- **来源**: 物理治疗 + 矫形器行业指南 [Primary]

### B8. 体外冲击波 (ESWT — Extracorporeal Shockwave Therapy)
- **典型场景**:
  - 慢性足底筋膜炎 (规范保守 6 周以上无效)
  - 慢性跟腱病 (尤其 insertional)
  - 钙化性肌腱炎
- **关键参数**:
  - 焦点式 (focused) vs 散射式 (radial), 焦点式深部更强
  - 通常 3-5 次治疗, 每周 1 次
- **证据等级**: 慢性足底筋膜炎 + 跟腱病 是 A 级证据 (多个 meta-analysis)
- **来源**: ACFAS Heel Pain CCS / 国际 ESWT 应用指南 [Primary]

## C. 新兴 / 实验 (5 个)

### C1. AI 辅助影像诊断
- **One-liner**: AI 看 X 线 / MRI 标记可疑区域 (骨折 / 软组织异常)
- **Status**: 早期 — 部分 FDA cleared 产品 (主要骨折), 足踝特化 AI 仍在临床验证阶段
- **代表产品**: 国际 — RBfracture, AIDoc; 国内 — 推想科技 / 联影 / 数坤 (主要胸 / 头, 足踝特化少)
- **避坑**:
  - 必须由医生最终判读, 不能裸用
  - 训练数据偏倚 (常见骨折训练充分, 罕见情况 / 复杂解剖识别差)
  - 不能替代体格检查 + 病史
- **来源**: FDA 510(k) 注册 + 国内三类医疗器械注册 [Primary]

### C2. 可穿戴步态监测
- **One-liner**: 鞋垫 + 传感器持续监测步态 / 负重 / 平衡
- **Status**: 起势中 — 多款消费级 (Garmin Foot Pod / Stryd) + 部分医疗级
- **典型场景**:
  - 运动员长期伤后回归监测
  - 糖尿病足风险评估
  - 术后步态恢复定量
- **避坑**: 消费级精度不足以替代医疗评估
- **来源**: 运动医学行业资料 + 中华运动医学杂志 [Reference]

### C3. 3D 打印 (定制鞋垫 + 矫形器)
- **One-liner**: 个性化鞋垫 / 矫形器 / 截骨导板
- **Status**: 部分专科应用 (上海六院 / 北医三院 / 华西已开展)
- **典型场景**:
  - 复杂足部畸形定制矫形器
  - 截骨术前 3D 打印模型预演
  - 定制鞋具 (糖尿病足 / 严重畸形)
- **价格**: 定制鞋垫 ~¥3000-8000, 截骨导板手术增项
- **来源**: 国内三甲 3D 打印骨科应用 [Primary]

### C4. 富血小板血浆 (PRP) 注射
- **One-liner**: 慢性肌腱病 / 软骨损伤 增长因子治疗
- **Status**: 临床应用扩展中, 证据等级仍 B 级 (部分情况 A 级)
- **典型场景**:
  - 慢性跟腱病 (规范保守失败)
  - 距骨骨软骨损伤 (OLT) 微骨折术中辅助
  - 慢性足底筋膜炎 (有限证据)
- **关键参数**:
  - 离心制备 — leukocyte-rich (LR-PRP) vs leukocyte-poor (LP-PRP)
  - 跟腱病用 LR-PRP 证据更强 (国际共识)
  - 通常 2-3 次, 每 2-4 周 1 次
- **避坑**:
  - 不是「神药」 — 失败率 30%+
  - 制备工艺差异大, 国内不同医院 PRP 浓度差距大
  - 急性损伤效果不明确
- **来源**: AOFAS PRP position statement / FAI 期刊 PRP meta-analysis [Primary]

### C5. 干细胞 / 生物活性材料
- **One-liner**: 软骨修复 / 骨愈合促进
- **Status**: 实验阶段 — 国际部分研究, 国内严格监管
- **典型场景**:
  - 距骨骨软骨损伤 (OLT) 大缺损
  - 关节融合促进愈合 (BMP)
- **避坑**:
  - 国内干细胞临床应用受国家监管严格 (除经批准临床试验)
  - 「干细胞包治百病」是营销, 不是循证
  - BMP (骨形态发生蛋白) 部分应用 FDA 警示
- **来源**: 国家药监局 + FDA 干细胞临床应用指南 [Primary]

## D. 选型决策树

| 主诉 / 场景 | 工具优先级 |
|------------|----------|
| 急性扭伤 | 病史 + 体格 + Ottawa rules → X 线 (如阳性) → 必要时 MRI (≥6 周症状未缓解) |
| 慢性足底痛 | 体格 + X 线 → 整体力学评估 (步态 + 鞋具 + Coleman/Silverskiöld) → 必要时 MRI |
| 跑步损伤 | 病史 + 体格 + 步态分析 → 鞋具评估 → 必要时 MRI |
| 急性怀疑跟腱断裂 | 体格 (Thompson 试验) + 超声 → MRI 确认范围 |
| 复杂骨折 | X 线 → CT 三维重建 → 软组织评估 |
| 拇外翻评估 | 负重 X 线 (HVA / IMA / 籽骨位) + 体格 → 必要时 MRI 排关节炎 |
| 平足评估 | 负重 X 线 + 体格 + 步态 → MRI 看 PTTD 程度 |
| 终末期踝关节炎 | 负重 X 线 + CT 力线 → 决定置换 vs 融合 |
| 不明痛源 | 体格 + X 线 → MRI → 必要时 SPECT-CT / EMG |
| 术后随访 | X 线 (规范节点) + 体格 + 必要时 MRI / CT |

## E. 避坑总览

1. **影像异常不一定是病因** — 临床体征 + 病史先行 (incidental finding 极常见)
2. **Ottawa rules 阴性不要做 X 线** (减辐射 + 成本)
3. **AI 辅助诊断不能替代医生判读** (训练数据偏倚 + 罕见情况识别差)
4. **慢性病变只查症状点不查整体力学** (失衡不解决, 复发率高)
5. **跟腱内激素注射禁忌** (诱发断裂) — 周围 ≠ 内部
6. **PRP 不是神药** — 失败率 30%+, 制备工艺差异大
7. **干细胞临床应用** 国内严格监管, 警惕营销 / 非法应用
8. **3D 打印 / 步态实验室** 不是每个医院都有, 不要让患者绕远路
9. **AOFAS Score 包含医生评估部分**, 不是纯患者主观, 解读时注意
10. **超声依赖操作者** — 不是「便宜替代 MRI」, 看适应症
11. **SPECT-CT 辐射剂量高** — 严格指征用
12. **MRI 报告写「肌腱信号异常 / 骨髓水肿」** 需结合临床, 不是直接病因
