> Wave 1 Track 04 — essential books / theory / canon documents. Industry: 现代建筑设计 (建筑学 + 当代实践), locale=global, profile=practitioner.

> 结构约束 notes (诚实): (1) 早期 modernist 文本绝大多数是法/德/意原文 → 英译延迟 20-50 年, 当代中文学者读的几乎都是二次翻译, "原意 vs 译义" 是已知 caveat; (2) 中文当代理论 canon 起步晚, 一线学者的"正式专著"普遍 1980+ 才系统出现, 1990 之前的中文 canon 以教材为主 (彭一刚, 梁思成《中国建筑史》战时稿后出版) 而非理论; (3) 部分 starchitect monograph (El Croquis / GA Document) 不开源、定价 €60-200、library-only, 但因学院普遍使用列入 secondary bucket; (4) Pritzker laureate ceremony PDF 是 verified_primary (官方一手); (5) 中文 canon 比例 ~20% (在 15-30% 现实区间内); 严格不收知乎 / 百度百科 / mp.weixin / CSDN 等任何 blacklisted 站点.

### 📖 1. Vers une architecture (Towards a New Architecture)

- **作者**: Le Corbusier (Charles-Édouard Jeanneret, 瑞士-法国)
- **首版年**: 1923 (Etchells EN 1927; Goodman EN 2007 *Toward an Architecture* — 更准确)
- **流派 / 类别**: Modernist 奠基 manifesto
- **一句话核心论点**: 「住宅是供人居住的机器 (la maison est une machine à habiter)」— 用工程师的精神重新定义建筑.
- **为什么是 canon**: Vincent Scully 称其为"自 Vitruvius 以来最重要的建筑写作"; 全球建筑院校史课通识; 几乎每一本 Frampton/Giedion 都引用; 1929 已被巴黎学界视为"一代人的 canonical text".
- **代表性洞见 / 概念**:
  - 五项新建筑要点 (底层架空 / 屋顶花园 / 自由平面 / 自由立面 / 横向长窗)
  - 工程师美学 vs 学院派
  - 用 Parthenon 对比 Delage 跑车 — 普世形式法则
  - 建筑或革命 (Architecture or Revolution) — 收尾警句
- **可建造性 vs 理论性 标签**: hybrid (manifesto 但配 5 点可操作)
- **推荐读法**: 完整通读 (薄, 200 页内); 配合 *Modulor* 看其晚期数学/比例转向.
- **争议 / 局限**: 与法西斯/极权主义有 documented 接触; "machine for living" 被后世归为去人化的 modernism 原罪; Goodman 2007 译者明指 Etchells 1927 译本失真 80 年.
- **来源**:
  - [T04-S002] Dover Publications 出版 listing — verified_primary
  - [T04-S003] Getty (Goodman 2007 trans.) publisher listing — verified_primary
  - [T04-S001] Wikipedia 历史脉络 — reference

### 📖 2. Space, Time and Architecture

- **作者**: Sigfried Giedion (瑞士)
- **首版年**: 1941 (Harvard Charles Eliot Norton Lectures; 多版增补至 5th 1967)
- **流派 / 类别**: Modernist 历史 / CIAM 派
- **一句话核心论点**: 现代建筑不是断裂, 而是 Renaissance 透视空间→ Cubism 多视点空间的连续演进.
- **为什么是 canon**: 第一本系统把 modernism 写入"伟大叙事"; CIAM 秘书长 Giedion 用学院权威把 Le Corbusier / Gropius / Mies / Aalto 写进历史; 反复出现于 1950-1990 院校史教材; Frampton 1980 直接接过其衣钵并改写.
- **代表性洞见 / 概念**:
  - 时空 (space-time) 概念 — Einstein/Cubism 类比
  - "Constituent vs transitory facts" 区分长效与时髦
  - 把美国 (Wright, Chicago school) 写入欧洲叙事
  - 工业革命 → 桥梁/百货公司是建筑前夜
- **可建造性 vs 理论性 标签**: theoretical (史学)
- **推荐读法**: 史课选读; 完整版 800+ 页, 摘要每章首尾.
- **争议 / 局限**: Banham 后期批为"CIAM 御用史学"; 对国际式之外 (e.g. Expressionism) 边缘化; 5 版叠加导致结构松散.
- **来源**:
  - [T04-S032] Harvard UP 5th ed book listing — secondary
- **last_checked**: 2026-05-16

### 📖 3. Modern Architecture: A Critical History

- **作者**: Kenneth Frampton (英国-美国)
- **首版年**: 1980 (5th ed 2020, Thames & Hudson "World of Art")
- **流派 / 类别**: Modernist 历史 + critical regionalism 提出
- **一句话核心论点**: 现代建筑是技术理性与文化抵抗的辩证 — critical regionalism 是当代唯一伦理出路.
- **为什么是 canon**: 全球建筑院校史教材 #1; "World of Art" 平装系列让它人手一本; "Critical Regionalism" 一词由此扩散; 5 版 (2020) 重写全球化章, 仍是 standard.
- **代表性洞见 / 概念**:
  - Critical Regionalism — 抵抗"国际无差别"
  - 三个文化 transformation (cultural development / building production / 现代建筑 program)
  - Tectonic vs scenographic 区分
  - Resistance architecture (Botta / Ando / Mendes da Rocha)
- **可建造性 vs 理论性 标签**: theoretical (但章节后附实例)
- **推荐读法**: 通读一遍后, 把"critical regionalism"章与其 1983 essay (in Foster ed. *The Anti-Aesthetic*) 一起读.
- **争议 / 局限**: "regionalism" 被 Schumacher 等批为浪漫主义; 1-3 版女性建筑师严重缺失, 4-5 版补.
- **来源**:

### 显著分歧 / 学派对立

- **Critical Regionalism (Frampton + 王澍 + 朱锫 + Kuma + Pallasmaa)** vs **Parametricism universal style (Schumacher)** — 全球化时代是该用普世数字 epochal style, 还是用在地材料/气候/文化抵抗.
- **乐观 pragmatic populism (Venturi + Ingels + Jencks)** vs **批判否定 neo-Marxist (Tafuri + de Graaf)** — 建筑师与资本/大众的关系: 拥抱还是反抗.
- **Object-making (Mies / Eisenman / Hadid)** vs **Anti-object (Kuma + Zumthor + 王澍)** — 建筑的本质: 完美对象, 还是消融在环境/场地里.
- **Phenomenology craft (Zumthor + Pallasmaa + Ando)** vs **Data-driven design (Carpo + Schumacher)** — 设计的认知方式: 手感/记忆/直觉, 还是算法/数据/计算.
- **Total design utopia (Le Corbusier *Vers une*)** vs **Collage / incremental (Rowe & Koetter + Aravena + Alexander)** — 计划 vs 渐进 — 现代主义自身张力.
- **Western canon-dominated (Frampton 1980 1-3 ed / Giedion / Banham)** vs **Pluralist / global / 中文当代 (5th ed Frampton 2020 / 王澍 / 朱锫 / Ito / Aravena)** — 后期 canon 的去殖民化趋势.
- **Image of the city (Lynch — 心理图)** vs **Architecture of the city (Rossi — 类型图)** — 城市分析的认知 vs 历史路线.
