## Glossary entries (75 entries)

### Category A. 项目交付 / 阶段 / 流程 (13 entries)

#### 1. Schematic Design (SD)

- **中文**: 方案设计
- **类别**: A. 项目交付
- **定义**: AIA 5-phase 服务的第 2 阶段 (Programming 之后, DD 之前). 输出 site plan / floor plans / sections / elevations / 渲染或 model, 配以总尺寸 + initial 造价估算. 目的是与业主对齐 design intent, 之前不要陷入节点细节.
- **流派 / locale 适配**: AIA SD 通常推到 "design intent finalized" — 总尺寸 + 总材料策略明确, 但还没有 spec / 节点 / consultant coordination. 中国设计院体系下「方案设计」往往边界更前: 概念 + 体块 + 基本平面 + 总图; 多数院子 SD 不到 AIA 那么深, 部分 AIA SD 内容下移到 中国 "初步设计 / 初设" 阶段. RIBA Plan of Work 2020 中相当于 Stage 2 (Concept Design).
- **交付物**: 草图 + 总图 + 平面 + 立面 + 剖面 + 体块 model (实体 / 数字) + design report + 初步造价
- **常见反模式**: SD 阶段直接做 DD-level 节点 (浪费 fee); 跳过 SD 直接 DD (业主无法 sign off intent, 后期改图灾难)
- **来源**: [T06-S001], [T06-S002], [T06-S064]

#### 2. Design Development (DD)

- **中文**: 扩初 / 深化设计 (中国设计院常对应 "初步设计")
- **类别**: A. 项目交付
- **定义**: AIA 5-phase 第 3 阶段. 把 SD 的 design intent 推进到 MEPS (mech / elec / plumb / struct) + 建筑细节明确 — material types / window 和 door 位置 / 主要节点 strategy.
- **locale**: 中国 "初步设计" 阶段 + 部分 "施工图初稿". RIBA Plan of Work 2020 大致对应 Stage 3 (Spatial Coordination) + 部分 Stage 4 (Technical Design) 起步.
- **交付物**: deepened drawings 含 spec outline / consultant 协调图 / refined 造价
- **来源**: [T06-S001], [T06-S064]

#### 3. Construction Documents (CD)

- **中文**: 施工图设计
- **类别**: A. 项目交付
- **定义**: AIA 5-phase 第 4 阶段. 出 contractor 用于 build 的完整 set — 全 dimensions + notes + spec + 协调 (struct / MEP / 其他 consultants). 目的: 把 design intent 翻译成 buildable instructions.
- **locale**: 中国 "施工图" 阶段. RIBA Stage 4 (Technical Design). 中国施工图额外要过 "设计文件审查" — 见 entry #66.
- **来源**: [T06-S001], [T06-S064]

#### 4. Construction Administration (CA)

- **中文**: 施工配合 / 监理协助 (国内不等同于 "监理" — 监理是独立第三方)
- **类别**: A. 项目交付
- **定义**: AIA 5-phase 第 5 阶段. 施工期间 architect 处理 RFI / submittal review / site observation / change order / punch list / substantial completion / final completion. 不是「监工」.
- **关键工具**: G716 (RFI), submittal log, change order, punch list, substantial completion certificate
- **locale**: AIA 中 CA 是 architect 标准服务 (除非业主单独 contract 出去); 中国施工阶段 architect 更多以 "设计交底 + 现场服务" 形式介入, 第三方监理走另一条 contract (《工程监理企业资质管理规定》).
- **来源**: [T06-S001], [T06-S022], [T06-S024], [T06-S037]

#### 5. Programming (Pre-design)

- **中文**: 任务书 / 策划
- **类别**: A. 项目交付
- **定义**: SD 之前的 phase, 把业主 需求 → 量化的 space program (room list + 面积 + 关键 adjacency). AIA 把它视为 basic services 之前的 additional services 或 项目早期任务.
- **locale**: 中国对应 "设计任务书" — 通常业主提供, architect 协助修订. RIBA Stage 1 (Preparation and Briefing).
- **来源**: [T06-S001], [T06-S016]

#### 6. Level of Development (LOD) 100 / 200 / 300 / 350 / 400 / 500

- **中文**: 模型深度等级
- **类别**: A. 项目交付 (BIM)
- **定义**: BIMForum 维护的 reference, 用于约定 BIM model elements 在不同 phase 的 content + reliability. 原始 LOD 100/200/300/400/500 由 AIA 定义; LOD 350 由 BIMForum 工作组补入, 专门描述 "interfaces between systems" 以 enable 协调 (clash detection).
- **逐级**: 100 = conceptual massing; 200 = generic system / approximate; 300 = specific, accurate geometry; 350 = + interfaces / coordination ready; 400 = fabrication-ready; 500 = field-verified as-built.
- **常见混淆**: LOD ≠ Level of Detail (LoD 是 visual richness, LOD 是 model element reliability)
- **来源**: [T06-S014], [T06-S015], [T06-S063]

#### 7. IFC (Industry Foundation Classes)

- **中文**: 工业基础类
- **类别**: A. 项目交付 (BIM 互操作)
- **定义**: buildingSMART International 开发的 open data schema, 让 BIM 数据在 Revit / Archicad / Tekla / Bentley 等不同 software 之间无损交换. 2013 年成为 ISO 16739.
- **来源**: [T06-S025], [T06-S063]

#### 8. COBie (Construction Operations Building information exchange)

- **中文**: 建造-运营信息交换格式
- **类别**: A. 项目交付 (handover)
- **定义**: 非专有 data 格式, 把 design + construction 阶段产生的 equipment lists / product data sheets / warranties / spare parts / preventive maintenance schedules 结构化交付给 owner / FM. 设计师在 spec 阶段需指定 COBie deliverables. 由 Bill East (US Army Corps of Engineers, 2007) 设计.
- **来源**: [T06-S026]

#### 9. ISO 19650 / PAS 1192 / BS 1192

- **中文**: BIM 信息管理国际标准 (ISO 19650)
- **类别**: A. 项目交付 (BIM workflow)
- **定义**: ISO 19650 是 BIM 信息管理 international 标准 (2019 1 月起 publish; Part 3 资产管理 / Part 5 安全管理 2020 出, Part 4 信息交换 2022 出, Part 6 H&S 2025 出). 起源于 UK PAS 1192 系列, BS 1192 是其前身. UK 老 BS 1192 已被 ISO 19650 superseded, 但很多 UK firm 仍沿用其命名习惯.
- **来源**: [T06-S025], [T06-S026]

#### 10. RFI (Request for Information)

- **中文**: 信息请求单
- **类别**: A. 项目交付 (CA 工具)
- **定义**: 施工期由 contractor 发给 architect 的 formal 文档, 要求澄清 contract documents 中的问题. AIA G716 是 industry-standard RFI 表格. 关键: RFI 本身既不授权增加成本也不授权增加工期 — 那是 change order 的活.
- **来源**: [T06-S022], [T06-S023]

#### 11. Submittal

- **中文**: 报审件 (shop drawings / product data / samples / mockups)
- **类别**: A. 项目交付 (CA 工具)
- **定义**: contractor 提交给 architect review 的材料 — shop drawings / product data sheets / samples / mockups — 用于证明拟用材料 / 装配 与 contract documents 一致. Architect 的 review 是 "for general conformance with design intent", 不承担 means/methods/safety 责任.
- **来源**: [T06-S022]

#### 12. Change Order / RFP / Punch List

- **中文**: 变更单 / 询价书 / 缺陷整改清单
- **类别**: A. 项目交付 (CA 工具)
- **定义**: **Change Order** — 正式修改 contract 范围 / 成本 / 工期 的文档. **RFP (Request for Proposal)** — procurement 阶段向 contractor / consultant 征求 pricing 提案. **Punch List** — 接近完工时 architect 列出待修复缺陷的 list, completion 之前必须 close.
- **来源**: [T06-S022], [T06-S024]

#### 13. RIBA Plan of Work 2020 — Stages 0-7

- **中文**: RIBA 工作计划 2020 (8 阶段)
- **类别**: A. 项目交付 (英国体系)
- **定义**: RIBA 维护的英国行业 standard process. Stage 0 战略定义 / 1 准备与简报 / 2 概念设计 / 3 空间协调 / 4 技术设计 / 5 制造与施工 / 6 移交 / 7 使用. Stage 7 后可循环回新 Stage 0.
- **vs AIA 5-phase**: RIBA 把 use / handover 显式纳入流程, AIA 把它放在 "additional services". RIBA 把 design 分得更细 (Concept / Spatial / Technical).
- **来源**: [T06-S016], [T06-S017], [T06-S018]

### Category B. 设计 / 制图 vocabulary (12 entries)

#### 14. Parti / Parti Pris

- **中文**: 设计立意 / 核心构思
- **类别**: B. 设计 vocabulary
- **定义**: parti 是 parti pris (法语 "决定 / 立场已表态") 的简写. 一张 simplified schematic 图, 把建筑的 organizing idea 简化到最少 line/形 — 通常 single sketch 就能传达 (axis / massing strategy / void / 序列). Parti 不是平面 — 是 idea.
- **来源**: [T06-S053], [T06-S054]

#### 15. Poché

- **中文**: 实墙涂黑 / 黑实部分
- **类别**: B. 设计 vocabulary
- **定义**: Beaux-Arts 法语术语 — 平面图中 涂黑或填充 的 solid 厚墙 / 结构 / 服务空间. 视觉上区分 served space (空) vs servant space (实). Louis Kahn 著名地使用 poché 把楼梯 / WC / 设备塞入厚墙. 看到 parti + poché 一起谈是常见.
- **来源**: [T06-S053]

#### 16. Datum

- **中文**: 基准 (线 / 面 / 高程)
- **类别**: B. 设计 vocabulary
- **定义**: 一条 line / plane / volume 作为构图组织的参照轴, 让 disparate elements 通过共享 reference 形成关系. 例: 长廊 / cornice line / 水平水池. Ching 的《Architecture: Form, Space, and Order》经典词条.
- **来源**: [T06-S053]

#### 17. Massing

- **中文**: 体量 / 体块
- **类别**: B. 设计 vocabulary
- **定义**: 建筑的 overall 三维 composition / shape / size — 类比绘画的 composition 但是 3D. 在 SD 阶段, massing model (physical 或 BIM) 是核心输出.
- **来源**: [T06-S053]

#### 18. Fenestration

- **中文**: 开窗 / 窗洞布置
- **类别**: B. 设计 vocabulary
- **定义**: 建筑围护中任何 opening 的 design / construction / 存在 — windows / doors / vents / wall panels / skylights / curtain wall / louvers. 不是 "windows" 同义词, 是 envelope opening 综合策略.
- **来源**: [T06-S053]

#### 19. Plan / Section / Elevation

- **中文**: 平面 / 剖面 / 立面
- **类别**: B. 设计 vocabulary
- **定义**: 建筑三大 orthographic projection. **Plan** 水平剖切 (通常 cut 在 +1200mm); **Section** 垂直剖切; **Elevation** 正交立面投影 (没有透视). 在 SD/DD/CD 各阶段精度递增.
- **来源**: [T06-S053]

#### 20. Axonometric / Isometric / Oblique

- **中文**: 轴测图
- **类别**: B. 设计 vocabulary
- **定义**: 3D 表达 spatial 关系 + massing 而 无透视畸变. **Isometric** 三轴等角 (120°); **Axonometric** 严格是 平面 旋转后立起来 (保持平面真实); **Oblique** 一面平行另一面斜推. Stan Allen / SANAA / Dogma 等当代 firm 大量使用 axon.
- **来源**: [T06-S053]

#### 21. Tectonics (in drawing sense — 见 #34 for tectonics as theory)

- **中文**: 建构 (制图 / 节点)
- **类别**: B. 设计 vocabulary
- **定义**: 在 制图 / 节点 sense 下, tectonics 是 the art and science 后藏 building 的 assembly — 材料 connection / joint / detail 的 制作 + 装饰 + 装配. 见 #34 把它作为 theoretical 范畴 (Frampton).
- **来源**: [T06-S053]

#### 22. Programmatic / Program

- **中文**: 功能 / 功能配置
- **类别**: B. 设计 vocabulary
- **定义**: "Program" 在 建筑英语中 = building 内 functional 用途 list (room types + 面积 + adjacency). "Programmatic" 形容词 = 与 function 相关的. Programming 阶段产出 program brief (见 #5).
- **来源**: [T06-S001], [T06-S053]

#### 23. Circulation

- **中文**: 流线
- **类别**: B. 设计 vocabulary
- **定义**: 人 / vehicle 在建筑内移动的 path. 平面分析常区分 primary / secondary / service circulation. 中国"防火规范"严格管 evacuation circulation (见 #56).
- **来源**: [T06-S053]

#### 24. ADA (Americans with Disabilities Act) / Universal Design

- **中文**: ADA 美国残障人士法案 / 通用设计
- **类别**: B. 设计 vocabulary + 监管
- **定义**: **ADA** (1990, 美国 federal civil rights 法案) 强制 accessibility 要求 — door clearance, ramp slope ≤ 1:12, ADA-compliant 卫生间, lever handle, 转弯半径等. **Universal Design** 是 Ron Mace (1997) 提出的 broader design philosophy — 让 product / 环境 尽可能多人可用, 不靠 retrofit.
- **locale**: 中国对应「无障碍设计规范」GB 50763-2012, 部分指标更宽松, 部分更严.
- **来源**: [T06-S053]

#### 25. 'Good Bones' (informal)

- **中文**: 底子好 / 结构 + 体块 + 流线 三件套到位
- **类别**: B. 设计 vocabulary
- **定义**: 业内非正式用法, 形容一栋建筑 (常指既存改造对象) 拥有合适的 structural grid + 楼层高度 + circulation core + massing — 即使 finishes 老旧也值得 invest 改造. 用于 adaptive reuse 评估.
- **来源**: [T06-S053]

### Category C. 立面 / 建构 / 材料 (12 entries)

#### 26. Curtain Wall — Stick System

- **中文**: 杆式 (现场组装) 幕墙
- **类别**: C. 建构
- **定义**: 非承重 exterior cladding, 在现场逐件组装: vertical mullions 先装, 横向 transoms 接入, glazing + spandrel panel 最后嵌入. 灵活 / 适用 small-medium 项目 / 修改容易, 但现场 labor + 质量控制风险高 / 接缝多.
- **来源**: [T06-S044], [T06-S045], [T06-S046]

#### 27. Curtain Wall — Unitized System

- **中文**: 单元式 (工厂预制) 幕墙
- **类别**: C. 建构
- **定义**: 大型预制 module 在工厂完成 (glass + mullion + spandrel + seals), 运到现场快速吊装. 优点: 现场施工快 / 工厂级质量控制 / 适合超高层. 缺点: 单元尺寸受运输限制 / 现场不能改 / 单价高.
- **典型应用**: 上海中心 / Burj Khalifa 等所有 supertall.
- **来源**: [T06-S044], [T06-S045]

#### 28. Mullion / Transom

- **中文**: 竖框 / 横框
- **类别**: C. 建构
- **定义**: **Mullion** = 幕墙 / 窗系统中的 vertical 框 — 分隔 panel 并主要承担结构功能. **Transom** = horizontal 框, 连接 mullion 并把立面分成 glazing / panel grid.
- **来源**: [T06-S044], [T06-S045]

#### 29. Spandrel Panel

- **中文**: 实体 (不透明) 板块 / 防火带
- **类别**: C. 建构
- **定义**: 幕墙中 opaque 部分, 用于 hide 楼板边 / 吊顶 / HVAC plenum / 防火带. 常用 ceramic-fritted glass / 金属背板 / 石材. 防火工程上常对应 "防火带 / 实体段" 要求.
- **来源**: [T06-S044], [T06-S045]

#### 30. 半隐框 / 全隐框 / 明框 幕墙

- **中文**: 中国行业幕墙 sub-classification
- **类别**: C. 建构
- **定义**: 按 外侧 mullion 可见性分: **明框** (mullion + transom 全部可见 — 类似 stick 暴露版) / **半隐框** (横向 transom 隐藏, 竖向 mullion 露 — 常见 unitized) / **全隐框** (mullion + transom 都被 glass 盖住, 视觉 frameless). 中国《玻璃幕墙工程技术规范》JGJ 102 + 《建筑幕墙》GB/T 21086 管这套.
- **来源**: [T06-S046] (UK 政府 cladding 安全 reference); 同源中国 JGJ 102 不在本 manifest

#### 31. CLT (Cross-Laminated Timber) / Mass Timber

- **中文**: 正交胶合木 / 集成材
- **类别**: C. 材料
- **定义**: 多层 kiln-dried lumber 板, 每层垂直交错铺设, 结构胶粘接 + 压制成 solid 大板. 板可用作 floor / wall / roof, 平面 + 出面强度均高, 抗震表现良好. 立面用时 CLT 需 protected (额外 cladding / 涂层), 不能直接日晒雨淋. Mass timber 含 CLT + Glulam + NLT + DLT.
- **代表项目**: Mjøstårnet (挪威, 85m), Ascent (Milwaukee, 86m)
- **来源**: [T06-S055], [T06-S056], [T06-S057]

#### 32. GFRC / GRC (Glass Fiber Reinforced Concrete)

- **中文**: 玻璃纤维增强混凝土
- **类别**: C. 材料
- **定义**: 高强度水泥砂浆 + AR 玻璃纤维, 比普通混凝土 thin (10-20mm) 而保强度. 常用作 façade panel — 可塑性强, 复杂曲面 / 模仿石材效果. 重量约为 precast concrete 的 1/4.
- **代表项目**: Heydar Aliyev Center (Zaha Hadid) 大量使用.
- **来源**: (本 manifest 未单列 GFRC primary source — 当 inline 术语 cross-reference 给 T06-S053 教学 glossary)

#### 33. ETFE (Ethylene Tetrafluoroethylene)

- **中文**: 乙烯-四氟乙烯 (氟塑膜)
- **类别**: C. 材料
- **定义**: 透明 / 半透明 fluoropolymer 薄膜, 拉到 cushion (双 / 三层 + 充气) 或 single-layer 张拉, 重量约 同 size glass 1%, 透光率高, 抗紫外 / 自洁. 代表项目: Allianz Arena (2005), Beijing Water Cube (2008 奥运), Eden Project (UK).
- **来源**: [T06-S053]

#### 34. Tectonics / Tectonic Culture (theory sense)

- **中文**: 建构文化
- **类别**: C. 建构 + E. 智识 vocabulary
- **定义**: Kenneth Frampton《Studies in Tectonic Culture》(1995) — 把 tectonics 上升到 architectural value: 建筑的 ethical 意义 不在 image, 而在 the poetics of construction (Gottfried Semper 谱系 — earthwork / framework / cladding 区分). 当代用于 反 image-driven 建筑.
- **来源**: [T06-S041], [T06-S042], [T06-S040] (Schumacher 把 Tectonism 视为 Parametricism 演化最高阶段, 跟 Frampton tectonics 不完全同义但相关)

#### 35. Board-Formed Concrete

- **中文**: 木纹清水混凝土
- **类别**: C. 材料
- **定义**: 用 sawn timber 板做 formwork, 拆模后 concrete 表面留下 wood grain + 板缝 texture. Tadao Ando / Peter Zumthor / Herzog & de Meuron 常用. 表达 Semper 谱系中 "earthwork" 的物质性.
- **来源**: [T06-S053]

#### 36. Terrazzo

- **中文**: 水磨石
- **类别**: C. 材料
- **定义**: cement 或 epoxy 基 + 大理石 / 玻璃 / 金属 碎粒混合, 浇筑后打磨抛光. 经历 1920s-50s 黄金期, 1980s 衰退, 2010s 复兴 (Instagram-friendly). 当代 epoxy-based terrazzo 用色谱广 / 接缝少.
- **来源**: [T06-S053]

#### 37. Vertical Lapping / Standing Seam / Rainscreen Cladding

- **中文**: 立缝咬合 / 雨幕系统
- **类别**: C. 建构
- **定义**: **Standing seam** = 金属屋面 / 立面 板与板之间立起的咬合缝, 雨水从面板上流走. **Rainscreen** = 外层 cladding 与 内层 air/insulation/waterproof 之间留 cavity (通常 25-50mm), 让 wind-driven rain 通过 pressure equalization 不进入 内层. 当代 vented façade 通用做法.
- **来源**: [T06-S046]

### Category D. 可持续 / 性能 (14 entries)

#### 38. LEED v4 / v4.1 / v5 (USGBC)

- **中文**: 美国绿色建筑评级体系
- **类别**: D. 可持续 (国际)
- **全称**: Leadership in Energy and Environmental Design
- **当前状态**: v4.1 (2018) → v5 (正式发布 2025, full registration 切换 2026 年 7 月起). 商业 BD+C / ID+C / O+M rating systems 从 2026-07-01 起仅 v5 可注册.
- **认证级别**: Certified (40-49 pts) / Silver (50-59) / Gold (60-79) / Platinum (80+)
- **核心 categories** (v4): Location & Transport, Sustainable Sites, Water Efficiency, Energy & Atmosphere, Materials & Resources, Indoor Environmental Quality, Innovation, Regional Priority
- **v5 新焦点**: decarbonization (operational + embodied + refrigerants + transportation), quality of life (health + resilience + equity), ecological conservation & restoration. Platinum 新增 electrification + minimum energy efficiency + renewable + embodied carbon 阈值.
- **常见误读**: LEED Platinum ≠ 真正低碳 (operational vs embodied 区分, modeled vs measured 差距已被第三方研究反复指出)
- **来源**: [T06-S003], [T06-S004], [T06-S005], [T06-S006]

#### 39. BREEAM (BRE Environmental Assessment Method)

- **中文**: 英国绿色建筑评估方法
- **类别**: D. 可持续 (国际, 主导欧洲)
- **机构**: BRE Group (UK)
- **认证级别**: Unclassified (<30%) / Pass (>30%) / Good (>45%) / Very Good (>55%) / Excellent (>70%) / Outstanding (>85%). Outstanding 代表 top 1% 项目.
- **评估 categories** (10 项): Energy, Water, Waste, Pollution, Transport, Materials, Health & Wellbeing, Management, Land Use & Ecology, Innovation
- **vs LEED**: 加权机制 + 强制 minimum standards per rating level + 区域 surveyor 评估; LEED 偏 美国 + 北美市场, BREEAM 偏 欧洲 + 英联邦.
- **来源**: [T06-S007], [T06-S008], [T06-S062]

#### 40. WELL Building Standard (v1 / v2)

- **中文**: WELL 健康建筑标准
- **类别**: D. 可持续 (健康 / 人本)
- **机构**: IWBI (International WELL Building Institute, 2014 推出, Delos 创立)
- **认证级别**: Silver / Gold / Platinum
- **7 concepts** (v1): Air / Water / Nourishment / Light / Fitness / Comfort / Mind. v2 调整为 10 concepts (+Movement / Materials / Sound / Thermal Comfort / Community).
- **关键 distinct**: WELL 是 performance-based + 现场测量 (空气 / 水 / 光 / 声), 不是纯 design checklist; 必须 recertify 每 3 年.
- **来源**: [T06-S011], [T06-S060], [T06-S065]

#### 41. Living Building Challenge (LBC)

- **中文**: 活建筑挑战
- **类别**: D. 可持续 (最 stringent 标准)
- **机构**: ILFI (International Living Future Institute)
- **机制**: 必须 building 运营 ≥ 12 consecutive months, 凭 actual measured performance 评 (不是 modeled). 7 performance areas (称 Petals): Place / Water / Energy / Health & Happiness / Materials / Equity / Beauty.
- **关键要求**: Net Positive Water + Net Positive Energy + Red List 材料禁用 + 当地材料 + biophilic. 全球认证项目数 ~100 (vs LEED 十万级).
- **来源**: [T06-S012], [T06-S061], [T06-S066]

#### 42. Passive House — PHI (Darmstadt) vs Phius (Chicago)

- **中文**: 被动房标准 (两个独立体系)
- **类别**: D. 可持续 (低能耗住宅 / 商业)
- **PHI 核心 criteria**:
  - Annual heating demand ≤ 15 kWh/m²·a (或 peak heat load ≤ 10 W/m²)
  - Annual cooling demand ≤ 15 kWh/m²·a
  - Airtightness n50 ≤ 0.6 h⁻¹ @ ±50 Pa (blower door)
  - 一次能耗 (含 plugs + appliances) ≤ 60 kWh/m²·a (PER, classic) 或 ≤ 120 kWh/m²·a (Plus/Premium)
  - 机械通风 + 热回收 (典型 ≥ 75% sensible efficiency)
- **Phius 差异**: 美国 Chicago, 把目标按 climate zone 调整 (PHI 是 global flat target). Phius CORE / Phius ZERO 两级.
- **设计指标**: 高保温 + thermal bridge free + 三玻 + ERV/HRV + 朝向最优化
- **来源**: [T06-S009], [T06-S010], [T06-S047], [T06-S059]

#### 43. Net Zero Energy / Net Zero Carbon

- **中文**: 净零能耗 / 净零碳
- **类别**: D. 可持续
- **定义**: **Net Zero Energy** = annual 现场可再生 产能 ≥ annual 总能耗. **Net Zero Carbon** 更严 — 全 lifecycle 或 operational 阶段 carbon 排放 ≤ 抵消量. AIA 2030 Commitment 把 2030 年 net zero operational 列为行业目标; Architecture 2030 是 Ed Mazria 推动的倡议.
- **来源**: [T06-S049], [T06-S050]

#### 44. EUI (Energy Use Intensity)

- **中文**: 能耗强度
- **类别**: D. 可持续 (KPI)
- **定义**: 年度能耗 / 总建筑面积. 单位: kBtu/ft²·yr (US) 或 kWh/m²·yr (公制). benchmarking 工具 (ENERGY STAR Portfolio Manager, CBECS). 当代 office EUI typical 50-90 kBtu/ft²·yr, Passive House < 20.
- **来源**: [T06-S013], [T06-S049]

#### 45. sDA (spatial Daylight Autonomy) / ASE (Annual Sun Exposure)

- **中文**: 空间日光自治 / 年度阳光暴露
- **类别**: D. 可持续 (日光性能)
- **定义**: LEED v4 EQ Credit 8.1 引入. **sDA300/50%** = 空间中 ≥ 50% occupied hours 达到 ≥ 300 lux 的 floor area %. **ASE1000,250** = ≥ 250 直射小时 / yr 收到 ≥ 1000 lux 的 floor area % (越低越好, 防眩光 / 过热). IES LM-83-12 是底层 metric.
- **来源**: [T06-S003], [T06-S013]

#### 46. LCA (Life Cycle Assessment)

- **中文**: 生命周期评估
- **类别**: D. 可持续 (方法论)
- **定义**: 系统方法评估建筑各阶段 (原材料提取 → 制造 → 施工 → 使用 → 维护 → 处置 / 回收) 的 environmental impact. ISO 14040 / 14044 是基础方法标准. EN 15978 是建筑专用. 输出含 GWP (kg CO2e), AP, EP, ODP 等多 indicator.
- **来源**: [T06-S048], [T06-S051], [T06-S052]

#### 47. EPD (Environmental Product Declaration)

- **中文**: 环境产品声明
- **类别**: D. 可持续
- **定义**: 制造产品的 "环境营养标签" — 显示 functional unit 产品的 embodied carbon 等 impact. 由 LCA 算出 + 第三方 verify. 通常 EPD for structural material 覆盖 A1-A3 (cradle to gate). LEED v4/v5 MR Credit 给 EPD 加分.
- **来源**: [T06-S048], [T06-S051], [T06-S052]

#### 48. Life Cycle Stages A1-A5 / B1-B7 / C1-C4 / D

- **中文**: 生命周期阶段模块 (EN 15978)
- **类别**: D. 可持续
- **逐模块**:
  - **A 产品 + 施工**: A1 raw material supply / A2 transport / A3 manufacturing / A4 transport-to-site / A5 construction-installation
  - **B 使用**: B1 use / B2 maintenance / B3 repair / B4 replacement / B5 refurbishment / B6 operational energy / B7 operational water
  - **C 处置**: C1 deconstruction / C2 transport / C3 waste processing / C4 disposal
  - **D 边界外回收 / 再利用收益**
- **常见 scope 简称**: A1-A3 = "cradle to gate"; A1-A5 = "cradle to practical handover"; A-C = "cradle to grave"; A-D = "cradle to cradle".
- **来源**: [T06-S048], [T06-S051], [T06-S052]

#### 49. Embodied Carbon vs Operational Carbon

- **中文**: 隐含碳 vs 运营碳
- **类别**: D. 可持续
- **定义**: **Embodied** = 建筑投入使用前 + life cycle 中非运营产生的 CO₂ (A1-A5 + B 部分 + C). **Operational** = 使用阶段 能源 + 水 (B6 + B7). 全球建筑业占年 CO₂ 排放 ~40%, 其中 embodied ~13% 且 share 在增长 (电网清洁 + 运营降, 但 embodied 在前期 lock-in).
- **AIA 2030 / RIBA 2030**: 都对 operational + embodied 设阶段目标; SE 2050 / MEP 2040 行业 commitments.
- **来源**: [T06-S048], [T06-S049], [T06-S050]

#### 50. ASHRAE 90.1 / IECC (energy code baselines)

- **中文**: ASHRAE 90.1 能源标准 / IECC 国际节能规范
- **类别**: D. 可持续 (强制 baseline)
- **定义**: 美国两大基础能源 code — ANSI/ASHRAE/IES 90.1 (商业 / 大型 多层住宅) + IECC (Inter­national Energy Conservation Code, 含住宅 + 商业). 各州采纳其一或并用. LEED v4/v5 + ASHRAE 189.1 (高性能) + 当地 stretch code 在其上 layer.
- **关键 envelope 指标**: max U-factor + max SHGC (Solar Heat Gain Coefficient) 按 climate zone 分级; mandatory 日光控制 + 3% skylight on roof (in 2019+ 版本).
- **来源**: [T06-S013], [T06-S067], [T06-S068]

#### 51. AIA 2030 Commitment / Architecture 2030

- **中文**: AIA 2030 行业承诺 / Architecture 2030 倡议
- **类别**: D. 可持续 (倡议)
- **定义**: Architecture 2030 (Ed Mazria 创立) 设 2030 Challenge — 新建 + 改造 项目 在 2030 年达成 carbon neutral (operational). AIA 2030 Commitment 是 AIA firms 签署 + 每年报告 portfolio EUI / embodied carbon 的 framework. 2024+ 已扩展含 embodied carbon reporting.
- **来源**: [T06-S049]

### Category E. 智识 vocabulary (流派 / 理论, 9 entries)

#### 52. Parametricism

- **中文**: 参数化主义
- **类别**: E. 智识 vocabulary
- **定义**: Patrik Schumacher (Zaha Hadid Architects) 2008 提出, 在 2010 Venice Biennale 进一步阐述为 "global style for architecture and urban design". 把 parametric design 工具 + agent-based + script-driven 流程 上升为 epochal 风格. Schumacher 列出 5 sub-style 演化: Folding → Blobism → Swarmism → Tectonism. **Tectonism** 是 Schumacher 视为最成熟阶段 — fabric modulation 延伸到 tectonic articulation. (注意 Schumacher tectonism ≠ Frampton tectonics)
- **批评**: Daniel Davis / 当代评论者 指出 Parametricism 多 style/aesthetic 化, 缺少社会 / 政治维度.
- **来源**: [T06-S038], [T06-S039], [T06-S040]

#### 53. Critical Regionalism

- **中文**: 批判性地域主义
- **类别**: E. 智识 vocabulary
- **定义**: Alex Tzonis + Liane Lefaivre (1981) 提出, Kenneth Frampton (1983) "Towards a Critical Regionalism: Six Points for an Architecture of Resistance" 发扬. 介于 国际现代主义 + 浪漫地方主义 之间 — 不 nostalgia 复古, 而是 mediating universal civilization 用 place-defining regional elements (材料 / 气候响应 / 场地 / 触觉) 抵御 "counterfeit commercial regionalism". Frampton 称为 "synthetic contradiction".
- **代表建筑师**: Alvar Aalto / Luis Barragán / Tadao Ando / 王澍 (部分讨论)
- **当代 evolution**: "Ultra-Regionalism" — 一些学者认为 civilization vs culture 二元 已被全球化 dissolve.
- **来源**: [T06-S041], (Frampton 文本 cross-ref 至 T06-S053 教学 glossary 链接的 Frampton 入门)

#### 54. Phenomenology of Architecture

- **中文**: 建筑现象学
- **类别**: E. 智识 vocabulary
- **定义**: 反 image / scenography, 强调 atmospheric experience + multisensory + 时间感. **Juhani Pallasmaa**《The Eyes of the Skin》(1996) — 反 ocularcentrism, 主张 触觉 / 听觉 / 嗅觉 同等; **Steven Holl** — phenomenal transparency + 光-时间-材质; **Christian Norberg-Schulz**《Genius Loci》(1980) — 场所精神. 哲学根源: Husserl / Heidegger / Merleau-Ponty.
- **来源**: (无 dedicated primary in manifest — 当 cross-reference 给 T06-S053 教学 glossary + T06-S041 Koolhaas 出版方 关于 "place" vs "space" 讨论)

#### 55. Tectonic Culture (Frampton)

- **中文**: 建构文化
- **类别**: E. 智识 vocabulary (见 #34 building sense)
- **定义**: Kenneth Frampton《Studies in Tectonic Culture: The Poetics of Construction in Nineteenth and Twentieth Century Architecture》(1995). 沿 Gottfried Semper 谱系 (earthwork / hearth / framework / cladding 四元素) 把 the poetics of construction 上升为 architectural ethics. Counter-discourse 反 image-driven / 反 spectacle.
- **代表分析对象**: Auguste Perret / Wright / Mies / Kahn / Scarpa / Utzon
- **来源**: [T06-S042], [T06-S043] (Koolhaas Junkspace 是反向案例)

#### 56. Deconstructivism

- **中文**: 解构主义建筑
- **类别**: E. 智识 vocabulary
- **定义**: 1988 MoMA "Deconstructivist Architecture" 展 (Philip Johnson + Mark Wigley 策展) 定型. 7 位建筑师: Eisenman / Gehry / Hadid / Koolhaas / Libeskind / Tschumi / Coop Himmelb(l)au. 借 Derrida 哲学 (deconstruction) + Russian Constructivism, 视觉特征: fragmented geometry / 非欧氏 / 倾斜 / surface 与 structure 解耦. Koolhaas 是该 movement 长期遗产 carrier.
- **来源**: [T06-S041], [T06-S063]

#### 57. Bigness (Rem Koolhaas)

- **中文**: 大 / 巨大性
- **类别**: E. 智识 vocabulary
- **定义**: Koolhaas《S, M, L, XL》(1995) 第三篇 "Bigness, or the Problem of Large". 当建筑超过 critical scale, 它逃脱 architecture 经典工具 (composition, scale, proportion, detail) 的支配, 变为 自治 实体 — 立面 与 内部 解除关系, 程序混合, 总 transparency 成为 ideology. 当代 mall / airport / hub 的描述.
- **来源**: [T06-S041], [T06-S042]

#### 58. Junkspace (Koolhaas)

- **中文**: 垃圾空间
- **类别**: E. 智识 vocabulary
- **定义**: Koolhaas 2001 同名 essay (October magazine + Harvard Project on the City). modernization 的 残留 — mall / airport / 候机楼 / 会议中心 / 酒店大堂 共同的 空虚 / 无序 / 装饰过剩 / 永远 under-renovation 的 空间. 后 Augé "non-place" 谱系. 反 phenomenology / 反 place.
- **来源**: [T06-S041], [T06-S043]

#### 59. Minimalism / High-Tech / Postmodernism / Neo-vernacular (movements 一并)

- **中文**: 极简主义 / 高技派 / 后现代主义 / 新乡土
- **类别**: E. 智识 vocabulary (movements)
- **定义**:
  - **Minimalism**: John Pawson / Tadao Ando / SANAA — 去除装饰 + 材料纯粹 + 光为媒介
  - **High-Tech**: Norman Foster / Richard Rogers / Renzo Piano — exposed structure + service externalization + 工业美学 (Centre Pompidou 经典)
  - **Postmodernism**: Robert Venturi / Charles Moore / Michael Graves — 反 Mies "less is more" → "less is bore", 复合 / 历史引用 / irony
  - **Neo-vernacular**: 当代用 local 材料 + typology 的 contextual 路径, 与 critical regionalism 重叠 但 less critical
- **来源**: [T06-S041], [T06-S063]

#### 60. Generic City / Blob Architecture

- **中文**: 通用城市 / 团状建筑
- **类别**: E. 智识 vocabulary
- **定义**: **Generic City** — Koolhaas 1995 essay, 描述当代亚洲新城等 没有 history / center / identity 的城市, 是 21 世纪现实. **Blob Architecture** — Greg Lynn 1990s 提出, NURBS-driven curvilinear 形态, 是 Parametricism 早期 sub-style.
- **来源**: [T06-S041], [T06-S038]

### Category F. 中国本土合规 / 规范 (15 entries)

#### 61. 容积率 (FAR / Floor Area Ratio)

- **中文**: 容积率
- **类别**: F. 中国合规 (规划指标)
- **定义** (按 GB 50352-2019): 在一定用地及计容范围内, 建筑面积总和 与用地面积的比值. 是 控规 (控制性详细规划) 最核心指标. 例: FAR 2.5 = 用地 10,000 m² 总建筑面积 25,000 m².
- **国际对照**: FAR (US) / Plot Ratio (UK) / Coefficient d'Occupation des Sols (FR)
- **来源**: [T06-S031]

#### 62. 建筑密度 (Site Coverage)

- **中文**: 建筑密度 (建筑覆盖率)
- **类别**: F. 中国合规 (规划指标)
- **定义** (按 GB 50352-2019): 在一定用地范围内, 建筑物 基底面积总和 与 总用地面积 的比率 (%). 控制 site footprint 留白.
- **来源**: [T06-S031]

#### 63. 限高 / 退线 / 间距 (三个规划硬约束)

- **中文**: 建筑限高 / 用地退线 / 建筑间距
- **类别**: F. 中国合规 (规划指标)
- **定义**:
  - **限高**: 控规规定 max 建筑高度, 受机场净空 / 历史保护 / 城市天际线 / 消防车云梯 等约束
  - **退线**: 建筑 外墙(含 阳台 + 屋顶 outline 投影) 必须 退离 用地红线 / 道路红线 / 绿线 一定距离 — 各市不同
  - **间距**: 建筑之间最小距离, 取 max(日照间距 + 消防间距 + 通风采光 + 视线卫生). 当地《城乡规划技术管理规定》具体落实.
- **来源**: [T06-S031]

#### 64. 日照分析 (大寒日 / 冬至日)

- **中文**: 日照分析
- **类别**: F. 中国合规 (居住 + 老年 + 学校)
- **定义** (按 GB 50180-2018《城市居住区规划设计标准》):
  - 住宅: 大寒日满窗 ≥ 2 h (新区) 或 ≥ 1 h (旧区改建)
  - 老年人居住建筑: 冬至日 ≥ 2 h
  - 校园 (教室): 冬至日 ≥ 2 h (按《中小学校设计规范》)
- **计算工具**: SUNLIGHT / 众智日照 / Ecotect-legacy, 按 GB/T 50947-2014《建筑日照计算参数标准》
- **来源**: [T06-S027], [T06-S028]

#### 65. GB 50016《建筑设计防火规范》(2014, 2018 版)

- **中文**: 建筑设计防火规范 (国标 强制性)

#### 66. 防火分区 / 疏散距离

- **中文**: 防火分区 / 疏散距离

  - **防火分区**: 由防火墙 + 楼板分隔的最大允许建筑面积区域; 多层穿透时 累加计算. 民用建筑各类用途的 max 分区 不同 (一类高层 1500 m² 等).

- **中文**: 工程监理
- **类别**: G. 执业 (中国, 与 architect CA 独立)
- **定义**: 中国独有的 强制第三方制度 (《建设工程监理范围和规模标准规定》). 监理 ≠ architect — 监理是 独立单位, 受业主委托, 监督 contractor 施工 + 质量 + 安全 + 进度. Architect 的 "施工配合" 不替代监理. 监理工程师 必须 注册 (《工程监理企业资质管理规定》).
- **来源**: [T06-S037]

## Standards & Regulations (国际 + 中国 对照)

### International

#### LEED v4 / v4.1 / v5 (USGBC)

- **全称**: Leadership in Energy and Environmental Design
- **机构**: U.S. Green Building Council
- **当前版本**: v4.1 (2018) → v5 (2025 launch); 商业 BD+C/ID+C/O+M 2026-07-01 起 仅 v5
- **认证级别**: Certified / Silver / Gold / Platinum
- **核心指标**: site / water / energy / materials / IEQ / innovation; v5 三大焦点: decarbonization / quality of life / ecological restoration
- **争议**: Platinum 不等于真正低碳 (operational vs embodied; modeled vs measured 差距)
- **来源**: [T06-S003], [T06-S004], [T06-S005], [T06-S006]

#### BREEAM (BRE Group, UK)

- **全称**: BRE Environmental Assessment Method
- **6 levels**: Unclassified / Pass / Good / Very Good / Excellent / Outstanding
- **10 categories**: Energy / Water / Waste / Pollution / Transport / Materials / Health & Wellbeing / Management / Land Use & Ecology / Innovation
- **来源**: [T06-S007], [T06-S008], [T06-S062]

#### WELL Building Standard (IWBI)

- **3 levels**: Silver / Gold / Platinum
- **performance-based** — 现场测量 + 3 年 recertify
- **来源**: [T06-S011], [T06-S060], [T06-S065], [T06-S066]

#### Living Building Challenge (ILFI)

- **机制**: post-occupancy 12 月 actual performance
- **7 Petals**: Place / Water / Energy / Health & Happiness / Materials / Equity / Beauty
- **来源**: [T06-S012], [T06-S061]

#### Passive House (PHI Darmstadt / Phius Chicago)

- **PHI 核心**: heating ≤ 15 kWh/m²a, airtightness n50 ≤ 0.6 h⁻¹
- **Phius**: climate-adjusted targets; Phius CORE / Phius ZERO
- **来源**: [T06-S009], [T06-S010], [T06-S047], [T06-S059]

#### ASHRAE 90.1 / IECC (US energy codes)

- ANSI/ASHRAE/IES 90.1-2022 是 最新; IECC residential + commercial; daylight mandatory; envelope U / SHGC per climate zone
- **来源**: [T06-S013], [T06-S067], [T06-S068]

#### NCARB AXP + ARE (US licensing path)

- 4 steps: Education / Experience / Examination / Registration
- **AXP**: 3,740 hours across 6 experience areas, ~half under licensed architect supervision (Setting A); up to 1,860 hours under Setting O
- **ARE**: multi-division exam, accepted by all 55 US jurisdictions
- **NCARB Certificate**: facilitates reciprocity among 55 Member Boards
- **来源**: [T06-S019], [T06-S020], [T06-S021], [T06-S058]

#### RIBA Chartered (UK) + ARB (UK)

- RIBA = professional body (membership); ARB (Architects Registration Board) = statutory regulator (license to call self "architect" in UK)
- Path: 5 yr education + 24 mo practical experience + Part 3 exam
- **来源**: [T06-S016]

#### BIM standards (ISO 19650 / IFC ISO 16739 / COBie)

- ISO 19650 information management (origin: UK PAS 1192 / BS 1192)
- IFC = open schema (ISO 16739, by buildingSMART)
- COBie = handover data spec
- BIMForum LOD 100/200/300/350/400/500 是事实标准
- **来源**: [T06-S014], [T06-S015], [T06-S025], [T06-S026], [T06-S063]

### China 本土

#### GB 50016《建筑设计防火规范》(2014, 2018 版)

- 类别: 强制性国家标准 (GB)
- 主管: 公安部 + 住建部
- 核心范围: 防火分类 + 耐火等级 + 防火分区 + 安全疏散 + 防排烟 + 防火间距
- 必看章节: 第 5 章 (5.1 防火分区, 5.5 安全疏散) / 第 6 章 建筑构造 / 第 8 章 消防设施
- 2018 修订: 高层公建 / 老年人 / 儿童活动 / 仓库 等加严
- **来源**: [T06-S029], [T06-S030]

#### GB 50352-2019《民用建筑设计统一标准》

- 替代 GB 50352-2005 (《通则》)
- 容积率 / 建筑密度 / 绿地率 控制指标对接控规
- **来源**: [T06-S031]

#### GB 50180-2018《城市居住区规划设计标准》

- 日照标准: 住宅大寒日 ≥ 2h (新区) / ≥ 1h (旧区), 老年人 冬至日 ≥ 2h
- 间距 + 配套 + 路网
- **来源**: [T06-S027]

#### GB/T 50947-2014《建筑日照计算参数标准》

- 太阳赤纬 / 时角 / 大气消光 计算参数
- 日照分析软件计算依据
- **来源**: [T06-S028]

#### GB/T 50378-2019《绿色建筑评价标准》(2024 版)

- 4 级: 基本级 / 一星 / 二星 / 三星, 60/70/85 分
- 5 类指标: 安全耐久 / 健康舒适 / 生活便利 / 资源节约 / 环境宜居
- 一二三星 强制 全装修
- **来源**: [T06-S032]

#### GB/T 51129-2017《装配式建筑评价标准》

- 装配率 A / AA / AAA 三级 (60-75 / 76-90 / 91+)
- **来源**: [T06-S033]

#### 注册建筑师 / 设计资质 / 工程监理 (3 个 监管文件 cluster)

- 《中华人民共和国注册建筑师条例》 — 一级 / 二级 注册建筑师 制度
- 《工程设计资质标准》 — 设计院 甲 / 乙 / 丙级 资质
- 《工程监理企业资质管理规定》 — 独立第三方监理体系
- **来源**: [T06-S034], [T06-S035], [T06-S036], [T06-S037]

## Phase 2 接口

**最高频术语** (跨多 figure / canon / tool / workflow 出现, 候选 voice DNA + Agentic Protocol 维度):

- **Schematic Design (SD) / Design Development (DD) / Construction Documents (CD)** — 5/5 工作流强相关, 任何 practitioner 的 daily 词汇
- **Parti / parti pris** — figure 教学 + 评图 + 评图 critique 都用; 与 "concept" 在 voice 上区分明显
- **Massing** — SD/DD/render/critique 都出现
- **LEED v4/v5 + Passive House** — 当代项目可持续 baseline 之上的 brand 标签
- **Embodied carbon / operational carbon** — 2020s 后最热的 单一 issue; AIA 2030 / RIBA 2030 都围绕它
- **BIM LOD 100-500 + IFC + COBie** — workflow 端
- **GB 50016 防火分区 / 疏散** — 中国实操 必经
- **容积率 FAR / 退线 / 限高 / 间距 / 日照** — 中国规划报建 5 件套
- **Tectonics (Frampton sense)** — 流派 / 评论 / 学术 vocabulary 高频
- **Critical regionalism** — 跨 figure (Frampton, Aalto, Ando, 王澍) 的接合术语

**locale 差异显著的术语** (国际 vs 中国 不同 — Phase 2 必须 双 mode 处理):

- 设计流程: SD/DD/CD/CA vs 方案/初设/施工图/施工图审查 (流程切分 + 第三方制度都不同)
- 评级体系: LEED vs 绿建三星 (评分逻辑 + 控规挂钩程度 不同)
- Construction Administration: AIA CA = architect 主导 vs 中国 architect 现场服务 + 独立监理 双轨
- 资质制度: 美国 firm 无 federal 等级, 中国 甲/乙/丙
- 节能 vocabulary: EUI (kWh/m²·yr) vs "节能 65%/75%/80%" 相对 reference 建筑表达
- 防火: IBC fire compartment + occupancy classification vs 防火分区 + 民用建筑分类 (一/二/三类高层)
- 制图节点术语: mullion / transom / spandrel vs 中国行业 龙骨 / 横料 / 实墙; 半隐 / 全隐 / 明框 是中国独有 sub-classification
- 雨洪: LID / SuDS vs 海绵城市 (政策驱动度不同)
- 装配建造: prefab / modular / DfMA vs 装配率 (中国把它强 KPI 化)
