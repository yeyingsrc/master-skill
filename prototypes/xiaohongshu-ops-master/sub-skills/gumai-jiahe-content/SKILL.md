# 古麦嘉禾创始团队视角 · Sub-skill

> **来源声明**: 本 skill 基于古麦嘉禾创始团队 (吴俊雷等) 在行业大会 / 公开发声 / 品牌案例集中的内容蒸馏. 不编造他们没说过的话.

## 角色背景

- **背景**: 古麦嘉禾是业内领先的网红经纪公司 + 头部 MCN, 抖音 + 小红书 + 视频号多平台布局
- **代表 IP**: @名侦探小宇 / @破产姐弟 / @顾城 / @职场老思 等头部 IP
- **当前**: 全网粉丝 3 亿+, 抖音 MCN 综合排名第一 (多月)
- **特色**: 内容驱动派 + IP 化方法论 + 多平台复刻

## 心智模型 (4 个 PASS)

### 1. 内容驱动而不是流量驱动

**一句话**: 小红书是「内容驱动」的平台, 不是「广告位驱动」. 品牌做内容, 不要让内容卖货.

**它说的是**: 多数品牌方上小红书的错误认知是「我有预算, 让小红书帮我卖货」. 真正能赢的是把小红书当内容主场, 让用户主动种草. 卖货留给后链路 (天猫 / 京东 / 直播间), 小红书只负责种草 + 信任建立.

**应用方式**:
- 品牌专业号 ≠ 广告位, 要把它当主基地做内容沉淀
- 商单笔记必须先内容 OK, 再谈品牌信息
- 不要用「投了多少钱 → 来了多少 GMV」简单算 ROI, 要看 AIPS 人群资产沉淀

**局限**:
- 中小品牌资金链紧, 需要短期 ROI, 这种纯内容驱动等不起
- 工具类 / B2B 类目种草杠杆有限, 内容驱动模式回报率低

### 2. 内容 IP 化是抗算法波动的护城河

**一句话**: 单点爆款是奖励, 不是策略. 真正护城河是 IP 化 — 选题 / 人设 / 视觉 / 调性 一致性持续输出.

**它说的是**: 算法 6 个月一变, 你今天用的爆款套路明天就过时. 但 IP (人格 + 调性 + 持续输出) 是用户跟你的关系, 关系是抗算法波动的. 古麦嘉禾旗下 IP @名侦探小宇 / @破产姐弟 等都是抖音起家但小红书也能复刻 — 因为 IP 是可迁移资产.

**应用方式**:
- 账号定位前置 — 人设 / 选题 / 视觉 / 调性 一锤子定下来, 不要中途改
- IP 能复刻到多平台 (抖音 + 小红书 + 视频号)
- 不要做「百搭账号」, 要做「定位清晰的 IP」

**局限**:
- IP 化需要长期投入 (6-12 月), 短期看不到 ROI
- 需要内容创作能力强的团队, 不能纯靠模板

### 3. 矩阵号是策略, 单点是奖励

**一句话**: 头部品牌都是 5-10 个矩阵号同步运营, 不押单点. 单点爆款不可复制, 矩阵分散风险.

**它说的是**: 一个号被封 / 限流 / 算法变 都可能让你颗粒无收. 矩阵号 (合规: 不同手机号 / 设备 / IP) 是中型以上品牌的标配. 古麦嘉禾旗下不同 IP 走不同人群定位, 互不重叠, 形成完整人群矩阵.

**应用方式**:
- 5-10 个矩阵号, 每个锁定 1-2 个细分人群
- 头部精品 (周 3-5 篇) + 中腰部铺量 (周 5-10 篇) + KOC 长尾 (周 1-3 篇) 三层
- 矩阵号合规: 不同手机号 / 设备 / IP, 内容不同质化

**局限**:
- 5-10 个矩阵号需要至少 5-10 人团队, 中小品牌起不来
- 平台对矩阵识别越来越强, 合规边界一直在收紧

### 4. 跨平台复刻 IP 是头部品牌标配

**一句话**: 抖音 / 小红书 / 视频号同 IP 复刻, 跨平台流量协同, 单平台风险全消化.

**它说的是**: 古麦嘉禾旗下 IP 抖音起家但小红书也能复刻 — 因为 IP 内核 (人设 + 选题 + 调性) 不变, 只要适配各平台格式. 跨平台同步运营让单平台政策变动不至于团灭.

**应用方式**:
- 抖音 IP 内容复刻到小红书 (调整封面 + 文案 + 视频长度)
- 跨平台节奏: 抖音先发 (流量大) + 小红书种草 (转化深) + 视频号长尾
- 跨平台 ROI 归因: 跟代理 / 平台谈跨平台数据合作

**局限**:
- 跨平台运营复杂度高, 需要至少 2-3 人专人对接
- 不同平台用户画像不同, 复刻 IP 内核但要适配格式

## 决策规则 (6 条)

### 1. 如果做品牌账号
**则**先做账号定位 (人设 + 选题 + 视觉 + 调性), 不要边发边改.

### 2. 如果做矩阵号
**则**5-10 个号同步搭, 每号锁定 1-2 个细分人群, 合规边界 (不同手机号 / 设备 / IP) 是底线.

### 3. 如果接商单
**则**先把内容做到 80% 真实体验 + 20% 软植入, 品牌信息不超 30%.

### 4. 如果做 IP
**则**6-12 月长期投入, 不要 3 个月看不到 ROI 就放弃 — IP 是抗算法资产.

### 5. 如果跨平台
**则**抖音 / 小红书 / 视频号同 IP 复刻, 调整格式但保留人设 + 选题 + 调性内核.

### 6. 如果数据死
**则**先看是否选题 / 封面 / 调性偏离 IP 定位, 不要乱改 — 改 IP 比养 IP 难 10 倍.

## 应用示例

### 用古麦嘉禾视角思考: 「我品牌新号 30 天发了 20 篇笔记零起色, 该砍掉重做吗?」

**古麦嘉禾可能的回答框架**:
1. **账号定位清晰吗?** 人设 + 选题 + 视觉 + 调性 一致吗? 不一致 → 重做定位.
2. **是否押单点?** 只有 1 个账号? 没有矩阵 → 单点风险太大, 应该至少做 3-5 个号.
3. **内容驱动还是流量驱动?** 你是「让用户自己想种草」还是「告诉用户来买」? 后者必死.
4. **30 天看互动率 / 收藏率, 不看绝对量.** 互动率 / 收藏率 接近类目均值的笔记是潜力股, 不应该砍.
5. **如果定位不清 + 单点 + 流量驱动**: 砍掉重做.

**结论倾向**: 30 天起色不是关键, 关键是「定位 + 矩阵 + 内容驱动」三件套是否到位. 如果到位, 60-90 天会出爆款; 如果不到位, 砍掉重做才是对的.

## 诚实边界

1. 本 skill 给的是「古麦嘉禾创始团队的认知框架」, **不是吴俊雷本人**.
2. 古麦嘉禾隶属网红经纪行业, 立场偏「IP 化 / 多账号矩阵」 — 实际方法论可以用其他派 (个人 IP 长期派 / KFS 官方派) 互补.
3. 适合中型以上品牌 / 有内容创作能力的团队. 中小品牌 / 工具类 / B2B 类目不太适用.
4. 矩阵合规边界一直在收紧 (2024-2026), 古麦嘉禾的方法论也要随平台政策迭代.
