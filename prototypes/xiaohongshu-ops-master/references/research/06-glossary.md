# Track 06 — Glossary (小红书运营)

> 调研日期: 2026-05-02。高频术语 + 平台规则 + 商业化合规。

## 元数据
- 高频术语: 30
- 平台规则: 8
- 最近一次重大变化: 2024 蒲公英政策收紧

---

## 平台基础

### 内容
- **笔记 / 视频笔记 / 图文笔记**: 小红书内容单元 (说「帖子」是外行破绽)
- **封面图**: 笔记第一屏, 决定 70% 点击率
- **正文 / 标题 / 标签**: 笔记三要素
- **薯条**: 平台原生付费推流, 50 元起 (给单笔记)
- **信息流 / 聚光**: 信息流投放高级版 (品牌 / 大账户)

### 角色
- **KOL** (Key Opinion Leader): 头部博主 (≥ 50 万粉)
- **KOC** (Key Opinion Consumer): 中腰部 / 素人 (≤ 5 万粉)
- **专业号 / 普通号**: 品牌账号 vs 个人账号
- **矩阵号**: 同主体多账号
- **企业号 / 个人号**: 注册主体类型

### 流量
- **流量层级**: 初始 (< 1k) / 中等 (1k-10k) / 大流量池 (10k+)
- **完读率**: 看完笔记的比例 (≥ 35% 算优)
- **收藏率**: 收藏 / 曝光, 反映「有用」
- **互动率**: (赞 + 收藏 + 评论) / 曝光
- **CSF / COSMO**: 算法迭代名

### 商业内容
- **报备 / 非报备**: 商单是否走蒲公英
- **蒲公英**: 官方品牌合作平台
- **品牌词 / 品类词 / 长尾词**: 关键词层级
- **种草 / 拔草**: 引导购买 / 完成购买
- **黑科技**: 灰色玩法 (限流风险)

### 投放
- **CPC / CPM / CTR / CVR**: 信息流广告通用指标
- **达成成本**: 单次互动 / 转化的成本
- **冷启动期**: 新计划前 24-72 小时, 系统在学习人群

## 平台规则 (重点)

### 蒲公英商单规则
- 所有商单笔记必须在蒲公英报备 (从 2022 起)
- 不报备发商单 → 限流 + 账号风控
- 蒲公英抽佣 (10-15%)
- 报备后笔记会被打 #与 X 品牌合作 标签

### 笔记内容规则
- 引流到站外 (微信 / 电商) 限流甚至封号
- 联系方式不能直接放笔记
- 部分品类有禁词 (医美 / 金融 / 培训)
- 真实分享 vs 软广的判断 — 商业内容必须报备

### 流量分发机制 (公开版)
- 笔记初始流量池 ~500-1000
- 互动率达标 → 进入下一流量池
- 笔记发布后 7 天是黄金窗口

### 蓝 V (品牌专业号) 规则
- 品牌注册需提供营业执照 + 资质
- 蓝 V 账号有专属功能 (商品橱窗 / 私信回复 / 数据统计)

### 重点禁忌品类
- 医美 / 整形 (大量限制)
- 教育 (双减后部分内容下线)
- 金融 (P2P / 投资类禁止)
- 保健品 (严控功效宣传)
- 化妆品 (不能宣称药妆 / 治疗)

## 算法 / 推荐机制

| 概念 | 一句话 |
|------|------|
| **CES 模型** | Click + Engagement + Stay 综合评估笔记 |
| **打标签** | 平台给账号打的兴趣标签 |
| **冷启动期** | 新笔记前 1-3 小时的初始流量池 |
| **递增曝光** | 数据好的笔记进入更大流量池 |
| **滞后爆文** | 发布几天后突然爆量 |
| **用户画像** | 平台对每个账号粉丝的画像 |

## 最近一次行业 / 平台规则更新

### 2024-2025 主要变化

- **2024-Q3**: 蒲公英商单平台规则收紧, 报备机制升级
- **2024-Q4**: 矩阵账号管控加强, 同 IP / 同设备多账号被批量降权
- **2025-Q1**: 「聚光」投放平台升级, 信息流跟搜索流量进一步分化
- **2025-Q3**: 站外引流 (私信导微信) 严打, 大量账号封号
- **2025-Q4**: 视频笔记权重上调, 图文权重相对下降

### 衰减速度

- **平台规则衰减极高** (3-6 月一变): 商单 / 矩阵 / 引流规则反复调
- **算法变化中等** (6-12 月): CES 模型大方向稳定, 细节调整频
- **黑话相对稳定** (1-2 年): KOC / 笔记 / 流量池 等核心术语长期不变

## 外行破绽

- 把「笔记」叫「帖子」
- 不知道蒲公英 / 报备
- 把投流叫「投广告」
- 把刷量当核心策略
- 不懂 KOC vs KOL 区别
- 不知道站外引流的严重性
