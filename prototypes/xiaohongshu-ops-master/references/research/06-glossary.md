## A. 平台机制 / 算法 (20 项)

### CES 评分系统

- **CES 评分**: 笔记推荐权重模型 — **点赞 ×1 + 收藏 ×1 + 评论 ×4 + 转发 ×4 + 关注 ×8**
- **CES 含义**: Community Engagement Score (社区参与度评分)
- **流量池**: 笔记按 CES 进阶 — 0-200 (基础) → 200-500 (二级) → 大流量池 (爆款)
- **完读率**: 视频 / 图文笔记看完比例 (2025 年新增隐性指标)
- **互动率**: (点赞 + 收藏 + 评论) / 曝光 — 类目均值 1.5 倍以上算「好」
- **收藏率**: 收藏 / 曝光 — 高于点赞率 = 内容有用 (重要信号)
- **完播率**: 视频笔记看完比例 (2025 重点权重)
- **转化率**: 看到笔记 → 点击 → 加购 / 进店 / 转化
- **CTR (点击率)**: 曝光转点击, 8% 以上算「好封面」
- **24 小时数据期**: 笔记发布后 24 小时关键期, 决定能否进二级流量池
- **48 小时复刻期**: 爆款笔记 48 小时内复刻 2-3 条同主题
- **Honeymoon Period (蜜月期)**: 新号前 14-30 天的算法扶持期 (类比抖音机制)
- **限流**: 推荐量被算法压低 (违规 / 矩阵识别 / AI 内容 / 不报备商单等)
- **降权**: 整体账号权重降低, 影响后续所有笔记
- **封号**: 严重违规导致账号被禁止使用

### 推荐机制

- **基础流量池**: 0-200 浏览量 (笔记初始流量)
- **二级流量池**: 200-500 浏览量 (CES 高的笔记进入)
- **大流量池**: 500+ 浏览量 (爆款笔记)
- **搜索流量**: 平台超 65% 流量来源 (关键词匹配)
- **推荐流量**: 算法主推 (CES + 兴趣匹配)

## B. 商业化 / 蒲公英 (15 项)

### 蒲公英平台

- **蒲公英**: 小红书官方报备 + 商单交易平台
- **报备**: 商单笔记必须通过蒲公英平台报备 (合规标识)
- **千粉门槛**: 2022 起千粉素人允许接单 (开放 KOC 池)
- **信用等级 Lv0-Lv3+**:
  - Lv0: 无接单权限
  - Lv1: 仅接单权限
  - Lv2: 接单 + 官方优先推荐
  - Lv3: 金牌种草员, 去掉商业笔记数量上限
  - Lv3+: 「人间种草机」, 全开放
- **抽佣**: 平台抽佣 10% (品牌方付给博主金额的 10%)
- **72 小时审核**: 笔记通过社区审核需 72 小时
- **7 天确认**: 品牌方笔记发布后 7 天内确认订单结束
- **私单**: 不通过蒲公英的私下交易 — 平台一查就限流甚至封号
- **品牌专业号**: 企业认证账号, ¥ 600/年, 用千帆后台
- **企业认证**: ¥ 600/年, 解锁品牌专业号工具
- **报备笔记标识**: 商单笔记带「赞助」标识 (但不影响算法权重)
- **博主商单数量限制**: Lv1-Lv2 有数量上限, Lv3+ 全开放
- **创作者池**: 蒲公英内可被搜索的博主总池
- **代理服务商**: 蓝标 / 利欧 / 引力等品牌代理 — 大品牌走代理通道

## C. 营销方法论 (15 项)

### KFS 三件套

- **KFS**: KOL × Feeds × Search (官方推荐三件套)
- **K (KOL)**: 通过博主种草 — 选博主 + 内容生产
- **F (Feeds)**: 信息流投放 — 薯条 / 聚光跑量
- **S (Search)**: 搜索词渗透 — 品牌词 + 类目词 + 痛点词
- **B+ KFS**: B 端 (品牌专业号) 加 KFS 的进阶版 (2024 年提出)
- **KFS 比例**:
  - 阶段 1 内容摸索: 7:3:0
  - 阶段 2 内容验证: 5:3:2
  - 阶段 3 规模化: 4:4:2

### IDEA 早期方法论

- **IDEA**: Insight (机会洞察) / Decision (目标设定) / Execution (内容投放) / Audit (效果度量)
- **历史**: 2021 年蒲公英平台上线时官方提出, KFS 之前
- **价值**: 框架仍然好用 — KFS 是落地策略, IDEA 是策略框架

### AIPS 人群资产

- **AIPS**: A (认知 Awareness) / I (兴趣 Interest) / TI (深度兴趣 Targeted Interest) / P (购买 Purchase) / S (分享 Share)
- **价值**: 把种草「玄学」变成可度量的人群资产
- **TI → P 跳转**: 深度兴趣到购买的转化是关键节点
- **第一方数据**: 头部 2100+ 品牌跟小红书有第一方数据合作通道
- **种草联盟**: 跟阿里 / 京东 / 唯品会 / 美团等的数据合作 (小红星 / 小红盟 / 小红链)

## D. 内容创作 (10 项)

- **种草**: 让用户产生想买 / 想试的冲动 (vs 直接卖货)
- **拔草**: 用户购买后的体验反馈 (好 / 不好)
- **真实分享**: 平台底色 — 真实使用 / 真实经历
- **活人感**: 2025 平台关键词, 反对伪素人 / 反对硬广
- **软植入**: 品牌信息植入比例 ≤ 30%, 真实使用占 ≥ 80%
- **硬广**: 直白宣传产品的内容 (平台限流)
- **伪素人**: 伪装成素人的商单内容 (平台识别越来越强)
- **爆款**: 互动率 ≥ 类目均值 1.5x 的笔记
- **复刻**: 数据异常笔记 48 小时内复刻 2-3 条同主题
- **选题反推**: 从用户搜索词 / 痛点反推内容方向

## E. 投放工具栈 (10 项)

- **薯条 (Boost)**: 自助投放, ¥ 75 起投, 自然流 + 信息流双轨
- **聚光 (Spotlight)**: 专业级投放, ¥ 5000+ 起投
- **千帆**: 品牌专业号管理后台 (免费)
- **千瓜**: 第三方数据头部 (qian-gua.com)
- **新红**: 第三方方法论 + 数据 (xh.newrank.cn)
- **蝉小红**: 信息流投放案例库
- **灰豚**: 跨平台数据 + 视频拆解
- **CPC**: 单次点击价
- **CPM**: 千次曝光价
- **ROAS**: 投放回报 (Return on Ad Spend)

## F. 矩阵 / 跨平台 (10 项)

- **矩阵号**: 多账号同步运营 (5-30 个号)
- **矩阵合规**: 不同手机号 / 不同设备 / 不同 IP
- **品牌专业号**: 企业认证账号 (主基地)
- **个人 IP 长期派**: 真实人格 + 持续输出 (反矩阵派)
- **跨平台联动**: 小红书种草 + 抖音 / 视频号 / 天猫 / 京东闭环
- **种草直达**: 笔记直接跳第三方电商
- **后链路 ROI**: 跨平台成交归到种草内容
- **私域承接**: 公众号 / 微信社群承接小红书人群
- **跨人群投放**: 聚光的多人群同时投
- **大促**: 双 11 / 618 / 38 节等关键节点

## G. 平台规则 / 监管 (15 项)

### 内容合规

- **社区规范**: 小红书官方内容标准
- **违规处罚**: 限流 / 降权 / 封号 三级
- **政府监管**: 国家市场监管总局 / 网信办
- **互联网广告法**: 广告内容合规底线
- **未成年保护**: 涉及未成年内容严格审核
- **医美 / 保健 / 减肥**: 高敏感类目, 商单审核更严
- **金融 / 证券**: 不允许直接推荐
- **政治 / 宗教**: 平台禁区

### 商业化合规

- **报备义务**: 商单必须报备
- **真实性义务**: 不得虚假宣传
- **明示标识**: 报备笔记带「赞助」标识
- **抽佣合规**: 平台抽佣 10% 透明化
- **税务合规**: 博主收入需要个税申报
- **限流红线**:
  - 不报备私单
  - AI 直发不编辑
  - 矩阵号同设备
  - 刷量 / 刷互动

## H. 跨平台对照 (12 项)

### 小红书 vs 抖音

| 概念 | 小红书 | 抖音 |
|------|-------|------|
| 投放工具 | 聚光 / 薯条 | 巨量引擎 / 千川 |
| 数据工具 | 千瓜 | 蝉妈妈 / 灰豚 |
| 报备平台 | 蒲公英 | 星图 |
| 信用等级 | Lv0-Lv3+ | (类似分级) |
| 内容类型 | 图文为主 + 视频 | 视频为主 |
| 用户结构 | 一二线女性高净值 | 全民 / 下沉 |
| 商业化 | 种草 → 站外 / 站内闭环 | 直播带货为主 |
| 算法核心 | CES + 搜索 | 完播率 + 互动 |
| 跨平台联动 | 小红书种草 + 多平台闭环 | 抖音种草 + 抖音闭环 |
| 人群资产 | AIPS | (类似模型) |
| KOL 议价 | 中腰部 ¥ 5k-3w | 中腰部 ¥ 1w-5w |
| 流量结构 | 65% 搜索 | 主推荐流 |
