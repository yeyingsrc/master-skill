## 头部 MCN / 操盘手

### 古麦嘉禾创始人 (吴俊雷等)
- **核心一句话**: 小红书是「真实分享驱动的种草平台」, 不是流量平台。运营核心是「让品牌做内容」, 不是「让内容卖货」。
- **代表作**: 古麦嘉禾官方品牌案例 / 多次行业大会演讲
- **值得听**: 「小红书品牌运营的 3 个共同认知」keynote

### 群响 / 蜂群文化代表
- **核心一句话**: 矩阵号 + 多账号矩阵投放是中型品牌的标准做法。单点爆款是奖励, 不是策略。
- **代表作**: 群响私域运营年会 / 蜂群文化案例集
- **争议**: 矩阵账号合规边界一直在收紧

### 千瓜数据创始团队
- **核心一句话**: 「数据驱动 + 工具套路」是品牌方上小红书的杠杆。但工具也救不了内容差。
- **代表作**: 千瓜白皮书 / 月度行业报告
- **值得读**: 千瓜美妆 / 母婴 / 食品类目年度报告

## 头部投手 / 信息流操盘

### 小红书信息流投放圈头部 (匿名常见)
- **核心一句话**: 信息流投放是「让算法找到对的人」的工具, 不是「让流量来」的工具。
- **代表作**: 灰豚 / 蝉小红 数据驱动投放案例
- **值得读**: 「信息流投放从 0 到爆款」类 case study

### 头部代理服务商 (蓝标 / 利欧 / 引力传媒)
- **核心一句话**: 大品牌投小红书的标准玩法是「品牌专业号 + KOL + KOC + 信息流」四位一体。
- **代表作**: 各家代理年度品牌案例集

## 头部博主 / KOL

### 程十安 / 多位 100w+ 美妆博主
- **核心一句话**: 真实使用 + 个人审美 + 持续输出 = 头部 KOL 的护城河。
- **价值**: 看头部博主商单密度可推断品牌投放趋势

### 反共识声音 (反矩阵 / 反伪素人派)
- **核心一句话**: 平台对矩阵 / 伪素人内容的识别越来越强, 真实人格 / 真实经历是长期主义。
- **代表**: 部分头部个人 IP 博主反复强调

## 流派分裂

| 流派 | 主张 |
|------|------|
| **内容驱动派** | 选题 + 创意决定 80%, 工具是辅助 |
| **数据 / 矩阵派** | 多账号矩阵 + 数据迭代是品牌长期路径 |
| **跨平台联动派** | 小红书种草 + 天猫 / 抖音直播闭环 |

核心分歧: **单点爆款 vs 矩阵 + 数据迭代** — 决定了品牌投预算的长期路径
