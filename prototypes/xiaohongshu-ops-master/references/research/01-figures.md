## A. 平台官方代表 (战略级最高一手)

### 1. 小红书 COO 柯南 (CONAN)

- **核心一句话**: 「真实分享 + 商业化效率」是小红书永恒的张力, 必须同时拿稳, 不能为短期 GMV 牺牲社区. 2025 是「种草进入效果化」元年.
- **代表内容**:
  - 2025 WILL 商业大会 keynote「种草进入效果化时代」(2024-12-18)
  - 2026 WILL 商业大会 keynote (2025-12-22, 核心主题: 种草数据 → 直接 GMV 归因)
  - 「全域种草」官方提案 — 跨淘宝/京东/唯品会/美团的种草直达
- **代表观点**:
  1. 种草度量从「玄学」到「确定性」 — 2024 推 AIPS 人群资产 (认知 A / 兴趣 I / 深度兴趣 TI / 购买 P / 分享 S)
  2. 平台跟阿里 / 京东 / 唯品会 / 美团合作做「种草联盟」, 把后链路成交数据反哺给品牌
  3. KFS (KOL × Feeds × Search) 是商家投放的官方推荐组合, 不再单押 KOL
- **争议**: 商业化加速跟「真实分享」社区底色之间的张力 (晚点 / 36氪 长期跟踪)
- **来源**: [Primary] 2025/2026 WILL keynote + 财经媒体覆盖

### 2. 小红书蒲公英平台官方运营团队

- **核心一句话**: 商业笔记必须报备 + 信用等级决定接单上限, 私单一查就限流封号.
- **代表政策**:
  - 蒲公英信用等级 Lv0~Lv3+ (Lv0 无接单权限 / Lv1 仅接单 / Lv2 优先推荐 / Lv3 金牌种草员去掉商业笔记上限 / Lv3+ 「人间种草机」)
  - 商单流程: 品牌方 7 天内确认订单结束 + 笔记审核 72 小时
  - 千粉门槛准入 (2022 起千粉素人允许接单, 拓宽 KOC 池)
- **来源**: [Primary] 蒲公英官方帮助文档 + 平台公告 (pgy.xiaohongshu.com/help/docs)

## B. 头部 MCN 操盘手 (业内一手)

### 3. 古麦嘉禾创始团队 (吴俊雷等)

- **核心一句话**: 小红书是「内容驱动」的平台, 不是「广告位驱动」的平台. 品牌做内容, 不要让内容卖货 — 卖货留给后链路.
- **代表案例**:
  - @名侦探小宇 / @破产姐弟 / @顾城 / @职场老思 等头部 IP (古麦嘉禾旗下)
  - 多月获抖音 MCN 综合排名第一, 全网粉丝 3 亿+
  - 同步小红书布局 — 把 IP 类内容方法论复刻到笔记
- **代表观点**:
  1. 小红书品牌运营 3 个共同认知: 真实 / 长期 / 矩阵
  2. 「单点爆款是奖励, 不是策略」 — 真正杠杆是矩阵 + 数据迭代
  3. 内容 IP 化是抗算法波动的护城河
- **来源**: [Primary] 古麦嘉禾品牌官网 + 行业大会 keynote

### 4. 群响刘思毅 (群响创始人)

- **核心一句话**: 小红书是 2024-2025 私域 / 矩阵流量最值得深耕的平台之一, 但矩阵合规边界一直在收紧.
- **代表内容**:
  - 群响私域运营年会 (年度) keynote
  - 群响公众号 + 知识星球 (私域操盘手社群头部)
  - 群响夜话播客 (周更, 头部操盘手访谈)
- **代表观点**:
  1. 矩阵号是中型品牌标配, 单 IP 时代过去
  2. 平台对「伪素人 / 矩阵 / 黑科技」识别越来越强, 老套路活不到 2026
  3. 私域 (公众号 + 微信社群) 跟小红书内容池的协同是新增长点
- **来源**: [Primary] 群响公众号 + 群响夜话播客

### 5. 蜂群文化 (莫力洋)

- **核心一句话**: 矩阵 + 数据 + 多账号策略是中型品牌打小红书的标准做法.
- **代表案例**: 蜂群文化品牌案例集 (美妆 / 食品 / 母婴类目大量案例)
- **代表观点**:
  1. 一个品牌至少养 5-10 个矩阵号, 内容差异化覆盖人群细分
  2. 头部矩阵 + 中腰部铺量 + KOC 长尾 — 三层结构
- **争议**: 矩阵规模化跟「真实社区」张力 — 蜂群被多次质疑伪素人
- **来源**: [Primary] 蜂群文化官网案例集 + 行业大会发声

### 6. 千瓜数据创始团队

- **核心一句话**: 数据驱动 + 工具套路是品牌方上小红书的杠杆, 但工具救不了内容差.
- **代表内容**:
  - 千瓜数据月度行业报告 (美妆 / 母婴 / 食品 / 服饰各类目)
  - 千瓜白皮书 (年度, 头部品牌投放数据复盘)
  - 千瓜功能地图 + 笔记数据趋势图 (2025 上线)
- **代表观点**:
  1. 「先数据后内容」 — 选题先看竞品爆款数据, 不要拍脑袋
  2. 互动率 ≥ 类目均值 1.5 倍才算「好笔记」
  3. 收藏率 > 点赞率 = 内容有用 (重要信号)
- **来源**: [Primary] 千瓜官网 + 月度报告

### 7. 新红数据团队

- **核心一句话**: 工具方法论必须跟平台规则同步迭代 — 平台政策 6 个月一变, 工具方法论也得 6 个月一改.

## C. 头部投手 / 信息流操盘 (实战一手)

### 8. 头部代理服务商代表 (蓝标 / 利欧 / 引力传媒)

- **核心一句话**: 大品牌投小红书的标准玩法是「品牌专业号 + KOL + KOC + 信息流」四位一体, 缺一不可.
- **代表案例**: 各家代理年度案例集 (美妆 / 个护 / 食品大量品牌)
- **代表观点**:
  1. 品牌专业号是「主基地」 — 长期内容沉淀 + 后链路承接
  2. KOL (头部 ≥ 50 万粉) 解决破圈, KOC (千粉~10万粉) 解决说服力
  3. 信息流 (薯条 / 聚光) 解决精准触达, 自然流解决长期价值
- **来源**: [Primary] 各代理年度案例集 (蓝色光标 / 利欧 / 引力 公众号)

### 9. 朴朴超市 (KFS 实战标杆品牌)

- **核心一句话**: 3 个月从 0 到拿成果的 KFS 组合营销 — 内容摸索期 K:F:S = 7:3:0, 然后逐步加 F + S.
- **代表案例**: 朴朴超市小红书 KFS 组合营销 (2024 年, 数英 / 36氪 报道)
- **代表观点**:
  1. 内容摸索期目标: 累积 10-20 篇潜力爆款笔记 (互动率 ≥ 类目 1.5x)
  2. 单品爆款验证后再上 Feeds (薯条 / 聚光), 不要一上来就投流
  3. 搜索词 (S) 是品牌词 + 类目词 + 痛点词 三层
- **来源**: [Primary] 数英 + TopMarketing 案例报道

### 10. 卓尔数科 (年度影响力合作伙伴)

- **核心一句话**: 「种草进入效果化时代」, 数据归因是品牌方现在最关心的事.

## 流派分裂 (关键认知分歧)

| 流派 | 主张 | 适合 | 风险 |
|------|------|------|------|
| **内容驱动派** (古麦嘉禾代表) | 选题 + 创意决定 80%, 工具是辅助 | 中小品牌 / 内容能力强 | 算法变天就被动 |
| **数据 / 矩阵派** (蜂群文化代表) | 多账号矩阵 + 数据迭代是品牌长期路径 | 中大品牌 / 有预算 | 合规边界收紧 |
| **跨平台联动派** (新消费品牌代表) | 小红书种草 + 天猫 / 抖音直播闭环 | 头部品牌 / 跨平台 | 操盘复杂度极高 |
| **KFS 官方方法派** (朴朴 / 平台代理) | KOL × Feeds × Search 三件套 | 中大品牌 / 标准化 | 同质化严重 |
| **个人 IP 长期派** (反伪素人派) | 真实人格 / 持续输出 / 长期沉淀 | 个人 IP / 创作者 | 短期 ROI 看不到 |

**核心分歧 1**: **单点爆款 vs 矩阵 + 数据迭代** — 决定品牌投预算路径
**核心分歧 2**: **内容驱动 vs 数据驱动** — 决定团队配置
**核心分歧 3**: **小红书闭环 vs 跨平台联动** — 决定运营复杂度

## 可信度判断 (诚实边界)

1. **「头部 MCN 操盘手」公开发声有立场偏见** — MCN 自身是矩阵派受益者, 不会主动唱衰矩阵
2. **「平台官方」keynote 是战略宣示, 不等于落地执行** — KFS / IDEA / AIPS 模型实际效果跟品牌情况强相关
3. **「头部博主」个案具有偶然性** — 程十安停更不代表所有头部博主路径
4. **类目差异极大** — 美妆 / 母婴种草强逻辑, 工具类 / B2B 完全不同
5. **2026-2027 政策窗口高度不确定** — 商业化加速 vs 社区底线张力还在演化
