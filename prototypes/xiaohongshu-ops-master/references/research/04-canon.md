## A. 必读书 (12 本)

### 平台战略 / 创业类

#### 1. 《种草》 (虎嗅 + 36氪 长篇报道合集)
- **作者**: 多位记者 + 36氪深度
- **核心**: 小红书从「站外种草 → 站内闭环」的演化史 + 「真实分享 vs 商业化」张力主线
- **何时读**: 入行 / 想理解平台战略
- **用法**: 看完能理解平台政策为什么钟摆 — 商业化加速 → 社区底线收紧 → 再加速

#### 2. 《小红书营销手册》 (业内主流教材)
- **作者**: 多位代理 / 操盘手合著
- **核心**: KFS / IDEA / AIPS 等官方方法论拆解 + 实战案例
- **何时读**: 操盘手入门期 (1-3 年)
- **用法**: 当工具书查 — 哪个方法论适合品牌当前阶段

#### 3. 《种草经济学》 (社科类)
- **核心**: 种草心理学 + 用户消费决策路径 + 内容驱动消费
- **何时读**: 任何阶段, 理解用户心智
- **用法**: 看完会更尊重「内容真实」, 不会想走捷径

### 内容创作类

#### 4. 《爆款笔记如何写》 (实战合集 / 多版本)
- **核心**: 标题 / 封面 / 正文 / 互动设计 4 件套
- **何时读**: 内容创作者入门
- **用法**: 当 checklist 用 — 每条笔记发布前过一遍

#### 5. 《人间真实》 (内容创作哲学)
- **核心**: 真实分享的力量 + 反对伪素人 / 反对硬广
- **何时读**: 任何阶段, 校准内容方向
- **用法**: 想偷懒走捷径时翻一遍, 校准回真实

#### 6. 《故事公式》 (西方经典, 移植到小红书)
- **作者**: 各家创意 / 编剧
- **核心**: 故事结构 (起承转合 / 痛点 → 转折 → 解决) — 笔记本质是 60 秒短故事
- **何时读**: 内容创作者进阶期
- **用法**: 没思路时回到基础结构

### 投放 / 数据类

#### 7. 《数据驱动的小红书投放》 (代理 / 操盘手手册)
- **核心**: KFS 三件套 + AIPS 人群资产 + 数据归因
- **何时读**: 投放操盘手入行
- **用法**: 工具书 + 模型库

#### 8. 《信息流投放方法论》 (跨平台通用 / 巨量引擎为主)
- **核心**: 出价 / 创意 / 人群三件套. 抖音巨量引擎逻辑可类比小红书聚光.
- **何时读**: 投放进阶期
- **用法**: 跨平台对照看, 看出小红书跟抖音的不同

### 平台规则 / 合规类

#### 9. 《小红书社区规范》 (官方文档, 必看)
- **作者**: 小红书官方
- **核心**: 内容规范 / 商业内容标准 / 违规处罚
- **何时读**: 入行第一周必读
- **用法**: 不读这个等于不知道平台底线在哪

#### 10. 《蒲公英平台规则手册》 (官方)
- **作者**: 小红书蒲公英官方
- **核心**: 商单流程 / 信用等级 / 抽佣 / 违规处理
- **何时读**: 接商单前必读
- **用法**: 不读这个接商单等于走钢丝

### 行业 / 宏观类

#### 11. 《互联网平台经济学》 (学术 / 政策)
- **核心**: 平台经济的供给侧 / 需求侧 / 网络效应 / 双边市场
- **何时读**: 想做长期 / 想做策略层
- **用法**: 看完能更宏观判断小红书未来 5-10 年方向

#### 12. 《新消费品牌操盘手册》 (新消费创业实战)
- **核心**: 完美日记 / 钟薛高 / 元气森林 / 蕉下等新消费品牌的小红书打法
- **何时读**: 跨界做品牌 / 操盘手必读
- **用法**: 案例库 + 反面教材库 (这些品牌的成功 + 翻车都有大量复盘)

## C. 50 个核心概念

### 平台机制 (15)

1. **CES 评分**: 点赞×1+收藏×1+评论×4+转发×4+关注×8 (推荐权重模型)
2. **流量池**: 0-200 (基础) → 200-500 (二级) → 大流量池 (爆款)
3. **完读率**: 视频 / 图文笔记看完比例 (2025 新增隐性指标)
4. **互动率**: (点赞+收藏+评论)/曝光 — 类目均值 1.5x 以上算「好」
5. **收藏率**: 收藏/曝光 — 高于点赞率 = 内容有用
6. **限流**: 推荐量被算法压低 (违规 / 矩阵识别 / AI 内容 / 不报备商单等)
7. **报备**: 商单笔记必须通过蒲公英平台报备 (合规标识)
8. **千粉门槛**: 2022 起千粉素人允许接单
9. **信用等级**: Lv0~Lv3+ (蒲公英内创作者评级)
10. **品牌专业号**: 企业认证账号 (¥ 600/年, 后台 + 数据 + 千帆工具)
11. **矩阵号**: 多账号同步运营 (合规边界: 不同手机号 / 设备 / IP)
12. **KOC**: 千粉-10 万粉, 偏「素人感」博主
13. **KOL**: 10 万-1000 万粉, 头部博主 (内有头部 ≥ 50 万 / 中腰部 ≤ 50 万分层)
14. **薯条 (Boost)**: 自助投放工具, ¥ 75 起投
15. **聚光 (Spotlight)**: 专业级投放, ¥ 5000+ 起投

### 营销方法论 (15)

16. **KFS**: KOL × Feeds × Search (官方推荐三件套)
17. **IDEA**: 机会洞察 / 目标设定 / 内容投放 / 效果度量 (官方早期方法论)
18. **AIPS**: 认知 / 兴趣 / 深度兴趣 / 购买 / 分享 (人群资产模型)
19. **种草直达**: 笔记直接跳第三方电商 (天猫 / 京东 / 美团等)
20. **种草联盟**: 跟阿里 / 京东 / 唯品会等的数据合作 (小红星 / 小红盟 / 小红链)
21. **第一方数据**: 头部品牌 (2100+) 的私域数据接入小红书人群定向
22. **B+ KFS**: B 端 (品牌专业号) 加 KFS 的进阶版 (2024 年提出)
23. **闭环**: 站内成交 (vs 站外种草 → 站外成交)
24. **冷启动**: 新号前 30 天的关键期
25. **爆款复刻**: 数据异常笔记 48 小时内复刻 2-3 条
26. **选题反推**: 从用户搜索词 / 痛点反推内容方向
27. **软植入**: 品牌信息植入比例 ≤ 30%, 真实使用占 ≥ 80%
28. **标准化 brief**: 卖点 / 禁忌词 / 投放预期 — 商单 brief 三要素
29. **内容摸索期**: KFS 阶段 1, K:F:S = 7:3:0
30. **规模化期**: KFS 阶段 3, K:F:S = 4:4:2

### 数据指标 (10)

31. **CTR (点击率)**: 曝光转点击, 8% 以上算「好封面」
32. **CPC (单次点击价)**: 信息流投放成本指标
33. **CPM (千次曝光价)**: 信息流投放成本指标
34. **ROAS**: 投放回报 (跨平台需归因)
35. **互动健康度**: 评论 / 转发占互动比例 (高 = 真实流量)
36. **粉丝画像**: 性别 / 年龄 / 地域 / 兴趣 / 消费力
37. **内容种草力**: 蒲公英 2025 新增指标 (取代单看粉丝量)
38. **类目均值**: 千瓜 / 新红行业 benchmark
39. **GMV 归因**: 跨平台后链路成交归到种草内容
40. **AIPS 转化率**: A → I → TI → P → S 五层转化率

### 运营管理 (10)

41. **矩阵管理**: 5-30 个号统一运营 (千帆 / 第三方工具)
42. **私域承接**: 公众号 / 微信社群承接小红书人群
43. **跨平台联动**: 小红书种草 + 抖音 / 视频号 / 天猫闭环
44. **活人感**: 2025 平台关键词, 真实人格 / 真实经历内容 (反伪素人)
45. **后链路 ROI**: 站外成交归到种草内容 (跟代理 / 平台谈)
46. **KOL 议价**: 头部 ≥ ¥ 5w/篇, 中腰部 ¥ 5k-3w/篇, KOC ¥ 200-3k/篇
47. **类目渗透**: 美妆 / 母婴 / 食品 / 服饰 — 渗透率最高的 4 个
48. **内容差距评估**: 跟头部对标差距 (选题 / 封面 / 文案 / 视频质感)
49. **流量池判断**: 笔记当前在哪个层 (基础 / 二级 / 大池)
50. **投流性价比**: 类目 CPC / CPM benchmark + 自家达标率
