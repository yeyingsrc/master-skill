## 核心概念 (15 个)

### 平台 + 内容
- **笔记**: 小红书内容载体 (行业内说「笔记」, 不说「帖子」 — 说「帖子」是外行破绽)
- **薯条**: 笔记加热 / 投流入口
- **信息流投放**: 大盘投流入口, 比薯条专业 + 复杂
- **蒲公英**: 商单官方报备平台
- **报备 / 报白**: 报备 = 商单走蒲公英. 报白 = 平台对账号特殊类目准入

### 角色
- **KOC** (Key Opinion Consumer): 中小博主, 真实分享感强
- **KOL** (Key Opinion Leader): 头部博主, 影响力广
- **矩阵号 / 品牌专业号**: 矩阵号 = 多账号同步发. 品牌专业号 = 平台官方品牌账号体系

### 流量
- **流量池**: 笔记发布后进入的初始流量层级
- **完读率 / 互动率 / 收藏率**: 衡量笔记质量的核心指标. **收藏率 > 点赞率** 是平台最看重的「内容有用」信号

### 商业化
- **跑流量 / 起量 / 抢流量**
- **黑科技**: 灰色 / 非合规手法 (买赞 / 刷量 / 假评论). 平台严打
- **测评**: 找真实买家给好评 / 试用. 平台规则边缘
- **软广 vs 硬广**: 软广 = 自然种草感. 硬广 = 直接卖货话术. 硬广平台限流严
- **流量分发逻辑**: 小红书偏「内容质量 + 账号垂直度」, 跟抖音的「快速完播」逻辑不同
