# Track 02 — Tools (小红书运营)

> 调研日期: 2026-05-02。中文圈内容运营工具栈。

## 元数据
- 必备 4 / 场景特化 5 / 新兴 4

---

## 必备层

### 1. 千瓜数据
- **One-liner**: 中文圈头部小红书数据 SaaS — 选题 / 对标 / 投放数据。
- **Tier**: 必备
- **License**: SaaS, ¥299-3000/月
- **典型场景**: 选题反推 / 头部对标账号拆解 / 行业 benchmark / 投放数据复盘
- **避坑**: 数据深度高但价格门槛也高, 个人博主用免费版即可

### 2. 灰豚数据
- **One-liner**: 千瓜直接竞品, 数据维度近似, 价格略低。
- **Tier**: 必备 (替代品)
- **License**: ¥199-2000/月
- **典型场景**: 千瓜替代方案 / 双数据源对照

### 3. 蒲公英 (官方报备平台)
- **One-liner**: 商单笔记必经的官方报备入口。
- **Tier**: 必备 (做商单必装)
- **License**: 平台抽佣 + 部分服务费
- **典型场景**: 商单笔记发布 + 数据后台
- **避坑**: 不报备发商单 → 限流风险极高

### 4. 小红书品牌专业号后台
- **One-liner**: 品牌账号官方运营后台。
- **Tier**: 必备 (品牌方)
- **License**: 免费 (需品牌资质 / 营业执照)
- **典型场景**: 品牌专业号自营运营 / 数据看板

---

## 场景特化层

### 5. 蝉小红 / 蝉妈妈
- **One-liner**: 小红书 + 抖音跨平台投放数据洞察。
- **Tier**: 场景特化 (跨平台运营)
- **License**: ¥399-3000/月
- **典型场景**: 信息流投放数据复盘 + 跨平台对比

### 6. 新红 / 新榜小红书
- **One-liner**: 行业排行 + 热点选题工具。
- **Tier**: 场景特化
- **License**: ¥299-1500/月
- **典型场景**: 实时热点跟踪 + 行业账号排行

### 7. 巨量数字 (信息流投放后台)
- **One-liner**: 字节系投流后台。
- **Tier**: 场景特化 (大盘投流)
- **License**: 后台免费 + 投放预算
- **典型场景**: 大盘信息流投放运营

### 8. 飞瓜小红
- **One-liner**: 小红书博主 / 笔记 / 商业数据综合工具。
- **Tier**: 场景特化
- **License**: ¥299-1800/月

### 9. ERP / 矩阵管理工具 (微小宝 / 蝉小红矩阵版)
- **典型场景**: 多账号矩阵 (10+ 账号) 内容统一管理

## 新兴层

### 10. AI 笔记生成器
- **One-liner**: ChatGPT 包装 + 小红书爆款标题模板。
- **Status**: 2024-2025 起势, 风险高
- **避坑**: 平台已能识别 AI 内容, 必须人工编辑 + 加真实素材

### 11. 智能投放工具
- **One-liner**: 蒲公英 + 第三方智能投放联动。
- **Status**: 2025-2026 起势

### 12. 视频号 / 直播工具配套
- **One-liner**: 小红书直播运营工具。
- **Status**: 平台直播能力还在演化中

### 13. 数字人内容工具
- **One-liner**: AI 数字人 + 自动化内容生产。
- **Status**: 平台对数字人内容限流加严

## 选型决策树

| 角色 | 必备工具 |
|------|---------|
| 个人博主 (粉丝 < 1w) | 平台后台 + 千瓜免费版 + 蒲公英 |
| MCN 矩阵账号 | 千瓜专业版 + 蝉小红 + 蒲公英 + ERP |
| 品牌方自营 | 品牌专业号 + 千瓜 + 蒲公英 + 信息流后台 |
| 投放代理 | 蝉小红 + 巨量数字 + 千瓜专业版 |

---

## 避坑清单

1. ❌ 商单不报备 (蒲公英) — 限流风险极高
2. ❌ 多账号矩阵不用 ERP 拼 Excel — 数据混乱
3. ❌ AI 笔记直发 — 限流概率上升
4. ❌ 只看千瓜不看灰豚 — 数据偏差
5. ❌ 投流不看 3s 完播率, 只看曝光 — 跑不出量
