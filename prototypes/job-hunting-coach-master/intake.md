## 2. 核心子领域候选(Phase 1 调研维度)

1. **求职定位与职业叙事** — 目标岗位 / 行业的选择与收敛、个人定位与卖点、职业故事线、personal branding。求职的第一性问题,定位错了后面全白做
2. **简历 / Resume / CV** — 简历写作 / 量化成就 / ATS 关键词与机器可读性 / 国内简历(含照片年龄婚育)vs 海外匿名化简历 的格式分野 / cover letter
3. **求职渠道与投递策略** — 招聘网站直投 / 公司官网 / 内推 / 猎头 / 校招渠道 / hidden job market;『海投』vs『精准投』之争
4. **职场社交与内推** — networking / 信息面谈 / LinkedIn 与脉脉运营 / 如何要到内推 / 弱关系的力量(海外权重高,国内校招相对低)
5. **行为面试** — STAR / 故事化表达 / 高频行为题库 / 把经历讲成『证据』而非『流水账』/ 反向提问
6. **专业 / 技术面试** — 技术岗 coding 与系统设计 / 咨询 case / 产品 PM / 金融 / 设计作品集;以及教练如何处理自己不精通的专业领域(连接资源 vs 不越界)
7. **面试心理与表达** — 面试焦虑 / 沟通与表达 / 视频与远程面试 / 模拟面试与复盘 / 临场应变
8. **谈薪与 offer 决策** — 薪资信息搜集 / 谈薪时机与话术 / 多 offer 与竞争性 offer / offer 横向对比与决策 / 背调环节
9. **校招 / 应届生求职** — 国内春招秋招的工业化节奏 / 海外 new grad rolling / 实习转正 / 无经验如何找第一份工作
10. **转行 / 跨行业求职** — 可迁移技能 / 转行叙事 / 解释空窗与频繁跳槽 / 从 0 进入新行业的路径
11. **特殊群体求职** — 海归 / 国内 35 岁+ / 职场妈妈回归 / 裁员后求职 / 国际学生与签证 / gap year(在结构性约束下最大化)
12. **求职教练的执业经营** — 定价模式(按次 / 套餐 / 训练营 /『包 offer』争议)/ 1v1 vs 团体 / 获客与口碑 / 与猎头和培训机构的区别和利益关系 / 教练的能力边界

## 6. 警示与边界(写入 SKILL.md 诚实边界)

1. **『包 offer / 内推保过』是这个行业最大的雷**。国内大量『包 offer』『内推保过』是营销话术甚至黑产(买内推 / 代写造假 / 面试作弊代面)。教练能提升命中率,不能保证 offer;代写造假 / 面试作弊是红线,会害了求职者(背调暴露、试用期崩盘、职业污点)。
2. **简历可以优化,不能造假**。ATS 关键词、量化成就、措辞提升是合法且应该做的;编造经历 / 改头衔 / 虚构数字是造假,背调和试用期会暴露。SKILL.md 要把『优化』和『造假』的界线划清楚。
3. **求职结果受市场周期强烈影响,不全是技巧问题**。2023-2025 全球科技业裁员 + 国内就业承压 + 应届生规模创新高。教练优化的是『可控部分』,但宏观周期、行业冷热、年龄学历门槛是真实约束。不能把『找不到工作』全归因于个人 —— 那是 victim-blaming 也是不诚实。
4. **国内外求职体系差异极大,必须按 locale 分章**。简历格式(国内带照片年龄婚育 vs 海外匿名化反歧视)、招聘渠道(BOSS 直聘聊天式 vs LinkedIn / recruiter)、校招机制、谈薪文化都不同。不能把一套套到另一套。
5. **面试『押题 / 背题库』短期有用但有上限且有风险**。尤其技术面,背题撑过面试但能力不匹配,试用期会崩。要区分『呈现优化』(合理)和『能力造假』(危险)。
6. **网申 vs networking 之争没有唯一答案**。取决于行业、层级、地区、目标公司。海外科技 / 咨询 networking 权重高;国内校招『网申 + 笔试』仍是主通道。SKILL.md 给『什么场景用什么』的判断框架,不站队。
7. **就业歧视是真实的结构性问题,不是『优化简历』能解决的**。年龄(国内 35 岁)、性别婚育、第一学历、户籍、gap、海归 vs 本土。教练可以教『约束下最大化』,但不能假装约束不存在,也不能把全部责任压给求职者。
8. **求职教练不是猎头也不是 HR,利益关系要讲清**。猎头由用人方付费、HR 代表公司、求职教练(理论上)站求职者一边。但『教练 + 卖内推 + 抽成 / 与机构绑定』的混合模式有利益冲突,要标注。
9. **『35 岁』『第一学历』这类国内职场焦虑被部分培训机构刻意放大变现**。制造焦虑→卖课。SKILL.md 给真实图景(约束确实存在但程度因行业 / 岗位而异),不贩卖焦虑也不粉饰。
10. **AI 正在快速改变求职两端**。求职者用 AI 改简历 / 模拟面,招聘方用 AI 筛简历 / AI 初面。2024-2025 变化快,任何『具体工具 / 话术 / 模板』保鲜期短;工具与战术章节必须标时效,教『原理』而非只是『当下话术』。
