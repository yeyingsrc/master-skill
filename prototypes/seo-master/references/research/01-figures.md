## A. 平台官方代表 (战略级最高一手)

### 1. Google Search Central / John Mueller / Gary Illyes / Danny Sullivan

- **核心一句话**: People-first content + E-E-A-T 是 Google 长期主线 — 不要为搜索引擎写, 为用户写.
- **代表事件**:
  - 2022-08 Helpful Content Update 上线
  - 2023-12 「Experience」加入 EEAT (从 E-A-T → E-E-A-T)
  - 2024 Helpful Content System 并入核心算法
  - 2024 AI Overviews (前身 SGE) 推全球
  - 2025-08 大型 Helpful Content + 反 AI 内容更新
- **代表观点**:
  1. People-first content > Search engine first content
  2. E-E-A-T (Experience / Expertise / Authoritativeness / Trustworthiness) 中 Trust 最重要
  3. Helpful Content 是 site-wide signal — 几页垃圾页拖累整站
  4. AI 生成内容不必然差, 关键是 helpful + originality + E-E-A-T
- **争议**: AI Overviews 让 26% 用户读完不点击, 把流量留在 Google
- **来源**: [Primary] Google Search Central 官方文档 + Twitter (John Mueller / Gary Illyes / Danny Sullivan)

### 2. Bing / Microsoft + Microsoft Copilot

- **核心一句话**: AI Search (Copilot) 是 Bing 反超 Google 的杠杆 — 但 SEO 底层还是内容质量.
- **代表内容**: Bing Webmaster Guidelines + Microsoft Copilot 集成
- **来源**: [Primary] Bing Webmaster Tools 官方

## B. 西方 SEO 头部 (思想 + 工具体系)

### 3. Brian Dean (Backlinko 创始人, 2022 卖给 Semrush)

- **核心一句话**: SEO 是「让 Google 找到你的最佳答案」. 不是技术游戏, 是内容 + 链接 + 用户体验.
- **代表内容**:
  - Backlinko 博客 (SEO 头部博客之一)
  - Skyscraper Technique (摩天楼技术) — 找最强对手内容 → 做更好版本 → 抢链接
  - Exploding Topics (新兴趋势工具)
  - 2022 Backlinko 卖给 Semrush, 但博客继续运营
- **代表观点**:
  1. 长内容 > 短内容 (1500+ 词为基线)
  2. Pillar + Cluster 内容架构 (主题枢纽 + 长尾分支)
  3. 链接质量 > 链接数量
  4. SEO 投入 6-12 月才看到效果
- **净资产**: $23-25M (Backlinko 出售 + Exploding Topics + 咨询业务)
- **来源**: [Primary] Backlinko.com + Brian Dean Twitter

### 4. Neil Patel (Neil Patel Digital / NP Digital 创始人)

- **核心一句话**: SEO + Content + PPC + Social 全渠道整合是头部品牌增长杠杆.
- **代表内容**:
  - Neil Patel 博客 (个人头部博客)
  - NP Digital (企业级 SEO 咨询机构)
  - Ubersuggest (SEO 工具, 中型品牌实惠版)
  - 多本 NYT 畅销书
- **客户**: Amazon / Google / NBC / Viacom 等
- **代表观点**:
  1. SEO ROI 跟内容质量强相关 — 不投内容只投技术 SEO 没用
  2. Content marketing 跟 SEO 不可分割
  3. 跨渠道协同 (SEO + PPC + Social) 比单渠道压倒性 ROI
- **来源**: [Primary] neilpatel.com + NP Digital 官网

### 5. Rand Fishkin (Moz / SparkToro 创始人)

- **核心一句话**: SEO 不只是 Google — 是「在用户搜索的所有地方被找到」.
- **代表内容**:
  - 2004 创办 Moz (原 SEOmoz) — 行业头部工具 + 教育平台
  - Whiteboard Friday 视频系列 (SEO 教育里程碑)
  - 2018 离开 Moz, 创办 SparkToro (受众洞察工具)
  - 著作《Lost and Founder》 (创业者必读)
- **代表观点**:
  1. Domain Authority (Moz 指标) 是 SEO 行业事实标准
  2. SEO 视野要扩大到「所有用户搜索地方」 (Google + YouTube + TikTok + ChatGPT)
  3. 「Zero-click search」是 SEO 新现实 — 用户在搜索引擎里就拿到答案
- **来源**: [Primary] Moz.com + SparkToro.com + Rand Fishkin Twitter

### 6. Aleyda Solis (Orainti 创始人 + SEOFOMO)

## C. AI Search 新流派 (2024-2026 兴起)

### 9. AI Search / GEO (Generative Engine Optimization) 派代表

- **核心一句话**: 传统 SEO「rank and click」转向「visibility and citations」 — 在 AI Overview / ChatGPT / Perplexity 被引用比传统排名重要.
- **代表数据**:
  - 30% 美国桌面关键词触发 AI Overviews (2025-09)
  - 26% 用户读完 AI Overview 不点击
  - AI Overviews 平均链接 13.3 个源
  - 76.1% 被引用 URL 也排 Google 前 10
- **代表观点**:
  1. Generative Engine Optimization (GEO) / LLM Optimization (LLMO) 是新学科
  2. 公司用 GEO 6-12 月看到 300-500% ROI
  3. 内容优化 GEO 后 AI search 可见度提升 30-40%
- **来源**: [Primary] Position.Digital / Search Atlas / SEO.com 行业报告

### 10. AI SEO 工具厂商代表 (Surfer SEO / Frase / Clearscope / Jasper)

- **核心一句话**: AI 内容辅助工具是中型品牌效率倍增器, 不是替代品.
- **代表工具**:
  - Surfer SEO (内容评分 + AI 大纲)
  - Frase (问答 SEO + AI 内容)
  - Clearscope (语义关键词)
  - Jasper / ChatGPT (通用 LLM)
- **来源**: [Primary] 各工具官网

## 流派分裂 (关键认知分歧)

| 流派 | 主张 | 适合 | 风险 |
|------|------|------|------|
| **Content-driven** (Brian Dean / Neil Patel) | 内容 + 链接 + 用户体验三件套 | 中型品牌 / 内容能力强 | 短期 ROI 看不到 |
| **Technical SEO** (Aleyda Solis / Cyrus Shepard) | Core Web Vitals / Mobile / Schema / 国际化 | 大站 / 跨语言 / 复杂架构 | 单纯技术不带流量 |
| **Authority / E-E-A-T** (Marie Haynes / Lily Ray) | 作者权威 / 信任信号 / 真实经验 | YMYL 类目 (健康 / 金融) | 起量慢, 适合长期品牌 |
| **AI Search / GEO** (新流派 2024-2026) | 在 AI Overview / ChatGPT / Perplexity 被引用 | 头部品牌 / 信息类内容 | 工具方法论快速演化 |
| **Black-hat / Spam** (淘汰中) | 链接农场 / 内容农场 / Spam | 短期暴利 (淘汰中) | 一更新就团灭 |

**核心分歧 1**: **Content vs Technical** — 决定团队配置
**核心分歧 2**: **Traditional Rank vs AI Citation** — 决定 2026+ 策略
**核心分歧 3**: **White-hat vs Black-hat** — 决定长期可持续性

## 可信度判断 (诚实边界)

1. **「Google 官方」声明是政策宣示, 实际算法变化更快**. John Mueller / Gary Illyes / Danny Sullivan 公开发声偏「人话」, 实际更新有时跟公开口径有差距.

2. **「头部 SEO 专家」公开发声有立场偏见**. Brian Dean / Neil Patel 卖工具 / 课程, 不会主动唱衰 SEO.

3. **AI Search / GEO 是 2024-2026 新学科**, 工具方法论 6-12 月衰减, 跟最快.

4. **中文 SEO 跟西方 SEO 生态差异极大**. 百度算法跟 Google 不同, ZAC / 卢松松 的中文方法论不能直接套到 Google.

5. **2025-2026 政策窗口高度不确定**. AI Overviews / SearchGPT / Perplexity 让传统 SEO 流量结构发生巨变.
