## 高频术语

### 排名信号
- **E-E-A-T**: Experience / Expertise / Authoritativeness / Trustworthiness
- **YMYL**: Your Money or Your Life
- **Core Web Vitals**: LCP / FID / CLS / INP
- **Mobile-first indexing**: 移动端优先索引
- **HTTPS**: 安全协议
- **Helpful Content**: Google 强调的内容理念

### SERP 元素
- **SERP**: Search Engine Results Page
- **Featured Snippet**: 摘要框 (位置 0)
- **PAA (People Also Ask)**: 相关问题框
- **Knowledge Panel**: 实体知识框
- **Local Pack**: 本地搜索三结果框
- **AI Overviews / SGE**: AI 生成的 SERP 摘要 (2024 起势)

### 关键词 / 内容
- **Long-tail / Head term**: 长尾词 / 头部词
- **Search Intent**: 信息 / 导航 / 商业调研 / 交易
- **Topic Cluster / Pillar**: 主题集群 / 支柱页
- **TF-IDF**: 主题相关性
- **Content gap**: 内容空白机会

### 技术 SEO
- **Crawl Budget**: 抓取预算
- **robots.txt**: 抓取规则
- **sitemap.xml**: 站点地图
- **canonical**: 规范 URL
- **hreflang**: 多语言标签
- **noindex / nofollow**: 索引 / 链接控制
- **redirect (301 / 302 / 308)**: 重定向类型
- **Schema markup / Structured data**: 结构化数据
- **JSON-LD**: schema 实现方式

### 外链
- **Backlink / Inbound link**: 反链
- **DR / DA / UR**: 域名权重指标
- **Referring domain**: 引荐域名
- **Anchor text**: 锚文本
- **PBN**: Private Blog Network (黑帽)
- **Disavow**: 拒绝外链

### 监控
- **Manual Action**: Google 人工处罚
- **Algorithmic penalty**: 算法处罚
- **Index coverage**: Search Console 索引覆盖

## 重大算法更新 (近 5 年)

| 时间 | 名称 | 重点 |
|------|------|------|
| 2019-09 | BERT Update | 语义理解 |
| 2021-06 | Page Experience Update | Core Web Vitals 加入排名 |
| 2022-08 | Helpful Content Update | 内容质量打击 (尤其 AI 滥用) |
| 2023-04 | Reviews Update | 产品评测内容质量 |
| 2023-09 | Core Update | 多领域排名洗牌 |
| 2024-03 | Core Update + Spam Update | YMYL E-E-A-T 强化 |
| 2024-05 | AI Overviews 上线 | SERP 大变 |
| 2025-Q1 | Core Update | 持续 E-E-A-T 信号强化 |

## 行业近期更新 (近 12 个月)

| 时间 | 事件 | 影响 |
|------|------|------|

## 标准 / 协议

### 1. WCAG (Web Content Accessibility)
- **范围**: 全球
- **要求**: 可访问性, 部分国家 (US / EU) 法律要求

### 2. GDPR / CCPA (隐私)
- **范围**: 欧盟 / 加州
- **要求**: Cookie consent / 用户数据控制

### 3. Schema.org
- **范围**: 全球
- **要求**: 结构化数据标准

### 4. PageSpeed Insights / Lighthouse
- **范围**: Google 官方测速
- **要求**: Core Web Vitals 监控基础
