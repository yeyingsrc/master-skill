## A. 必备工具栈 (任何 SEO 项目最小集)

### 1. Google Search Console (官方, 免费)

- **类型**: 官方
- **核心定位**: 站点跟 Google 的官方对接 — 索引 / 性能 / 错误 / 链接
- **价格**: 免费
- **代表功能**:
  - 性能报告 (展示量 / 点击 / 平均位置 / CTR)
  - 索引状态 (哪些页被索引)
  - 站点地图提交
  - 移动友好 / Core Web Vitals 报告
  - 链接报告 (谁链向你)
  - 手动操作 / 安全问题通知
- **必备程度**: ⭐⭐⭐⭐⭐ (任何站长必装, 不可替代)
- **来源**: [Primary] search.google.com/search-console

### 2. Google Analytics 4 (官方, 免费)

- **类型**: 官方
- **核心定位**: 站点流量分析 — 用户行为 / 转化 / 渠道
- **价格**: 免费 (GA4) + 企业版 GA360 ($150k+/年)
- **代表功能**:
  - 实时 / 历史流量
  - 渠道归因 (organic / paid / social / direct)
  - 转化跟踪
  - 用户路径 / 用户分群
- **必备程度**: ⭐⭐⭐⭐⭐
- **来源**: [Primary] analytics.google.com

### 3. Ahrefs (第三方头部)

- **类型**: 第三方
- **核心定位**: 链接分析头部 + 关键词研究 + 竞品监控
- **价格**: $129+/月 (Lite) → $499+/月 (Advanced) → 企业定制
- **代表功能**:
  - Site Explorer (链接 / 关键词 / 流量)
  - Keywords Explorer (关键词难度 / 流量 / 长尾)
  - Content Explorer (内容主题挖掘)
  - Site Audit (技术 SEO 审计)
  - Rank Tracker
- **链接库**: 35T 链接 + 96 次/天爬取 (15 分钟一次)
- **优势**: 链接数据最全 (比 SEMrush 强 35%, 比 Moz 强 68%)
- **缺点**: 2025 改为 credit-based 计费, 大客户成本暴涨
- **来源**: [Primary] ahrefs.com

### 4. SEMrush (第三方, 综合营销套件)

- **类型**: 第三方
- **核心定位**: SEO + PPC + Social + Content 全渠道
- **价格**: Pro $139.95/月 / Guru $249.95/月 / Business $499.95/月
- **代表功能**:
  - 关键词研究 (25B 关键词数据库, 行业最大)
  - 竞品分析 (organic + paid)
  - Site Audit + Position Tracking
  - 内容营销工具 + AI 助手
  - PPC 工具 (Google / Bing Ads)
- **关键词数据**: 25B 关键词 (业内最大), 89% Google 相关性
- **优势**: 综合性最强, 适合营销团队
- **来源**: [Primary] semrush.com

### 5. Moz Pro (第三方, 教育向)

- **类型**: 第三方
- **核心定位**: SEO 入门工具 + 教育平台
- **价格**: $49/月 (Standard, 年付) → $99/月 (Medium) → $179/月 (Large) → $299/月 (Premium)
- **代表功能**:
  - Domain Authority (DA) + Page Authority (PA) — 行业事实标准
  - Keyword Explorer + Rank Tracker
  - Site Crawl + On-page Grader
  - Spam Score
  - 30 天免费试用 (Ahrefs / SEMrush 没有)
- **优势**: 入门友好, DA 是事实标准
- **来源**: [Primary] moz.com

## 工具栈坑点 (诚实边界)

1. **第三方数据 ≠ Google 真实数据**. Ahrefs / SEMrush / Moz 都是反推, GSC 才是 Google 给你的真实数据.

2. **不要被工具绑架**. 工具是辅助, SEO 核心还是内容 + 链接 + 用户体验.

3. **Ahrefs 2025 改 credit 计费**, 大客户成本可能暴涨, 评估时算清楚.

4. **AI 内容生成工具被算法打压**. Jasper / Copy.ai 直发会被 Helpful Content Update 限流, 必须人工编辑 + E-E-A-T.

5. **Domain Authority (DA) 不是 Google 指标**. Moz 的 DA 是行业惯用, 不等于 Google PageRank.

6. **跨语言 SEO 工具差异大**. 中文 SEO (百度) 跟西方工具 (Ahrefs / SEMrush) 数据池不同, 跨境品牌需要双轨.

7. **AI Search 监控工具是新兴学科**. Otterly / Profound / Peec AI 都是 2024-2025 才有, 方法论还在迭代.
