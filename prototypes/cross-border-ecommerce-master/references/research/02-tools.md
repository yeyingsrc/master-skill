## 必备层 (≥ 80% 严肃从业者用)

### 1. Helium 10 — 工具集事实标准
- **One-liner**: 亚马逊运营全栈工具 — 选品 / 关键词 / 跟卖警报 / PPC 自动化 / Walmart 与 TikTok Shop 部分支持
- **License**: SaaS, 月费分级 (年付有 17% 折扣):
  - **Beginner**: $33.25/月 (年付) / $39/月 (月付)
  - **Platinum**: $99/月 — 大部分中小卖家用这一档
  - **Diamond**: $279/月 — 4 个店铺, 多用户, 库存管理
- **Maturity**: 7+ 年, 工具厂商行业头部地位稳固
- **2025-2026 关键功能**:
  - Cerebro (反向 ASIN) — 关键词竞品反查
  - Adtomic — PPC 自动化
  - Frankenstein / Scribbles — listing 高阶撰写
  - Refund Genie — FBA 退款追溯
  - Walmart + TikTok Shop 多平台支持
- **典型场景**: 跨境新品选品 (Black Box) / 关键词矩阵 / listing 优化 / 反查竞品
- **优劣** (基于 Austinplease / EcomCrew 2025 评测):
  - 优 — 关键词 / SEO 工具栈最强 / 数据广 / 多平台支持
  - 劣 — 学习曲线陡 / 中小卖家用不到 1/3 功能
- **避坑**: 中小卖家先用 Beginner 或 Platinum, 不要直接上 Diamond
- **来源**: [Primary] [Helium 10 vs Jungle Scout 2025 — Austinplease](https://www.austinplease.com/blog/helium-10-vs-jungle-scout) / [Helium 10 官网](https://www.helium10.com/) / [Helium 10 vs Jungle Scout — EcomCrew 2026](https://www.ecomcrew.com/helium10-vs-junglescout/)

### 2. Jungle Scout — 选品工具老牌
- **One-liner**: Helium 10 最直接竞品, 选品 + 销量预估行业标杆
- **License**: SaaS, $29/月 (基础) - $49/月 (高级), 年付节省 50%
- **Maturity**: 7+ 年, 行业地位稳固
- **2025-2026 关键功能**:
  - Supplier Database — 私自标签直接对接工厂 (Helium 10 没这个)
  - Million Dollar Case Study Season 5 — 持续真实开店纪录
  - Chrome 插件 — 实时看 BSR / 销量
- **数据准确度**: 第三方独立测试约 84-86%, Helium 10 略低 (5-out-of-10 vs 4-out-of-10)
- **典型场景**: 早期选品 + 类目天花板评估 + 工厂寻源
- **优劣**: 优 — 选品 + 工厂对接最强 / 学习曲线友好. 劣 — 关键词 / PPC / 高阶功能弱于 Helium 10
- **决策树**: 入门 / 选品阶段优选 Jungle Scout, 已运营 + 关键词矩阵复杂时优选 Helium 10. 多数大卖两个都用
- **来源**: [Primary] [Million Dollar Case Study Season 5](https://www.junglescout.com/mdcs/season-5/) / [Jungle Scout vs Helium 10 — projectfba 2026](https://projectfba.com/helium-10-vs-jungle-scout/)

### 3. Amazon Seller Central + Brand Analytics — 平台官方后台
- **One-liner**: 任何严肃 Amazon 卖家的底层基础设施
- **License**:
  - Seller Central: 卖家协议 (Professional $39.99/月)
  - Brand Analytics: 免费 (Brand Registry 准入)
- **核心模块**:
  - **Brand Analytics SQP (Search Query Performance)** — 真实搜索词 + 转化漏斗
  - **2025-02-26 起 SQP 通过 SP-API 程序化访问** — 不再依赖手工下载
  - 三种粒度: WEEK / MONTH / QUARTER
  - 品牌级最多 1000 个 query, ASIN 级最多 100 个
  - **Search Query Score** — 内部排名, 反映哪些 query 真正驱动销售 (不是搜索量最大的)
  - Item Comparison / Demographics / Repeat Purchase Behavior
- **典型场景**: 看真实搜索词 + 转化率 + 重复购买行为 (远超第三方反查的精度)
- **避坑**: 必须先做 Brand Registry, 否则 Brand Analytics 锁死
- **2025 大变化**: SQP 全面 API 化, 工具厂商可直接订阅
- **来源**: [Primary] [Brand Analytics — Amazon 官方](https://sell.amazon.com/blog/brand-analytics) / [Search Query Performance 解读 — Kapoq](https://kapoq.com/search-query-performance-report-explained/) / [API Access to SQP Reports — ClearAds](https://clearadsagency.com/api-access-to-search-query-performance-sqp-reports/)

### 4. 卖家精灵 (Sellersprite) — 中文圈头部
- **One-liner**: 中国版 Helium 10, 中文界面 + 国内支付友好
- **License**: SaaS, ¥149-799/月 (中文卖家友好定价)
- **Maturity**: 7+ 年, 中文圈事实标准
- **典型场景**: 关键词调研 + 反向 ASIN + 库存预警 + 差评监控
- **优劣**: 优 — 中文 UI + 国内卖家习惯. 劣 — 数据深度比 Helium 10 略低, 关键决策应交叉验证
- **避坑**: 跟 Helium 10 数据偶有差异 (3-15%), 关键决策双源验证
- **来源**: [Primary] 卖家精灵官网 + 中文跨境社区共识 (海猫跨境 / 跨境老炮 反复推荐)

### 5. Keepa — 历史数据权威
- **One-liner**: 亚马逊历史价格 + BSR 曲线最权威, 几乎所有竞品分析都靠它
- **License**: 19 EUR/月 (Pro)
- **Maturity**: 10+ 年, 行业事实标准
- **典型场景**: 看竞品定价历史 / 销量估算 / 跟卖判断 / 季节性分析
- **避坑**: Free 版本数据严重阉割, 决策必须 Pro
- **来源**: [Primary] keepa.com (中文圈 + 英文圈共识工具)

## 场景特化层

### 6. Pacvue — 企业级 PPC 自动化
- **One-liner**: 服务大品牌 / 大代理的 retail media 平台, Amazon DSP 整合
- **License**: 企业定制 (基于广告支出 / 市场数 / 支持级别), 一般月费 $1000+, 起步价不公开
- **Maturity**: 行业 #1 PPC 工具 (David Zimmerman 等 PPC 专家公开认可)
- **2025-2026 关键功能**:
  - 整合 Amazon Search + DSP + AMC 单一执行层
  - **Pacvue Agent** — 用自然语言生成 AMC SQL 查询, 然后翻回大白话解读
  - "200x faster commerce media workflows" 宣称
  - 100+ retailers 跨平台支持
- **谁用**: 月广告费 ≥ $5000 USD 的卖家 / 代理 / 大品牌
- **决策**: 月广告费 < $5000 USD 用不上, 中小卖家用 Helium 10 Adtomic 或自家 Sellerise
- **来源**: [Primary] [Pacvue 官网](https://pacvue.com/marketplaces/pacvue-for-amazon/) / [Pacvue Agent 200x — PPC.land](https://ppc.land/pacvue-agent-promises-200x-faster-commerce-media-workflows/) / [Pacvue Pricing Guide — Atom11](https://www.atom11.co/blog/pacvue-pricing-guide)

### 7. Sellerise — 中型卖家 PPC + 数据
- **One-liner**: Helium 10 / Pacvue 中间档定位, 偏 PPC 自动化 + Brand Analytics 增强
- **License**: $40-200/月
- **典型场景**: 月销 $30-300k 卖家自动化 PPC + 反向监控对手 listing 变化
- **决策**: Helium 10 PPC 不够强 / 但又不到 Pacvue 量级时用

### 8. Stackline — 多渠道品牌情报
- **One-liner**: 品牌方做亚马逊以外 marketplace (Walmart / Target.com / 自有 DTC) 的数据整合
- **License**: 企业定制
- **典型场景**: 多渠道品牌 (不只亚马逊) 的全景情报

### 9. SellerBoard / FeedbackWhiz / Helium 10 Profits — 财务 + 利润
- **One-liner**: Amazon 后台财务模块太弱, 需第三方算清楚单 SKU 真实利润
- **License**: SellerBoard $19-69/月; FeedbackWhiz $30-200/月
- **典型场景**: 多 SKU 卖家做月度财务复盘 (扣除 FBA / 仓储 / 退货 / 广告)

### 10. ERP (店小秘 / 旺店通 / 马帮 / 通途)
- **One-liner**: 多店多平台订单 / 库存 / 财务统一管理
- **License**: SaaS, ¥3000-30000/年
- **典型场景**: SKU > 30 或店铺数 > 3 时必装
- **避坑**: 切换成本极高 (历史数据迁移痛苦), 选定后 12 月内不要换

### 11. Pingpong / Airwallex / WorldFirst — 跨境收款
- **One-liner**: 跨境收款 + 多币种结算 + 退税路径
- **License**: 按交易额抽佣 / 多币账户费
- **典型场景**: 美金 < $50K/月用 Pingpong; 欧元多用 Airwallex; 大额用万里汇手续费低

### 12. Avalara / hellotax / Avask — VAT + 合规
- **One-liner**: VAT 注册 + 申报 + 欧代服务 + GPSR 应对
- **License**: 各国 VAT 注册一次性几百到几千 EUR + 月度申报 100-500 EUR
- **避坑**: 选服务商先看「能不能在所有目标国注册」, 单国服务商坑多

## 新兴层 (近 12 个月起势, 慎用但要关注)

### 13. AI Listing 工具 (Aitomic / Helium 10 AI / Stackline AI)
- **One-liner**: AI 自动生成 / 优化 listing 文案 + 关键词覆盖
- **License**: $10-100/月
- **典型场景**: 大批量 SKU (≥ 50) 上架的 listing 初稿
- **风险**: 纯 AI 内容遇到 Helpful Content Update 类政策会被限流, **必须人工编辑 + 加真实使用场景细节**
- **2025-2026 趋势**: 工具厂商都在嵌入, Helium 10 / Jungle Scout / Sellerise 都已加 AI 模块

### 14. TikTok Shop 卖家工具 (FastMoss / EchoTik)
- **One-liner**: TikTok Shop 选品 / 达人匹配 / 直播数据
- **License**: $30-300/月
- **典型场景**: 拓展 TikTok Shop 但不知从哪开始的卖家
- **2024-2025 大事件**: TikTok Shop 美国开放, 工具生态快速形成

### 15. Smartscout / DataDive — 类目 + Market share 分析
- **One-liner**: 亚马逊类目结构 + 卖家份额 + 品牌 share 拆解
- **License**: SaaS, $99-500/月
- **典型场景**: 找细分类目机会 + 竞品 share 拆解 (Helium 10 Black Box 替代选项)

### 16. Pacvue Agent / 智能投放托管 (各家 AI Layer)
- **One-liner**: AI Agent 翻译业务问题为 AMC SQL 查询, 然后用自然语言解释
- **Status**: 2025 最热的方向, 还在快速演化
- **风险**: 大账户慎用, 关键决策不能完全托管

### 17. Temu / Shein 全托管运营平台
- **One-liner**: 全托管模式 (ROAS / listing / 物流都平台搞), 卖家只供货
- **Status**: 2024-2026 商业模式趋势, 不是工具但生态变迁
- **典型场景**: 中国供应链卖家想出海但运营能力弱时
- **风险**: 利润空间被平台挤压 + 议价权差

## 避坑清单

1. **不上 ERP 拼 Excel 管 SKU > 50** — 库存乱 + 多平台同步问题, 长期吃血
2. **Helium 10 + Jungle Scout 同时订阅** — 数据有重叠, 入门选 Jungle Scout, 进阶选 Helium 10
3. **AI listing 直接发** — 必须人工加真实 use case + 作者署名
4. **欧洲站不装 VAT / 包装法代理** — 自己处理几乎 100% 出错
5. **TikTok Shop 没工具直接干** — 选品 / 达人匹配靠人工根本撑不起规模
6. **Pacvue 太早上** — 月广告 < $5000 用不到 1/3 功能, 性价比差
7. **Brand Analytics 没用 SQP** — Amazon 内部数据准确度远超第三方反查
8. **Keepa Free 版做关键决策** — 数据严重阉割, 必须 Pro

## 最近 12 月主要变化 (按时间)

- **2024-12**: 欧盟 GPSR 实施, VAT / 包装法代理需求暴涨
- **2025-02**: Amazon Brand Analytics SQP 通过 SP-API 程序化访问, 工具厂商可直接订阅
- **2025-Q2**: Pacvue 推出 Agent (AMC SQL 自然语言生成)
- **2025-Q3**: TikTok Shop 美国半托管转半托管模式, 工具生态成型
- **2025-Q4**: Helium 10 / Jungle Scout / Sellerise 都嵌入 AI listing 模块
- **2026-Q1**: Temu 全托管向北美扩张, 倒逼传统 Amazon 卖家重估利润结构
