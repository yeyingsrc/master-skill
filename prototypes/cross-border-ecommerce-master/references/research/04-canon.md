## 必读书 (≥ 3 个独立来源都点过)

### 1. *The Lean Startup* — Eric Ries
- **领域**: 创业方法论 (跨界经典)
- **跨境电商相关**: MVP + iteration, 直接套到选品 / listing 测试
- **谁推过**: Steve Chou / Greg Mercer / Andrew Youderian / Anthony Lee 反复引用
- **核心概念**: Build-Measure-Learn / Pivot / Vanity vs Actionable Metrics
- **入选理由**: 跨境圈最被引用的非行业书

### 2. *The Personal MBA* — Josh Kaufman
- **领域**: 商业基础
- **跨境电商相关**: 营销 / 销售 / 财务 / 运营全覆盖, 入门跨境必读
- **谁推过**: 美国独立卖家圈 + 中文老炮圈反复推

### 3. *Influence: The Psychology of Persuasion* — Robert Cialdini
- **领域**: 心理学 / 营销
- **跨境电商相关**: listing 文案 / review 策略 / 主图设计的底层心理学
- **谁推过**: Bradley Sutton / Liran Hirschkorn / 中文圈雨果跨境
- **核心**: 6 大说服原则 — 互惠 / 承诺一致 / 社会认同 / 喜好 / 权威 / 稀缺

### 4. *Built to Sell* — John Warrillow
- **领域**: 商业资产化
- **跨境电商相关**: 想做能退出的跨境品牌必读
- **谁推过**: Andrew Youderian / Bill D'Alessandro / Empire Flippers 圈

### 5. *The 4-Hour Workweek* — Tim Ferriss
- **领域**: 自动化 + 外包 + 远程
- **跨境电商相关**: 早期 dropshipping / muse 业务的方法论, 影响一整代美国跨境创业者
- **谁推过**: Steve Chou / 几乎所有 DTC 老兵
- **争议**: 部分内容已过时 (2007 出版)

### 6. *Profit First* — Mike Michalowicz
- **领域**: 现金流管理
- **跨境电商相关**: 「跑得快不如活得久」 — 跨境圈最被引用的财务书
- **谁推过**: Bill D'Alessandro / 群响私董会

### 7. *DotCom Secrets* — Russell Brunson
- **领域**: 漏斗营销 + email list
- **跨境电商相关**: 独立站长期复利的圣经
- **谁推过**: Andrew Youderian / Steve Chou (DTC 路径)
- **争议**: 偏推 ClickFunnels, 但漏斗思维扎实

### 8. *Crossing the Chasm* — Geoffrey Moore
- **领域**: 产品营销
- **跨境电商相关**: 早期采用者 vs 主流市场 — 选品阶段的认知分水岭
- **谁推过**: 阳萌 (Anker) / 多家品牌创始人

### 9. *Trust Me, I'm Lying* — Ryan Holiday
- **领域**: 公关 / 媒体
- **跨境电商相关**: 跨境品牌做 PR 必读
- **谁推过**: Andrew Youderian / Anthony Lee

### 10. *The Million Dollar Case Study* (Jungle Scout 出品)
- **形式**: 多季视频 + 文档 (免费)
- **领域**: 真实开店纪录
- **核心**: 老案例 (2017-2018) 但选品 / listing 框架仍适用
- **现状**: Season 5 在跑 (Lenny Smith 主持 + Greg Mercer / 多位 8 位数卖家嘉宾)

### 11. *Crushing It on Amazon* — Tim Jordan
- **领域**: 亚马逊长尾运营
- **核心**: SKU 矩阵管理 + long-tail 选品的实操方法论
- **谁推过**: Helium 10 expert 圈 / 多家 7-8 位数卖家

### 12. 《亚马逊跨境电商运营宝典》(中文圈)
- **领域**: 中文实操经典
- **核心**: 亚马逊运营全流程 SOP, 数据驱动方法论
- **谁推过**: 中文圈跨境运营圈反复点
- **状态**: 多次再版, 部分内容已过时但框架稳定

## 核心概念 (50 个)

### 平台 + 角色

#### 1. ASIN
Amazon Standard Identification Number, 亚马逊每个商品的唯一 ID. 一个 ASIN 可对应多个 SKU (变体).

#### 2. SKU
Stock Keeping Unit, 卖家自己的库存编码. 独立于 ASIN.

#### 3. BSR (Best Sellers Rank)
类目销量排名. **跨类目不可比** — 不同类目天花板差距大.

#### 4. FBA / FBM / SFP
Fulfilled by Amazon (亚马逊代发) / by Merchant (自发) / Seller Fulfilled Prime (自发但挂 Prime). FBA 流量倾斜 + Buy Box 友好.

#### 5. Buy Box (购物车)
商品页面默认下单按钮. 跟卖时争夺. 90% 销量从 Buy Box 出.

#### 6. Brand Registry / Project Zero
亚马逊品牌备案. 不备案 → A+ Content / Brand Analytics / Sponsored Brands 全锁死.

### 流量与排名

#### 7. A9 / A10 算法
亚马逊搜索排序算法. 对关键词 + 转化率 + 销量综合考量. A10 是业内非官方说法.

#### 8. Search Query Performance (SQP)
Brand Registered 卖家可看的真实搜索词数据. **2025-02 起通过 SP-API 程序化访问**, 工具厂商可直接订阅.

#### 9. Search Funnel
曝光 → 点击 → 加购 → 购买四步漏斗. SQP 显示完整漏斗.

#### 10. Honeymoon Period
新品上线后 14-30 天的算法扶持期. 利用这段期集中投流 + 累积评价是头部共识.

#### 11. Velocity (销量速度)
单位时间内的销量. 算法看「速度增长」, 不是「累积总量」.

#### 12. Conversion Rate (CVR)
listing 流量 → 成交比. 头部卖家 12-20%, 新手 < 5%.

#### 13. Click-through Rate (CTR)
搜索曝光 → 点击比. 主图 + 标题质量信号.

### 投流

#### 14. PPC (Pay Per Click)
Sponsored Products / Brands / Display 三类亚马逊广告.

#### 15. ACOS (Advertising Cost of Sales)
广告花费 / 广告销售额. 越低越好.

#### 16. TACOS (Total ACOS)
广告花费 / 总销售额 (含自然流量贡献). 比 ACOS 更全.

#### 17. ROAS (Return on Ad Spend)
广告销售额 / 广告花费. ACOS 倒数.

#### 18. CPC (Cost per Click)
单次点击成本.

#### 19. DSP (Demand Side Platform)
亚马逊站外程序化展示广告.

#### 20. SP / SB / SD
Sponsored Products / Brands / Display, 三类广告类型.

### 物流

#### 21. Pan-EU FBA
亚马逊欧洲一站式 FBA, 商品在多国轮流补货. 前提是 OSS + 各国 VAT 都搞定.

#### 22. AWD (Amazon Warehousing Distribution)
亚马逊仓配 + 第三方分销, 2024-2025 起普及.

#### 23. MCF (Multi-Channel Fulfillment)
用 FBA 库存发其他渠道订单 (独立站 / Walmart 等).

#### 24. IPI (Inventory Performance Index)
亚马逊库存绩效指数. < 200 限制入仓.

#### 25. EORI / IOR / EOR
欧盟海关注册号 / 进口主体 / 出口主体.

### 财务 / 跨境收款

#### 26. CAC (Customer Acquisition Cost)
获客成本.

#### 27. LTV (Lifetime Value)
客户生命周期价值. DTC 重指标.

#### 28. AOV (Average Order Value)
客单价.

#### 29. Cash Conversion Cycle
现金转换周期. 跨境特别重要 (海运周期长).

### 合规 (尤其欧站)

#### 30. VAT (Value Added Tax)
欧洲增值税. 各国独立 + OSS 一站式可统一申报.

#### 31. OSS (One-Stop Shop)
欧盟内 B2C 跨境一站式申报. 2021-07 启用.

#### 32. IOSS (Import One-Stop Shop)
单笔 < €150 跨境进口包裹简化申报.

#### 33. VerpackG (德国包装法)
2019 起, 卖家或 Amazon 代缴费. 不注册 listing 直接下架.

#### 34. WEEE (Waste Electrical and Electronic Equipment)
电子电器废弃物指令. 各国独立注册.

#### 35. EPR (Extended Producer Responsibility)
生产者责任延伸. 法 / 西 / 德 已实施, 2024-2025 多国扩展强制范围.

#### 36. GPSR (General Product Safety Regulation)
**2024-12-13 实施**. 欧盟内必须有责任人 (Authorized Representative) + listing 显示责任人地址 + 高风险品类更严合规.

#### 37. CE / GS / RoHS / Reach / Battery Regulation
欧盟产品认证组合. CE 强制大量类目, GS 是德国安全, RoHS 限有害物质, Reach 化学品, Battery 电池.

#### 38. Sales Tax (美国)
各州独立, Amazon 代征 marketplace 部分; 独立站需自行处理 (Avalara / TaxJar).

### 品牌与运营

#### 39. A+ Content
品牌注册后的增强 listing 模块. 7 个图文模块, 提升转化率 5-15%.

#### 40. Brand Analytics
亚马逊后台品牌洞察. 含 SQP / Demographics / Item Comparison / Repeat Purchase Behavior.

#### 41. Vine (亚马逊评价计划)
品牌备案后, 30 个免费评价. 大幅提升早期信任度.

#### 42. Manage Your Experiments (MYE)
亚马逊官方 A/B 工具. 测主图 / 标题 / Bullet 哪个改动最有效.

#### 43. PL / RA / WS
Private Label (自有品牌) / Retail Arbitrage (零售套利) / Wholesale (批发分销). PL 是主流严肃路径.

#### 44. OEM / ODM
Original Equipment Mfg (代工) / Original Design Mfg (设计 + 代工).

#### 45. MOQ
Minimum Order Quantity, 工厂最低起订量.

### 行业黑话

#### 46. 跑量 / 起量 / 衰减 / 死号
跑量 = 大销量但不一定盈利. 起量 = listing 销量开始增长. 衰减 = 流量下降. 死号 = 账户失活.

#### 47. 跟卖 / Hijacker
别人挂在你 ASIN 下卖货, 抢 Buy Box.

#### 48. 黑科技 / 测评 / 站内站外
黑科技 = 灰色优化手段 (刷单 / 测评 / 操控评分). 测评 = 找人买后给好评 (Amazon TOS 灰区). 站外 = 站外引流.

#### 49. DTC / D2C
Direct to Consumer, 品牌直营独立站.

#### 50. Long-tail vs Hero SKU
长尾选品 (100+ SKU 摊薄风险) vs 爆款选品 (1-3 个 hero SKU 撑起 80% GMV).

## 反复出现的争议

- **品牌 vs 套利**: 长期建品牌 vs 快速套利不同 SKU? 头部都建品牌
- **Amazon vs 多平台**: All-in Amazon vs 多平台分散? 越来越多人选后者
- **数据派 vs 直觉派**: 严格 A/B + 数据决策 vs 经验判断? 前者大势
- **AI 内容 + Amazon SEO**: 能不能用? **共识: 必须人工编辑 + 加真实经验**
