## 平台运营术语 (按使用频率)

### 平台 + 角色

| 缩写 / 术语 | 含义 |
|------|------|
| **ASIN** (Amazon Standard Identification Number) | 亚马逊每个商品的唯一 ID |
| **SKU** (Stock Keeping Unit) | 卖家自定义的库存单位 |
| **FBA** (Fulfilled by Amazon) | 亚马逊代发货 |
| **FBM / MFN** (Fulfilled by Merchant / Merchant Fulfilled Network) | 自发货 |
| **SFP** (Seller Fulfilled Prime) | 卖家自配挂 Prime 标 |
| **MCF** (Multi-Channel Fulfillment) | 用 FBA 库存发其他渠道订单 |
| **AWD** (Amazon Warehousing Distribution) | 亚马逊仓配 + 第三方分销 (2024-2025 普及) |
| **PL** (Private Label) | 自有品牌 |
| **RA** (Retail Arbitrage) | 零售套利 |
| **WS / Wholesale** | 批发分销 |
| **OEM / ODM** | 代工 / 设计 + 代工 |
| **MOQ** (Minimum Order Quantity) | 工厂最低起订量 |
| **DTC / D2C** (Direct to Consumer) | 直连消费者 (独立站) |

| 术语 | 含义 |
|------|------|
| **A9 / A10** | 亚马逊搜索算法 (A10 是业内非官方说法) |
| **BSR** (Best Sellers Rank) | 类目销量排名 (跨类目不可比) |
| **LD** (Lightning Deal) | 限时秒杀 |
| **BD** (Best Deal) | 7 天爆品促销 |
| **SQP** (Search Query Performance) | Brand Registered 卖家可看的真实搜索词数据 (2025-02 起 API 化) |
| **Search Funnel** | 曝光 → 点击 → 加购 → 购买四步漏斗 |
| **Honeymoon Period** | 新品上线 14-30 天的算法扶持期 |
| **Velocity** | 销量速度 (算法看「速度增长」, 不是「累积总量」) |
| **Buy Box** | 商品页面默认下单按钮, 跟卖时争夺 |
| **Hijacking / 跟卖** | 别人在你 ASIN 下卖货 |

## 合规框架 (按地区, 重要程度排序)

### 欧盟 (重灾区)

#### 1. VAT (Value Added Tax) 增值税
- 欧盟 27 国各自税率 (15-27% 不等)
- 不注册 → 平台冻结资金 + 罚金

#### 2. OSS (One-Stop Shop)
- 欧盟内 B2C 跨境一站式申报
- 2021-07 启用, 在 1 国注册, 月度统一申报全 EU 销售
- 适用 B2C 跨境年销 ≥ €10k

#### 3. IOSS (Import One-Stop Shop)
- 单笔 ≤ €150 货物的进口 VAT 简化申报

#### 4. VerpackG (德国包装法)
- **2019 起, 2022 扩展所有商家**
- LUCID 注册 + 加入双系统 (Duales System)
- 不注册 listing 第一天就被举报下架
- 罚款 €50000 起

#### 5. WEEE (Waste Electrical and Electronic Equipment)
- 电子电器废弃物指令
- 各国独立注册 + 回收义务履行
- 含电池产品强制 (Battery Regulation 2024 加强)

#### 6. Battery Regulation (2023/1542)
- **2024-08 生效**, 含电池标签 + 注册新规
- 适用所有含电池产品 (移动电源 / 玩具 / 家电)

#### 7. EPR (Extended Producer Responsibility) — 生产者责任延伸
- 法 / 西 / 德 / 奥 已实施
- 包括: 包装 / 电池 / WEEE / 纺织品 / 家具
- **2024-2025 法国扩展玩具 / 家具 / 纺织** 等强制

#### 8. CE 标志
- 欧盟一致性标识, 多类目强制
- 自我声明 (部分类目需第三方 testing — 玩具 / 高功率电气)

#### 9. UKCA (英国合规)
- 脱欧后, 部分类目从 CE 转 UKCA
- 自我声明

#### 10. GPSR (General Product Safety Regulation)
- **2024-12-13 实施** (重要)
- 必须有「欧盟境内责任人 (EU Responsible Person)」
- listing 必须列出制造商 + 责任人信息
- 高风险品类 (玩具 / 化妆品 / 电子) 触发更严合规
- **不合规 listing 可能被亚马逊批量下架**

#### 11. DSA (Digital Services Act, 欧盟)
- **2024 起** 平台传递: 卖家身份 / 联系方式公开

#### 12. EU AI Act
- **2026 完整实施**
- listing 用 AI 生成需标注

### 美国

#### 13. Sales Tax (各州)
- 大部分州 Amazon 代征 marketplace 部分
- 独立站需自行处理 (Avalara / TaxJar)

#### 14. SHOP SAFE Act (立法讨论中)
- **目标**: 平台对假货 / 不合规商品负连带责任
- **影响**: 平台对中小卖家审核可能更严

#### 15. Prop 65 (加州)
- 含化学品需警告标签

## 平台政策 / 标准 (重点)

### Amazon

#### Brand Registry
- 商标 + 品牌官网 + 主图带 logo
- 解锁: A+ Content / Brand Analytics / Sponsored Brands / Vine / 透明计划

#### Variation Policy
- 变体规则: 色 / 尺寸 / 不同款产品不能强行变体

#### A-to-z Guarantee
- 买家保护机制

#### 2024-2025 重要变更
- IPI 标准提高
- Vine 扩展 / 测评限制加严
- AI 生成 listing 限制
- FBA 入仓限制 (尤其旺季)

### TikTok Shop

#### 达人合作模式
- **Hands-Free**: 平台撮合, 自动结算
- **Targeted**: 卖家主动找达人
- **Free Sample**: 寄样, 达人决定是否带货

#### 小店等级
- 影响曝光权重

#### 2024-2025 重要变更
- 美国市场 GMV 快速增长
- 商家保证金 + 服务标准提高
- 半托管转半托管

### Walmart Marketplace

#### 卖家增长
- 2025 卖家 200k+ (前 5 月新增 44k)
- 95% 商品来自第三方 (Marketplace Pulse 数据)

## 外行破绽 (用了立刻被识别)

1. **把 ACOS 当 ROAS** (反向数字, ACOS 是费用率, ROAS 是回报)
2. **把 FBA 当 FBM** (物流模式分不清)
3. **不知道 Brand Registry 是什么** (没备案多数高级功能用不了)
4. **把刷单当合规营销** (Amazon TOS 严格禁止)
5. **不知道 GS / CE / WEEE 区别** (合规知识缺失)
6. **把 Brand Registry 简化为「上架就能用」** (实际要商标 + 验证)
7. **把跑量等同利润** (GMV 高不代表赚钱)
8. **把跟卖当合法** (亚马逊允许但同行不齿)
9. **把 OSS 当所有欧洲税务一键解决** (各国仍有本地法规)
10. **不知道 SQP** (Brand Analytics 已升级为 API 化, 不是手动下载)
