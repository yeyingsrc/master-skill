## 中文圈

### 12. 阳萌 (Steven Yang) — 安克创新 (Anker) 创始人 + CEO
- **背景**: 北大计算机系学士 + 德克萨斯大学硕士, 前 Google 高级软件工程师 (Founder's Award 获得者)
- **2011 年回国**: 在湖南创立海翼电子商务, 在加州注册 Anker 品牌
- **2025 重大动态**:
  - 完成品牌战略升级: Anker (智能充电) / eufy (智能家居) / soundcore (音频) 三大品牌矩阵
  - 提出「浅海战略」 — 深耕特定品类不追风口
  - 推出全球首款个人级 3D 纹理 UV 打印机 eufyMake E1, 创 Kickstarter 众筹纪录
  - 以 385 亿元财富列 2025 胡润百富榜第 157 位
  - 向港交所递交主板上市申请
- **核心方法论 — 「国际品牌打造三段论」**:
  1. **品类聚焦** — 不追风口, 深耕特定品类做透
  2. **本地化运营** — 真正了解目标市场用户的认知 + 习惯
  3. **研发驱动** — 研发投入 ≥ 销售收入 7%, 不是纯营销驱动
- **核心观点** (来自多次访谈):
  - 「跨境不是『把中国货卖出去』, 是『按当地消费者心智去做产品』」
  - 「我们做和自己禀赋相匹配的事 — 长期主义不是口号是商业现实」
- **争议**: 安克模式资本密集 (研发 + 品牌投入), 不适合中小卖家直接复制
- **来源**:

## 流派分歧 (基于上述 figures 的真实立场)

| 流派 | 主张 | 代表 |
|------|-----|------|
| **数据派 (亚马逊深耕)** | 数据驱动选品 + 高度系统化的 listing / PPC | Greg Mercer / Bradley Sutton / Tim Jordan / Liran Hirschkorn |
| **品牌派 (长期资产)** | 不在别人平台上盖楼, 自有品牌 + 独立站才是长期资产 | Steve Chou / Andrew Youderian / Eric Bandholz / Bill D'Alessandro |
| **教育 / 高端培训派** | listing 高阶 + 品牌差异化 | Kevin King / Steven Pope |
| **品牌出海派 (中国)** | 国际品牌三段论 + 研发驱动 + 本地化运营 | 阳萌 (Anker) |
| **运营实战派 (中国)** | 选品决定 90%, 运营放大空间 | 海猫跨境 / 跨境老炮 / 老魏 |

**核心分歧**: 「Amazon 是流量市场还是品牌孵化器」 — 决定是否值得投入品牌建设资源.

**2025-2026 共识**:
1. 跨境电商已结束疯狂增长期, 进入「质量 + 运营 + 合规」三高时代
2. 单平台押注风险大, 多平台分散 (Amazon + Walmart + Temu / TikTok Shop + 独立站) 成为头部共识
3. 欧洲 GPSR / EPR / VAT 合规升级是 2024-2026 重塑跨境格局的关键力量
