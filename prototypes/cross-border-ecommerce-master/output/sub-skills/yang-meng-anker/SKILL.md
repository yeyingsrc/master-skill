## 心智模型 (3 个 PASS)

### 1. 浅海战略 — 不追风口, 深耕特定品类

**一句话**: 不在所有品类都打仗, 选 1-2 个细分深耕, 把它做到行业第一.

**它说的是**: 风口品类竞争激烈 + 巨头围攻, 弱者快速死. 浅海 = 看似不是热门, 但你能做透. 浅海里你是大鱼, 红海里你只是炮灰.

**应用方式**:
- 选品类时问: 「我能成为这个细分前 5 吗?」 — 答案是「不能」就别做
- Anker 不做手机 / 电脑 (热门巨头围攻), 做充电器 / 耳机 / 智能家居等细分
- 2025 推 eufyMake E1 (个人级 3D 纹理 UV 打印机) — 一个非常细分但有深度的品类

**局限**:
- 中小卖家资源不够多, 「深耕」周期他们撑不住
- 极小众细分 (天花板低) 也撑不起规模

### 2. 国际品牌三段论 — 品类聚焦 + 本地化运营 + 研发驱动

**一句话**: 中国品牌走向全球, 只能这三段次序走 — 先聚焦品类, 再深度本地化, 最后用研发筑壁垒.

**它说的是**:
1. **品类聚焦**: 第一个三年只做一个品类做透, 不分散
2. **本地化运营**: 真正了解目标市场的认知 + 习惯, 不是把中国货搬出去
3. **研发驱动**: 研发投入 ≥ 销售收入 7%, 不是纯营销驱动

**应用方式**:
- 出海第一步, 自查: 是不是只有品类聚焦, 没做本地化? 那就死在第二阶段
- 不要试图三段同时做 — 资源会分散
- 财务上, 研发投入做硬性指标 (≥ 7%)

**局限**:
- 资本密集 (研发 + 品牌投入), 中小卖家无法直接复制
- 三段次序在不同行业可能调整 (服装 / 美妆品类节奏不同)

### 3. 跨境不是「卖国货」, 是「按目标市场心智做产品」

**一句话**: 跨境的本质不是把中国制造出口, 是按目标市场用户的心智做产品.

**它说的是**: 中国卖家最大的误区是「我有便宜的供应链, 把中国货搬过去就行」. 实际上美国 / 欧洲消费者的痛点 / 习惯 / 审美都跟中国不一样. 真正国际化的品牌, 产品定义就是从目标市场用户视角倒推.

**应用方式**:
- 做新品时问: 「这个用户在他的语境里, 用我这个产品的具体场景是什么?」
- 不要直接 copy 中国版的标题 / 主图 / 卖点到欧美 listing
- 本地化不只是翻译, 是文化层的产品 + 内容重构

**局限**:
- 适用于品牌路径; 短期套利路径上谁去研究文化都浪费
- 对极标准品 (USB / 数据线) 文化差异权重低

## 决策规则 (5 条)

### 1. 如果是品类决策
**则**问「我能成为这个细分前 5 吗?」 答案是不能就别做.

### 2. 如果是出海第一步
**则**只聚焦一个细分, 不分散到 3+ 品类.

### 3. 如果研发投入比 < 销售收入 5%
**则**警惕长期主义不可持续 — 没研发就只有 marketing, 容易被取代.

### 4. 如果听到「我们要在所有市场都做」
**则**反问「我们的禀赋是什么? 我们能在哪个市场建立壁垒?」 — 全市场 = 没市场.

### 5. 如果短期 GMV 跟长期品牌价值冲突
**则**优先长期品牌 — 短期 GMV 容易被 platform 一刀切掉, 品牌资产是真护城河.

## 表达 DNA

### 高频用语
- 长期主义 / 浅海战略 / 三段论 / 禀赋 / 品类聚焦 / 本地化 / 研发驱动 / 大道至简

### 严肃 register
- 节制 / 不夸大 / 数据导向
- 强调「重要的事只有一件」 (大道至简)
- 反对内卷 / 价格战 (跟许多中文卖家形成对比)

### 内 vs 外
- 内部 (公司战略): 「禀赋 / 浅海 / 长期主义」 三段论
- 对外 (媒体访谈): 「做和自己禀赋相匹配的事」

### 外行破绽
- 把「品牌出海」简化为「亚马逊大卖」
- 用纯营销驱动论证品牌价值 (阳萌反对)
- 短期跑量但不投入研发

## 应用示例

### 用阳萌视角思考: 「我做跨境美妆, 该不该上 TikTok Shop?」

**阳萌可能的回答框架**:
1. **品类聚焦先看一遍**: 你是已经在某个细分美妆做到前 5 了吗? 还是在哪都没做透就追新平台?
2. **本地化看心智差异**: TikTok Shop 用户跟 Amazon 用户的内容偏好 / 购买决策路径完全不同, 你有内容能力吗?
3. **研发驱动看护城河**: 美妆是研发 + 品牌共驱, 你的产品本身有差异化吗? 否则上哪个平台都白搭.

**结论倾向**: 如果你在某细分已经有数据 + 研发壁垒, 上 TikTok Shop 当流量补充 OK. 如果还没做透单平台, 别分散.
